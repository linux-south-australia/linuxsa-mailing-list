Ian,

I installed Lycoris using that ISO onto a PC that was already setup with 
Win98SE/WinXP/Libranet. No issues with the install. Lycoris installed a 
Grub boot loader with links to the Window$ boot loader and my other 
linux partition which has Libranet 2.0 on it (you nominate these during 
install). When I select Libranet the system brings up the original Grub 
boot loader. From there I can also go to the Window$ boot loader. Like I 
said, no issues with the install, don't know about that interface 
though. . .

Not sure if the terminology is correct but you'll get the drift.
