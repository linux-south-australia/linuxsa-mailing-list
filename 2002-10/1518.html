Did you know that the KDE printing manager has an 'add special printer' 
option in the dialogue that allows you to set up a PDF-creating printer 
for an automated operation?

postscript data into batch-mode ghostscript and dumped the resulting PDF 
into a shared folder for windows clients that used the 'Apple 
laserwriter II-NT' driver.
