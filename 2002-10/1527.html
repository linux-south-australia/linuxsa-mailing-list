Re: VLAN question

Wilkinson,Alex wrote:

>Are there any Unix tools to find out (query a switch) for what VLAN a Unix host
>is in ?
>
Telnet is probably the most useful command in this instance. When you're 
'inside' of a vlan on a host, there's very little direct evidence to let 
you know that you're a part of one.

On a Cisco switch, you would use 'show vlan' to get a list of vlans and 
the interfaces configured to use them. Then if you don't know what 
interface your unix host is in, you can use a 'show mac-address' command 
(depending on the type of switch) and see which port entry matches the 
arp table for the known IP address of the host.

Like telnet, you might also be able to get some information from the 
switch via SNMP if you have a layer-3 path to the configured IP address 
for the switch.
