The CDs are organised so that the most common apps are on earlier disks. 
You _might_ be able to get away with 1. I'd be reasonably confident that 
3 would be ok for what you want, but to be sure, I think there are lists 
of what packages are on the CDs online, but I can't find them ATM. The 
other option is to get the minimal install iso / floopy images 
(depending on drivers I think the minimum you need is 2-4 floppys 
worth), and let debian download the packages you install on the fly.

see http://www.debian.org/CD/ for more info.
(the FAQ here suggests you probably only need the first 2 CDs)
