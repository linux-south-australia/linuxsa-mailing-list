> Internode has their own time server?  So it will use that as primary and

Actually, to get that behaviour you should use the "prefer" option, 
which does what you describe:

server ntp.internode.on.net prefer
server ntp.saard.net

It's probably a good idea to set up access control, and decent logging, 
so on as well. I've yet to get around to doing that. :(
