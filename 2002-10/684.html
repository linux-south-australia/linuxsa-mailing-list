DNS and unknown host

Hi,
I have a private network and a RedHat 7.3 machine running Bind as a
nameserver.
My configuration of Bind provides the correct name resolving for all the
machines on my LAN except the nameserver itself.

for instance; on any other machine on the network I can use 'ping a' and it
will correctly ping a.mydomain.mynet

In resolve.conf on all machines I have set the search path to mydomain.mynet

But on the nameserver, when I use 'ping a' it reports "ping: unknown host a"

If I run 'dig +search a' on the nameserver it will resolve the correct IP
address.

If I run 'ping a.mydomain.mynet' on the nameserver it will resolve the
correct IP address.

I have tried changing the resolve.conf entry from search mydomain.mynet to
domain mydomain.mynet without any luck.

Any suggestions for how I might debug this?
