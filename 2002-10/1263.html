RE: Mandrake on APC Mag

Works fine for me - both on my laptop and desktop machine - in my opinion
Mandrake is perfect for beginners.  Please consider subscribing to their
Mandrake Club or purchasing something from them if you can too.

Just be careful with the security settings - setting it one inch higher than
you need can cause all sorts of dramas - like sound not working, being
locked out, etc.

And if you have an Nvidia based card - make sure you install the Nvidia
Linux drivers ASAP for kick butt graphics (Quake3 runs MUCH MUCH smoother on
my Dell laptop @ 1600x1200 resolution than under XP!!!! The graphics are SO
smooth!)

And also INSTALL ALL THE GOODIES off the CD's - there is STACKS of great
stuff not installed by default - so go the individual package selection
route.
