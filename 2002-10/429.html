Re: New Server

Quoting Troy Davis <tdavis@comstech.com>:
> I am about to build a server for web, mail and DNS using Red Hat Linux
> My Questions are...
> 1. Should I go buy the greatest P4 1.8gig system or stay lower end say
> celeron
> 2. What main board would people find they have least hassles with for a
> Intel solution
> 3. Red Hat 8 seems to be much better for desktop use, Would it also be
> better as a server than say Red hat 7.3

1. Do you want to future-proof your investment?  Buy the best you can 
(especially I/O if it's going to be a server).
2. RH 8 has several different "types" of installation.  One of them 
is "desktop", but another is "server".  I would be installing the latest (RH8) 
as it'll have the latest software => no patches (yet) to be applied for 
security.  If you go with RH7.3 (or any other distro released > 6 months ago), 
you'll have quite a bit of patching to make it secure.
