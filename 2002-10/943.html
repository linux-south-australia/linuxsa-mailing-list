Re: [OT] Watchdog eyes broadband limits

Richard Russell <richard@yellowgoanna.com> wrote:
> We need a second protocol (for both email and sms) that includes some
> garantee of delivery somehow...

That's not possible.  Proof by counter-example: You send me an SMS but
my mobile burns before it arrives, and I don't replace it.

An achievable alternative is some sort of receipt that guarantees the
user has read it.  We've long had delivery receipts, but that's not
good enough.  Microsoft mailers can also give read receipts, which I
think is a darned good idea.  All mail readers should honour that.
