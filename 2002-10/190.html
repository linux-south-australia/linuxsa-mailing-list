Re: Contributing back to the open source community

On Thu, 3 Oct 2002, Mcleod, Ian wrote:

> True - I always make a point of running talkback on Mozilla - both on Win XP
> and Linux and providing feeback wherever I can.

Funny you should mention that. Along these lines the other night I partly
finished a HOWTO document aimed at helping people produce better feedback,
including debugging and trivial fixes to stuff. Someone who has doodled
with hundreds of packages at the ./configure level and traced through
complicated log files and crash records can handle C if they stop and
think about it. But a lot of really competent Unix users never quite get
to the point of using the code they have available. I'm thinking of
calling it HOWTO Use the Source.

I know there is a big difference between a monster package like Mozilla
and some little commandline utility, but still there are quite a few ways
of using the source to good effect.
