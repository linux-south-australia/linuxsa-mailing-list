Re: booting Linux

Hmmm, or since you are using Mandrake 9.0, boot in with the boot disk, then 
run Mandrake Control Centre (should be under the main app 
menu/configuration).  Click on the Boot icon on the left, then Boot Config in 
the middle of the icons on the right.  Then run the Lilo/Grub setup wizard.

Once the wizard is finished, you can look at the lilo.conf file in /etc, just 
to see what's in it or to customise it further.

Give me a hoy if you have any problems with it.

Cheers
