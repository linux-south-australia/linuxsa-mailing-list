Agree, PowerPoint is a real PITA for incompatibilities. I have built 
presentations expecting that files and fonts might not be on the target 
machine and still had it mess up my links and bullets.

It is a cow of a program and so inconsistent. Now I make sure I carry 
copies of particular fonts if I need them. I also burn a copy of the

presentation.

my presentation and how it comes out when run.

Then I can happily use Impress and blow MS away <sigh>
