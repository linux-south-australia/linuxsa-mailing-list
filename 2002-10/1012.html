Re: Email address via uid...?!?!

Andrew Galdes wrote:
 
>This is how it goes. A user on a windows system connects to a share on
>samba. Samba knows the uid of this person now (from the %u variable). Samba
>now needs to send an email to this user who has an account on the exchange
>server. Samba needs to know what the email address is which corisponds to
>the uid it has recieved in order to send that user an email. 

Exchange can be an LDAP server. So long as there is some common key you
should be able to pull out the email address using the commandline
OpenLDAP tools. Maybe you have unique common keys in the Unix 'full name'
field, or maybe you can teach the Exchange LDAP server about the uids you
are using on Unix, or maybe it already knows about the uids because you
have a common username scheme across all machines in the company.
