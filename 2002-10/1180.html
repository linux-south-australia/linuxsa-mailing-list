Linux user group announces installfest

October 21 2002

The Linux Users of Victoria will hold their seventh Linux Installfest on
23 November, according to a media release.

An installfest is an event to which people can take their computers and
receive help to install Linux, or obtain assistance with the
configuration of an existing Linux system.

LUV media spokesman Graeme Cross said a wide range of Linux
distributions, including Debian, Red Hat, Mandrake, SuSE and Gentoo,
would be supported.

Mr Cross said there would be demonstrations of Linux running a variety
of software and presentations on what Linux is, so people curious about
Linux could experience it first-hand.

The installfest will be held at Strathcona Baptist Girls Grammar School,
Claremont Crescent, Canterbury (Melway map reference 46 D12) between
10am and 4pm.

LUV has set up a Web pag
(http://www.luv.asn.au/installfest2002/register.php3) to provide more
information and allow people to register online.

Melbourne Wireless will join the LUV event for their first wireless
installfest and antenna field day. More information is available at the
Melbourne Wireless Web site (http://www.wireless.org.au/wiki/?installfest).
