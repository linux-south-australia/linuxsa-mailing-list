Re: NTP Servers

Mcleod, Ian wrote:

> How can one run a time server on a gateway to sync machines around the 
> house
> to one ntpd synced clock on the gateway?  Say with Mandrake 9.0 if
> possible..


Run ntpd.

Install the ntpd package (it's a core part of FreeBSD, under Debian the 
package is called ntp-simple) for Mandrake.

Set up a config which looks somthing like (for me, an Internode customer):

#
# /etc/ntp.conf
#
#
server ntp.internode.on.net
server ntp.saard.net
#
# driftfile
#
driftfile /etc/ntp.drift

On your UN*X workstations, do the same, but use a ntp.conf file which 
looks the same but points to your new ntp server.

And that's all there is to it, or at least, that's how I did it."I'd 
rather be anywhere, doing anything"
