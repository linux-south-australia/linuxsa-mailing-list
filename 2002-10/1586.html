Completely agree, ECC for absolute mission critical stuff (I don't know
how true this is, but I've been told that once you start getting too far
over 1GB of memory, one needs to use ECC or thing become unstable...) 
ECC *is* over-the-top for your average desktop machine.

Some Intel boards (mostly their really expensive server varieties) will
not accept memory unless it is ECC and registered.  Obviously, this
isn't the case here - exchange the memory for normal RDRAM if possible,
but chances are it was ordered in specifically and is not returnable.

Jordan.
