Hello,

You should modify the file "/etc/lilo.conf" to make sure that it is looking
at everything it should. -That it knows what OSs are installed to which
partition.

#vi /etc/lilo.conf
#lilo

(note the use of vi :)

Lilo is mostlikly installed but not configured (as a boot loader should be
configured...)
