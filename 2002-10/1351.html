Yes, INSTALL FROM SOURCE on this one.

I have a fairly recent card (Gf4MX-440) with a Nvidia
chip that was cheap, fast at Quake3 but lacking new
technology like GPU programming[1]. Once you have the
framebuffer and X libraries from Nvidia installed FROM
SOURCE you can begin coding 3D of your own in a fast
and at least open and helpful if complex environment.
If you do it carefully, you can compile this code on
MacOS-X with /no change/. MacOS developer tools are
sweet. It's possible to get it to build on windows
too.

Most of Nvidia.com is aimed at 3D Hackers or at least
the level of 3D gamers. They have great slide shows, 
speed vs other resource benchmarks, stuff like that.
There are plenty of other online resources that have
things like technical papers, source code and forums.

[1] The technology that these cards are lacking is one
of the more active area of 3D graphics at the moment.
If you go by the (PC Centric) $generation = "John
Carmack's occluded space 3d engine major release." it
won't be so good for next $generation based games.
