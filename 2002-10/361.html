Re: KDE 3 with RedHat 8.0

This is in /etc/X11/gdm/Init/Default:

#!/bin/sh
# Xsetup for Red Hat Linux
# Copyright (c) 1999, 2000 Red Hat, Inc.

#/usr/X11R6/bin/xsetroot -solid "#356390"
/usr/X11R6/bin/xsetroot -solid "#000000"

/sbin/pidof -s kdm &> /dev/null && [ $? -eq 0 -a -x /usr/bin/kdmdesktop
] && /usr/bin/kdmdesktop && sleep 1s

# only set the background if the xsri program is installed
if [ -x /usr/bin/xsri ]; then
    /usr/bin/xsri  -scale-height -center-vertical -center-horizontal
-keep-aspect \
	/usr/share/pixmaps/redhat/customised.png
    #/usr/bin/xsri  -geometry +5+5 -avoid 300x250 -keep-aspect \
	# /usr/share/pixmaps/redhat/redhat-transparent.png

--> take special note of "only set the background...

This is in RH 7.1 but I don't think they'd be too different. xrsi might
not be behaving itself of the png at the end (my customised.png is just
as it says -- customised) might not be there. Yours will probably look
similar to the two last hashed out parts.
