* One important thing I forgot about the ps files, there are a series of 
 page numbering lines in a postscript file that tell Ghostview or 
Acrobat Reader how many pages are in the document and what page number 
you are at.

You need to determine how many pages of postscript you have combined, 
and number each page like so:

Page: 1 11

All this means is Page 1 of 11 Pages.


Page  2 11

etc.

You can search for ----->    Page: with emacs.  


I should have mentioned that it is most useful when using Wine 
applications or non printable hard copies that need to be scanned, but 
the GIMP couldn't convert the scan to postscript.

I had to do all of that today because we had to deliver a set of 
documents to a client in PDF, that was originally made up of some 
documents that we could print to postscript, and some had to be scanned.

After I scanned the docs and converted them to ps, I had to manually 
combine  the ps files by using Emacs.  We have Adobe Acrobat 5, and it 
is able to PDF files. Unfortunately, the PDF file lost its formatting.
