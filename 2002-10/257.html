Re: Open Source Distribution Service

Brian,

> My question - Is this a Red Hat Open Source Distribution Service ?
> or are others allowed.

This would be up to the open source community as a whole.

I would urge all distributors to consider distributing anything that has
an open source approved licence. This licence would include:

* GPL
* Apache Licence
* X Licence
* Perl Licence
* Artistic Licence
* BSD Licence

I think what will limit some distributors is bandwidth and time, however
if we get a suitable network up we should be able to distribute almost
anything within reason.

Notice I have said "urge" not "tell" - I only intend to facilitate this
little venture :-)
