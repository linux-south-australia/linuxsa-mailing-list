AFAIK, it doesn't. the PostScript format was only designed to send 
info to printers so it only describes, fonts, colours, images, vectors 
etc (Why would a printer need to know what a hyperlink is).

With PDF, it's sort of pushing PostScript to it's limits so any of that 
stuff needs to be done POST-PDF'ing.

This is just in my experience so if anyone can prove me wrong,
