Re: New Server

When it comes to building a server I would not even consider ATA.  The fact
that most SCSI drives come with 5yr warrenty vs 6months/1 year for OEM IDE
drives.  IDE utilise cpu for IO, SCSI have built on controllers, SCSI has
known and grown defects lists in the firmware .. etc etc.  When it comes to
protecting data and ensuring reliability, the extra money for a SCSI based
system is justified IMHO.

If this system is going to be just for DNS, two 4.5 GB drives in RAID 1
would be more than plenty.

I have an old DPT PM1554 budget raid card in my SuSE server with 2 18GB 10K
Atlas drives and have benchmarked it (with mySQL?) faster then a very
expensive dell server my mate was building for a large business as a
database server.  My system is a p3-600/128, the dell server was a dual
p3-1.13GHz/512.

Just my 2c worth.

Mick Campbell
