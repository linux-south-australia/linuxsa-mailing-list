linux on compaq dl360 kernel panic

If anyone tries to run Red Hat 7.3 on a compaq dl360 G2 and gets the
following kernel panic:

ServerWorks OSB4 in impossible state

when mounting the cdrom drive AFTER installation, the solution is to turn of
DMA for the cdrom before mounting it.

Eg hdparm -d0 /dev/hda

Passing nodma on the kernel parameters line in lilo / grub won't work for
Red Hat 7.3 since the /etc/sysconfig/harddisks script turns it back on.
You'll need to make a script for the cdrom drive and disable dma -
instructions are in the /etc/sysconfig/harddisks script.

Hopefully this saves someone else some time.
