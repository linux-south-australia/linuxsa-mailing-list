RE: Lets pass the hat for Linux - about time..

Not sure if this has been mentioned already in this thread...

In the latest Linux Journal, there is a letter to the editor suggesting the
Open Source community adopt a fund raising method that the NRA uses. This
person suggests that when someone orders something such as a linux distro,
tux coffee mug etc, the vendor asks if the customer would like to add $1 to
the order as a donation to something such as the FSF. The vendors then send
the collected money to the charity / foundation / project.
