Peter,
 
Did you download the binary or the generic package with source? I have a
connexant HCF 56k modem and on RH7.3 I found that the binary package
didn't work for my particular setup. If you get (or have to get) the
source package you will also need the Kernel source for your system as
the package compiles as a Kernel module.
 
But anyway, get the rpm and install it. Then run /usr/sbin/hcfpciconfig
and all should be well.
 
I will provide any further help if you need it.
 
John Rayner
