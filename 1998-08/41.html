Re: binding mulitple ip address to the one interface

> how do you ?

If you're using Red Hat Linux, load up the network configurator, click
on interfaces, pick your device, and click on alias.  Take a look in
/etc/sysconfig/network-scripts to see what it did.

If you're not using Red Hat Linux, you can do 'ifconfig eth0:0 ...'
for the first alias, :1 for the second, etc.

The above presumes you have already compiled in support for network
aliasing (Red Hat's kernel includes it).
