Re: printing man pages

Ben 
man cat|lpr
       ^
       this is the pipe character
       lpr is an interface to the print spooler
       this prints the man page for cat on the default printer
you can pipe any thing that appears on the screen (standard output)
to lpr, which reads standard input if you don't give it a file name

caveat:
 some things on the screen are from standard error not
 standard output so to get them printed as well you need to 
 redirect error to standard (2>&1 in bash) so the errors also get piped
viz:
 > [root@grissom scripts]# rpm -V fred | lpr
 > package fred is not installed
 > lpr: stdin: empty input file
 > [root@grissom scripts]# rpm -V fred 2>&1 | lpr
 > [root@grissom scripts]#
 the last command printed a page with just one line - 
    package fred is not installed
