for a better explanation, netscape is an X-windows based program. That is
it needs a windowing environment with full widget set to run. If you are
running X (via startx or xinit or xdm [each has it's own uniqueness and
benifits]) then as the privilaged user (read the user running the client
or root) then you can run netscape from Virtual Terminal(VT) provided the
DISPLAY environment variable is set to the correct xserver (usually via
export DISPLAY="localhost:0.0") or by passing what would be the DISPLAY
vaule as a parameter (i,e. netscape -display localhost:0.0 &). Alternatly
once in X you can run an xterm and just run netscape there as part of it's
initiation is to set the DISPLAY env. var. to the value of the controlling
server.
