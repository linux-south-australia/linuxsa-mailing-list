On Fri, 21 Aug 1998, Robert Mibus wrote:

> is there any way to watch what is going through my modem when using usernet/
> nettools to run a ppp connection?

You can monitor the connection process by having "debug connection"
selected in the ppp setup, and then watching the log file as you attempt
to connect using tail -f /var/log/messages (need to be root for this).
