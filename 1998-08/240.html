Re: finding dependencies

> I am having a few problems installing some RPMs. When i try to install or
> upgrade some files, it tells me I need some other package, so i obviously
> try to install that one.
> 
> Now, there's one that got a failed dependency, and it says:
> 
> libXpm.so.4 is needed by package-whatever-is-was
> 
> anyway, I tried for a while to find out how to get this information, however
> I was unsuccessful. I managed to get a list of those things the RPMs on the
> CD provide, but it didn't tell me which file provided it, and I couldn't
> find a may to single out the one i needed.
> 
> could someone tell me how please? :)

(insert Red Hat CD)
% mount /mnt/cdrom
% cd /mnt/cdrom/RedHat/RPMS
% ls -la *xpm*
-rw-r--r--   3 root     root       29598 May 11 17:58 xpm-3.4j-2.i386.rpm
-rw-r--r--   3 root     root      143422 May 11 17:58 xpm-devel-3.4j-2.i386.rpm
% rpm -qp --provides xpm-3.4j-2.i386.rpm
libXpm.so.4

I hope the above helps?
