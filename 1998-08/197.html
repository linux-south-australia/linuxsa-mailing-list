Re: Customised msg on ftp

Hi there,

you wrote:

> I've created a file ftplogin.msg in /etc 
[snip]
> and added the following line in 
> ftpacess:
> 	message /etc/ftplogin.msg            login
> 
> but nothing seems to happen, ie the ftplogin file was not shown when a 
> user is connected.

When an anonymous ftp user logs in, their root directory is set to the
home directory for security reasons.  Therefore, if you write
"/etc/ftplogin.msg", that really means "/home/ftp/etc/ftplogin.msg".
Change the line to:
message /ftplogin.msg login
then put the file in /home/ftp and it should work.
