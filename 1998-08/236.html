Re: 1:modules,2:inet

> since upgrading from rh5.0 to 5.1, when loading, my computer says:
> clock: modprobe: can't locate module char-major-10
> 
> also, when loading network stuff at boot, it says 'can't find module =
> net-pf-3'
> (and net-pf-4)
> 
> any ideas?

It sounds like you're not running the kernel that comes with 5.1.
char-major-10 would probably be referring to /dev/rtc, the real time
clock device.  net-pf-3 and net-pf-4 are the amateur radio and ipx
protocol families respectively.  The Red Hat Linux 5.1 kernel should
include all three of these devices.  If the messages trouble you, the
solution to get rid of them has already been posted.
