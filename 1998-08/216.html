Hi Steve,

You can do this on sendmail configuration with "generics".  You could visit:
http://www.sendmail.org/virtual-hosting.html
for more information.  Point 6 has information on how to set up generics
support.

You will probably have to install (or get) sendmail source to get hold of
the original configuration files (*.mc files) and add the "FEATURE" stuff
and then run through m4 to get a sendmail.cf.  Information on how to do this
is available at:
http://www.sendmail.org/m4/readme.html

You will probably not have to recompile sendmail as this is just
configuration stuff.
