Re: Clever things with a Linux Quake server.

On Thu, 6 Aug 1998, Dave Lawrence wrote:

> I have a quake server on a LAN, named 192.168.0.42 and I want people to
> be able to connect to say quake1.lan.net How do I do this? I assume it's
> just a matter of summoning some daemon.

If my assumption that the clients are Windows boxes is correct, the
easiest way is to go to each one and create a file in the c:\windows
directory called HOSTS and put something like this in it:

192.168.0.42	quake

Then they just connect to hostname 'quake'. Easy, no messing about with
DNS and the like.

This is my recommended solution because I don't understand DNS and the
like.

:-)

Test it works by creating that file and typing ping quake. Should get
the usual 4 replies. And in a first for Windows, you don't have to reboot
for it to recognize the changes. Woohoo!

See also the file c:\windows\hosts.sam
