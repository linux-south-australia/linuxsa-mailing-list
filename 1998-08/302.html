Hi Richard,

This filter uses ghostscript to talk to a laserjet.

If you need any hints, feel free to ask.

Cheers,

ig


#!/bin/sh
#This File by Todd Burgess (tburgess@uoguelph.ca)
/usr/bin/gs -q -r300 -dSAFER -dNOPAUSE -sDEVICE=ljet2p -sOutputFile=- -
