Most recent distributions of Linux use PAM. All security like logins,
permissions, the whole lot should go through pam. Therefore, by using
something like PAM_LDAP, you can authenticate against an LDAP directory.
Also grab NSS_LDAP. It allows you to use ldap with the services in
/etc/nsswitch.conf. You need it for PAM_LDAP, but it means you could
keep quite a few of your /etc/* files in an LDAP database.
