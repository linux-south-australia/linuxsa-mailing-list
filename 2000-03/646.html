Re: logging

Nick Stock wrote:

> Hi all,
>
> I want to capture some log files and effectively 'tail -f' them
> to the /dev/tty1x
>
> How do I do this and which file eg, /etc/rc.d/rc.local do I
> add the commands to?

Add a line to /etc/syslog.conf. eg if you wanted to capture everything
which goes to /var/log/messages, take the line which looks like :

*.info;mail.none;news.none;authpriv.none             /var/log/messages

and change it to:

*.info;mail.none;news.none;authpriv.none             /dev/tty1x

Then restart syslog, and you're away!
