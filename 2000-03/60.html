Re: Recovering lost files

> Accessing a Linux box across the network, as a user, accidently deleted some
> files - you guessed no backups. Is there any way to recover them?

Not from hard disk.

You could search the hard disk to find all the data blocks, if you can
recognise them, but that's not really practical unless the data is
immensely valuable.  Don't write anything to your hard disk until after
you have accepted that the file is lost.
