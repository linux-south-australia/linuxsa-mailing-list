 All fixed. I was running RedHat 6.1 and I never had the same problem on
 my old laptop with 6.0, so I replaced the kernel-pcmcia-cs rpm with the
 RedHat 6.2 beta one. It doesn't replace any kernel modules, just the
 card services stuff. It now works like a bought one.
 
 Someone else (Bruce Moffatt?) was having problems with the internal
 modem working with an ethernet card plugged in? I can be transferring
 stuff across my LAN, while dialling out with the Internal modem (I
 haven't tried forwarding). I've even got the maestro sound driver
 working at the same time. Oh yeah, and Windows 98 was never booted and
 there aren't any non-linux partitions. A truly marvelous laptop.
