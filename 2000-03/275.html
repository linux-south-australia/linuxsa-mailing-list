In concept: LDAP is a read-optimised database that has a defined network
interface. You don't know how to get to an SQL database without more
information (like how to talk ODBC to it for example), however all you
need for an LDAP database is an IP address, just like a web server.
There's lots more to LDAP (I've got a 650 page reference manual.)
