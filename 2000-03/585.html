Re: Asset tracking software.

> 
> Would some one know of code that may do similar to the following. 
> 
> Ping a range of ip-addresses and get the computername of a workstation.
> search an input file for that computername then get the serial number. 
> Connect to that workstation and get the last login of that workstation.
> Output the ip-address, computername, serial and last user to a text file.

nmap, nmblookup (from samba) and a script should be able to do this.

Basically ping a host and resolve it's IP to name.  If you are using WINS
records in your DNS you will have the Machine name.  From there you can
do an nmblookup on the IP address and specify the machine name or the domain
and retreive all the names the windows gives that system.

That should be enough to get you started.
