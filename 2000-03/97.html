Re: modem

On Mon, 6 Mar 2000, Galdes, Andrew (PIRSA) wrote:

> Im after advice on a good modem. money is an issue but so is performance. i
> am looking at Diamonds Supra 56e/pc. 

If you want the modem to run under Linux then you must make sure you don't
get a `win modem`. These are modems which use the cpu and ram of the
computer rathern than the modem doing the work itself. There are a few
which will work under Linux but I'd try an external as it's 99.9%
guarenteed to work.

I recommend you read: 

http://www.o2.net/~gromitkc/winmodem.html 

before you buy a modem. There is a number of Supra modems listed in the
table from this page.
