Re: "Web Folders"

> Just a wild stab - but does anybody know exactly what makes up a
> "Web Folder" under Microsoft's Office package?  Or even whether it's
> the Office package that installs Web Folders support?  What protocol
> is being used?
> 
> I was assuming it was just HTTP... but I haven't made that work.

I'm pretty sure you mean WebDAV, which is the standard replacement for the
hacks used by programs like PageMill and FrontPage to write back to web
servers. Look in the refs given in http://linuxcare.com.au/~dan for the
RFC and related pointers.

> Linux connexion:  I want to know if I can utilise them in any way by
> using a Linux server.  It could come in handy to people, and me.

The Front Page Server Extensions for Apache (as implemented either as a
really buggy and insecure package by, yes, Microsoft or as implemented by
Mrs Brisby - see previous URL) was the old way to do this with MS
FrontPage. Using the mod_dav Apache module you can use this with the
Windows webtop thingy, late versions of Frontpage and anything else that
implements the WebDAV protocol.
