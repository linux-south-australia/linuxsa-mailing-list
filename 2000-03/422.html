router and firewall for 200 machines. It is an AMD K6-2 300 with 128MB
RAM and 8GB HDD. It has a 50MB RAM cache and 500MB disk cache. We have
never seen the machine stress. Not even when I compile the latest Samba
CVS ;-). It's running squid 2.2 and will be an LDAP server and use that
