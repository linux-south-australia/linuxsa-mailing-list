Java

Hi All,
    Last week (maybe the week before - I forget) there was a
discussion about Java and JDK. I remember suggesting to try
a commercial version by Borland (Inprise) called J Builder
3, and that apparently there was a Linux version available.
Well there IS a Linux version available, AND IT IS FREE
(well for the cost of the download time, or a British
magazine). I discovered a British magazine called PC PLUS
has this month (April) on the cover disk a copy of J
Builder3 for Linux (and Win NT). You do need to obtain a
copy of J2SDK from Sun still (it's free), but there is a
fully integrated development environment (like Visual C++ or
Delphi) included as part of J Builder. I was so impressed
that I DID buy the software, and it IS worth it. The
magazine was just on the shelf at the news agent and was
$19.95. So for those interested, give it a try, you will not
be disappointed.
    Cheers!
        A.
