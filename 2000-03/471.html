Re: /dev/fd0 again

On Sun, 19 Mar 2000, rob wrote:
> okay ,
> 
> created fd0 using makedev - no problem , can write to the floppy , but
> when i try to mount the floppy it wiil not let me . ie  (root) mount -t
> ext2 /dev/fd0 /mnt/floppy

mkdev should not have been necessary

"mknod /dev/fd0 b 2 0"

do an ls -l /dev/fd0. it should read
brw-rw-rw-   1 root     root       2,   0 Apr  4  1999 /dev/fd0  

if 'b' isn't there, delete it and start again. Then;

mkdir /mnt/floppy
chmod 666 /mnt/floppy
chmod 666 /dev/fd0

because you already have an entry in fstab all you need to type is

mount /dev/fd0   OR
mount /mnt/floppy

Naturally. an ext2 formatted disk is expected to be in the drive
