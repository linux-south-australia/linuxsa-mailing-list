converting a binary number to string on command line

Hi,

I just cannot think how to do this at present.

I am investigating the creation of FAT32 partitions (into which one would
restore Win98) from Linux.

mkdisfs can make one, but to make it bootable requires a bit more work.

The boot sector(s) have to be written to the partition.  This is easy.
There are three boot sectors, and one can simply copy them from an existing
bootable Win98 partition:

   dd if=/dev/hda1 of=fat32.boot bs=512 count=3

and then copy them back again:

   dd of=/dev/hda1 if=fat32.boot bs=512 count=3

But wait.  There is a backup boot sector(s).  The location (sectors) of the
backup boot sector(s) is given at byte 50 of the boot sector.  It is a
16-bit quantity.

I can get the location out easily:

  dd if=/dev/hda1 bs=1 count=2 skip=50

and it is output on stdout, but as a binary number.  I want to get that as
a sting and be able to write to the correct location.

So, how do I do it?
