Re: WHICH JDK

Evan Bourlotos <evan@cs.adelaide.edu.au> wrote:

>I am totally confused atm I want to get the JDK I have RH 6.1 but there
>seems to be a version of jdk for everyone and his dog. How do I find
>which one will run on my version of Linux

At this page
  http://www.rpmfind.net/linux/RPM/JByName.html

You'll find a link to this rpm which works OK on my RH6.0 system
   jdk-1.2-prev2.i386.rpm
(there may be a more recent version, I haven't looked in a while)

While you're there, get a recent rpm for "Jikes" which is an IBM
bytecode compiler to use instead of javac.  The first time I tried
Jikes, it compiled my code so fast that I thought it didn't work.

I'm using jikes-1.06-1.i386.rpm   (701k)
