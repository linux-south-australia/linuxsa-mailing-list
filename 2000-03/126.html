>I downloaded the all the files for Mandrake distribution for linux from
>ftp://mirror.aarnet.edu.au/pub/linux/mandrake/7.0 and copied them onto a
CD.  I
>used the rawwritewin program in the dosutils directory to create a boot
floppy
>from windows.  This boot disk starts up the linux installation but the
>installation stops since the files on the CD rom is not recognised as a

Hmmm, usually, you copy the ISO image, not the individual files, but I bet
the UNI does not like you copying a 500-600 MB image :-)

You will have to make sure that you have all the files, and then use a
Linux system, or some other system with the mkisofs command, and make an
ISO cdrom image ...

Check out the mkisofs man page.  You will learn a lot :-)
