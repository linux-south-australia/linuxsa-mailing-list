On Wed, 29 Mar 2000, Dan Shearer wrote:
> Not completely, especially if you are using a webmail client that likes to
> fetch the contents of each message as you cursor down a mail index.

There can be no such thing.  You might be able to write html using some
extensions, and therefore be able to do that for a limited range of
browsers, however it is not possible to do that with standard html.
The limited range of browsers may include recent versions of Internet
Explorer, and maybe also of Netscape Communicator, and thus most people
might have such a browser; the argument that everybody's browser supports
these extensions does not make it html.

Leave aside that sort of picky argument, if you use a webmail client that
likes to fetch the content of each message as you cursor down a mail
index then don't cursor down the mail index.  This problem is specific
to that sort of webmail client, and it occurs just as much with small
folders as with large folders.
