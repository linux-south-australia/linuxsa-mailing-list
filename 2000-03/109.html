Re: converting a binary number to string on command line

Richard Sharpe:
> I want to get that as a sting and be able to write to the correct
> location.

Dieter Jansen:
> Well, it's a little agricultural, but:
>   dd if=/dev/hda1 bs=1 count=2 skip=50 2> /dev/null | \
>     od -t x2 - | \
>     awk 'NR==1 {print $2}'

But isn't this so much clearer (and more portable)?

 main() {
  struct { char offset[50], unsigned short location; } data;
  read(0, data, sizeof data);
  printf("%u\n", data.location);
 }

Every unix system should come with a C compiler!
