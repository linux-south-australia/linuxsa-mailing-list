Hey there...

Dunno 'bout Sonys, but as of 2000, all drives are supposed to be RPC2 -
which is a hardcoded region setting. - Older 6ish speed drives were RPC1
(pre-2000) you could find some apps that allow you to run 'multizoning'
apps - ie, that let you keep on changing region...
I have a Pioneer 6.2x  last of the Pioneer RPC1s - basically left the jumper
on RPC1 (changing it means hardcoding the region setting). Have a few
windows apps that allow me to change the region..
Be wary of multizonal software - some allow you to change region a set
number of times, then you're stuck with your last change.. there are apps
available which will let you keep doing it.
