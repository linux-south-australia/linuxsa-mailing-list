It can be. If you move say a kernel compiled for a late model pentium to
a 386, the code just won't work. If it has the wrong network card, it
will probably boot, but without network support. Using default kernels
that come with distributions are typically easy to port (but using
default kernels really isn't a good idea). Modules help.
