When you are using ip-masquearding, things like ICQ, X-windows and ftp
only work if you have the specific kernel module. Here is  list of those
modules one of my boxes

find /lib/modules/2.2.5-15 | grep ip_masq
./ipv4/ip_masq_autofw.o
./ipv4/ip_masq_cuseeme.o
./ipv4/ip_masq_ftp.o
./ipv4/ip_masq_irc.o
./ipv4/ip_masq_mfw.o
./ipv4/ip_masq_portfw.o
./ipv4/ip_masq_quake.o
./ipv4/ip_masq_raudio.o
./ipv4/ip_masq_user.o
./ipv4/ip_masq_vdolive.o


So, to make it work, you would need to get an "ip masqurading module for
X windows" try this in google.
