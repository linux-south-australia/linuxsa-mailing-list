Re: Newsletter

Greetings,

I have been running a web-based community news site (aka. slashdot)
for Mawson Lakes for about a year now - www.mawsonlakes.org

It uses static pages, but thats only because they have been located on 
a firewall.. (this will change whne the house get built).
The articles are stored as regular html files, and a
postgresql database looks after the titles/categories/authors/urls.

A php3 script accesses the database and creates the pages. 
If url?mode=static produces the current pages for publishing
(otherwise, locally they're produced dynamically) 

The current project is to produce a suitably looking hardcopy. 
As a first effort, I've been producing 'TeX' output.  It works well,
except that I'm now trying to get a two column output.. (which
involves working with TeX's output commands..)

More than happy share what I've done... (it's all GPL'd)
There are similar 'community news-site' downloadable packages out
there but I haven't tried any of them. 
(I'd really like to implement a calender feature!)

Happy Hacking,
PaulS
