Re: Linux Agents/Robots ??

On Mon, 6 Mar 2000, Subba Rao wrote:

> 
> Are there any command line search agents/bots/crawlers for Linux? I want
> to search a few engines at the same time for some key topics.

You want the perl module WWW::Search

You should be able to install it with:

   perl -MCPAN -e "install WWW::Search"

It comes with a perl script called 'WebSearch' which will query all the
major search engines and spit the results out to your screen, one per
line. Or you can use the module in your own scripts for more advanced or
custom stuff.

Do 'WebSearch "your name here"' and be ready to cringe :-)
