Re: X-terminal

Alex,

I'd categorise the message as a warning, not as an error.  Apparently the
x-terminal tries to load the boot image using NFS first, and if that fails
it tries using TFTP.  Setting up TFTP is much simpler than NFS and I
recommend you simply ignore NFS (for the x-term).  Did you setup tftpd to
permit retrieval of /usr/local/tekxp/boot/os.350?
