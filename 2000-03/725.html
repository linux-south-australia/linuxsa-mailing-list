Re: VMWARE on Solaris

VMWare is a BIOS emulator, not a CPU emulator -- it will _ONLY_ work on x86
machines. If you want to run x86 binaries (of any sort -- Windows, Solaris,
or whatever) on a SPARC box, you need to emulate a x86 chip in software.
However, if you want to run Windows x86 binaries on a Solaris x86 box, this
should be possible, assuming VMWare is available for Solaris (you could
always put VMWare on Windows and run Solaris in a VMWare session, or maybe
use Linux, and run both Windows and Solaris in VMWare)

HTH

rr

PS: You email appears to have Solaris and SPARC confused... SPARC is Sun's
chip architecture, analagous to intel's x86. Solaris is Sun's OS, analagous
to Microsoft's Windows. Solaris will run on either SPARC or x86 (although I
have been lead to beleive that Solaris x86 has a limited lifespan...) You
can also run Linux (eg RedHat) on SPARC, but if you're doing this, you
aren't running Solaris (just like if you run Linux on your x86, you aren't
running Windows ... unless you use VMWare or something similar to run
both...)
