As Brad and others mentioned for a one shot change to server 'whatever' 
then use the command line as above. If you want to change the default, 
then place the server name at the top of the list in 
/usr/lib/irc/ircII.servers (at least mine is here (redhat). Try 'locate
irc' or 'ircii' if you can't find it).
