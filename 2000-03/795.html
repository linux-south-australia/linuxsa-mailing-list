Re: basic Command line scripting

On Fri, 31 Mar 2000, Jake Hawkes wrote:
> $ for i in `ls`; do

This is better:

  $ for i in *; do

Consider the following:

  $ touch "a b" c
  $ for i in *; do echo $i; done
  a b
  c
  $ for i in `ls`; do echo $i; done
  a
  b
  c

Other than that one issue I commend you on a useful tutorial.
