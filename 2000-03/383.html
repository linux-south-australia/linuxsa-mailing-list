Re: login scripts

samba.org. Check out the NT-DOM FAQ (there should be a link somewhere).
Also try looking at the documentation for Samba.

OR

You are trying to set up a Linux box as an NT Primary Domain Controller
(good for you!). You will need the following lines in your smb.conf file
if using samba-2.x.x:

workgroup = DOMAINNAME
domain logons = yes
domain master = yes
local master = yes
preferred master = yes
encrypt passwords = yes
os level = 33

wins support = yes

Then create an account for each user that will be using samba / the
domain. These account don't need to be enabled, they just need to exist
in the /etc/passwd file. Then create the NT account with 'smbpasswd -a
username'.

Then on your Windows box, add the IP address of your linux box as the
primary WINS server, double-click the Client for Microsoft Networks bit
in Net Properties and select 'Log onto NT Domain' and type the name of
your domain in the box underneath. Then Go to file and print services
(again, in netcpl) and turn the 'browse list' option to disabled.

This should pretty much be all you need. You may want a netlogon share
as well as profiles and home directories.

I apologise for any inaccuracies, but I haven't booted Windows in a
while. ;-).

Hope it helps,
