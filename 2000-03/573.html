Re: scripted login

If you mean how do you disable CHAP, then use the -chap +pap options for
pppd.

If the other system is using CHAP though, you will be required to set up
your chap-secrets file:

Something like:

#client		server	secret		IP Address
username	*	<password>	

works for me. I think that a * in the IP address field may be necessary.
