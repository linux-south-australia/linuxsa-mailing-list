Re: unix/linux command utilities

On Wed, Mar 15, 2000 at 01:19:23PM -0800, emyeu wrote:

 > Is any know how to limit user to access certain commands
 > in unix/linux
 > for example ; to eliminate user access w or who command
 > or finger command

What are you actually trying to achieve?  Limiting access to a command
is "difficult" because if the user can't run the one that's already
on your system, they can grab the same command from another system and
copy it onto your box and run the copy.  Generally speaking, you should
engineer your security on the assumption that users can run any software
they damn-well please.

It actually sounds like you don't particularly want to limit command
execution at all;  It sounds more like you have some *other* motive 
which you haven't told us about and you think that limiting command 
execution is the best way of achieving that endpoint.  Perhaps it'd
be more useful if you explained why you want to limit command execution,
then we can tell you an alternative way to achieve that aim.
