You can find RPM for Solaris (as in a Sun package that installs the RPM
package manager) at:

http://www.rpm.org/solaris.html


There was a guy who maintained quite a few GNU and other software packages
in RPM format for solaris at:

http://www.solaris.rpm.org/

His site now just points you back to the RPMFIND web site, but there is a
link to his FTP site where there are still a fair few archives of Solaris
RPMS.

I used to use RPM under Solaris, and it was OK (it was easier for me to
build packages than Suns native package manager, as I had used it before),
however if you just want to install some GNU software without actually
building it yourself (which can be quite hard, as Sun do not ship a C
compiler with the OS. So you have to go through the process of downloading
gcc etc), then check out:

http://sunfreeware.com/

 Where you can download muchos GNU and other free software as native Solaris
packages.

BTW you really should try Google (http://www.google.com/). I had used, but
forgotten where to find, all of this stuff. Entering "solaris rpm" in Google
gave the RPM site above as the top match, and "solaris gnu package" as the
top match for the freeware site.
