> What is the all-round preferred TeX system for MS users?

If you want to pay, then Y&Y are good.  See

  http://www.yandy.com/

IMHO TeX is about to die for anything but precision layout work
and back-end text flowing.  On the bright side, MS Word may share
the same fate.

I'd suggest you think about begining to write documents in XML.

The DocBook XSL stylesheet looks good for technical
documentation, it is documented in "DocBook: the definative
guide" from O'Reilly.  Backend tools for rending XML+XSL
into HTML and PDF are at:

  http://xml.apache.org/

  http://www.jclark.com/

I've just started to use it, and there's a current lack
of user authoring tools, but the backend is very very attractive.
