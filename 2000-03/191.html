Not that I know of. LDAP's just like any other database, it's just that
there exist modules and programs that make use of it for their purposes.
You can set up an LDAP address book, which both OutLook and Netscrape
Mail will talk to (amongst other clients). Samba can use it as it's
password backend, so can PAM (in fact LDAP can be used as a complete
replacement for NIS). There is a sendmail LDAP HOWTO. I think it can be
found at sendmail.org (I only saw it in passing).
