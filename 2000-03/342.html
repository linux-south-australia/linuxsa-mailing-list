Re: WHICH JDK

Hi,
    I have been doing some work recently with Java, and I
would offer 2 suggestions. Get the latest version of JDK
1.2.X from the link off the Sun website, or possibly look
into the new Borland JBuilder, which has apparently just
been released for Linux. I use the version for (Sniff) Other
Operating system (mainly because I didn't know the Linux
version was available) and it is very good. If you want
plain java from the ground up, low (no) cost, then go for
sun. If you want a pretty GUI that allows quick easy
development of other GUIs, and don't mind paying a small
amount ($129 AUD), go for the Borland version. O.K. so it
isn't Open Source, but it is a product I don't mind paying
for.

    Cheers!
        Andrew.
