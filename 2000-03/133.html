Re: converting a binary number to string on command line

At 07:11 AM 3/7/00 +1030, Dieter Jansen wrote:
>Hi Richard,
>
>> I can get the location out easily:
>>
>>   dd if=/dev/hda1 bs=1 count=2 skip=50
>>
>> and it is output on stdout, but as a binary number.  I want to get that
>as
>> a sting and be able to write to the correct location.
>>
>> So, how do I do it?
>
>Well, it's a little agricultural, but:
>
>   dd if=/dev/hda1 bs=1 count=2 skip=50 2> /dev/null | \
>     od -t x2 - | \
>     awk 'NR==1 {print $2}'
>
>will do it.

Which just goes to show that there is always more than one way to achieve
things under Linux/UNIX :-)
