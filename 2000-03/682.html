Re: Permissions

On Tue, 28 Mar 2000, Nick Stoic wrote:
> I get this message in RedHat 6.1 that I need to change permissions of
> file /home/ftp/pub from 42755 to 42555.

These numbers are file (permission) modes.  You can read all about them in
section 2 of the manual under stat.  (The command is "man 2 stat")

The 4 means directory; the 2 means set GID bit is set; and the 755 means
read, write and execute permissions granted for user, but only read and
execute for group and other.  555 corresponds in meaning to 755.  In short,
it says that /home/ftp/pub needs to have write permission taken from user.

Do read "man 2 stat".
