basic Command line scripting

Here is a quick little guid to writting shell scripts on the command
line. I will be using bash, and note that there are differences between
the shells.

to invoke bash:
[jake@hyper-ig ~]$ bash
[jake@hyper-ig jake]$

i find the most commonly used command line script I write, is the for
loop:

$ for i in `ls`; do
> cat $i
> done

this first expands everything between the ` and passes the output into
the foor loop. The for loop will then loop around fro as many times as
there are items in the list, assigning $i to the next item in the list.

so to do something more usefull (like convert every file in the
directory using html2ps:

$ for i in `ls *.html`; do
> html2ps $i
> done.

Now, what if the program html2ps needs an output filename? ie, you need
to call html2ps like this:

$ html2ps foo.html foo.ps

then you would need to do some translations:

$ for i in `ls *.html`; do
> html2ps $i `echo $i | sed s/\.html/\.ps/`
> done


html2ps: the program to run
$i: the input file
`echo $i | sed s/\.html/\.ps/': the script to construct the output
filename


to explaion the last bit, we echo $i (in this case a filename like
foo.html) into sed which translates every instance of ".html" with
".ps". So foo.html will become foo.ps, which is what we wanted.

the sed syntax is s/string1/string2/
s: replace strings
string1: what to replace
string2: what to replace it with

the '\' in the above example escape the "." which is a special character
used in pattern matching.

another really cool script that you may need to use later on (if you
deal withj copyinbg files rom MSDOS infected disks is:

$ for i in `ls`; do
> mv $i `echo $i | tr [:upper:] [:lower:]`
> done

which will rename every file in the current directory to lowercase -
thankyou to Mr Tester for that one many years ago - in fact my first
every exposure to shell programing.
