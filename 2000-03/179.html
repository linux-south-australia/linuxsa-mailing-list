> 
> su
> linuxconf
> 
> Select from the tree
>   "Users accounts"
>   "Normal"
>   "User accounts"
> 
> Select "Add" button
> 
> Fill in parameters:
>   [X] The account is enabled
>   Login name: gdt
>   Full name: Glen Turner
> 
> That's all you need to get started.  If you are doing this at work,
> grab your correct user name, user ID and group ID from a UNIX
> system administrator.  Consistency will save trouble later on.
> 
> 
> Press "Accept"
> 
> 
> > 2) can't find the dial=up utility.
> 
> Set it up under linuxconf
> 
> It's pretty straightfoward, except that it's very old-fashioned.
> So you need to tick the PAP box (which isn't picked by default,
> and enter the user ID and password you ISP gave you into the
> PAP tab.  Delete any "chat script" lines. 
> There's a set of PNG screen dumps (4 x 10KB) following this
> message to you mailbox (but not to the list, it upsets people).
> 
> You then say:
> 
>   ifup ppp0
> 
> to start and
> 
>   ifdown ppp0
> 
> to end your session with your ISP.  There are also fancy graphical
> utilities like usernet.
