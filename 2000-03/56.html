Dieter,
If Linux RH61 does not give the option to use fdisk (cant remember 
if it is still there or not) or even if it is there. I always used fdisk to 
set up a system the way I want when I want to control the layout of 
the disk(s).

I will go into fdisk and assign the partitions and tag them as 
required. If it is not with the RH6.x install, then before I install RH I 
use another fdisk. Either the OS/2 one, or the Linux one (from a 
floppy with tomsrtbt utility disk), or whatever, or a DOS tool called 
part.exe (which now can boot off its own OS'less floppy. If I use the 
OS/2 fdisk I use part or linux fdisk to tag the partitions the way I 
want, eg: 83 for e2fs and 82 for swap etc..., and others for wierd 
experiments.

Then, I use the disk druid (you have to anyway) to assign what 
filesystem where to the already existing partitions. Eg: root to hda1 
or /usr to hdb5 etc...
