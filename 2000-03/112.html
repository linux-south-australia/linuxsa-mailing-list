Re: linux pre-installed

I have done this in the past and found that as long as the kernel was
compiled for the correct processor everything else worked fine.
As far as the network cards etc go, you might find that the first time you
boot, you can boot into single user mode ('linux single' at the LILO:
prompt) and correct this sort of information before starting the system
normally.
You might also want to check if and services that you choose to start (eg
httpd, bind etc) rely on DNS and correctly configured network devices to
start correctly. Basically, as long as the IP addresses and network
devices are set right, everything else should follow suit.

Not sure about SCSI adaptors or drives. Does anyone else know what happens
if you move between different vendors adaptors?
