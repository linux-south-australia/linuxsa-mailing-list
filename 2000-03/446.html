Re: Linux Security FUD

> What's needed isn't a virus scanner (except on incoming e-mail),
> but better machine management tools.

Including package management. There's a lot of nice things about rpm
(especially if you've used commercial attempts at the same thing!) but
installing or upgrading is often far too hard. How often have you seen
"rpm -i something" fail with a dependency error, you then try to install
the dependency only to find it has a dependency (or a clash) and so on.
There are rpm shells that try to do this sort of resolution for you but
the mechanism has some things missing.

This is the one area where I think the apt system works better.
Maintaining servers is just so much easier.
