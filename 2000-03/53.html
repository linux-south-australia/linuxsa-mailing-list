Re: Can't start XWindows

On Thu, 2 Mar 2000, Martin Whiteside wrote:
> Now when I try to start XWindows I get the following message:
>                 "Xauth: error in loading shared libraries: libxmv.so.6:
>                 Cannot open shared object.
>                 file: No such file or directory."
> Any suggestions please.  I could post my XF86Config file if required.

The error message sounds quite clear to me.  It cannot open the shared
library libxmv.so.6 because a file is missing.  It's not completely clear
whether that file is the shared library itself, or something named "file".
Have you done something stupid with your config, like naming a file, "file"?

If it refers to the shared library, is the library file really there?  (Look
for it.  If it's not there then you know what to do.)  If the file is there
(unlikely) the problem will be where does X look for shared libraries?  Find
out how the systems locates them and make sure X is looking in the right spot.
