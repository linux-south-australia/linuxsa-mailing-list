Lilo boot probs

Suse 6.3 install

Am having problems booting a new Suse install from the Lilo prompt. Both
RH and Win 95 boot ok from hda. When booting with the below config, Suse
loads and all goes well but the login shows it is using the 2.2.5-15
kernel. It will boot ok from the boot disk, but I believe it is using
the kernel on the floppy. As this is a new install on a new disk it can
be changed without much problem.


Brian Marr
