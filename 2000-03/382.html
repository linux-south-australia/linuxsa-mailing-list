Re: hard disk speed

run "hdparm /dev/hda"
and see what it says.
You may then want to try increasing the speed with the following
commands
hdparm -a16 /dev/hda <-- read ahead
hdparm -m16 /dev/hda <-- multiread sectors
hdparm -c3 /dev/hda <--32 bit w/sync transfers


Those should work fine. Although try reading a few files to double
check.
DMA make a big help too, but it may already be enabled, depending on
your kernel version. 
hdparm -d1 /dev/hda
will enable it for you. Note that if you've got a dodgy harddrive or
motherboard, that this is dangerous and could result in loss of data.
Works for me fine, but YMMV.
