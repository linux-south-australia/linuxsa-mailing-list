Once upon a time, boys and girls, there was this thing called a "shar"
file, which was used all the time for sending arbitrary files as news or
mail.  A shar file is a shell (usually Bourne Shell) script that outputs
the appropriate files, be they text or binary.  They can certainly be used
to transport binary files, such as gzipped x86 binaries.

Here's a simple shar file:

#! /bin/sh

uudecode << 'EOF'
Readme
EOF
