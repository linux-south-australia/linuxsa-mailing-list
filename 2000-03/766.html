> Any Ideas?

If it's a tcp wrapper authorisation issue (you don't state what the
inetd.conf line is, and I don't run Mandrake) then it will log the
rejection via syslog to authpriv.  This means grep your /etc/syslog.conf
file for authpriv (or *.[facility]), check which logfile it's being
recorded to (on RedHat, /var/log/secure) and check for messages there.
Or recreate the event, and ls -lrt /var/log to see which log file
changes.
