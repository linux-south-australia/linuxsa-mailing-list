Re: unix/linux command utilities

> Is any know how to limit user to access certain commands
> in unix/linux
> for example ; to eliminate user access w or who command
> or finger command

There are tools to prevent users other those who belong to the right 
groups from accessing past or present status information.

If you don't trust your users, follow the following tips to make your 
machine obnoxious.

last:
  chmod 0660 /var/log/wtmp /var/log/utmp
  chown root.utmp /var/log/wtmp /var/log/utmp
so only root can access the last login records.

w who:
  http://www.openwall.com/linux/ kernel patch
The "Restricted /proc" part of the patch will prevent users from 
accessing ps, top, who, w information about others.  If you do not trust 
your users, then this patch is a good start to securing your box.
Also, see last.

dmesg:
I forget, maybe it accesses /proc.

Don't forget to chmod /var/log, eg:
drwxr-x--x   8 root     root         5120 Mar 12 04:02 /var/log

And remove any unnecessary pesky setuid/setgid programs.

For administrative guidelines, see 
  http://www.ugcs.caltech.edu/~werdna/sysadmins.html
