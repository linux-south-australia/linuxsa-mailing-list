Re: Xwindows on an NT client

spr@ozemail.com.au said:
> Can I run an Xwindows session on the NT Workstation? My telnet
> software [KEA] only supports a character based session. If it can be
> done what software do I need? 

You could use an X server on your WinNT machine to connect to your linux box.  
There is a free one available at:
http://tnt.microimages.com/www/html/freestuf/mix/
I found it a little annoying to use, as it doesn't do logins via rexec or 
xdmcp automatically (so you need to telnet, or otherwise connect to your box 
to login).  As always, your mileage may vary.

Starnet have a better X server for win32 (unfortunately for the financially 
challenged it's payware) called XWin32.  It supports XDMCP/rexec and also has 
Low Bandwidth eXtentions (so it works better over dialup).
It is available at
http://www.starnet.com

Alternatively, you could use VNC, which has some neat features that I like 
(such as leaving your desktop exactly as you left it when you connect, 
providing your machine hasn't rebooted in the meantime).  It has server 
software that runs on linux (as well as WinNT) and client software for linux, 
Win32 and java.  It works moderately well over dialup.
Available at:
http://www.orl.co.uk/vnc


Hope this helps,
Martin
