Re: What od I need to istall Linux

> Can anyone tell me if 420mb dis is sufficient.

Sufficient for what?  420MB is sufficient to run Linux and do lots of
stuff, but it's not sufficient for a bundle of other stuff too.  It
all depends on what _you_ want to do with it.  I live my computing
life on Linux, so I need many gigabytes; someone else might just use
it as an Internet gateway and have it doing what they want in 40MB.

> Wil it be better to dedicate the computer to linux (old 486) or is it
> necessary to have win 95 running on the same computer to run the programs?

It's not necessary to have Win95 running on the same computer.
DOS/Windows and Linux are completely separate operating systems.  You
can have either or both on there -- whatever you want.

> If I dedicate the computer  do I still nbeed redhat?

If you want to run Linux, you'll need some sort of Linux distribution.
Red Hat is one of them, but there are others such as Debian and
Caldera.
