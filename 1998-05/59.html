Re: Xwindows on an NT client

> Can I run an Xwindows session on the NT Workstation? My telnet software
> [KEA] only supports a character based session. If it can be done what
> software do I need?

You need an X server for NT.  There are a few commercial ones out
there, but not being an NT user myself, I'm unable to recommend one.

> Finally, and completely unrelated to the above; does anybody know if Oracle
> 7 - Oracle 8 will run on the Linux O/S. And if it does, do you know anyone
> who is actually running such a setup.

The SCO version of Oracle will apparently run on Linux (using iBCS),
but there is no Linux port (at least not one that has been released).
Do you require Oracle?  There are a number of databases available
natively for Linux.
