Want to be a Linux Documentation Project author? 

The Linux Documentation Project has a number of HOWTO documents that
are now unmaintained and in need of someone to bring them up to date
and keep them useful. This is an excellent way of becoming involved
in a truly useful Linux project. You can find the unmaintained documents
listed at: http://www.linux.org.au/LDP/HOWTO/. Some of the important
documents that are currently unmaintained are the Serial-HOWTO, the
News-HOWTO and the UUCP-HOWTO. If you need more information on how to
contribute to the Linux Documentation Project email Tim, the Linux
Documentation Poroject Coordinator, at linux-howto@sunsite.unc.edu.
