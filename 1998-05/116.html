Crontab not working

Daniel,

If crond is running, try attaching to that process.
Every minute crond should do something.  To attach to 
crond do the following.

	ps ax | grep crond

Then as root

	strace -f -p <pid>

Where pid the process id given by the first command.

Every minute it should go and look at a number of files.
Look for lines starting with 'open' as the system call.
If they are returning EPERM or similar then have a closer
look at those files.

If it is not doing anything every minute, in another window
do a 'crontab -e' and see if it reloads the crontab files.

Hope this helps.

Matt.

PS If you have problems decoding the strace output, contact me
so I can look at it for you.
