Re: Exchange replacement for Linux

If you want to replace with more bloatware possibly Notes, Samsung 
Contact, Oracles mail/calendar stuff etc... or wait for Novell/Ximian 
Groupwise on Netware7/Linux.

If you want a hack there are connectors like Bynari(Outlook to Imap) or 
Ximian Evolution (Evolution to exchange)

Or the ideal solution(IMHO):
Mail server side use cyrus IMAP and exim or postfix.
OpenLdap for your global address book.
Mozilla Thunderbird for mail client (removes the spam that spamassassin 
misses)
Get hacking on Mozilla Sunbird for your calender client.
And webmail for the tourists, Squirrelmail does nicely.

Bonus would be if someone wrote a nice calendar server because although 
synching ical files to/from a web site is a good way to 
publish/subscribe a footy schedule it seems kind of restricted for 
office appointments.

I really recommend people look hard at the Mozilla birds and test those 
nighty builds hard. Evolution or Kmail are OK for the converted but the 
way to advance the linux desktop is to get those windows users using 
Mozilla and OpenOffice.
