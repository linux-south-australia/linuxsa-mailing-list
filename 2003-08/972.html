Luke,


> I'm new here, and new to Linux. I've got a pentium II 233 with 2 x 3079MB 
> hard drives, and I wanted to know what would be the best way to setup my 
> partitions, and how big to make them. I've read through David Lloyd's
> notes on partitioning, but wanted to know if there is anything I should
> look out for. Should I install debian 3 first, or redhat 9. Also,
> apparently I can share the swap partition between the OS's?

My notes on partitioning? I don't suppose you have a URL for them...because
I can't find them :(

I'd be inclined to install the Debian 3 first and then the RedHat 9.

Why?

 * Debian's installer is a lot less pretty than RedHat 9's
    - and it will be easier to work out what *not* to zap using RedHat 9's
      custom install

You are able to share the swap partition between OS's. Just make sure that
you install Debian with the "bf24" kernel (type bf24 at the initial boot
disk) because I'm not sure whether a 2.4.X swap partition is the same as a
2.2.X swap partition.

As for how to partition them, I'd be inclined to give each OS one disk each.
