All except the lover case.  And I'm not so sure about expalin, either.
 
I'm sure this has been posted to the list before ... using a Bourne-type shell
 
for file in $(find . -type f -print); do
  lcname=$(echo $file | tr [:upper:] [:lower:]
  if test -f $lcname; then
    echo "file $lcname already exists"; continue
  else
    mv $file $lcname
  fi
done
