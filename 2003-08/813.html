Re: Vector Linux SOHO 3.2 problemo

[rant mode]
why, oh why, do distros omit the kernel source from the default package 
install list? (redhat, mandrake, i'm talking to you.) there are so many 
times when it's needed; i've had 3 or 4 times personally where it's come 
up either at work or home that it hasn't been installed and people have 
been mystified as to why something wouldn't compile.

i'd say the kernel source, while it takes up a fair bit of space, 
*needs* to be installed by default.
[/rant]

Corey, what distro are you using? if it's redhat or mandrake (the only 
two i have any experience in, really), you should be able to run their 
respective package managers - under "main menu -> configure -> package 
management" in mandrake, or under "main menu -> system settings -> 
add/remove applications", i think, in redhat. *hopefully* the kernel 
source package will be listed in there; you can select it to be 
installed and it should prompt you for the correct CD.
