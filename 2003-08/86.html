SuSE Linux 8.2 KDE Upgrade to 3.1.3 with APT/Getting Rid of AnnoyingKonsole Font Warnings

Here are the procedures and results of the upgrade to KDE 3.1.3 on my 
SuSE Linux 8.2 Professional desk:

1. I already have apt for SuSE installed and configured on my desk. 
Basically it is a matter of going to this page:

ftp://ftp.gwdg.de/pub/linux/suse/apt/SuSE/8.2-i386/RPMS.extra

and downloading and installing the following files in the following order:

   1. lua
   2. apt-libs
   3. apt

You will also need a sources list from here:

http://ftp.gwdg.de/pub/linux/suse/apt/SuSE/8.2-i386/examples/sources.list.FTP

If you use Netscape, the file can be saved in your home directory by 
selecting "File/Save As" from the menu bar. Make sure that you rename 
the file "sources.list."

Open up your console and log in as root by typing su, and then supplying 
a password when asked.

Then type this command:

cp sources.list /etc/apt/

Next, you should remove the SuSE 8.1 sources.list from 
/ect/apt/sources.list.d/

rm cp sources.list /ect/apt/sources.list.d/

Then type this at the console:

apt-get update

Its a command you should use periodically to update your list of 
available applications.

To install an application just type:

apt-get install (packagename)

For example, if I wanted to install a package called Gimp, I would type:

apt-get install gimp

Note that APT is not a requirement for the upgrade, its just easier to do.


Once apt is installed, you should go to this page:

http://www.suse.com/us/private/download/linuks/i386/update_for_8_2/index.html

The base packages must be installed first, then you can choose the 
packages of your liking.

I made a script to automate the procedure and it is available here:

http://prdownloads.sourceforge.net/automagic/SuSE82Kde313.sh?download
