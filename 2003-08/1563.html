You haven't said what distribution of linux you're using, not that that
should matter too much.

http://qtables.radom.org is the home of the most moron friendly firewall
generating script around. Just unpack the tarball and run the
quicktables script, answer the questions, and it will produce a decent
firewall script you can then execute by doing...
chmod +x rc.firewall ; ./rc.firewall ; iptables-save

If you use redhat then quicktables has a contrib/redhat-install.sh to
install the chkconfig and quictables scripts into all the correct places
so that everything starts up properly on every boot.

Other distro's have different methods for starting their firewalls.
Please supply more detailed questions.
