How long to set up a server to do..

How many hours would it take someone experienced to set up a server
(on SDSL w/static IP) to handle:

0. / + /boot on RAID1.

1. Apache, multiple domains (2 or 3), php, mysql,etc

2. DNS for said domains.

3. Email (multiple domains w/eg. exim), + webmail client eg IMP.

4. X applications to display on an X server/terminal elsewhere on
the LAN.

5. Squid.

And be running securely, reliably, and efficiently. This may involve
setting up UML to keep web serving separate from DNS separate from
email separate from X applications.
