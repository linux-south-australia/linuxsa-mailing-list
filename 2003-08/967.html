RE: satellite

first add the Sat ip as an alias on eth0

assuming a Sat ip of 201.11.11.1

ifconfig eth0:5 201.11.11.1 netmask 255.255.255.255 up

then

/sbin/iptables -t nat -I POSTROUTING -o ppp+  -j SNAT --to 201.11.11.1

Now EVERYTHING going out any ppp interfaces will be given a source ip of
201.11.11.1 and will come back via your Sat IP
