Tony,

> Will  chmod 2750 <directoryname> (for example) do the trick? (The man
> pages explanation of the first digit is not clear to me)

If you set the sticky bit on a directory (the "2" you're talking about)
then the files created in that directory may only be deleted, renamed or
move by the file's owner.

Root can do whatever root likes of course.

You'll often see it set in the /tmp directory. This is so that users
can't accidentally or maliciously delete other people's temporary files.
