>>downloaded. How do we set this up so it can be run from the command
>>line or from a Kde menu ?
>>
>>Brian Marr
> 
> 
> Brian,
> 
>      I don't run KDE so I can't answer that question for you.  I can
> however answer the command line question.  All you have to do is add the
> /path/to/the/command you use to start MozillaFirebird to your $PATH. 
> This command for you would most likely look like: 
> 
> bourne compatible shell
> `export PATH=$PATH:$HOME/MozillaFirebird/MozillaFirebird`
> 
> csh compatible shell
> `setenv PATH $PATH:$HOME/MozillaFirebird/MozillaFirebird`
> 
> To get this functionality everytime then you will need to edit your
> shell's init files eg: .profile, .bash_profile, .cshrc, .tcshrc ... etc.
> If you are unsure on how to do this last part then just let me know.
