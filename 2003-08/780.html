Re: Sendmail Question [buildaddr: unknown mailer unknown]

How'd ya build the sendmail.cf file?  from the m4 file?  If so, you actually 
(at least on RH) need to tweak it a tad.

In the ".mc" file you'll need (under Rh at least) to add:

define(`PROCMAIL_MAILER_PATH',`/usr/bin/procmail')
FEATURE(local_procmail)dnl
MAILER(procmail)dnl

[at least this is what I did and it works for me]

This is because different unixes have different ideas about what's a local 
mailer.
