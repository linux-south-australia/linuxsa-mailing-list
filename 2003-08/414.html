Hi Andrew,

It sounds like your 'UserDir' directive or the permissions on the
'/home/andrew' directory,

apache is probably running as 'nobody' or similar (a user with limited
permissions) so at least:
directories need r-x for 'other'
files need r-- for 'other'

If you like send me your httpd.conf file and I'll take a look,

hope this helps,
