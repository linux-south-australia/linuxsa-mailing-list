Re: Anyone got a booting kernel 2.6-test .config

"make oldconfig" doesn't work from 2.4 configs because there is a new option 
for Virtual Terminals that must be enabled. Try going to this url:

http://www.codemonkey.org.uk/post-halloween-2.5.txt

Site doesn't seem to work for me at the moment, but you'll probably find a 
mirror for that file on google anyway...

I have a working .config for 2.6.0-test2 if you really want it. There's also a 
patch you should apply for improved scheduler performance:

http://people.redhat.com/mingo/O(1)-scheduler/sched-2.6.0-test1-G6

(it says test1, but it applied to test2 no problems and improved performance).

Cheers,

Damien
