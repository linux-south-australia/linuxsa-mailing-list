(From the Ximian FAQ)

A:  Ximian Evolution provides all standard email handling features, plus
powerful mailbox management using vFolders\x{2122}, user-defined filters,
and quick searches. The address book works with mailer for autocompletion of
addresses and easy addition of address entries. The calendar supports
multiple date views and allows users to create and confirm group meetings
online, even with Microsoft Outlook and Lotus Notes users. Both address
cards and calendar events can be sent and received by email, and PalmOS
devices can synchronize with the calendar, address book, and to-do list.

Additionally Ximian Evolution includes sound alerts, a bounce-mail function,
faster performance, and advanced filter criteria. Ximian Connector expanded
feature set include scheduling group meetings, viewing free/busy time,
access to public folders, access delegation, direct resource booking,
ability to notify people when you are out of the office and a smart account
assistant for easy setup. Check the application manuals for full details.

New Ximian Evolution 1.4 is based on GNOME 2 and takes advantages of the
desktop platform with noticeable improvements in font handling and printing
as well as enhanced performance.

Q: If we are a Windows based company with Unix users, can we use Ximian
Evolution? A: If your server uses standard open protocols like LDAP, IMAP,
POP, and SMTP, you can use Ximian Evolution with it. You can share addresses
with vCards and calendar items with iCal appointments. If you use Microsoft
Exchange 2000, you can use Ximian Evolution with the Ximian Connector add-in
as a Microsoft Exchange 2000 Client.
