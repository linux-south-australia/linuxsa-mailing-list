Jim,

The sales@ addresses prove nothing. I get heaps of spams addresses to
sales@domain for all of my domains, but I don't have a single user called
"sales". It's just pure speculative addressing.  The same is true with
"webmaster". I get spams to these although I deleted all such user names a very
long time ago, and I've never exposed these addresses at all on web pages.
Another such speculative e-mail address I've seen is jobs@domain.

If anyone wants to get rid of such spam for commercial mailing addresses, it's
only necessary to change the addresses to something a little more imaginative,
and then use some sort of "IQ > 0" test like this:

http://www.alankennington.com/extra/email.php

which I've admittedly posted to this list before.  This approach reduces my
spams to about one very 3 months.  I must say I do miss the nigerian scam spams
though, and some of the other equally hilarious ones. But the SNR has improved
from about -10 dB to about +25 dB.

Cheers,
