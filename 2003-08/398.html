mv /etc/xinetd.d /etc/xinetd.bak
mkdir /etc/xinetd.d

Then retry
service xinetd reload
then 
grep xinet /var/log/messages

to check the output

If you still have the error check /etc/xinetd.conf!

If that's OK then move the files back 1 at a time checking each time
ie;
mv /etc/xinetd.bak/wu-ftpd /etc/xinetd.d/wu-ftpd

retry. Eventually you will find the file with the error
