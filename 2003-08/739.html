Hey,

I recommend Mandrake 9.1. I'm using it (and I've tried others, including 
Debian), and I'm think it's by far the best one to use to get all your 
devices working. I don't think the capture card or printer will be a problem, 
dunno about the modem/scanner/digital camera (just make sure they're all 
plugged in an turned on when Mandrake installs). The GF4 will work like a 
treat, just go to www.nvidia.com and download the drivers for Linux on IA32, 
then run the install script, modify /etc/X11/XF86Config-4 so that it says 
Driver "nvidia" rather than Driver "nv", restart X and enjoy.

As far as package management goes, go to 
http://plf.zarb.org/~nanardon/urpmiweb.php and select the "Contrib" and "plf" 
package repositories, run each command it spits out as root (type "su" at a 
terminal and enter your password to become root) one after the other.

Then to install software, go to k->Configuration->Packaging->Install software. 
Then search or browse the package lists for additional software (some are big 
downloads so make sure you click "Maximum Information" so you can see what 
source your package comes from, i.e. internet or CD). I recommend getting 
mplayer from plf and the win32-codecs package so you can play videos. I also 
recommend kmplayer as a frontend to mplayer.

For DVD playback, I recommend "ogle". The version in plf is 0.8.5, which is 
okay, but if you want 4/5.1 channel surround sound (assuming you have a card 
that supports this under Linux), then you need version 0.9.1 which can be 
found at:

http://www.dtek.chalmers.se/groups/dvd/index.shtml

The RH 7.x rpms seemed to work for me.

Another download you'll want is the freetype2 package from plf, this will 
upgrade your mandrake True Type Font renderer so it supports byte-code 
interpretation (a patented technology that can't be included in the 
distribution), this makes your fonts look heaps better. Then in the Mandrake 
control centre, go to System->Drakfont and select "Get Windows Fonts". You 
may need to restart xfs (type "service xfs restart" as root) and/or X (press 
ctrl-alt-backspace) for the new fonts to be recognised.

Cheers,

Damien
