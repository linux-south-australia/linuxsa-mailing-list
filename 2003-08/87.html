Re: SuSE Linux 8.2 KDE Upgrade to 3.1.3 with APT/Getting Rid of AnnoyingKonsole Font Warnings

I forgot to add the information about getting rid of the annoying...

"Font `-misc-fixed-medium-r-normal--15-140-75-75-c-90-iso10646-1' not 
found. Check README.linux.console for help."

...message that appears after upgrading KDE. The readme file is useless.


I found the answer I needed on this page:

http://www.linuxquestions.org/questions/archive/5/2003/02/2/32978

Pertinent information includes:

Posted by: kenb

> There are several places within Linux to affect the fonts.
> However, to effect the Konsole you must select the Konsole "Settings" 
> menu and then "Font".
> To correct this problem, choose "Custom" under "Font",
> then select "Fixed", of which there are three varieties.
> I am using "FIxed [Misc]"
>
> Hope this helps,

What is missing is how to save the settings. After you change your font 
settings, select the Konsole "Settings" menu and then "Save Settings."



Hope this helps.
