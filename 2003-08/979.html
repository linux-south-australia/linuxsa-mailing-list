Re: Mini-ITX anyone?

Hiya,

well I have been playing with them for a little while now. Yes, their
perfomrance isnt all that great when compared to what else you can get
for similiar money. But for office apps (windows, for the financial
officer here), it works a treat, and it comes in a funky little cube.
We are actually using them for developing small PBX systems (4 station 2
trunk) with echo cancelation for VOIP. They have a enough grunt to do
it, just :) Changing the code to use 3D-Now! optimization for the
floating point calculations certainly helped things out, but nothing
like it did on a Athlon/Duron :)

I like the fact that you could, if you wanted to, run them of a 12v gel
cel. Check out this link for a heard of little linux robots using
mini-itx mobo's http://www.ai.sri.com/centibots/

Cheers,

Ben :)
