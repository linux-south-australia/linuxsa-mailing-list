Re: Non-passive FTP behind NAT?

> Hiya,
> 
> I'm needing to get a proprietry (grr) application that uses non-
> passive FTP to work behind a Linux box doing NAT; previously, I've 
> always used passive FTP, which works fine, but in this case, I need 
> to get non-passive working.

Don't worry, got it working, turned out I needed to load:

ip_tables
iptable_filter
ip_conntrack
ip_conntrack_ftp
ip_conntrack_irc

(which I thought should have been loaded with the other module, but 
obviously not).
