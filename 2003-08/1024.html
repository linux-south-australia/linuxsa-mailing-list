Re: Knoppix Linux

Knoppix mounts detected hard drive partitions as read-only by default, 
if you want to write to them you have to re-mount them as read/write.
I think it give the option to do this from a right click on the (KDE) 
desktop. But there is still one gotcha, it won't give write access 
(quite sensible really) to NTFS partitions.
