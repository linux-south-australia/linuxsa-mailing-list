David Hill wrote:

> If you are using vim, 'gq}' is quite a useful command.  It usually makes
> cleaning up other people's poorly formatted paragraphs much less painful.

If your editor is smart enough to be able to call an external program, 
try this perl module:

http://search.cpan.org/author/DCONWAY/Text-Autoformat-1.12/lib/Text/Autoformat.pm

It's rather smart; see the examples.

You don't need to know any perl to use this module, this little 
one-liner will give you a filter that does the job:

perl -MText::Autoformat -e'autoformat'

Bind that to a key in your editor of choice and you're away.
