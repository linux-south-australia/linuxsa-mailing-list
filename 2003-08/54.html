Re: What may be a silly SAMBA question

On Fri, 1 Aug 2003, Colin Rose wrote:

> I am running Mandrake 9.1 with a dual boot system. The other system is
> XP Professional and is running NTFS. Apparently writing to an NTFS
> partition from within Linux is dangerous making my question: is it safe
> to write to this partition if it is mounted as a samba share?
> I did do a bit of a search and couldn't find a solid answer to this
> question. I heard a rumour that someone on this list is a world famous
> samba guru and would know the answer.

It will be safe. The issue will be mounting the share on Linux. You might 
have to turn off sign&seal on XP Pro, which is on by default, I believe.

There are some .reg files in the docs directory for Samba, one of which 
should help.

It is possible also that the latest version of Steve French's CIFS VFS 
module understands sign&seal.
