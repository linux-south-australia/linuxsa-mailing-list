More FreeBSD 5.1 Current Solutions& Notes

	:##############Notes#######################
	:1. Netscape 7.2
	:* To install Netscape 7.2 in FreeBSD 5.1, just make
	:  sure that the Linux compatability layer is installed.
	:  Download the Netscape installer from
	:  http://www.netscape.com and use these commands:
	:** Netscape Setup
	:Just go to www.netscape.com and download Linux version.
	:Then:
	:gzip -dc netscape-i686-pc-linux-gnu-installer.tar.gz |
	:tar xvf -
	:cd netscape-installer/
	:./netscape-installer

Why ? when you can just do:

shell> cd /usr/ports/www/netscape7
shell> sudo make install clean

Always try to build software from the ports tree first.


	:2. CVSUP
	:* To use CVSup, use the following commands:
	:*FreeBSD CVSup setup
	:cd /usr/ports/net/cvsup-without-gui/ && make && make
	:install && make clean

FYI. This is one piece of software that many people recommend to install the package rather than
build the port. Why ? Because CVSup depends on the Modula-3 compiler and runtime system and it
is huge (to build and download). So over a 56k modem downloading the package would be a smarter
option.


	:*default release=cvs tag=RELENG_4

Aren't you tracking the head of the tree ? ie -CURRENT ? If so, you are using the wrong tag.
