Re: Web site load (was: [SCO] some code sees the light)

FWIW,
I run a reasonably popular website on an ADSL service at home.  It has some 
jpg and mpgs (it's a site about a type of Peugeot, see below).

In Feb it peaked at 21,109 hits (350MB traffic) [according to webalizer].
[average is around 5000 hits, 127MB per day]

This works reasonably well on a 512/128kbps ADSL link - the main killer is 
images being downloaded, hence my recent ADSL shaper scripts.

So, you can run a reasonably popular website on quite a low bandwidth link 
as long as you don't have a lot of images etc.
