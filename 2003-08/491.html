Tony Nabkey wrote:
 
> How do I set the permissions on a directory so that any files added will
> inherit the permissions of that directory?

Under Unix, you can only inherit ownership, not permissions.

> Will  chmod 2750 <directoryname> (for example) do the trick?
 
That causes any files to be created in <directoryname> to have the
group id of <directoryname>.
