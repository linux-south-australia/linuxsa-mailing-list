I serviced a dead machine with one once, so I did a little research
on them. Plus I had an earlier chip from them and after that
experience, I won't touch them. I don't know if they're improved
since the Cyrix MII but those were awful - in floating point
operations, a Pentium 166 with 16MB could beat the MII-300 with 64MB
by miles. Via have since bought Cyrix. On the other hand, didn't Via
release that software CPU with the self-editing emulator? At first
boot it benchtested at roughly half the speed in Pentium, but after
an hour or so had rewritten the emulator to be only a MHz below the
same speed Pentium in performace. So, all up, I'd have to say do lots
of research.

http://www.tomshardware.com/cpu/02q2/020605/
http://www.via.com.tw/en/viac3/c3.jsp

This one classes it as "low end", so I'd be wary
http://www.digit-life.com/articles/viac31ghzlowend/

Tom's hardware says "At first glance, the C3 processor seems to be
nothing but a low-end competitor to Intel and AMD CPUs; how could a 1
GHz processor compete with a fast Pentium 4 at roughly half the clock
speed?"
But points out that the main advantage it has is that it runs so much
cooler - I seem to recall a fanless system being based on one, once.
Designed as an entertainment system. So, great for cheap, cool
running. Not so hot if you need serious grunt work
