Re: Vpopmail and .qmail forwarding through odeiavir

nik [tm] wrote:

> I am really unsure exactly where the .qmail should be, plus its contents.


Hi Nik,

The .qmail file should be in the 'home' directory for the user, which is 
usually the directory above 'Maildir'.

If you're using vpopmail, it's probably using /var/qmail/users/assign to 
lookup which directory is 'home' for a given mailbox, have a look in there.
