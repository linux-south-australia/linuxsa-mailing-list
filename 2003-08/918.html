RE: NFS

Make sure that the target computer (the one exporting the share) is running
portmap and nfs. D o this:

#service portmap start
#service nfs start

and make sure your /etc/exports file is well writen - on the target
computer. Ie:

/home/shared 10.0.0.1(rw)

Or something like that. Once thats done then you can go into the /etc/fstab
stuff. Make sure it works the aboe way first.
