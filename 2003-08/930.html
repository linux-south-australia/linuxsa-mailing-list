If you have:

 /dev/hda1 / defaults,data=writeback 1 1

...in your fstab which isn't the default (the default is data=ordered),
if you don't pass the kernel:

 rootflags=data=writeback

...the kernel goes haywire and your system is, essentially, unusable.
Which is a total pain because it mounts itself as read only and there's
all sorts of problems that happen when this occurs.
