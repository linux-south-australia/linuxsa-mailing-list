Hi there

not too shure what the problem is, are you just setting up a normal ftp
server.....if so what version os OS you running there


nmap, will show you if the ftp port is there  ...eg
21/tcp     open        ftp
22/tcp     open        ssh
25/tcp     open        smtp
53/tcp     open        domain
80/tcp     open        http
110/tcp    open        pop-3
111/tcp    open        sunrpc
443/tcp    open        https
514/tcp    open        shell


a couple of things to check for are, /etc/hosts.deny   /etc/hosts.allow

if Redhat ...../etc/xinetd.d/vsftpd

make shure no firewall rules are blocking it
