Re: Exchange replacement for Linux

Hey Brad,

What version of exchange are you running? If it's version 5.5 or below, I 
think you will find that the transition will be easy. Unfortunately, later 
versions of Exchange, generally implement only with Active Directory (for 
full functionality).

Now, if you have Active Directory installed, then I think the transition 
will be a hard one, and you'll probably have to start from scratch anyway, 
because the Exchnage server schema will be a part-of/addition to the Active 
Directory schema.

Obviously, if you have gone down the linux path, this may not apply.
