1. Netscape 7.2


* To install Netscape 7.2 in FreeBSD 5.1, just make
  sure that the Linux compatability layer is installed.
  Download the Netscape installer from
  http://www.netscape.com and use these commands:

** Netscape Setup

Just go to www.netscape.com and download Linux version.

Then:

gzip -dc netscape-i686-pc-linux-gnu-installer.tar.gz |
tar xvf -

cd netscape-installer/

./netscape-installer

2. CVSUP

* To use CVSup, use the following commands:

*FreeBSD CVSup setup

cd /usr/ports/net/cvsup-without-gui/ && make && make
install && make clean

mkdir -p /usr/local/etc/cvsup/sup

emacs /usr/local/etc/cvsup/sup/supfile

emacs /usr/local/etc/cvsup/sup/refuse

emacs /usr/local/bin/cvsrun

chown root:wheel /usr/local/bin/cvsrun

chmod u+x /usr/local/bin/cvsrun

/usr/local/bin/cvsrun

cvsup -v

* Contents of /usr/local/etc/cvsup/sup/refuse:

ports/chinese

ports/french

ports/german

ports/hebrew

ports/japanese

ports/korean

ports/russian

ports/ukrainian

ports/vietnamese

* Contents of /usr/local/etc/cvsup/sup/supfile:

*default host=cvsup10.FreeBSD.org

*default base=/usr/local/etc/cvsup

*default prefix=/usr

*default release=cvs tag=RELENG_4

*default delete use-rel-suffix

*default compress

src-all

ports-all tag=.

* Contents of /usr/local/bin/cvsrun:

*default host=cvsup10.FreeBSD.org

*default base=/usr/local/etc/cvsup

*default prefix=/usr

*default release=cvs tag=RELENG_4

*default delete use-rel-suffix

*default compress

src-all

ports-all tag=.
