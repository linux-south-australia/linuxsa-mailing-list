RE: .shtml

Troy,

> The cgi script runs if I call it by it self and the 
> .shtml shows the rest of the html thats in it.

This is because your server is not "parsing" the file and executing the command you have in the .shtml file. 

You need to setup server side includes on your webserver.

In your httpd.conf look for something like:

    AddType text/html .shtml
    AddHandler server-parsed .shtml

Google should be your friend...

http://www.google.com/search?q=enable+server+side+includes+apache
