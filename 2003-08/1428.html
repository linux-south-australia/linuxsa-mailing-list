What like ctm? I don't know that many people are using that anymore.

But cvsup is incredibly easy to get going and requires little bandwidth 
(updating over dialup is not onerous).

Install the package when you install freebsd, get networking going, and 
type:

cvsup -h cvsup.au.freebsd.org /usr/share/examples/cvsup/ports-supfile

That's not hard.
