RE: ftp server setup

Have a look at /etc/xinetd.conf specifically to check the 
include dir.

------------------------
#
# Simple configuration file for xinetd
#
# Some defaults, and include /etc/xinetd.d/

defaults
{
        instances               = 60
        log_type                = SYSLOG authpriv
        log_on_success          = HOST PID
        log_on_failure          = HOST
        cps                     = 25 30
}

includedir /etc/xinetd.d

------------------------
Also 
service xinetd reload
then 
grep xinet /var/log/messages to check the output

What does it say?
