A few examples for you, (watch line wrap).

This one specifies source and dest IP and DNATs to the internal address.
For instance if you have multiple live IP addresses on your gateway box,
you can specify which one gets dnatted. You can have multiple sources
specified, just make a new rule for each.

iptables -t nat -A PREROUTING -s 203.0.0.1 -d 202.100.1.100 -p tcp
--dport 3389 -j DNAT --to 192.168.100.1

This one only specifies the destination, so will work from any source.

iptables -t nat -A PREROUTING -d 202.100.1.100 -p tcp --dport 3389 -j
DNAT --to 192.168.100.1

Make sure you don't have any other rules blocking inbound that will
over-ride this.
