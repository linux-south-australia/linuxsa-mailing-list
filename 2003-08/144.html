Re: Mounting Windows partition

Vern,

You'd probably be better mounting it under a directory, to avoid clutter...

Perhaps edit your /etc/fstab file (which controlls filesystem mounting), 
and add a line like (i think):

/dev/hda1 /mnt/windows vfat defaults 0 0

(you might need to change 'vfat' to 'ntfs' depending on your windows 
partition type (format)).

Then you go into /mnt and create a 'windows' directory (you'll need to 
be root, probably). then unroot, and type 'mount windows'.

This oughta work, I think... and won't give allow you to write to your 
windows directory unless you're root - of course, you can change this if 
you wish  ;-)
