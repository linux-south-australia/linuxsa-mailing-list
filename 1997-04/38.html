Re: comms program

> can someone give me a URL to get a comms program that will support 8N2 -
> minicom wont let me change the stop bits :(

How about seyon?  It's an X-based comms program and it at least has an
option to set 2 stop bits (I can't test it though :).  Seyon comes
with Red Hat, so either you have it installed, or you can mount your
Red Hat CD and run this command to install it:
rpm -ivh /cdrom/RedHat/RPMS/seyon-*.rpm

If you don't use Red Hat, you can download Seyon from
sunsite.anu.edu.au:/pub/Linux/X11/xapps/comm/Seyon-2.14c-tar.gz
