Re: [linux-security] pentium bug makes security under linux impossible (fwd)

Don't forget that just searching for the instructions in executables won't
protect you.

eg.
/* Kill sequence = 0xf0,0x0f,0xc7,0xc8. Listed as (0, -1, -2, -3) to avoid
	detection. */
char x[5] = { 0xf0, 0x0e, 0xc5, 0xc5 };

main ()
{
	int i;
	void (*f)() = x;

	for (i = 0; i < 5; i++)
		x[i] += i;

	f();
}

And this is a very simple example.

Yes people this is indeed a serious problem.
