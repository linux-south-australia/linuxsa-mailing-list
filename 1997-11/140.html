Re: Discovering an IP address

>  Hi all,
>  Well call me stupid, ignorant, unlearned or whatever you like, but I'll ask
>  this question anyhow.
>  
>  If I know someone is getting on to the 'net via a dial-up link, how do I
>  find out the IP address for that connection.
>  I wish to be able to ssh into a machine that dials in to an ISP at least
>  once a day, but will need to know its IP address, which of course is
>  different each time it dials up.  I'm sure it's a simple command/procedure
>  but right now I'm suffering a severe case of mental blockage.  Any helpers?
>  regards....Don

Really depends how automated you want it. When the machine dials into the
ISP they could set off a telnet to a specific port that could use tcp
wrapper to report the ip address to the anybody by email.
If you look under section 5 man page for hosts_access you can get
inetd/tcpd to fire off just about any program when a connection is made to
a specific port.
