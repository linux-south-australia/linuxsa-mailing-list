> The problem I'm having is that if I turn off (disable) the keyboard and/or

> All I end up getting is "LI-" and lots of noise from the speaker. How do I
> tell LILO to ignore the keyboard and video hardware settings??
> I really don't wish to have to have a keyboard and monitor on this beast.

Hi Don,

There might not be a solution.  It seems that "it just works" for some
computers and "it just doesn't work" for others.  I had a server that
I _had_ to have a keyboard attached to (regardless of the BIOS
setting), otherwise Linux wouldn't load.  The replacement server
didn't need a keyboard.

Have you tried booting with the keyboard unplugged, both with and
without the BIOS keyboard setting enabled?

The "LI-" is unusual though.  The "LILO start message" section in the
LILO documentation mentions "L", "LI", "LIL?", and "LIL-" but not
"LI-".
