The pristine sources thing means that the original untainted source
code plus patches are put inside the source RPM.  This is as opposed
to having patched source code included.

Maximum RPM (available in paper form from NetCraft or postscript from
www.rpm.org) has some stuff on the advantages of using pristine
sources (page 115-116).
