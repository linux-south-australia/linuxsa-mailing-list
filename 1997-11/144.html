> MS and need to at least start accessing the net, either send email, which I
> tried with PINE but couldn't get the pine to activate my modem like Eudora

Pine is simply a mail-user-agent (MUA). What this means is that its' only
useful purpose is to send and receive e-mail. It will /not/ dial for you. 

You need to enlist the help of another program, called pppd (for PPP) or
dip (for SLIP) to /dial/ netcom, and a program called fetchmail to get
/mail/ off netcom.

Pine includes a basic SMTP delivery agent, so you don't need to worry too
much about setting up a proper delivery agent properly.
