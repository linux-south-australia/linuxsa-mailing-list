Re: RPM's

> Is there a RPMS converter thingy so taht i can install
> .rpm's on my slackware box?

Do you really need a converter?  You can install RPM yourself pretty
easily, and then you at least get the benefits of it for the packages
you install with RPM.  To install it, something like this should do:

Grab ftp.netcraft.com.au:/pub/redhat/code/rpm/rpm-2.4.12-1.i386.tar.gz
Unpack it from the root directory
Create the directory /var/lib/rpm
Run rpm --initdb

If you do want a converter, there is rpm2cpio.  You can get a binary
of it in the same file that I mentioned above.
