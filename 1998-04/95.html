Re: Customising the Afterstep desktop

> I'm brand spanking new to Unix and Linux, and struggling.

We're here to help!

> I've installed the RedHat 5.0 version of Linux, loaded xwindows and selected
> the Afterstep desktop. Can anybody tell me how to customise the buttons that
> appear down the righthand side of the screen?

I don't use it, but I think that's called the "Wharf".  If you want to
customise the default Red Hat 5.0 desktop, you need to look at the
files in /etc/X11/AnotherLevel, and if you want to modify one of those
files, you copy it to your home directory, then rename it so it has a
dot in front of the name.  Eg. if you want to modify the
fvwm2rc.defines file in particular then you'd do:

cd
  (takes you home)
cp /etc/X11/AnotherLevel/fvwm2rc.defines .fvwm2rc.defines
  (copies and renames the file)

> In particular I would like to remove the email button, and add a button to
> launch Netscape Navigator.

For customising the Wharf, you want the fvwm2rc.modules file, but I
find it strange that you don't have Netscape there already.  I just
switched to the AfterStep look, and it appears on my desktop just
below the email button.

If you want to customise your desktop much, you might want to start
with a configuration file a bit simpler than the Red Hat default.  To
do this, you need to get startx to use fvwm2 with the regular fvwm2
configuration file, then you need to set up your own config file.

Create a file .Xclients containing just the following two lines:
#!/bin/sh
fvwm2

Make the above file executable:
chmod +x .Xclients

When you do "startx" now, fvwm2 will start up with its default
configuration (not Red Hat's) (stored in
/etc/X11/fvwm2/system.fvwm2rc).  If you want to play with this, do:
cp /etc/X11/fvwm2/system.fvwm2rc .fvwm2rc

For information on the format of fvwm2's configuration file, run "man
fvwm2".
