> 
> Is there a RPMS converter thingy so taht i can install
> .rpm's on my slackware box?
> 

There is nothing stopping you compiling rpm for slackware.

You may need to manually confirm the dependencies and use
--force and --nodeps a bit, but that would be the best form.

Or alternatively you could get alien (IIRC) that allows you
to convert between various package formats.  I have never used it,
but you should be able to find them fairly easy on your
nearest sunsite. 

Finally since RPMs are made from pristine sources you should be able
to download the source from sunsite or similar (or install the 
source RPM and build the puppy yourself.
