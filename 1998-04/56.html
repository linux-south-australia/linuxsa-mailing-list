Hi Michael,

Like Tom mentioned, there is support for Joliet since kernel version
2.1.60, but there is not yet support in the standard kernels.

You can download a kernel patch that will add Joliet and FAT32 support
to 2.0.31-33 kernels from:
http://www-plateau.cs.berkeley.edu/people/chaffee/joliet.html
