SOURCEWARE PIONEERS HOLD SUMMIT

* SOURCEWARE PIONEERS HOLD SUMMIT
Numerous well-known developers of open source software (often called
freeware) recently held a summit to explore ways of expanding the use and
acceptance of their products. Tim O'Reilly, chief executive officer of
O'Reilly & Associates, organized the meeting.
  At the summit, open source software was defined as "software whose source
code is available so that users can customize or extend it." Attendees
agreed on several key points that underlined the importance of freely
distributing program code for scrutiny and development. Open source
applications such as Domain Name System (DNS) and Sendmail continue to
support huge portions of overall Internet functionality, and open source
applications such as Apache Web Server are the dominant platforms of choice.
  According to O'Reilly, spreading the word about the importance and value
of open source software is vital to the group's efforts. "Until Netscape
announced that it would release the source code to Communicator, open source
software received little attention in the press," said O'Reilly. "Now
everyone wants to know about it. It's important to realize just how
successful and widespread open source development is. Much of today's most
innovative and important software has been built using this [open source]
model."
  Notables who attended the summit include Linus Torvalds (Linux), Tom
Paquin and Jamie Zawinski (mozilla.org), Larry Wall (Perl), Brian Behlendorf
(Apache Web Server), Eric Allman (Sendmail), Paul Vixie (BIND DNS), and Phil
Zimmerman (PGP Encryption).
