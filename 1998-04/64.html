Have you got the kernel source installed?  If so, I think you forgot
to read /usr/src/linux/README.  If not, install the source (or at
least the include directory) and type:

cd /usr/include
rm -rf asm linux scsi
ln -s /usr/src/linux/include/asm-i386 asm
ln -s /usr/src/linux/include/linux linux
ln -s /usr/src/linux/include/scsi scsi
