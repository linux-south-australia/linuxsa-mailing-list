Ben, bash has very good "built in" help for those who are scripting.  For
example, if you type "help for", you will see that you are missing the
"done" statement.  Try "help if" and "help test" also.

You are also missing a semicolon; there should be one immediately after
the closing square bracket.  A semicolon is also required after NAME, i.e
before the "fi" - except if "fi" was on a separate line.

Regarding your need to "translate a number into a name", via an array -
this kind of array is called an "associative array".  Typical usage of an
associative array is as below (showing the reverse case to yours).

number["joe"] = 156
number["fred"] = 65
number["bill"] = 319

print number["joe"] + 1
157

Bash does not have associative arrays!  For this kind of task I use awk.
A good example of when to use associative arrays is when you wish to
convert a month number into a month name (a common requirement in the kind
of script you are doing).
