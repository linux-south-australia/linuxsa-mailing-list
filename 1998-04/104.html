Re: Language compilers in Linux

The DLX assembly language compiler is available from the Adelaide
Uni CS ftp server ftp.cs.adelaide.edu.au. I have not used this so I can
not help any more than this.

The ADA95 compiler, GNAT, is available from ftp.canberra.edu.au in the
/pub/ise/gnat directory. There are problems with getting GNAT to work with
RedHat 5.0 using the standard installation procedure. This is because of
the use of the new glibc 6 in RedHat 5.0 which is not yet supported in
GNAT. It can, however, be made to work by installing in a non-standard
location (e.g. /usr/local) and setting the environment variables as
described in the documentation that comes with GNAT. To compile Ada code
the option -b i486-linuxlibc5 must be added to the end of the gnatmake
command line. I have created a shell script which automatically sets the
environment variables, correctly executes gnatmake, and the unsets the
environment variables to avoid problems when compiling C code.

These difficulties do not occur using an older version of RedHat (e.g.
4.2) and will be resolved when a release of GNAT is made with support for
the new glibc 6.

Hope this helps,

Phil
