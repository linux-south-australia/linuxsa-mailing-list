> WHat is it?

> don't hve working btw)
> Is it an addon thing to xwindows?

Afterstep is a window manager for X, like fvwm, enlightenment, and
others.  The "Afterstep" desktop as supplied by Red Hat is a set of
configuration files for fvwm2 that make it look like the Afterstep
window manager.
