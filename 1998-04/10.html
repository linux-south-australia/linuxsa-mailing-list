good!

> BUT
> 
> I want to play with some of the startup files.
> 
> I have two problems
> 
> I think that the /etc/rc* files are the appropriate startup files (is this
> correct?)
> I cannot access these files ?  I can see them using "ls -lsa" but I cannot
> "cat" them or edit them ?
look in /etc/rc.d/init.d   The startup files are actually in there. The only
startup files in /etc/rc.d are rc.local rx.serial and rc.sysinit. If this is
different maybe you can send us an ls -lR output of /etc/rc.d

The DIRECORIES in /etc/rc.d have to do with runlevels. the file in all of
them (except init.d) are symlinks to file in init.d

> Questions are
> 
> am I looking at the correct files ? and if so how do I edit them?
you should be able to use vi or pico -w to edit them.
