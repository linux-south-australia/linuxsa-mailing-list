Re: Customising the Afterstep desktop

Depending on which version of afterstep you have
the configuration file is either ~/.steprc OR
~/GNUStep/Libraries/AfterStep/wharf

Look for a line that says somthing like
*Wharf asclock nil MaxSwallow "asclock" asclock -12 -shape &
and comment it out (place a # as the first char of the line). This will
remove the mail pixmap.

To add netscape to the wharf add the following line to your config file
(just after the clock line above)

*Wharf Netscape netscape3.xpm Exec "-" netscape &

Note that this assumes that your netscape binary is in your path.



FYI some good afterstep resources can be found at
http://www.afterstep.org
http://www.via.ayuda.com/~smw/afterstep/

hope this helps
