Re: Staroffice 4 & redhat 5.0

Hi,
  Just a little report on redhat 5 & staroffice 4.
I downloaded it about two weeks ago and after configuring the new libs
(just dumped all the libs from libc.5.4.44 in /usr/local/lib & added it to
/etc/ld.so.conf) it installed and worked fine.

It is a vast improvement over the previous version (and MUCH easier to
install). It has a cool integrated desktop - with build in preview mode 
for many image types including JPG, word docs (word 95 or lower) and many
other formats. It's web browser is very, good. It still doesn't top
netscape though (java is not up to scratch). The word processor has built
in spell check ie red underline like word 97. 

All up it's very good and I've been using it instead of word for a while.
It needs some refinements but overall it is very stable and apart from the
75mb Installation space (150 with help) required, it rules!
