Re: Dos and Samba

kjfarrell@ozemail.com.au wrote:

>I have some computers that are just going to act as public access for a library. The library software is dos based, so I thought I would find 3 of the crudiest computers and load dos on them and network them up to my Samba server that is acting as a PDC.
>
>Is it possible to log on to the server in dos and map a network drive so I can run the library software?
>  
>
There was a 'workgroups for dos' or similar client, although I can't 
remember if it did TCP/IP. Maybe check what the library software was 
designed to run with. Many old dos databases did not have very good file 
locking/sharing routines that could result in data corruption or poor 
performance at best if you're trying to share the same database between 
3 PC's.

Maybe you could attack the problem from a different angle and try 
running the software on linux with 'dosemu'.
