Something to check out might be CensorNet, I have recently installed a
box running this in a rather large organization and am finding it very
nice indeed. It does need another box to run on, as it is an entire
distro based on Debian I think, and uses DansGuardian and other gadgets,
with a nifty web front-end. It protects an entire network rather than
running locally for each box. And it gives nice reports too :-)
