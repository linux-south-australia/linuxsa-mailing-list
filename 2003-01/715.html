Re: tarball size limit?

Given that tar's can be gigabytes long if they happen to be on a tape
there should be no problem. For a tar that is REALLY large, though, it
could take REALLY long to get anything off it.

(tar's store data sequentially in the archive - they're not random access
and to read them the tar program has to read the tar from one end to the
other)
