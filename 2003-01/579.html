Re: Reinstalling Windows while keeping Linux

Remember that Redhat 8 uses Grub by default! Also, if you boot from the distro 
CD your harddrive will be mounted in /mnt so it may be a good idea to chroot 
/mnt before you do anything (so ure using the correct tools, conf files, etc)

for grub in redhat you can simply do

grub-install /dev/hda

and as far as I know it can even detect new OS's and add them to its grub.conf 
(on my friends machine, Mandrake was added to the grub boot menu when I 
reinstalled grub (he has RH8, MDK9, XP installed), at least I think it was, i 
Can't remember putting it in grub.conf...)

On Mon, 13 Jan 2003 9:34 pm, Andrew Pullin wrote:
> Hi There,
>     It is fairly straight forward to do what you want quite painlessly. Get
> your Linux distro and create a boot disk if you haven't got one already, or
> just boot direct from Disk 1 CD. At the appropriate time, type in Linux
> Rescue - you will know when, it prompts you for it. The disk will then do a
> few things and search for your Linux install. When it finds it - mount to
> it. When you have booted the Linux system in rescue mode, just run lilo,
> and it will write to the boot sector again. Reboot the machine and lilo
> should launch as normal and windoze should be blissfully unaware of it as
> always.
>
>     The command is just:
>         /sbin/lilo
>
> Hope this has helped.
>     Cheers!
>         Andrew.
>
> P.S. there are instructions in the Linux Pocketbook on how to do this. It
> is available from Australian Personal Computer Magazine still and the
> latest version supports RH 8.0.
