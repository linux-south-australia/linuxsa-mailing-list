One of the reasons why Linux isn't such an easy target as Windows at the
moment, though, is that:

a) you can't really use it out of the box so it forces you to understand a
   little bit about what you're doing
 - and that's a good thing

b) it's not widely used on the desktop
 - and that's not necessarily a bad thing

The moment you get default PostgreSQL or Oracle database packaged with
default users such as "administrator" with no password then it will start
happening. The moment you get users insisting on running as root [AND yes
they do--I won't even install any open source systems on a few of my
friends' machines because they refuse to not run as root and I basically
say you may as well stay with Windows if you don't get with the
program]--then you'll get problems.

The more untrained, non-administrators using a system the more likely it's
going to be fun to compromise that system on a worldwide basis.

Don't worry--I'm just waiting for the first Linux based Internet killer.
It's not something that might happen, it WILL happen at some stage...
