Supposing you already have an apt-capable cdrom with your packages on it,
"apt-cdrom add" will be enough to read the cd's packages, add it into to the
machine's database, and add the required line into sources.conf.

If I'm following what you're trying to do though (supply custom packages on
a cd), you're going to need to use certain debian tools to create packages,
then generate Packages.gz type files and have the right directory/file
structure on the CD.  I'd imagine these tools don't come with Lindows, but
you may be able to install debian stable (woody) packages via dpkg.

I'm not completely sure on what would be needed though; if anybody does, I'd
be keen to hear about it as well.
