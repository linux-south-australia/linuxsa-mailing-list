Re: El Torito Bootable Disks - Getting the Boot Sector

On Tue, 2003-01-14 at 12:12, David Lloyd wrote:
> 
> I am trying to extract the boot sector from an El Torito bootable disk.
> 
> I've tried:
> 
> # dd if=/dev/cdrom of=foo.img bs=2880k count=1
> 
> ..but when I burn that back as the boot sector it fails. Anyone know how
> to get such a beast off the CD?

IIRC, it's just a file on the CD. When you burn it to CD, you need to
specify (amongst other things) the '-b pathtoimageonthecd/whatever.img'
option to cdrecord. Cdrecord then writes a couple of bytes to a magic
part of the CD telling the BIOS that it should make use of that file as
an emulated (or not) boot image. You're also not guaranteed of the size
being 2880k.

Hope it helps,
Matt
