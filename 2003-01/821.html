Re: OpenOffice.org cant find X server

To reply to this topic, a number of things had happened:

1) Somehow all the authentication in the "startx" script had usefully
   been deleted or was not there
 - hence "startx" would eventually start X but then you'd get "client
   can't connect errors"

2) Somehow libpng got upgraded from 1.0.8 to 1.2.X
 - and Ian claims he has no idea why

After this it was a simple case of setting the DISPLAY variable
correctly.
