Re: Northern Neighbourhood Networks

> I was thinking the AP's could go on top of the building. If it got to 
> the stage of building towers, then paying for a Telstra link would be 
> more of an option. An extra $100,000 buys a lot of Telstra link - 
> especially if the PAPL option is still lurking.

An AP ontop of a building:

a) won't have a great deal of range - maybe a few hundred meters - you need 
a good height for distance (look up freznel zone).
b) won't allow you to communicate to another AP (you need a wireless bridge 
for that).

If you run links via a telco then $100k is going to get used up pretty 
quick.

> 
> I also wouldn't run low loss antenna cabling, since it'd be cheaper to 
> run an Ethernet link (fibre if necessary) up to the roof then place the 
> AP local to the dish. This is presuming the wireless is only for 
> connecting the buildings together.

Umm, much of a muchness.  Fibre's not cheap.  Probably more than low loss.
(termination costs mostly)
