Re: Smoothwall

Smoothwall's just a linux box - you can login to it (abeit via ssh on port 
222 rather than 22) and no doubt load whatever you want.

SW's not a full version of linux, so a lot is missing so it's quite likely 
that although you can get stuff onto it, it may not work due to missing 
libraries etc.

If running stuff on it is important, I'd recommend running a full install of 
<insert favourite distro> instead because it'll be easier in the end.
