Re: Courier-IMAP and Outlook

Sam Silvester wrote:
> 
> Next question: Can anyone recommend a good replacement for Outlook - it has
> to have IMAP support, multiple accounts, and a calender (preferably so that
> more than one person can add appointments etc)
> 

On Windows, I take it? Mozilla has excellent IMAP support, or at least 
I've never had any trouble with it, I've been using it since M18 or so. 
Download the calendar <http://www.mozilla.org/projects/calendar/>; add on 
(soon to be part of the default build) and you get calendaring support 
as well. The calendar still needs a bit of work, and it doesn't really 
support shared calendars, but it can use iCal files from the network via 
HTTP and (IIRC) WebDAV.
