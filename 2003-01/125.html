Re: kernel

It sounds like you haven't compiled module support.

Go into /usr/src/linux (or wherever your kernel is) and see if your
.config file looks like this:

#
# Automatically generated by make menuconfig: don't edit
#
CONFIG_X86=y
CONFIG_ISA=y
# CONFIG_SBUS is not set
CONFIG_UID16=y

#
# Code maturity level options
#
CONFIG_EXPERIMENTAL=y

#
# Loadable module support
#
CONFIG_MODULES=y
# CONFIG_MODVERSIONS is not set
CONFIG_KMOD=y

...you're interested in Loadable module support.

I suspect yours will say n.
