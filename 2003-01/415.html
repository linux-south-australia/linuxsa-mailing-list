Re: [ADSL] Technical question

More specifically, for ADSL based on Telstra xDSL DSLAM's (which is 
almost always the case), it is a Telstra decision that there has to 
be a paid, active, underlying voice telephony service on the copper.

You can pay someone else (AAPT, Primus, whoever) for it instead of 
Telstra if it makes you feel better, but you have to have it active 
and running in order for the xDSL service to be active and running 
too. If you cancel the voice service, the xDSL service automatically 
gets cancelled as well.

This is an economic decision, not a technical one, on Telstras' part. 
The voice service cross subsidises the xDSL service (or it would 
actually be -even more- expensive!)
