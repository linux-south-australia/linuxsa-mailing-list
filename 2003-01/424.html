Re: VPN and forwarding

On Fri, Jan 10, 2003 at 01:17:00PM +0930, Wayne Simes wrote:
> The symantec box has vpn settings which pass the connections through to the
> server 192.168.1.1, but since installing the linux box between the symantec box
> and server, vpn sessions do nt work. I guess this has something to do with the
> masqurading or forwarding and my lack of/incorrect settings.
> 
> Do I need to set up vpn on the linux box or should I be able to forward the
> requests via iptables ?

it depends on "which" of the VPN protocol/ports/layer you are using.
it may not actually be using TCP or UDP. some use other link layers
like IPsec does ...

if you can find out which type of VPN it is you could port forward
from either face of the your linux box to the internal and external 
servers/services respectively.

or terminate them on the linux box

or maybe you could find an alternative VPN/tunneling package ...
