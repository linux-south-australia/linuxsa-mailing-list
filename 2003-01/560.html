Re: Reinstalling Windows while keeping Linux

On Mon, Jan 13, 2003 at 09:04:37PM +1000, Andrew Pullin wrote:
> Hi There,
>     It is fairly straight forward to do what you want quite painlessly. Get
> your Linux distro and create a boot disk if you haven't got one already, or
> just boot direct from Disk 1 CD. At the appropriate time, type in Linux
> Rescue - you will know when, it prompts you for it. The disk will then do a
> few things and search for your Linux install. When it finds it - mount to
> it. When you have booted the Linux system in rescue mode, just run lilo, and
> it will write to the boot sector again. Reboot the machine and lilo should
> launch as normal and windoze should be blissfully unaware of it as always.
> 
>     The command is just:
>         /sbin/lilo
> 
> Hope this has helped.
>     Cheers!
>         Andrew.
> 
> P.S. there are instructions in the Linux Pocketbook on how to do this. It is
> available from Australian Personal Computer Magazine still and the latest
> version supports RH 8.0.
