A more difficult way -- not at all. But there are many other 
"undifficult" ways. Adding "exec startx" to the end of 
$HOME/user/.bash_profile is an excellent way for a newbie to become 
familiar with Linux. He is greeted with a console in the traditional 
manner: if he logs on as root he remains at the console; if he logs on 
as user he goes straight into his default GUI.

Has the added advantage that user has actually done something useful 
with a Linux config file -- quite painlessly and without bringing an 
end to civilization as we know it.
