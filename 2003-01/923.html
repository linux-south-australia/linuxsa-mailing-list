Well, after piddling about a bit, it wasn't that hard...

For those wanting to know, the basics (as I said, I had only simple procmail
recipes) are as follows:

in a file ~/.mailfilter, the procmail recipe:

:0
* ^X-Loop.*linuxsa@linuxsa\.org\.au
$MAILDIR/.lists.LinuxSA/

can be translated as follows:

if (/^X-Loop.*linuxsa@linuxsa\.org\.au$/)
	to $MAILDIR/.lists.LinuxSA

It's fairly simple... there are commands like "to" and "cc", which are
self-explanatory, and the pattern matching is similar to grep (though
not quite the same), and the pattern should be contained within /
characters, like perl. That was about it for my set of translations --
I'm sure a simple translator could be written by someone who likes
their perl, though more complex recipes would require some thought.
