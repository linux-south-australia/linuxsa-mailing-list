Re: OpenOffice.org cant find X server

Hi Ian,

first determine if your display is set at all! You can do this by the following command:

]$ echo $DISPLAY 
:0

If you dont get a simialar result, you need to set the variable. This can be done in this manner:
]$ export DISPLAY=:0

I hope this helps!

Cheers,

Ben.
