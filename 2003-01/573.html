Re: RH default firewall settings

At a root prompt type...   setup
That'll let you config all sorts of things.

Or you could type lokkit for just the firewall settings. Choose
customise to open various ports.
I personally on my rh8 box use MonMotha's firewall available from
http://monmotha.mplug.org as it only takes a few minutes to setup and
have a full working lan environment.

If you use a third party firewall you should run (again as root) the
"service iptables save" command to save the rulesets to
/etc/sysconfig/iptables file for use on furture boots.

HTH
