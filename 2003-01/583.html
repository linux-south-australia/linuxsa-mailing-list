Procmailrc problems

Greetings all...
Been a long time since ive posted to this list.

Im trying to get procmail to compress emails as it recieves them.
But ive been having problems, so im now using this test procmailrc:

-----------------------------------------------

#assignments
PATH=/usr/bin;/usr/local/bin
SHELL=/bin/sh
MAILDIR=$HOME/.mail
DEFAULT=$HOME/.mail/INBOX
LOGFILE=$HOME/procmail_log

#  write everything to a file
:0 c:
$MAILDIR/test/EVERYTHING

# archive all mail to the file
:0 cw:
| cat >> $MAILDIR/test/TEST_CAT

# archive and compress all mail to the file
:0 cw:
| gzip >> $MAILDIR/test/TEST_GZIP.gz

# linuxsa list
:0:
* ^(From|Cc|Bcc|To).*linuxsa
LINUXSA

-----------------------------------------------

EVERYTHING, LINUXSA and the normal INBOX are all working fine.
But TEST_CAT and TEST_GZIP.gz are always just empty files (0 bytes).
According to the many procmail howto's and man pages ive read, my cat and gzip
rules should be working.

Can anyone explain why they are not?
