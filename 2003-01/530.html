Re: LCD Screen

Hi.

I have a Benq 17 inch (752?) which is a renamed Acer 751. It works
natively at 1280x1170 or something like that. The only issue I have had
is if you set it up to run in a different resolution in X, it looks like
crap, because the screen is having to map the new resolution onto it's
native resolution.

For the stuff I do (mostly internet and text based stuff, no 3d), it's
brilliant. Best monitor I've ever had! I've run RedHat, Mandrake and
(now) Debian. I did accidentally test it with one of those feelthy
closed source OS's and there were no technical problems with the display
at least. <grin>
