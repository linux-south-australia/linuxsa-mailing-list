RE: XMMS not working

On Mon, 2003-01-06 at 11:05, Andrew Galdes wrote:
> 
> >  * Check to see which sound system is configured / in use by RedHat
> > (under Gnome, it'll probably be esd and probably arts under KDE)
> 
> Other KDE sounds work - like on events.

Check the preferences (under the options menu when you right-click on
XMMS) and make sure that the output plugin is set to aRts. While you're
at it, it's probably worth checking that there's an input plugin that
looks like it's able to decode whatever you're trying to play (eg, the
libvorbis.so module if you're trying to play .ogg files).

Hope it helps,
