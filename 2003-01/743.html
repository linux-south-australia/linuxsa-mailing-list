Re: No More AMOR please

Others have hinted at searching for startup agents.  My humble advice is to
look at the session manager.  Both gnome and KDE (and CDE - Solaris) have
these things and if you are not aware of them, you may be looking in the
wrong place.

I'm currently using KDE and have done this:

control centre -> system -> session manager -> don't restore previous session

There may be another part to the picture whereby the user is able to tell the
session manager to remember the current session; but I'm unable to find it
after a quick look, any clues out there??
