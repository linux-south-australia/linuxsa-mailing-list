User groups

I would like all files being created in a directory to belong to group
ADMIN_STUFF but the creators of this file cannot be in that group.

The point of this is to make the files accessible by those in the
ADMIN_STUFF group and the owners but not others. The creators of the files
in the directory cannot have access to each others files. The ADMIN_STUFF
group members have access to all.

In the following example, "myuid" in not a member of the "admin-stuff"
group. How can I automate this?

-rwxrwx---    1 myuid    admin-stuff    20585 Jan 29 14:08 myfile

Files created in this directory are created through the use of Samba, remote
NT machines.
