Re: [ADSL] The Broadband Battle

Adam Smith wrote:
> 
> I've always wanted to be able to directly host
> my own domain, so I can run my low traffic personal website and collect
> my own email etc.  But I can't do that efficiently, because I have a
> dynamic IP.  

Well I'm on a regular dialup and don't have my own domain, but I just
get one free from www.no-ip.org and run my own web server at
http://awol.no-ip.org. I also get my MX record setup there too, so I can
run my own mail server and mailing lists etc...

My isp has one of those busy killers that kicks me every 6 hours too,
but no-ip.org gives you a simple C program that you run just the once
and it auto-updates the dynamic IP everytime it changes. There are
plenty of other free Dyn Hosts around too. Google as always is your
friend.

I know it's not quite the same as a static IP and domain, but hey! It
works for me!  ;-P
