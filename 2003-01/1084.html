Im on the dev list, and they've been developing it quite quickly for
several years now. It grew out of dvgrab. I havn't used it in a very
long time though, although I would give it another shot. It used to have
some annoyances with problems with compatibility between some of the
supporting libs versions, and each other, and one particular bug in the
MMX code of the dv playback library (which you had to patch the source
to disable mmx permanently to use it at all). That said, I think those
problems are now sorted out. Give it a shot!
