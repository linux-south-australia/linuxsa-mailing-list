I'd say try debian anyway - It has the most number of open-sourced 
packages packaged of any distro by a country mile, and if there isn't 
something officially packaged there are often un-official versions 
around, or you can always build it yourself. Using the testing branch 
gives you a decent stability/cutting-edge trade-off (provided you have 
decent internet connectivity), and you can always get sources from 
unstable/experimental and build them locally, or mix testing and 
unstable - although this can get messy if there are too many  related 
packages that need updating.
