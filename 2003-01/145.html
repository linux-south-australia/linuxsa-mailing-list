Re: XMMS not working

On Mon, 2003-01-06 at 10:26, Andrew Galdes wrote:
> I have custom installed RH8.0 over a previous install of suse and found that
> xmms isn't working from any user including new users. 
> 
> When starting xmms, it loads fine. however, it will not play anything.
> Pressing play will bring up the "Open" window asking me to select a song to
> play. Selecting a song and pressing play just repeats the story. 

 * Check to see which sound system is configured / in use by RedHat
(under Gnome, it'll probably be esd and probably arts under KDE)
 * Ensure that XMMS is using the sound system it's supposed to be
 * Check that your users have write access to /dev/dsp (not sure if this
is necessary when using something like esd or artsd)
 * Can you make noise using a different program? Try playing an mp3 or
ogg vorbis file using mpg123/mpg321/ogg123 in a terminal window and
playing around with the -d option (check the man page for more details).

Does XMMS look like it's playing (timer incrementing, animations
animated, etc)?

Hope it helps,
