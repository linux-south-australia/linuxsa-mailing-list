Re: Dvd Region Setting under Linux

Brian Marr wrote:

> Current DVD firmware allows 5 changes to be made to the region setting or your
> dvd drive. While there is an abundance of Windows tools to hack dvd firmware
> or change the region setting, stuff for Linux is harder to come by.

[...]

> suse:/home/blackbird/dvd_disc_20000215 # ./regionset /dev/dvd
> Current Region Code settings:
> RPC Phase: II
> type: SET
> vendor resets available: 4
> user controlled changes resets available: 4
> drive plays discs from region(s): 4, mask=0xF7

I'm not sure if you're inferring it or not, but it doesn't look like
that software gets around the fact that RPC II drives hold the
"region change" counter in the firmware on the drive itself, and
the drive is still limited to the "5 chnages and that's it" policy.
The only way to circumvent that has been to load RPC I style firmware
onto the DVD drive, if you can find it.
