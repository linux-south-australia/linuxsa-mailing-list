Linux window managers

At tonight's (15-07-97) LinuxSA meeting someone asked about different 
windows managers .

http://www.plig.org/xwinman  is a good place to start looking at as it 
details, compares and provides links to several (about 14) different 
windows managers. It also provides example screen shot for each manager.

http://www.plig.org/xwinman  also has links for icons & textures etc that 
can be used with your window manager of choice.

My personal favourite is Afterstep, but Window Maker could be interesting. 
 For out and out "wow" check out Enlightenment (Australian grown).
