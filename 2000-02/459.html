Samba out of the box and browsing

Hi,

Something that is bound to bite people with Samba is the default smb.conf
file.

Most Linux distributions are shipping with Samba 2.0.5a and they have the
following in the smb.conf file (some way down)

;  os level = 33
;  local master = yes
;  domain master  = yes
;  preferred master = yes

They are not all together.

The result of the above, and the fact that prior to 2.0.6, the default for
os level was 0, means that you will not be able to browse a Windows network
in which Samba is the only server.

If you uncomment the above statements and then restart your Samba server,
things should be a whole lot better.
