>>>>> "Ant" == Ant  <fineales@ozemail.com.au> writes:

    Ant> Can some name a program that I can run on an NT machine to sync
    Ant> the time with a linux server?  I know it (or they) are out
    Ant> there but finding them is near impossible as all the key words
    Ant> turn up pages of irrelevant crap!

If you have samba running on the Linux box you can use "NET TIME
\\LINUXBOX /SET /Y" from the NT machine without doing anything much at
all.

Otherwise, install xntpd on the Linux machine, and have a look on
www.tucows.com for an 95/NT NTP client.
