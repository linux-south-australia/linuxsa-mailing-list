Re: automatic root login

Michael Neuling wrote:

>> Basically I want the machine to boot to run level 3 and then
>> automatically log root in (say on tty1).

> in your /etc/inittab replace
> 
> 1:2345:respawn:/sbin/mingetty tty1
> 
> with somethine like
> 
> 4:3:respawn:/bin/bash

(ignoring the typo, which you subsequently corrected).

This won't associate bash with a terminal (unlike getty, which attaches
itself to the terminal supplied as a parameter), so you'll most likely
wind up with a shell attached to /dev/null (or whatever init passes to
the spawned executables on descriptors 0 to 2).  You would also need to
set the terminal to some sort of default state (like the equivalent of
a "stty sane").  I do believe someone wrote a utility for doing this
with
any executable (not just bash), but its name escapes me :-(.
