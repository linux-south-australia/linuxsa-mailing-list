Re: packages

Robyn Manning wrote:
> Which packages do I need to install for a basic Redhat 6.0 installation?

A really basic installation needs /boot/*, /etc/init, /lib/modules/*,
maybe /lib/ld-linux.* and /lib/ld.so*, and /bin/sh.  You'll need to
write an appropriate /etc/inittab, two lines should do it.  This is
very minimal.
