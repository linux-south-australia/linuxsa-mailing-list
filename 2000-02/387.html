RE: Squid vs Browser cache

> I have installed Squid on my system. How do I set the
> browser to not cache any file and use Squid's cache
> instead? This will also prove my Squid's caching cofiguration
> to me.
> 
> Thank you for any pointers. 

In Internet Explorer, go to the Tools menu and choose Internet
Options, then muck about in the General/Settings tab and the
Connections/LAN Settings bit.

In Navigator, the process is similar.

Change the memory and disk caches to 0 bytes, and all should be
as required.
