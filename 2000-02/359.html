Samba and hide dot files

Hi,

Today in the Samba course that I ran over the last two days, we came up
against an interesting problem.

It appeared that the hide dot files parameter in Samba, which by default is
on, was not working.

However, I decided to look into this tonight, and found the following.

On my system, file that start with a dot (ie, the dot files that are
supposed to be hidden) are indeed marked with the DOS/Windows hidden
attribute.  However, if you have configured explorer to show all files, and
it may be that Win98 does just that, all files are shown, including dot
files :-)

So, I think the problem is resolved ...
