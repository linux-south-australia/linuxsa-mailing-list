Re: Linux on Laptop - yippee

On Wed, 23 Feb 2000, Bruce Moffatt wrote:
> Hi all,
> 
> The installation hw probe found the S3 ViRGE/MX 2MB card, and selected
> the SVGA server. Should it be running an S3 server?

Nope. SVGA is fine. the S3 virge was fully incorporated as of Xfree 3.3.3 and
your xfree is probably later than that. The S3 server you refer to is for 486
style VESA and earlier card combos including the Trio series. It will run ok if
you choose to use it but X uses autodetection to select 'the card at the moment'
type approach. Although not applicable to a laptop as such, in theory, you can
swap out your vga card for another one, and the svga server will detect in and
run accordingly. That's the theory.
