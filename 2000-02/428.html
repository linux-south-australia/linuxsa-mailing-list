Install using the working Kernel. Recompile the kernel to make one that
is customised for your machine. APM is not essential (but handy), but
you want to have PCMCIA support. You will need to have the kernel
sources and gcc, etc. to be able to recompile the kernel (yet, contrary
to popular belief, you don't need to be a programmer),
