It depends upon the implementation you want.

The quickest minimal solution, is to run an online database
(Postgress/MySQL), and have it speak ODBC to the existing code.

Alternatly, using the Postrgess/PHP3 compination provides for a very
portable interface, as the only requirement is a Web Browser.

Whatever you do, a significant ammount of coding will be needed, as the
only portable structures are the Table Layouts, and the internal data. The
VB is verry MS windows and Language dependant.
