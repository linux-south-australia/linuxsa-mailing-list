Re: Squid config

Subba Rao wrote:

> How does Squid keep track of the files being cached? How do we know if the cached files have
> changed on the original server?

Squid is smart enough to work out when to re-read a web document based
on a number of factors, such as "expires" headers. When a document is
requested, squid can use the "HEAD" function in HTTP to see if the
document has been modified since it was cached. It should only retrieve
it if it is "fresh". Hitting the refresh button in IE (shift+reload in
Netscape) sends squid an instruction to check the page again.

You can override this using refresh patterns or squid's offline mode.
