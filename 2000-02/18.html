How is your DNS set up? check your /etc/resolv.conf to make sure you have a DNS server set up.
Also check to see if that DNS server knows where mail.chariot.net.au is. You can do this with
'nslookup mail.chariot.net.au'. If this works it's most likely your mail configuration, else it's
your DNS configuration.

Hope it helps,
