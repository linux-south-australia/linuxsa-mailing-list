Suggestion for Andrew ... just create a new user - get into the dialer &
you should have a nice shiny new default conf file ... if that fails -
reinstall whichever dialer you were using (rpm -i --force kppp*  or
similar to force it over the top) - and.... delete any dialer conf files
from your  users directory first

Also check /var/log/messages & run the debug option with your dialup
app.

Hint: before dabbling you should copy conf files :))
