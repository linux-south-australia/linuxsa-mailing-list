Re: Short Message Service for Mobiles

Peter Fitzsimons wrote:

> Has anyone implemented a gateway (SMTP /POP3 / Web) to the Australian Telcos
> ?. I have installed sms_client which provides the basic connectivity to the
> telco, but want some way that non linux users can easily send messages..

I'm not sure quite what you want, but SMTP to SMS gateways
are pretty popular.

Internode has one, which they charge a moederate amount for.

The usual application is alert you, via your mobile, of
outages.  OpenView, Patrol and so on will happy produce
e-mail alert notifications, which you can forward to
the SMS gateway.

The standard network manager e-mail messages do tend to be a
bit wordy for reading on a phone.
