The last mention on the kernel mailing list suggested the maximum size
linux is able to format and use with a filesystem was 2TB (Terrabytes).

The largest report I've heard of someone actually using was 206 GB.
If you exceed this (hmm, wouldn't that be nice) let me know :)

As for a single drive there is many reports of people using 27GB drives
without problems.
