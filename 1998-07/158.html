Problem connecting to my ISP.

I recently installed rh 5.1 linux on my system. i can not, however, seem
to get connected to my isp. i followed the 'how to hook up to your isp'
HOW-TO almost verbatim, but i still can not connect to my isp.

here is what happens:
i've tried several methods of connection. first, i started up minicom. i
typed 'atdt 555-5555', the number of my isp. when it connected, it asked
me for my user name and password. i entered both correctly. now, minicom
prints nothing more on the screen, except maybe jarbled computer symbols
when i press a key. usually it will disconnect after a minute or so.
while it IS connect, i pressed ALT-F2, and run lynx, but the program can
not access the WWW.

next i tried using the how-to. i set up the name server, i edited
/etc/hosts and /etc/networks and etc/ppp/pap-secrets (substituting my

/etc/ppp/chatscript file exactly as the how-to instructs. next i went
into x windows system and i entered:

exec pppd connect \
'chat -v -f /etc/ppp/chatscript' \
-detach crtscts modem defaultroute \
user ELN\padrino \
/dev/modem 38400

using the internet, i would truly appreciate it.
