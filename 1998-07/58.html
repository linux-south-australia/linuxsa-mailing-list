Yep.  If you are using run level 5 (ie have a graphical login screen for
linux
using xdm) you need to make these changes to 
/etc/X11/xdm/Xsession :

After installing KDE, edit /etc/X11/xdm/Xsession and add
the following lines after the line " # clean up after xbanner " :

KDEDIR=/opt/kde    # if kde is in this dir
LD_LIBRARY_PATH=$KDEDIR/lib
PATH=$PATH:$KDEDIR/bin

Edit your own .xsession, or .Xclients and add 
exec /opt/kde/bin/startkde

I also found that I had to chmod u+x ~/.xsession

This should get you going.  

If you don't use the Linux Graphical Login screen, (use normal 
text on black), then you won't need to do the above.  You will only
need to do what they sugest in the instructions re your personal 
.xinitrc file (ie exec /opt/kde/bin/startkde).
