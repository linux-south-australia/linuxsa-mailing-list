Re: copying HDDs

On Wed, 8 Jul 1998, James How Lum wrote:

> Was wondering how to easily mirror the contents of one HDD to another
> identical HDD. Have the ability to install both to the one machine. The
> second HDD already has linux installed, but I'd like it to have all the
> packages and configurations as the original. Heard the 'dd' command was an
> option?
first make sure the partitioning is exactly the same on each drive. Then
say you wish to copy hda onto hdc 
foo# dd if=/dev/hda of=/dev/hdc

read the dd man page for more information on options but dd is the tool
that you wish to use :)
