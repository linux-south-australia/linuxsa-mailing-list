Re: linux and novell

works a treat - man ncpmount (if you use redhat its in the ncpfs
package)
you also need IPX networking setup, the easy way is to install mars-nwe
package
(netware emulator which turns a linux box into a novell file and print
server)
which will bring up the IPX routes etc when it starts

I use these options

ncpmount -S <servername> -U SUPERVISOR -P <novelPassword> -u
<ownerOfMountpoint> -g <groupOfMountpoint> -f <mask> -d <mask>
<mountPoint>
the result looks like:
ncpmount -S NW_ADL -U SUPERVISOR -P MYSECRET -u adrian -g adrian -f 775
-d 775 /mnt/novellbox

slist shows novel servers on network viz:
> [adrian@loraine adrian]$ slist
>
