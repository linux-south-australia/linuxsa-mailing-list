Re: copying HDDs

> > Was wondering how to easily mirror the contents of one HDD to another
> > identical HDD. Have the ability to install both to the one machine. The
> > second HDD already has linux installed, but I'd like it to have all the
> > packages and configurations as the original. Heard the 'dd' command was an
> > option?
> 
> You can use dd, but cat is easier to remember.  If you want /dev/hdb
> to end up identical to /dev/hda (and provided that the disks are
> identical), type:
> 
> cat /dev/hda > /dev/hdb


ditto - this works so well it stands up in court ;-)
