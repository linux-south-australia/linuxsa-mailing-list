Re: memory

On Tue, 28 Jul 1998, Ben Kramer wrote:

|I have just installed more memory to 96MB but linux doesnt recognize
|more than 64MB...  how do I tell it that there is more....

You need to tell the kernel at boot time how much ram you have.
A parameter such as "mem=__M" or "mem=_____k" should work. If you use
lilo, add this to /etc/lilo.conf and rerun /sbin/lilo. If you use loadlin,
just add it on the command line.

Make sure you get it right though ...
