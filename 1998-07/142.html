RE: LCD screen cost

I bought a 2 row by 16 character LCD (POWERTIP) kit from Altronics for about
$40-ish (they also sell a 2x80 LCD kit).

It plugs into the parallel port, is currently powered by a 9VDC (via a plug
pack) which is then dropped down to 5VDC by a separate 78L05 5VDC 100mA
regulator - a very PC compatible voltage I thought at the time :-)

Currently, the only software they supply is for dos (I've written a VB4
version - uses third party parallel port DLL).

Time permitting (big problem), I intend to write a version (or see if
LCDproc will handle with a suitable patch etc) for Linux over the next 4-8
weeks.

I may be able to do it sooner but work, home-life (proud father of a 10 week
old boy), plus TAFE will be keeping me busy.

Whilst the POWERTIP display does not(?) appear to have all the flexibility
of the Matrix item, I think it might be able to achieve most of the
functionality, and the price is a bit more affordable (for me at least).

Feature set (from memory so I may have left something out etc):

	own on-board processor
	accepts straight ASCII code
	parallel port compatible
	8 end-user customisable characters
	240 characters inbuildt (a..z, A..Z, 1..0, plus Japanese(?) chars, plus
some useful graphics)

Altronics supplied some (very average, IMHO) code in MS-QBasic which gives a
starting point for driving the unit.

If anyone wants more info let me know and I'll bring it along next meeting.

NOTE: I got Altronics to send me more tech. info on the board so if anyone
wants a copy let me know and I'll photocopy for the next meeting.  I found
it very handy as it details the full capabilities of the LCD display.

Cheers,
