Re: Starting PPP

Tony Pearce wrote:

> I have just installed RedHat 5.0 and have got it up and running,
> including Netscape etc, having avoided scripts entirely thus far. Its
> probably safer that way!!!
>
> It is however a little difficult to log on to my ISP. The only working
> method at the moment ia as follows:
> boot
> open X (as me, not root)
> su root to access Control Panel
> activate PPP interface
> open lynx/netscape etc

In RH 5.0, there is a program called "usernet" which you can use if you
allow the ppp to be activated by normal users. (In netcfg from
control-pannel, you can set the ppp interface to allow user to activate
it.)

The other solution is "sudo" (in contrib directory) which will allow
certain users to run program as root (like halt/reboot/control-pannel).
