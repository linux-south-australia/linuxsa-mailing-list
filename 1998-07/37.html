Getting KDE Working (HOWTO)

Hi all,

just as a matter of interest, I thought that I would write
and let you know how to get KDE working under Linux
run level 5.

After installing KDE, edit /etc/X11/xdm/Xsession and add
the following lines after the line # clean up after xbanner :

KDEDIR=/opt/kde    # if kde is in this dir
LD_LIBRARY_PATH=$KDEDIR/lib
PATH=$PATH:$KDEDIR/bin

Edit your own .xsession, or .Xclients and add 
exec /opt/kde/bin/startkde

I also found that I had to chmod u+x ~/.xsession

This should get you going.  

If you want to try out KDE, check out http://www.kde.org/
It is in beta 4, but seems pretty stable.  Any old OS/2'ers amongst
the list will probably like it heaps - it has templates & Drag 'n' Drop
like Presentation Manager.
