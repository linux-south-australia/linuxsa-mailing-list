Re: PPP and IP Numbers

> Hi all,
> 
> I would like to thank everybody for their prompt replies to my netscape
> problem a fair while back. I now have another question.
> 
> What I would like to do is create a shell script that updates a web page
> with include my current (dynamic) IP address and then to FTP the page to
> my server. I know the HTML and it is a simple matter of having the shell
> script echo it out to a file (it's only a few lines for a redirect page)
> but I don't know how to make it put in my current IP address.

When a PPP connection comes up, the /etc/ppp/ip-up shell script gets
called with a number of parameters, one of them being your IP address.
There is also the added bonus of not needing to run a shell script
manually when you connect.

If you have a Red Hat system, you should put the stuff you want in
ip-up.local (as is documented in ip-up).
