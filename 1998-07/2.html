Do you have the linux kernel source installed and the
/usr/include/{asm,linux,scsi} symbolic links set up? 

Once the linux kernel source is installed under /usr/src/linux, which is
normally a symbolic link to /usr/src/linux-2.0.34 or whatever version you
happen to be using, run:

                cd /usr/include
                rm -rf asm linux scsi
                ln -s /usr/src/linux/include/asm-i386 asm
                ln -s /usr/src/linux/include/linux linux
                ln -s /usr/src/linux/include/scsi scsi

These commands were taken from /usr/src/linux/README and will set up
the symbolic links so gcc can find the header files it wants. Another
way to get the links set up is to start configuring the kernel. The first
thing it will do is make sure the links are in place.
