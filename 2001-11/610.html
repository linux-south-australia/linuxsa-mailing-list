Re: forwarding web requests

ipmasqadm isnt that great though. It seems to work, but only on the
first interface that it finds in your system. For example I was trying
to use it to redirect to a trial mail server, and for external hosts it
worked find, but on the internal interface the mail port just stopped
receiving connections. Since then I've found nportredir which works like
a charm.

Sucky site, but you can get it from 

http://www.asymmetrica.com/
