James,

I'm not familiar with mysql but it's a generic problem.
Generalisation here:
Most SQL databases maintain a table of table definitions which also
maintain a table of column names.
You need to find this table issue a select on it to extract your column names.

eg <poetic license>
select columnname from syscols where tablename = "product" and columnname
like "extra_*"

Alternatively just issue "select * from products" and loop through only
extracting the values from the appropriate columns whose name matches your
filter via some form of case statement/if statement.
