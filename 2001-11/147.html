RE: help on telstras cable

Good place to start is

http://whirlpool.net.au

The "easiest" way that I know to share Cable securely is to set up a
smoothwall box that will do everything for you
www.smoothwall.org

Here is a link to setting up a smoothwall box using BPA Cable. You only need
a 486 with 16 MEg Memory and 100 MEg Hard drive as a secure Router/Firewall.
If you run Smoothwall on a better PC you can activate Squid for extra
performance.

http://whirlpool.net.au/howto/smoothwall/

I've got 2 friends using BPA Cable and they know nothing about Linux until
now but they got a smoothwall box up and running quite easily.
