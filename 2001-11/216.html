RE: I need help using linux

I would suggest that the little old Ladies stick to Windows 95/98. There are
accessibility options that can make fonts and the like REALLY big.

Windows 98 is a lot less complicated than most Linux distributions.

As for Hardware requirements there is not really that much difference with
the current versions of Linux or Windows. Both require large hard disks ( at
least 2 gigs ), 128 Meg of RAM and a fastish Pentium II or better processor.

The only exception to that is if you don't want any graphical interface, but
I think little old ladies will need a graphical interface !!.

Windows 98 is not complicated... It is quite different from Windows 3.11 but
there is a lot of help available both online and from the community
including libraries etc.
