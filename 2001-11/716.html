Re: Routing - Best Practices

Matthew Western wrote:
> 
> what's the best way to do routing????

Best practice is to use gated or zebra to dynamically
learn routes.  Use OSPF with authentication.

/etc/sysconfig/static-routes is correct if you don't
want to dynamically learn routes.

If you only have the default route then the best
practice is for the network to tell the host
about this in a DHCP response.

If you need to hard-code the default routes, use
GATEWAY and GATEWAYDEV in /etc/sysconfig/network.

> > what's the best way to do routing????
> ...use FreeBSD.

isn't a useful response.  The API is the same and
the procedures broadly similar.
