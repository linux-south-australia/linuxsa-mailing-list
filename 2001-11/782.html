Re: Linux ISDN Problems

>We're having some troubles setting up Traverse ISDN Netjet cards under
>linux. We are trying to run 3 cards in the one box for DOV dial-in for
>three separate sites. However, when we have three cards in
>simultaneously computer freezes whenever we try to initialise the Hisax
>module.

The traverse cards use DMA to move bytes around the motherboard, which is
often a cause of conflict if more than one device tries to do it at once.
Some UDMA controllers interfere with the Traverse cards as well.

If you can, try the 3 cards in a vanilla PCI motherboard, like a Pentium 133
setup, and see if you still have the same problem.
