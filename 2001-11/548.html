Re: Family Internet Connection

Aristotle wrote:
> 
> Hi people,
> 
> I am running Mandrake 8.1 - how can I have an internet connection that
> my girlfriend can bring down in one account if it was brough up in
> another?  /sbin/ifup ppp0 doesn't seem to work - the modem flashes
> briefly and then the command stalls.
> 
> Any ideas?
> 
Assuming there has not been a reply for this.. try:

sudo /sbin/ifup ppp0

You will need to edit /etc/sudoers
user	ALL= PASSWD: /sbin/ifup

Unfortunately, I don't have a dial up connection here to test it..
