RE: telnetd

The other way is to go to
http://www.chiark.greenend.org.uk/~sgtatham/putty/ and grab a copy of
PUTTY, which is a SSH (Secure Shell) client for Windows (I assume you
are trying to telnet from a windowz box :) ).  To use ssh from another
redhat box, just type ssh <host> (assuming you have the ssh client
installed).

SSH is enabled by default on Redhat 7.x boxes as far as I recall.

If you dont know what SSH is, it is basically an 'encrypted' telnet
session, where all traffic (including usernames and passwords) are
encrypted between client and host.

Hope this has helped.
