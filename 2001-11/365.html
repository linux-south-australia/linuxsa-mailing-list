Re: RH7.1/7.2 Install and Promise ATA 100 raid NEED HELP

pmullins@bigpond.net.au wrote:

> Does anybody know the 'correct' way the make it do a hardware array 
> using the promise controller..

Yes. You need to patch the BIOS of the Promise controller. There are 
plenty of links around about how to do this.

However, I'd strongly advise that you don't, and install your Linux 
setup using software RAID. I've discovered the hard way that the Promise 
RAID controller is not worth the effort, as you'll be in big trouble 
when a drive fails and you want to recover the array from the other drive.
