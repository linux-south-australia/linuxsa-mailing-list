RH7.2 / ext3 / kernel 2.4.13 issues??

Hi all

I've installed a RH7.2 box & am putting Samba 2.2.2 thru its paces.  
So far everything works great with a Win2000 SP2 workstation.  
The roaming profile problems I had with 2.2.1a have disappeared and its
working great.  Kudos to the Samba team, they've done it again.
(dang! another nail in NT's coffin, my heart bleeds........ :o)

Now, RH7.2 installs with ext3 as the default filesystem (i did a
server install, so YMMV with the workstation/custom installs)
So far it's got my vote, I started copying about 200mb of files
to one of the Samba shares then powered off the server halfway 
through, came up without a hitch, did lose some of the files, 
but all in all quite impressive.  (Most of the Samba boxes we run
at present use reiserfs)

Anyway, I've got kernel 2.4.13 and am about to compile/install/test it.
I'm searching around for any known bugs/issues with 2.4.13/ext3
etc, and thought I'd run it by the list out of curiosity to see
if anyone's played about with this combination yet.  
If so how did it go?

Feedback welcome, TIA.
