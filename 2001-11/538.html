you're doing it all wrong (and you don't need any drivers or
modules or whatever for modems in linux).
Go to the smoothwall modem or dial or whatever it's called page not the
network settings page.
if you plugged it into the first serial port, select COM1 (which is
ttyS0).  second serial port, select COM2 (ttyS1).
I just played with smoothwall last night and did exactly this
(although I can't remember the name of the web page in smoothwall
but it's obvious it's for a modem).
