Gettting the latest FreeBSD sources ...

Hi,

A 30-GB IBM diskdrive was delivered by FedEx last week. It contained 
FreeBSD-4.3-current.

"I can handle this," I thought, so I put it in my 750MHz Duron and ran 
some netbench tests over a 100Mb/s link between Linux 2.4.13 and FreeBSD 
4.3.

A 10-client NetBench run gets about 60Mb/s with Linux as a server, while 
with FreeBSD 4.3-current, it gets about 20Mb/s.

Now, I know that FreeBSD 4.4 is out, so this strikes me as an unfair 
comparison.

Can anyone tell me how I can quickly upgrade the kernel to the 
4.4-current version?

Observations made during the runs suggests that Linux is getting more 
concurrency that FreeBSD (perhaps FreeBSD has a big file system lock or 
something). FreeBSD had a load average always under 1, while Linux got 
up to 7 or 8 during the run.
