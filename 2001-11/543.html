[ANNOUNCE] LinuxSA November 2001 Meeting

Hi all,

Time for the LinuxSA November 2001 Meeting...

Here are the usual details:

  When:   7:00pm (doors open 6:45pm) on
          Tuesday, 20th November, 2001
  Where:  2nd Floor
          Compaq Computer
          139 Frome Street
          Adelaide SA
  Cost:   FREE
  Who:    Anyone and everyone.
          No pre-registration necessary.

Presentation:

  Paul Schulz will be giving a talk on User Mode Linux:

  User Mode Linux is a build of the Linux kernel which allows it to be
  run on a Linux system as a regular process. It is an excellent tool
  for learning about the low level internals of the Linux operating
  system and UNIX in general, without rendering your system useless
  should you make a mistake. 

  This talk intends to be an introduction to User Mode Linux.  There
  will be some discussion on its uses and possible usefulness and some
  demonstrations (networking and file systems).

Extras:
