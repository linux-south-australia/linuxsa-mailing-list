> Well, nothing formal [man xrdb] shows how to incorporate its changes.
> 
> For X toolkit apps, you can use editres [/usr/X11R6/bin/editres] to 
> find out what can be set, and even enter it into [.Xdefaults] for you.
> 
> Try loading editres and xedit, then in the menus [Command][Get Tree] 
> and click on xedit's window. This gives a tree of resourses, click on 
> the one you want to change [try the quit button] as a test then go to 
> the menu item [Command][Show Resource Box], select something like 
> [background] in the list then in the text box at the bottom type a 
> colour name. Hit [Apply] and the item should change its background 
> colour. Up the top of the window is shown the resource string that you 
> would place in .Xdefaults. You can choose to save it if you want.
> 
> XEmacs can show you a few sample things.
> 
> Each option is application specific, so I don't think there is any real 
> list of things you can do with it. Even different versions of programs 
> handle options differently. Netscape used to be able to be changed 
> using X resources, although this does not seem to work anymore.
> 
> Anyway, I hope this helps. :-)
