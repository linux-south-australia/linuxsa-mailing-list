> i need to setup a fortran compiler here on my computer at uni. I am
> running RH6.2. The closest thing (i think) i have come to actually
> getting my hands on a compiler is a rpm on the RH6.2 disk called:
> egcs-g77-1.1.2-30.i386.rpm
> i tried installing in (and the other rpms it needed) via:
> rpm -i egcs-g77-1.1.2-30.i386.rpm
> 
> when i type:
> rpm -q egcs-g77-1.1.2-30.i386.rpm
> it says it isnt installed.

Did you try:
* rpm -q egcs-g77
* rpm -qa | grep egcs-g77
* rpm -qf /usr/bin/f77 (or similar)

I think you'll find that you have to drop the version number and/or
the .rpm extension when performing a query.

Another handy RPM command is:
* rpm -ql egcs-g77
which will show all the files installed as part of that package.

Hope this helps.
