Peter Clifford wrote:
> 
> i need to setup a fortran compiler here on my computer at uni.

I notice you've found g77, which is OK.

If you are going to use FORTRAN for serious number crunching
you'll need to look elsewhere; see the products from Lahey, Absoft,
Portland and the new compiler from Intel.  All are commercial,
available for free evaluation and beat the GNU compiler hands
down for speed and language support.  The Intel compiler is
free for non-commercial use, and unis are non-commerical :-)

See
  http://www.nikhef.nl/~templon/fortran.html

The retailer with the best support is said to be
  http://www.polyhedron.co.uk/
Have a look at the benchmarks there.  You'll see that
there is no one "best" compiler.  So if you have
a number-crunching application in mind then benchmark
the lot using evaluation licenses.

First stop for any science application on Linux is always
  http://sal.kachinatech.com/
