Virtual Linux servers

I don't know if this has been already posted, because I have only just
rejoined the list and the web archives are out of date. Anyway...

A recent post to the Linux kernel mailing list introduced a patch creating
virtual servers for Linux. Not partitioning, not a virtual machine, see
http://www.solucorp.qc.ca/miscprj/s_context.hc for more details. I'm
downloading hoping to dive in for a look now but thought I'd mention it
since I know there are quite a few here interested in virtualisation.

I have tried out User Mode Linux and various emulators (including
Linux/390 under Hercules on Linux :-) and VMware GSX server, but this
looks architecturally like it could be better than any of them, depending
on what you are trying to do. Of course, there are some things unique to
each approach so opinions will vary.
