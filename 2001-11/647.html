Got some help from the kylix list, tho it's not overactive. Seems the 
LD_LIBRARY_PATH has to be set right. ie something like

LD_LIBRARY_PATH=/usr/local/kylix/bin:/usr/lib

as a responce from env... where the appropriate kylix friendly libs are in 
/usr/local/kylix/bin, and these are the libs you can freely share, so I 
imagine it's a matter or sending these 'free' libs out with the app and doing 
the path thing and then the app should run on a PC without kylix on it. 

I haven't tried any of that yet, but at least I can now get the compiled apps 
to run on my own box...
