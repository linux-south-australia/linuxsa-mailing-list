RE: virus software

We use MIMEDefang to quarantine all undesirable file attachment types (
.mp3, .avi, .mpg, .dll, .exe, .shs ) and the rest.
http://www.roaringpenguin.com/mimedefang/


MIMEDefang also calls upon McAffee ( nai ) to check any attachments that
aren't quarantined ( ie .doc, .xls etc etc ).

MIMEDefang quarantines on average about 5-10 virus infected .exe files a day
( Mainly Snowwhite and the Seven Dwarves ). It also discourages people from
choking the network with "funny" attachments. MIMEDefang sends a reply to
the user saying the file was quarantined.

Hope this helps
