Re: GUI HTML program

Hi Joshua,
    Have you looked at Screem, Site Creation Emulation tool. Not exactly gui
last time I downloaded but it did allow for drag and drop of components and
easier swapping between views. It is still designed more round the editor
than the Page view but allows for easy tabbing between views. Sort of an
advanced Homesite(Commonly used with Dreamweaver) Otherwise Bluefish or
Quanta, again both are designed from the editor style but allow easy
swapping between views.  Very similar to HomeSite or Arachnia

ps I'm still using fairly old versions (18mths) and believe their are a lot
more additions to Screem since I last got it.
 nb. none of the above alter code on existing pages unless you intend them
to.

Hope it helps
David Delyn
