Re: Poor internet connection on Mandrake 8.1

Hi there.

I had the same problem when connecting to iPrimus.  The connection would be 
fine for about 20 mins then slow down to a stop.

I found that disabling BSD compression in pppd did the trick.

in the pppd man page, search for bsdcomp to see how to disable it and see if 
that helps.

Cheers
Damien.
