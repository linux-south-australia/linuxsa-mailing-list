Re: cdrom burner

Hi

Add the below line to you lilo.conf file, making sure the ? is the correct
IDE port
then run /sbin/lilo. (note this work for lilo of your using grub or some
thing like it read the manual)

append="hd?=ide-scsi"


Now after boot do a 'modprobe ide-scsi sg' now run cdrecord -scanbus record
the dev numbers.
