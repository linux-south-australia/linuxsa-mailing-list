Denise,

Which web site did you try to find a FAQ at? RedHat.com? Their site is
generally very corporate and appear to use liberal amounts of DHTML so it
may not appear correctly on older browsers. Try www.mandrake.com and
www.suse.com.

On a 486DX (33Mhz?) machine with 32Mb of RAM, the latest distros for full
blown GUI desktop use would probably feel like time has stopped. I think
around the Red Hat 5 era would be fine though.

Generally the most recent really viable desktop Linux distributions like
Mandrake, Suse, and Red Hat would go well on machines with 128 MB RAM and a
300Mhz Pentium, if desktop use is what you're after.

For moderate emailing and a great web browsing experience, Mozilla and
Mozilla's own email client would do the job rather well.

Windows 98 with IE 4 and Outlook Express would suit them well too,
especially considering its good accessibility support. I've seen a nearly
blind guy at the state library navigate Windows (rather slowly) with a mouse
cursor about 1/4 the size of my notebook's 1024x768 desktop. Accessibility
is one of Windows' strong points, I would have to admit.

Cheers,

Joseph Tan
