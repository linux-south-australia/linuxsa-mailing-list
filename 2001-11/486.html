Untitled DocumentWhy do you need 2 Network cards ? Are you running servers
on a DMZ ?
For a modem setup ( sharing Internet ) you would only need a single ethernet
card.
If the 2 NEtwork cards are the same you will need to manualy specifiy the 2
IRQs and IOs.

On the Smoothwall PC run the command
ifconfig

this will show the IP addresses and the IO and IRQ that it is trying to use.
Check to see if it shows it being up. Should look like this

eth0      Link encap:Ethernet  HWaddr 00:A1:49:1A:41:D5
          inet addr:162.145.11.82  Bcast:162.145.11.255  Mask:255.255.255.0
          UP BROADCAST RUNNING MULTICAST  MTU:1500  Metric:1
          RX packets:12927526 errors:0 dropped:0 overruns:0 frame:65577
          TX packets:9700687 errors:351 dropped:0 overruns:0 carrier:351
          collisions:843767 txqueuelen:100
          Interrupt:11 Base address:0x9000

the important thing to look for is the interface is UP and the Interupt/Base
address


Before you worry about the modem you need to establish Network connectivity
between the Smoothwall PC and the non-smoothwall PC.

Check postings on the Smoothwall mailing list for problems with Internal
Modems. I personally wouldn't bother trying to use an internal modem. They
are more trouble than they are worth.
