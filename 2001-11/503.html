That looks like it's trying to use Frontpage extensions. You can get
them for apache, if you're willing to compromise with an out-of-date
version. There are some other frontpage emulation systems out there as
well.

Another other option is samba, windows can access a file share across
the Internet if you enter a URL of the form into the 'start' menu:

\\192.123.45.6\sharedir

If you setup a WINS server and configure it for the client, it'll show
up in 'network neighbourhood' for them.

You would want to control that with appropriate username, password, and
ACL's if possible.

Lastly you could look at webdav. IE5 installs a thing called 'Web
Folders' into the explorer system, which allows Webdav to be used like
local windows directories for editing stuff.
