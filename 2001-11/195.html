Re: Virtual Linux servers

>A recent post to the Linux kernel mailing list introduced a patch creating
>virtual servers for Linux. Not partitioning, not a virtual machine, see
>http://www.solucorp.qc.ca/miscprj/s_context.hc for more details. I'm
>downloading hoping to dive in for a look now but thought I'd mention it
>since I know there are quite a few here interested in virtualisation.

I don't know if you have any specific application in mind, but if you're
interested in unix-style virtual servers it might be worth checking out the
"jail" program/function in FreeBSD.

This effectively creates a virtual FreeBSD server on a FreeBSD server,
without the overhead of emulation or a virtual machine. According to the man
page, it first appeared in version 4.0
