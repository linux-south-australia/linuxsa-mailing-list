RE: SmoothWall External Modem Install Problems

You shouldn't have to do ANYTHING else.
Can you still ping the IP of the Smoothwall box once the modem has
connected.

The only thing that happens after it dials is to set up a PPP connection.
The LAN side of things should not be affected in any way.

What is the output if you type
ifconfig
on the Smoothwall box once the modem is connected.

I'm not sure why you are having so much trouble. THe default installation is
a breeze as long as you get communication happening from the SMoothwall NIC
to your other PCs NIC. If youve been manually trying to reconfigure various
stuff I would seriously think about starting the Smoothwall install again.
The whole process only takes 15 minutes from start to finish.
