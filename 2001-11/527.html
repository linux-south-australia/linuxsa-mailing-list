IPTables & FTP

Greetings, 

I am having troubles setting up FTP with masquerading using IPTables.
Webbrowsing, mail and other programs work fine.

Here are the ftp kernel modules loaded:

>lsmod | grep ftp
ip_nat_ftp              3648   0  (unused)
iptable_nat            18196   1  (autoclean) [ip_nat_ftp ipt_MASQUERADE]
ip_conntrack           16940   2  (autoclean) [ip_nat_ftp ipt_MASQUERADE 
ipt_state iptable_nat]


Here are my iptable rules: EXTIF is eth1, INTIF is eth0

$IPTABLES -A FORWARD -i $EXTIF -o $INTIF -m state \
                    --state ESTABLISHED,RELATED -j ACCEPT
$IPTABLES -A FORWARD -i $INTIF -o $EXTIF -j ACCEPT
$IPTABLES -A FORWARD -j LOG
$IPTABLES -t nat -A POSTROUTING -o $EXTIF -j MASQUERADE


FTP clients connect OK, but are unable to perform a directory listing.

I am running Redhat7.1, Kernel 2.4.12, iptables v1.2.3
