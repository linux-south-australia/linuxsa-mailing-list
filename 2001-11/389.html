Re: Internet Connection Sharing

Hi there,
    Go to the RedHat site and download the How to set up an
ISP using RedHat document (the exact name escapes me). Read
the Networking HOWTO, the Security HOWTO and other HOWTOs
suggested within them. This will get your network set up and
your security setup to give you base protection. Then read
any security documents to plug any holes left over. The
reading will take you a few days, but is well worth it
BEFORE you start. The actual setup will take about an hour
or two. You will need networking cards for each machine
(read the Networking HOWTO to get suggestions for supported
cards) and I would suggest lashing out and getting a 100Mb
hub (usually no problem with any of these). This is a
relatively straight forward operation to set up, so go to
it.
    Cheers!
        Andrew.
