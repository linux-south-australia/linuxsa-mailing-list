>Having only used OS X for about 2 days now, I'm pretty impressed, so 
>much so that if/when I buy another desktop it will probably be a 
>Mac. Why? Well I have a personal need for Unix (my desktop PC runs 
>Linux probably 85% of the time (Win2k for gaming)), but the Aqua 
>interface really surpasses Gnome or KDE in just about every aspect. 
>It is, my ideal OS (at the moment anyway). All I need now is a Shiva

I've used it since I first got my hands on a public beta some time 
back, until now.

You'll be even more impressed, the longer you use it.

I run it on all three of my own computers (two fixed, one laptop). 
Absolutely rocks. Real Unix, a nice GUI, and it runs all the 
'mainstream' apps I like as well (because it's also a Mac, and all 
the decent 'maintstream' commercial apps have long since appeared on 
both PC and Mac platforms).

The best of all possible worlds in my view. I am -such-  a happy 
camper using OS X.

[and because it's on an open source core, more open source stuff is 
turning up for it - I just got told that Xfree86 has been ported to 
it, for instance, so it can run X windows stuff as well]

Its so nice having a Mac where I can pop up a terminal window and 
write code in perl, where the machine is so amazingly -stable-, and 
where it only looks like unix when you want it to - otherwise it just 
looks like the ultimate MacOS.
