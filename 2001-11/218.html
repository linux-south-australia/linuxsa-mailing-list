Re: I need help using linux

On Thu, Nov 08, 2001 at 04:21:58PM +1030, Denise Tzumli wrote:
> So my questions are:
> Everything I have read says that I need a pentium myself to run Linux,
> and the latest thing I've read says I need at least 1.3 GIG of my
> hardrive to load it?
> Is this true?

depends on which linux distribution you're talking about,
eg. RedHat, Debian, Mandrake, Suse, Turbolinux etc etc
The minimum is a 386, just depends how fast you want to
run it.  Same with disk space, depends how much you want to
install.

> what can I do on a 486? with 32meg of RAM?

more or less everything!
 
> Does the Linux display allow for mega large high contract display for
> the visually handicapped?

as far as I'm aware, you can set up Xwindows to whatever
size etc you want.

For your requirements, you might not even need to install an 
operating system.  Eg. download the QNX demo disk onto a floppy
disk and you can surf the web and more without even a disk
in your machine (http://www.qnx.com/demodisk/index.html).

Otherwise, I'd suggest an older, stable, simple linux distro
without the "heavy" window manager environments, or one of
those simple small distros (eg. see:
http://www.linuxlinks.com/Distributions/Mini_Distributions/)
