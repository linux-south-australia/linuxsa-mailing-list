Re: Very, very newbie-please be gentle

Try editing /etc/inittab so your default runlevel is 3, not 5 -- this
will enable you to get in without running KDE. (you may need to do
this in single user mode)...

look in /var/log/* for any messages that indicate a problem...

See if it's stable without X at all...

then run startX...

if/when it locks, try ctrl-alt-f1 to get to a virtual console...

try ctrl-alt-del to kill X...

HTH
