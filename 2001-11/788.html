RE: ssh forwarding

There are instances where ssh will use a source port lower than 1024.
It is the only TCP-based service I've seen doing this. This is a
carryover from the days when it was assumed that only a root user of a
multi-user system could open a source port < 1024 so it was deemed
more trustworthy. Nowadays, this is a naive assumption.
