> I am trying to figure out a way to add X as an after thought after a 
> RH7.1 system was installed without X.
> 
> Does anyone know a simple way?

I suspect the easiest way is to perform another installation and select
upgrade, then be sure to select some X-related packages (for instance
X Windowing System, KDE, or GNOME).

Otherwise, you can try installing the necessary components using rpm,
but there's a whole lot of dependencies to worry about: you'd probably
need or want the following:
* X server (XFree86)
* Red Hat X tools (Xconfigurator)
* libraries (Qt, Xlib, GTK, and so on)
* window manager
* desktop environment - optional
* GUI applications - optional
