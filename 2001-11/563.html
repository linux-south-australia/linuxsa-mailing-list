regular expression help

Hi All, I'm trying to put together a set of regular expression that will
identify the longest string matching the definition of a protein coding
sequences:

The base RE is:

/(?:TAA|TAG|TGA)(?:\S\S\S)*(ATG(?:\S\S\S)*)(?:TAA|TAG|TGA)/

with $1 being the answer. My first question is how do I ask the RE engine to
return the longest $1 match (I remember seeing this in a manual somewhere, but I
can't find it again.

The gene sequences that I am looking at may be truncated at either end, so I
need a set of REs that can make guesses about what is coding, this brings me to
Q2.

The second question is how do I specify a string of triplets /(?:\S\S\S)*/ that
does not contain /(?:TAA|TAG|TGA)/? If it were a single character, to avoid it
would be easy, but I can't see a way to get this with frame-wise triplets of
characters?

any help would be greatly appreciated
