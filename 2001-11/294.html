Toby,

So far I have had no problem with my XP/RH71. dual boot system. I have not
used GRUB, but you should have no problem. The XP bootloader looks as though
it is the normal trashy NT/2000 bootloader, so there should be no problem. 

I can still map my SAMBA shares and get my DHCP stuff from my Linux Server.
I haven't tried to mount XP partitions from Linux, so I can't help you there
and my PC's are at home so I can't test it now.

Perhaps the only thing that you have to think about is whether to put NTFS
or FAT32 as your XP file system.
