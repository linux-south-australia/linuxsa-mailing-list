Re: [OT] Concaternate command for DOS

Hi Michael,
    copy file1 + file2 + file3 targetfile

to tell copy that all the files are just ASCII text, make it:
     copy /a file1 + file2 + file3 targetfile

or if any of the files are binary you need to tell it as such:
    copy /b file1 + file2 + file3 targetfile

the /v option tells it to verify the copy went correctly so what you probably
need is:
    copy /a /v file1 + file2 + file3 targetfile

Good Luck.
