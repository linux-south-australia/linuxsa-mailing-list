Re: Any Questions to Red Hat, Mandrake You'd Like To Ask?

note all, that this was a question about RH's and Mandrake's _plans_
for the future... Yeah, apt and rpm are different in functionality,
and there are various projects to make an apt-like system for rpms,
but that's not what I was asking about. Note also that to have an
effective system like apt, you need a strict policy on package
specification -- they all need to be thoroughly tested, and have
dependencies clearly specified, for a start... and this is something
that is more to do with the maintainers of the package repository than
the actual package management software itself...

Anyway....

I'm sure someone else may have a useful question or two as well...
