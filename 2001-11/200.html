Worked fine for me when I tryed it a while ago, Win2k + Nero.  It's
simple, File, Burn Image, select all files, select the .iso file and hit
enter a few times.

> I presume when burning the image file is expanded into normal files onto the
> CDROM which is them readable as normal -no?

Sort of.  The ISO image is a 1:1 copy of a CD, when written to a CD, the
CDROM reads it as many normal files - but the image itself is not
compressed.

> Anyway - could someone give me a few hints?

You may have a dud download.  Have you checked the MD5 checksums?

sanctuary> md5 smoothwall-0.9.9.iso
MD5 (smoothwall-0.9.9.iso) = eb74019e8eab939637f0b4adba60ff73

The above checksum should match the one on SmoothWall's web site (the
bottom of http://www.smoothwall.org/gpl/get/download/ ).
