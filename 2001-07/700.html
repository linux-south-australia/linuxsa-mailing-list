James,

It looks as if you are running the Zeus engine.  If so the port 9090 is the
Admin port for the Zeus server.

Port 5050 is usually multimedia conference control tool.

Port 554 is usually Real Time Stream Control Protocol

so if any of these things are not on your server (you did not install it)
then you may have a problem.
