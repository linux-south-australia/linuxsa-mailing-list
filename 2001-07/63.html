>>>>> "CG" == Conrad Gittins <conrad@iweb.net.au> writes:

  CG> DD =  David Drury wrote:
  CG> Hi, I'm trying to find the correct settings to put into the
  CG> /etc/printcap file for a certain type of printer.  How can I
  CG> find out about compatibility or find out where and what
  CG> information is required so that I can write a filter for it?
  CG> It's a Brother 8600.

  DD> If you have RedHat, use RedHat's printool. It will do everything
  DD> you need !

  CG> It's Debian, but would the redhat printool files contain
  CG> information that I could use?  Thanks, Conrad.

You find out about the "compatibility" either by scouring your
documentation (the Hardware-HOWTO is a good place to start), or simply
search for it on the net.  For example: 
  http://www.google.com/search?q=linux+printer+brother+8600+driver

To save you the trouble, visit:
  http://www.linuxprinting.org/printer_list.cgi?make=Brother
(This site also has some useful hints about what printing systems are
supported by the different distributions.  Check out the Debian
section.)

The web site at http://www.brother.com/us-fax/INFO/mfc8600/8600.htm
says it handles PCL4, so if all else fails, treat it the way you would
a HP LaserJet.
