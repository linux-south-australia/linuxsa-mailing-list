RE: ezbios

This only works if the disk is a Western Digital HD - the utility checks and
if its not you can only access certain functionality.

I had the same problem recently and I used Partition Magic's PTEDIT and
zapped the disk to tell it was something else (FAT 16B). Linux liked this,
reformatted accordingly and installed.

You an still buy the original EZ Drive software but it equates to about $50
AUS. I figured my option was a little cheaper.

Hi Robyn,

Goto www.westerndigital.com, goto the support section and download the
western digital datalifeguard tools. Run the dlgmaker.exe file from a
windows box, and it will create a self bootable floppy. Boot off the
floppy and there is a menu option on the disk to remove ezbios.
