> If I'm putting in a Firewall for a network connected to two LANs and

> DMZ (such as ours), would it be advisable to run DNS on the border
> firewall machine?

Do external machines need to query this name server?

If yes, then you have to put it in the DMZ.

If no, then there is nothing to be gained by putting it on the firewall. Put it behind 
the firewall or keep it up to date.
