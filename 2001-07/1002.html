> disk . Is there any way I can get around this small problem.

This is a common problem actually, and I should write an FAQ on it.

Take the following steps:

1) Insert the redhat boot disk
2) When the boot loader comes up (you see the lilo: prompt)
3) Type lilo rescue

After answering a few basic questions you'll get a command prompt.

4) Type lilo -r /mnt/sysimage/etc

Lilo should install.

Now, if the installer(the graphical one) doesn't say mounted system
under /mnt/sysimage talk to us again.

> I also have a lappy with debian on it, so could i create a boot disk on this
> machine and use it on the other.?

Probably, but try the above first.
