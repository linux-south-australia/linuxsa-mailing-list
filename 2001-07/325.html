Re: MySQL root passwd unknown

Michael Davies writes:
> I have a MySQL installation where the root password for MySQL is unknown
> (administrator left without passing on the info).  I can't re-install
> MySQL as the database schemas are in the database and cannot be
> retrieved without the root password for MySQL.  That will be my first
> task once I get in.  I do have the root password for the (Linux) box.
> MySQL version is almost the lastest, somewhere in 3.23.X.

The best way (the one on the book) is as follows 

You must bring the server down temporarily (the mysql daemon), restart the 
daemon using the '--skip-grant-tables' option which allows you to connect as 
root without a password, then  at the command line type
mysqladmin password "newpassword"
then restart the daemon as you normally would.
