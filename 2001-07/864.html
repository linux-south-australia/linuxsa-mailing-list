If you modify the ks file with a text prog by adding a line

mouse --device (where the mouse is) eg stty0  and then the name eg generic3
(for 3button mouse)
So, in your file (i guess) replace the 'generic3ps??????' after mouse with
generic if it is a wee 'generic' beasty... :-)

Actually, download the 'Official RedHat Linux Customization Guide' from the
RedHat site, it has a good section on kickstart and will most likely
overcome any problems.
