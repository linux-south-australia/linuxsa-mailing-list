Re: Blocking the SirCam & Code Red viruses

Keep a list of offending IPs and drop the connection as soon
as you hear from them.  When this is all over, you can delete
them from the list.

I use the Indy components in Delphi/Kylix to write web servers,
and you can do the same thing.  Using Apache limits you to the
standard line of connection methods, which means you have
more risk of 'DoS' attacks, as you put it.  Using your own
server methods lets you respond by blocking IPs again the
next time a new virus/worm screws up your system.

By the way, I'm not responding directly to you because
I don't want to be put into your address book!

-Rich
