How much traffic does the email account(s) that you run this software on
have? How do you define "works"? Is it "It gets the message through most

> I am not proposing blocking filters or censorware or things like junk
> buster.

No, but the concept is the same. Censorware works on the concept of
finding certain expressions within URLs, HTML pages, etc. and performs a
user-defined action accordingly.

For example, if Joe's CensorWare Software finds the string 'Hot XXX
Action' in the HTML page that it has scanned, it may be set to deny
access to that page (and replace it with an "Access Denied" screen).

Or there is the option of the CensorWare package to log the time and
date of the attempt and let it all go on. It's the concept of filtering
that remains the same over both implementations.
