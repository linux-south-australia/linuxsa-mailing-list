Re: Why do we hate HTML mail?

David Newall wrote:

> Why do we hate HTML?


Here's my guesses, they may not hold true in all cases:

1) some mail clients can not read HTML mail (not sure which ones)

2) not all mail clients produce standards compliant HTML,
    so mail may appear differently than intended
3) when replying to HTML mail (in plain text mode), one
    must remove all the HTML markup
4) HTML encourages persons to use ridiculous background images
    and font colors, sometimes making the mail less legible, and more
    system-dependent
5) HTML messages are somewhat larger for an equivalent message
6) some HTML messages can contain scripts, code, or images designed
    to do malicious things (not sure whether this is unique to HTML mail)
7) HTML mail may be harder to parse using scripts and filters
