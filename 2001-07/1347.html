There has been some stuff about long range wireless networking on the web
lately.

> > What would you's recommend for such a situation.

The best article:
http://www.pbs.org/cringely/pulpit/pulpit20010628.html
In particular "The connection would be using 802.11b, which normally has a
range of 100 feet or so. But that range is using an omnidirectional antenna.
Using a 21 dB Yagi directional antenna, other people had built links as long
as 14 kilometers" A follow-up :
http://www.pbs.org/cringely/pulpit/pulpit20010712.html

Some others:

http://www.adelaide.air.net.au/
Local Linux / Wireless Networkers. They may be able to help you with your
wireless set-up / questions.

http://www.oreillynet.com/pub/a/wireless/2001/03/06/recipe.html
Home wireless networking recipe

http://www.oreillynet.com/pub/a/wireless/2001/05/03/longshot.html
Extra long range wirless ( over 20 miles)

http://www.hpl.hp.com/personal/Jean_Tourrilhes/Linux/Wireless.html
Wireless LAN resources for Linux

http://www.hpl.hp.com/personal/Jean_Tourrilhes/Linux/Tools.html
Linux wireless tools

http://www.google.com/search?hl=en&safe=off&q=linux+wireless+lan
Heaps of other links courtesy of Google.com
