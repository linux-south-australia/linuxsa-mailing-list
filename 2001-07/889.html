For the IT Scholarships - get the MCP qualification for $500 - contact

for the MCP accreditation.

For the Youth and IT project, it is still getting underway but expect 
the ad for the Project Coordinator in this (or next) Saturday's 'Tizer

board. This project will add Trainer (level 1) qualifications to young 
people 25 and under who already have some good IT skills. Again the 
applicants should live in or close to the Salisbury council area. Keep 
in touch with me for more details.
