RE: Dump

wlsimes@tell.net.au wrote:

> The impression that I got from some other replies was that it is also
> possible to duplicate the hard drives using the dump command while
> in multiuser mode.

You can, but you will still get file inconsistencies if the file is
being modified whilst being read for backup.  Being in single
user (and read-only) is the best.

> Whilst I have done this with single partitions, I couldn't figure
> this out for doing hard drive to hard drive as a whole.

You can't - you have to copy the partitioning info across onto
/dev/hdc (or presume it already exists and is accurate), then
copy each partition across.  It's not necessarily a manual
process, though, because that's why shell scripts were invented.
