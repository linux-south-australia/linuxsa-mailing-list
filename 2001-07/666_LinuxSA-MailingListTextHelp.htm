Setting your email application to send plain text emails.

Outlook Express

Select Tools, then Options... or press Alt, t, o
A box will pop-up with many tabs at the top. Select the Send tab.
Near the bottom is a heading Mail Sending Format.
Simply select the radio button next to the word plain text, press Apply then OK.

Netscape Mail

Select Edit, then Preferences... or press Alt, e, e
A box will pop-up with Mail and Newsgroup Options.
Select the Send Format from the list in the left pane.
Select the radio box next to the words Convert the message to plain text

Other Applications

Usually you will find help in the applications documentation.
This is normally found under Help on the menu bar or by pressing the F1 key.
If this doesn't help, please send an email to texthelp@linuxsa.org.au with the name of the application you use.
We will do our best to find out how to set your program up and post the information to this page for other new list members.
