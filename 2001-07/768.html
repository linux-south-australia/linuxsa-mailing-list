Hello Bryan,
This has turned out for me to be surprisingly simple. I have setup 2 
adsl connections in the last month and both have been very simple. Both 
these were on redhat 6.2.
The first was a telstra connection. I looked up pppoe on a search 
engine on the web, downloaded the roaring penguins rpm followed the 
instructions and it connected. It set ipchains to close everything 
except web and mail so that needed to be changed and we changed the 
script to email us everytime the link goes down so that we get the new 
ip address and set it to start on bootup.

The one to internode took longer because we expected there to be more 
to it. It is just routing another ethernet connection. You therefore 
need a 1 in the /proc/sys/net/ipv4/ip_forward file and setup the 
routes. Our main problem is getting the DNS entries changed over.
