There are benefits to current Linux users if there is widespread use
of linux in the community. For example it would be good to be able to
easily access all government and commercial web sites. There are an
annoying number of web sites where they attempt to reject you if
you're not running windows, or even worse require you run a windows
program to communicate with them properly.

I'd also like to see games being written and released for Linux rather
than ported to it six to twelve months later (if ever). As for
businesses using Linux there are other advantages - I can get employed
to work on Linux all day rather than it be something I do when I get
home at night :-)
