That's a valid point, but not if you're talking about the standard,
run-of-the-mill stuff that most people will find themselves using.

The one problem that you may have is the issue of cross-firing beams.
For example, if I set a link up between point A and point B, and someone
else sets one between point C and point D with a beam going smack-bang
through the other, you will probably experience problems with signal
loss and (possibly) noise.
