Unreal Tournament and Linux - Was: Linux and games

I was wondering if anyone on this list has had any experience running
Unreal Tournament under Linux (Mandrake 8.0). More specifically it is
using a TNT card with the 1.x drivers from NVidia.

While it works and I achieve reasonable 3D results using the GL drivers, I
find that performance is not as good as the Windows version on the same
machine - occasional puases. This is running in single user mode (with
xfs started) using the 'failsafe' X window manager (xterm), in order to
remove all other influences on the game (such as KDE Arts etc...).

Are there some tweaks that I may have missed? What results have other
Linux UT players had?
