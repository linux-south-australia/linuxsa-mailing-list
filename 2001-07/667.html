Re: how to uninstall any program

Prakash!

> i have installes ucd-snmp from the tar.gz module now i want to upgrade
> it.

I see.

You could try starting the configure EXACTLY as you did before, then
doing "make uninstall".

[root]# ./configure INSERT_FLAGS_YOU_USED_HERE_IF_ANY
...
[root]# make uninstall

That would get rid of what it installed at the start, but it may not get
rid of old configuration files or it may not even work. Many makefiles
don't include the uninstall option.

> i need to know how to uninstall this installed module.

You could also do similar to the above, but instead of make uninstall:

[root]# make install >& make.log

Then read the "make.log" and work out what it's installed (it should say
so).

> i know rpm -u command,but it works only if the module is installed
> from the rpm package.

That's true.

> it is risky to directly upgrade the existing version since i am not
> sure the new version will put its files

What is the new version?

If it's an RPM package, you can do:

[root]# rpm -qlp [the rpm file name]

You may want to pipe that to less or more.

That tells you what files the RPM installs.

If the thing has a home page, you could try visiting the home
page...they may have a discussion there about upgrading.
