>
> As I see it, good mail etiquette includes:
> * including a meaningful subject
> * sending mail as plain text
> * quoting the relevant portions of the previous message, then adding
>   your reply beneath it
> * begining your signature with "--" then a new line
>
> This is what I would do to configure Mozilla to default to sending
> well written (as perceived by most knowledgeable Unix users) mail:
>
> Edit->Preferences->Mail and Newsgroups
> MESSAGE DISPLAY
> * Use this font when displaying plain text messages: Fixed width font
>
> MESSAGE COMPOSITION
> * Forward messages: As Attachment
> * Automatically quote the original message when replying.
>   Then,: start my reply below the quoted text
>
> SEND FORMAT
> * When sending messages in HTML format...:
>   Ask me what to do OR Convert the message to plain text
>   (according to tase)
> * Add "*" to Plain Text Domains
>   (not sure if this has the desired effect)
>
> Someone may wish to compile similar instructions for other MUAs,
> altho there's probably already something like this somewhere.
