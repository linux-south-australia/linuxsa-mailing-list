Re: how to include library in @INC

Hi There!

Look up the Perl documentation. You want to find one that talks about
the pragma:

* use lib [insert a path here]

Very quickly, I often do things like:

# /usr/bin/perl

use lib "./modules";

use mymodule
....

This adds "./modules" to @INC. Using use appears to be the approved way
of doing things but there are other ways...
