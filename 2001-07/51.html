Re: graphics tools

GIMP
from http://gimp.org

lay down your line markers over the image, then select the plugin, and it
outputs an html file and a bunch of files into a directory of your choosing.

It doesn't have much support for automatic rollovers, etc. though. But it
does do the image editting part extremely well. I prefer GIMP over
photoshop easily.

You mentioned you already use the gimp, but perhaps you just haven't
discovered this function? It's under Filters/Perl-o-tine in my copy, but i'm
running a rather ancient cvs version, so its possible it has been relocated
or renamed.
