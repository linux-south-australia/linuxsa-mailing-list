Re: Basic NFS Question

David Lloyd wrote:

> Is it possible to symbolic link to an NFS mounted directory?

Unix is very consistent. An NFS mounted directory is going to be mounted 
on a mount point, ie, a directory.

So, you link to the mount point, and you have it.
