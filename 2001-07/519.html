RE: using linux as a hub

>Yes I've worked out how to bridge 2 networks together, I guess what I'm
>really asking is if it's possible to have 2 nics in the same machine
>belonging to the same network (in my case, 192.168.0.0/255.255.255.0) and
>have the data flowing across the 2.

In my experience (thanks D.C.!!!) it is much easier in terms of routing 
packets if you make the machine on the other ethernet card on a different 
network, ie. so you have the ethernet cards on 2 different networks. like 
192.168.0.0/255.255.255.0 and 192.168.1.0/255.255.255.0

------

to answer your question.... you can but you possibly need to add in 
individual routes for each machine on your network so that the linux box 
knows where to send the packets...

someone correct me if I am wrong but this is what I had problems with 
earlier in the year...
