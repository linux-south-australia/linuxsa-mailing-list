Re: Modem and other hardware recommendations

On Fri, 6 Jul 2001, Brian Marr wrote:
> My current install of  Mandrake 8.0 does not have the kernel sources either. 
> There is nothing to "make menuconfig" with. Can anyone say how we get these 
> on our systems ? What packages are required ?

The simple graphical way involves starting the Mandrake Software Manager
from the Gnome/KDE configuration item on the Gnome/KDE 'start' button.

Choose the the uninstalled list. (From memory) Look under the development
section for 'kernel source', tick this then choose install.

The Manager will sort out all of the dependencies for you, ask you to put
in the required CD (I have INSTALL and EXTRAS), then install it for you.
If that works as advertised, you will be laughing.

You should then have the linux directories in /usr/src.

There are some other tools that do a similar job from the command line
(not mandrake) and I want to find out what they are.
