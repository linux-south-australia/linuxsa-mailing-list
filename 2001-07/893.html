Re: passwd

Fülöp Balázs wrote:
> 
> i am working on a script that can manage users.
> can anyone tell how i can list the users of the system?

man getpwent

Note that /etc/passwd need not contain all the users on
a system, and probably won't in any environment with
more than a handfull of users.

getpwent() will traverse all users on a system regardless
of what mechanism (file, ldap, nis, etc) is used to
register them.

Perl and other systems-administration languages have
packages that use getpwent().

If you are finding yourself writing scripts to manage
users it's probably time think about moving away from
/etc/passwd to another user registration mechanism.
