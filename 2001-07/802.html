Re: Help for Community Project

The redhat kickstart program can be tailored for a custom install. Select the
packages and make the kickstart file.  Once made the rest of your installs can
be identical.  Spend the time learning kickstart with redhat and there is no
need to even have CDROMs. All the installs can be done over a network if
desired.

There is also graphical program called ksconfig,  it might need installed
(RPM'd) that will get you going with ease. It's a matter of a few hours
reading and then saving tens/hundreds of hours, on future installs.

I think Suse has a good install program too,

Good luck


Steve
