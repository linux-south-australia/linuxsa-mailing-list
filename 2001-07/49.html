graphics tools

does anyone know of an open-source program that does an equivalent to what
adobe imageready does?

(for those who haven't seen it: imageready is a component of photoshop that
basically lets you take an image and cut it up into "slices", then output
those slices to html/images (it makes all the table html and stuff, and cuts
up the big image into slices nicely). it also does other stuff like
rollovers (html+javascript) and animated gifs, and probably lots more stuff
that i don't know about.)

the question comes from the issue of seeing if we could move to a linux-only
desktop. we've got most of the tools we need under linux - office-type tools
(word processors, spreadsheets etc), mail readers, browsers, database tools,
nice text editors, the gimp, etc - but nothing to cover this app. we could
go back to doing it manually, but it's a pain in the bum ;-)
