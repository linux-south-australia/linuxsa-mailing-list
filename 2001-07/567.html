Wednesday, July 11, 2001, 10:26:52 PM, Dale wrote:

> Which good 'well behaved' mail clients do people use?

I  personally  use  "The  Bat!"  from  RitLabs (www.ritlabs.com) as it
supports multiple accounts, massive amounts of customizations (down to
per  folder  settings  for From and Reply-To headers, templates, etc),
excellent  filtering,  intuitive  user  interface, able to handle both
munged and non-munged lists ;) and probably a few other reasons too.

It is, the single real application I bother to run Windows for, and if
I  could  find  a similar or better client in the Unix world I'd quite
happily use it.
