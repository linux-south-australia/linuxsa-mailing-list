Re: mounting cdrom/floppy

found out how to do it...
edit /etc/fstab
/dev/fd0                /mnt/floppy             vfat    noauto,user    
0 0
/dev/hdd                /mnt/cdrom              iso9660 noauto,ro,user 
0 0
adding the user option
