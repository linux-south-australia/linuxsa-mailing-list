> 
> Question is, how to I stop the "virtual" scrolling bit. I want to be
> able to see all the info on one physical screen but also at high
> resolution (using RedHat 5.2).
look at /etc/XF86Config. `man XF86Config` will explain the format and
options, but basically find the right set of server options for your
server (probably accel or svga), which should something look like this:
Section "Screen"
    Driver      "accel"
    Device      "Mach64 GD (Rage Pro)"
    Monitor     "Samsung SyncMaster 17GLi"
    Subsection "Display"
        Depth       32
        Modes       "1024x768"
        ViewPort    0 0
        Virtual     1024 768
    EndSubsection
{etc}

The Virtual line is the size of your virtual desktop - set it to the
same size as your resolution you run at, and voila! no more scrolling
around. You might have several modes listed - eg "640x480" "800x600"
"1024x768", if so you can change res in X by pressing ctrl-alt-+ (on
keypad) and ctrl-alt--.
