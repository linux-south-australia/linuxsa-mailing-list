Yes, I've seen that on my 4.2 system. You need to upgrade rpm.

There's a catch: if you look in the upgrades for RedHat 4.2, you will
see two rpm upgrade packages - you need to install them both, in
numerical order, using rpm -Uvh rpm-xxx...rpm (where xxx... is the
actual name of each package)

I may have the packages at work, but it's probably easier for you to
get them from ftp.netcraft.com.au or ftp.redhat.com

Good luck,
