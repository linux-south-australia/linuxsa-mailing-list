Given that you need to burn an image with the rock ridge format, as yet
unsupported by Microsoft, I'd say so.
grab it to your linux box, make sure the permissions (eg. execute, read)
are correct, and use mkisofs with the -r rockridge option, to create the
ISO image.
You can use either linux or windows software to burn the iso image to a
cd.
