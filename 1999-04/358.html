Re: running KDE and Enlightenment -- how?

Matthew Mills wrote:

 > Hi , I am interested in trying out enlightenment on my Redhat machine and
 > want to know if there is some way how I can run both.  What I am wanting is
 > to be able to type startkde to get into X using KDE as my window manager and
 > startenl to start enlightenment as my window manager.  Is this posible or do
 > I need to add another user and log in as a different user to use different
 > window managers such as these two?

If you already have kde installed, you can use kdm instead of xdm.
With kdm, there's a pop-up menu to ask you what kind of desktop
environment you want, which is passed as a command line argument to
the Xsession script in /usr/X11R6/lib/X11/xdm/

You can also do the same thing with xdm, but a bit less intuitively:
you can configure it so that if you use a key other than "ENTER" after
typing your password, a command line argument can be provided to 
the Xsession script (by default it includes support for using F1 as
an "Enter" key on the password prompt to start a failsafe session).
