Hi Richard
start here http://www.toms.net/rb/home.html
tomsrbt (toms recovery & boot disk) is a great tool
loaded with everything you need. They build RAM based
file systems so 8MB RAM or more is usefull. From the FAQ
"Support for 4M machines [is] borderline"

MuLinux is another floppy based linux more focused on
a usable client system on floppy. With the one floppy and
one command you can install a dosemu based linux in a 
directory on a dos windows box.
With a second floppy ou can even run X11!

There are several other similar single floppy based
linux dists. including the LRP linux router project.

You won't believe how much they can get on a floppy
