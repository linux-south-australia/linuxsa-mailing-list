I don't know about your first problem, but the second one sounds
suspiciously like X is no longer SUID root. I'm not sure where the suid
root is supposed to be though, but maybe someone else will.
I don't see how this could really have occured though. But it's just a
thought that X DOES need to be suid root to access the video card
directly, and so if a user can't run it, it looks like it doesnt have
it's root priveleges.
