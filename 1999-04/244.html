Seti@Home

Not sure if anyone here is interested, but the SETI@home project is
finally operational. This project allows for worldwide distributed
processing of data from the Arecibo observatory over the web, through
pcs that are online, allowing a greater harness of processing power for
the sadly underfunded SETI (Search for ExtraTerrestrial Intelligence).

It has Linux (and Un*x) files to d/l, to allow a b/g process.
Win users are catered to as well.*shrug*

I have not noticed any noticeable drop in system performance (as you
would naturally expect from linux:)))))  )
if interested, go to:
http://setiathome.ssl.berkeley.edu/
