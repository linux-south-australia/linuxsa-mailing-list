Re: Windows NT and Samba

NT uses encrypted passwords, as does Win98 by default. I was having this
problem as well. In the Samba documentation there is a couple of registry
patches, for NT and Win98 that set them to use cleartext passwords. Apply
those patches to the Windows machines and reboot them, and they should see
them on the network.

Also, just make sure the user that you logged into the Windows machine
actually exists on the Linux machine too ;)
