Re: smbmount

Two things :

1) your "/" should be "\" and will need to be escaped at the shell
prompt ie:

smbclient \\\\<NB Name\\<share>

Remember that Netbios names are 15 chars in length

2) Try :

nmblookup -U <Ip address> -S \*

which sends a wildcard Netbios name lookup to the machine with the
specified IP address. If you are using an older version of Samba, use
the "-B" param, rather than the "-U", with the same format.

HTH,
mark.
