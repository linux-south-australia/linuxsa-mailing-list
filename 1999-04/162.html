Re: Text Editors

Jordan Schutz wrote:
> 
> I am rather new to Linux and I am wondering how I edit /etc/inittab to start
> Xwindows automatically. I know what line to change but with what do I change
> it?

There are more text editors on Linux than you can poke a stick at. 
If you've never used an editor in Unix before I think pico might be the
gentlest introduction. From the promp type 'pico'. It's self
explainatory. Next you can :
try vim (or just vi). After loading vim, press 'i' to start entering
text, esc to stop, then if you want to save type ':w'. To quit press
':q', you can combine commands too, like to write and quit type ':wq'.
oh yeah, to quit without saving type :q! 
Or you could try emacs... just type emacs and follow the tutorial
there...
