Re: prompt

Alex Wilkinson wrote:
> 
> Howdy
> 
> can anyone tell me how to change the colour of my prompt?
> I'm using bash.

Off the top of my head, put something like this
into ~/.bashrc

# PS1 is not always set, and we shouldn't set it
# as lots of utilities use PS1 to see if the
# stdin is interactive.
if [ ${#PS1} -ne 0 ]
then
  if [ ${TERM} = xterm ]
  then
    # xterm understands ANSI terminal controls
    # as do VT100s, etc.
    export PS1='\033[1;36m\w \$\033[0m '
    # and so on for PS2..4 if you want
  fi
fi


The ANSI colour control sequence is:

  ESC [ <background> ; <foreground> m

and the colour values can be found on the man page
for ls(1) ["color ls"].
