Hey There,

I had fun getting my sound to work on my machine. Upgrading the kernel to
the latest version solved all the problems. My sound card is PnP Vibra16.

don't support PnP, whereas the latest kernel does.

It isn't that hard to compile a kernel. Trust me, if I can do it, then
anyone can. Just read through the Kernel-HOWTO. If you need any help, my
