Re: Text Editors

I, personally, use pico. It does everything I need, and I know it quite
well, so I don't have to think about how to use it. When you load it up, it
puts a list of all the commands down the bottom, so they are pretty much
always on the screen. Also, if you use pine to read your email, I believe
pine is just an email front end to pico. However, I could be wrong there.

Now to actually edit the /etc/inittab file to load up x. I'll assume you're
using pico and a redhat machine. As the superuser (root) type in:

pico /etc/inittab

Make the change to start x. Namely, change the 3 on the runlevel line to 5.
Then press "Ctrl + X" and it will ask you if you want to save the changes.
Type "y" and then it will prompt you for a name to save it as, and you can
just press enter. Then next time you reboot, it should start x.
