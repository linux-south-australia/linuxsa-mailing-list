Re: NVidia driver pain

> Next problem was a message saying /usr/src/linux/include/linux/modversions.h 
> didn't exist, I had a look and there was a modsetver.h, so I did 
> ln -s modsetver.h modversions.h

You need to run at least a 'make dep' on your kernel source to generate
this file properly.

Assuming /usr/src/linux exists:

# cd /usr/src/linux

# make menuconfig

 ( set up your kernel if required, using this configurator )

# make dep


Now you should be able to build/install the NVIDIA driver.
