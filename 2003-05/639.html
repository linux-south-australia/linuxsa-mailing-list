Is the dialup ISP the same one as the Satellite ISP?

Best current practice for dialup ISP's includes ingress filtering, which 
means denying outbound packets from non-local IP addresses at the ISP's 
router. Sometimes the IP used by a satellite link is not one that a 
different ISP will consider 'local'. Some Satellite Internet systems use 
an IP tunnel to get around the problem, usually with some sort of 
windows software.
