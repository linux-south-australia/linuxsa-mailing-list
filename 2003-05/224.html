Re: LDAP text referance book

On Wed, 2003-05-07 at 10:09, Steve Sloan wrote:
> I am looking for a good book on LDAP.
> I see O'Reilly's have one in their range which looks like it might be OK.
> Can anyone recommend this one or any other books.
> I am interested in using LDAP for more then just an address book ie. 
> authenticating various Linux boxes and maybe some Windose ones as well.

O'Reilly's book is a great book if:

	* You want reasonable overview of LDAP
	* You're probably planning on using Open Source tools
	* You want an idea of the sorts of things people use LDAP for

There's also the LDAP bible, Understanding and Deploying Directory
Services (or similar name - Google knows). This book is huge and is
probably most useful if:

	* You plan on changing the LDAP schema regularly
	* You don't care too much about vendor implementation specifics
	* You need an indepth understanding of LDAP

Hope it helps,
Matt
