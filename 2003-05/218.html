Re: Redhat 9.0 vs 8.0, Comments?

I've upgraded from RH 8.0 to RH9 recently. Really I can't see the difference 
compared to RH 7-8 or even more so 6-7. I would almost call RH9 RH8.1. It is 
nice to have all the latest versions of programs, saves a huge RPM update, 
and so for that much is worth it. 

My install went smoothly, so unless you've got a highly customised system, I 
wouldn't expect too many problems.
