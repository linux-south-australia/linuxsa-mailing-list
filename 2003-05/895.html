Re: Knoppix on old Hardware

Yeah, I think the instructions to make a boot floppy are on the CD.

In fact I'll copy the relevant instructions here:

"In order to start the CD, first adjust the boot list in the BIOS Setup of 
your computer to boot from CD (before the hard drive). Put the CD in and 
restart the computer. If your computer does not support this option, you must 
use a boot diskette, which you can create with the help of the boot image 
/KNOPPIX/boot.img available on the CD. Use the mkfloppy.bat script for the 
regular boot image (German version). Use the command rawrite2 -f boot-en.img 
-d A (in the KNOPPIX directory) to create an English boot disk. "

You can also boot a machine without a CDROM drive if you have a server running 
knoppix and you make it an LTSP server (which is already set up, you just 
have to click a few things to turn it on).
