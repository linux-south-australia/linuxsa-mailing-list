Re: srtipping chars from filenames...

I'm writing a script to do it... I think it's ugly but it gets the job 
done :$ I invole it from the command line: ./stripchars * and it does 
the lot.

The only thing I have left to do is not perform the mv operation if the 
two names are identical... I think that's cosmetic (to supress error 
messages), but you never know... any hints on how to improve it?


#!/bin/sh
# Shifty behaviour


echo $temp

while [ $# -ne 0 ]
do
        filename=$1
        oldfilename=$1
        filename=`echo $filename | tr ':' '_'`
        export filename
        export oldfilename
        mv $oldfilename $filename
        shift

done
echo
exit 0
