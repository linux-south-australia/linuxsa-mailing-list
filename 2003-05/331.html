Re: Putting it all together / office multifunctionals & Linux

James Leone wrote:

> I need to be able to send a message to the multifinctional before it
> will respond by ftping the files I described earlier.
> 
> "220 Scaner server ready"  From port 3670 to port 1048.  Is there a
> shell command to do this?

Not really, but this is the standard banner greeting from FTP (try
"telnet localhost 21", then type "quit<CR>" to exit on an FTP enabled
machine).  If the printer is looking for that specific greeting, then
some of the more funkier FTP daemons (like ProFTP) may enable you to
alter the greeting, or at least give you the source code to allow you
to alter the greeting.  :-)
