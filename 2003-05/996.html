Re: Gentoo

I've been using Gentoo on my machines exclusively for almost a year
now.
There's not really any such thing as the "latest distro", because
even if you install 1.0a from CD, when you emerge -u world you get
all the latest stable ebuilds.

 It does mean though, that you'll be downloading about 1G of source
files if you want to install lots of stuff (Moz, Gnome, KDE, X,
OpenOffice.org, etc etc etc)..
 
 But, I have the latest (by yesterday) sources+portage tree, so if
anyone wants a copy then get me 2 blank CDs and I can burn it all,
along with the installation image.
