James,

At last someone asks a linux question that I know the answer to!
You must be trying to be kind to an old bloke.
The help messages are in 

Documentation/Configure.help

Strange place for it, don't you think?
One might delete the Documentation directory, thinking it was just
documentation, but it contains stuff for the kernel build.
And just to complete the picture, the file

X/Y/Config.in

has the button instructions for the "make xconfig" menu for
the modules in directory X/Y, e.g. X/Y = drivers/net.
But I expect you knew that already. And

X/Y/Makefile

contains the necessary back-end rules to make the button choices
in "make xconfig" actually do something useful, which I assume you
also knew already.
