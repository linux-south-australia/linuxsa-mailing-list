Re: Lilo

Sure! I once had Win2k, Red Hat 8.0, Mandrake 9.0, Debian 3.0 and FreeBSD 5.0 
on my system :)

I had a separate swap partition (primary partition) so that all three Linux's 
would use the same swap, Mandrake, Debian and Redhat are self-contained in 
their own partitions, and same with FreeBSD... Remember you can only have 4 
primary partitions. Redhat seems to by default create separate primaries for 
various partitions, whereas Mandrake will create one primary, and have a 
bunch of extended partitions inside of it (the preferred method). Debian sat 
in one of the extended partitions in the Mandrake primary partition. I used 
lilo to boot them all.
