Re: multiple KDE monitors experience?

Quoting Alan Kennington <ak.linuxsa@topology.org>:

> Does anyone have any recent experience of using 2 or more monitors
> attached to one linux box using KDE and a single mouse and keyboard?

I use such a setup every day at work, the extra screen real estate is great for 
getting things done. The easiest way to get into such a setup is with the 
Matrox graphics cards - some of them have an option for multiple outputs on the 
one AGP card, and a piece of software called "powerdesk for linux" that offers 
an easy GUI way to set up xinerama without having to know how the XF86Config 
file works.
