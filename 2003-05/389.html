Redhat 8.0 shipped with a known bug in the LANG settings file as
explained at http://redhat.cltonline.com/fom-serve/cache/14.html

Short answer is edit the file /etc/sysconfig/i18n and set the LANG= to
suit your needs. The file /usr/share/doc/initscripts-*/sysconfig.txt
explains more about the  redhat system configuration files in the
/etc/sysconfig/ directory.
