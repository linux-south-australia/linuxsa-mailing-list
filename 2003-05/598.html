I currently am using 3 monitors with a single desktop of 3072x768 (I
only have crappy monitors). The window manager needs to support the
xinerama extensions to X though. KDE uses kwin as it's window manager
and I haven't had any problems with it, but I use fvwm and fvwm-themes
myself. Incidently fvwm has both gnome and kde compliance, xinerama
support, session management, and much much more....

The man XF86Config page has information on the ServerLayout section and
as an example my XF86Config can be viewed at
http://awol.no-ip.org/awol/configs/XF86Config-4.triple. I have 3 video
cards but the minimum requirement AFAIK is 2. One has to be AGP and the
other PCI. You can also run two different windows managers on each
monitor but then you can't drag windows between them.

The really cool way to use xinerama is with vmware running windows. Then
you can have the left monitor for example running vmware windows in full
screen mode and linux on the right monitor, with the mouse moving
seamlessly between the two operating systems. Lots of fun!  ;-P
