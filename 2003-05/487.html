Re: PDF Create Script for Samba Printer

On Thu, 2003-05-15 at 09:39, Matthew Western wrote:
> -----smb.conf
> [pdf]
>         comment = PDF Printer
>         path = /etc/samba/pdfoutput
>         public = yes
>         writeable = yes
>         printable = yes
>         print command = /etc/samba/pdfwrite.sh %s %J
>         lpq command =
>         lprm command =

is samba passing it to the pdfwrite script in one piece?
if not, maybe try:
	print command = /etc/samba/pdfwrite.sh %s \"%J\"

if so, then it's the pdfwrite script you need to modify,
just quote/escape the var names, eg.

cp $OUTDIR/"$1".temp $OUTDIR/"$2".pdf

and so on (untested, might need some further fiddling).

[consider translating spaces to underscores or something
too, spaces in file names are a pain for lots of things,
eg. if you then want to web serve the files]
