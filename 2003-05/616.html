Re: multiple KDE monitors experience?

On Mon, 2003-05-19 at 00:20, Alan Kennington wrote:
> Does anyone have any recent experience of using 2 or more monitors
> attached to one linux box using KDE and a single mouse and keyboard?

Go for it Alan, once you've tried it you'll never go back.

I've run KDE in Xinerama over 2 monitors for about a year now.
KDE is one of the more Xinerama friendly DE's in that it's easily
configured so that windows won't be drawn with half the window on one
monitor & half on the other.

The keyboard & mouse behaviour you speak of happens by default, "so that
the mouse moves off monitor 1 to monitor 2 when you go off the right
edge, and the keyboard input follows the mouse cursor."

It only uses the one display. eg. 2 monitors @ 1280x1024 creates a
desktop resolution of 2560x1024.

The only downside is that you only get 3D hardware acceleration on the
1st head. This is an X limitation & only really an issue if you game
quite a bit (as I do). There are commercial X servers out there that
tackle this, XiG for example.
My setup is a GeForce4 MMX420 (AGP) & a Riva TNT2 (PCI).

On the Windows side of things, it automatically detects the 2nd gfx card
& attached monitor at boot-up like everything else. In Win98se anyway.
