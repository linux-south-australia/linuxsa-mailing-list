Re: mplayer

use ogle for dvd (xine sucks as a DVD player, mplayer is quite good but my 
version doesnt have menu support, ogle is fantastic and supports menu's, and 
has played every DVD i've ever thrown at it flawlessly)

http://www.dtek.chalmers.se/groups/dvd/

download the red hat 7.x rpm's... they worked for me.. this version allows 
multichannel  surround sound. If you have a sound card that supports this 
(such as a soundblaster live) please see my previous post on setting ogle up 
to use this feature.

also for mplayer, use the plf rpm's... im using mandrake 9.1, so im not sure 
if they'll work for you. but the plf packages for mandrake 9.0 should work.

You can use this page to add package sources to urpmi easily:

http://plf.zarb.org/~nanardon/urpmiweb.php

I recommend downloading the win32-codecs package too for more codec support. I 
also recommend kmplayer as a front end to mplayer (works really nice).
