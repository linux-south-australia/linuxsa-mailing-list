Re: Backing up data using tar

On Wed, May 14, 2003 at 09:22:20AM +0930, Martin Stacey wrote:
> Is it safe to backup the dev directory using the tar or should I avoid doing
> this?

I believe so, though I have never tried it. It certainly won't damage
your existing /dev, so you could always try it and just kill the
process if it starts writing an enormous tar file.

It is certainly safe to make a duplicate of /dev using cp -a:

  cp -a /dev/* /floppy/devbackup
