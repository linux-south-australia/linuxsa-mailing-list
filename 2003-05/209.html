Freeman, Peter (ERHS) wrote:
> 
> What have people used out there, ie: for student mail typically, is it
> successful, drawbacks, positives, ease of administration etc.

I have used Squirrelmail <http://www.squirrelmail.org/>; for some time. A 
few people who have a vee.net email addresss use it as their primary 
email client with not too many complaints.

On the positive side:

- uses IMAP to access user mail
- has a large number of plugins
- is easy to configure and upgrade
- is an active open source product
- can interface with an LDAP directory for user address books (via a plugin)

On the negative side:

- is written in PHP ;)
- doesn't handle large (10k messages) mailboxes very well
- probably others that I cannot think of ATM.
