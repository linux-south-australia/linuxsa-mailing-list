Re: odd .forward file syntax

Yep,
Seen it used it.

- it stops loops.
- if you had a .forward file that had in it "username" for a user called 
"username" then it'd go around and around.
- "\username" stops this.
- it allows you to do something like "\username,someotheruser" which will 
forward a copy to someotheruser as well as leaving it in the mbox for 
username.
