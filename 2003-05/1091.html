An excellent suggestion! An improvement would be to have the error text 
point to the URL of a web page. This web page would contain a form which 
allows you to add your email address/domain/ip-range to the server's 
whitelist.

Since most spammers forge the "From" address, they would rarely see the 
error text themselves. Other controls could be put in place to make the 
form unusable by bots and ineffective to be used by the spammers that do 
discover it.
