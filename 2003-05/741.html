Hi,

Just received one of the new Dell Inspiron 8500 machines.  One word:
Nice.  It was around the $4,500 mark I believe, but that's with options.
You could easily scale that back to $4k and still have a nice machine.
My only complaint is the slightly flimsy keyboard, but with the
brilliant 1920x1200 widescreen I can get over the keyboard issues easily
enough  :)

One thing tho, I still havent gotten around to installing Linux on it so
I don't have first hand advice on that, but Dell machines are usually a
breeze to install Linux on with the only gotcha usually being the video
drivers, and in this case the video card is an Nvidea GEForce4 4200 for
which there is Linux support so there shouldn't be any problems there.
