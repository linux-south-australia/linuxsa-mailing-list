You'd have to change the bios each time for whichever OS you want to
boot to. The better solution would be to get another cheap second hand
monitor. I'm not sure how the windows games would handle that though, as

aware.

Sounds like too much trouble to me.

> 
> In the meantime, if you could give any hints on booting with different
> video cards, it would be much appreciated.

Get to know your bios. Configuring for X is easy, X -configure works
just fine for me. BTW: X wont care which is the boot device but will
allocate Screen0 to it regardless which one you choose.
