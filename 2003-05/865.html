Richard,

Seriously, www.php.net is THE place.
http://php.resourceindex.com/ has some nice examples.

Or look on www.freshmeat.net  for php impementations of similar tasks
you're trying to achieve & pull that code apart to see how it's done.
For example, you may benefit from looking at the php code for a file
browser.

There's always #php on irc.freenode.net. Lots of gurus willing to help
out in real time. Provided your questions aren't too newbish.
