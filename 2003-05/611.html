X uses the terms "client" and "server" backwards compared to what most
people expect.  Using X terminology, a server is a graphic display, and
a client is an application, such as an email program or a clock, that
sends its output to the server.  Hence WinAxe is an X-server, not an
X-client.

I doubt that helps you, however I note that WinAxe's evaluation version
is limited to 1 user, and has the warning, "If you want to have more than
one simultaneous WinaXe users in same network environment, you need to
purchase commercial version (license) of WinaXe."
