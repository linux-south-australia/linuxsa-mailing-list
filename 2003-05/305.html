Re: [OT] Dynamic web content

In the immortal words of "Andrew Galdes" <AGaldes@elders.com.au>...
> This may be a simple one for the vb, js, asp, dhtml, perl etc people.
> I would like to be able to dynamically change the size if an image by
> entering a new value into an input box and hitting the "Calculate"
> button. 
> 
> What is the best way to do this? Any ideas?

The best way? depends on what resources you have available. If you have
some degree of control over the web server hosting this particular page,
I would recommend PHP with the GD libraries compiled in, and use
something like this snippet (grabbed from one of my projects) ....

------------------------------------------
    $source = "$mod_dir/photos/$gid/$img";
    $imgsize = GetImageSize($source);
    $width = $imgsize[0];
    $height = $imgsize[1];
    $new_width = 110;
    // $new_height = 70;
    $new_height = ($height / ($width / $new_width));
    $im = ImageCreateFromJPEG($source);
    $new_im = ImageCreate($new_width,$new_height);  
ImageCopyResized($new_im,$im,0,0,0,0,$new_width,$new_height,ImageSX($im
),Ima geSY($im));
    ImageJPEG($new_im, "$mod_dir/photos/$gid/thumbs/$img");
    imagedestroy($im);
    imagedestroy($new_im);
---------------------------------------

On the other hand, if you don't have access to the server you might have
to settle for a client side solution (JS or VBS), both of which will
produce results, just not necessarily good ones.

It all boils down to what you have to work with.  Maybe some more
information would help us give you a better answer.
