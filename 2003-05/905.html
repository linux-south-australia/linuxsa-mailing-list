I think you need to read the email again carefully - you've mucked up the
command and missed the identifier that preceeds the login, hence imap is
confused by what you're telling it.  Try reading the original at
http://www.linuxsa.org.au/mailing-list/2000-05/452.html - the "a01" "a02"
etc. bits are important (essentially, imap commands aren't strictly
sequential - it's more of an asynchronous protocol, which allows for
multiple commands to be sent before being completed).
