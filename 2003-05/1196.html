Re: Can Linux handle really new systems?

I tried Mandrake 9.1 on an Nvidia nforce2 box about a month or so ago, and it 
wasn't entirely successful. The Hard drive is running at ata-33... but I did 
get the modem, network, sound and graphics working (though the sound card 
only allows one program to open /dev/dsp at a time). I had some weird 
problems which I blame on IRQ sharing not working properly... e.g. when doing 
a cd burn, I couldn't make any applicaiton run in fullscreen (I think that 
was the problem, I can't quite remember now). Burns would sometimes fail too, 
not sure why.

Anyway, I think the nforce2 is a bit flakey with Linux at the moment (I had 3 
proprietry drivers on that thing! Winmodem, network card, and graphics card). 
Though it's probably "useable". If you had a VIA chipset or something, it 
might be a different story.

Damien
