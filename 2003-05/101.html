Re: Combating Spam in a relatively large organisation

Spamassassin is in the FreeBSD collection but it can catch the wrong
type of spam...nonetheless it can add a header with a certain amount of
"stars" which Outlook Express and Outlook can filter on. Then each user
can setup their own "spam tolerance" level and you could have a
company-wide default "tolerance" level [which you'd probably want to set
fairly high].
