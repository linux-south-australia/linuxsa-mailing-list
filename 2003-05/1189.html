> "no connection permissions"

I think that RedHat 7 uses LPD. Do a man LPD and work out how to get LPD
to:

 a) NOT go into the background when run
    - not run as a daemon
 b) Print copious amounts of debug messages

(I know it can be done but I've forgotten the exact switches)

 c) STOP lpd
    # service lpd stop
    (or whatever the printing service is called - look in
     /etc/rc.d/init.d)

 d) Start it using the options you found in (a) and (b)
    # lpd {options options}

 e) Go looking for debug messages

The debug messages are likely in /var/log/lp*...look for any messages
there FIRST before doing a-e above btw.
