RE: Bash function query

>I recently created (in Bash) a test function that I called oops, but
>having created it, where is it and how can I edit it or delete it? 
>Running "updatedb" and then running "locate oops" doesn't find the
>function anywhere (that I can see).

locate only tells you about _files_, so unless you saved it to a file,
locate won't find it...

>
>So, what happens to functions that one creates "on the fly" 
>(i.e. at the
>command prompt)?  Where are they stored?  (Yes, I checked the bash man
>page for this, but it doesn't mention how to manage user-created bash
>functions.  BTW, as superuser I cannot use the function "oops" at all,
>which is strange 'cause I thought the root user could see and do
>*anything*.)

it's saved _in the running bash process_ ... you can put a bash function in
a file (say your .bashrc), and then any time after sourcing or running that
file, you can access the function. Superuser can't run it because it's not
defined in the superuser's running shell... It's like an alias in that
sense...
