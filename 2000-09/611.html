Re: Perl Syslog Problem

I've discovered my problem. According to the documentation, Syslog.pm
will try an INET socket by default. Because I'm running syslogd locally,
it's using a UNIX socket (more eficient apparently)....hence I need to:

use System::Syslog qw(:DEFAULT setlogsock);

setlogsock('unix');
...


Then it works!
