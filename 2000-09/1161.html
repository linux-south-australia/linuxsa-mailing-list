RedHat 7.0 on mirror.aarnet.edu.au

Today is actually "national install day" :-)

Red Hat Linux 7.0 is now available for download from
  http://mirror.aarnet.edu.au/pub/linux/redhat/redhat-7.0/

AARNet apologises for the rather confusing error message that
people encountered if they attempted to access the distibution
before its formal release date.  Red Hat did not wish mirrors
to allow downloads until the formal release date, that allowed
all mirrors to refresh before Red Hat's capacity was flooded.

Performance is likely to be somewhat slow today.  Shortly
the mirror will move to new hardware attached to a gigabit link
in Canberra, fixing future performance problems (thanks to
Sun and CSIRO).
