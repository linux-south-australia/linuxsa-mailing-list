RE: Missing Files

[quoting from random people here]
> > Today's question is, which one do I install?
> >
> > Is there a command to search for a specific file in the
> > rpm's that have not been installed yet?
>
> rpm -q -l -p *.rpm
>
>    -p <package_file>
>               Query an (uninstalled) package <package_file>.  The
>               <package_file> may be specified as an ftp  or  http
>               style URL, in which case the package header will be
>               downloaded and queried. See  FTP/HTTP  OPTIONS  for
>               information  on  RPM's built-in ftp and http client
>               support.
>
>  -l lists the file sin the package.

As someone pointed out before, this won't work.  Look further down the
man page for the --filesbypkg option, and try this:

rpm -qp --filesbypkg *.rpm

> A colleague of mine says there is a way to customise rpm output, but that
> seems like a lot of work for a one off query.

The above command with '--filesbypkg' is equivalent to:

rpm -qp --qf '[%-25{=NAME} %{FILENAMES}\n]' *.rpm
