Re: Vi or Emacs?

Emacs is clearly the only text editor worth learning.  You can do anything
with it, run any program, control any system.  Don't doubt me: it includes
a full-featured lisp interpreter.  It's more than just a simple text
editor: it's a mail reader, news reader, file browser and so much more.
Emacs is the Microsoft Word of text editors.

By contrast, vi is just a text editor, and of limited use if you want to do
anything else.
