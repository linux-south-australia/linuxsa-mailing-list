Re: Curious results with Linux 2.2.14 and 100Mb/s Ethernet

Hi,

As a postscript to this, I concluded that the problem was file system related.

I think I have verified this now, using the following technique:

  1. Create a 20MB file on the server
  2. Force it into the cache, using cat a couple of times
  3. Retrieve the file a few times using ncftp

I can get up to about 6.8MB/s using this method, which is a whole heap
better, and suggests that my pervious analysis is correct.
