Re: Vi or Emacs?

It matters less and less now. Pretty much the only difference is vi=small emacs=big.

Both allow you to edit multiple files
Both have syntax highlighting for more languages than most sane people use
Both have text and graphical interfaces
Both use key sequences that most mere mortals cry about when they first start out
Both take years of practise to use half of the features
Both have O'Reilly books about them ;-)

Learn both, and learn ex and sed and everything else like that, because one day, you'll find
yourself working on a box in an environment where they don't allow things like emacs and perl
to be installed. Not that I can talk ;-)
