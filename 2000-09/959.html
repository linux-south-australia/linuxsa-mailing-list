G'day,

On Wed, 20 Sep 2000 23:50:40 MWP wrote:
> Does anyone know of a small util for linux (console only, no X)
> that will resize jpegs??
> GD is not an option at the mo :(

Yes, and it IS a command line program, and is part of ImageMagick. 

I use it in cron scripts for a couple different jobs. 

The program name is "convert", and it gets installed in the X11/bin
directory when you install ImageMagick. Since it's a command line
program, I put a symbolic link from it to /usr/bin so non-X users
could make use of it as well. 

Convert also does a HEAP of other stuff, all from the command line or
in script programs.  It's a very useful utility.
