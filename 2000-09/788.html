XConfigurator is a Red Hat-specific tool, in the same way that SaX is
SuSE-specific and turboxconfig is Turbolinux-specific.

The tool which comes with XFree86 is XF86Setup. This works by putting the
video card into VGA mode (which should work on very nearly all hardware)
if X is not already running and presenting a simple and helpful GUI. As
far as I know all distributions leave it in place when X is installed. It
works well and is nicely designed.
