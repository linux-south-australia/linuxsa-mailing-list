slack admins in the UK

Anyone who thinks that my modus operandi for linux
security is a bit slack might be amused by this little piece:

http://www.theregister.co.uk/content/1/13316.html

It's about 160 UK sites which have been hacked in protest
at the 72% tax component in their petrol price (as opposed
to our modest 47% or whatever).
This seems to show that 160 UK admins don't change
default passwords on MS database software.

<philosophy>
This starts to suggest to me that every linux distribution
should have a checklist with it to tell you what things
you need to turn off if you want to put the machine on the net.
Every distribution (and every version of every distribution) seems
to have different things that need to be closed down.
So if you're running several distributions, you can't really
be expected to study every one of them for security if you've
got some productive work to do.
</philosophy>
