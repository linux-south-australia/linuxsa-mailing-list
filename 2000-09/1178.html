Re: Wireless LANS, and the Samsung MagicLan

Hi

I been running a linux based wireless lan now for about 4 months.

I have 3 cards all PCMCIA, I also have an ISA Pcmcia interface card which is
installed in my linux router.

Two of my card are 802.11 (2Mb) and one is 803.11b (11Mb) the 11Mb on is in
my router.

You do not need to us an AP (bases station) if you do not want to, I have a
dhcp server which does all the connection configuration stuff.

I currently get about 300 Metres with the standard equipment.

I using the Zoomtel hardware for which I am a reseller, this equipment has
no problems connecting to a Airport. I did have some trouble getting the
driver to compile on my G3. I was able to fix this about 2 month ago.

Cheers
    Mike
