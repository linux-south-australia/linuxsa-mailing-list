Robyn,

if you've setup your firewall correctly, most of access to your box
should be "denied", except for web & mail services. Turn off telnet - do
not use, only use ssh.  The same goes with ftp, if you are going to use
ftp use PROftp, it is a more secure implementation.

You can also run ethereal (Richard Sharp) to see what's happening with
the back and forth network traffic and you can see what port are used by
by what services.

If you're not sure about you firewall email me and I'll check it out for
you.
