Key-bindings were removed from Blackbox ever since the 0.60.x
alphas.  Blackbox aims to me completely ICCM complient, and since
key-bindings are not specified, it does not support them.  However, there
is an excellent little program called 'bbkeys' that handles not only what
blackbox used to, but much more.  It even has a nice Qt config tool.  Grab
it from http://movingparts.thelinuxcommunity.org/bbkeys.shtml.

You can get the rest of the bbtools from
http://bbtools.thelinuxcommunity.org.  There are some very useful
applications here, including a pager.  They make blackbox a very
comfortable working environment.

There are lots more over at http://bb.themes.org.  A guy called Artwiz has

http://artramp.org/artwiz/themes.phtml.

Not anymore.  Blackbox(1) has a very detailed (but simple and easy to
understand) man page.  It describes the simple syntax used in the menu
file, the format of the .blackboxrc file (most of which can be configured
from the run-time config menu anyway), and the resources used in the style
files.

Don't grab the 'stable' version.  It is VERY old.  The latest version is
0.61.0 and it is labelled as 'beta.'  Don't let this fool you, I run
Blackbox the entire time my PC is on (about 12-15 hours / day), and have
been for the last year or so, and it has never once crashed on me.
