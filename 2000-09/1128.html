Hmm.  I'm not sure this is a good idea.  If we work more closely
together, there will be some people who would potentially be able to
attend in multiple locations, for example national computer press
people.  Obviously this would only work if they were at different
times.

> Also, if we can make some sort of projection or target for numbers
> of people, PCs, and installs, it could generate interest...

Well, how do you estimate it?

With my AUUG hat on, what can AUUG do to help?  At our last committee
meeting we agreed that we would very much like to help in this kind of
event, and I think we're in a position to do so, especially since
we're a nation wide organization.  Questions to you:

1.  Were you happy with the AUUG involvement in the last Installfest?
2.  Is there something more we can do?
3.  Are there things which we could have done better, or just kept out
    of altogether?

One possibility would be to declare a nationwide installfest as a
cooperation between AUUG and each individual Linux user group.  As I'm
sure you know, the SA event was by far the most successful, and AFAIK
it was the only one in which AUUG was involved :-)
