Cron reader

Does anyone know of a tool that will read a crontab, and translate the
obscure syntax into plain English text? Preferably a command line tool, and
it has to deal with all legal cron syntax, and complain if the syntax is not
legal.

ie I want something that can tell me that:

0 0,4,8,12,16,20 * * * <command>

means "every day, every 4 hours, on the hour, run <command>"

(instead of "every 4 hours", "at midnight, 4am, 8am, midday, 4pm and 8pm"
would be fine)

I'd really like it to deal with things like "0-23/4" as well.

It would be ideal if there was a command line tool that enabled you to set
cron as well, but this isn't essential...

I have looked on freshmeat, and all I can find is GUIs for doing this kind
of thing...
