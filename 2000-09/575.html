NT will raise an exception which should be handled by the programmer.
ie in VB

On error goto BabyDidABoneHeadThing
    y=x/0
.
.
    Exit Sub
BabyDidABoneHeadThing:
  msgbox Err.Description

End Sub

In Delphi

try
    y:=x/0
except
    on e:EDivideByZero
      showmessage(e.Message);
end;

etc..

C++ also uses Try blocks... Of course you can do the same kind of thing as
Daryl shows under Windows with C and C++ as well, I assume the signal
library exists, not 100% sure on that one, but who really cares anyway!!!!
