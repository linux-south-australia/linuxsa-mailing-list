Mozilla Recompiled at Last

Heh!

I finally managed to get the "latest" mozilla snapshot at
ftp://ftp.mirror.aarnet.edu.au/ to recompile. It took me about 8
rebuilds because it kept on having compile time errors when I attempted
to prevent debugging and other such exotic configurations. This isn't
for the faint of heart though: the TAR ball is 26M and it takes about 20
minutes (at least) to rebuild on my Athlon 650M with 320M of memory...

* it's faster than M17
 - not that this would be difficult

* mail actually works
 - it crashes though, so I wouldn't trust it

* rendering is *very* fast
 - it renders at least as quickly as Internet Explorer

* Java
 - what Java? 

* JavaScript
 - kinda

Unfortunately it uses JavaScript internally to render the XUL (the user
interface, which is in itself an XML definition). This can make it
appear to be a little clunky :-( Other than that it's quite nice...
