RE: Squid mount full

What you are looking it is your squid proxy cache.

The space allocated to this cache is set in your squid.conf file

These files are created by squid (first created using squid -z) 

If you have much web use you may also wish to reduce the logging from squid,
or choose to logrotate with compression although this may cause issue if you
are using proxy log parsing tools.
