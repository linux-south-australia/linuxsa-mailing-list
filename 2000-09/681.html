Re: mbr or primary partition

Hello
Look at /etc/lilo.conf
it should say something like
boot=/dev/hda
(it could be hdb etc... sd? [i think] for scsi hard drives]
Basically, if it doesnt have a number after it, its in the MBR
If there's a number, lilo is at the start of that partition.

For example:
hda = whole hard drive [lilo in MBR]
hda4 = partition [lilo at start of partition]

Hope this helps :)
Ryan

P.S. Ideally, if you have no other boot loader, lilo should be installed in
the MBR.  If you wish to use a different boot loader you can tell it to load
linux off the partition, then load lilo into the start of that partition.
This means you will still be able to choose kernels / define kernel options
etc...
