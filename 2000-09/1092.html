Re: eeprom burner

> 1.  How does your friend know his BIOS is virus-infected?
> 2.  How did it happen if it hasn't been near a PROM burner?
> 3.  You don't need a PROM burner to flash a flash ROM.
> 4.  If it's not a flash ROM, what kind of PROMs are they?

I might make a guess at this: There are viruses that attempt to rewrite the
BIOS on machines with flash-writable BIOS chips.  These viruses often
result in a motherboard that cannot boot.  If your motherboard is in that
state you obviously can't use the manufacturer's supplied flash program,
because you need to be able to boot the machine in order to run the
program, hence you either replace the BIOS or re-program it using an
external programmer.
