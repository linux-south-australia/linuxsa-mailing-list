RE: sshd message query (from a newbie)

Paul,

It's complaining because it can't find the host public and private keys. You
can create them by running ssh-keygen (or it might be sshd-keygen - can't
remember) as root, hitting enter when it asks for a pass phrase and entering
/etc/ssh/ssh_host_key as the path when prompted.
That will leave the sshd running.
