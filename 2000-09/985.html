I've always been a vim person.  But, I've never really tried Emacs.  Does
anyone have a good learn-the-basics-of-emacs-fast link?

I typed 'emacs' at the command prompt, then couldn't work out how to quit,
so my only option was to cut the power to the building.

And can I just say..  Vim is damn cool.  As I said, I've not really tried
emacs, but vim does some incredibly cool things.  I can probably guarantee
that it's faster to do *any* sort of edit using vim/vi than using any other
text editor on the planet - its command structure is gorgeous (when you get
to know it :)

It colourises, it auto-indents, it understands loads of programming
thingies, it does line-numbering, the most commonly-used functions are all
single keystrokes, its search/replace/etc functionality is perfect, and
someone wrote a script to generate and solve a maze in it.  Now that is
just cool.  AND it loads in a split second.  I don't know about emacs.
Vi/vim is fast, sleek, inredibly powerful, and deceptively intuitive.  ;)
