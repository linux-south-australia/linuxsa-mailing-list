Re: AMANDA Clients

Hi,

> I look after a novell server, and I was wondering if it would be
> possible to write a 
> client for AMANDA that would work with NetWare 5. The reason that I ask
> is that 
> Netware uses NDS Database to store file perms etc.. Keeping the ACLs on
> files is a
> fairly big requirement when you have a server with 1100 users on it..

Backing up the NDS under netware is a bigger problem tghan backing up the
NT registry. It'd be equivalent (only moreso) of needing to backup and
restore a NIS database. If netware5 can dump the NDS to an export format,
this'd be the only way to do it, otherwise you need a backup agent thaty
has admin rights to the NDS, and its own method of dumping the NDS.

The best method of NDS backup is having another server as a replica. Loss
of one, means that the other can take over and restore the NDS once it has
returned.
