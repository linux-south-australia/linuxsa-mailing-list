Re: Boot from floppy

Evan Bourlotos wrote:
> 
> Hi all,
> 
> I am foreseeing a need to wipe one of my drives, (windoz) sadly this is
> where MBR/Lilo live. I reall had had to run loadlin etc after getting it
> from cd rom, but chances are primary drive will not see an o/s immediately
> so I still need to access linux. Is there a way of making a boot floppy
> which will allow me to boot as normal into linux.
> 

dd if=/vmlinuz of=/dev/fd0

Will copy /vmlinuz to the first floppy drive. It'll do it byte by byte
and create a bootable floppy (ar at least it used to), you may need to
do a 'make zImage' with you kernel source and dd the
/usr/src/linux/arch/i386/boot/zImage file to the floppy.

Hope it helps,
