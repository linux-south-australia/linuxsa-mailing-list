Re: ADSL Modem Alternatives

Hi,

If Telstra wanted to detect if you where using IP-Masquerading, they could:

  Check for a statistically high number of outgoing connections from
    ports 61000-65096 (to which I say, modify them in include/net/ip_masq.h
    before compiling your kernel).
  Check for unusual TTL values on your packets (fluctuations between 63,
    127 and 254 for various OS types behind the firewall).

But why bother, so they can sell you another IP for a few dollars?  
Technically it is not so difficult.  The public relations/physical 
investigation overhead would hopefully be too high.

Sean
