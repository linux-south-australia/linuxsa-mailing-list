> How's KOffice doing these days?

I have been looking at KOffice lately. It is extremely impressive in terms
of componentised design, and underneath the code looks quite easy to
follow in the bits I've scanned. 

In terms of functionality it does everything most people would need for
basic office suite functionality, but nothing like what everyone will
want.StarOffice has a very complete (and Microsoft-like) set of features,
but is quite fat and very frustratingly buggy once you start to get
serious (ie a lot of features work to 95% accuracy.)

KOffice isn't released yet, and the pre-release versions up until a month
ago (the latest I've tried) still crash a bit.
