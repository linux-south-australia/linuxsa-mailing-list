Hi Brian,

Encapsulated PostScript is a vector image format.

It can't be printed without embedding it into a PostScript document.

One slack way of doing this without losing resolution would be:

  epstopdf fred.eps | pdftops -paperw 595 -paperh 841 | lpr

The proper way it in Adobe's PostScript manual, which I think
is online.  Essentially, you extract the bounding box from the
EPS file, create a centered clip path on the page, save the
dictionaries, stack and state, set the translation and scaling,
redefine showpage, copy in the EPS file, restore the
dictionaries, and showpage.  Not too much Perl, which means a
search on Freshmeat should show up heaps of eps2ps utilities.
