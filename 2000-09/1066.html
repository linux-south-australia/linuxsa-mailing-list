Re: Debian and PPP

> I am using pppconfig and pon to connect to my ISP, however in pppconfig
> there is an option for nameservers but only ip numbers primary and
> secondary.  Where do I put the name?

If you think about it you will see that there cannot be anywhere to put
the name.  We're talking about how to configure system to access the DNS,
which translates names into numbers.  IP requires an address before you
can talk to a remote host.  If you only have the name of the DNS then
how do you find it's address in order to ask it to translate a name into
an address?  See: obvious, isn't it?
