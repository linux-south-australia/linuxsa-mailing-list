Hi,

> What is likely to make more of a difference is decent remote
> managment tools for system administration.  Unfortunately,
> UNIX is just as bad as Windows in this respect (for example,
> if you change mail software from sendmail to postfix you
> have to rewrite the configuration file), especially when both
> are compared to Novell NetWare.

Novell Netware has some very nice features of security. The inheritance
fetures are particularly nice, and likewise with the encryption on the
networking protocls, however I think the two main reasons it is so secure
are that it is no widely used compared to other types of systems, so there
are fewer people working on exploits of bugs that may exist, and also it
has only very recently started supporting TCP/IP in any major way, so
internet atacks on Netware hosts weren't possible till netware 5 in any
major way.
