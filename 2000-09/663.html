Viewing open files on Samba shares

Does anyone know of some good tools for administering open files/ active
users etc on Samba shares.
I'm setting up a couple drives on a Linux box for file sharing which will be
mapped as a drive in our main NT server logon script.

monitoring open files/users that the NT server manager has.
