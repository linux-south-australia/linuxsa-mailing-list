Re: Screen Dumps in Linux

On Fri, 22 Sep 2000, Evan Bourlotos wrote:
 
> I was wondering how I can get screen dumps under linux. I am using rh6.1
> and helix gnome.

Evan, I have found 'import' from the ImageMagick set of programs to be a
wonderful tool.

To grab the whole screen:
$ import -window root screenshot.png

To grab a single window, with decorations:
$ import -frame window.png (and then click on the window)

PNG, of course, is not the only format import can handle, it also does
JPGs and BMPs, plus some others.  You can also grab windows by putting
their window-ID into the command.  Check the manpage for more.

Hope this helps,
