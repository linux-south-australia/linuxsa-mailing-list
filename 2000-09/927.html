Java ssh client

MindTerm is an SSH client written in portable Java.

  http://www.mindbright.se/mindterm/

Basically, you place the MindTerm lite code on your web site
and call it from a web page on your web site.

Then from anywhere with a working Java-enabled browser
you can access your web page, which downloads the applet,
and you have a SSH terminal.  You can then access your
hosts.

If you allow only key access (that is, you never allow the
Password prompt), then you can configure MindTerm on the
fly to look for the keys on a floppy disk.

"Note that MindTerm does NOT work with Netscape versions
 before 4.06"

Alternatively, you can run the full version from the
command line (if you have installed Blackdown's Java
for Linux):

  java -cp mindtermfull.jar mindbright.application.MindTerm

The software is solid and fast.  The emulation is reasonable,
and the display is solid.  Some keys are not mapped.  vi
and pine work fine.

                         ------------------

One point not made last night is that it is worthwhile configuring
the SSH server to accept only public/private key authentication
and never allow a password as authentication.

Doing this allows you to accept SSH connections from anywhere
without giving the whole world the opportunity to guess passwords.
The world is welcome to guess the private key, as the sun will
probably explode before they find the right one.
