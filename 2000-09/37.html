Re: XFree86 3.3.6

Hi.

Yes, I have it running.

There were no binary rpm's at the time, and I doubted I'd get the source
compiled and installed without a weekend's work, so I grabbed the binary
install files from

mirror.aarnet.edu.au/pub/XFree86/4.0.1/binaries/Linux-ix86-glibc21/

If you get them all, it's a monster download of 28mb. I did.

I then installed as per the instructions in that directory, over the top
of the existing install. (yes, I know this breaks the rpm
install/upgrade ability).

The only hangup I had was that the new xfree uses a modified file format
for the XF86Config file, and the help is not complete. My monitor was not
in the database, so I had a bit of tinkering to get everything talking.

It works, I'm happy :)

Michael
