Re: Screen Dumps in Linux

The command 'xwd' is your friend.  Type it at the command line, click
on the window you want.. wait for the beep.  It sends it's output to
stdout (standard output) so you need to do something like..

xwd > file-to-output-to

if in doubt, man xwd

'Gimp' can then be used to crop, or otherwise edit the 
image and save it to another format.
