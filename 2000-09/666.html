Re: samba client for backing up to an NT box

David Lloyd wrote:
> Personally I'd take the tape out of the NT machine, and put it in a
> Linux machine running AMANDA (http://www.amanda.org/) but that's just my
> own choice...you've probably got other restrictions on what you can use
> and how to do it.

Given that it's in an NT/Linux environment, I feel that AMANDA would be
a bad choice, unless it's just a case of backing up files, and not
machines. From what I have seen, if you want full system recovery from
backups in a NT/Linux environment, using an NT server with 3rd party
software is the only solution.

I'm happy to be proven wrong if someone has other ideas/solutions.
