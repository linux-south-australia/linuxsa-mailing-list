RE: how to move a linked file and link it

Hi Russell,

If it is a standard RH script, it should have the chkconfig capability
already built in.

If, as root, you type:
	/sbin/chkconfig --list crond
You should see something like:
	crond           0:off   1:off   2:on    3:on    4:on    5:on    6:off

To enable crond at run-level 3, type
	/sbin/chkconfig --level 3 crond on

This functionality can be enabled for any script in /etc/rc.d/init.d by
adding the following lines to the script.

# chkconfig: 2345 40 60
# description: some helpful description of the script

This will allow chkconfig to do its stuff. Do a man on chkconfig to get all
the facts.
