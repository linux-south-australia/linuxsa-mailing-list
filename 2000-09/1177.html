Re: PCMCIA modems

> Do people know of *exact* model numbers & suppliers of real hardware
> PCMCIA modems.

Xircom Cardbus Realport 10/100+Modem 56
RBEM56G-100

I don't know what modem chips it uses (it uses tulip for the 
Fast Ethernet), but it appears as a serial port.

Might be a tad expensive for just the modem, but hey.. you 
get a fast ethernet port as well.

Problems: 
One ethernet problem at the moment.  Under Debian (2.2-woody) the card
needs to be put into promiscuous mode before it will work correctly.

This is a problem with the driver, not updating the arp table correctly
I think.  It's a MAC<->IP problem anyway.  It took ages to figure out 
why suddenly the networking would start working again, when I started 
to look at problem. (Read.. start using tcpdump.)  The whole situation
had a very quantum mechanical feel about it.. 

I've hacked the network scripts to automatically bring the card up
in 'promisc' mode but I would like to report these problems to the
Debian and/or Kernal mailing list. Anyone with experience know the best
approach..
