Addendum on Package managers

Hi,

An interesting asside on package management from today. I was upgrading a
RH6.1 box. I obtained the kernel 2.2.17 source and compiled the kernel,
loaded fine, then proceeded to try and update the NFS package. Despite the
fact I was running a 2.2.17 kernel, since I hadn't installed the >2.2.14
kernel RPM, rpm wouldn't let me install the package !!

I eventually fixed it by installing a kernel package in adition to the one
I installed.
