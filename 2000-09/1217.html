Re: Missing Files

> Is there a command to search for a specific file in the
> rpm's that have not been installed yet?

Yes/No.  There is not a specific command, but there is a specific
setup under which you can _efficiently_ answer this query.

Install, the package "rpmdb-redhat".  It is the database after
installing every single package.  You can the query this complete
installation database with commands using the "--dbpath" option.  For
example,

	rpm --dbpath=/usr/lib/rpmdb/i386-redhat-linux/redhat/ \
	    -q -f /usr/sbin/pppd
