Re: Debian 2.2 Problems

John,

I recommend that you install the package "kernel-package" and take a 
look at the README in the /usr/docs/kernel-package directory.  This
provides a way in which to make your Debian kernel into a Debian
package so that you can install and remove it with dpkg.  The process
will ensure that any Debian-specific conventions are followed, in
particular the post-install script is there to take care of modules.
However, (when I last used it) I had to sort out lilo manually.

Good luck,
