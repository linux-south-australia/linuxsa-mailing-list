This is pretty much exactly what Bastille Linux is.  I have seen some
comments about it in the replies, but I am not sure whether they had
run it, since it was passed off as a pre-rolled firewall.

It checks off the security problems on a stock RedHat machine,
provides a brief explanation, and offers to fix the problems.  It is
meant to harden the system and inform the administrator.  Also, it has
a developer community.

While it is designed for RedHat and Mandrake machines, each step
contains a brief explanation.  Furthermore, it can be run in test mode
so that you can follow the steps, read the explanations, and apply the
steps manually.

Of course, it was not invented here ;-).
