Okay, the menu editor is in the "(GNOME) Programs" menu, under Settings, 
Menu editor. So, if you go in and turn on this menu as a submenu, you 
will end up going to Programs > Programs > Settings > Menu editor.

If that still doesn't work, the program is called 'gmenu'. Just run it 
as root, and customise away :-)
