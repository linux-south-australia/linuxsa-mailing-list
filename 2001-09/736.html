On Wed, 2001-09-26 at 00:30, Shane Warner wrote:

> I have a network card that is detected as 'Vendor ID: EDI0119', yet I can not 
> seem to configure and use it.

Can you send the output of 'dmesg'?

> Most web paes I have read on similar problems, usually with other devices 
> such as modems, suggest putting values into file and recompiling kernels etc.

The GENERIC kernel usually covers a fair bit, but you may need to
recompile. To do that, I'd recommend you update your sources. The
easiest way to do that would be to install FreeBSD-4.4

> I performed a bare minimum installation on an old 486sx with the intention of

How did you install the software in the first place if there is no means
of connecting a CD-ROM?

> Does anyone have any suggestions as to how to go about manually configuring

Even without a PCI slot, I'm sure digging up an n-hand 3Com ISA card
wouldn't be too hard.
