 > Quoting Mark Newton <newton@atdot.dotat.org>:
 > > It MD5's the distfiles (source archives).  If the source you download
 > > doesn't agree with the MD5 from the CD-ROM ports tree (or the ports
 > > tree from ftp.freebsd.org, or whatever), then the package won't install.
 > 
 > Why can't a cracker update source and md5sums?

Because they'd need to break into the source repository to update
the source, then they'd need to break into your machine (and/or
ftp.freebsd.org) to update the md5sums.  

And if they've broken into your machine already, they probably don't
need to install a trojaned port to gain further access :-)
