Re: For Sale: SGI Indy

On Tue, Sep 04, 2001 at 09:53:51AM +1000, Michael Wardle wrote:

 > Naturally as the system is Unix, it is also possible to download and 
 > build the source for any GNU software.  If you are doing this, you may 
 > like to check whether you are getting a C compiler.  The SGI C compiler 
 > is called MIPSPro, however you should also be able to obtain gcc.

Trap for young players:  Vigilant observers will note that gcc is available
at http://freeware.sgi.com -- But what isn't mentioned is the fact that
the compiler from that source only works if you've already bought/installed
the MIPSPro compilers, because it needs crt0.o and all the .h and .a files
from the SGI compiler set.

... so don't think you can get a free software development environment
for IRIX by downloading gcc from freeware.sgi.com.  You'll need to find
it from somewhere else.

Happily enough, you don't need to care too much about this if you have
an IRIX 6.5 media kit (installation CD-ROMs):  6.5 was bundled with
the C compiler.  It complains about a missing license key but goes ahead
and works anyway.

You lose the compiler if you upgrade to 6.5.1:  The patchset which 
accomplishes that upgrade "fixes" the bug in the C compiler's license
checking code, so that when it complains about a missing license it'll
actually *not work*.  Sigh.
