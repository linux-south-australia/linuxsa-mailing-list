> > I then immediately try to mount it using	mount /dev/fd0 /mnt/floppy
> > and I get an error message saying it has an incorrect superblock structure.

I don't think that FFS is designed for floppy disks. Use MS-DOS' FAT file
system. If you are, you may need to specify that on the command line:

 mount -t msdos /dev/fd0 /mnt/floppy
