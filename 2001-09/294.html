Re: disapearing disk space problem

Horst,

A common reason for disappearing disk space is a program writing to a
file that you removed.  If a file is open, removing it (more properly,
unlinking it) only frees the disk blocks after it has been closed.
(By the way, that's a great way of maintaining a hidden file store for
worms and viruses.)

Check to see if you have any long running processes, and restart them.
