Re: Locating ethernet card I/O address

Nathan Millhouse wrote:
> Hi all, 
> 
> I'm setting up a RH 5.2 box with 2 ethernet cards in it. 
> 
> The first card is and 3COM card and is working fine, but I also want 
> to use a Surecom card
> using the NE2000 module ne.o. I know this card works because I have 
> another RH 5.2 machine
> working with an identical card. The problem is that to initialise the 
> secondary ISA card I need to supply options
> within /etc/conf.modules.
> 
> Here is the current file
>  I know I need to supply the I/O address wher I have placed NNN
> 
> alias eth0 3c509
> alias eth1 ne
> options ne io=0xNNN  // This line not actually in file yet
> 
> How do I find the I/O address?

Probe for it.

modprobe 3c509 io=0x280
modprobe 3c509 io=0x300

etc, until you find it.
