Re: Icons

John,

    I have only done this on the individual icon level, not on a whole
slurry of them, but I presume it works.  I simply transferred anything with
an .ico suffix and used Icon Editor in Red Hat.  It seemed to recognise the
icons flawlessly and I then converted the icon to .xpm format and inserted
it into the directory with all the other icons in it.  (usually
/usr/share/icons).  I found this technique particularly useful if I had a
windows icon for a windows device which I wanted to work in Linux.  I would
get the icon and make mounted and unmounted forms of it and convert it to
.xpm format.

    Because it all worked well, I haven't investigated further.  There is
nothing like a problem to scratch to help you learn more.  Have fun.
