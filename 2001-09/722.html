> shouldn't this be "HELO" instead of "EHLO"??

No. That's how you get the extra information from the SMTP server. If you
want to know more about this stuff, have a look at the SMTP RFC(s). There
are some cute things you can do this way that can give you more
information, start mail transfers and allow you to exploit security holes
:-).
