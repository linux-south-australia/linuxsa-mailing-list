Re: RAM strangeness on Athlon box

matthew@datadeliverance.com wrote:

> Anyway, I have recently upgraded it from 256M RAM (2 x 128M), to 512, by
> putting a 256M dimm into the last memory slot.  The BIOS recognises that it
> has 512M, but linux kernel panics intermittently if I run with "mem=512M" as

Probably not related to your problem, but I found recently that Linux
has a Kernel parameter that needs to be set to allow memory larger than
1Gb. Nothing that I'd ever paid any attention to before, because 512Mb
dimms weren't readily available until recently.

After trying the lilo mem=1500M tricks etc, I'd convinced myself that
the motherboard wasn't reporting the whole amount of memory to the OS. A
kernel compile later it (squid + debian) in this case) all runs great.
