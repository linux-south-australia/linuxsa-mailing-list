Shane Warner wrote:
> I have tried an SMC card and a ReadyLINK card as well neither have
> worked.
> 
> This card has both PNP (jumperless) and Non-PNP (with jumpers)
> modes.  I have tried all the settings I can think of.
> 
> With this card in I get an "ie0: unknown card found at <...>" error
> followed by "ie0: not found at 0x300".

With the old cards, the jumpers normally enable setting of three things.
* IO port
* IRQ 
* Memory location

If your having trouble getting this card detected, make sure that all
shadowing of memory ranges is turned off in bios (normally under
advanced setup menu). Otherwise, bios might be hiding the card.
