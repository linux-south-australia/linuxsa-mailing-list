Re: disapearing disk space problem

Hi Horst,

One of the features of ext2 is to reserve an amount of space on the disk
for a particular user, usually root.  The reason for this is to make it
possible to maintian/update the disk easier then when the disk is full. 
If you have a full disk it is harder for it to find contiguous space
which leads to degraded performance. [Someone correct me if I am wrong
here :]

You can get the stats on this by running tune2fs:

# tune2fs -l /dev/hda1 | grep Reserved
tune2fs 1.19, 13-Jul-2000 for EXT2 FS 0.5b, 95/08/09
Reserved block count:     24899
Reserved blocks uid:      0 (user root)
Reserved blocks gid:      0 (group root)

If you really want you can change the number of reserved blocks, see man
tune2fs for that. :)
