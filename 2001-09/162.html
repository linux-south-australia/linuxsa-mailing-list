When the cable modem is connected to the cable network and the computer,
the machine issues a DHCP request and is assigned a valid IP.  The catch
is that you may not send packets out of the system until you have logged
on using their software.  So even if it is not logged in, the machine will
recieve packets via the cable, like a Code Red attack.

As its just a network interface, running it on UNIX is easy, all you do is
run dhclient (FreeBSD) or pump (Linux) on the NIC its plugged into and
then get a program called 'bplogin' to authenticate against Big Pond.
The program isn't supported by Telstra but was written by a former Big
Pond employee.
