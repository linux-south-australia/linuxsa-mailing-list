Hi.

And the lesson is: Don't connect a box to the network until you have
finished installation/configuration.

I've seen internet users hitting an anon FTP server within about a minute
of setting it up.

As for not installing IIS, be careful - he may have been installing
something that _requires_ IIS, and therefore did. Sounds like he has left
the dark side now anyway, but if he repeated the exact same install
(without the cable connection), it would be interesting to see if IIs was
in there anyway...
