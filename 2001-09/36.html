Re: BSD Telnet Shells

I believe someone mentioned using a firewall... well that is probably
the best way to do it.  In FreeBSD:

ipfw add allow tcp from mickasaur.extremews.com to any uid mick
ipfw add reject tcp from mickasaur.extremews.com to any

I'm not sure if there is a way to simply negate the uid.  Just change
mick to whatever username you use on the box
