RE: General install question

Before we had such luxuries of bootable cd-roms, users would have to make a
bootdisk or number of them to boot of first, which would then go to a cd-rom
or other source install.
Depending on the type of distro you have there, they should have this
facility on the CD.
This is typically in the dos-utils directory on the first CD, but can vary
in diff distro's.
Look for the executeable file called rawrite.exe, this is used to write a
disk image (.img file) to floppy disks.
