Re: Problems with Netgear PS105 Printserverbox and HP 4000T

You may need to use smbprint-ing.
from memory even though it uses TCP/IP for connectivity, it
still uses net-bios names

EG:  \\PSXXXX\P1

   \\  Server name \ Port-name

Look at the bottom of the little beastie for the PSXXXX name,
it broadcasts in the "blundows network neighborhood" quite well.
If you can see it there, you should be able to add an
smb printer.
