Hi James,

The configuration file is /etc/lilo.conf

KDE has a graphical application to edit this and I think Gnome does now to 
KDE's is in the menu under configuration|boot and init|boot manager(lilo) and 
is laid out fairly comprehensively.

Otherwise edit the /etc/lilo.conf file direct using your favourite editor 
and change the line that reads

default=linux                     

to read

default=windows  
           
where windows and linux in the lines above reflect the label for the 
particular image you want to boot  this can be checked by reading further 
down in the same file
