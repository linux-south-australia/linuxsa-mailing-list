RE: Auto start squid

Hi Ken,

If you install the linuxconf package off the RH7.1 CD you can then find a
section called "Control service Activity". (This is a really good tool and I
don't know why they removed it from the default install.)

Squid usually has an entry in there where you can set what runlevel you want
it to work on. Just set it off at runlevel 3, 4 and 5 and all should be good
on your next reboot.

I would have to double check this at home because I don't have my Linux
boxes at work. I am sure that if I am wrong I will be corrected very
quickly.

Cheers,
