For Budding Programmers and Developers

I thought this would be of interest:

* http://wolfpack.twu.net/ManEdit/

I haven't looked too closely at it but it's a GTK+/X GUI application
that allows you to make "man pages". Hence, if you can compile it (it
built seemlessly on my RedHat 7.1 distro, there's a Debian package and a
Linux Mandrak 8.0 package) and you can run X and Gnome/Gtk+ the excuse:

"I can't write in that funny macro language"

Cannot be used to avoid writing documentation.
