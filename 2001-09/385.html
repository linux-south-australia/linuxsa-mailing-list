Re: FreeBSD

Richard,

> does anyone have experience with Debian and FreeBSD? I'd be interested to
> know how the ports collection compares to apt-sources...

I am a debian user through and through, however i built a machine with FreeBSD 
4.3-RELEASE on recomendation from our network admin, I forced my self to forget 
that the install would be like debian, and did some reading before hand.

I was impressed by the FreeBSD installer, and the /stand/sysconfig script (?? 
been a while since i did it..?) but the most impressive thing i found was 
indeed the Ports Collection. While a little slower that apt-get install 
<package>, i feel it provides for much more security, along with compiling 
specifically for your architechture/machine. 

Think of this, what if someone broke into the debian servers - however 
unlikely - and replaced for example the bash package with a new bash package, 
trojan and all.
Ports gets past this.

Ports is an excellent tool, however there is a Debian/BSD project - that would 
be fun.
