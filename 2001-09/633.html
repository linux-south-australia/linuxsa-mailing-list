> > I am trying to hack up a perl script to change all the occurences of an IP
> > (192.168.101.2) to another IP (192.168.0.200) in a text file.
> >
> > Can someone provide a quick example of a script to just do this subsitution? I
> > am trying to learn Perl - I am in a bit of a hurry with this one otherwise i
> > would put it together myself.
>
> while ($line=<STDIN>) {
> 	$line =~ s/192\.168\.101\.2/192.168.0.200/g;
> 	print $line;
> }
>
> Will accept data from STDIN, and write the modified data to stdout.
>
> Hope it helps,
> Matt

If it didn't have to be perl, you could do it easier with sed off the
command line.

cat inputfile | sed s,192.168.101.2,192.168.0.200, > outputfile
