Red Hat installer, keyboard options

Q: I am upgrading Red Hat Linux in text mode using a floppy
disk and HTTP server from a machine with no function keys --
a standard ASCII keyboard.  Can I get access to the command
prompt?

Why?  Because if I can get access to the command prompt
I can alter the machine's CMOS via /dev/nvram.  Then
I can change the BIOS from booting first from floppy
to booting first from hard disk.

And if I can do that, then I can dial in to the serial console
of a remote machine and upgrade it entirely without anyone
at the remote site touching the machine.
