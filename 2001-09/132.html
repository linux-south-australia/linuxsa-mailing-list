Re: dbopen and RedHat 7.1

On Wed,  5 Sep 2001 12:43, Andrew Hill wrote:
> Where has the dbopen function gone to in RedHat 7.1? I assume that
> it's a C function that's in some file that can be installed, but is
> not in a file called 'dbopen*', as I'm not having much luck in
> finding it in the rpms on my CDs....

Did you try "man 3 dbopen"?

If you look at said manual page, you'll see that to use dbopen(), you 
need to do these inclusions:
       #include <sys/types.h>
       #include <limits.h>
       #include <db.h>

rpm -qf tells me that you'll need these packages:
* glibc-devel
* db3-devel

If you didn't have those packages already installed, then it may have 
been a little more difficult to find this out.  :-)


Hope this helps.
