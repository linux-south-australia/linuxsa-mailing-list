Re: Compact Flash readers and Linux

On Thu, 2001-09-27 at 09:28, phatboy@ihug.com.au wrote:

> Sorry, I didn't explain myself properly, as usual! I want to be 
> able to take a cf disk from my Nikon Coolpix digital camera and 
> load all of the images directly into my Linux machine. 

That's what this USB CF reader does.

> I know about the ide-cf adapters that you can get, and I know that 
> Linux will see it as an IDE device - but can Linux read the storage 
> format that digital cameras use?? Can't find any info in the 
> archives regarding this, and google hasn't been a lot of help to 
> date, so I'm interested in any *firsthand* experiences that LinuxSA 
> folks might have had doing this. Has anyone tried the Sandisk USB
> card readers under Linux? Their web page says that they're supported,
> but I don't know anyone who's tried one...

The SanDisk works quite well with 2.4.x. It even worked with late
versions of 2.2 that had the half-assed USB support.

Most digital cameras use a "standard" format that nearly any OS can
recognise with very basic software. I know the Kodiak camera's that I've
used store their images in JPEG format.

I use a SandDisk USB reader when I download pictures from a Kodiak
DC2-hundred-and-something.

Have a look through http://www.linux-usb.org/ for more information on
supported USB devices.

If you read the entry (on the Linux USB website) for the supported
SanDisk reader, you'll notice that it doesn't appear as an IDE device,
rather a SCSI device appropriate to your system configuration.
