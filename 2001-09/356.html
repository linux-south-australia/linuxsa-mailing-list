Re: Install problem - Not enough memory(?)

On Mon, 17 Sep 2001 08:37, Shane Warner wrote:
> I am looking to install RedHat 7.1 onto an old 486sx with 16 Mb RAM.
>
> However the installer sayd there is not enough system memory to
> install RedHat.

I suspect the graphical installer requires at least 32 MB physical 
memory (RAM), have you tried the text-mode installer (boot: text)?

You should be aware that most of the graphical software on a Red Hat 
7.1 system (such as KDE 2.1 and GNOME 1.2) requires large amounts of 
memory, and the Linux 2.4 kernel (as shipped with Red Hat 7.1) also 
seems to use a considerable amount of memory, so I would suggest using 
a smaller window manager such as BlackBox or WindowMaker, or perhaps 
even installing an older Linux distribution.
