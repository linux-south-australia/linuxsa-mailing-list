At 11:59 PM +0930 7/9/01, Ryan Verner [festy] wrote:
>Sure.
>
>You'll need to write something to get your IP, and send this as a command
>line parameter to the dynamic ip update app.   This should happen every time
>your interface comes up.
>
>Download the dynamic update util, and assuming you're using ppp, checkout
>your ppp up local scripts, probably in /etc/ppp, depending on your distro.
>
>One way you can do this is to ipconfig <device name>, use grep to filter out
>some keyword on the right line of this output (grep <keyword>), use awk to
>print say the second or third word on this line (awk '{ print $2 }'), and
>then sed to replace any un-needed text with a blank (sed -e
>s/before/after/g).  Use pipes between these commands (|).
>
>'man' is your friend :-)
