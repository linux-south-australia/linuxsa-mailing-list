Re: NIS && NIS+

I am using NIS on linux with the UNIX services on
Windows 2000.  This works pretty well once you get it
setup.  When a person changes their password on NT (we
are running a domain) it automatically changes it in
the NIS database.  The other site that I am consulting
for I have Solaris 8 running NIS+ and Unix services
running on NT4.0  This also works well.
