Re: Dynamic DNS

Mikey, 

> Hello Group. Is it possable to setup a Dynamic dns SERVER?? For example I
> have pheermy.homeip.net automatically updating everytime I logonto the
> internet.

You don't want a DNS server on your machine, you want an automatic
update client for the homeip service.  Check out
http://support.dyndns.org/dyndns/clients/.  I use ddclient from
http://freshmeat.net/projects/ddclient/

Add something like this to your ip-up script (you'll need to add a the
http proxy option if you have to use a http proxy):
   ddclient -retry -wildcard -ip $4 -retry 

> Can I add sub domains to this dynamic address?  

Depends on what homeip.net offer.  Checkout their documentation.  I
know you can do wildcards so that www.pheermy.homeip.net will be the
same as mail.pheermy.homeip.net or qwerty.pheermy.homeip.net.
