Re: RAM strangeness on Athlon box

Hi.

I had a similar problem with a PII system and 128mb. Apparently, if there
is any shortage of real memory compared to what you tell the kernel,
things get reather pear-shaped in a hurry. I backed off my mem= statement
by a few megs and all was sweet (actually still is). The fact that the
system works with mem=384M tends to support this suggestion, but YMMV.

Try 490M or something, <jealousy> you have an obscene amount of memory
anyway </jealousy>
