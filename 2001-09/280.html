That's great.

You should checkout:


* http://www.winehq.com/

(or is it http://www.winehq.org/)

Look for the documentation there.

Also, try looking for the same or similar documentation in

/usr/share/doc

OR

/usr/doc

[depending on what distribution you have]

When you've untarred it, look for files such as:

INSTALL
README

Or a variation thereof. They are normally text files so an editor will
be needed.

Try pico or vi or emacs from the commandline.

If you're running X then there are a number of graphical ones you can read
text files with.

I probably should mention - VERY NICELY - that you should do your
best to get it setup using this documentation that I've pointed you at.

If you get an error try to be as speicfic about the error as you can.

Hope this helps :-)
