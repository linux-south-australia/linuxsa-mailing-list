Re: AFS experiances?

AFS is a file sharing system kinda like NFS (but really more like Coda
and Intermezzo). It was developed by a university and IBM got hold of it
somehow. Now IBM has released the source code for all the
clients/servers/etc which makes this very interesting. It support the
normal features such as disconnection mode, caching reads, on mount
point (all the same as Coda). It also suppose to have stable WinNT and
Win98 clients and be used in lots of places. The source has been ported
to Linux by IBM/Others .
