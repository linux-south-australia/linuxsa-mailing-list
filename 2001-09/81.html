Doubtless most prospective buyers will already know this, but SGI 
packages the most commonly requested third-party freeware packages and 
makes them available for download here:
<URL:http://freeware.sgi.com/>;
or the local mirror:
<URL:http://sgifreeware.planetmirror.com/>;.

Naturally as the system is Unix, it is also possible to download and 
build the source for any GNU software.  If you are doing this, you may 
like to check whether you are getting a C compiler.  The SGI C compiler 
is called MIPSPro, however you should also be able to obtain gcc.
