Yes, Cygwin comes in handy under Windows when a Unix machine isn't 
available!

I remember reading somewhere that Cygwin works well under Windows 
NT/2000 (which is partially POSIX compliant), while Cygwin is not 
fully-functional under Windows 9x, so check that out before you decide 
to use it under 9x.

Among the many Cygwin ports projects is XFree86, which will be very 
interesting (perhaps a potential replacement for X-Win or VNC) when it 
becomes useable.  It seems to be lacking some useful features like "X 
-query" and had some problems with colors last time I tried it.
