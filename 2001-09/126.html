Re: File Handlers / RH7.1

On Wed,  5 Sep 2001 10:50, Mark Day wrote:
> Anyone know how many file handlers are compiled with a basic server
> install of RedHat 7.1?

If I remember correctly, this information is shown by reading the 
contents of /proc/sys/fs/file-max ("cat /proc/sys/fs/file-max").  Doing 
this on my system returns 32768.

> And also how to increase these handlers? They are complied into the
> kernel right?

It may be possible to tune this by simply changing the contents of this 
file (such as "echo 64000 > /proc/sys/fs/file-max").  Before doing 
this, it would be wise to check whether this number needs to be a 
multiple of a particular number.


Hope this helps.
