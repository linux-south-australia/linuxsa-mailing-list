telnet might be unsuccessfully trying to perform a name resolution on the
ip address you are using (assuming you are using the ip address not the
name). This (as you suggest above) does eventually time out, but life is
too short for timeout's. Try the -N flag to turn this off ie. 

telnet -N x.x.x.x
