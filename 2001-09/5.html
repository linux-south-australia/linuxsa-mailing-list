Hi Benjamin,

A good 'local' resource for Windows questions is the Australian Visual Basic 
Users Group. They have a tech support page that can be found at:
http://www.avbug.com.au/vbtech.htm

This site lists Windows related newsgroups and mailing lists, but I am not 
sure how current it is.

There is also a local branch of AVBUG that meets once a month at Ngapartji. 
Ironically the last meeting was about building a list server using Microsoft 
products. The local Microsoft agents attend these meetings, and they're 
pretty good with finding solutions to Windows related problems,. They don't 
like it if you yell out "Windows sucks, Linux rocks!"  ;)  

In a pinch you could email me, since Windows related stuff my day-job, but 
then I'll know you're scraping the bottom of the barrel  ;)
