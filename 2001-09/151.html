Hi,

I'm running a Mandrake 8 box (Default 2.4.3-20 kernel), hardware as
follows:

Cel 466 / 384Mb
Abit BE6-II mboard
NE2000 pci NIC
Ensoniq es1371
NVIDIA TNT2 M64

This box has been running fine until yesterday, it nows refuses to bring
up the eth0 interface (/var/log/boot.log):

Sep  3 17:16:32 lister network: Setting network parameters:  succeeded
Sep  3 17:16:33 lister network: Bringing up interface lo:  succeeded
Sep  3 17:16:33 lister network: Bringing up interface eth0:  succeeded

Sep  5 21:10:36 lister network: Setting network parameters:  succeeded
Sep  5 21:10:36 lister network: Bringing up interface lo:  succeeded
Sep  5 21:10:37 lister network: Bringing up interface eth0:  failed

As it shows it does not appear to be network settings issue (It's a very
basic desktop machine).

`modprobe ne2k-pci` returns this:

/lib/modules/2.4.3-20mdk/kernel/drivers/net/ne2k-pci.o.gz: init_module:
No such device

/lib/modules/2.4.3-20mdk/kernel/drivers/net/ne2k-pci.o.gz: insmod
/lib/modules/2.4.3-20mdk/kernel/drivers/net/ne2k-pci.o.gz failed
/lib/modules/2.4.3-20mdk/kernel/drivers/net/ne2k-pci.o.gz: insmod
ne2k-pci failed

except for full duplex/debugging etc. support.

Here is a copy of my modules.conf:

pre-install pcmcia_core CARDMGR_OPTS=-f /etc/rc.d/init.d/pcmcia start
