Re: ipchains redirect to another host?

Hi Andy,

this is a case where you really want to start using iptables.

ipchains was good for some of what it did, but unfortunately things like
masquerading didn't fit terribly well into the scheme of things, and port
forwarding with ipchains was just a total kludge!

It is possible to do port forwarding with ipchains, but there are various
catches and oddities about it.

Doing the same with iptables is much simpler and makes more sense. In fact,
the whole system makes more sense once you know it.

You will need to be using a 2.4.x kernel to use iptables though.. If you're
running an old linux distribution then there are quite a few things to
upgrade before you can install it.

If you decide to go with iptables, and get stuck, email me and i'll post
some examples. However, I suspect you'll be able to figure it out yourself,
since the man page is pretty clear.

-Toby
