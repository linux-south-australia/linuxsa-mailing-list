Re: Quiet PCs...

Hmm, this might sound pretty obvious, but when it comes to fan noise, you
can get power supplies which make hardly any / no noise.  Not sure what to
recommend though, some of the Aopen cases have very quiet fans.

Regarding cpu fan noise, its probably possible to buy a high end celeron,
and underclock it, hence no fan needed.  (Done this with amd k6-2's before.)

Fujitsu hard drives I've found to be _very_ quiet.

If you're interested, I know of a supply of a whole heap of old Compaq
machines (p200, 32mb ram, 2gig hd afaik) we're actually using for Linux
boxes as gateways (so I can vouch full support).  Inbuilt network card, tlan
driver.   They're so silent it doesn't even sound like they're switched on,
hence why we're using them.
