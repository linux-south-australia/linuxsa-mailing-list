Re: PCI USB cards in old box

The cheap version - ports on a backing plate work ok. I have one of these on 
an old A-Open board. It just plugged into jumpers on top of the board. I had 
to make changes in the bios - then it was detected ok. The uhci module seemed 
to recognize and use the ports ok. My usb modem is currently using this set 
up.
