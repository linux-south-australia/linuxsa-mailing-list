Re: Username/Password Import

On 02 Sep 2001 16:13:28 +0930, Patrick Browne wrote:

> Anyway now that I'm setting up various Linux boxes (mail & web serving etc)
> around the campus I was wondering is there a Linux (I'm running RH 7.1)
> utility and/or script that will import great long delimited lists to create
> heaps of new users on the fly?

While I'd highly encourage a non-Novell based network, when you're
working with 1000+ user networks, you'd probably want to centralise the
authentication.

Rather than having a seperate set of usernames/passwords for the Linux
machines, try using the PAM module that we've written (contact me
off-list for more info) which will authenticate users against your NDS
tree.

It means you've not got to worry about account management and such on
the Linux machines.
