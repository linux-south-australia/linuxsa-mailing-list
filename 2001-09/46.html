Sorry I did not get back to you earlier. I don't really get to use my email

Anyhow, just as a tip, get SAMBA working on the box. This would be good so
you can demonstrate that they could use the Linux box in the Windoze
network. They probably have a NT Domain. Samba can work beautifully on an NT
domain, I use it at home. Make sure that you have both the server and client
working so that the people can map to the workstation and map from the
workstation. 

As with the GHOST image, let me know what happens. Unless the boxes are all
the same I am wondering if you are going to have a problem. Never tried it.

Another thing is maybe stop a few useless services. Might make the box run a
little faster.
