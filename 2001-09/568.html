I suspect that the interface is proprietary.  

All the smarts is (was?) done by a program that runs under MS-DOS on 
the PC.  So, even if you did have the interface specs, you'd need to 
dig up the data sheet/programmer's guide for the chip that you wished 
to program so that you could write lots of code to perform the 
programming algorithm [and perhaps also the flash erase algorithm].  

I suppose you could try running JED's program under an emulator... 
but there might be timing problems.  

My information is quite old; you might find that JED have done 
more with the product in the last few years... ask them!  (and 
let us know of any updates!)
