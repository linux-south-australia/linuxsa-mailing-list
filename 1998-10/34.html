Re: Kodak digital camera's

Hi Paul,

Oliver Hartmanns DC120 web page at
http://home.t-online.de/home/oliver.hartmann/dc120kdc.htm has some
information on the kdc format, and some code for converting to JPG or BMP.
From the way the kdc format is described on this page, it sounds like it may
also be useful for you with the DC50.

This page is a mine of information about serial formats and so on for kodak
cameras in general.  I was able to get a web-cam going with my DC25 using
the information about serial protocols and data formates from this page.

Hope this helps,
Martin
