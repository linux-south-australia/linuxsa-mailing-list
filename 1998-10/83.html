RE: Installing via FTP

Hey.

I installed my Redhat 5.0 from my 98 box using FTP over a LAN. But this was
because the copy I had made of the RedHat CD had the directory structure was
all mixed up, so I had to check my ISPs FTP site to get the correct
directory structure, then arrange it like such on my hard drive.

As for the FTP server, I run a web server on my 98 box anyway, so I just
enabled the FTP option in that. My server is running Sambar Server 4.1. I
set up the anonymous users FTP root directory to the appropriate directory
on my hard drive. Then the RedHat installation had no trouble installing
from FTP.

In my installation, the directories were:

Anonymous root:   d:\www\redhat  (visible over FTP as  ftp://192.168.1.2/)

that directory is the equilavent of the root directory on the CD. Then the
other important directories are, I believe (And what they are visble as from
FTP):

d:\www\redhat\i386\RedHat\base    (ftp://192.168.1.2/i386/RedHat/base)
d:\www\redhat\i386\RedHat\RPMS   (ftp://192.168.1.2/i386/RedHat/RPMS)

Dunno if this helps any...  :)
