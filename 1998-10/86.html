RE: AGP support

My machine at work uses an AGP Cirrus Logic 5465 card.
It does not work if you use acceleration, making it very slow.

When running with acceleration the X server works for a
while (5-30sec) then when it has to update a line of text
(the clock on the status bar or an xterm prompt etc) it
prints random garbage instead of text and then locks
up the system.

I have spoken with others who have experienced this problem
with different cards and also according to the XFree86 docs
the chipset is support and tested which makes me feel it is
an AGP problem not a chipset problem.
