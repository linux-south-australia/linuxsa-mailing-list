Frontpage (ack-spit!) server extensions

Has anyone used Frontpage server extensions for Apache?  If so,
is there any simple way to get URLS of the form http://server/~user/
to work with the extensions?

The FrontPage "Publish" menu option seems to expect the server 
extensions to be available in whatever directory pages are published
to, meaning copies of (or symlinks to) the extensions need to be 
present in each user's public_html directory for ~user-style webroots
to work.  Although I expect ugly kludges from Micro$oft, it'd be 
nice if there were less-ugly alternatives...
