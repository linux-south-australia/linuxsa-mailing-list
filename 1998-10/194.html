	How do I turn off this annoying beep I get every time I press the
tab key and
	there is more then one file that matches my query?

man bash
	bell-style (audible)
	Controls  what  happens when readline wants to ring
	the terminal bell.  If set to none, readline  never
	rings the bell.  If set to visible, readline uses a
	visible bell if one is available.  If set to  audi-
	ble, readline attempts to ring the terminal's bell.

If you want to do it for a tcsh shell
man tcsh
	nobeep (+)
	If  set, beeping is completely disabled.  See also
	visiblebell.

	visiblebell (+)
	If set, a screen flash is  used  rather  than  the
	audible bell.  See also nobeep.

You can control the bell when you launch an xterm (assuming
you are using X).

man xterm...
	set-visual-bell(on/off/toggle)
	This action toggles the visualBell resource and is
	also invoked by the visualbell entry in vtMenu.

man nxterm
	-vb     This option indicates that a visual bell  is  pre-
	ferred  over  an  audible one.  Instead of ringing
	the  terminal  bell  whenever   a   Control-G   is
	received, the window will be flashed.

Anyway, all this is just straight out of the man pages.
Just do a search in each page for bell and like magic...
enlightenment can be yours!
