Or alternatively, dmesg can be used to view recent messages.  It can
also be run by non-root users.

	dmesg | mail

Is sometimes useful for mailing out to mail to someone helping you
with boot problems.  Note that it only prints recent messages, other
messages should be in /var/log/messages (but on some systems you need
to be root to view it.
