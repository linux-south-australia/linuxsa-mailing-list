> 
> tar cf tarfile.tar <list of files to include in archive here>
> Will take list of files and tar them to a file called tarfile.tar
> 
> Using your example,
> prompt% tar cf files.1998.tar files.1998*
> 
> should produce a file called files.1998.tar which contains
> all files in the current directory matching the wildcard.
> 
> The option c tells tar to create a new archive, and the
> option f tells tar to use the file who's name appears immediately
> after the option instead of outputting on stdout.
> 
> This is how I would use tar. Please correct me if this is not right.
