Greeting All.

Having installed Redhat 5.0 on my machine ( 686 150), I discovered that =
I could not coax my Legend 64 video card to display in 16 or 24 bit mode =
under XFree86.  I borrowed a Matrox millenium card which worked well =
except the X versions of programmes like mc and the print tool would =
crash the system. The only recovery was the reset button.  The matrox =
card also meant I could no longer use my resource hungry orchid SW32 =
sound card.  The programmes often ran without crashing the system if =
executed from the command line in a X teminal window. I have since =
borrowed a diamond stealth S3 card which works ( immediately fixed the =
crash problem) in 24 bit colour but it is little better in windows than =
the legend card.  The sound card seems not to like the presence of the =
S3 card either.
* Has anyone successfully run X with a legend card - I would like my =
orchid card back in the system. =20
* Does anyone know how to make the system behave with matrox card - if I =
must lose the orchid card, I would rather lose it to the matrox card, =
the S3 is definitely second best by comparison.

Thankyou

Ian Anderson
