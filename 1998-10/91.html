"InterBase 5 for Linux Now Shipping"

FYI, INPRISE (formerly Borland) tells me that:

"InterBase 5 for Linux Now Shipping

InterBase Software Corporation is now shipping
InterBase 5 for the Linux operating system (Red
Hat 5.1). InterBase is a relational SQL
database that supports business-critical
technology. InterBase 5 for Linux combines the
traditional strengths of InterBase: ease of
installation, use, and maintenance, with new
SQL and server benefits that give InterBase
true scalability, stability, and superior
concurrency. Some of the new features are the
InterClient JDBC driver, Unicode-FSS and
character sets for Chinese and Korean, standard
function library, SQL roles, and cascading
referential integrity. InterBase is designed to
be embedded into Value Added Reseller (VAR)
applications."
