Before you try any networking with the linux box check the current
firewall rules with: ipfwadm -Il
		     ipfwadm -Ol
		     ipfwadm -Fl
		     ipfwadm -Ml
you need to run each on seperatly as ipfwadm will not allow you to
check more the one stream at a time. Next if the defaults are deny
change them to accpet (if you want to set up a firewall then let your
rule script change this) with:  ipfwadm -Ip accept
				ipfwadm -Op accept
				ipfwadm -Fp accept
				ipfwadm -Mp accept
this means it will accept all connections. then you need to check that
the netcard is actually there, just run ifconfig and see that eth0 is
there, if not the you will need to manually set it up. for this
example I will use 192.168.105.76 as the machines ip and 192.168.105.0
as the network:

ifconfig eth0 192.168.105.76 braodcast 192.168.105.255 netmask 255.255.255.0 up
route add -net 192.168.105.0

As this box will be the gateway it doesn't need a default gateway yet,
that will happen when you get ppp working.
