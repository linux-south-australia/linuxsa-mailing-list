Re: How do I dial up my service provider as user?

Hi Brenton,


>I am able to dial up my service provider as root using the networking
>window ppp activation (X Windows-Redhat 5.1) but when I go user the
>networking window does not automatically open and when I try to open it
>manually it will not open. Does this have something to do with root
>privileges and file permissions? If so where and what parameters do I
>need to alter and what command, if any, do I then need to use to dial up
>as user?.

You don't. The ppp daemon (pppd) on most systems won't let you use it as
anything but root. I suggest typing "ifup ppp0" while at a command line
interface (such as a shell window in x) as root (su to root when using x as
a user). Type "ifdown ppp0" to shut it down.

Michael.
