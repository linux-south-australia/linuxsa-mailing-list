RE: Database linking Webpages

Dump the access database to ascii, and import into PostgreSQL or mySQL
or something.

Then you have a world of choices to do fancy things on the linux end.

- Use PHP to do an ASP type thing
- Write perl cgi scripts
- Write C .cgi programs to query the data and generate the 'dynamic'
HTML

The performance of something like postgreSQL or mySQL compared to Access
is much, much better.
