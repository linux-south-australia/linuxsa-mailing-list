What's wrong with my network configuration.?

Hi,

I have been persevering for some considerable time now trying to setup
internet services with Redhat 5.1 with only partial success. The problem
is that I am able to send and receive mail and to access newsgroups but
otherwise unable to see past my service provider's home page
(camtech.net.au).

I have tried diligently to do my homework before seeking help to the
extent that in addition to Redhat specific documentation I have read all
the Linux networking How-To's. However, I have found that to the
uninitiated new Linux user the volume of information not to mention the

standalone Intel PC internet connection.

My service provider's help desk, who does not officially provide
technical support for the Linux O/S, advised me to edit auto

version under Win95. If  the proxy is the answer to my problem I suspect
it has to be configured in a specific Linux network file somewhere.?

that comes with X Windows.I log into my service provider as root.

My current configuration settings are as follows;

domain=camtech.net.au
nameserver=203.5.73.2
          =203.5.73.1
host IP=127.0.0.1
host name=localhost
interface=ppp0
nickname=localhost.local.domain
routing=no entries

process.
