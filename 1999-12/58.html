Re: ISP connection

On Thu, Dec 02, 1999 at 04:30:00PM +0100, ALLEN, David wrote:

 > I can get a ppp conection to my ISP - virgin.net - but when I fire up
 > Netecape I 
 > get the error "Problem with your name server, you may need to set $SOCKS_NS
 > ...."

That probably indicates that your resolv.conf file isn't set up correctly.
The info you received from your ISP when you signed up should have mentioned
IP addresses for one or more nameservers (nslookup tells me that the 
primary DNS server for virgin.net is 194.168.4.220, but that might not
be the one they want you to use...)

You'll need to edit /etc/resolv.conf on your Linux box and insert the
following:

    nameserver 194.168.4.220
    domain virgin.net

(change the IP address of the nameserver to match the one you're supposed
to use).

If your ISP has provided more than one nameserver IP address, list them
all on separate "nameserver" lines.

That should be all there is to it.
