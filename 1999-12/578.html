>Hi,

>answer.
>
>  Can SAMBA password authenticate again with my 
>Window NT server password ? 

Yes, it can ...

   security = server
   password server = <ntserver NetBIOS name>

is one of the simplest ways to do it.

You could also use:

   security = domain

but then you have to make your Samba server a member of the NT domain which
is more effort.
