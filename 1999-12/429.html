It depends on what you want to run and it which point you want it run. If you
want it after you log in, try the ~/.bashrc or ~/.profile (depending on what
shell you are running - check you local man page for details), if you want to
run stuff when you graphically log in or when you start your favourite Window
Manager, try the Window Manager's documentation / Config tool (gnome has an
option in it's config tool that allows this - you can also just get it to
save settings when you exit). The above are user specific.

If you want to add something to a specific runlevel, have a look at the shell
scripts and links in /etc/rc.d/init.d/* and /etc/rc.d/rc?.d/* the rc?.d/*
scripts are links to the init.d/* scripts. The ones that start with a 'S' are
run when system boots, the ones with 'K' are when system halts / reboots.
Reading these scripts is a good way of finding out how to make them work.
This is a system-wide thing (non-user specific)

Also try linuxconf. It's RedHat specific, but very good and I think there is
something in there which will do what you want (then again I could be
mistaken). Also try webmin (http://www.webmin.com/). It's very good.
