Re: Having a File System problem....

Disable DMA.  DMA works 95% oif the time, but on my old drive/PC
combination, it did things like that. hdparm has options to turn it on
or off from the command line, but you really want to have it disabled
from startup, which means you need to add something to your lilo config.
Check the hd doco in /usr/src/linux/Documentation anyway for more info.
If you recompile the kernel you can turn off the "Enable DMA by default"
option, but that's overkill for you.
