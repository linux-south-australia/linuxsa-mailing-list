Adaptec 1740 - Can't install RH

Hey All,

I've been given the task of setting up a mySql server, on an Acer EISA
server, which has an Adaptec 1740 EISA SCSI card in it. 

When the machine boots up, it shows: (note, '>' denotes screen output)

>Adaptec AHA-1740 BIOS v1.40
>Copyright 1992 Adaptec, Inc.
>[Enhanced Mode]
>
>Target 0 - SEAGATE ST11200N - Drive C: (80h)
>Target 4 - NEC CD-ROM DRIVE:841

When the linux kernel boots, it comes up with:

>scsi : 0 hosts.
>scsi : detected total.

Sometimes, when it does an autoprobe, it dies and says:

>Kernel panic: aha1740.c: Irq from unknown host!
>In interupt handler - not syncing

I know all the termination and everything is set, as I had to fix it

CDROM and stuff like that until I set the termination properly. Now the
BIOS finds all the drives 100% of the time.

More info about the computer is:

Acer EISA 486/50 (1 CPU installed)
64Mb RAM
1Gb SCSI HDD

Netgear EA201 (Ne2K compat.) NIC
