You shouldn't need much. If you want to have access to NT as a file server,
include smbfs. I'm not sure how you'd go accessing the scanner remotely (in
fact I'm not sure how you'd do that under just Windows). I have got my burner
so that I can read over the network, but haven't tried to write (I only use
Linux for burning - so NT doesn't have software installed).

You don't need Special software for NT to get standard file and print (and
authentication) services (Linux -> NT of NT -> Linux).

Oh yeah.... you'd probably want the correct drivers in your linux kernel to
allow network connectivity ;-)
