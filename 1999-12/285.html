Have these lines in your smb.conf

Unix password sync = yes
passwd chat = [crud]
passwd program = /bin/passwd
encrypt passwords = yes
smbpasswd file = /etc/smbpasswd (the file created by pwdump)

the smb.conf manual page will help a lot. Also try reading the NT Domain
documentation / Encrypted password stuff in /usr/doc/samba-2.0.5a.

I have found the NT-DOM Samba mailing list more than helpful as well.

Good luck.
