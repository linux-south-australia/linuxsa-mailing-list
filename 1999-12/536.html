Re: internet connection as user

On Mon, 27 Dec 1999, Ashley Ruehland wrote:

> I am having some trouble getting a dial up connection to the internet as
> a user.
> 
> I can set one up as root and it is fine but when I did the same thing as
> a user I get a message about not being able to open the lock file.
> 
> I have changed the lock file permissions and also as root removed "lock"
> from out of that file.

Try giving everyone permission to create the lock files:
As root do:
  chmod a+rw /var/lock
  rm /var/local/*modem*

then try and dial as a user again
