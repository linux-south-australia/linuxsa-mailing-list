RE: U Settings up RedHat 6.1 box as mail server for Windoze PC's

> Hi Kristian,
> 
> I have done this at two sites now and it works extreamly well.
> However I did find it very tricky to setup the first one but all as
> been working without problems for about 6 months now.
> 
> Basically on the RedHat box you will need to run the following:
> 
> 	sendmail				# to deliver local mail
> and outgoing internet mail to your ISP
> 	fetchmail				# to receive internet
> mail from your ISP
> 	diald					# to perform diald on
> demand
> 	openldap				# if you want to have a
> local email address directory
> 	apache or squid			# if you also want to run web
> and ftp proxy to the Windows PC's
> 
> On the Windows PC's I use Microsoft Outlook Express 5, using IMAP to
> connect to the Linux mailboxes rather than POP as this allows the mail
> to remain on the server and also allows people to move from PC to PC
> and still access their mail.  Outlook Express 5 also supports LDAP
> which allows local addresses to be verified using openldap on the
> Linux box.
> 
> The main difficultly in setting up the system is configuring sendmail
> and fetchmail.  Diald can also be a bit of a pain to get working with
> sendmail and fetchmail because of timeout problems with dial on
> demand, but I wrote a couple of shell scripts that are run by cron
> that resolved this.
> 
> I hope this may be of some help.  Let me know if you need further
> help.
> 
> Regards,
> Steve Tee.
