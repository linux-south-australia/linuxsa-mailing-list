A guess is that the insmod program isn't installed, or is
installed and can't be found (for example, the directory
it lives in isn't in the PATH, or the file isn't marked
executable).

If you are running Red Hat Linux and the insmod program is
missing, then you might need to install the RPM file.

$ uname -a
Linux teatree 2.2.12-20 #1 Mon Sep 27 10:40:35 EDT 1999 i686 unknown
$ cat /etc/redhat-release 
Red Hat Linux release 6.1 (Cartman)
$ which insmod
/sbin/insmod
$ rpm --query --file /sbin/insmod
modutils-2.1.121-14
$ ls -l /sbin/insmod
-rwxr-xr-x  root  root  /sbin/insmod
