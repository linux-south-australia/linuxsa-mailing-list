To do this, you will need to find a CDROM that allows you
to set the Block Size to 512bytes instead of the now standard
2048bytes.

The Older sparc's will only boto from a 512byte block CDROM.

I have a Sparc SLC, with 16mb at home running OpenBSD 2.5 quite nicely.
and I used to have a Sparc Station 5 with 64M running SparcLinux (Redhet 4.2)
it was _very_ much faster than running Solaris on the box.

As for the booting from tape, you've obviously figured out that you can
use "stop a" to get a prom prompt on boot, and then type 'boot tape'
I'm unsure how to get into the install from there, normally it should
proceed into the setup or at least run a script and ask questions.

Although I've never had to install SunOS from tape. Which I assume is
what you have there..

Personally, I'd recommend saving yourself the headaches, and get SparcLinux or OpenBSD/NetBSD
whichever is your fancy. It'll be more fun :)
