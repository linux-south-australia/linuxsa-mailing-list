Linux C++ Mailing Lists [ANNOUNCE]

LINUX C++ MAILING LISTS
-----------------------

In an unprecedented week, the Linux C++ mailing lists, created by David
Lloyd (mailto:tuxCPProgramming-owner@listbot.com), have grown from 390
members to 510. The list's subject is programming Linux with C++ and no
subject - easy or hard - appears to be too difficult for it. Before
these lists existed there appeared to be no mailing lists about this
subject as successful as this one.

To subscribe to the lists send an empty email to:

  tuxCPProgramming-subscribe@listbot.com

I hope that you will find the lists a useful and FREE linux resource.
