Re: Command for mounting windowsNT4sp4 shares? (using Samba)

here's one that works for me

smbmount //ntserver01/logs password -I 10.1.1.1 -c 'mount /mnt/smb' -U
jake -n linux01

-n = use this netbios name
-U = use this username, eg jake
password = jake's password
-I = use this IP address
-c 'mount /mnt/smb' = execute this command, ie., mount the thing here.
