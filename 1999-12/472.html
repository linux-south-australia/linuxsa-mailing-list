LaTeX - section numbering problems

Hello everyone!

Anyone here experienced in the use of TeX?

Consider the following example:

- 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 

\documentclass{report}
\begin{document}

\title{This is the title}
\author{This is the author}
\maketitle

\tableofcontents

\part {Introduction}
Blah blah blah

\part {Considerations}
	
	\section {Hardware}
	
		\subsection {ARLAN Microwave Bridges}

		\subsection {Network Switches}

	\section {Software}
	
		\subsection {Fujitsu Application}

- 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 8< - 

and so on.

I 'compile' this document by running "latex docname.tex" two or three
times.  It outputs "docname.dvi" and I'm happy.  It looks GORGEOUS.

-BUT-

This is what happens with the section numbering:

==
Contents

I Introduction                            x

II Considerations                         x
  0.1   Hardware                          x
        0.1.1  ARLAN Microwave Bridges    x
        0.1.2  ARLAN Microwave Bridges    x
  0.2   Software                          x
        0.1.1  Fujitsu Application        x
==

Now that's kinda annoying - I don't want the 0.1, 0.2, I want 1 and 2.

Can anyone tell me what I've done wrong?  I think I've exhausted the info
document on LaTeX....

Or - does anyone have any good LaTeX FAQ's that actually address this?  I
would think that someone else must have come up with the problem, but the
FAQs I searched didn't even mention section numbering problems :(

Out of interest - when I add an \appendix command at the end and add a
few new sections, the numbering restarts (obviously), BUT the "0" is
dropped.

ie - 

.1  Section 1                x
    .1.1  Subsection 1       x
    .1.2  Subsection 2       x
.2  Section 2                x
    .2.1  Subsection 1       x
    .2.2  Subsection 2       x

etc.

Argh!

Btw - once I've got this problem sorted out, TeX is going to become my
only typesetting program..  Word can finally get the kick :)  Hoorah :)

And luckily there are enough free DVI viewing programs to make this
relatively safe :)

Is there such a thing as a DVI-to-Word.DOC converter?
