Quickly though -

Do you recognise this?

-rwxrwxrwx         - it appears when you do an `ls -l`.

- r w x  r w x  r w x
- 4 2 1  4 2 1  4 2 1
- [ U ]  [ G ]  [ O ]

U = User, G = Group, O = Other

The first column?  It defines systemy things that I don't know enough about
to write about.

Basically - add up the numbers of the relevant permissions and use them in
the chmod command.  eg -

to set -rwx------ (ie user can read, write, execute, but nobody else can..)
use `chmod 700`

to set -rwxrwx--- (ie user and users in the file's group can r,w,x, but
others cannot..) use `chmod 770`.

to set -rwxr-xr-x (ie everyone can read and execute, but only owner can
modify) use `chmod 766`.

That will start you off.  Want to know more?  (Starship Troopers...  gotta
buy that DVD..)  Read the man page etc.
