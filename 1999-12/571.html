Permanent Internet connections

After reading what you guys wrote about permanent connections, I 
thought I'd ask my ISP (BOLDWeb - bold.net.au) what, if anything, they 
could come up with that would suit me. The following two options are 
what they came up with:

1)     Line rental: $25/month + $40/month + 22c/Meg
       "total cost": $65.00 plus data.
       $100 = 159 Meg/month (5.22 Meg/day average)

And the option I've chosen:

2)     Line rental: $25/month + $60/month (to 150 Meg) + 22c/Meg
       "total cost": $85.00 (to 150Meg) + data.
       $100 = 218.2 Meg/month (7.17 Meg/day average)

Benefit: A permamnet (live) IP number.

Limitation: These deals are only available for PSDN connections.

Requirement 1: Two weeks (maximum) to set things up (lines are only 
being
               connected as required). 

Requirement 2: These accounts need to be held for a year (to allow for 
cost
               recovery). 

Well, that's it. If anyone is interested (or wants further 
information), just send some email to support@bold.net.au .
