Re: NIC help

On Wed, 29 Dec 1999, Bryan Wetton wrote:
> RH 6.1
> 
> I had to take a PCI NIC from my Linux box the other day and have now
> replaced it with an NE2000 ISA card.
> This is not detected - the message I get is 'ne2k-pci-c - No usable cards
> found!'
> How do I add a driver for the ISA card and set it up as eth0?

remove the ne2k pci statement from /etc/modules.conf (conf.modules) it is for
pci versions only

add

alias eth0 ne
options ne <ioaddress> <irq>

you *must* run ISAPNP first before attempting this.

in case you didn't read above

you *must* run ISAPNP before even attempting to install an isa card of any
description. sound, nic, or otherwise.
