I just tried to do exactly this on a machine this evening! I'm turning
an ex linux server into a windows machine for my brother. I booted
Win98, ran fdisk, deleted the hda1 and tried to delete the extended
partition, but it said it couldn't "delete partition while logical
drives exist", checked, and it couldn't see anything in the extended
partition.

Solution? I pressed the reset button before it wrote the partition table
to the disk, booted linux into single user mode, and then ran linux's
fdisk to delete the partitions.
