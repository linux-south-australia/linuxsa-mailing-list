Re: Filename star

To handle files with metachars, single-quote the filenames, ie. touch '*' will 
create a file with an asterisk as filename. To remove this file, use rm '*'.

The following command will find all files containing an asterisk in its 
filename:

find / -name '*\**' -ls

And this will search for question marks:

find / -name '*\?*' -ls
