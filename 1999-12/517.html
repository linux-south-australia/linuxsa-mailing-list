The PPP-HOWTO says how to do it, in the last chapter: 27 Using PPP across a
null modem (direct serial) connection.

I've used it several times to quickly get a connection between my laptop and
desktop, because my PCMCIA network card died a while ago :(

The good thing about the howto is that it has example command lines, so if
you know where to look, you can copy and paste the command lines you need,
without reading (or even understanding) what it does ;)
