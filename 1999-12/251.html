Re: Connecting MySQL.....?

ok - you need more info here.

are you using an ODBC connector from windoes Access? in this case,
check:
/etc/hosts.allow and /etc/hosts.deny

and make sure you windows box can "see" your linux box.

then read up on ODBC. i think you'll find that MySQL doesn't come with
an ODBC driver by default. You'll have to download and install one.

Here is what I think is happening now:
you Windows Access program is trying to connect to the loinux box on
port XYZ, but inetd is not configured for port XYZ and there is no stand
alone program listening on XYZ and thats why you get the "connection
refused"


conclusion: go to the MySQL website and read up pon ODBC connectors and
download the ODBC program.
