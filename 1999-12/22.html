On Wed, 1 Dec 1999, Nick Morrison wrote:
> -rwxrwxrwx         - it appears when you do an `ls -l`.
> The first column?  It defines systemy things that I don't know enough
> about to write about.

It indicates the type of file.  A dash is a normal file; a 'd' is a
directory; an 'l' is a symbolic link; and a 'p' is a named pipe.
