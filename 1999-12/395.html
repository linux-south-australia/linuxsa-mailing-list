Re: New Linux Mandrake Setup

If you are after a small distibution for the 486, don't go with 2.2
Kernel-based dists. Try something like RedHat 4.2. It's "old" and not
completely comaptible with newer systems, but you can still do stuff. Also
Caldera OpenLinux is quite good (the older versions). If you get "Linux for
Dummies" I think it comes with Caldera 1.3 and KDE plus an old Star Office.
This should be quite adequate.

Running X is a bit crap with 4Mb RAM (I hate HAVING to upgrade hardware, but
it was necessary), you will find it quite slow and running applications will
suck.
