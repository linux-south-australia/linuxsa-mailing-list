I'd start by looking at your login files (and system log in files) and see
if there are any strange programs being run. I'd also check the version
date of the login programs, and telnetd of various boxen. I'd also check
the terminal emulation, those with much more Unix experience then me would
probably know better, but I think you can copy all tty data to printer.

Maybe try installing and running ssh on all boxxes, then everything thould
be encrypted.

Are you going from a PC to the SunBox ? If so check the client
configuration on it, it is possible to set some clients to send every
keystroke to the printer.
