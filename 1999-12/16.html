chmod has a symbolic mode for ages (well, as long as I can remember,
which dates well back to last week ...).

You want to set group read/write permissions (leaving any other permissions
for group intact)?

    chmod g+rw filename

Remove execute permission for others?

    chmod o-x filename

Set user (owner) to only write to a file, stripping off other permissions?

    chmod u=w filename

Upon reading my RedHat 6.0 manpage for chmod, this is laid out in the
first page.  Leave octal to ex-assembler weenies like myself.
