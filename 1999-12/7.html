Apache Mailing List [ANNOUNCE]

APACHE MAILING LIST
-------------------

Chuck Mead has created a mailing list specifically to help you with the
web's most popular web server. Currently in its infancy there are
already a number of "experts" able to help and give advice on this web
server to many varied questions. The lists are hosted by Linux's most
favourite list manager - majordomo - so the subscription process goes
like this:

1) Send an email to :-
  mailto:apache-request@moongroup.com

Make sure you have no footer/signature or subject. Add a single line to
the body:

  subscribe apache

2) Majordomo will then process your request and send you back any
result(s); follow its instructions

Chuck hopes this will become a valuable resource for all of our
communities.
