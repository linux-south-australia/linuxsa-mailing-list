Re: X :1 -query 'pluto'

> When I want to run a second X-windows session from a second console <ie
> Control-Alt F2>
> I run as root:
> 
> X :1 -query 'pluto'

I assume 'pluto' is the name of your machine?

If so, you need to enable the gdm (the gnome equivalent of xdm).

To do this you need to modify /etc/X11/gdm/gdm.conf.  In the [xdmcp] section
there is 'Enable=1', that should be setup by default to be 'Enable=0'.

Incidently, if this is a normal occurance, you can get gdm to handle your
X servers on both displays.  Same file, down the end [servers] section.
You might need to add a full '/usr/bin/X11/X :1' or it might handle it nicely.
