Re: JAVA SERVLETS

Hi David
Apache has a module for Java Applets called Jserv
  see http://java.apache.org/
I've been running it in production since 1997. It works a treat.
All the servers I run it on happen to be Linux boxes, but it should
should be OK on any system capable of running Apache & a Java Virtual
Machine.
It has been a bit fiddly to install, but I they say the latest version
has addressed this.
