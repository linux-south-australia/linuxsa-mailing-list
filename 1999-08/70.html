In answer to your question: after installing SuSe I changed only the setup
value for the Accton ISA card which had a default value of  0x300 and which
produced startup error messages. I changed this to io=0x240 irq9 in line with
my Windows working setup. No other aspects of the proposed Linux network have
been configured as I am unsure how to proceed.

For the record cat /proc/ioports shows the NE compatible Accton at 0240-025f
lying between ide0 (01f0-01f7) and serial(auto) (02f8-02ff). I assume it is
not intruding into these addresses as it seems to have been probed OK on
startup.
