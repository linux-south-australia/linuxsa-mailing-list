Re: postgresql-data

I've while downloading 'rpm' can be easy, I've found that
doing the '.tar.gz' and './configure; make; make install'
very useful in learning how postgres works...
the INSTALL file (although long) does give a
good set by step guide.. and...

1) you get to choose where you put everything...
      /usr/local/postresql-6.4.2
2)where you data goes
      /erwin/pgsql
3)you get to run a test/regression suite
    just to make sure everything works
4) ... and ...
     by adding -lcrypt to a particular make file,
     (after doing ./configure, before make)

    file: src/interfaces/pgtclsh/Makefile
    change line : SHLIB_LINK= -L../libpq -lpq -lcrypt

    you can run 'pgaccess'.. a cool tcl/tk script
    which will give you a nicer interface than
    'psql'.

   This is broken in the RH distribution.
