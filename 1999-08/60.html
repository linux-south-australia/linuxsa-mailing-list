Hi.

Well, the best way to install it is to run the StarOffice setup program with
the '/net' option...ie './setup /net'...you must do this as ROOT ! Install
this somewhere common such as /opt/Office51 or /usr/local/Office51.

Then, once that's done...each user (including root) who wants to use
StarOffice must run setup from the StarOffice bin directory...ie
'/opt/Office51/bin/setup'. This will copy only the relevant config files to
an Office51 directory within that user's home directory....about 2MB.

Then the user just has to run $HOME/Office51/bin/soffice !

All done.
