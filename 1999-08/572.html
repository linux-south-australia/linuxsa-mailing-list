Linux C Programming Lists

There are currently two linux c programming lists. I have created one
using the free services of ListBot. Litbost *may* append advertisements
below all messages and all messages must be below 100kB (estimated 15
000 words). ListBot does not resell emails. You can join this list by
sending this email:

mailto:linux_cprogramming@listbot.com

I have developed a basic home page at:

http://users.senet.com.au/lloy0076/linux_c_programming/index.html

This list should be archived. I have tested it myself and the resend
speed appears to be acceptable. 

****
Chuck Mead who hosts the redhat-list has also set up a list using
majordomo. In a previous email he has said that one should send this to:

"Subscriptions are available by sending the words "subscribe c" in the
body of a
message to c-request@lunar-penguin.com"

Unfortunately this list is unarchived. 

****
I'm not certain how we should deal with the two lists. From zero lists,
there are now two. I would be inclined to use the ListBot list because
one is able to access it's archives at:

http://www.listbot.com/archive/linuxCprogramming/

****
Either way I think that these lists will prove useful.
