If I just had one more bit of info, I think
I could advance to the third nirvanic stage of
buddhic enlightenment.

Since the disk format programs in linux always seem to
offer me to start at cylinder 1, and the LILO boot sector
is in cylinder 0, apparently, and there are
separate boot sectors for hda (MBR), hda1, hda2...
(primary partitions), does this mean that only one sector
of cylinder is in use?

Cylinder 0:	sector 0 is MBR, rest unused.
Cylinder 1:	sector 0 is hda1 boot sector,
		the rest used for real data.
....
First cylinder of
partition hda2:	first sector is hda2 boot sector,
		the rest used for data.

If this picture is correct, then one could write
a nifty new improved LILO system to make use of
those empty unused sectors in cylinder 0.

That woudl result in a luxurious amount of space to
store the secondary boot loader, and
linux installation could rise to a new level!
