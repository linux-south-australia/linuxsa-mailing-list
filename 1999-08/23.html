Re: What file system do I mount a RAMDISK as?

On Sun, 1 Aug 1999, Jarrad S wrote:

> I have noticed while recompiling my kernel that I can mount memory as a
> ramdisk.
> 
> I hav tried but I do not know what filesystem type to specify in 'mount'.
> 
> does anyone do this and how do I do it!

You can mount it as any filesystem you like.
The easiest way to use it is:


dd if=/dev/zero of=/dev/ram count=1M          (Null the memory)
(change the count to the size yuo want)

mke2fs /dev/ram      (for an ext2 filesystem, but you could use any other)

mount -t ext2 /dev/ram /mnt/somemountpoint
