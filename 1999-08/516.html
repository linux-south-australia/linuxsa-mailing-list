Re: MOUNTING MUSICAL CD'S

You don't need to mount the CD to play it.
Mounting data CDs on a FILEsystem is so you can access the files
contained on it.
An audio CD does not contain files, thus it has no filesystem.

There are 'psuedo-filesystems' available that you can use to allow you
to copy the tracks off as raw data, but they aren't available in the
standard kernel. It's a far better idea to use cdda2wav or cd-paranoia
if you want to rip the audio to an audio file on your harddrive.

Note that the default windows cd filesystem that you see isn't a true
system - you can't actually do much with the files except click them to
play tracks.
