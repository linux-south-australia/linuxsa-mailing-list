Robyn,
	If you don't want to go to redhat 6 I suggest you ignore upgrading
to glibc 6 (2.1.1). Since it is the main library that Linux uses it breaks
many, many things. 

If you want to run wine (or any application) but can't due to dependancy
problems such as above. Then download the .src.rpm file instead.
Once you have it type:

rpm --rebuild wine-[somenumbers].src.rpm

This will compile it for you.

Once finished have a look in: /usr/src/redhat/i386/ 
and you should find the version you can install.
