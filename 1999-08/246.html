Re: Finding Duplicate Records

Nick,
In the abscence of knowing if there is such a tool to do this
specifically, I'd do a sort (see `man sort`), after which it is trivial
to make something (sed, or perl) run thru the script comparing lines.
Eg.
$lastline=<STDIN>;
print $lastline;
while (<STDIN>) {
   if (!/^$lastline$/) {
	next;
   }
    print;
}
