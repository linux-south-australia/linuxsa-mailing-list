LinuxSA Digest List

Hi all,

A while ago some people asked about having a digest version of the
LinuxSA list.  I've had a go at implementing it, so I'm now looking
for some guinea pigs :-).

Subscribing/unsubscribing to/from the digest is the same as for the
main list (ie. send a message with a subject of
subscribe/unsubscribe), but use the address:

linuxsa-digest-request@linuxsa.org.au

instead of

linuxsa-request@linuxsa.org.au

The posting address (linuxsa@linuxsa.org.au) has not changed, and
should work whether you are on the main or the digest list.

In summary:
- if you're happy with things how they are, then you don't need to do
  anything.
- if the list volume is too heavy for you (we averaged ~27.5
  emails/day last month) then you can unsubscribe from the main list
  and subscribe to the digest (but maybe you should wait a few days
  until the brave people have helped me iron out the bugs :-).

I've set the digest to go out when there's >16K worth of messages or
it has been >1 day since the last digest, but that may change to be
less frequent once I'm sure things are working.

Let me know (by private email) if you have any troubles with the
digest.
