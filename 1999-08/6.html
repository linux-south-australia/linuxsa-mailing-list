RE: vmware

Thanks for you replies to my VMware questions I hadn't realised that there
was a demo available, 
Just a summary of my installation for those interested. 
The software can be downloaded at
http://www.vmware.com/download/download.html you are then required to
register then a return email is posted with a licence file that you place
into the directory from which the vmware binary is run. You then start
vmware and create a config file including all your hardware choices
and you can then start the VM and use a dos boot disk in the VM window to
fdisk and format your partition
Which seemed fairly idiot proof as I still have all my existing partitions
phew!
Then you go ahead and install your OS, in my case WIN95, after that I
installed the VMware tools which provided a vmware display driver, It was
then running like a dream I haven't installed any other programs to this
partition as yet, but I am already happy with the results. It almost
seemlessly intergrates into my Gnome/Enlightenment desktop so much so that I
can move my mouse over the edge of the VM window from Gnome to the VM and
hardly notice any difference I can then execute any icon in Windows..

A couple of bad points were; I couldn't work out how to use an already
created Fat16 partition and I also had trouble with the licence file, it had
expired before the first time I used it. I think that this is because the
difference in US mm/dd/yyy format and ours, so 28/7/1999 was read as
7/28/1999 I logged this error with vmware and they soon replied with this.
