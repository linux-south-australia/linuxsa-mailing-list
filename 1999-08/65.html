Re: Linux as an internet gateway

Hi... I run a system similar to that which you mentioned.... at home I
have a Linux Box as my permanent dialup machine  & then have 5 other
pc's in my house/radio shack on the LAN , all of which access the
internet via the dialup (behind a firewall)... the other boxes are 1
windoze95, 1 triple boot(win95/linux/freebsd, 1 dual boot(win/linux) & 2
linux only boxes

I also run a siomilar system at my cafe - 1 linux dialup/router with 2
other boxes (1 FreeBSD/WIN95 & 1 LINUX/WIN95) on the LAN.

If you have any probs/questions give me a yell...BUT try to do it on
your own first, cos if you ask me dumb questions I'll probably tell you
to RTFM! (BTW... a dumb question is one which indicates to me that you
havent tried to set it up & havent even opened the "M" part of RTFM!) If
you want some hints..:

1. Set up all the windoze boxes with 192.168.1.x addresses on the
network cards
2. Set the gateway address on the window boxes to the IP number of your
dialup linux box.
3. Run ipfwadm on the linux box   (deny ALL first , then accept windoze
boxes ;))

Cheers, Phil
