Re: Listing all packages installed.

On Fri, 27 Aug 1999, Paul Cleland wrote:

> 
> Hello everybody, im normally a debian man and I am used to typing dpkg
> -l to get a list of all packages installed..... I had to setup a redhat
> machine for a mate and i cant get the hang of doing the same with rpm,
> can anybody help?
> 

rpm -qa                      = list all packages installed
rpm -qi packagename (ie zip) = give info about that package
rpm -ql packagename          = list files used by that package

rpm -i  packagename          = install package
rpm -U  packagename          = upgrage package(I use it insteadof install)
rpm -e  packagename          = remove package

That should help a bit :)
