Re: SWAP SPACE

Hi,

> Is swap space necessary for any programs at all? I have 192mB of memory now, 80%
> of which at the most is used so I've actually turned off the swap file. I guess
> I'm wondering whether this would actually make any programs simply run more
> inefficiently.

My understanding is that a swap file is necessary, but the memory
management of Linux is superior to windoze, so if it doesn't need it, it
won't use it.

The 64MB in my PC are rarely totally used up, unless I open a large number
of programs (or Star Office). Top amoungst oter commands will tell you
your memory and swap file utilisation.

One observation I've had is that a fully running RH6 box takes about 10MB
more memory than the equivalent RH5.2 box.
