Re: Floppy Drives

There are ways around this. 

Some File Managers (X-Files in my case) enable configuration of buttons.
The string "mount -t vfat /dev/fd0 /mnt/floppy" can be set up to execute
with a button click. The files will show in the list box.
Unmount can be set up in a similar manner.
