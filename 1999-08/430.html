Re: EXTENDING / PARTITION

Hi ...... just symlink directories on other partitions/hard drives....
Iif you dont know how to do it you can either RTFM :) on symlinks   OR,  use "mc"
(press F9 & select the appropriate options)

I do it all the time when I run out of space :) I either add another hard drive or
trash/shrink a win95 partition etc and then voilla!! All my smaller (ie/ 400-540Mb
hdd) boxes have old 100-200Mb hdds attached with lots of symlinked directories :-) I
usually symlink /usr/src & /usr/local onto the additional hdd's.

Just create the directory (eg/ /usr2/local)  - ensure /usr2 is added in to
/etc/fstab if its a separate hard drive! ) then COPY all the files from your current
/usr/local/* (for example) to /usr2/local/  .... at this point be aware that IF
you're currently running any apps in any /usr/local directories & you delete
everything in /usr/local , you might have a "LITTLE" problem with lockups/crashes :)

You now need to delete /usr/local/* & then symlink it to /usr2/local.   IF you are
not confident abt this process then ensure you merely mv /usr/local/*
/usr/local.old/*  so you can "reconstitute" it should it go wrong.

OK.... clear as mud!!  :)  Sounds complex but its not... especially if you use "mc"

Cheers, Phil
