Re: redhat 5.2 box httpd cannot find local host name.

On Mon, 16 Aug 1999, Khai Dao wrote:

> 
> HI ALL !
> I have a problem bring up httpd ( appach 1.3 )
> a message when boot up "Cannot  determine local hostname. Use the Server
> Directive to set it manually."
> /etc/rc.d/init.d/httpd start  when it run from the command line also
> give the same above error message.
> What do i do to fix it. I could not find the config file that indicate
> the local host file.
> Any Help is appreciated.
> 

If you have been attempting to alter networking lately you've probably
setup localhost incorrectly. 

If you haven't have a look in /etc/httpd/conf/httpd.conf and as it says,
manually set the ServerName directive. (Set it to the name of your
computer).
