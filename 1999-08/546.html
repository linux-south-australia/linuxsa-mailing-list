Re: Binary to Decimal Converter

David Lloyd wrote:

> Last night whilst reading my new book, "Using & Managing PPP", I
> discovered I lacked a simple CLI binary to decimal converter.

[dt@orac dt]$ bc
bc 1.05
Copyright 1991, 1992, 1993, 1994, 1997, 1998 Free Software Foundation, Inc.
This is free software with ABSOLUTELY NO WARRANTY.
For details type `warranty'. 
ibase=2
> 1010
10
> 1101
13
> 110
6

where the "> " prompt distinguishes what I typed in from the
system output - it doesn't appear in real life).  Specifically,
I started up bc, set the input base to binary ("ibase=2"), then
entered the values to be converted.  bc also performs arithmetic on
arbitrarily (sp?) long numbers.  You can also set the output base
("obase") to something other than decimal, but if you change ibase,
be sure to do it last (left as an exercise to the reader to understand
why).

bc - yet another standard Unix tool that widens the gulf between
Unix and Microsoft products.
