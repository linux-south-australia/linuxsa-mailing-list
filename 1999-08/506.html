>>>>> "Michael" == Michael Wardle <michael_wardle@hotmail.com> writes:

[...]

    Michael> When I run top, it reports that all of my physical memory
    Michael> (32 MB), as well as 100 MB of my 256 MB swap is being used.

You might want to consider the possibility that you need to install more
RAM to get good performance with the software that you are running. Some
people suggest that 1:2 RAM:swap ratio is about the limit for acceptable
performance.  Your ratio seems more than 1:3.

    Michael> It reports that X is using 52 MB which is 174% of my
    Michael> physical memory.  I am also running Netscape Communicator,
    Michael> which I believe may have something to do with my problem,
    Michael> top reports SIZE 18548 RSS 8368 %MEM 27.2

The usage that X reports includes all mem mapped by X, including the
video RAM aperture.  You can probably subtract the size of your video
memory from this figure to get the amount of "core" used.

You can run X at a lower bit depth ("startx -- -bpp 8" or similar) to
reduce the size of X somewhat.
