RedHat & PPP Howto

I have developed a HOWTO for connecting a RedHat Linux Box to the
Internet. It doesn't assume that you have KDE or GNOME but does assume
that you can run X as root. Provided you can run the following tools:

1) X
2) modemtool
3) netcfg
4) linuxconf

You should be able to get a connection going with this HOWTO. As it is
approximaltely 150kB I am not sending it through these lists. Please
email me directly to get your review copy. The review copy is useable,
but lacks an exaplanation of CHAP (primarily because I don't know how to
use CHAP !!).
