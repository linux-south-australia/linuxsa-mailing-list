RE: Ethernet card setup

edit /etc/conf.modules - insert or modify the following two lines:

alias eth0 ne
options ne io=0x240 irq=9

You'll probably find that the second line is set to  io=0x300 - which is
obviously wrong :)

When a module is loaded - for example, ne.o, /etc/conf.modules is checked
for options first.
