Re: Virtual Private Networks

Brett,

> Does Linux (Redhat) support VPN's ? Any ideas on where I can look for 
> articles/case studies etc ?

Linux does support VPN, you just have to choose a standard.  The good
thing about standards is that there are so many to choose from.

I have no experience with VPN myself, but have found the following
resources on Linux VPN support by searching the howtos and
www.freshmeat.net.

If anyone has tried any of these approaches then I'm sure they'll post
about their experiences.

There is a mini HOWTO on Virtual Protected Networks at
  http://mirror.aarnet.edu.au/linux/LDP/HOWTO/mini/VPN.html
that might be a good starting point.  It deals with VPN over SSH.

If you want MS PPTP compatibility, to talk to NT boxes, have a look at the
PPTP-Linux pages at
  ftp://ftp.rubyriver.com/pub/jhardin/masquerade/ip_masq_pptp.html
There is also some information there about the MS PPTP protocol.

A PPTP client for Linux can be found at 
  http://www.pdos.lcs.mit.edu/~cananian/Projects/PPTP/

You can set up a VPN connection using blowfish encryption using VPNd from 
  http://sunsite.auc.dk/vpnd/
which has the benefit of not being patented.
