Easy way to get ppp working on a bsd box is use the the program ppp (it uses
the tun0 device to seend stuff in and out of the kernel),

You need to edit the file ppp.conf which is in /etc/ppp
ppp.linkup as well

(You can actully test that ppp is working with out a ppp.conf
Just type ppp
at the ppp command line type term, then type normal commands to suit your
modem to make it dial, logon then when ppp starts at the other end it goes
into ppp mode auto you can then test it =].)

There is section in the FreeBSD handbook on how to do this (its on your
machine if you installed the docs in /usr/share/docs/handbook or on
www.freebsd.org/handbook/ )

Then once the config files are setup all you do is
type
ppp -ddial (and the service name here)
To have ppp without ip aliasing (or or fowarding or what ever yoou wish to
call it )

ppp -ddial -alias (and the service name here)
To have ppp with ip aliasing.
You dont have to setup any ipforwarding rules or stuff)

e.g i type  ppp -ddial rebel  (rebel is the name of the section int he
ppp.conf file that corrsesond with the isp i want to dail)
