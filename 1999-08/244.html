Re: Shell Scripts and variables

If you check your manual pages, I think that one way to put the output
of a command into a variable in sh or bash (which sh is linked to on
RH), is by doing
DATE=`date +%Y%m`
(where ` is the backtick under the ~ key)
Also, you don't need to export the variable if you're only using it
inside the local scope of the script.
