SuSE 8.2 Preview video.

Hi Folks,

I got home kind of late and was just getting ready to go off to bed, 
 but I decided to check my email quickly before going off.
 When I opened my Netscape browser which points to SuSE I noticed that 
SuSE had a video preview available for SuSE Linux 8.2.

Now, I don't usually want to bother with plugins, so I usually try to 
find a way to capture a .ram (Real Player) file either through my 
browser or in /temp.  It is just a text file that contains a web-like 
address that points to where the video is streaming from.

I captured the .ram file with Opera, and then clicked on it to play it 
with Real Player (I have the associsation set up in KDE - right click, 
open with...remember...)

Anyway I start to play it and it says....

...."This video sponsored by Microsoft WindowsXP tablet PC"

Also it never got to the video, but I did get to see 2 Windows XP 
commercials....  LOL

I edited it with emacs and named the text file SuSE.ram.  The contents 
of the non commercialized portion read:

 
rtsp://a980.v06745.c674.g.vr.akamaistream.net/ondemand/7/980/674/3ccdd04/cnetnews.download.akamai.com/674/t0326linux_2_hi.rm?start="00:00:00"&end="00:16:30"

If you want to see the video, just copy & paste that into your favorite 
text editor, and save it as suse.ram

Ok, I am off to bed.
