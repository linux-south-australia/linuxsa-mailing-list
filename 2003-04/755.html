RE: LaTeX -> PDF -> download from Apache -> error on Mac OSX

Hi all,

I think I may have fixed it... rather than using dvipdf, which I was
using, I now use this:

dvips -Ppdf -Pcmz -Pamz file.dvi 
ps2pdf file.ps 

This embeds PostScript Type 1 fonts in the document (also fixing the
ugly bitmap Type 3 fonts I was getting in certain places). Using Acrobat
(under Win XP), I can hit <C>-<A>-f, and it gives me a listing of the
fonts used, and they are all embedded Type 1...

The question remains, however, whether these actually work properly on a
Mac OS X 10.1 with Preview... woudl someone mind checking
http://www.yellowgoanna.com/~richard/test.pdf ...
