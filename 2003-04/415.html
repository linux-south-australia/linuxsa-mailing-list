Re: Linux server hardware

Quoting Kim Hawtin <kim@freesolutions.net>:
> Now, I've not had any experience with Serial ATA yet, and was wondering
> if anyone has experienced any "gotchyas" with it.

Experienced?  No.  But there are supposedly some compatibility problems still 
around the place.  Some drives and some motherboards don't talk, since some 
manufacturers released products before the serial ATA spec was ratified.

So best bet is to buy controllers and drives together, with the seller 
verifying that they'll talk together.

As for Linux/FreeBSD support - no idea.
