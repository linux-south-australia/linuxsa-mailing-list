Re: Upgrade from SuSE 8.1 to 8.2/Review of 8.2

Ok a follow up on this.  Not having the Yast2 control center installer 
module caused me to feel it worth reinstalling SuSE 8.2. I also had an 
issue with Jpilot not being able to start because it could not find fonts.

Solutions:


    * The mere presense of Windows fonts in
      /usr/X11R6/lib/X11/fonts/local breaks Jpilot in SuSE 8.2.
    * If you upgrade your QT3 3.1.1 that comes with SuSE 8.2 in the
      boxed set to QT3 3.1.2 that will install via apt, the Yast2
      installation module will be broken.

Hope that helps. As for me, I have all the answers I need to go ahead 
and upgrade the other machines. :-)
