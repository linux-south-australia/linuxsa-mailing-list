On Friday, April 18, 2003, at 07:51 PM, Brian Astill wrote:
> I have copied all the CD files onto that 1G partition, but of course 
> it won't boot from there.
> Q. How could I prepare a boot disk which would find Knoppix on that 
> second partition and boot up?

Boot up using the Knoppix CD, and you'll likely be able to pass 
something to the default kernel, that'll let you boot from the hard 
drive (you can then configure & run lilo).

rescue root=/dev/hda1

possibly?  (assuming hda1 is your root partition).

Note: even if you get it booting, that's probably not going to work, 
knoppix uses lots of references to the cd to decompress things on the 
fly.  It's not going to be as easy as you're thinking, imo.
