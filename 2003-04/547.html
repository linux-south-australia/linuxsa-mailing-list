Re: Uninstall rpm files

Blangberg wrote:

>Hi,
>How can you uninstall a program once its been installed on Mandrake using
>rpm to install it, I installed mrtg and some of the files corrupt, so now it
>wont install over with a fresh copy because it says conflicts with the older
>files
>Thanks
>Ben Langberg
>
>  
>
Ben
Open a shell editor and type rpm -e mrtg. Provided that this is the 
right name of the rpm file you are trying to remove, this should do the 
trick. If you want a full list of commands under rpm, type rpm --help. 
You can also use KPackage which is under Mandrake, Configuration, 
Pakaging, KPackage in Gnome (this assumes that you are running Mandrake 
9.0, like myself).

Hope this helps
