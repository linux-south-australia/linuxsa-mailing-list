RE: LaTeX -> PDF -> download from Apache -> error on Mac OSX

Hi Richard.

I can duplicate the problem on a Mac OSX 10.1 box I have here!

If I just slick the test.pdf, it opens by default with 'Preview' on the
Mac. If I open Acrobat Reader, and open test.pdf from the file menu, it
opens fine.
Preview is one of the bundled OSX system apps. I don't know why it doesn't
work in this instance, but I've generally found Acrobat to be better
behaved.
So, get your OS person to save the file to their desktop, and open it with
Acrobat Reader and they should be fine.
Regards,
