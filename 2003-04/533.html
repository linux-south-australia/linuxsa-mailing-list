Re: gateway modem connect/sudo

"Ben Williams" <benw@webmedia.com.au> wrote:

> the only problem lies in this line in the script:
> 
> sudo echo 1 > /proc/sys/net/ipv4/ip_forward

The redirect occurs before the process is executed,
by your current shell with its lowly privileges.  All
you are doing here is running "echo" as root.  You
need the other shell (with root privileges) to do the
redirect.  I haven't run sudo in a while, so the
following may or may not work:

sudo "echo 1 > /proc/sys/net/ipv4/ip_forward"

or if not,

sudo "sh -c 'echo 1 > /proc/sys/net/ipv4/ip_forward'"
