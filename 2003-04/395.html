Hi, I have seen this message when I have compiled a new kernel and 
included ext3 or my scsi card as a module. The kernel is unable to see 
the root file system. I notice that many distro's use a initrd image to 
get the root file system seen at boot while retaining the 'everything as 
a module' kernel style. Something I didn't do.

I recompiled without using modules for those things the kernel needs to 
have to mount the root file system and it was okay. I also don't use the 
lilo initrd =
option for my own kernels. I haven't used grub.
