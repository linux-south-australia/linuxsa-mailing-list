Curiously, I once created a mutli-boot monster machine that mutli-booted in
the following manner:

     Windows or FreeBSD   [BSD Bootloader]
       |
       | Windows XP or Windows 98   [NTLDR]
			 |
                         | Windows 98 or MS-DOS 6.22   [Win98 Startup]
					     |
                  [MS-DOS config.sys Menu]   | MS-DOS 6.22 or Windows 3.11
                                                           or Windows 2.03
                                                           or Windows 1.01

It was kind of a cool experiment.  Because I had a FreeBSD box as my
gateway, I actually got TCP/IP working on both MS-DOS and Windows 3.11,
which was kind of cool.  I ran IEXPLORE 2.0 in Windows 3.11 :)

Anyway that lasted all of a few weeks before it all got blown away :)
