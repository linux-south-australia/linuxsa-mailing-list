Re: changing isp and ip numbers

Danny,

> It is  just a normal modem dialup with 2 static ip's .
> the isp is telling me that  it is because of my old ip's that are
> still in my system..

I would try this:

# cp /etc/wvdial.conf /etc/wvdial.BACKUP

...then use the RedHat GUI tool (the start menu->internet or wherever it
is) to set it up if you are running X. If you are NOT running X, then
you could looking at the settings in:

/etc/sysconfig/network
/etc/syconfig/network-scripts/ifcfg-ppp0

...and comment out any line that happens to make mention of an IP
address. Do this by placing a hash (#) in front of the line:

EG:

IP=10.0.0.100

...would become...

#IP=10.0.0.100

...but honestly it sounds more like an authentication error.

man wvdial.conf
man wvdial

...and work out how to get wvdial to turn on ridiculous level of
debugging. From memory it logs to:

/var/log/messages

BUT it could be:

/var/log/ppp.log
/var/log/syslog

It would be worth checking /var/log/messages for clues anyway.
