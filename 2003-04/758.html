[NEW] "tiny desktop computer" with embedded Linux

Flyweight Linux PC comes out swinging 

Taiwanese hardware maker Via Technologies and U.S.-based PC
maker Mini-Box have unveiled a tiny desktop computer featuring
an embedded Linux operating system. The Mini-Box M-100, a
general-purpose computer built around Via's EPIA Mini-ITX
mainboard, is about the size of a dictionary and weighs about 2
pounds. Besides being used as a desktop, it can also be used as
"embedded" hardware--housed within a larger machine to perform
a specific computing task.
