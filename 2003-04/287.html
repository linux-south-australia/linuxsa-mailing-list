Re: Samba/Win2K - browse lists over different subnets

On Thu, 2003-04-10 at 15:10, Ryan Verner wrote:

>    remote announce = 10.10.2.1
>    remote browse sync = 10.10.2.1

Have you tried setting the above two options to the broadcast address
for the subnet (10.10.2.255)?

Do you know for certain that 10.10.2.1 is the *only* machine on that
subnet that thinks it's the master browser?

I'm pretty sure you mentioned that the workgroup name was the same on
both subnets. If this isn't the case, you'd probably want something like
this for both options:

	* = 10.10.2.255/DOMAINNAME
