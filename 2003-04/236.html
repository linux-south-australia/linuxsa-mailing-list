I'm looking for a small footprint stand alone (not PICMG) P4 or Pentium-M
based single board computer that would have the following key on-board
features:
- 1x PCI-slot (32-bit is good, but 64-bit is better)
- CF and/or DOC socket
- 10/100Mbit Ethernet controller ( 1Gb is a plus ) with PXE support
- 2x USB 2.0
- Audio controller
- Serial ports
