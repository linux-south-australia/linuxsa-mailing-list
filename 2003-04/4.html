These three reasons are why I'm looking forward to Ximian's version of 
OO. If you are running Gnome2, Ximian's version should fix all of the 
above. It will look (and feel?) like a GTK2 app, it uses the Gnome VFS

Fontconfig for getting at fonts. This plus a whole swag of other 
improvements. It's supposed to be faster, too.

It sounds like Ximian are coming up with the version of OO that should 
have been released in the first place. It sounds *really* good.

effort <http://ooo.ximian.com/>;.
