Re: ntfs

Hi Alex, 

Linux certianly can access NTFS partitions.

Mandrake has had NTFS read support compiled into the default kernel
since at least 7.2, Redhat has no suport in the kernel at all.
I'm not sure about other distros.

As for writing to an NTFS partition this is SERIOUSLY not advised,
and classed as extremely experimental. NTFS write support can be 
enabled when compiling a new kernel but it is suggested you back up 
all the data on the partition before you perform any writes. 
It is also recommended that the file system be scanned
for errors afterwards.

When mounting use mount -t ntfs /dev/[device] /mnt/[mountpoint]

Hope this helps.
