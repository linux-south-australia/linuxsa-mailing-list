Re: ntfs

Alex!

> Can Linux access NTFS partitions ie removable hard disk with ntfs on it and
> I want to mount it?

The answer is yes, but I don't believe that it would be built into a
default kernel in one of the generic distributions.

If you:

1) cd /usr/src/linux (or wherever the kernel actually is)
2) make menuconfig
3) goto the file systems option
4) read the help on "NTFS"

Note: NTFS writing is flagged as "beta" and likely to corrupt data, but
reading appears to be OK.

Also:

1) man mount
2) man fstab

Should help...

(

mounting ntfs should be something like:

mount -t ntfs /dev/some_device /some/mountpoint

)
