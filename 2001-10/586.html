This is Microsoft's advertised minimum spec.

> RH 7.1 needs :
> 286 processor
> 32Mb ram

> CDrom and/or Floppy

> 
> RH 7.1 can't run on that ..... can it?

Perhaps it's 266MHz processor? I would say that this is a pretty
inaccurate description of the minim hardware spec (depending on what
you're using the box for). Remember too that the Windows 98 spec does not
allow for any applications.

I'd say (from personal experience) that the following hardware is
suitable for a RH 7.1 box:

 486DX CPU
 16MB RAM
 400MB HDD
 combination of Floppy drive / CD-ROM / Ethernet card

You wouldn't run X on this machine very happily (unless you possibly use
twm as the window manager). It will work as a thin X-Term or a simple
router / gateway / firewall for a modem.

I don't believe that a 15" monitor is required (Glen Turner's been
installing RedHat Linux on machines without monitors at all). I've also
seen Windows 98 running with a 9" VGA monitor.
