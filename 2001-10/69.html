Try not to use Network browsing as a diagnostic tool, because it is damn
near useless for it. :-)

You can use the machine's netbios name to check SMB connectivity by doing
something like Start->Run and typing \\MACHINENAME in the box and hitting
enter. If SMB is set up properly, you should be able to "see" everything
that permissions allow you to. If this fails for whatever reason, you can
then check your Samba logs for more clues.

Hope it helps,
