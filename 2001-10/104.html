 > * do it graphically,
 > * > but I don't have X handy. Are there any tricks out there to
 > * access this

Try mount -t iso9660 -o session=x /dev/yourdevice /yourmountpoint, where 
x is the session number, if kernel version >2.3.4
(or read man mount)
