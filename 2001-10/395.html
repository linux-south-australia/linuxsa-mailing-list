Try xls2csv. It may work for you, but I daresay its going to spit at you
if you try anything fancy.

The main website is currently offline for some reason, so I extracted
the tarball from the source RPM and put the source on my website --
http://www.plug.cx/misc.html. Just extract and type "gmake".

You'll have to wrap some PHP around it --

<?php

/* This function requires a the contents of the Excel file in $in, and 
 * will spit out CSV to $out
 */

function xls2csv ($in) {
	$fn = tempnam ("/tmp", "converter");
	$fp = fopen ($fn, "w");
	fwrite ($fp, $in, sizeof ($in));
	$out = system ("/usr/local/bin/xls2csv " . $fn);
	return $out;
}

?>

You may have to play with the ``system()'' line, as I'm not sure if
tempnam returns the full path of the temporary file or not. Perhaps you
could add a note on the function reference for tempnam() on the PHP
website when you've found out.
