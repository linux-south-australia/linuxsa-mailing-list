I run the same software for the same reasons...

Ipac has the option to sum the data up in to one file, rather than 1 file
for each time fetchipac runs. Have a look in the README file, in the section
titled Cleaning Up. If you add the cron entries as it specifies, that should
speed things up. Or you can just tell it to sum a period up as follows:

ipacsum -r -s 20010101 -e 20011024

which would summarise all the data in to one file, for the time period 01
Jan 2001 to today.
