Re: SHared Memory

"Davis, Harvey" <Harvey.Davis@dsto.defence.gov.au> wrote:
> I was wondering if there was a way of cleaning up shared memory segments and
> semaphores left over by programs that could be passed a wildcard meaning all.

The point of unix, that makes it better than other operating systems, is that
it encourages a tool mentality, which permits you to string together simple
commands to achieve a more complex result.  You don't need a tool that has
every option that will ever be wanted, and indeed such a tool will never be
written.  Here's the idea:

  vi `find . -name '*.h' | xargs grep -l hwif_ide | xargs grep -l LOCK`

(edit all .h files that contain the strings "hwif_ide" and "LOCK".)

Now think along the lines of: ipcrm `command to list all IPC ids`
