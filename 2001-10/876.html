Re: How do I restore my X login?

> Thanks for your reply, but it is set to run level 5.
> What else could gave gone amiss?

all that runlevel 5 means is that it will run the scripts in
/etc/rc.d/rc5.d (or some variation of that path) as follows:

if the script starts with "K", it will run it: `script stop`
if the script starts with "S", it will run it: `script start`

So... you could check for the xdm (or gdm or whatever) process in the
rc5.d directory, and if it's not there, create a (soft)link to
/etc/rc.d/init.d/gdm there, starting with "S" and a number, similar to
the rest of the scripts in the rc5.d directory... 

Hmm... I'm sure that's a little vague and incomplete, but it will do
for now :)
