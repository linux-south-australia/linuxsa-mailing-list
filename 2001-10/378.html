Re: Performance evaluation

On Thu, 11 Oct 2001, Shane Warner wrote:

> Can anyone point me in the right direction to a HOWTO or a NHF that describes 
> performance monitoring and optimization of Linux and/or any suitable tools to 
> perform the same.

Just about any generic unix-based performance documentation will help you
(www.google.com is a good place to start looking for all things
web-based). There are also some operating system independent things that
you should always consider. I'm referring to things such as turning off
services that aren't in use and the like. You can also use tools such as
top, ps and vmstat to help work out where a bottleneck lies.

If you can give the list some kind of indication as to where the problem
lies, I'm sure you'd get some other suggestions too.

Hope it helps,
Matt
