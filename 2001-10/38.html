Re: Debian pretty-boy config utilities ...

On Sat, Sep 29, 2001 at 08:02:38PM +0930, Richard Sharpe wrote:
> Hi,
> 
> In an effort to liven up proceedings in this mailing list, I would like 
> to enquire whether Debian has any pretty-boy config utilities like 
> netcfg and netconf under RedHat, or do you have to be a Macho Man and 
> config it with the files?

I also have wondered this, but have not found anything... I wonder if
anyone knows any reasons for this... I postulate that it's for one of
the following reasons:

1) Debian is more flexible than other distros, so you can't make the
sorts of assumptions that you would need to to implement said
prettyboy config files.

2) Debian simply hasn't gotten around to it yet -- it's not important
to the generally highly technical userbase (and developerbase) of
Debian, so it hasn't been done.

3) Maintaining such a system requires substantially more effort than
the Macho system, and reduces flexibility....

I do wonder if there's any better reasons... Anyone know?
