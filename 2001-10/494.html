> Malsync is the software which talks to the AvantGo (or other MAL server)
> website to sync your Palm device.  You select the channels you want by
> visiting the AvantGo website.

Yes, I set it up so it is called directly by jpilot. Have a look on the
jpilot pages for more information.

Alternatively, you can use 'plucker' which is a set of software with two
components, a palm reader and a unix 'spider' program which builds the
plucker documents.

This is a nice alternative, you can generate plucker documents out of any
old HTML you have lying around, creating useful little e-books on whatever
you desire.

http://plucker.gnu-designs.com/
