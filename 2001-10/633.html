Hiya Sjaak,

I set up the Sun Java 2 SDk 1.3.1 as a tar.gz file from the site a few
weeks ago.

I would imagine that all you have to do is type in
./j2sdk-1_3_1_01-linux-i386-rpm.bin to execute the code.

I moved the jdk directory to /usr/local/ and added the path to my
.bashrc file.

Also make sure that when you type in java that it actually runs the
newly installed jdk. Mandrake has its own java vm, I had to un-rpm it.

Overall I'm a happy JEdit, Sun JDK 1.3.1 user.
