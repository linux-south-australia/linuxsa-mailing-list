I'm sure most people would...

However, you'd want to run it within VMWare for the same reasons as if
you wanted to run anything within VMWare...

- if you have need for more than one machine, but only have one
  physical machine available
- if you need to switch between Windows or Linux and Solaris
  regularly.
- if you are developing installation software for Solaris -- it's easy
  to maintain a copy of a clean VMWare install, which you can
  duplicate and install, and so on...
- etc...
