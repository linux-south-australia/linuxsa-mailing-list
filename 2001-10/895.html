RE: Boot CDRom Creation - Dual Boot XP / RH7.2

On Mon, 29 Oct 2001, Freeman, Peter (ERHS) wrote:

> While Vmware is a great piece of software, and I've had success with it in
> the past, I'd imagine that on a laptop running with 64mb, it may not be the
> best option

Urk. Missed the 64Mb bit. VMware needs some CPU and plenty of memory. You
can skimp when allocating memory to guest OSs (eg NT in a 32Mb VMware
session is fine - 48 for win2k - it just involves a little waiting) but
skimping on the host system is asking for trouble. 96Mb is workable, 128Mb
more reasonable, 256Mb lets you run a whole LAN without too poor
performance.

VMware isn't just useful for Windows. I don't know of any practical way
other than VMware to run *BSD under Linux, and the same the other way
around (there is a VMware port for BSD). Does anyone else?
