Re: Red Hat 7.2 file system

> Red Hat announced it was going ext3 instead of ReiserFS, but on
> balance it was probably the right decision.

RedHat actually had ReiserFS modules and utilities in the 7.1 release.
So if you really wanted to use it with 7.1 you could.  It's just not
supported in the initial installer.

I assume 7.2 is the same, although 7.2 now uses GRUB instead of LILO
and GRUB doesn't support booting from images sorted on ReiserFS
filesystems.  This can be worked around in a couple of ways of course.

BTW I've been running ReiserFS on my laptop for quite a while.  It's
easy to migrate from ext2 if you have ample disk space (like twice as
much as the data you have) and GNU parted and the ReiserFS tools to
resize things.  The only problem is that my laptop never crashes under
Linux so the usefulness of a journalling files system is sorta lost
:-)
