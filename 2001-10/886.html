RE: Boot CDRom Creation - Dual Boot XP / RH7.2

>I been reading some workarounds for a Win2K / RH Dual boot and 
>these involve
>creating a boot floppy with Lilo and then manually carrying out some dd
>commands and editing of boot.ini file to let the Windows Boot 
>manager boot
>into Linux.

Depends on which one you install first.  Install Win2k (and XP I presume, 
tho I haven't tested XP yet) first, then Linux second.  Linux should 
recognize the Windows boot partition and put the entry in LILO for you.
If not, you can put in the entry manually after you've installed Linux.
This worked fine for me with Win2000 & Mandrake 8.0 & 8.1, haven't tried
it with RH7.2, but should be very little difference.
Generally its only when you install Windows after installing Linux that you 
have problems.

>Question 1: I assume although I have never tried it that you

>boot floppy for
>Win98 has - Need this to be able to run fdisk /mbr and keep XP

If you use EasyCd (I'm not familiar with any other CD writing packages under
Windows), it has the option of creating a bootable CD.  You just need to
have
a boot disk handy for it to use.  I use a Windows 98 boot disk to create 
bootable cds with Ghost images for use when i don't have access to a
network.
This should get you out of trouble.
