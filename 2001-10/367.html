Im not much of a shell scripter, Id probalby use perl for soemthgn like
this, but I think an expression like:

grep [0-9].*M
will match "111M"

and
grep [0-9].*k
will match "99k".

grep will return 0 if it matches - the return value of the last command
exectued is stored in the variable $? in sh and bash.

This might be enough to help you discern the 'M's from the 'k's.

Hope that is vaguely helpful.
