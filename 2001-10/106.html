Getting linux to serve ftp to an iMac

I have some success to report. I loaded up netatalk from the rpmfind site
and it worked without any configuration. This allows the linux box to serve
AppleTalk, so I can now load files back and forth from the Mac quite
happily. This will allow us to leverage each computer for backups and save
a lot of sneaker rubber.

It still leaves me with the issue of getting ftp to work, however this has
been demoted to being an "intellectual" problem as opposed to a "practical"
one, and so it is less urgent.
