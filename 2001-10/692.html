Sometimes bz2 files are downloaded as text files, as the extension is 
not old enough yet to be registered all over. This means that the data 
in the file becomes corupted. :-(

Try using a different program to download the file. If the file is on 
an FTP site, try using a real ftp program rather than a web browser. 
The other option is to download the GZIP version, there is just about 
always a GZIP version, as GZIP is quicker to use, but produces larger 
files. The files are not that much larger though, unless you get a file 
above a few MB.

Anyway, I hope this helps.
