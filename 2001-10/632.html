Yeah I've tried Opera before - it's certainly quick and feature rich, but I

Have you tried version 0.9.4? I have been using Mozilla as my default
browser in both WIndows 2000 and Linux for 2 months now and the latest
stable builds have been great. You could say that I am a very heavy web
user. I get around the same MTTF for both Mozilla 0.9.4/Mandrake 8.1, and IE
5.5 under Windows - generally this is a good thing.

Had to install seperate RealPlayer, Flash/Shockwave plugins though. One
thing I haven't gotten to work: I can't get the Java VM to work though,
Sun's Netscape 6 plugin bombs upon loading.

Mozilla's speed have been improving over the past few months, but it is
slighly sluggish on my machine compared to IE 5.5 under WIndows 2000 - about
on par with Konqueror 2.2.

Mozilla 1.0 final, like Evolution 1.0, and Star Office 6 ARE things I'm
looking forward to.
