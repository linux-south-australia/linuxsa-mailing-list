Re: Boot CDRom Creation - Dual Boot XP / RH7.2

On Mon, 29 Oct 2001, Michael Selge wrote:

> I'm not planning on leaving the dual boot setup permanent. Just want to
> check out some of the "features" of XP eval copy - The notebook only has 64
> Meg RAM so XP will run like a real dog anyway ( actually RH7.2 probably will
> also if a GUI is used  but hopefully not as badly ).

If the machine matters you, dualbooting may not be the best option.  An
alternative is payware vmware, which does completely isolate the guest OS
from your host OS. www.vmware.com. There are drawbacks to VMware but it
does work quite well for what it tries to do, if you disregard the sales
literature.

(If you had wanted to install and test another version of Linux vmware is
not a good option, because apart from doing closed-source things behind
the kernel's back it emulates an entire PC including BIOS. A much better
solution is User Mode Linux, user-mode-linux.sourceforge.net, which runs
Linux under Linux as just another process.)
