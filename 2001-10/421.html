Re: Installing Debian via FTP

James,

If you have already installed debian on another machine..
have a look at: 

file:/usr/doc/debian-guide/html/index.html

The step's that I'd use..
1. Create the boot and root floppy from the images on the CDROM
	Also create the 'module' floppys. There may be up to 5 of these.

2. Boot your system using the boot and root.. 
	Follow the bouncing ball.
	(If you have any questions about any of the configuration
	options, eg. partitioning, read the manuals, then ask linuxsa :-)

3. Load the appropriate drivers from the other floppies.
	Not sire what the Xircom card needs. It may be the tulip
	module.(?)

	Unless this module has been fixed, after the network has been
	configured, it helps to put the network interface into
	promiscious mode. Change to another console
	<ALT>-F2, and type 'ifconfig eth0 promisc'. Change back with
	<ALT>-F1

4. Load the CD's onto the server machine and make the contents
	available via ftp or http.

5. Configure the laptop's apt methods. Have a look at
	/etc/apt/sources.list
	
6. Install the rest of the OS
