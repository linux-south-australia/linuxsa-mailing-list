On Fri, Oct 05, 2001 at 12:58:13PM -0500, Javier Morquecho Morquecho wrote:
> Hi...
> 
> 	I'm sorry if my question is so easy, but I need urgently solve
> this........how can I solve a domain to a directory of WWW??......

apache's virtual hosting.

set your DNS records so that both domains resolve to the same IP
address, and ad a virtual host directive to your httpd.conf file...


eg:

----
<VirtualHost www.domain.com>
ServerAdmin you@somewhere.com
DocumentRoot /home/httpd/html/domain/
ServerName www.domain.com
ErrorLog /var/log/apache/www.domain.com-error.log
TransferLog /var/log/apache/www.domain.com-access.log
</VirtualHost>
----

I don't know if there's any advantage to having all of them set up as
virtualhosts (as opposed to having one being the default and one being
the virtualhost...)
