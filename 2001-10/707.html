I use a linux (almost) from scratch. The doco is very good but it can be
quite time consuming to build it.
The good things are:-
You only build what you want
You can pick the apps to suit your needs
Everything is built from the authors source code, no funny patched versions
(although in many cases you can cheat and install the latest stable binary)
You can control when and what you upgrade
Your system will be a lot smaller and faster (I use a Pentium 75 as my
target machine)
The bad things are:-
It really can take an awful lot of time
You need to control what you install and maintain compatibility
With so many choices of apps it can be a lengthy process to find what you
like, or what suits your need

One of these days I want to build a full monty Linux from scratch in 3 stages
1. floppy boot and rescue
2. lightweight console based for dial up home user
3. X Win addition for dial up home user

one of these days...... :-)
