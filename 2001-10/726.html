Re: Problem with eth0 dissapearing

Hi Adam

Have you check to see if its an irq conflict. I had great fun Friday night
with a system which would reboot ever time we loaded the Ethernet with
traffic.

I looked at the irq stuff first, and check the docs for the network card and
it supported irq sharing (and I knew that Linux does also) so I thought it
must be a CPU or the motherboard or the memory or the power supply, or a bad
network card.
So after changing everything including doing a BIOS upgrade, I move the
network down on slot on the PCI bus. FIXED

The problem must have been with the on board Intel video, and irq sharing.

Two hours of playing around.

So a lot of fun.

Cheers
    Mike O'Connor

P.S. so all I can say to you is you'll will need to go thought a similar set
of testing to find the problem.
