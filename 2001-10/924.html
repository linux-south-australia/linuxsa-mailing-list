Well, if this was my problem, I'd:

1. Move the machines close to each other in the same room/powersource etc.
2. plug them into a known good hub or switch in the same room with nothing
else attached.
3. see if the problem persists.

If the problem stays, I'd be surprised, but I'd then start swapping out
cables/nics etc.
