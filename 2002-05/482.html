> Can anyone tell me how to configure X with Slackware?

If you're running Slackware, you should know how to configure X by
hand. ;-)

If you're looking for a GUI configuration tool, have you tried the
configuration option that comes with XFree86 4.x?
"XFree86 -configure"

Other possibilities include
- xf86config
- XF86Setup

I hope this helps
