On Thu, 2002-05-16 at 12:29, Andrew Speer wrote:
> On the off-topic of NT and Linux administrators, I am surprised no-one has 
> mentioned the near automatic habit NT admins have of pressing 'CTL-ALT-DEL' 
> on the keyboard as the first thing they do when they touch any server.

What they are doing is using Windows's Secure Attention Key.  This is
a key (sequence) which is guaranteed to elicit a response from the
operating system, and not from any random program which happens to be
mimicking the login prompt.  SAK's are a good idea and it's surprising
that linux doesn't use one for logon (I guess you can't be above C level
secure, maybe even C1, without SAK.)  If you were going to be clever
you would replace init's ctlaltdel handler with one that presented a
login/out prompt.
