Re: How to stop and start network services

> Hi Gurus,
> I've been having a specific problem.
> I've installed CVS on a server running Linux 6.2 and added entries to 
> /etc/inetd.conf and stopped /etc/init.d/network and started it but the 
> server still does not listen to port 2401. Can any of you masters of Linux 
> suggest me a solution for this specific problem.
> 
> Any suggestions will be greately appreciated.

You need to re-start the inetd, not networking.
To do so type the command
ps ax |grep inetd
Note the processID then do the command
kill -HUP <processID>

all should be well.
