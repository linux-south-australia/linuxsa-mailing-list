Re: Perl Question

Quoting Toby Corkindale <tjcorkin@sa.pracom.com.au>:

> PERL5LIB, and also PERLLIB. The former is prefered.
> 
> I believe you could also adjust @INC in your script prior to including the
> modules that can't be found.

'use lib' does this:

#!/usr/bin/perl -w

use strict;
use lib "/home/me/perl/modules";
use MyModule;
use CGI;

Non-absolute paths work relative to the directory that you're running the script
from, not the directory that the script is in. Use the 'FindBin' module if you
need to specify relative 'use lib' directories.
