Craig Chandler wrote:
> How do you turn that on? wait a second, this is so off topic i'm now asking
> how to use win2k

Set the CompletionChar value to REG_DWORD value 9 (i.e. tab) in the 
HKEY_CURRENT_USER\Software\Microsoft\Command Processor section thingy.

Should work with WinNT, Win2K and WinXP. Don't forget to re-start your 
DOS windows before it will work! :-)
