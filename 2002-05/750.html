The LinuxSA maintainers (is it still Geoffrey?) have made the simple 
procedure for unsubscription readily available.  Instructions can be 
found at <URL:http://www.linuxsa.org.au/mailing-list/>, which is 
presumably where the person first discovered how to subscribe to the list.

Instructions can be found at the bottom of EVERY message sent to the 
list, where it should be.

It is also quite common that when subscribing to a list, the list server 
sends information about the list including instructions for 
unsubscribing from the list; I assume LinuxSA also does this.

If two or three methods were insufficient, a Google search for "Linux 
unsubscribe" returns the relevant page as the first result.

I have not been following this thread thus far, so it is not my intent 
to denigrate anybody, however the information you write of is clearly 
very simple to find.
