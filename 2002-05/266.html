Re: Printer Problem

On Tue, May 07, 2002 at 12:49:09PM +1000, David Tanti wrote:
> The problem I am having is that when I try and print to the printer my
> output gets stuck in the print queue for some reason.

Type 'lpc status' on the command line (or xterm). You should see something
like this

lp:
	queuing is enabled
	printing is disabled
	no entries
	no daemon present
psp:
	queuing is enabled
	printing is disabled
	no entries
	no daemon present

Both my boring old lp printer and my boring old psp (poscript) printer have
queuing enabled but printing disabled, meaning jobs queue but don't print.
Type 'lpc start all' and all printing will be enabled.
