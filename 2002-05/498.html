RE: Powerbook 1400cs

Hi

Disclaimer: I'm no Mac user, but no-one else appears to have responded,
so I thought I'd offer some generic suggestions...


> I am looking of setting up a linux distribution on my 
> Macintosh Powerbook 1400cs, however I'm told this may be 
> impossible to do, especially seeing it doesnt have a CD ROM drive.
> 
> Any thoughts on this ('throw it away doesn't really count') 
> would be appreciated!

Do you have a floppy drive (and a network card)? If so, I imagine you
should be fine -- surely some of the ppc distros have an ftp or nfs
install mode from floppy?

Can it boot from the network card (if it has one)? Set up a bootserver
on a PC or Mac on your LAN... make sure you don't have anything else set
to boot from net at the time :)

Do you have access to another similar/identical machine _with_ a CDROM?
Is the hard drive removable? Swap HDD's, install onto your HDD using the
other powerbook, swap HDDs back, edit /etc/fstab.
