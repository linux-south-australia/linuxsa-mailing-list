Re: sendmail command

Well if you delete all files in /var/spool/mqueue it will clear the mail
queue. Although generally only undelivered mail is queued.  

If you want sendmail to not deliver outgoing mail you could edit your
config file and change 0 DeliveryMode line to O DeliveryMode=queueonly
then it will queue all outgoing mail and you would need to run sendmail
with the -q switch to process the queue.
