> there and that is were I put the arg. -nolisten tcp and that done the
> job for me.

RedHat 7.1 has:

(from /usr/bin/X11/startx):

if [ -f $userclientrc ]; then
    defaultclientargs=$userclientrc
else if [ -f $sysclientrc ]; then
    defaultclientargs=$sysclientrc
fi
fi

...therefore I think it would be better to have:

export $userclientrc=-nolisten

(or if you use a shell different to a bourne shell whatever gets that
user variable into your environment) because next time you upgrade X, it
probably won't undo your change again.
