RE: Open Office 1.0 - was [OT] Bitchin about M$

Ive done some comparisons with build 641d on windows and with doc files.
I dont have the specs, but I noticed that with a several mb doc file
word was quicker to load, but it hadn't loaded all the graphics, and
when you scrolled to a page you had to wait a second for the graphics to
apprear. 

OO seemed to load the lot up front, and so even while holding down page
down all the graphics where there as you flew past. Which method is
better is probably debatable, but similar reasons can probably explain
the speed difference between excel and scalc (?).
