I had this problem with a linux server I set up at my local high school.
(We used it for disk recovery, it's since been retired to a MP3 player
in my lounge room :)

I fixed it by editing /etc/inittab to run /usr/local/bin/ctrl-alt-del.sh
when C-A-D was pressed.

IIRC the script put a message to all ttys saying "This is Linux and not
NT! :-)" and something vaguely insulting ;-D
