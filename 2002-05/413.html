If you can define the messages that you want to be alerted about fairly 
well, then I'd recommend NetSaint.

NetSaint's setup to do alerting etc etc and has a plugin to do this.

Even if it feels like a bit of overkill, it can be very handy if you want to 
have SLAs met etc.
