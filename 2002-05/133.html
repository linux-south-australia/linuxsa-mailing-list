Re: X11 material

I use a dual head desktop but had no problems whatsoever getting it to
work. When I did X -configure both my vid cards were detected and a file
called /root/XF86Config.new was written fine. I merely had to tweak that
file to get my mouse to work since "auto" didn't work for me.
The old xfree-3.x.x tools (like Xconfigurator, xf86config, and/or
XF86Setup) will not work with xinerama I believe, and they use the old
syntax anyway.
You could try doing xf86cfg though, as well as the recommended XFree86
-configure method.

For more details on dual head displays see
http://www.tldp.org/HOWTO/Xinerama-HOWTO.html

Alex
