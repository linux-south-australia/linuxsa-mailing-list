If you plug the Jetdirect EX into your printer and press the "test"
button on the Jetdirect EX it will actually print a test page on which
the IP address is noted.
 
If, like the two Jetdirect EX's I've come across lately it is set to
0.0.0.0 you will need to download the configuration software from HP
(about 18Mb) to change the IP address to something in a usable range.
(If like me it then loses those settings each time the power is lost you
will simply throw it away and get something else)
 
Best of luck.
 
Matt
