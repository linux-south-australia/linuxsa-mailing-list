RE: nothing left to fiddle

Well Actually,

I installed RedHat 7.3 on my desktop Celery400 last weekend, and I
must say that it is now at the stage that it is boring. So far the
only thing I've had to do is install a CSS plugin so that Xine can
play encrypted DVDs. (Just an off note, if you want to play DVDs on an
IDE drive, turning on DMA **REALLLY** helps!)

Load up Xawtv and the USB webcam works without doing anything. XFree
was installed and configured without any intervention. Network was
configured automatically. Haven't had to log in as root yet to do
anything except that Xine plugin. KDE 3 looks pretty good.

Oh, there was one other thing I had to do... Point Evolution to my
IMAP server... For some unknown reason it couldn't guess my password
all by itself!

Darryl
