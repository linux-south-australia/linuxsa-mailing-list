I found Gentoo to be good for the beginner, as it has step-by-step
instructions to build it from the ground up, so you get to learn about the
structure of linux.

it is also ports-based, which makes it easy to install applications,
ensuring they are up to date and getting the dependancies for you.
an ADSL connection to the net makes it more bearable on download times! but
you can start from as little as a 16Mb* downlaod for an i386 setup

www.gentoo.org
