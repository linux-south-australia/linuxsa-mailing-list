Re: How to restore from QIC Tape !

On Mon, May 13, 2002 at 09:48:52AM +0930, Linh Yeung wrote:

 > I know for sure that these tapes contain tar files, the files were backuped
 > using a RS6000 on unix. Could this be a possibility that I need the same
 > machine to restore them.

You probably need to reverse the byte order:

   dd if=/dev/st0 bs=512 conv=swab | tar xvfp -

The "conv=swab" bit does byteswapping (e.g., 0x1234 will be changed 
into 0x3412).  This is needed when you're taking a tar archive from
which has been written by a processor with one endianness to be read
on a processor with a different endianness.
