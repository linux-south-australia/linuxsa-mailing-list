yeah sorry - I was referring to Woody at the moment.  Once a release has
been "released" it is called the stable release so yes only security
fixes are back ported (my firewall/gw runs potato still so only has
security updates).  It goes unstable -> testing -> stable in decreasing
order of "bleeding edginess" which is currently sid -> woody -> potato.

BTW kde is a kinda special case, there's a guy in debian who packages
the latest kde "in advance" of it being officially incorporated into
debian so you can get download from him.  This happens a bit where
there's interest in something, use the search on packages.debian.org
to find things like this (or maybe google).
