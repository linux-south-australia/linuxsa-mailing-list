Re: Man Pages

Andrew Galdes <AGaldes@elders.com.au> wrote:
> Is there a way to search man pages for a phrase?

The man pages are stored in files in /usr/share/man/man*.  Each page is
compressed, so you need to walk that directory tree, gunzip each file
and grep the output:

  find /usr/share/man -type f -print -exec gunzip {} \| grep KEYWORD \;
