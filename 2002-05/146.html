Re: Tight VNC & ssh

On Sat, 04 May 2002, dalpets@ozemail.com.au wrote:
> I simply want a secure encrypted path to a remote Windows box (server)
> from my Linux box (client) when initiating a TightVNC session with the
> VCNviewer from Redhat 7.2.

You need to have an ssh daemon running on your windows box, so that you
can connect to it.  I think the cygwin utilities http://www.cygwin.com/
has an ssh daemon for Win32.

Once you have the daemon running, and can connect to it with a normal
ssh client, you can try this (requires the TightVNC vncviewer):

$ vncviewer -via loginname@windows.ip.address localhost:0

The -via option is only available with recentish versions of the
TightVNC client.
