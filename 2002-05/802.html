Linux on Ps2 - anyone heard anymore?

Hi List
I came across this in my trolling the web

Sony has begun shipping a kit for turning the PlayStation 2 into a Linux
console, at this stage the kit is only available in the US and Europe and
includes hardware and software for essentially turning the PlayStation 2
games console into a Linux PC, with a 40GB hard disk, 10/100Mbps network
adapter, USB keyboard and mouse, monitor cable and a Linux installation
DVD-ROM. The hard drive can't be used with PS2 games.

Anyone know more about it?
