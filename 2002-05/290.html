>I wish to create PDF documents from a W2k machine.

<snip>

>Could people give me an example of how to configure 2-3 queues on the
>same host under Linux.

If you're after a gui interface to do all this, check out the new printing
manager in KDE3, it's great.  Not sure about other distros, but under
Mandrake's install it has a print to pdf, print to email etc
already setup.  You could print to this from your W2k machine by 
configuring your cups server on the linux machine to "share" the printer 
(excuse the Windoze terminology ;) (this can also be achieved using the 
printing mgr in KDE3).

Or of course if you're a CL person just check out the man pages for cups
and set it up that way (probably easier in some ways too ;), the PDF printer
setup under Mandrake uses a program called ps2pdf at the backend of its
print
to pdf option.

HTH
