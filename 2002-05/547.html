On Thu, 16 May 2002, jakehawkes2001@yahoo.com wrote:
[...]
> Does anyone run linux on a Mac?

I have run yellow dog linux on an iBook (http://www.yellowdoglinux.com).

It runs fine.  This is probably a good solution for those working in a
mostly Linux/Unix environment, and without the grunty machine required
to run Mac OSX.

For what I do day to day, I found myself running Mac-On-Linux
(http://www.maconlinux.org) most of the time, which seemed kind of
silly.

If you have the specs to run MacOSX, then you should do this.  It's
really, really good.  Unfortunately, you should not try this if your CPU
is slower than a 500MHz G3, and/or you have at less than 256MB of RAM,
as the performance (or lack of) will probably scar you for life.  To get
the best out of it, you really need a G4 - the difference in performance
between a 600MHz G3 and a 667MHz G4 is huge - I've been told it's the
AltiVec unit that speeds Quartz (part of the display system)
performance.
