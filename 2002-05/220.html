RE: Still [OT] Do Realtek NICs drop packets? Was: [OT] DEC Tulip NICs

Tuesday, May 7, 2002, 11:28:21 AM, Patrick wrote:

> I have used the $20 Surecom cards for a while now in workstations.
> Realtek chipset, lifetime warranty, hard to say no at that price. The
> mini hubs ans switches are ok too.

> I have heard however that they drop quite a few packets (the Realtek
> chipset that is) but I haven't done any tests myself.

> Anyone know if it's true?

> Things seems to run fine in the Windows, Linux, Novell environments I
> have been in.

Yes, Realtek chipset based NICs drop packets. But, having said that,
only under load.

I've experienced varying degrees of packet loss, from 2% to as much as
30% depending on load and the speed of the machine the card was
installed in.

Tests were conducted under Linux (SuSE 6.4, 7.0, 7.1, 7.2 and 7.3)
using the default kernel and with updated, custom compiled kernels
(2.4.17 being the latest tested).

Realtek 8029 based cards were tested in a Windows environment.

All card performed well for day to day operations in a low load
environment, however, a simple flood ping resulted in lost packets,
as did network intensive applications (eg some games, some file
transfers, etc).

Of course, your mileage may vary.
