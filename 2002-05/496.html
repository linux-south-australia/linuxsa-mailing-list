Re: Telstra ISDN Home

>> Yes, Internode run a separate number specifically for DoV. This is so

>most (if not all?) ISPs that do DoV do that, well all the ones
>I've looked into anyway.

Some ISP's can't do it, certain types of dialup server don't have the
ability to differentiate called numbers, and/or answer 56K ISDN DoV calls on
the same number as modems, along with their 64K ISDN data.

We used to have 2 different numbers just for DoV, so that we could
differentiate the 2 different types of bearer-capability we saw (audio or
voice) and not answer if the incorrect one was presented. Then we changed
telcos, and all calls came in as audio regardless. Then just last week we
discovered an exception where if you route your DoV (voice, not audio) calls
through Australia's regional phone network in an abnormal way, it comes
through with a different bearer cap again (but still 64K..).
