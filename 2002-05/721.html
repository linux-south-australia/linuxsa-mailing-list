indeed.  You must really want to use Mandrake!
what you've described doing is setting up an apt-proxy (as you
may already know).  I have done the same thing at home for when
we keep upgrading all the machines to the latest woody packages
we don't have to download them multiple times.  But of course
using debian I just did:
apt-get install apt-proxy
edit it's config file appropriately
edit the other machines /etc/apt/sources.list to point to the apt-proxy
and that's it.
