Re: Damn you dhcpd

On 14 Aug 2001 23:10:57 +0930, Craig Keogh wrote:

> My dhcpd configuration file is simple. It looks like:
> 
> # /etc/dhcpd.conf
> subnet 192.168.0.0 netmask 255.255.255.0 {
>    range 192.168.0.10 192.168.0.100;
> }

While I'm no DHCPd expert, I know enough to think that the netmask
and/or subnet is incorrect.

You've specified a netmask for a /24 subnet, while the subnet is set to
what would be a /16.

If you want to be able to use 192.168.*.*, change the netmask to
255.255.0.0. As it is currently, it doesn't look particularly healthy.
