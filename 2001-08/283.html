Re: Redhat 7.1 and Win2k

Michael, 
	this will work fine, but the win2k boot loader cannot boot linux.
You must use lilo <or another linux boot loader like grub> to boot the
win2k boot menu and then use this menu to boot win2k. 

I have a tri boot system. The lilo boot allows selection between
windows and Redhat 7.1. If you select windows it then loads the win2k
boot loader and allows selection between win2k and 98SE. If you have the
win2k boot loader installed to the mbr and then install linux, this
process
takes place automatically.
