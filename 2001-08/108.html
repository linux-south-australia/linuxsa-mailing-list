I like to use (as root):

`find <dir> -exec /sbin/fuser {} \; -ls | less 2>&1`

try the man pages for the various commands to tell you what is actually
happening here... (make sure you understand the command before running it as
non-nobody, so you can be sure I haven't maliciously embedded an rm -rf in
it...)
