Just to concur: We're running netatalk quite happily here for this purpose.

Also, Steve Jobs announced that one of the features of the upcoming 
MacOS 10.1 (looks like only about 30 days away, beta copies are 
already on warez sites :) ) is that it will fully support a Macintosh 
being able to access smb file servers.

The phrase Jobs used was something close to '... will make a Mac into 
a first class citizen on a Windows file server network'.

So if you aren't in a hurry, and you don't mind moving up in 
operating systems to the latest and greatest, that might be the 
single simplest approach.

[my guess is that this means they've ported Samba into OS X and 
instrumented it into the GUI to make it friendly to use in that
