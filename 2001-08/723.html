On 18 Aug 2001 12:29:52 +0930, steve wrote:

> Take pleasure in the excitement, you have probably one of the best small

> user friendly Linux disto out there, no self confessing Internet user

I put e-Smith through its paces for about a week, and I found it to be
extremely slow.

The way they've decided to go about the server configuration is very,
well, bad. Changing something as simple as the IP address requires a
reboot, something that I think is just plain wrong.

The web-interface you use to drive it is extremely slow. Looking at
'top' would indicate that they're using perl way too much. e-Smith has
some nifty features, but I'm not fond enough to recommend it to anyone.
