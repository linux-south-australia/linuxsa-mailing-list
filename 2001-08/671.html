Re: Debian APT source password?

Ryan,

> I'm wondering whether its possible to somehow password protect an
> apt server, in the regard of having a username/password list to be
> able to use this server to update from?
>
> How would apt-get prompt for this?

You can setup the apt server as a normal ftp server with user access
only (ie. no anonymous access).  Then in the sources.list you can
specify the useranme and password like ftp://user:pass@server:port/.
You can do a similar thing for http access (see man 5 sources.list)

I don't know how to (or if you can) get apt to prompt for this
password interactively but the source is there... go hack :-)

Hope this helps,
