Re: logging in without root

Hey Rob.

If you're talking X logins (I'm assuming you mean this... and not an app
which uses svgalib), gdm/kdm (one of these) has options for an automatic
login.

Go and look :-)  I forget where it is, heh, just apt-get install 'em & poke
around man pages / config files.  I'm sure there's something in
/etc/X11/kdm/config (or something like that) which has a AutomaticLoginUser=
in it, as I configured a machine to do this on a BSD box a while ago.

Why use root?  I'd really recommend creating another user, as Richard
suggested.  You can chown / chmod files / devices, so certain users do have
access to these, if this is the concern.

Ryan Verner
