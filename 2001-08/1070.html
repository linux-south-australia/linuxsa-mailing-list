> I have been playing around with an 'embedded'-style x86 board. It is a 
> small-form-factor board (about 5.25") with all the mod-cons. Since it 
> has an x86-compatible processor, the biggest advantage is it has an MMU.
> 
> This has allowed me to build a script that can put a real Linux kernel 
> on there, so one can develop embedded stuff in a real environment, 
> including Kylix, test under a real Linux distro, and then build for the 
> embedded environment.
> 
> I am down to about 7MB for the embedded environment, which easily fits 
> on a 16MB flash.
> 
> However, I have to give the board back soon, and I want to continue
