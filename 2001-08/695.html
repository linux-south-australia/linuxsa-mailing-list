Hmmm, in the very early stages of the install, I don't think you are 
running init (as in /sbin/init).

The install loads an initrd, which runs the linuxrc script in the 
initrd. This does a few things, and pretty quickly mounts, using the 
loopback device, a root file system image from the CD which contains the 
second stage ...

Have you tried other CDs? Other Distros?
