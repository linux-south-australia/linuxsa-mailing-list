> My questions is how do i force the network card to 100Mb.
If you are forcing the driver to full duplex, one would assume that
you are forcing the card. The comments in the driver code insinuate
that setting this option is not necessary if you have an
autonegotiating switch. If you hard set the switch you would imagine
that setting that dull_duplex option is all that is required.
> We are backing up a linux box thru samba and getting heaps of errors
> so I'm making both the Cisco 1900 switch and the linux box 100mb
> Full Duplex.
A Cisco 1900 can't do 100Mb. It is a 10Mb switch. That could be your
problem.
> We had a problem a while ago and Cisco recommends that all servers
> and links to other switches be not set to negotiate.
Routers and switches, yes, but autonegotiate is fine for most servers
and workstations.
