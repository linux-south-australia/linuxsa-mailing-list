loopback mounted crypto file system!

Can anyone comment on the wisdom of using the
"loopback mounted crypto file system"?

This is mentioned here:
http://www.suse.com/us/products/susesoft/suse72/new.html

This item:
http://kt.linuxcare.com/kernel-traffic/kt20010112_102.epl#15
http://www.uwsg.indiana.edu/hypermail/linux/kernel/0101.0/1227.html
seems to hint at some sort of crypto in the linux kernel
running over the loopback device.

But I can't find any other references.

-------------------------------------------
'l'     00-3F   linux/tcfs_fs.h         transparent cryptographic file system
                                        <http://mikonos.dia.unisa.it/tcfs>;

would be good for laptops that get stolen.
