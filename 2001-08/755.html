Re: virus infects on boot sector

On 21 Aug 2001, Andrew Reid wrote:

> Boot off of a floppy disk and do a low-level format. Some BIOSes have an
> inbuilt function to do this.

I once heard someone mention something about sledgehammers and cracking
walnuts. Why not just use DOS FDISK? If the disk is still seen by the
BIOS, you can remove a virus from the MBR with the DOS command:

 FDISK /MBR

Obviously, you'd want to do it after having booted from a write-protected
floppy.

I can't remember whether Linux' fdisk has a similar function, but I would
guess it did. OpenBSD fdisk does, but that also overwrites the partition
table (bad).

Once you've done that to the boot sector, you'd probably want to use some
kind of anti-virus software to clean the rest of the disks (when I last
checked, most had some kind of AV system that could be run from DOS).

I don't think there's any need to do a low level format.
