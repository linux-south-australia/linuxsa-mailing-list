Your explanation of everything was good - I was aware of how G++ deals with
virtuals, but I'm sure a lot of ppl on linuxSA have learnt something new.

However, I don't believe your example code was "true c++".. As you commented
yourself, the strcpy() was bad. strcpy() is a C function, and I was saying
that *c++* functions are safer.

Try again with something more like:


#include <string>

class notSoCrap
{
        private:
		// Our internal buffer
                string notSoCrapString;

        public:
                void fill_buffer(const char *str)
                {
			notSoCrapString.assign(str);
                }
		void fill_buffer(const char *str, const int length)
		{
			notSoCrapString.assign(str, length);
		}
                virtual void output_buffer()
                {
                        cout << notSoCrapString << endl;
                }
};
