Re: Apache and Perl script

On 02 Aug 2001 15:25:40 +1000, Adam W wrote:

> Can't locate MIME/Base64.pm in @INC (@INC contains:
> /usr/lib/perl5/5.6.0/i386-linux /usr/lib/perl5/5.6.0
> /usr/lib/perl5/site_perl/5.6.0/i386-linux /usr/lib/perl5/site_perl/5.6.0
> /usr/lib/perl5/site_perl .) at smb2www.pm line 29.
> BEGIN failed--compilation aborted at smb2www.pm line 29.
> Compilation failed in require at /usr/local/smb2www/cgi-bin/smb2www.pl line
> 23.
> BEGIN failed--compilation aborted at /usr/local/smb2www/cgi-bin/smb2www.pl
> line 23.
> [Thu Aug  2 16:13:55 2001] [error] [client 192.168.0.1] Premature end of
> script headers: /usr/local/smb2www/cgi-bin/smb2www.pl

Doesn't that say it all? It's evidently not been able to find
"MIME/Base64.pm" and as such, has not been able to compile. The sequence
of events would indicate that failure to load "MIME/Base64.pm" has
halted the compilation process.

You'll need to build and install the MIME::Base64 module (which can be
found by using http://search.cpan.org/).
