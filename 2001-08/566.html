> I found similar problems with my installation of Mandrake, if you set up a
> network or Webserver when you run the installation.

> The solution I found , No guarantee's, was to In kppp select the following
> items/buttons   (Note this is all from KDE not the terminal)
> Start kppp select setup / select the account / edit / DNS page tab
> - check that "disable existing DNS servers during connection" is selected
> I've found this is all that is needed in most setups.
> Occasionally the following is also needed.
> from the same place as the above select the Gateway tab and check that
> "Assign default route to this gateway" is selected.
>
> I'm not sure why this works but perhaps one of more technically minded can
> let us both know.
