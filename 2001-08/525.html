Prakash!

> i want a sample running code in c to access any socket

Macmillan publish a book called "Linux Programming Unleashed". I think
they've got the source code on their site (http://www.mcp.com/) and that
book definitely has a number of socket programming examples.

You might want to subscribe to:

* http://lists.linux.org.au/listinfo/linuxCprogramming/

Furthermore, you might want to look at the source code to something like
"ping", "BIND" or any other program that is likely to use sockets (i.e.
most of them, just pick one and go find the code). If the source code
you've picked is written in C you can guess where the socket programming
is by searching (grepping) for words such as:

* "SOCKOPT"
* "net\.h"

Try any other function that pops up and is mentioned by "man socket".
