Depending on your budget, try ArcserveIT, not cheap as OSS goes, but works
very well.
Should support the autoloader ok and you don't need windoze clients,
the main interface is java based thru a browser, so its platform
independant.
 
We've been using it here on a smallish Samba server (30-40gb storage / 150
users)
for the last 12 months or so, only problem we've had was a compression one
in 
which Arcserve didn't disable software compression when hardware compression
was available, resulted in very corrupted backups & a few headaches for us
when 
we tried to restore some data..................once we turned off software
compression
it was fine (their support reckoned that bug would be fixed in updated
releases, we're
running v6.61 Ent)
