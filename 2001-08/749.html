Re: Finding a package that contains a program/file under Debian?

> Under RedHat (and Mandrake and TurboLinux), if I need to find the
> package that contains a particular file, and that package is not yet
> installed, I can use a short shell loop to do the job ...

  rpm -qlp blah.rpm

or on a bunch of files:

  find . -name '*.rpm' -print -exec rpm -qlp {} \;

> What facility exists under Debian to do the same job?

You can do a similar thing with:

  dpkg --contents blah.deb

or run auto-apt which does it all for you.
