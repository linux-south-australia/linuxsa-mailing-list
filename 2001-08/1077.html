[OT] Review of running Windows on Linux

ZDNet has an interesting review of 
alternatives for running Windows on Linux.  

"We called on five Windows-on-Linux solutions--VMware GSX Server, 
VMware Workstation, Win4Lin, Wine, and WinToNet--to strut their stuff 
for us. With an emulator (or, in the case of Wine, a Windows 
compatibility layer), you can maintain the Linux foundation while being 
able to call up Windows applications as needed. And WinToNet takes a 
different tack; it's a Windows NT/2000-based product that can serve up 
Windows applications to any client with a browser--including Linux. 
Although each of these packages has its own set of quirks, none is 
terribly difficult to use, and all tend to work to varying degrees, 
depending on what you're looking for. 
"
