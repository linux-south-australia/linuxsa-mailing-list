RE: Kickstart Installation Question

Davis, Harvey wrote:

> Does anyone know how you specify, in a kicksatrt script, to install
> everything EXCEPT certain packages?

I think you're outta luck; there doesn't appear to be exclude package
option (although having recently looked into Anaconda, the changes
required to add it wouldn't be that significant).  You could use the
"select group of packages" function to get the majority, then select
the non-excluded remainders, or attack the RedHat/base/comps and
build your own component groups.
