you may find that console-apt is nicer than dselect, although the interface
is still odd (I don't know what it is with Debian developers and their
lobotomised interfaces -- they are both non-intuitive and unique... It seems
that every time someone designs an interface for debian (well, on dselect
and console-apt at least), they come up with a key for everything, which
happens to be different to every other program that ever existed... if only
they could decide what to use then use it... puh!)...

anyway, using console-apt, hit '/' for search, and then type 'task', and
then 'n' to go to the next item that matches (sounds like some other
programs, right, except that you can't search backwards, as far as I know...
puh!)... most of these "task-*" packages are empty packages that just
install a set of packages...
