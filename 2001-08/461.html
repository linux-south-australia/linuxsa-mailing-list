If the terminal emulator in question is not actually gnome-terminal 
(which I, at least, have always found uncomfortable), but an xterm, rxvt, 
or aterm, then fonts can be configured for them using the $HOME/.Xdefaults file.

In mine, I have the following lines:
*font:          lime
*boldFont:      lime

They configure the fonts for any application (hence the '*') which reads 
from the .Xdefaults file to be the 'lime' font.

For more information, especially with regard to the X resources that can 
be configured, read the manual pages for the terminal emulator in 
question - the ones for xterm(1), aterm(1) and rxvt(1) each outline the 
different resources that can be set for them.
