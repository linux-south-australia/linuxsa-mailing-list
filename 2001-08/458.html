Re: Linux (Debian) WebMail

Hi All

IMP can be configure to talk to as many different Pop/imap hosts a you like.
The long problem is that by default the package allow uses to set the host
and protocol, this is a risk because any one could use the server to talk to
any other mail server.

This is easily fixed, by editing the defaults.php3 and servers.php3 to lock
the server list down
