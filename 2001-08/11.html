Re: acls

Fülöp Balázs wrote:
> 
> hi
> i wanted acls. and there were acls. it works fine under linux. so if you
> ssh to my host, you can set acls to your files. but how can i ask samba
> to set acls? so how can a windows user set acls on a linux share?

* Compile Samba 2.2.1a from source
* Read the various threads found on this topic in the Samba mailing list
archives (www.samba.org)
* read the Samba manuals.

Apparently, Samba supports a number of different ACLs on a number of
different platforms. The two Linux based ones that I know work are the
EXT2FS patch and XFS.

You should find it quite simple to get something working without too
much hassle.

Hope it helps,
