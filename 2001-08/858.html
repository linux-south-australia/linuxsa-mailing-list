michaeld@senet.com.au wrote:
> I have a 2 CD set of debian here with me.  I'd like to know what .deb 
> on the CD
> provides the "iwconfig" command, that is, the command needed to 
> configure 
> wireless networking.  (without installing everything on the CD first) 

AFAIK, do it by CD.

cd /to/the/place/where/all/the/groups/are
for f in */*.dep
do
   if [ "`dpkg --contents $f | grep 'what-you-want'`" ]; then echo $f; fi
done

or something like that will do it :-)
