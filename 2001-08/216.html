Hi All,
    Yes, but YOU have to make it do that, it isn't a "built
in" function. If you set up the appropriate listener ( and
all the other event stuff) for the event you want to copy,
cut or paste the text then your Java application will do
that part of it, then you need to find out where the
clipboard buffer is under whatever desktop you are using and
copy the selected text there. Pasting is the opposite.

    If you know someone with a copy of Borland JBuilder
(version 3.5 and above I think comes with a Linux version
free, version  4.x certainly does) then the Tutorial is how
to build a text editor (bloody good idea if you ask me) so
have a look how they do it there. Hope this helps.
    Cheers!
        Andrew.
