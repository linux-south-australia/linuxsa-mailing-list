Well, seeing as we're only wanting to use the last 255 addresses, the
subnet of 255.255.255.0 is more correct than 255.255.0.0. The only
reason that you'd want to use a subnet of 255.255.0.0 is when you have a
network that needs to use more than the 255 addresses provided by the
class C.
