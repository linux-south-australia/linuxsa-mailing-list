Re: Tail Recursive

Hi Daryl,

Tail recursion occurs when the call happens near the end of the routine.
eg

float recurseFunc(float x, int n)
{
    if(n==1) return v;
    v = recurseFunc(x*x,n/2); // this is at the "tail"
};

the if statement tests the condition that makes the recursion "unwind"
after reaching the base state.  ( i used a function similar to X^N)

me some code to see if I can help :)
