TeX-borne linux viruses?

While hacking away at some TeX and MetaPost and
stuff like that, it's occurred to me that the linux
world (and the rest of the world) should be about
as worried about TeX-borne viruses as about
e-mail borne viruses, or Word-file or PDF-file viruses.

Has anyone heard of an actual TeX-borne virus?
Since every TeX file may contain write-commands
(like \openout and \write) to any file in your file
system, surely that means that a devious person could
write code in TeX to overwrite your various executables
with viral code.

It occurs to me that the only reason this might not be
very frequent is that almost no one in the world
uses TeX or LaTeX (or MusiXTeX etc. etc.).
The people who do use TeX-based software tend to
be urbane, civilised, erudite, sensitive souls
who wouldn't dream of stooping to virus-writing.
But if TeX had taken off as the world's favourite
document preparation system, I'm sure the world
would be awash with TeX viruses by now.

So.... has anyone heard reports of this sort of thing?

Cheers,
Alan Kennington.
