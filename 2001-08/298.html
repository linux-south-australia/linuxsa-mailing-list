Disclaimer: I'm still using 2.2r0 so some things may
have changed:

The big difference to what you have been using is that
Debian uses DEB packages not RPMs. They work in a very
similar manner, except Debian has a tool called
dselect  (equivalent to GnoRPM except that I
personally makes it a lot easier to navigate whats on
the system)
I fyou want to install an RPM you have to convert it
to debian using a tool called "alien".

The core Debian system is also based on pure open
source philosophies so depending on where you got your
distro from you may be missing Netscape & Java.

Some etc files have different names or are in
different places from RedHat. 

There is a tool called modconf that makes it a bit
easier to install modules into the kernel.

Also (as of 2.2r0) I found some patches to the kernel
didnt work properly and I ended up using the standard
kernel instead of the Debian disto'd one
