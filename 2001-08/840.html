> Does anyone know what the MINIMUM RAM needed to install RH 6.x is?
> None of the documentation that came with my box set of RH 6.0 says.

the HCL:

http://www.redhat.com/support/hardware/intel/62/rh6.2-hcl-i.ld-2.html#ss2.6

says:
----
16 MB minimum for the text install (requires 32 MB swap space). 24 MB or 
more is recommended for better performance and for the GUI installer. If 
you wish to run GNOME and Enlightenment or KDE, it is recommended that 
you have 48 MB or more RAM. 
----

RAM could, indeed, be your issue...
