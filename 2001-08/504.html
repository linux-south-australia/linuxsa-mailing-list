Re: [Somewhat-OT] ssh clients for windows

On Mon, Aug 13, 2001 at 11:17:08AM +0930, Toby Corkindale wrote:

> I use TeraTerm + SSH, and I prefer it.. One of the reasons why is because it
> supports the port forwarding, and X authentication forwarding.. So, if you
> absolutely have to use Windows, then you can run Xwin32 + TeraTerm + SSH,
> and still run your linux apps off remote hosts.

Another interesting piece of software that I was shown by Mark Newton on
the BUGA list is Mindterm. I've downloaded and set it up at
http://www.plug.cx/mindterm/

Essentially, it's a Java SSH client that supports port forwarding and a
whole bunch of other cool stuff.

You can use it from anywhere, simply because it works in any
Java-compatible browser. I assume it'd also work in the Applet viewer
included in the JDK, but I've not checked.
