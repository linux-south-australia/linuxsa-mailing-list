Hi

You will need to use setserial to set the base address and interupt.

something like

setserial /dev/ttyS2 port 0x3e8 irq 5

from the spec of the card I would say that its an isa which mean it probably
has jumper on the board which allow it to be set to a irq and serial
no/port.

Remeber that in linux the serial ports start at Zero so DOS serial port 1 is
/dev/ttyS0 :)

Ocne this is done use the program pppconfig configure you connection, then
type pppd <link name>
