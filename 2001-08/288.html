Re: Redhat 7.1 and Win2k

Nathan Millhouse wrote:

> Michael,
>         this will work fine, but the win2k boot loader cannot boot linux.

You can boot linux from win2k boot loader, provided that you didn't install lilo
in the MBR, and after installing linux you do something like

dd if=/dev/hdxx of=/bootsect.lnx bs=512 count=1   ,where hdxx is the partition
where you boot linux from,

and copy the file bootsect.lnx in win2k system partition, and add the entry in
boot.ini like c:\bootsect.lnx="linux"

Just in case, there is a HOWTO about this procedure.
