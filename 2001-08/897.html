Re: Gigabit Routing?

John Edwards wrote:

> One more thought, your switches may be able to restrict the
> mac-addresses seen on a given port, in that case you might want to make
> sure that only known Server mac-addresses are seen on the Gigabit ports.
> That might reduce excess broadcast traffic from workstations, and stop
> any P2P abuse of the network that might occur.

Everyone must be busy today, because no-one's pointed out the flaw in
the above yet..

The servers need to find a way back to the clients.. which can't happen
if you apply a hardware address filter on the Gigabit ports.

I think it will work if you're able to put it on the switchports for the
local network, so that data only reaches the 'server network' if it's
destined for it. Either way it will require that your switches can do
something intelligent with mac-addresses.
