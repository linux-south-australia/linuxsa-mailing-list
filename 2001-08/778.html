I'm also using a Dynalink TV card that also goes under the description
of Magicview I think. Works perfectly with Mandrake 8, and programs
like xawtv.

I found with early versions of Mandrake that I needed to manually run
the modprobe command on the bttv module with the identifier for the
card. Mandrake 8 with a 2.4.8 kernal finds it automatically though.
If you have the kernel source installed, have a look at:

/usr/src/linux/Documentation/video4linux/bttv/CARDLIST

and see if a card model matches yours. If so, try reloading the bttv 
driver like so:

modprobe bttv pll=1 card=24

This tells it to use PAL mode, and card number 24 from the card list.

Finally, make sure you have the sound output from the TV card plugged 
into the line input of your sound card. Most TV cards only work this 
way.

Hope this helps. TV cards under Linux aren't quite as smooth as they
should be just yet.
