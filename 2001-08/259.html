Re: Debian Question

Debian shall rock thy world, but before you install it :
1 make sure you know how to operate the package management system: apt-get,
apt-cache (ie apt-* )
2 get together a list of apt-get sources (to store in
/etc/apt/sources/)(special debian urls to download updated packages) before
you install it ... I dont know how many times ive had to scratch around the
net for local debian (ie .deb) mirrors, wishing that id gathered them with
my previously more operational install :)

Actually, I suppose only #2 is important if you intend to get the latest
packages, otherwise you would take them off the CD(s).  Personally, I prefer
to do a very basic install (fs, file system structure, kernel and boot), and
then manually install everything with the package management system.  This
way I have complete control over what gets onto my system without having to
use dpkg [a program that analyses what packages you have installed and
presents a list of them, along with all the other packages it knows about
and lets you select things to install/uninstall] - which isnt the best
package selection system ive ever used :0/
