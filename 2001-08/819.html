Jay,

No the Terminal Server stuff in Win2k is a blatant rip of Citrix
Winframe. I believe (but don't know as fact) that part of the License
agreement Citrix took with Microsoft when they developed Winframe, meant
Microsoft got the Citrix technology. This was back in the days of WinNT
3.51, the next release of WinFrame was still using the 3.51 interface
because Microsoft brought out WinNT 4.0 Terminal Server Edition, which
was basically WinFrame with the new NT 4.0 interface. Of course, I might
be completely wrong, it was some years ago...

As for SMP, unless anyone can definitevly remember, it might pay to
troll through kernel.org to find the first SMP kernel.
I do know however, that many OS's, even in the Intel sector were
supporting SMP way before Windows of any variety (Netware springs to
mind, but there were others).
