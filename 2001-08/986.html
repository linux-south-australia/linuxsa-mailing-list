> My problem is that i have set up SAMBA, and the user home directory shares
> are operating from the /homes/home/$username  directory, is there a way
> of root mapping the users share to the /$username rather than
> /homes/home/$username  ???? (i have had a fair bit of experience with
> netware not linux so sorry if i found a bit vague on my termonology)

If you are running Samba 2.0.7, you will have a copy of the book 'Using
Samba', which is a pretty good reference.

When you create the share(s) (I'm assuming from your mail that you are
using the [homes] share to share out the home directories). When you go to
connect to this share from another machine, the path you need to connect
to is:

  \\servername\sharename$

The actual file system path (on the linux machine) where this share points
to is the same as the user's linux home directory.

Hope this helps,
