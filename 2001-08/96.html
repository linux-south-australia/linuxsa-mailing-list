RE: Kickstart Installation Question

> Does anyone know how you specify, in a kicksatrt script, to install
> everything EXCEPT certain packages?

I don't think there is a mechanism to do that, but you could simulate it by
installing everything, then removing certain packages... alternatively, (and
I don't know enough about kickstart to know if this would work), build a
list of packages to install by using grep -v <package-you-don't-want> on the
complete list of packages (built with find or ls or whatever)...
