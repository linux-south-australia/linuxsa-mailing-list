On Mon, 6 Aug 2001, Gamunu Dassanayake wrote:

Hi,

> I have a problem to copy  some files to my web server using FTP command.
> The problem is that since LINUX is case sensitive, I have to go and
> adjust each and every page.  How Do I settle this problem ?

Many ftp programs (ws ftp for example) has an option to force everything into
lower case as you ftp. This might help you.

> 
> I have a another thing to know.
> 
> Assum that I  have two servers. One is Lunux & other one is WINdows NT. I
> want to put main page (index.html) to linux server and other pages to
> WIN-NT server. Is it possible to do ? How do I do my links in index.html
> file ?
> 
> Is theer any reference sites, pl let me know.

Three options spring to mind
Run Apache on both the linux server and the Windows NT box. Refer to the other
server with links from the index page (assuming both have full 'net access).
Use an outomatic ftp program (eg ncftpget) to transfer the files automatically
to the web server if access from ordinary users via an Win Nt network is
necessary.
Use smbmount to mount the web directories of the WinNT server to the LinuxBox
and directly reference them.
