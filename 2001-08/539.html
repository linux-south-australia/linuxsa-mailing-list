On Tue, 14 Aug 2001 rob wrote
> Hi,
> Im trying to compile a small  curses program and
> when running make I come up
> with the error
> gcc -o  iqf.o -lvga -lcurses
> /usr/bin/ld: cannot find  -lvga

Looks like the library is missing or incorrectly
installed.

If you installed a library not using dpkg then you
probably need to update the linker system:

add the library to /etc/ld.so.conf ;
and run as root:
ldconfig
