"A thank you to Microsoft"
Thank you for disabling Java in IE6 (according to
http//www.anfyteam.com/ms_protest.html).
As a dedicated Netscape user who has been unhappy about the tricks used
by Microsoft to make IE the dominant Web browser (specially having
FrontPage create Web pages only IE browsers can read!), I can only
applaud this decision. IE users may now see Microsoft for what it is --
a monopolistic company which will do anything to kill off the
opposition. However, competition keeps companies healthy.
The sooner Microsoft is split up into two companies -- one supplying the
operating system and the other supplying the software which is used on
that OS -- the better. Maybe then Microsoft will become healthy again
and start to build a future for itself and the billions of Microsoft
product users throughout the world.
Until then, Microsoft will continue to erode and the cancer it has
created will be the cause of its own death.
Graham Pockett
