Hi -

I dual boot win98 (work requirement :( ) and linux, in an arrangement that looks like this:
/dev/hda1 / reiserfs noatime,notail 1 1
/dev/hda3 /win vfat defaults 0 0
/dev/hda5 swap swap defaults 0 0
/dev/hda6 /home reiserfs noatime,notail 1 2
/dev/hda7 /data vfat umask=0 0 0  # So I can share data between linux and 'doze

Windows went berserk and mangled the filesystem on /dev/hda3 beyond recognition. So it seems I need to backup, partition from scratch, and then install linux and 'doze in that order. The linux boot still works perfectly and I really don't want to have to reinstall it.
