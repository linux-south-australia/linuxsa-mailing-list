Hi There,
    If you go to the RedHat site, there are a whole heap of
Redhat specific howtos and help sheets and tutorials. Trawl
for the one about mail servers. The biggest problem you will
have is setting up sendmail correctly, but last time I did
it, I found a setup script on Freshmeat.net. Just search for
configure and sendmail. It is a script that sets up all the
hard things automatically, like mail relays and such like so
you don't get used for spam and such things. You will need
to have fetchmail installed to actually run the script, but
you will not need fetchmail at all once sendmail is set up.
The Redhat Tutorial from the web site will tell you how to
set everything else up. Hope this has helped.
    Cheers!
        Andrew.
