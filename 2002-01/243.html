Re: Serial port cards + MS outlook-like software

Hi There,
    The only thing you should have to do is set the serial
port speed etc. This is done with the setserial command. The
exact syntax eludes me at the moment, but there is a man
page for them. The drivers are default built in to the
system. If you have one serial card, then you have drivers
to cope with 4 serial ports. The only time you *might* need
special drivers is if you are installing one of the super
serial cards with 8, 16, 24, pick a number of ports. These
cards use special interrupts etc to work and so need special
drivers. Hope this helps.
    Cheers!
        Andrew.
