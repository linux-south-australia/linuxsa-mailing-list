Another distribution which offers something similar to Sorcerer (which 
I've not investigated much yet) is Gentoo.  It uses an advanced ports 
system called `Portage,' which has features such as dependencies, fake 
installs, config file protection, and USE variables.  It seems to be a 
very modern and interesting distribution - the only thing stopping me from 
giving it a burl is the lack of a .au mirror.
