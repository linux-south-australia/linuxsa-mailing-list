Re: Requirements for NAT?

For what its worth, a few years ago I setup a linux gateway for some
people that was:
* file serving
* dialling up via 56k modem
* providing NAT and firewalling
* providing DNS and nmbd services to local machines
* providing web interface for configuration and dialup/hangup

This was running on a 386sx-16 with 4 Mb of ram.

It worked fine, and could happily shuttle packets around to suit a 56k
modem. File serving wasn't terribly fast, but for small files it was fine.

Ye olde 386 power :)

TJC
