RE: "find" frustrations ;-(

Sounds like you're trying to cd into the directory that contains
pkgtools.conf...

Try just:

cd /usr/local/etc

;)

Alternatively, and this is certainly not the easiest way, but it works
(for me)...

cd `sudo find /usr/local/ -name pkgtools.conf | sed -e
's/pkgtools.conf//'`

Of course, if there are two pkgtools.conf's in /usr/local/, then you
will end up cd'ing to the first one...
