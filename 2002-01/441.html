Re: Blocking Spammers.

>> There are various solutions.. One is to always give out your email as
>> realusername+fnork@your.isp.com.au
>
>Provided your ISP runs Sendmail and has plus names enabled (and a
>delivery agent that copes with it).  Qmail uses '-',

In practice, Qmail works with any substring that is more specific than any
other substring in the users/assign file. For some reason the documentation
mentions '-', which is probably a good habit to get into anyway.

Without it, users with short names will end up with the spam from longer
versions of a similar name if those longer names are removed from the system
later.
