RE: su ??

Your problem here is in your path -- I'm 99% sure that su would exist on
your system (at least, if it is a "standard" install of some
distribution), but you can check with 'locate su', and if it does, it
should be in a path somewhere... You may need to chift it into a
directory that is in your path, or add the directory it's in into your
path -- although, if it's in your path by default when you are in a
shell, then the KDE version of the program must be getting its path from
somewhere else... You may need to read man pages and even source code to
find it...
