See what you get with lpc status, mine looks like this
~$ lpc status
lp:
    queuing is enabled
    printing is disabled
    no entries
    no daemon present
psp:
    queuing is enabled
    printing is disabled
    no entries
    no daemon present

for both these printers lp (the boring one) and psp (poscript printer)
printing is actually disabled so prints jobs will only be queued (because
I don't actually have a printer).
Type "lpc start all" and "printing is enabled" will displayed and hopefully
things print.
