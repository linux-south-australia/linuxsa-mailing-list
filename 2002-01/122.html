Evan Bourlotos wrote:
> 
> Sorry for this but I have noticed a few people lately have been trying to
> run glquake etc using the nvidia cards under linux and have given up in
> frustration they were trying Linux I guess but they let one bad experience

> and installing things.
> 
> Anyway, I was wondering if anyone has managed to achieve this and if so are
> there any specific issues to be aware of?
> 
> Most people seem to be using Rh 7+ or MD 8 I have nfi about the nvidia
> cards as I run a voodoo2 and it works for me :D I just read the how-to
> which is old and 3dfx specific. So I thought I'd ask someone here
> 

GLQuake (quake with opengl support) requires glide in order to work.
Glide is a library which is specific to 3dfx cards (voodoo1, voodoo2,
etc). I doubt you can get it running on a nvidia card. My recommendation
is try quake3 if you have a nvidia card.
