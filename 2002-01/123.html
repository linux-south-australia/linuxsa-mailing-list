Hi Evan,

I have the GLX drivers working with both a nVidia TNT and TNT2 card.  I
know of people that have used them with success using GeForce2 cards.

The trick seems to be to use the nVidia AGP driver (built into the
kernel module - use rmmod agpgart before starting X) and also to
recompile the kernel module on the machine it will be used on.

Oh, all the successes that I know of have been with a 2.2 kernel.  This
is not to say that it wont work with 2.4, I just don't know of anyone
that has used it with 2.4.

Martin
