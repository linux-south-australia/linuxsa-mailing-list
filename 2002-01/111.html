Try the TIS (Trusted Information Systems) Firewall Toolkit:

http://www.fwtk.org/fwtk/

There is even some doco about setting it up under Linux.

When I set it up about 6+ years ago it contained an authenticating FTP proxy 
server. IT was a bit convoluted to setup, but seemed to work well once done. 
It is not transparent to Windows users (ie it has no knowledge of NT domain 
logins etc), but tools like ws-ftp can handle it easily. It can also we used 
with the command line FTP client as long as the correct syntax for proxied 
FTP connections is used.

When I looked at it last the TIS FWTK was free as in beer, but not as in 
speech (ie code available at no charge, but not GPL). This may or may not 
have changed. Read the license (if you can find it) carefully to ensure it is 
OK to use in your circumstances.
