Sometime ago I had an interesting encounter with Windows XP Pro. I was
trying to install it on my laptop, that just happened to be conncted via
my LM box to the Internet. I just didn't take off the PCMCIA card. I had
a sniffer (tcpdump) running on that LM machine.

Sometime during the installation I noticed that my laptop is trying to
obtain an IP from my LM box. XP was "Copying files, blah-blah-blah...",
but the machine was looking for a DHCP server using its NIC hardware
address.

I don't have a DHCP server on my LM machine, so I waited to see what
will happen next. The laptop OBTAINED an IP FROM MY INTERNET
PROVIDER!!!! I didn't know it was possible. And it sent something to
another IP. I saw with my own eyes on my LAN traffic from 64.*** to 209.***
