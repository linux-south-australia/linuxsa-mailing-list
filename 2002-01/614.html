Re: Sawfish not running

Hi Sue,

If you can start a terminal (maybe using Programs / Utilities / Terminal
from the gnome menu), then you may be able to get more information about
why sawfish hasn't been able to start by starting it manually.  Just
type "sawfish" and press enter once the terminal has started.

I would expect that you would get a couple of error messages at this
point, and these messages will help greatly in finding the problem.
