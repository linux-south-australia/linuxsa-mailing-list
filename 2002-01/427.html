Re: Request for tips on keeping distros up-to-date

In the immortal words of David Fitch <davidf@parachilna.com>...
> there's the problem: rpms!
> one of the reasons I switched to debian.
> I run "apt-get update ; apt-get upgrade" and it goes and downloads
> and installs the latest versions of all the packages that I already
> have installed.  No more dependency problems, apt-get works it all out
> for you.

Or for those of us using *BSD, portupgrade -a works rather nicely as
well, for source isntallations and portupgrade -a -P for binary packages
if available.
