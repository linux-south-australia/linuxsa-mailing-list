Re: Linuxconf

Martin Stacey wrote:
> 
> Has Linuxconf been removed from Red Hat 7.2? If so, what has it been
> replaced with?
> 

The console ncurses driven linuxconf is deprecated and no longer
included with redhat anymore, although it still ships with the gnome
version gnome-linuxconf which you can run from an xterm.

To administer servers etc... you need to use the chkconfig command to
set what starts at various run levels(do chkconfig --list), and/or use
the service command (as in service --status-all); or you can just use
the ntsysv tool.

I'm not sure what redhat will replace linuxconf with, if anything.

Hope this helps.
