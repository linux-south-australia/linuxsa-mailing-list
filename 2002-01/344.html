Re: IMac CDFS file format

Axel Lingstadt wrote:

> I'm unable the read a CD which has been burned on an iMac. RH7.2 does 
> not recognise the CDFS file system. The given file system is not 
> recorded as format in the mount man pages either.
> 
> Has anyone an idea how to extract the data from the CD?


Perhaps have a look at the mount(8) and mkisofs(8) man pages for some 
clues about various file systems that exist, where they're from, and how 
you might read them.

If it's not a standard ISO9660 CD-ROM file system, I'd try mounting the 
CD as an HFS or UDF file system.  As I understand it, HFS is commonly 
found on Apple Macintosh systems, and UDF is a format used for packet 
writing, commonly used on CD-RWs and DVDs (I could be wrong).

Is "CDFS" a file system type?  Could it be synonymous with ISO9660? 
What told you the CD was CDFS?

Do file systems have some sort of header or code indicating their type? 
  Can the file system type be easily deduced?
