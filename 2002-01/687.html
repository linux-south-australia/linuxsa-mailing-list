Re: ext2 to ext3 or reiserfs conversion

Hi Jason

Converting ext2 to ext3 is very easy (on redhat 6.2) after compiling a new
2.2.19 kernel with the ext3 patch and installing the upgraded e2fsprogs.

run the command on say your home partition.
1. tune2fs -j -c0 -i0 /dev/hd?? (hda1)
2. then adjust the /etc/fstab file to indicate ext3 instead of ext2 for the
home partition.
3. reboot
4. check if you home partition is mounted if it is then you in business and
can repeat the above for all partitions.

If you need to I can make a 2.2.19 kernel with fast IDE and ext3 available
for you to download.

Cheers
