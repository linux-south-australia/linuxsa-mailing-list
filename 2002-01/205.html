Re: Reinstating lilo

Quoting Sam Pohlenz <retrix@dingoblue.net.au>:

> Hi,
> 
> I am currently running a dual boot of Windows XP and Debian (sid), using
> LILO as the boot loader.
> I recently reinstalled Windows XP (overwriting the MBR), and unfortunately
> didn't make a boot disk, assuming that I would be able to get back in by
> booting off the CD.
> However, I found that this isn't working as my Debian partition is running
> ext3.
> Is there any way of getting back into the partition? All I need to do is
> run
> 'lilo' with my current /etc/lilo.conf.

ext3 is backwardly and forwardly compatible with ext2 so you _should_ be able 
to boot back to the patition using any bootable Linux CD which supports ext2 
(Red Hat, Debian, Linuxcare boot disk [my favourite], etc etc etc).  My guess 
is that Red Hat 7.2 even supports ext3.

If none of that works, get a hold of Tom's Root/Boot disk.  One floppy is all 
you need to get back your beloved Linux box.

And while you're at it, nuke Windows eXPlode :-)
