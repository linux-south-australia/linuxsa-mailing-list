Re: Linux system -> image -> linux system

I have had some experience using GHOST to manage hard drives with multiple 
OSes and partitions. Ghost runs under the MSDOS  command line so you can 
boot from a M$dos or Win9x boot floppy. GHOST has support for 
networking  and CD burners so you can choose to burn the image or store it 
on a networked computer. I am not sure about how it will handle reiserfs it 
works well on ext2. Unless you do the whole disc you lose the MBR and will 
have to rebuild the lilo or whatever boot manager you use if it is not in 
the partition boot record. You can restore a partition to a different size 
partition assuming the contents fits. I found GHOST very slow with non M$ 
partitions.

Bill
