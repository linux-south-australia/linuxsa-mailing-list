Re: php and apache

On Wed, 2002-01-23 at 14:51, thanhnhan... wrote:
> nik, I have already installed the package of apache and
> php4 and mod_php4 
> in /usr/local/etc/apache/httpd.conf
> have content these line .
> LoadModules  php4_module  linexec/apache/libphp4.so
> AddModule  mod_php4.c
> 
> ...
> ...
> ScriptAlias /php/ "/usr/local/www/php"

If your using the module you shouldnt tell apache to execute it. The
module will work that out on its own. Plus, if you chmod +x the file
then you'll get the error you mention as it trys to execute the file
rather than internally interpret it.

Take this line out, chmod -x the file, and then it'll probablty work.
