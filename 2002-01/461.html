Re: "find" frustrations ;-(

Wilkinson,Alex wrote:

> Can anyone see what I am typing wrong here ?
> 
>>sudo find /usr/local/ -name pkgtools.conf -exec cd .. {} \;


It might be necessary to shell escape the braces like this:
sudo find /usr/local/ -name pkgtools.conf -exec cd .. \{\} \;

or

sudo find /usr/local/ -name pkgtools.conf -exec cd .. "{}" \;

You might also be able to achieve a similar effect with:
find ... | xargs ...

or

find ... && cd ..

(or something more elaborate based on the return value)
