Re: Linux Text File Viewer

> Hello Linuxers,
> 
> We have a need to view text files on a Linux system similar to the way 
> LIST does it for DOS systems.
> 
> The features we are looking for are:
> 1. A "Sliding" virtual view of the file if it is wider than the screen.
> 2. Being able to Arrow or Page Up/Down up and down through the file.
> 3. Ideally, the viewed file should NOT be able to be altered.
> 
> Does anyone know if such a tool exists for Linux?

mc does 1 (F3 to view a file, then F2 to not wrap lines) and 2, but
not 3 (you can use F4 to edit a file instead of F3 to view it).

vim does 1 (type ":set nowrap" or set the environment variable EXINIT
to "set nowrap") and 2 (set scrolloff=999 may give you closer
behaviour to LIST).  3 can be done if you invoke it as "view", but
that can be overrided by editing the file then doing ":w!".

To prevent 3, is it an option to remove write permission from the
file?
