Re: X

> hi!
> 1: how can you use the s3 x server, and change colour depth on the fly?
> i currently have a script to copy over different  config files.

Provided that your config file has all the different colour depths you
want to use defined, use the '-bpp n' switch to X.  For example, when
using startx, do:

startx -- -bpp 16
(the '--' means "pass the following arguments on to X")

> 2: how can i choose a default screen res when i load x?

Create a .xserverrc file in your home directory which contains one
line that looks like:

exec X -bpp 16
