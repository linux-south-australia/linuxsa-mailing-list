Re: Cant get terminal

Bill,

Sounds like you have an rc script that is not exiting.

Once it has started various services they are free to operate 
within the system.  But the console isn't finishing
running all the rc scripts.

I would try modifying /etc/rc.d/rc to emit a message when
it starts up or brings down a script.  I am assuming a somewhat
normal sysv start up routine.

I don't have a linux box handy, so I can't tell you what you would 
most likely have to do.  But look for somewhere where the script
loops through a number of files starting with 'S*' and then 
before it starts the script put an echo line to print output
of the variable name.

If you have a problem, mail me your rc file and I show you where
your modifications should be.

Cheers,
