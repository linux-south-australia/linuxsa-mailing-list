Re: swapon

> Hi
> I have just started playing about with Linux. The installation went
> smoothly however, swapon doesn't seem to run automatically. I have got a
> swap patition but need to type swapon /dev/hda3 after logging on. Where
> does the swap command go? I tried it in the /etc/rc.d/rc.local file but it
> returns command not found. Using an older version of Linux, nowhere to put
> Redhat 5 just yet. Any help would be greatly appreciated.
> Cheers and beers, Kevin.

Somewhere in your /etc/rc* files there should be a command which will
turn on all known swap partitions:
swapon -a
(or perhaps /sbin/swapon -a).

swapon figures out where your swap partitions are by looking in
/etc/fstab.  You should have a line there that looks something like:

/dev/hda3  swap  swap  defaults 0 0
