Text editors for Linux

I've installed RedHat 5.0 on an old 486 with a 200Mb hard drive so that I
could have a look at the Linux O/S. To start working on the Linux machine I
need a simple text editor similar to TextPad - my favourite Windows text
editor.

I looked at vi, Emacs and Joe. For someone brought up on Windows wysiwyg
software I found vi and Emacs just awful. vi was just too difficult to use,
and Emacs is almost as difficult as vi and it's size [16Mb] takes up far too
much room on my limited hard drive. I don't have the time, and I'm not
willing to invest the effort needed to master these editors. I just need a
glorified typewriter.

Can anybody recommend a simple, efficient, easy to use, wysiwyg, text editor
similar to TextPad, UltraEdit, Super NoteTab, etc that will run under X. I'm
happy to pay [TextPad only cost me $US27] but I don't need the full
Applixware suite, as I already have Office 97 on my other machine [NT 4.0
workstation]
