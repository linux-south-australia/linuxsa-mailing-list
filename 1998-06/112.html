UltraPenguin

Okay,
So I had 20 minutes to spare today, so I had a Sun Ultra 450
sitting unloved temporarily by my desk, so I had to do it.

The Machine: Sun Ultra 450 (1x250Mhz UltraSparc II/1MB Ecache,
                            256MB Ram, 16GB disk, 3 UW SCSI)

I didn't have time to do an install or anything interesting,
but it does boot the UltraPenguin Kernel across the net.

It gives a lowly 495 BogoMips.

[Unfortunately, the machine I wanted to put it on, which is
 similar to the above, but with 2x250Mhz, 1GB Ram, 5 UW SCSI
 controllers and 20 4.2GB UW disks has already been installed
 as our proxy server .. ]

FYI: UltraPenguin 1.x is a release of Sparc linux based upon
Redhat Linux 5.0 for the Sun Ultra series of machines.

(UltraLinux is http://ultra.linux.cz)
