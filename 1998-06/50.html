By the name of the package, I'm assuming that you are using RedHat.

According to my system, the xpm-devel package provides the file xpm.h,

$rpm -q -f /usr/X11R6/include/X11/xpm.h
xpm-devel-3.4j-1

You will also need the xpm package.
If there is anything else missing that either of these packages depends
on, rpm will tell you when installing the package.


Hope this helps.
