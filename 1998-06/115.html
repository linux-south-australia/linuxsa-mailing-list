Linux Vs Others

distributed.net run a little code cracking game online, some of you might
have heard of it.

They have a list of the operating systems used by the users that are
playing this 'game'.

They are rated in order of blocks they process in the game ... while
windows 95 is miles in front ... its good to see a large number of "unix"
type operating systems being used.

You will be happy to know that LINUX is 3rd out of many :)


Go have a look :)
