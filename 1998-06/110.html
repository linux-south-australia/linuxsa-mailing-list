Okay... this makes more sense.

The problem you ran into is that 'rpm -Uvh *' asks rpm to install
everything that it can find in your current directory, but you weren't
in a directory with any RPMs.  The RPMs are located in
/mnt/cdrom/gnome/RPMS/i386 and /mnt/cdrom/gnome/RPMS/noarch.  If you
want to install everything, you could do:

cd /mnt/cdrom/gnome/RPMS
rpm -Uvh */*.rpm

but I would suggest following the instruction in the README file in
the /mnt/cdrom/gnome directory which says:

To install GNOME just run the ./install-gnome script.
