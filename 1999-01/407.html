The rpm file has all the files inside it.  The rpm file is not 
copied anyway, the files are just extracted.  To find out what files
have been extracted from the rpm file you type the command 
	
	rpm -ql <package name>

So in you case the rpm file is still on the CD :)

To find out the rpm file name (in case the CD-R software changed the
filenames) type in...

	rpm -qa | grep XFree

And it should return a number of lines with something like 

	XFree86-libs-3.3.1-14
	XFree86-3.3.1-14

To view what is in each package, type in everything up to the version number.
In the above example do a 

	rpm -ql XFree86

which will present you with the listing of files in the XFree86-3.3.1-14
package.  To view one page at a time, do a 

	rpm -ql XFree86 | less

To find out what rpm a file has come from type in 

	rpm -qf

followed by the *complete* file name.  For an example, to find out what
package /bin/ls is from type in

	rpm -qf /bin/ls

Which will return something like 

	fileutils-3.16-9
