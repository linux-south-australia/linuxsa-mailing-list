Wojciech ,

Try a different host.  When I first set up my machine I used
ntp.adelaide.edu.au on advice from this list.  However, on the
following day it was down so I tried my ISP's address which worked 
and I've been using it ever since.  Don't expect just any old host
to respond -  to provide a decent ntp service, the Sys Admin has to 
set up some time keeping software and run a server.  Oh, and BTW, 
are you behind a firewall? maybe its not letting the ntp protocol 
through!

If you need confirmation or otherwise that your intended host will
respond, post (or email me) the host name.

Hope this helps,
