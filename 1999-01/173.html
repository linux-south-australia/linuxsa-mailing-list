Spell checker for Vim

Ive been looking for a spell checker that works with Vim, when running Vim
on NT/Windows95, as well as Linux

Dr. Charles Campbell sent me this URL. Ive found the file very useful and
thought it might be of some interest to the Vim users on this list. It also
works well as the user dictionary when using the Ctrl-X Ctrl-K commands. I
think it has more words than the linux.words file.

I did have a few problems getting the \ep map working, if you have similar
problems send me an email and I'll pass on my work around.

+++ Begin post from Dr. Charles Campbell +++

<engspchk.vim> works under Vim -- that is, on any platform that runs vim,
so will engspchk!  (English spell checker)

You can get a copy of the Vim-based English spell checker via the web from

    http://www.mv.com/users/kgg/engspchk.vim     <- plain text
    http://www.mv.com/users/kgg/engspchk.zip     <- zip'ped
    http://www.mv.com/users/kgg/engspchk.vim.gz  <- gzip'ped

    (thanks to Kenn Goutal for the webspace)

After getting <engspchk.vim> installed someplace, while in Vim, just
    :so (whatever path)/engspchk.vim

Mis-spelled words are highlighted as Error; you may put your cursor on any
such word and enter it into a permanent user dictionary (ie.
usr_spchk.vim) or just into the temporary (this-session-only) database if
you feel that its spelled correctly.  These deeds may be accomplished via
two maps in <engspchk.vim>:

 \et : put word under cursor into database (temporarily - ie. just this file)
 \ep : put word under cursor into database (permanently)
       Requires CVIMSYN to be an exported environment variable pointing to
       directory holding database: $CVIMSYN/usr_spchk.vim
