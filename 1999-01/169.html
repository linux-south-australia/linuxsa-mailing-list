Re: Cron

Hi

As root, you need to edit /etc/crontab and add a line like this:


30 23 * * * root cat /var/log/httpd/access_log | mail -s "Web Server Log 'date'" root

That's it. Cron watches the crontab file and any changes are active as
soon as it is saved. the '30 23 * * *' are the time/date fields for
when the job will be run, this is set to 11:30pm. man 5 crontab for
details...

Michael.
