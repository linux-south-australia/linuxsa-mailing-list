I'd reccomend reading the mouse MiniHowto. On the few Linux boxes I've set
up, I've found the specification of the mouse is wrong. One problem was
fixed by speifying the correct /dev device then the correct sort of mouse
(not what the install program said), and the other was fixed with the
mouse autodetect program (Its in Redhat, I don't know whether it is in
Slackware). Turns our the lable on the serial port in the BIOS was wrong
in that case.

GPM and X works fine in every case I've ever seen.
