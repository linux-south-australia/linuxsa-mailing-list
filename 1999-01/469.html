First, you have downloaded a .src.rpm. This contains the source code
for the application and requires building. There are (AFIK) no 
executable parts of the package except for (maybe) a script or two
to patch the source tree and update whatever files need updating after
the installation.

To install this package you will need to use
'rpm -b kdethemes-2.1.0-1.src.rpm'. You will probably need to specify
some options to rpm to build it, so look in the man pages, or try
http://www.rpm.org 
for more information.
