Re: palmtop and handheld "conduit" software for linux

> Question:
> Can anyone tell me of any software they know of that will run on
> linux/PC for exchanging files down a serial line to a
> handheld PC (windCE 2.0, for instance) or a palmtop?

Try running an ftp server on your wince machine once you have a PPP connection
going between the linux machine and the Wince machine.  This is not ideal but
you will be able to get files on and off the wince machine from linux.  A simple
script would enable you to do basic syncronising or you could you fmirror (FTP
mirroring program which comes with the standard Debian 2.0 distribution).

You can find a a good how-to for connecting a wince machine to a linux machine
at
http://home.utah-inter.net/clalor/linux_ce.html

and you should be able to find an ftp server for the ce machine at
http://www.cemonster.com/internet.htm
