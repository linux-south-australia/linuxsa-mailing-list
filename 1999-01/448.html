On Thu, 28 Jan 1999, Darryl Ross wrote:

One I use is called XFmail - it's not absolutely perfect, but it does fit
the bill - and even better, I'm able to still use Pine as well (so if I'm
not using Xwindows I can still read mail without upsetting anything). 

I have occaisonal crashes with it, but I've never lost any mail to it -

hard on the eyes).
