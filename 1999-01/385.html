Personally i would just put a socks5 server on the linux box

mirc runs through socks
browser runs through socks
icq runs through socks

most stuff will run through socks and it can be set up quite securely
with access lists and ipfwadm blocking external access to socks port.
Also everything is logged :)

probably better off setting up socks that masquerading IMHO :)
