I have a Matrox Millenium AGP (8Mb) G200. About a month ago I installed 
Slackware (Version???). However, this distribution only came with XFree86 
3.3.2 (??). This doesn't support the G200 (or so I believe). I downloaded 
and installed (at least I tried to - don't know if I did it right or not!)

XFree86 3.3.2 probably does have support for the older Matrox chipset - so 
if you're using that version of X and selecting the Matrox Millenium - but 
you actually have a G200 (or G100) - you're not selecting the hardware that 
you actually have and you'll probably run into problems.

the video RAM ..... I took to directly hacking into the xconfig file (yeah,
