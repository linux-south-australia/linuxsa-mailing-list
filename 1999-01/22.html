Hi Simon,
My understanding of using FIPS is that you need to load it on your own
boot/rescue disk that you hopefully created when you installed Win 95.  I may
be wrong but I dont think you can use someone elses rescue disk and to use
FIPS you need to boot with the rescue disk and then use FIPS which you have
copied onto it.
If you need more info the Linux Pocketbook has excellent instructions, if you
don't have access write me I'm happy to help.
Hope this is useful, Gordon
