Re: Spell

> Since installing RedHat 5.1, the Spell button cannot be used to check
> the spelling of an e-mail before sending it. Even selecting Tools from
> the menu, is missing from there as well.

You didn't mention which package/etc you are trying to run 'spell' from.
Most console applications will use the application 'ispell' to do their 
spell checking.  

To check to see if you have the ispell package, use 'rpm -qa | grep ispell'.
If you don't then install it from the RH CD.

If it is another application then we need to know what application it is
and we should be able to get it going.

> Even though is a very good practice to make sure you spell the words
> right, it comes handy to have a spell check.

Always is, but if anyone has seen some of Rasterman's (of Enlightenment fame)
emails you know it isn't always the case :).
