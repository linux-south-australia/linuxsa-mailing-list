You need the Mesa OpenGL libraries installed to run Wine now. These
are available from:

ftp://rufus.w3.org/linux/powertools/5.2/i386/Mesa-3.0-2.i386.rpm

Install this package before you install Wine 990110 and you will have
no problems.
