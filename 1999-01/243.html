Re: download limited

Adam Crawford wrote:
> 
> Ive just got a new net account unlimited time, but a 200 meg download limit,
> I was wondering if anyone knew of a way to monitor my downloads on this
> account, I use multiable inet accounts and these dont have limits on these,
> so would need it to monitor this account only, im currently using a ppp
> script on ozemail's web page

I had this problem too. Try ipfwadm to set accounting on the ppp 
interface for that account.

Try 'ifwadm -A -i -W ppp0'
where ppp0 is the ppp interface name for that account. To check the
counters, use 'ipfwadm -A -l'. This will account for traffic in both
directions. There are options for rules to look at incoming and out-
going traffic separately. 'man ipfwadm' for more info.

You will also need to use some sort of script to save and restore the
counters on shutdown/boot otherwise they will be lost.

The easiest option is kppp (from KDE). It will run in any window
manager (just an annoying little icon floating around in a tiny
window because of no taskbar) and provides accounting for both in
and out traffic as well as for call costs and otherwise. It is
pretty easy to set up and resembles the windows95 dialler in look.
Good for the beginner who needs a quick and dirty fix for this
problem.

Hope this helps.
