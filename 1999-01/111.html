Re: GNAT Ada Compiler

Hi All,

    As far as I know, GNAT is a version or variant of gcc or g++. We have
been using these programs to write ADA at uni for years without any problems
on Sun Boxes. From what I have been told gcc and g++ will also compile
Pascal, Fortran, C and C++ (of course) and a few others, so you should
already have these on your Linux distribution. There was a new compiler
shipped with Linux in RH5.1 apparently and this may also do the job. The
name of it escapes me but it is mentioned in the docs somewhere.

    Cheers!
