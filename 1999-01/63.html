Re: X question

Ben Kramer wrote:
> 
> I have a couple of linux boxen and I want to run pgramms on one but
> display it on another in x.
> ie run netscape on big box, but display on little box..
> can some one point me in the right dirrection ?
> Ben.......   :-)

i did some stuff with this:

in an xterm on the little box, type
  xhost +[hostname/ip of big box]
so my system would be:
  xhost +bigbox
if your big box's name is "bigbox"

then on the big box (or a telnet to it)
type:
  nxterm -display littlebox:0
presuming name of little box is "littlebox"
and an nxterm will pop up on the little box.

other programs also accept the -display thing, but i think it is
possible to:
  set DISPLAY="littlebox:0"
on the big box's xterm before running an app and it should come up on
the little box.

Hope this helps
