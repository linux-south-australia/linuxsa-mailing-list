Re: Portscan Attacks

it's best to do...

#deny all services
ipfwadm -F -p deny
ipfwadm -I -p deny
ipfwadm -O -p deny

# flush all commands
ipfwadm -A -f
ipfwadm -F -f
ipfwadm -I -f
ipfwadm -O -f
ipfwadm -M -f

# then only let in and out the ports you want e.g
ipfwadm -I -a accept -b -P tcp -S 1.2.3.4/12 1024:65535 -D 0.0.0.0/0 80
...etc.
