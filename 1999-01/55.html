Re: Changing video resolution

> I've read in Mark Sobell's "A Practical Guide to Linux" and the "Linux
> Pocketbook" that I can change the display resolution in Linux by
> pressing CTRL-ALT-KEYPAD+ or CTRL-ALT-KEYPAD- That doesn't work on my
> machine - can anybody tell me how to get this working?

You need to add the other Modes to the "Modes" line.  The modes you
can have there are defined in the Monitor section.

> Section "Screen"
>     Driver      "accel"
>     Device      "Trio32/Trio64"
>     Monitor     "My Monitor"
>     Subsection "Display"
>         Depth       8
>         Modes       "1024x768"    <-- add other resolutions here
>         ViewPort    0 0
>         Virtual     1024 768
>     EndSubsection
>     Subsection "Display"
>         Depth       16
>         Modes       "800x600"    <-- or here, depending on the
>                                      colour depth you use
>         ViewPort    0 0
>         Virtual     800 600
>     EndSubsection
> EndSection

Personally, I don't think the feature is very useful; it changes the
physical resolution, but your desktop will stay the same size.
