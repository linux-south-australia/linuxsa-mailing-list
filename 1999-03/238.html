Forgot to mention in last post that "usermod -P[encrypted pw]" will also

"useradd" and "usermod" are not supported by EARLY versions of the shadow
passwd suite (it would need to be from a package off an early dist CD I guess,
because I have never encountered the problem with downloaded versions of the
package).
