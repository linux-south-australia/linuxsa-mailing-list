RE: X windows & video card

You'll need to get hold of XFree86-3.3.3 or newer.  I've got the same video
card (onboard SiS 6326 w/ 8M) and xfree works perfectly using the SVGA server,
but with  OPTION "noaccel" in the XF86Config file to disable acceleration,
otherwise it don't work.

The key thing is to get the NEW version of XFree86.  Chances are you'll
currently have 3.3.1 or so, which is the one that comes with redhat
5.2/pocketbook.

You can probably get the new version from ftp://mirror.aarnet.edu.au/, which
is my online resource for pretty much anything :)  There are a number of files
to download, so read the readme file.

Good luck :)   If you need any more info, give me a hoy, because I was exactly
where you are several weeks ago (ie at a loss) and now have a gorgeously
working 1024x768 X.
