The parallel device drivers have changed with 2.2.x of the 
kernel... and by default arn't configured 

The following worked for me...

In 'make xconfig' (or whatever),
under [general setup] you need to enable [parallel port support]
with [PC-style hardware]. (Choose to compile them as modules)

Under [Character Devices], enable [Parallel printer support].

Then (repeat after me):

make dep; make clean;
make
make modules
make modules_install

and edit /etc/conf.modules to include
alias parport_lowlevel parport_pc
options parport_pc io=0x378,0x278 irq=7,5

or similar (I have two parallel ports installed)
see: /usr/src/linux/Documentation/parport

Then..  run modprobe -a
(not completely sure about this step.. I think I just rebooted the
machine which had this in it's /etc/rc.d/rc.local)
