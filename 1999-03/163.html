Re: Linux on IBM233MX

At 12:37 11/03/99 +1100, Kevin wrote:
>Matthew Mills wrote:
>> Hi all, I am intending to upgrade my 486 DX4-100 box to a VXpro
>> motherboard and place an IBM 233MX ($82 new!) chip in it.  I would love
>> to get an AMD or an Intel chip but I just cant justify to my wife the
>> money spent.  What I would like to know is is this a worthwhile
>> expense?  or is it not compatible with Linux Redhat running the 2.0.36
>> kernel?
>
>Good price and should work fine. That is, I had no problems with the same
>hardware

Same here, we have more 686 boxes than Pentiums and (with the exception
of the chips with 75MHz bus speeds, eg: 150/P200+ = 2x75Mhz), they all run 
very reliably. You would want to avoid the n x 75MHz chips anyway, as
motherboards don't always support it. So the 233MX should run like a gem ;-)

Also, this could just be my paranoia, but I would suggest running a 
heavy duty CPU fan on them as they can get quite hot with standard fans.

*NOTE: This was under Slackware but I highly doubt that the dist would
make much difference in this matter.
