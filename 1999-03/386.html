Re: linux login restrictions faulty?

On Tue, 23 Mar 1999, Alan Kennington wrote:
> How else do you get an account to be ftp access only?

Assuming you use an ftpd which consults /etc/shells, that file
lists all valid login shells.  Ftpd only permits ftp access for
users with a valid login shell, so if you give them a shell which
does nothing, for example /bin/false, and put that in /etc/shells,
they can ftp but not usefully login.
