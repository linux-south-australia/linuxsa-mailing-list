You definately have a refresh/timing problem.

There is a way to fix it as well :)

We must delve into XF86Config...

First and VERY important are the VerticalRefresh & Horizontal refresh
rates. Dig out that old monitor manual, blow off the dust and find the
correct settings. 

Now in XF86Config there is many lines starting with: 
Modeline "somerezhere"
Workout what rez you want to run at the comment out ALL modelines execpt
one. Preferably one with a low refresh rate

Ie:
Modeline "800x600"   50 800 600 .....
                     ^Refreshrate 

Now start X - you should get a working X setup. 
Time to fine tune it - xvidtune will help here.

At this stage it is VERY VERY VERY VERY important you have you
Hor/Vertical refresh rate correct - if you don't do the following steps!

Click auto,
Now adjust the screen how you like it - ie strech it, pull it, flatten it
Once you are happy - try changing consoles CTRL+ALT+F1 the do CTRL+F7 if
the screen hasn't jumped on you click SHOW.

This will put the modeline in the console. Copy this in to XF86Config,
comment out the other line and your done!
