Thanks for your input.

Eventually I went onto the Corel support newsgroup and tracked down the
answer. It was quite simple actually but it took quite some time to track
down.

If youre using a printer driver for a non-Postscript printer as I am
(specifically the Canon bjc600) then you can't send it to the Postscript
printer (-Pps) so that it goes through Ghostscript. It has to be sent to
-Plp which doesn't use a filter and by adding the -l to the Lpr options in
the destination dialog in WP8 Printer Control this tells lpr to print no
matter what file it is.

So now WP8 prints in colour in my Caldera OS.:-)

Thanks again.
