RE: Library files not loading

You get the same problem trying to upgrade Java to 1.1.7 or better.

The solution is to get the latest XFree86 kit, which includes libXp (for
printing).  You can hack your way around it and not upgrade to the latest
XFree, but you will increasingly run into this problem as new software comes
along - just take the plunge.

Unfortunately, upgrading to the latest XFree86 is non-trivial
