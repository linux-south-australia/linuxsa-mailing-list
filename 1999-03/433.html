Re: Top stats emailed via cron.

Check if you Linux has the command unix2dos and if it does do the following

top | unix2dos | ..... the rest of the command line you are using.

If not check out www.mag-sol.com/Articles/index.html which gives a
description of the problem and a perl script to handle the files for you.
