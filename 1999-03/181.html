Re: X gateway

Alejo Acosta wrote:

 > I don´t know if I made my self clear.
 > What I need is a X gateway that runs on Solaris, and when it receives a
 > packet from the PPP connection of that workstation, it reroutes this GUI
 > packet to a determined PC.
 > Hope this time I have been more clear.

Go to ftp.tis.com and get the firewall toolkit (fwtk).  There's a 
program there called "x-gw" which does what you want.

Alternatively, you can use X tunnelling in ssh:  Login to the
Solaris box then, from there, login to the system you want to run
X clients on;  the X client calls will be passed back over the
encrypted session ssh sets up and end up on your display.
