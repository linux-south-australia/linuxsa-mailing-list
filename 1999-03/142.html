Well, firstly, harrass them a bit about their lack of knowledge etc, and
then explain to them that PAP and PAP-secrets is the same thing.
PAP-secrets is just the file you need to store the passwords in, under
linux, for use with PAP. If you've only got the 1 ISP you dial, you'll
just need an entry like * * * yOUrPaSSword inside the file.
then make sure theres a 
user yourusername
in the options or appropriate peers file.
Once you've done that, just dial up as per normal and it should work for
you.
With PAP you don't need to get a manual login/password, so your chat
script just needs to initialise the modem and dial in.
If you read either the PPP HOWTO (see /usr/doc/HOWTO) or the pppd manual
page (`man pppd`), it can explain in more detail and give you accurate
examples.
