Re: linux+linux

Wong Jen Choong, Jimmy wrote:

> Hi,
> I have 2 linux machines.  One contains all my users and the second is a
> newly setup machine.  Is there any way that I can allow users to login
> into the first linux machine and be able to access the 2nd machine.
>

Yes. Set up hosts.equiv on the second machine. Then users on the first
machine can use rsh rcp etc to acces sthe second machine.

Alternatively and safer, use NFS to mount those parts of the second m/c
you want available on the first.
