Re: Questions

RPM is telling you that you cant install that particular package (GNOME) cos
it needs to have libraries and stuff from another package. In this case its
"xscreensaver". What i would say is load up your redhat cd:
mount -t iso9660 /dev/cdrom /mnt/cdrom
cd /mnt/cdrom
cd RedHat
cd RPMS
ls |grep screensaver
rpm -i *screensaver rpm name*

This should install the appropriate dependancy's and installing gnome wont
be a problem :)
