Re: Masquerading under 2.2.x

My box is Masqurading under 2.2.x thru Ip-chains..
Ipfwadm no longer works for ip masquaring, so you need to grab a copy of it.
I just got the rpm from www.freshmeat.net.
A great site that has plenty of info for masqurading under 2.2.x is
http://www.tor.shaw.wave.ca/~ambrose/
they have the latest version of the IP-Masqurading HOWTO.
Anyway.. what worked for me was:
ipchains -P forward DENY
ipchains -A forward -j MASQ -s 192.168.1.0/24 -d 0.0.0.0/0
For the old ipfwadm.. and these lines for ICQ masqurading (demonstrating the
new autofw code)
ipmasqadm autofw -A -r tcp 4000 4011 -h 192.168.1.24 -i
ipmasqadm autofw -A -r tcp 5000 5011 -h 192.168.1.25 -i
btw - autofw, portfw, mfw are now part of "ipmasqadm" - a replacement for
the old forwarders. I think this is included in the ip-chains package.

I think Ip-Chains is HEAPS better then the old ipfwadm code, but others are
sticking with 2.0.x masq'ing.
Its got some nifty new things in /proc/net/ip-masq. The only bummer is i
dont think netstat -m tells you all masqurading connections. Next version eh
guys? :)
