RE: e-mail clients - Try Pegasus

Hi there

Pegasus v3.01d is a free eMail app. It can be configured for multiple clients. One nice feature I like is that mail can be selectively downloaded. I know that it works fine in a Unix/Samba environment so should be OK with Linux. It has some close links with Netware but works fine without it.

The official site is http://pmail.usa.com

There is more information at http://www.caiw.nl/~jaabogae/han/p300_index.html and a mirror at ftp://pegasus.topnz.ac.nz/pegasus/
