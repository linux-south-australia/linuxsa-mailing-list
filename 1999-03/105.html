TCP port 10752

Question:
Does anyone know what TCP port 10752 is for?

Someone on a German site www.steckhan.de
who was trying to mount my NFS file systems
made an access to this port this morning:

10:46:03.960000 www.steckhan.de.22837 > rat.topology.org.10752: S 1861050474:1861050474(0) win 512 <mss 1460>
10:46:03.960000 rat.topology.org.10752 > www.steckhan.de.22837: R 0:0(0) ack 1861050475 win 0
10:46:14.610000 www.steckhan.de.22839 > dog.topology.org.10752: S 2678589231:2678589231(0) win 512 <mss 1460>
10:46:14.610000 dog.topology.org.10752 > www.steckhan.de.22839: R 0:0(0) ack 2678589232 win 0

But I can't find this port in /etc/services.

I'm sure I saw it in _some_ documentation recently.

I wonder if this is another port I need to block.
