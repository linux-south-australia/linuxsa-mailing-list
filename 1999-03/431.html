Hi Dale,
unlike in windblows, netscape doesnt come all setup with all the
'helpers' for different file types. You will need to add them in your
self with your favorite application for that file type.
go..
edit->preferences->navigator->applications
this is where you will find the different mine types. choose the one you
are getting errors from and then add some like htis in to the
application section
eg for real audio etc
rvplayer %s

good luck,
