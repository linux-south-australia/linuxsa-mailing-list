Net news Microsoft article

BIG BROTHER MICROSOFT
Microsoft today braced itself for a massive backlash over news that
Windows 98 
can secretly build up enormous databases of personal information. The
Windows 98 
registration wizard generates  a Global Unique Identifier, a number
which 
identifies the user's computer and Word documents created on it. It
works even 
if you have turned it off! Microsoft hurriedly posted an open letter
promising 
to modify the wizard immediately and to provide a software tool for
download so 
you can delete information from the Windows registry. Microsoft says
it's all a 
big mistake, but critics claim it's deliberate. The open letter is at  
http://www.microsoft.com/presspass/features/1999/03-08custletter.htm
