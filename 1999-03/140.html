RE: Cyrix 6x86 chip

I recently had the same issue :

New Cyrix M2 chip (300 Mhz on a via mboard).  Managed to boot the TurboLinux
CD (which is just RH 5.1 with some bits added), and found that LDLINUX would
not go any furthur than the prompt.

Solution was to take out the new disk, format it and load linux on it from
my other AMD K6 machine, and then stick the new disk into the Cyrix machine.
The Cyrix boots it fine. - strange stuff.
