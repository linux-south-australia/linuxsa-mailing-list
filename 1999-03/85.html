Re: linux+linux

"Wong Jen Choong, Jimmy" wrote:
> 
> Hi,
> I have 2 linux machines.  One contains all my users and the second is a
> newly setup machine.  Is there any way that I can allow users to login
> into the first linux machine and be able to access the 2nd machine.
In way do u want access?
You probably want to let them telnet in, or ftp, etc.
Possibly the easiest way is to just duplicate the /etc/passwd and
/etc/groups file.
Either by a samba or nfs share, or else just by ftpping it accross, and
then remembering to keep it updated.

Think about it - users get access based on their entries in passwd.
To allow users to access any given machine, they need two things:
1) The ability to be able to telnet to it, and for the inetd daemon to
let them in (tcp_wrappers,etc).
2) A valid username and password, stored in /etc/passwd (or shadowed
passwds, etc...)
