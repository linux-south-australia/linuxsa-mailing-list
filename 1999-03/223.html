> Hello everyone,

> Trying to install "kicq" form an ".rpm" comes up with the message that
> in the library is the file "icqlib-0.1.3 and libicq.so.0" are needed.
> What do I do, and where can I find them?

First you need to download the library. Go to http://filewatcher.org
and put icqlib*rpm in the search field.

Once you have downloaded it install it like any other rpm then install
kicq like normal.
