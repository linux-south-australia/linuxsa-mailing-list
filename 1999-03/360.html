Hey There,

First make sure that the hosts you want to send mail from can be resolved to
a name, either by putting entries in your /etc/hosts file or running DNS.
For example, all my machines can be resolved to the domain bachblue.com.au
so in the /etc/mail/relay_allow i have put:

/* Start /etc/mail/relay_allow */

bachblue.com.au

/* end */

And that should work. Hope it helps.
