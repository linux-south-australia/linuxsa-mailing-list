Fw: Setting a path

>Hi, Still learning the ropes and am having probs with installing apps in
>Linux in that they often get installed into /usr/local/bin or
>/usr/share/bin  which are both not in the standard PATH on a Redhat5.2
>installation, I have been searching on how to change my PATH, have tried
>ENV PATH=<paths appeneded here> but that only temporarily adds the path
>. There seems to be no SETPATH command under bash.
>Why is it that many/most apps install into a path that isnt listed in
>the path variables of a standard redhat5.2 Installation.


heres a quick solution.. edit /etc/profile

look at the path setting in there.. Just add in extra ones you one.

/etc/profile is a system-wide file.. itll be in affect for everyone.

or you could edit the /etc/skel/.bash_profile (these files get copied to all
new accounts)..making
sure that you update your already created .bash_profile's (in your home
dirs).

easiest thing though is to just edit the /etc/profile
