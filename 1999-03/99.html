Re: Cyrix 6x86 Chip

A friend of mine installed Redhat 5.2 off the Pocketbook CD onto a 
686PR200 system without any hassles. It didn't run as good as other
systems I have seen running Linux on Intel, but it did work without
any problems that either of could seel.

I believe that there are patches to the kernel specific to Cyrix to
enable the inbuilt cache or special features of that chip, something
like that. And also external program to do similar things and enhance
performance. Correct me if I'm wrong tho.
