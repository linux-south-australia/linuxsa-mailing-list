Stick with it Henry

Both lilo & grub are boot loaders (obviously) if you lilo.config then
run lilo -the next time you boot you'll use lilo - if you follw the
steps below the next time you reboot you will continue to use grub

To update GRUB you need to edit the "menu.lst" (It works very similar to
lilo.conf) file in /boot/grub/
Just copy the line that  has title linux & change the relevant details.
then when you've saved it go to the console & type   sh install.sh  
from the /boot/grub/ directory. This command writes your new
configuration to the master boot record of your HDD.
