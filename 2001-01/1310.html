Because I really care, I'm going to give everyone the exact, simple command
line that will work in 99% of RedHat 6.2 installations to fix this problem.

[as root]

# rpm -Uvh ftp://mirror.aarnet.edu.au/redhat/updates/6.2/i386/bind-8.2.3-0.6.x.i386.rpm

I'm not sure if you need to restart named or not..  just in case, I typed:

# /etc/rc.d/init.d/named restart

Everyone should do this ASAP, especially if they're in an educational
institution ;)  The bug is a *bad* one.

os's may vary.  E&OE.  Read at your own risk.  No responsibility taken.
