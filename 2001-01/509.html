Re: downloading probs

On Fri, Jan 12, 2001 at 10:42:06PM +1030, Robyn Manning wrote:

 > I'm having some trouble with the latest version of Mozilla, when I go to 
 > download a tar.gz file it downloads a file that is about 4 times the actual 
 > size in blisteringly fast pace ie 21kB / sec and the resulting file is 
 > unusable.

It's probably doing the same braindead thing Netscape used to do:  
Automatically gunzipping the file for you, but still naming the resulting
file with a .tar.gz extension.

See if you can manipulate the downloaded file with "tar" without using
gunzip first.

The blisteringly fast download rate would be because it's counting 
uncompressed bytes when it's working out how many bytes per second 
have been downloaded.
