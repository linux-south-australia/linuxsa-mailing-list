Hi all
Have had an interesting day.  I needed more PCI slots in my proxy server, so
thought the easy way out would be to swap m/b with the yet-to-be configured
mail server & all would be ok.

boot (the proxy server is setup with 2.2.17, /dev/hda5 root partition)
Errors as follows:
	
	.......................<snip>................
	Calibrating delay loop 1399.19 BogoMIPS
	Memory: 257292K/262144K available (1544k kernel, <etc etc>)
	Kernel panic: Attempted to kill the idle task

above.
However just after it mounts the root filesystem it comes up with the
following:

	......................<snip>...................
	Freeing unused kernel memory: 68k freed
	Warning: unable to open an initial console
