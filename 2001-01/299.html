RE: Mylex Raid and RH7.0

I presume you are talking about the Acceleraid 150.

If so, they work great.  I've just been setting up a RH6.2 server with this
card + 4 SCSI drives.  Its all autodetected once you've set the controller
up in its BIOS.
Make sure you pick 8gb disk geometry (the default is 2gb), also, depending
on how many drives you have & what RAID config you want, give yourself a
couple or more logical drives (do an expert setup in the BIOS).

Mylex have a nice server monitoring package for Linux as well, and there's a
couple opensource ones which are very good also.
(a dac960 module for webmin & a package called Varmon, check freshmeat.net)
I had lots of fun killing & rebuilding drives with these the other day.

I'd agree with Richard 100%, skip RH7.0.  This is the 3rd server I've set up
since 7.0 was released & all have been setup with 6.2.
