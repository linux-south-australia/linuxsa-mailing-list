> 4    (OFT) Where would I find information about setting up my modem
> and configuring Linux for the Internet plus setting up sound security.

Use the RedHat PPP dial up configuration tool, very easy. I use it
because I can't be stuffed setting up PPP scripts every time I screw
this machine up :) 

It's under the Internet menu in Gnome or you can start it with "rp3".
There is another command to configure different connections but it
currently escapes my mind.

It is based upon "wvdial", a easy to use command line PPP dialer which
you can also use if you prefer.

If rp3 is not installed, open up your RH6.2 cd and go to /RedHat/RPMS/
and type "rpm -ivh rp3-####.rpm"
But it should be there unless you did a Custom install.
