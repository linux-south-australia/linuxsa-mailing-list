Re: Samba logon script

Alex,

You need to add the following to your smb.conf file:

domain logons = yes
logon script = %m.bat

then make sure you have a netlogon share defined

eg

[netlogon]
path = /home/netlogon
comment = Network Logon Service
guest ok = yes
writable = no
share modes = no

Then write your logon script and give it a name matching your machine name
(in smb.conf, the %m gets expanded to the machine name. This lets you
define a logon script per machine)

For more information have a look at the smb.conf man page. Especially the
options like 'domain master', 'domain logons' and 'logon script'. If you
are interested in doing roaming profiles, then have a look at 'logon path'

Good luck!

Sam.
