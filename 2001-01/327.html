Type

  file fred.bin

where "fred" is the file you downloaded.  If it is an executable
or a shell file then you'll need to set the execute bits:

  chmod a+x fred.bin

and then do

  ./fred.bin

If it is a tar file, if can be extracted with

  tar xvf fred.bin

It might be compressed.  In that case run

  gzip -d fred.bin

and run "file fred" again.

It probably the file permissions being not set to execute.
If so, your KDE file manager executing it is a bug and
should be reported (I think KDE has a bug report tool).
