Re: Backup of Server

Hi.

I think I can comment on this, but I don't think the task can be done with
available tools. I think your friends best option is to either do as you
first suggested and netatalk a linux box as a file server for important
files, keeping a vanilla as possible Mac system on the iMac; or grab a
copy of retrospect and schedule a nightly backup via ftp to the linux box.
(actually retrospect ftp works very well, I used it on the last major
upgrade of my G3)

A virgin Mac system install is pretty quick, and there is not much to do
once you kick it off until you reboot the Mac with the system installed.
Your friend could have a spare drive on the shelf with the system on it,
just waiting for the failure..

BTW, what is he trying to protect against? Hardware failure is pretty rare
on the (ide) drives on iMacs...

Regards,

Michael
