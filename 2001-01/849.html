> Hi, i finally got an NIC working under my mandrake machine. I had two
Intel
> pro 100's in my gateway, so i swapped one for the SMC network card, which
is
> now used to get the cable connection to my gateway and the intel pro 100
> server adapter used to connect to my LAN. Anyway, enough of the details. I
> have got it all working, but i cant get an Internet connection working as
i
> dont know how to set the "default gateway" as you would do in windows.
> Do i have to use the route program?

Temorary:
route add default gw ip_of_gateway

> Or is there a file which holds this information?

Permanently:
add
GATEWAYDEV=eth0
GATEWAY=ip_of_gateway
to your /etc/sysconfig/network
