Yep.

I see you've installed RedHat 7.0 which means you'll probably have to
ferret in the text configuration file unless they've managed to roll
XF86Setup (a graphical setup program) into the distribution...it's worth
having a look.

> What do I need to do to get X to display in the correct resolution?

Well:

* try xvidtune (but don't set go overboard with it)
 - once you've worked out what your LCD display likes, then muck around
with Xconfigurator to set it permanently

* try looking at /etc/X11/XF86Config
 - back it up before you do anything
 - look for monitor lines, card lines and so forth
 - the documentation (I think it's in that directory too) should help
you
