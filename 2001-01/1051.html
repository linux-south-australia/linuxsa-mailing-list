RE: Ghosting off a unix server...

Try using the multicast server which comes with Ghost.
We have been using this to set up machines for the past 12 months or so, and
all our images are stored on a Samba box.
The multicast server uses TCP/IP so you dont have to stuff around with the
MS Client, plus it allows you to specify how many machines you want to do at
once, whether you want to load an image to a client or suck an image off it
(very handy).
The boot disks are very easy to configure, Symantec has downloadable disks
for most of the common PCI cards, all you need to set up is the ip address
on the disk.
