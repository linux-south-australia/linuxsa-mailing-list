Re: perl books / resources

For reference Programming Perl OReilly.
Aka the camel book.

It is the Perl reference book.
It is to perl what K & R is to C.

Ie written by the language's author and the standard language reference
guide.

For details man perlbook,


If you know  some programming already ie the concept ofa  variable,
iteratoin (looping) selection (if) then you may well not need anythign
else.

I can't make any suggestions for a more tutorial style book.
I would suggest that perl is probably not the best language to learn with
if you are a complete programming beginer.
