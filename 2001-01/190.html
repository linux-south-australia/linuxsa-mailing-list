Most of the packages you down load make things very easy:

  bust it out
  ./configure
  make
  make install

This does require that you have a bunch of things installed, like:

  gcc/egcs
  autoconf
  make
  bison and flex

and the list goes on. If you have the disk space you can install
everything, or you can sellect the correct option during install
(workstation) and you will get what you need.

As far as libraries are concerned, you need to be aware that the run time
loader uses a cache of libraries that is configured via /etc/ld.so.conf. If
you add new libraries, you may need to run ldconfig to get the system to
notice the new libraries.  If you need to pick up a new location, like
/usr/local/lib, then add that directory to /etc/ld.so.conf and rerun ldconfig.
