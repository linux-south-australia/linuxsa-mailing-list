> Does anyone have any expirience with Novell's NDS CE product?

> 
> Can and how do Linux volumes become/mount as Netware volumes?
>

I'd reccommend support.novell.com.au, its the best netware related
documentation and knowledgebase.

> How is security implementated on volumes
> 
> How it integrates with Samba for user security on files?

SAMBA is separate. SAMBA uses windows networking protocols. The file
security depends upon the config of samba, but seems to default to default
unix security rwx to user, default user/group.

I'm not sure what you mean by NDS CE product. Is this NDS E-Dir 8 for
Linux ? I will be looking into that later this year personally. If it
isn't then do you have a URL for the product ?

Netware volume mounting is available through a couple of utilities I
think. One comes default with RedHar (mars a netware 3 emulator), and
there is another one I'd have to check the name on. Its available on
freshmeat.

Hope this gives you a start.
