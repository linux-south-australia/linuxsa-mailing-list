Backup of Server

Hello All,

I was approached yesterday by a friend who works for a primary school
who wanted to know how he could backup his iMac 'Fileserver' to a Linux
box.

I initally suggested using netatalk and copying across the user data
each night, efectively keeping a copy.

This is apparently not satisfactory. If the server dies, he wants to be
able to go to the Linux Server, rip out the backup data hard disk, and
put it into the iMac.

I thought maybe using rsh to pipe the contents of the hard disk to the
remote one, but I then remembed that it needed to run on a PPC
Processor.

I know that distributions such as LinuxPPC and MkLinux exist, but I
wasn't sure how I would go about getting a small (TomsRTBT sized) Linux
onto a CD (as the iMac has no floppy) and booting up into Linux, so that
I could run a small shell script to do the rsh stuff.

I'd like to be able to give my friend some advice, but not having all of
the answers myself, I was hoping that someone from LinuxSA might have an
idea.

Your thoughts, suggestions and comments are all greatly appreciated.
