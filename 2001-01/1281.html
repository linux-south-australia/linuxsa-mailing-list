Re: Can Lilo work without Linux?

> Hi,
>
> I have a system with 2 HDD. One has Win95 (primary), the other Linux.
> Lilo is being used to boot between the two disks. I need to remove the
> second (linux) disk for a short time. I don't want to remove lilo from
> the first disk as I will be refitting the second disk back into the
> system later & want my linux system to work again!
>
> >From my understanding of the pc boot process, the boot sector is read
> in at boot time. Lilo is (in essence) pointing to the different
> boot images on the machine. So regardless if linux is installed or not
> on the second drive, I should still be able to boot to windoze by
> selecting "dos" from the lilo boot prompt from the first disk.
>
> I ask this because I have removed the second disk and now if  I try
> to boot from the first disk I get a string of 07 across the screen
> and the machine does not boot.
>
> What could the problem & possible solution be ?

LILO says LI when boot sector loaded and then trying to load rest from linux
partition and only then says LO:
So if read unsuccessful you will get error code running across your screen.
It means that Linux disk MUST present for LILO operate properly.
