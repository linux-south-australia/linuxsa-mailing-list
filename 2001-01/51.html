RE: Installing Packages

The answer to both your questions is no.  : - )
You can put an rpm file in any directory and install it from there
 	
rpm -ivh /anydir/packagename.rpm

	run "man rpm" for the full list of options
 
NB.  About the only time you would need to reboot after installing a package
would be if you upgraded your kernel with an rpm from Redhat/Mandrake or
whoever (tho generally I'd recommend recompiling your kernel from source, it
gives you a lot more control)
