Unplugging of HDDs can be avoided. Do the following way:

Add the following lines to:
--- cut here ---
other=/dev/fd0
    label = floppy
    map-drive = 0x80
        to = 0x81
    map-drive = 0x81
        to = 0x80
--- cut here ---

lilo -v

Then insert DOS boot floppy, reboot, on LILO: say "floppy" and pray :) If
you have good LILO system will boot from floppy and you can start to install
windows. When windows will need to reboot boot linux again and modify
"other" statement to "other=/dev/hdb1" and change "label" to
"label=windows". Don't forget to run "lilo -v". Then reboot, say "windows"
on LILO prompt and pray more. :) On next windows reboot you will need to say
"windows" again... And so on until you will have your windows going...

Fancy?! But works! :)

Windows stupid enough and will update MBR on drive 0x80 but it really will
be 0x81. So you will not loose your linux mbr and will have windows going.

Regards,
Vladimir

PS: Anyway if you can detach your HDDs - do it... Safer way in such cases
will help you to avoid some headaches...
