Re: Upgrading to 2.4.0

David Lloyd wrote:
> 
> (eg: ipchains for 2.2.X doesn't work any more)

Yes it does, type "modprobe ipchains" and proceed
as normal.

I supplied a patch to Red Hat for their ipchains
init script ages ago.  It follows

/etc/init.d $ rcsdiff -r1.1 ipchains
===================================================================
RCS file: RCS/ipchains,v
retrieving revision 1.1
diff -r1.1 ipchains
10a11
> #   -- modified for Linux 2.4 by glen.turner+linux@aarnet.edu.au
31a33,37
> if [ "$KERNELMAJ" -eq 2 -a "$KERNELMIN" -ge 4 ] ; then
> 	# Linux 2.4 doesn't use IP chains, backward compatibility
> 	# is provided with a module.
> 	MODULAR="y"
> fi
36a43,46
> 	    # Install module if needed
> 	    [ "$MODULAR" == "y" ] && 
> 	  	action "Installing IP chains kernel module:" modprobe ipchains
>

> 	# Remove module if installed
> 	[ "$MODULAR" == "y" ] && 
> 	    action "Removing IP chains kernel module:" rmmod ipchains
