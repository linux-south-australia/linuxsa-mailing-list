Re: Autostarting CD's

Michael,

> Is there a way of creating an AUTOSTART.INF on the CD that will cause it
> to load the index.html in the default web browser automatically on W9x
> boxes, AND create that .INF file on linux. I have the source for a Visual
> Basic solution, but no VB..

Just create a file called autorun.inf and enter into it:

[autorun]
open=\path\to\index.html

then copy the file to the root directory of the CD and that should be all
you need to do.

This should open the file in whatever the default browser is on the windows
machine (IE, Netscape, etc)

Good Luck!
