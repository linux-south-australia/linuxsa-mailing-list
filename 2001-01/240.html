RE: Coupla Questions

I've been using Reiserfs on my home PC since Mdk 7.0.  I'm now using it with
Mdk 7.2 & have had no problems with it so far.
(note, I have kept /boot as ext2 just cause I'm paranoid, but
anyway.......every other partition, /, /home, /var is Reiser)

We have a proxy server at work (RH 6.2 w/ patched 2.2.17 kernel) which runs
its 2 cache drives on Reiser & also its /var partition.  Went with Reiser
for performance reasons, not stability.  Have had no problems with that
either.

For a home PC, go for it!  Just don't keep any state secrets or LinuxSA
manifestos on there without backing up ; - )
