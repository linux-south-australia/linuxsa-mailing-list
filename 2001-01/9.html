Re: RedHat 6.2 Install by remote control problem

yeah that would be one of the well documented "features" of lilo, easiest is to
create a small partition named /boot somewhere inside the first 1024 cylinders
of the harddrive if at all possible. the rest of the partitioning is most likely
failing because of this. on another note, try cfdisk. not at all user friendly,
but powerful as all hell.
you may need to shrink the first fat32 partition so that, make sure its is
backed up somewhere!!
