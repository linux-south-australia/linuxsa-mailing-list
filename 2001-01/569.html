Re: Question from Newbie

Henry Barton wrote:
> 
> Hi all,
>         I've just downloaded the kernel patch for Mandrake 7.2 that offers
> support for win4lin from the pocketbooks website. The instructions  on
> extracting the tar.gz file seem simple enough to follow. However, they also
> give some instructions on reconfiguring lilo as well. My situation is that I
> am currently using grub as my boot manager and I am unsure whether l still
> have to edit my system's /etc/lilo.conf file  as per the instructions, or
> need to reconfigure grub or both. If grub needs to be reconfigured, could
> some one please let me know how I go about doing this. Sorry for the
> inconvenience but  I'm still very much trying to find my sea legs in this
> great ocean of things linuxy.

Henry,

you run either GRUB or lilo, not both, so don't bother with editing
lilo.conf if you are already running GRUB. See if you can figure out
what the changes of lilo.conf mean, and make the equivalent changes to
the grub config file (/boot/grub/menu.lst) and run grub. Check out the
GRUB man page (man grub) and see if it makes sense. Welcome to the sea
of uncertainty :-))))

Bruce
