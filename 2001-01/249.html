You problem most likely is that you do not have a host line on each machine
for the translation of IP address to a host name.
This is the problem I have had since day one with the LPR system, its very
old and does not use today's standard for connections

With out a host entry or a DNS resolving Address I have never been able to
print a file on another machine.

There is one exception to this and that is if I use rlpr (remote lpr)
