Re: Question Re Compilers

David Lloyd wrote:
> 
> This is going to sound particularly weird, but if I were to compile a
> program with the same compiler and same compiler flags would it create
> exactly (I mean EXACTLY) the same machine code each time? Or could each
> compilation be subtly different?

In a previous career into this when looking for covert
channels for a crypto project.

As long as you don't use any timestamp macros in the
C preprocessor or the source control system (ie: the C
code actually *is* unchanged) then the same assembly
code is output (ignoring assembler comments).

The link loader may create differing load images from
this assembler code as COFF and ELF loader formats
have optional timestamps and user stamps.  Whether these
timestamps are inserted in practice depends on the linker
and its options.
