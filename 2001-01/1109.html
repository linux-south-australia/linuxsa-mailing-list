> server and run ghost to dump a 2GB image onto disk? Basically I'm going to
> use a server to house desktop images and suck them off onto (up to) ten or
> twenty machines at once. Novell does it very well. I understand that you can
> even get a boot disk organised for WinNT and run ghost off the server. How

while 1
	dd if=/path/to/image.dat | nc -l -p 9999
end

then, connect to port 9999 on the machine and be prepared for the image.
