Re: Console Root lockout

Steve Sloan wrote:
> 
> I had to visit one of my remote Linux boxes the other day and found I
> couldn't login as root from the console.
> I normally use SSH and login as root with no problem, which still works.
> I found I could login as a user and su to root OK.

You might want to check the file /etc/securetty 

it should only contain
tty1
tty2
tty3
tty4
ttyn

root can only login to the system direct using the terminal listed in this file,
hence if the file doesn't ocntain these entries (or ones like it) then you can't
login direct as root.  Conversely, if the file doesn't exist, you can log in
direct as root from anywhere

Hope this helps
