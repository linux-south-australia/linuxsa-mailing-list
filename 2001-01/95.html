At 09:07 AM 1/5/01 +1030, Evan Bourlotos wrote:
>Steve,
>
>I had to do similar once, all I required was to get lilo managing the boot. 
>I did this by running a boot from CD then mounting my root partition then 
>running liloconf which put lilo back onto MBR. The exact process for this I 
>got from a friend and don't recall it but I'm sure the gurus out there will 
>know the exact process.

The process is relatively easy from most install CDs these days. At the
boot prompt from an install CD, type something like the following:

  linux single initrd= root=/dev/hda3
          |       |         |
          |       |         +--- Or wherever your root file system is
          |       +--- Note the space after =, says don't do the install
          +--- Single user mode is enough
