Re: determining IP address of client

> Is there any way to determine the IP address of a telnet client from the
> server ?

Yes.  You can do getpeername on the socket, which is file descriptor 0,
1 and 2.  The telnet daemon undoubtedly already does this, and probably
records it in /etc/wtmp, so "who" should show the IP address or name.
