Re: Samba

At 05:54 PM 1/15/01 +1030, Mick Collins wrote:
>Hello Once again. I hope I may ask a question?
>
>Here it is. I have changed the name of a domain and i got this message.
>
>[root@madness /root]# tail -f /var/log/samba/log.nmb
>  Get_Hostbyname: Unknown host madness.center.net.au
>[2001/01/16 01:45:08, 0] nmbd/nmbd.c:main(689)
>  Unable to get my hostname - exiting.

Samba uses the gethostname call to pick up your hostname. In one of your
startup files (you haven't told use which distro you are using), you are
setting your hostname to madness.center.net.au, but you do not have it in
/etc/hosts or DNS, so Samba cannot find out important info it needs, and dies.

>I have gone through the smb.conf file and i cant find anything about the
>domain problem. I was wondering if it was in a file somewhere in Cache?  If
>there is a thing called cache for Samba???

There sure is, a directory cache and a file cache, but this is not what
your problem is.
