Re: Reinstating NT boot sector

Lyndon Zimmermann wrote:
> 
> Greetings,
> 
> I was endeavouring to reinstall Linux on my home PC (already running NT)
> and Red Hat Linux 7 overwrote NTs boot sector so now I have a
> semi-functioning Linux install and no NT.  I tried reinstalling NT but
> this didn't help.  I need dual (or is it duel?) boot.  How should I
> proceed?
> 

This happened to me last week. My fix was to boot the system up with an 
msdos (actually win95) boot disk. I then did an "fdisk /mbr" then
rebooted 
into windows nt fine. After that I simply used a linux boot floppy to
get
me going with linux and followed the instructions in the "howto"
to get the dual boot system working.

I think I had to reinstall the service packs on nt.

The mini howto on Windows NT/Linux is good. You will find it under
/usr/doc/HOWTO/mini-howto (or somewhere to that effect).
