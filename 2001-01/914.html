Well I just got my son one for Christmas.

MSI mobo with VIA 694x Chipset running Intell PIII 800E's
MSI Video Card (NVidia Genforce 2Mx)

The only problem is to run ATA100 IDE Drives they need to be plugged into
the terenary IDE interface on the Motherboard.

Windows (ME & NT) will work fine and boot like this, but, Linux will not and
if we plug an ATA 33 into the primary interface lilo will not boot Windows.

Vladimir saved me the research and solved this problem for me this morning.
(Thank you Vladimir).

It's running RH7.0 with the 2.4.0 kernel (to recognise the ATA100) with a
small ATA33 drive on /dev/hda for Linux and an ATA100 40gig for Windows &
data on /dev/hde

I've put the following entry in /etc/lilo.conf

--- cut here ---
other=/dev/hde1
    label = windows
    map-drive = 0x80
        to = 0x81
    map-drive = 0x81
        to = 0x80
--- cut here ---

Your local Leading Edge shop should be able to supply.

BTW IMHO Quake3 runs better on this system under Linux that Windows ME.
