RE: A quick Question re win4lin

If you wish to go the promise way and you are not doing it commercially buy
an ultra100 IDE controller it will save you many $$$'s. This can be modified
into a Fastrak 100 without too many issues if you can solder (see
http://www.geocities.com/promise_raid/raid100.htm) you must have
ptiflash.exe version 1.09, the latest that comes with the controller will
not do it. I have a copy and have modified quite a few of these cards
successfully.

I have not installed it on linux yet, but a friend of mine has, and the
redhat drivers from promise are apparently zillions times easier to install
then the fastrak66 - he is a relative linux newbie and he had it all going
first attempt! - (my instructions to get the FT66 going runs into pages and
is a real prick)

For dual boot, use lilo.
