Re: Cant access internet :(

Hi, could be as simple as setting the default route... IF you are
actually logged in to ur ISP using PPP & you have been authenticated
using PAP or CHAP then probably all that is wrong is you havent set the
default route for your packets!

do an ifconfig - IF your ppp interface is UP it will show and you can
then do this --->

route add default ppp0   

then try pinging something on the internet
