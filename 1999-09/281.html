Re: Relaying in sendmail8.9

On Fri, 10 Sep 1999, SaiGKrish wrote:
> if I relay thru; this null client, it passes it on to the Hub. I dont
> know how to prevent this.

Well pretty clearly you either need to implement anti-relaying rules on
your null client, or not do *any* mail forwarding on it.  There's really
no reason why every machine should be a network mail agent just because
it can.  Choose which machines *need* to provide that function and turn
it off on all others.  (In practical terms this means don't run an smtp
daemon, neither sendmail nor anything else.)
