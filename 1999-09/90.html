Warm boot from Linux to Windows

Hi,

someone asked if was possible to warm boot into Windoze from Linux (such an
ugly thought).

A number of suggestions were made, including:

   lilo -R winblows; shutdown -r now

It was pointed out that this does a cold boot, not a warm boot.

It seems to me that a warm boot into WinBlows is not possible (at the
present) as  Linux must be shut down properly, and Linux does not know how
to chain to another program when it halts.

In truth, loadlin can only do a warm-boot from DOS. If you do it from
Windows (I have not tried) you are likely to take file system damage in the
process.
