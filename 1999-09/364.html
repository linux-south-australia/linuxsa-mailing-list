RE: Motorola Linux support

This is great news..

I went to a Compaq seminar last night on new equipment that they have and
will be releasing soon.
All of Compaq's primary servers and all new bits that go with them, are now
supported.
i.e. RAID controllers, video, Network cards, etc.

I haven't been able to determine if the cool stuff which is available in the
Compaq NIC's are supported yet.
Things like network card aggregation ( 100Mb + 100Mb +... ) to a common
switch mesh or automatic port switch over based on availability of network
link.
These features are available on the NT version of the drivers.

The next version of there Smart Start config and driver CD will include all
the Linux drivers.
They are currently available on Compaq's WEB Site.

As an advantage to me, being able to provide a Unix and NT environment on
the same platform saves lots of money.
Also when on site sparing is needed this enables us to minimise the hardware
we need to have on site to support both platforms.

The other option to me at the moment is Solaris on Intel which is very
expensive to purchase and maintain.

The good news just keeps coming.
