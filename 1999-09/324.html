It seems to me that you are asking about Windows Sockets, which are an
entirely different kettle of fish from Unix sockets.  Well, in many ways
they are different.

You will want to write a winsock program to connect to the Linux machine,
and log in if need be.  However, you might consider rsh or something
equivalent.

Now, thinking laterally, why don't you create a special share on the Linux
machine using Samba.

The Icon you click on on Windoze runs a batch script that does a net use
etc to access the special share.

Then, in the special share, run a root preexec script which brings up the
network ...

What could be simpler :-)
