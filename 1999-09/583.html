Re: LI LO?

Hi.... grab a linux boot disk (even the install disk SHOULD do) and
reboot your pc with the floppy in the appropriate orifice. When you get
the lilo or boot prompt hit shift or tab (can never remember which one
it is cos I hardly ever boot my windoze partitions :))
You should see various "boot" options...just pick one (for this example
we'll call it linux ;))

type 
linux root=/dev/hdb1
and hit enter.. if you're lucky it will work & you will boot into your
installed linux.
Once logged in cd /etc/
and check lilo.conf is as it should be, then type lilo
If it returns with no errors you are laughing it (probably ;)) worked...
now shutdown , remove the floppy & try again.

If you get probs do an fdisk and make sure that /dev/hda1 is set
bootable(active) and ditto for /dev/hdb1

That should do it
