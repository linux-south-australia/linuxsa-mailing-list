On 8 Sep 1999, nick cradock wrote:

> Dont know if this has been asked before. Is there a way to install fonts on
> linux, I am using Wordperf 8 and would like to put more fonts on the system
> but not certain how to do it. Any suggestions would be good
> Nick
> 

If it is a pcf or similar X type font put them in
/usr/X11R6/lib/X11/fonts/misc 
and run: mkfontdir
in that directory.

If it is a truetype font (like windows uses)
You'll need to run a true type server and add it to the fontpath in the
X11 config file.
