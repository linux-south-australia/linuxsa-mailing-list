Mandrake Linux: Wow

Hi,

I have just installed Mandrake Linux, and here are a few first impressions.

INSTALLATION.

The installation process is basically a RedHat install.  The X
configuration process was easy and essentially worked. I had to use
xvidtune to change the position that X started up on.

DESKTOP

Mandrake uses KDE as the default, and it is great compared to what I have
been using.

This makes Mandrake look great when it starts up.  Of course, you can
choose things like AnotherLevel, but that looks clunky after KDE.

CONCLUSION

Mandrake Linux was very easy to install and looks great ...
