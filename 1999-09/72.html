Re: The eqvivelent of loadlin.exe

On Mon, 30 Aug 1999, James Morgan wrote:

> Just a quick question.
> 
> I am running a dual boot system running two operating systems Win95 and
> Linux (Deb slink). I select between the two operating systems with LILO.
> Currently if I have booted win95 I can run loadlin.exe to load linux.
> 
> Is there an equivalent linux program that allows me to go the other
> direction? (i.e. allow me to reboot in win95 with out doing a cold
> reboot.)
> 
> I assume LILO may have a feature like this but I could not find it in
> the man pages.

Yep, and it is a LILO option, just type:
lilo -R other | shutdown -r now
where 'other' is my windows partition (man lilo). Note that this performas
a cold reboot, but when I added 'reboot=warm' to /etc/lilo.conf, the
reboot to windows didn't work (YMMV). 
I have mine set up as an alias:
alias other='lilo -R other | shutdown -r now'
