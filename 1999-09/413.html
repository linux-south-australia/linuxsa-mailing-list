>I am using RH6.0 Is it possible to have two dial-ups at same time? I've
done a
>search on freshmeat.net, doesn't seem to have any program to do this.

Hmmm, not sure if you are asking:

1. Can you have multiple people dialing in at the same time or multiple PPP
dial-ins at the same time?

or

2. Can you dial-out via PPP multiple times.

In any case, the answer is yes.  There are lots of tools for doing this.
RH6.0 has got tools like netconf and netcfg that help you support dial-outs
and getty_ps can support dial-ins.  There are also other tools, like mgetty
that support dial-ins.

Just plug in multiple modems and away you go.
