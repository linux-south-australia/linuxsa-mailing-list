Installation/Recognition problem

Thanks to those that have given me some helpful tips with my
installation problem, however I now believe that I have identified the
problem. My BIOS and Linux (Redhat 6.0) see my HDD differently i.e.
Geometry as follows;

C/hda
1.6GB HDD    Cyl        Hds        Sec        Size        LBA
Block
BIOS               3146      16           63          1548
Yes            16
Linux               786        64           63          1547

D/hdb
1.4GB HDD    Cyl        Hds        Sec        Size        LBA
Block
BIOS               2792      16           63          1374
Yes            No
Linux                698       64          63           1374

E/hdc
1.4GB HDD    Cyl        Hds        Sec        Size        LBA
Block
BIOS               2792      16           63          1374
Yes            No
Linux              2792      16            63          1374

D and E are the same make and model. What I need to know now is how do I
get the BIOS and Linux to see the same thing. I have used the BIOS auto
