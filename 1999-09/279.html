Re: Use v.42bis over PPP? (Was Re: Mail collection problems)

On Fri, 10 Sep 1999, W. Sierke wrote:
> What's the consensus about using v.42bis (compression) over a modem
> PPP link?

My experience has been one of poorer performance with modem error
correction turned on, which is a prerequisite for compression.  Perhaps
it's because it takes a while to recover after an error, and with the modem
correcting your complete link "stops" until you've recovered, whereas with
TCP doing the error correction it's only the dead packets which are held
up.  Is there really any difference?  I don't know.  I've not done
measurements and can only offer these anecodatal impressions.
