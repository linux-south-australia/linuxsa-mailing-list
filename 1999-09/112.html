Re: PCMCIA support for Toshiba 310 CDS with Turbolinux 3.6.x

> Have just installed Turbolinux (full install with an APM kernel) on a
> Toshiba 310 CDS laptop with a 3Com (3c509) etehernet pcmcia card.
> install was fine, but pc card is not recognised - no beeps on boot etc.
> Probing around for pcmcia within the OS suggests thats pcmcia support
> has not been installed. Is this a problem particular to Toshiba?

My experience with RH is that unless the install explicitly uses the
PCMCIA bootdisk it will not enable it.  But to enable it it it was as simple
as installing the pcmcia kernel RPM (supplied with RH6) and modifying
/etc/sysconfig/pcmcia to say PCMCIA=yes and PCIC=i82365.

So with a bit of playing (and knowing TL's heritage in RH) you should be able
to get it going.
