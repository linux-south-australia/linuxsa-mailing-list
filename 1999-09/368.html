Mark Williams wrote:

> Hi guys...
>
> I have a bit of a problem with LILO.
> A ehile ago i moved the hdd containing linux out of the machine (and removed
> linux from it).
> Unfortunatly i forgot to unsinstall LILO on the computer, and now when i boot
> i just get "LI".
> At the moment the comp is only running win2k which is how its going to stay.
>
> How do i remove LILO from the MBR?
> I know i can do a "fdisk /mbr" in msdos but im not too sure if that will work
> too well with win2k which uses the NT4 boot loader.
>
> Anyone got any ideas???
>
> Thanks,
>  Mark Williams
>
>

Boot from a Win9x/DOS disk (floppy) and use fdisk as follows:   fdisk /mbr
That will trash it and leave a single boot DOS MBR.

If you have a boot disk but don't have fdisk on it then boot from the floppy and
access fdisk on the Hard Disk...usually in C:\DOS (pre Win9x) or
C:\WINDOWS\COMMAND.

regards ... Don.
