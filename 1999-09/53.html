Hi,

I assume you have it installed - do a rpm --query samba to see if this is so
and if nor install it from your installation cd by going into the RPM dir
and typing rpm -i package name.

If it is installed, type setup at the command prompt, go into system
services and run tool, scroll thru the list and make sure that there is an
asterix next to smb, by pressing the F1 key will give you more detailed
instructions on what these services do!

Should work fine now!  If you want to set up the samba.conf file there are
noe GUI based programs that help to make the job easier.  Have a look at
http://au1.samba.org/samba/samba.html  under the downloads section I think.

Hope this helps.
