Alan Kennington wrote:

 > The trouble is that SO loads the _entire_ office suite
 > even if you just wnat use a tiny part of it.

This is not true on any OS with a halfway modern virtual memory
subsystem:  Demand paging means that only the bits that are used
get loaded, the rest of it just exists as unmapped pages in a 
large, sparse virtual address space.

There are other reasons for StarOffice's speed deficiencies.
