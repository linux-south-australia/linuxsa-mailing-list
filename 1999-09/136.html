Re: SuSE

> As a general point of interest, you might find the apropos command useful.

For people using RH there is an application called 

	helptool

Which scans most of the resources that are installed (info/man/docs).

Even if there are some problems with the level and quality of the documentation
available for Linux and other Open Source applications as a user group we
can still effectively do indexing for people.

It takes only a few characters to say... 

	'For more information on what I have just said, visit
		http://blah.com/ or have a look at the documentation
		usually found in /usr/doc/blah-1.4.5/.'  

If anything it gives people an idea about where to get the information.

As an aside, how do people fare find niggly bits of information for 
Windows.  Whenever I have had a look at the documentation it tends to be
very light in detail.  And usually I end up trawling the net for a solution
to the really hard problems.
