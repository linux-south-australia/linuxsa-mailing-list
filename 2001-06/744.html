Re: cant find network device

Oh yeah, the other way is to download intels driver for it in tar.gz
format from their web site and compile that up. The intel driver is
called e100, so for the intel one you can just add a line like this to
/etc/conf.modules like:

alias eth0 e100

The inbuilt kernel driver is called eepro100
