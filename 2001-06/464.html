Samba with OS/2

Hi,
OS/2 file sharing uses SMB technology & supports TCPBEUI compliant to
RFCs 1001/1002.

Based on this information, do people think an OS/2 requester or server
will talk to a Linux Samba server?
Can Samba replace a PDC yet?
What testing have PDC replacements had with other than Windoze clients?

FYI
OS/2 can & does run TCPBEUI protocol without NETBEUI.

An OS/2 workstation or server configured as above will talk to a Windoze
workstation or server.
