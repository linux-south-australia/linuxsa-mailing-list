Robyn!

Reading the man page it looks like mkbootdisk calls mkinitrd. You can
specify arguments to mkinitrd using --mkinitrdargs <args>. So, something
like:

mkbootdisk --mkinitrdargs bzImage --device /dev/fd0 2.4.2-2

Would possibly work. It looks like the "2.4.2-2" is the same name as
whatever is in /lib/modules/ ...
