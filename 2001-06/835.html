Re: Network Install

"Davis, Harvey" wrote:

> I have recently set up my linux network (closed intranet) to allow network
> installs. Unfortunately partway through tyhe download (random) the system
> freezes (not the server but the client). Any ideas what might be causing
> this.
>
> System is Red Hat 7.1, link is fast ethernet machines are PIII 1Gigs

My guess would be that the FTP server is timing out the connection.
We get this complaint about installs from the AARNet mirror a
lot.

Since the FTP timeout is a tad essential and some disks can
take many hours to format, the most reliabile solution
is to use a HTTP server and the HTTP network installer.
