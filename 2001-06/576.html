Re: PHP as a Scripting Language

An example php shell script that I execute in my .bashrc
Easily done with a bourne shell but I wanted to use php
for fun.

#!/usr/local/bin/php -q
<?php
if (date("H") >= 0 && date("H") < 12) {
	echo "Good morning $USER.\n";
} elseif (date("H") > 12 && date("H") < 18) {
	echo "Good afternoon $USER.\n";
} else {
	echo "Good evening $USER.\n";	
}
echo "You have " . trim(exec("ls Maildir/new/ | wc -l")) . " new emails and " . trim(exec("ls Maildir/cur/ | wc -l")) . " read emails.\n";

?>
