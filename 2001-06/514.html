Re: ReiserFS and large files ... 4,294,967,295 bytes seems to bethe limit.

When you created the file system, which version did you pass?

I think it creates ver 1 FSs by default, but you need ver 2 to get the large
file support, etc..

Also, have you tried later kernel versions? reiser still seems to be getting
bugs ironed out of it.. the version that came with 2.4.2 had a few bugs, it
seems, and even 2.4.5 still needs an extra patch for reiser.
