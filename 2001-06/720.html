Re: rm return values

Ben Williams wrote:
> does anyone know what values 'rm' returns? i want to execute an rm statement
> from a php script and check the returned value to see if it worked properly.

Here's a couple of ways of determining this:

METHOD 1 - test to see whether exit status was zero or non-zero
-----
rm nonexistantfile && echo true
=>
rm: cannot remove `nonexistantfile': No such file or directory

rm existantfile && echo true
=>
true
-----

METHOD 2 - display exit status
-----
rm existantfile
echo $?
=>
0

rm nonexistantfile
echo $?
=>
rm: cannot remove `nonexistantfile': No such file or directory
1
-----
