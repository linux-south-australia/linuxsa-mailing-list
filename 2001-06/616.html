Re: Putty dropping connection

Try 3600 for an hour. That should cover it. The only reason the parameter is there is so the masq table gets cleaned of stale entries. Not really a problem if you have a small network behind being masq'ed.

If you have a program like a web spider running making lots of small connections, then you would want to lower the figure to take that into account.
