Re: booting reiserfs

Tim Fairchild wrote:

> Can reiserfs now be used when booting linux... or do I need an ext2 /boot
> partition?

I guess the question is, would you want to?  ReiserFS' advantage is when
you do lots of writing and changes to files/directories on a file system,
something that happens pretty infrequently on a boot partition.  Of
course, that's presuming that you have/use a separate boot partition
(I do, whether I need to or not, because I usually winding up with the
need ...).
