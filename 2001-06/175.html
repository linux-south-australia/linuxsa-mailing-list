I run a script like this to start the qmail pop3 daemon:

#!/bin/sh

#Number of simultaneous POP3 connections
POPCOUNT=100

/usr/local/bin/tcpserver -R -c $POPCOUNT -u 500 -g 500 0 pop3 \
/var/qmail/bin/qmail-popup mailhost.adam.com.au
/var/qmail/bin/chkpoppass.pl \
/var/qmail/bin/qmail-pop3d Maildir &


You can get the username with $ARGV[0] and the password with $ARGV[1] in
your script.
