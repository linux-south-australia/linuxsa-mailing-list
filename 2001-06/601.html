Re: how to install man pages

On my system (Debian), there is a configuration file

    /etc/manpath.conf

which has a whole lot of standard paths for man already
setup (I presume there is something similar for other
dists?).  Instead of setting a MANPATH variable, it would
be easier to modify the above configuration file to point
to the directory of the new man files.  Or better still,
move or soft-link them into the relevant places within
/usr/local/man and make sure a path exists to this.

Cheers,

Mark.
