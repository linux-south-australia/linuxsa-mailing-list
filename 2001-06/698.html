> What I usually do is this.(It may not be 100% correct as I'm writing this
> off the top of my head but the theory is there.)
> 
> 1) Boot from your RH 7.1 install CD
> 2) Run through the steps until you get the X Installer up.
> 3) Use Alt-Ctrl-F2 to get to the basic shell prompt
> 4) do a mknod /tmp/hd? to create the device you need to access your disk
> 5) mount /tmp/hd? /mnt/somedirname
> 6) chroot /mnt/somedirname
> 7) mount -a (Mount everything in your /etc/mtab)
> 8) verify the /etc/lilo.conf file is correct
> 9) rerun lilo
> 10)reboot

Ummm, this is the hardest way of doing it that I know :-)

What is wrong with:

1. Boot your RH 7.1 install CD
2. When you get the prompt, issue:

  linux initrd=  root=/dev/hdax single

where hdax is your root file system partition.

3. Rerun lilo ...
