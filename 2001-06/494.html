But, (to be pedantic) I'm not sure it was the point at all...  Wasn't
Andrew Reid talking about his BSD system when michaeld turned it into
a RH/Debian comparison?  (It's a bit hard to tell, since michaeld's

For linux PCMCIA information, check out the page at sourceforge: 
  http://pcmcia-cs.sourceforge.net/ftp/SUPPORTED.CARDS
(For example, it says there that the Xircom CEM II Ethernet/Modem is
NOT supported.) 

BTW, I installed my current debian system in early 1997 from 6
floppies (and an Internet connection).  (Back then, there were no
*really* good Linux installers that did all the auto-detection & etc.)
But in 4 years, I've never had to re-install, or buy an "upgrade" CD.
Now I run the 2.4 kernel, with XFree86-4.0.3, gnome-1.2 or kde-2.1,
gimp-1.2, etc.  I've never had to wait a year for any package to
"catch up" with the competitors.
