Re: Permissions on Win partns

Hi Bryan,

On my box I have one fat partition and I'm pretty much the only user so
I mount the partition as my user id (1000) using the following fstab
entry:

/dev/hda2       /fat            vfat    user,uid=1000,defaults  1       0

You can set various other options such as gid and umask if other people
need to access the drive also.  For example:

uid=1000,gid=60,umask=002

Hope that helps,

- Trev.
