Re: Network printing on the Adelaide Uni system

>>>>> "DS" == Dan Shearer <dan@tellurian.com.au> writes:

  DS> On Fri, 15 Jun 2001, I. T. Riley wrote:

  ITR> I am attempting shake of the Microsoft shackles but am not yet
  ITR> able to print on the network (Novell Netware) printers on the
  ITR> Adelaide Uni system.  Without access to printing facilities,
  ITR> becoming a full time Linux user remains a vain hope.  I am
  ITR> running Progeny Debian and am just a novice Linux user.  I
  ITR> would appreciate any helpful suggestions from those familiar
  ITR> with our situation.  Thanks, Ian

  DS> The best solution from many points of view is to get the Netware
  DS> admins to enable the Unix standard printing protocol lpd (or the
  DS> Internet Printing Protocol, if Netware is modern enough to do
  DS> it.) That way all kinds of of systems can print without knowing
  DS> anything about Novell. But they probably won't, citing
  DS> authentication or other reasons.

  DS> So the next best thing is to read the Netware bit of the Netware
  DS> Administrators Guide (NAG version 2) available at
  DS> linux.senet.com.au or other places as recently discussed. The
  DS> ncpfs package can do all you want and more. You existing Novell
  DS> logins should just work.

  DS> If you want a super-kosher Netware environment, use Caldera
  DS> Linux which has official Netware code in it.

Or you could install the debian ncpfs package:

Description: Utilities to use resources from NetWare servers.

   This package contains utilities to mount volumes from NetWare
   servers.  Also included are some little utilities such as nprint,
   which enables you to print on NetWare print queues, and pserver,
   which provides NetWare print queues.

   Package ncpfs-2.0.12 and above replaces package ncpfsx. This
   package will work with 2.0.x, 2.1.x, 2.2.x, 2.3.x and 2.4.x
   kernels.

   Also, NetWare Directory Services are supported in ncpfs-2.0.12 and
   above.

Also, check out these debian packages:

   Package: printtool
   Description: Printer administration tool.
    An X11 program for the easy manipulation of LOCAL lpd, REMOTE lpd,
    SMB(samba), NCP(netware), and DIRECT(hp jet direct) printers, and
    gs printer filter options. This version work both with lprng and
    lpr.

   Package: lg-issue34
   Description: Issue 34 of the Linux Gazette.
    This contains issue 34 of the Linux Gazette, including:
      How to print to a printer on a Netware server

All these answers were brought to you courtesy of the debian command:

  apt-cache search netware
