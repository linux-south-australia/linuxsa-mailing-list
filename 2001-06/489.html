Gnome 1.4 is in the current version of the ports, use cvsup to update them,
you can install cvsup-bin from the packages on the FreeBSD cd.
Then edit /usr/share/examples/cvsup/ports-supfile to change the host which
you update from(cvsup.au.freebsd.org is the one I use which is actully
mirror.aarnet.edu.au, does get busy after 12am and hard to get a
connection).

Then on the command line type cvsup -g -L 2
/usr/share/examples/cvsup/ports-supfile, and let it go.
