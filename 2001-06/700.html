I have done this so often now I should be expert in it. My apologies 

What is wrong with:

1. Boot your RH 7.1 install CD
2. When you get the prompt, issue:

  linux initrd=  root=/dev/hdax single

where hdax is your root file system partition.

3. Rerun lilo ...

This is pretty much what I did, except I couldnt remember partition number
so quick boot via cd found it then did this.
