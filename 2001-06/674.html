:> -----------------------------------------
:> /dev/hdc3       /                         ext2           defaults   1   1
:> /dev/hdc5       /usr                    ext2           defaults   1   2
:> /dev/hdc2       /e                       ext2           defaults   1   2
:> /dev/hdc4       swap                 swap           defaults   0   0
:> /dev/hda1       /c                       msdos         defaults   0   0
this line could be. ......

/dev/hda1       /c                       msdos         rw,defaults,user   0   0

:> /dev/hdb1       /d                       msdos         defaults   0   0
and this....
:> /dev/hdb1       /d                       msdos         rw,defaults,user  0   0

:> /dev/hdd         /cdrom               iso9660         ro,noauto,user 0   0
:> /dev/fd0          /floppy                auto            noauto,user 0   0
:> none               /proc                  proc            defaults   0   0

note there are other ways of doing not least replacing "user" with "owner".
You might also investigate useing vfat as a file system (but this won't alter you access problem)

man mount 
gives a terse but usefull account of how to properly mount each partition:
don't forget info mount and apropos mount.
