Re: Initial sound volume

Andrew,

> I have a problem where the sound volume "appears" to
> be automatically set to the maximum when my PC boots. 

The only way to this is to tell the sound card module to start with
certain mixer settings.  

Unfortunatley this is not easy as you need to modify the sound card
kernel source.  You need to look thru the source code for your sound
card and find where it sets the initial mixer setting.  For example
looking in the Maestro sound card driver (in drivers/sound/maestro.c)
for "mixer_defaults" shows clearly what needs to be changed.  Posting
your sound card type on the list may enable someone else to help you
do this.  

You will need to make your modification and then recompile the
module.  

It would be nice if the modules took initial sound settings as a
parameter but I have never seen this, not that I've looked very hard.
