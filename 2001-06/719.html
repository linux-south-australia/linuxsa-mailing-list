> What is wrong with:

> 
>   linux initrd=  root=/dev/hdax single
> 
> where hdax is your root file system partition.

If you don't know what the root partition is, a simpler version of the
hard way is:

1) Boot from the RH7.1 install CD
2) Type 'linux rescue'
3) Run through the steps until you get a shell
4) Run lilo -r /mnt/sysimage
5) Press ^D

The rescue mode will find your partitions and mount them
automatically.
