RE: PHP question

it sounds very much like you need a variable variable (i love these ;-) )
try doing this inside the loop:

$headervar = "header".$i
$textvar = "text".$i
$sql = "INSERT INTO latest (heading,text)
	VALUES ('\$$headervar','\$$textvar')";

that should (hopefully) work. i haven't tested that at all though.

what that variable variable thing does is evaluate a variable called "$(the
value of the variable called $headervar)". on the first loop say, $headervar
contains the text "header0", so $$headervar evaluates to "$header0", which
is what you want.

on a related note, you can also do variable function calls (dunno if that's
what they're called), like:

if($input == "1") $func = "echo";
else $func = "printf";
$func("this will print!\n");
