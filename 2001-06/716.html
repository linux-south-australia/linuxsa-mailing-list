Re: Convert Workstation to a Server.

On 22 Jun 2001 12:54:32 +1000, Valladares, Gerard C wrote:

> I however took the Workstation option.
> I would now like to make it a Web Server.

The 'Workstation' option is only a predefined set of packages installed
during the setup process. It doesn't stop you from installing other
packages that perform server-like opperations.

What you're probably looking for is Apache (http://www.apache.org/).
RPMs are available from the RedHat CD or AARNet
(http://mirror.aarnet.edu.au/pub/redhat/redhat-7.0).

Then use RPM to install the package (rpm -Uvvh /path/to/apache.rpm).
