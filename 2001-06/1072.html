Re: HOW to BURN CD

Anil,

I use a Debian system which I finally set up for CD writing (Debian
2.2 "Potato" with the 2.2 kernel).  It wasn't a trivial exercise 
because the standard kernel does not have the SCSI compatibility
drivers in it; these are needed by cdrecord which only knows
the SCSI interface.  I followed the instructions in the 
CD-Writing-HOWTO.  Hint, find the list of devices which must be
activated and DON'T be tempted to make all of them as built-in,
there are a few that must be built as modules (that took me ages
to figure out!).  

The commands that you described below look a bit different to what
I've been using.  Here is what I have in my notebook...

To create a CD image...
mkisofs -l -o /tmp/cdimage pathspec
                           -------- 
			   ^ ie this is what you want
To examine the image...
mount /tmp/cdimage /mnt -t iso9660 -o loop,ro

To burn the image...
cdrecord -v -dummy speed=4 dev=0,0,0 /tmp/cdimage
            ^1           ^2    ^3
1 first time around only
2 CDRW is 4, CDR is 8 (for my drive)
3 figure out 0,0,0 from cdrecord -scanbus

If your machine is hanging when you try to mount the image
using the loopback device, I would recommend checking that
the loopback is in the kernel (examine /proc/devices) and that
the "loop" devices are in the /dev directory. Refer to 
the HOWTO that I described above.

Hope this helps, and good luck,
