Hi.

Forgive me for pointing out something ambiguous with this whole hardware
versus software raid argument.

On one hand I am told that software raid is better cause the hardware
cards have a low spec cpu on them, and spare cycles on the system cpu are
a much better resource for doing the raid calculations.

On the other hand, when Mike says that software raid loaded up his box,
the response is that he must be mistaken, because 'they're not nearly as
cpu intensive as you would think'
