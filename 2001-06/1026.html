Re: SCSI cards

I'm using a DC315U in a RH 7.1 system now, and have had 
no problems either with kernel 2.4.2 or 2.4.5.  The 
SCSI card simply interfaces to the CD-ROM, not a 
SCSI disk, but booting off the SCSI CD was quite easy 
(configure the Tekram card to boot -- F2 on startup) 
and the CD was detected thereafter.  

I use the NCR53c8xx driver in the kernel to support 
this card.  

hope this helps,
