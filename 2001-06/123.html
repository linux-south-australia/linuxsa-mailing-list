Re: LDAP Authentication with dot usernames

Richard Russell wrote:
> 
> note that mail addresses and usernames don't have to be identical... you
> could have firstname.lastname for mail, and a serial number for username...
> I think... (don't know novell -- this is just a general principle)

Yes. We use sendmail's LDAP lookups for that. I created a few aliases
when I first set up the mail accounts, so that staff can receive mail in
the form of:

mgeddes@xavier.sa.edu.au
Matthew.Geddes@xavier.sa.edu.au
Mr.Geddes@xavier.sa.edu.au

There doesn't appear to be a limit on the number of maillocaladdress
attributes (the aliases) you have, but obviously, you can only have one
mailroutingaddress (the final destination of the mail message).
