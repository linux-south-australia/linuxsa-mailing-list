Re: Installing PHP4.0.0.5 on Suse Linux 6.3?

On 05 Jun 2001 13:19:34 +0930, Wharff, Bjorn wrote:

> I have a Suse 6.3 box currently running PHP 3.0.12-18 and wish to
> install/upgrade to PHP 4.0.0.5 but it seems there are far too many
> dependencies such as libjpeg which I upgraded which then caused invalid
> library calls from within X-Windows. Does anyone know whether this is meant
> to be possible/impossible or whether there is an easier way to do it?

Download the sourcecode for the required libraries and compile and
install them in some other location (like /opt). Then, point PHP's
configure script to the new libraries:

./configure --with-jpeg=/opt/libjpeg

Rememmber that you don't have to satisfy all of the '--with' options --
enabling them often only provides functions that you never normally use.
