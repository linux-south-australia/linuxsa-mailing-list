They should shouldn't they :-)

However if your laptop is a nice new up to date one it's possible that the
CS4281 is still a work in progress (as my own Intel ICH is) and ain't gonna
work yet :-(

The good news is.....ALSA.....Advanced Linux Sound Architecture, who seem
to have CS4281 working nicely. Avaliable from http://ftp.alsa-project.org/
You will need 3 source code packages, driver, utils and libs and follow the
INSTALL instructions for each package.

If you have any problems give me a yell, and thanks for the cat /proc/pci
it helped me find the correct ALSA modules.
 
(ALSA code is slowly being included in new kernels so CS4281 will be there
one day)
