Re: Getting a program to run as a user on startup

Darryl Ross wrote:

 > Got a friend who is running FreeBSD. He has a program that cannot be run as
 > root, but he wants it to start up automagically when the machine boots up.
 > The program is to be run under my account. How can I get it to do that?

In /etc/rc.local:

    echo programname | su username
