Re: FreeBSD vs Linux

Nick Morrison wrote:

 > I'm assuming BSD will happily co-exist with win/linux/etc?  Are there any
 > odd/unusual considerations for little old jump-in-why-the-hell-not me?

There's a HOWTO which describes how to get Lilo and FreeBSD's boot
manager to coexist;  I think it's included with most Linux distros,
but if not you can find it at 
http://www.linuxresources.com/LDP/HOWTO/mini/Linux+FreeBSD.html

It's rumored to be possible to boot FreeBSD via Lilo too, but I'm not
sure if the docs for that have been updated since the FreeBSD bootstrap
loader was updated a few months ago.

 > Hardware support is ok?  I've got fairly standard bits'n'pieces..  so can
 > you think of any last-minute warnings?

Not really; See ftp://ftp.freebsd.org/pub/FreeBSD/3.2-RELEASE/HARDWARE.TXT
for the list of supported hardware.
