Thoughts On Root

Most of the time I log in as "lloy0076" which is my user
account. The only time I ever log in directly as root is if
I know I absolutely must; generally only after a total
reinstall.

I then happily use my system and if I come across a
"permission is denied" or some type of message indicating I
need to have more power, I stop and think exactly what I'm
wanting to do. If I decide it's OK, I su, type the password,
do it and go on with life.

So, this way I've managed to prevent myself from doing silly
things like rm -rf /usr....just some thoughts...or thinking
out loud.
