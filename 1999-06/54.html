Hi,

If you want X to use 3 buttons (or emulate 3 buttons), I beleive you'll
need to edit the XF86Config file (stored in /etc/X11 on RedHat). I'm not
sure exactly what needs to be put in, since I always use other programs to
edit XF86Config for me.

If you're using RedHat 6.0, install the XFree86-XF86Config and
XFree86-VGA16 rpms and at the console type "XF86Config" and you'll be
taken to a config program. note, XF86Config has been totally changed. it's
no longer a text based config, it runs X in VGA and you can configure a
lot of things (mouse, monitor, video card and more).

good luck,
