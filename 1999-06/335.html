Hi,

> 
> anyone understand what this message means?
> this may seem blindingly obvious to someone else, but....
> 
> 
> httpd: cannot determine local host name.
> Use the ServerName directive to set it manually.
> 

yep, since you don't have name services configured (named) the apache
can't work out what the localhost name is. Easiest to uncomment the
hostname line in /etc/httpd/conf/http.conf and insert your host name.

check the apache documentation for the details here.
