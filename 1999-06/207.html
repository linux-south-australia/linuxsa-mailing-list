Check that E (enlightenment) works properly by putting :
exec /usr/local/enlightenment/bin/enlightenment
(or wherever you put E) as the only command in your .xinitrc

if E works then put the following command in your .xinitrc

exec gnome-session

It is important that you remove the enlightenment entry. The comment
character is the # symbol (for removing entries that you think you
better not forget).

Gnome will now load. Add E from the Gnome control panel. To get to this
pop up menu you have heard so much about click the middle mouse button
(or l + r together if emulating). The r mouse button still operates the
Gnome menus.

ummm I think that about does it. 

Let us know if it still doesn't work (you might need a later version of
E or Gnome or something).
