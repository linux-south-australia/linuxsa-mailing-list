Re: Virtual Screens

On Mon, 21 Jun 1999, David Lloyd wrote:

> When I place the mouse cursor very close to the edge of the
> screen, it jumps into the next virtual screen. How does one
> turn off this feature totally, or at least stop it from
> automatically jumping around?

Sounds like you are using fvwm or fvwm2, though I could be wrong.

If you are, have a look for a file .fvwm or .fvwm2rc in your home
directory. Try "ls -lad ~/.fvwm*" to see what you have. Edit the file and
change the line:

EdgeScroll XX YY

where XX and YY are some numbers, and change the XX and the YY both to 0.
That should stop it.

If you don't have any .fvwm* files in your home directory it might be the
system default that you are using. You need to find the file system.fvwmrc
or system.fvwm2rc on your system (maybe in /usr/X11R6/lib/X11/fvwm) and
copy it to your home directory as .fvwmrc. Then you can do the above edit.

I still think fvwm is enough Window Manager for anybody :-)
