Re: Displaying REMOTE_USER with JavaScript

Well, you could do it my way...
Use server-side-includes, (server parsed html), for the page with the
javascript in it.
So, you have the javascript look like
string username = "<!-- #include cmd="echo ${REMOTE_USER}" -->";

You can't just copy n paste that, but you get the idea of how to do it.
