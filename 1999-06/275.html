Re: help on modems for a mac

Your basic answer appears to be yes. You will need the apple modem connector
which is a mini din plug, should be able to pick one up from any good Mac
distributor. The problem is driver software, not all modems have Mac
software, you may be able to download some, of course this is usually fixed
by installing Linux and not MacOS anyway.

Some of the leading brand names do supply Mac versions of the modem which
basically come with Mac software and the correct cable, I am not sure which
anymore, check them out.

Rob.
