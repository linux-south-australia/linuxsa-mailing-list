Phearless Phils Quick/Dirty/Simple Home LAN
Ingredients
                :  3  pc's with at least ONE running linux
                   3  Ethernet Cards + coax

1: The DIALUP MACHINE must run LINUX  & MUST have the modem attached :))
2:  Give the ethernet interface on the DIALUP machine an ipaddress of
192.168.1.1
(NB: its ppp  interface ,when u r dialled in, will be either what your isp
allocates or your permament ip if you have a generous humane ISP ;))
3: Give the 2 other pc's the ethernet ip addresses of 192.168.1.2  &
192.168.1.3  respectively with the default route set to gw through
192.168.1.1 netmask 255.255.255.0 will do
4: set up IPFWADM to allow 192.168.1.2 & 192.168.1.3 to access 0.0.0.0/0
via ppp interface & you;re pretty much done
LINUX WILL DO THE REST... to the internet, it will appear that only your
dialup machine is accessing data but it is actually sending out data
requests for the other machines as well using 60000+ port addresses.

Think thats it... anyway, that will get you pretty much LAN;d up ;) Oh
btw, I assumed you wanted the LAN to allow more than one connect to the
internet :))

NB: You may (not sure) have a problem if 2 (or all) of the machines try &
access the same irc server at the same time ;) and some ftp servers may
not like you (pasv might fix that one - IF your ISP has ftp proxying then
it wont matter anyway)

Hope that gets you started,
