>My question is I also want to do PPTP(VPN) on my windows NT to go through 
>the Linux machine to connect to my work NT machine that has PPTP installed

A search for PPTP at www.freshmeat.net reveals the follwoing:

       PPTP Masquerade patch
       Kernel patch to add support for masquerading MS PPTP VPN protocol

       PPTP-linux
       a PPTP client for Linux

       Linux VPN Masquerade
       Masquerade support for IPSec and PPTP VPN traffic

       PoPToP
       PPTP Server for Linux

It looks like you want either Linux VPN Masquerade or PPTP Masquerade Patch.
