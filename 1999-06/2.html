Hi,

It seems if you add the deflate option, without adding the required
additional modules, pppd will be inactive. To fix this, if you have added
the option, remove the option from the options file, unload the ppp module
with modprobe -r, do a depmod -a ppp, and then reload the ppp module with
modprobe -a ppp.

This should make it work.
