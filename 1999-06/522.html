Send a HTTP "head" request to their server (i.e.: type the lines 
prefixed with ">>>")

>>> newton@atdot> telnet www.whitepages.com.au 80
    Trying 192.148.160.200...
    Connected to www.whitepages.com.au.
    Escape character is '^]'.
>>> HEAD http://www.whitepages.com.au/ HTTP/1.0
>>>
    HTTP/1.0 200 OK
    Server: Netscape-Enterprise/2.01
    Date: Fri, 25 Jun 1999 04:17:55 GMT
    Last-modified: Mon, 19 Apr 1999 04:02:34 GMT
    Content-length: 7460
    Content-type: text/html
    Proxy-Connection: close

    Connection closed by foreign host.

They ain't using the MS server (but they might be using the Netscape
server on NT).
