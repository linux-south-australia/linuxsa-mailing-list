It sounds like that disk may be faulty.
If your using redhat then you can start the install in two other ways.
1 - boot directly off the CD (may not be possible with your bios)
2 - provided you have the dos drivers installed for you cd drive, go to
dos mode and type:

d:                (or the letter of your cd drive)
cd \dosutils
autoboot

This will start the install off the cd.
