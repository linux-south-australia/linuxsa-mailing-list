Re: Virus via (HTML-format) E-Mail...?

C4IS wrote:
> 
> OK, it may take VB... is Linux affected?
> 
> http://www.developer.com/news/stories/112098_vbvirus.html

No.  Linux, and other UNIXen for that matter, are not
effected.

Basically the article suggests that a scripting language
enclosed in a HTML wrapper could be used to replicate itself,
either by uploading itself to the web or e-mailing itself.

There is no web-accessable scripting language under UNIX
that is allowed to have the poorly fettered access that
VBScript is given under Windows.

However, JavaScript comes close and JavaScript hacks are
sufficiently common for most of us to disable JavaScript
on our browsers.

Not that we should be complacent.  Linux's relative immunity
is a matter of conscious design and a willingness to say `no'
to requests to "make Linux easier to use" by:

 - removing the need to login

 - making configuration files globally writable

 - getting rid of the different between root and other users

 - having attachments run macros

Microsoft, being more `customer focussed' has said yes to
all the above at various times :-)
