FOLLOWUP: FP Extensions under Linux

Hi all,

Well my month-long saga has finally ended in success. :-)
I have successfully got the Linux FrontPage server extensions
to work completely. The bug was as simple as one single line of
code that failed to be patched into apache's http-request.c for
catching ".../shtml.exe/something.htm" requests and executing
the extension (instead of resolving it as an invalid path).

By the time I discovered this I was actually compiling the
NEW FrontPage 2000 (version 4.0) extensions into a freshly downloaded 
1.3.6 ver of Apache (clean slate kinda idea ;-).

Anyway, if anyone needs any help with doing the same or needs
a copy of the altered patch to suit Apache1.3.6, let me know.

Regards,
