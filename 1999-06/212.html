KERNEL VERSION 2.3.6

I recently installed the development
kernel 2.3.6. One Australian mirror site
is (to download the version I
have,simply log onto the InterNet, and
type in ):

  wget
ftp://ftp.au.kernal.org/pub/linux/v2.3/linux-2.3.6.tar.gz

I wasn't altogether convinced that a new
kernel would make a great difference.
However, my computer - a 300mHZ Cyrix686
with 32mB of RAM and 64mB swap space -
is running noticeably faster, especially
in reference to accessing the hard disk.
I'm certain that my other kernel (of the
2.0.3 family) had been configured
optimially for my system as I built
myself a custom kernel to optimise it
for my system.

I write this to indicate that the later
kernel does appear to function a lot
more efficiently, although I could not
say the technical reason why, and to
find out if anyone else has had the same
experience. Now, even my SlowOffice 5.1
(StarOffice) is starting to look like a
viable alternative to Microsoft Office
in terms of speed...
