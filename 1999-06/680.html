As a side issue, a number of utilities have been written for doing
authentication against an SMB/CIFS server (like a PDC, etc).  Most of these
are ultimately based on SMBlib (called smbval or something like that else
where). 

Unfortunately, SMBlib has a number of nasty buffer overflows in it, so take
care. 

I am working on fixing SMBlib, but have other priorities, like a Linux
