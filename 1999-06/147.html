Re: install problem

Hi Stephen,

once it has booted to x you can switch virtual terminals by holding down
alt ctr and then hitting F2. This should give you a normal terminal
screen to play with.
You then need to edit /etc/inittab, change the line...
id:5:initdefault: 
so that it has a 3 instead.
Next time you reboot, it should boot without starting X.
You can then go and reinstall the correct X-server for your video card.

Ben.
