Hi,
I am using the Creative Labs PCI64 card with the ensonic es1370
chipset.  Anyone using this card?
I've installed the alsa files (drivers, libs.utils) but no
luck yet.  The howto's claim that snd-audiopci.o is the module that
should be installed with modprobe and not snd-audiopci1370.o.  My
installation didn't create it, it only created snd-audiopci1370.o and
snd-audiopci1371.o.  There is source code for it in
/usr/src/alsa-drivers/cards (snd-audiopci.c).  It isn't listed in any of
the Makefiles.  Also in the file snd-audiopci.c there are a whole series
of lines that are commented out (#) but look relevant.  There don't
appear to be any instructions to cover this (or should I ignore it).

insmod snd reports unresolved symbol.
There are no conflicting modules/drivers loaded (dos, linux or
otherwise).  When I do use the modules that come with kernel 2.2.10
