It does also work under Linux.  I'm not using dbmmanage, but this
works for me using Perl:

#!/usr/bin/perl
use NDBM_File;
use Fcntl;

tie %db, NDBM_File, $dbmfn, O_RDWR|O_TRUNC, 0666;
...
untie %db;

With this in httpd.conf:

AuthDBMUserFile /whereever/whatever/
AuthName "Secret Area"
AuthType Basic
require valid-user

I don't recall changing anything related to DBM authentication when
compiling Apache.
