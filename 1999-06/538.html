Re: ppp-compress-21 module error

Hi!

On Sat, Jun 26, 1999 at 09:32:54PM +0930, David Lloyd wrote:

> [date blah] can't locate module ppp-compress-21     
> can't locate module ppp-compress-24
> can't locate module ppp-compress-26

I think that you probably have the modules (I can't remember under what
circumstances they're built, but I would think Red Hat probably included 'em)
... check that you have the file /etc/conf.modules setup correctly.
Specifically, for the PPP compression modules, it should contain (from the
debian conf.modules on my system):

alias ppp-compress-21 bsd_comp
alias ppp-compress-24 ppp_deflate

I don't recognise 26 ... checking ppp-comp.h, however, reveals that it's
another number for ppp_deflate, so you can probably alias ppp-compress-26
ppp_deflate as well. 

If this doesn't fix the problem, you may not have the module paths set
correctly, or you might just not have the modules (find /lib/modules -name
ppp_deflate.o), in which case you might wanna recompile your kernel.

They're not that important anyway, your PPP link will work without 'em - don't
worry too much if you can't solve the problem.
