RE: ls

This does not directly answer your Q, but still ..

# du -s *
Gives a summary of the directories from here down, showing the disk usage in
each

Here is a script that does something useful with each directory ..

# for i in *
> do
>    if [ -d $i ]      # see 'man test'  for info on the test command to use
with if :  -d is true if it is a directory
>    echo $i is a directory
> done

or this one : just lists the names of directories in this directory :

# file * | grep directory | cut -d: -f1

Which you could maybe alias to 'lsd' or something
