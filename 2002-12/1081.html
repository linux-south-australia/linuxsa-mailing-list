Hey Paul,

Sure have.  If you have a look in the Linux SA archives for "NT1 Plus II",
you'll find a past discussion on it.

The 'acm' kernel module is needed if you want to use the USB interface...
Otherwise you'll be limited to 115200bps using serial.

Once you setup the 'acm' module, the NT1 acts like a normal modem.
www.opennz.com.nz's site has a detailed manual with commands.

If you need a hand, lemme know =)
