SuSE Linux changing your RAT from usb to ps2

UG!

I had a Logitech USB mouse inistalled in SuSE Linux 8.1 . It was getting 
old, so I decided to replace it with a very inexpensive ps2 mouse.

 From that point on, every time I would log out of KDE, I would have to 
reconfigure the mouse. Furthermore, every time I would reboot the 
machine, ui-ncurses would come up and ask me to identify the mouse as a 
ps2 mouse.  Also, the KDE control module would show the logitech USB 
mouse as an available device.

This took a long time to solve.   I no longer have to reconfigure the 
mouse or deal with ui-ncurses, and I can live with the error message on 
boot up...

1. Go to the console and type sax2 -r
                (This is actually a nice feature that allows you to 
reconfigure some hardware)
2. Uninstall Yast2-ncurses, but leave ncurses installed


Now off to finish the upgrade!

Cheers
