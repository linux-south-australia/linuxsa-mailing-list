RE: Convert VC project to gcc make file(or "one small step to linux")

If I remember correctly the VC make files are for nmake. These are not all
that different from make files for GNU make. My inclination would just be to
edit the VC make files to fix up the path names to suit the Linux
environment and then try to run them with GNU make. I'm sure a few
iterations of fixing errors would get something you could use quite quickly.
