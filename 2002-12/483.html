Gumf.  I've done the following on Red Hat 7.2, 7.3 and 8.0.

1) Hit http://www.gnome.org/start/2.0/
2) Go to a mirror for the sources
3) Download tarballs
4) Follow the build order in the release note
   (http://www.gnome.org/start/2.0/installation.html#buildorder)
5) tar zxvf <pkg>.tgz
6) cd <dir>
7) ./configure  --prefix=/opt/gnome2
8) make
9) make install
10) cd ..
11) go to 5 for next package in 4

It's just standard GNU tool chain stuff.  Did it a lot when I was following 
release candidate tarballs.  Not that hard.  It you can't/don't want to do 
that, use the GARNOME scripts, or the vicious-build script.
