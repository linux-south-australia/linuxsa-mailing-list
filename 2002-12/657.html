I have a script in /usr.local/etc/rc./ called ap.sh (original huh?)
that runs automatically on bootup. You will have to create this path.
This script run my wicontrol and if config commands, in that order.
The rest of the stuff I kick off in /etc/rc.conf.

> 
> The sequence is important - it must go:
> 
> 1. kldload bridge
> 2. sysctl commands
> 3. wicontrol commands
> 4. ifconfig commands
