Re: Report tool

Andrew Galdes wrote:

>
>For example, i have a log file which has user names in it. I need a way to
>return a list of users which appear in the log. However, i don't want repeat
>mathes returned. 
>
Normally you can just pipe the output into 'sort' then 'uniq' to get a 
list of distinct items.

For example, if you knew that the username was in field 3 of a 
space-delimited file, you could use:

awk '{print $3}' < logfile | sort | uniq

to get a list of usernames.
