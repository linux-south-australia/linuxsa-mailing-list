Re: compiling the kernel

First you need to make sure you have the kernel source tree installed.
I'm assuming you're using a redhat linux distro (you haven't actually
told us) so look on your install cd's for kernel-source package.
Otherwise get the latest kernel from ftp.kernel.org and unpack it into
/usr/src directory and make a symlink like ln -s /usr/src/linux-2.4.19
/usr/src/linux
Then cd into /usr/src/linux and read the README as the instructions for
building kernels are in there. You should then be able to run make
menuconfig (it's the best way to do it) as make config will probably be
tiresome and confusing. I also recommend reading ALL the help for each
and EVERY kernel option even if you dont understand it but the help will
often/usually suggest a sensible default for that option.
It's also a good idea to keep an old and working kernel entry in your
lilo.conf to boot from until you get the new kernel working properly.
Building kernels can take a lot of practice, just like anything.

HTH
