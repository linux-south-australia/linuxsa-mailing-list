Sure Linux has a few problems;

1. Not as easy to use straight out of the box..... Time will fix this.

2. Any help needed in using it is not available from the fellow next door (in 
most cases).............. But there IS help available from many knowledgeable 
people, don't know too many very knowledgeable people using Windows,
for most people it doesn't encourage too much interest in the workings of the 
operating system.

3. Probably the biggest hurdle is that it is very difficult for the average 
computer buyer to buy a machine without Windows installed. If a new user was 
to start from scratch using say Mandrake 9 or similar without the habits 
gained from Windows use, the learning curve would be a lot 
smoother.........maybe another senate inquiry would fix this, or the Trade 
practice act.


Could anyone imagine anyone from the Windows camp telling someone that they 
would be willing to burn them as much Windows software as was needed for an 
exchange of a foreign home brew kit.  Last I heard that would probably be 
Illegal, still illegal is only a sick bird.
The willingness this community has in providing assistance to others will 
never be found in a system that has no interested backing.
