Automatically added rules can simply create a different way for a denial 
of service attack to be brought against you. Most attacks while annoying 
are harmless against  patched server, whereas an automatic rules can be 
used like a remote firewall control tool by a serious attacker - 
effectively locking out network access for anyone (such as the 
administrator) at their command by spoofing attacks.
