It'd work on any distro, cos they supply all the libraries and things they 
need. Strangely enough, the environment is being developed in Kylix 3. All 
the source is pascal. It may be possible to compile up the environment on 
Lazeras using the free pascal compiler once Lazeras goes v1.0 (Lazeras is a 
delphi/kylix clone). The cool thing is with the delphi-like environments, (at 
least with lazeras) is that u can use any GUI back end when you compile (QT, 
GTK, whatever...)... all the pascal headers stay the same, so the source for 
your project doesn't change... I had to use Delphi for 6 months work 
experience a while ago, and it's quite a nice environment. Kylix is awfully 
slow though, i didn't like it (apparantly it uses wine-lib, which would 
explain the sluggishness)
