Re: Report tool

On Saturday, December 7, 2002, at 08:15 AM, Andrew Galdes wrote:
> cat /my/log/file.log | sort | cut -d'.' -f1 | uniq -c

I'd suggest rewriting this to:

cut -d. -f1 < /my/log/file.log | sort | uniq -c

This removes a pipe and cuts the data down before sorting. Your code 
contains a possible bug in that the sort and cut are running on 
different parts of the input. This may have been deliberate, in which 
case my code is faulty.
