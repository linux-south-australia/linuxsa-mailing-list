I'm trying to do something that might sound really wierd, but it really does

Basically, the end result I want is to have a machine on my home network act
as if it was part of a remote samba domain, complete with a correct IP,
profiles, the lot.  Just like if it was physically plugged in.

What I've done is to setup poptop (pptp server) on the gateway on the remote
end, running Debian (which is also the PDC), and I can successfully VPN into
it from my XP workstation and act just as if I was on the local network.

I figured if I ran XP inside a VMWare virtual machine, I could trick it into
