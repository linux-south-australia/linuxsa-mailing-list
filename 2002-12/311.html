Nathan P. Millhouse wrote:

Dear Lycoris Inc, 

I am writing to congratulate you on a fantastic product. 
I have been using Linux for about 5 years now and I have used many distributions which are designed with all kinds of purposes; secure servers, firewalls, and also desktop workstations. I believe that 
your product is a fantastic step towards creating a Linux distribution aimed squarely at the workstation sector of businesses, especially for those with minimal or no Linux knowledge. Since 
discovering your product I have recommended it to countless other friends of mine as a great place to begin using a Linux environment as a replacement for Microsoft products both at home and in the 
workplace.

Unfortunately the main machine which I use at home has two major components which are not supported by the OS. 

Firstly I have an ASUS A7V333 MB with an onboard Promise 20276 ATA133 RAID controller which supports RAID 0 and 1. Although I have both my system hard drives plugged into this device neither of them 
are using the controller for RAID, both just function as separate IDE devices. My CD-ROMs are connected to the standard IDE controllers and they will not work on the RAID controller so changing them 
is not an option. You can imagine my disappointment when the installer refused to detect any HDDs on my system at all and refused to proceed any further. Being aimed at desktop PCs I can understand 
lesser a focus on RAID support, but many other current Linux distributions will automatically detect this device and use the drives as straight IDE without any issues.

Secondly, the NIC which I use is a DLINK DFE-530TX Rev B which is based on the VIA RHINE chipset. According to the Lycoris installer, it showed "None"

next to network adapters during the hardware detection process before no IDE controller was detected. The machine was probed for NICs during the beginning of the installation process so it is not 
possible that the card was going to be probed after the HDDs were detected.

The fact it appears that there is no support for my NIC has me completely dumbfounded as it is supported by every Linux distribution I have used over the last few years, and is also supported by the 
Linux 2.4 kernel itself. Also, with the product being based at workstations I would have thought that as much support as possible would have been built onto the distribution.

Does the company have any intention of including support for these devices in the near future releases of Lycoris? I certainly hope so as I would love to use the product as my main OS on my computer 
as I cannot use any other distros without windows as my wife currently has only basic Linux knowledge.

I understand that you may not be able to respond to this letter immediately, 
but I would really appreciate some form of response even it is only "Yes" or "No"
