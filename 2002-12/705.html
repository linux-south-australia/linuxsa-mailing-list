Re: ssh public key auth

Andrew Lord wrote:

> I tried:
> 
> tee /home/andrew/myfile.txt
> ssh -l andrew andrew@mydomain.com -vvv

tee is used to rediect the standard output of a command, so an example
would be "find /dev -type f -print | tee /tmp/find_dev.output" - this
would send the output of find to both your screen and to the text file.

If you want to capture both sides of the conversation, try:

$ script /home/andrew/ssh_output  # Warning!  This starts a sub-shell
$ ssh ...
$ exit               # To exit from the subshell

Everything typed on your screen, including editting, and displayed
by ssh will be captured to the ssh_output file.
