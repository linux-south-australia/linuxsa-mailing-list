Re: SAMBA in RH7.2

On Thu, 2002-12-12 at 07:00, Daryl Pilkington wrote:
> Hi,
> I need to quickly get a SAMBA server working on RH7.2
> I've read:
> http://au1.samba.org/samba/docs/Samba-HOWTO-Collection.html
> 
> But I probably need something more specific to the RH7.2 install.
> 
> Where does Samba install itself to?
> How do I find & read the man pages?
> Where do I find/put the smb.conf file?

I've always found the rpm command to be quite useful in such
circumstances. Try:

  $ rpm -qa |grep -i samba

Which will tell you which samba packages (if any) are installed. If
there aren't any, check to see if it's been installed under /usr/local
from source.

  $ rpm -ql samba

Will give you a listing of files in the package called samba. In fact,
this exact example is used in IIT Training's Linux course.
