Re: SAMBA in RH7.2

On Thu, 12 Dec 2002, Daryl Pilkington wrote:

> Hi,
> I need to quickly get a SAMBA server working on RH7.2
> I've read:
> http://au1.samba.org/samba/docs/Samba-HOWTO-Collection.html
> 
> But I probably need something more specific to the RH7.2 install.

Well, did you want to build from sources or install an RPM?

RPMs generally put things in FHS/LSB defined places, whereas a build from 
sources generally puts things in /usr/local/samba/... unless you specify 
--with-fhs when you run configure.

Is this enough of a hint?
 
> Where does Samba install itself to?
> How do I find & read the man pages?
> Where do I find/put the smb.conf file?

If RPMs, then typically in places that man can find them. If not, then in 
/usr/local/samba/man. Modify /etc/man.config or whatever, in that case.
