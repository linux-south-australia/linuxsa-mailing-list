RE: Router Question

A while ago I set up a router under Linux (routes packets between a 128k
ISDN link and a 10Mbit LAN) and it works well.  Before I did set up the
machine I asked the same question of the list, and the general consensus
seemed to be that if you're a multi-homed ISP, or routing between 10/100Mbit
LAN segments a "real" router would be a better option.  If you've got an old
box lying around a Linux router might be a good cheap option for you.

Other advantages of using a Linux box is that it can do other stuff a
dedicated router can't...  Our "router" also runs low volume web services,
and some network monitoring tools for example.

Jim.
