> Running RH6.0. Need to set environment variable 'setenv' ...........for 
> particular application.
> Ran 'which setenv" -there is no command present in bin sub-directory.
> 
> Where can I find this command  ???

setenv is a built-in shell function for tcsh.  But RedHat systems give
you bash by default, so to set an environment variable you need to do:
VARNAME=stuff ; export VARNAME
or
export VARNAME=stuff

where:
  VARNAME is the name of the environment variable to set and
  stuff is the required contents of the environment variable.  You can
  use quotes if you need to.  E.g. export VARNAME="more than one word"
