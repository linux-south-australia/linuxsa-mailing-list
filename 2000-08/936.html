Re: DNS problem

>Recently upon visiting several sites I have noticed
>that the address I am visiting is resolving without
>the www eg.
>httP://www.freshmeat.net becomes
>http://freshmeat.net
>
>How is this done?
>Is this a DNS issue or browser redirection?

All this is, is that they have not set the host www to resolve to the same 
IP address or the same server as the domain freshmeat.net

This is done by say setting freshmeat.net to 123.123.123.123 and then 
setting the host www for freshmeat.net to 123.123.123.123 aswell. so it 
resolves as host.domain.NET

hence www is the host and freshmeat.net is the domain.
