Having had an extremely dodgy hard disk in the recent past I had the opportunity
to use badblocks quite often. I had a windows partition with lots of bad blocks
reported by scandisk but when I used the same disk space for linux and used
badblocks (both in the setup process when the partition is formatted, and later
as bad blocks just kept on appearing), the results were always different.

Most important of all, *badblocks* supposedly can be used to mark these badblocks
so that they will not be used by the filesystem. This it never did!!!

I've always wondered how this program could be included in the distributions, and
for quite a long time, when it seemed to be less than useful.

Mind you, I seem to remember that the man page states that it has only been
tested on diskettes???
