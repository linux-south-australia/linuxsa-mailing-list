Hi,
setenv is a csh (c shell) built-in command which is used to assign values to
environment variables
of the form:

setenv NAME value
I assume it is the PATH variable so:
setenv PATH '/home/yourpath/etc'

you could probably find out more info if you
man cshrc - the resource file for the csh
hope it helps,
