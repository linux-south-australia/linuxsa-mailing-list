Don't use WU-ftpd, as it has to may security issues, use PRO-ftpd.

With tcp_wrappers, in hosts.allow put the address of the ip address(es)
that you want to have access to your ftp service. eg

ALL : 192.168.100.6

or 

ALL : 192.168.100

with the /etc/hosts.deny file kept at 

ALL : ALL

hope this helps
