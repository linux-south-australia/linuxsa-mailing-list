Re: DNS

At 15:50 28/08/2000 +0930, you wrote:
>Has anyone ever setup a Redhat box as a secondary DNS server to a WinNT 4
>Server (primary DNS)

Yeah, plenty of times

>I can resolve names, but can't get the dns tables to copy down onto the
>linux box...

There are two things that I would do - firstly, ensure that the NT Domain 
Service has actually refreshed (haven't got a NT box here and can't 
remember exactly where it is).  Then make sure that your named.conf file is 
correctly configured, eg...

zone "domain.bogus.au" {
         type slave;
         file "sec/db.domain.bogus";
         masters { 123.123.123.123; };
};

and all the other really groovy stuff that you put at the top of the file 
to let you know where localhost is and the like...

Hope this helps...
