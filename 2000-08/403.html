When you install Linux, try telling the install program to install LILO
into the partition that Linux is on. You should be able to tell System
Mangler to boot straight from that partition.

Check out the LILO HOWTO. It'll help a lot. You may also want to
consider not using the 3rd party boot manager, as LILO is more than
capable. It's also free. ;-)
