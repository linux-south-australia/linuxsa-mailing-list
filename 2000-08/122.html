Re: copying accounts...

I did this when I replaced a redhat 5.0 box with a new 6.1 box (some
time ago now).  I wrote a perl script that grabbed the appropriate info
and then generated a bash script that called adduser, groupadd, and
groupmod.  I had to use the bash script because perl wouldn't execute
these commands  (security feature?).  It also set a default password as
I didnt realised the passwd file was backwards compatable (doh!) After
reading Daryl's way I think mine sucked :) But if it comes to it it can
be done and I can send source (although I have learnt much unix since I
tryed this and wouldn't do it like this now).
