Re: WU-FTPD Problems

Jake et al:

> "Leonard Teo, Macs" wrote:
> > I've set the hosts.allow to ALL:ALL:ALLOW (does
> > that look right??).

The TCP Wrappers toolkit seems to look at the name of the file -
hosts.allow/hosts.deny to determine which is which [full points to
anyone who can guess which file allows stuff and which denies it :-PPP].
Hence the ALLOW at the end is obsolete and may very well confuse TCP
wrappers totally.

It should be:

ALL:ALL

Mind you, setting hosts.allow to ALL:ALL really defeats the purpose of
TCP Wrappers in the first place!
