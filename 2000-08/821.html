Not officially, but NT logons have worked since Samba 2 came out (I
first tried 2.0.3). If you want better NT support you'd want Samba TNG,
which is a test branch that contains an almost fully working set of the
MS RPC crud. The Samba TNG stuff will appear in the main Samba branch
over time and eventually, we will have Samba 3.

Samba TNG also supports Windows 2000 logins in NT mode.
