And this is the whole point - the basic licence/registration has very
little to do with the performance of cable installations, but an awful
lot to do with the laws about safety - to make sure that you don't cause
a fire, blow up some equipment, make a roof collapse, etc.

I've done plenty of cabling before (under supervision, of course), and
having done the course is great - I've learnt all sorts of things about
the safety rules. If you do cabling, I'd highly recommend doing it :-)
