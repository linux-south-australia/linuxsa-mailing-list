Re: DHCP static-routes

Nick,
In my dhcpd.conf I use the option:

option routers x.x.x.x;

which sets up a default route... I can then do all the static routing at the
router .  This works for dhcpcd on linux.

Unless your trying to set up dhcp to configure a gateway (under windows?)
this should do the trick,
