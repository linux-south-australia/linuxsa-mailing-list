Re: Screen bigger than my monitor....

Hi,
In the file XF86Config (it's location is distribution dependant so
you may need 'find' to find it) there is a line that looks something
like

Virtual 1280 1024

This sets the virtual destop resolution. If the numbers given are bigger
than your physical screen resolution it will cause the problem you mention.
Try changing these values to match your physical screen resolution.
There are probably several different 'virtual' entries in the file so to be
safe you can change all of them. (you also need to be root to edit the file)

Hope this helps.
