you may have the wrong mouse configuration in the XF86Config file. Not
all mice are the same - they use different protocols, some being more
common (logitech 3button) than others (dodgy brothers $1.20 brand).

there are many ways to change the mouse configuration. one (easy) way os
tp edit the /etc/X11/XF86Config file directly, but you need to know the
protocol of your mouse.

My fav X config tool has been XF86Setup (XFree86-XF86Setup-3.3.5-3.rpm)
'cos it allows you to make mistakes and try out different mouse
protocols, without having to restart X. its stable too.
