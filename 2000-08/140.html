Robyn,

I have not been following this thread closely so this may not be what you
need. I've been doing some stuff with pppd as was perusing the man page and
read this.

pppd supports the usepeerdns option. The server assigned DNS addresses are
passed to your /etc/ppp/ip-up script as environment variables DNS1 & DNS2

With this you should be able to use server assigned dns and configure your
firewall from the ip-up.local script

For various reasons ip-up.local is the script that starts my firewall.
