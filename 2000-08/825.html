RE: IP Chains problem & squid

Are you using BigPond Direct for your internet connection?, if so there are 
some help files for setting up Squid to use telstra's proxy server have a 
look at http://www.direct.bigpond.com/proxy/proxy.cfm.
This is a direct link to the squid info page: 
http://www.direct.bigpond.com/proxy/proxy-squid-single.cfm

If it goes through your firewall and you are using bigpond direct then you 
need to enable port 3128 for the telstra proxy.

These settings are only applicable if you are using Telstra BigPond DIRECT 
not HOME, BUSINESS or ADVANCE

Hope it helps,
