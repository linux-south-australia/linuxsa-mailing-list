I think you'll find that these freshmeat solutions use ipchains as their
base.


  1.0.0.1 ------
  1.0.0.2 ------  box  ----- 168.56.34.7 (default route to internet)
  1.0.0.3 ------

to count stuff going *to* the 1.* machines:
ipchains -A input -d 168.56.34.7 -d 1.0.0.1
ipchains -A input -d 168.56.34.7 -d 1.0.0.2
etc

to count stuff going *from* the 1.* machines:
ipchains -A output -s 1.0.0.1 -d 168.56.34.7
etc

then, to get your info:
ipchains -L -v

or, for the hard core people who want a bit more speed, read it
directly:
cat /proc/net/ip_fwchains.
(I can dig up the explaination of this file if you want)
