> In Gnome/Enlightenment (ie up to but not including RH 6.2) you can use the
> Gnome Control Centre to bind keys to commands.  In the control centre,
> select Desktop/Window Manager/, Choose Enlightenment from the list on the
> right, click the button "Run Configuration Tool for Enlightenment" and go
> from there.
>
> Sawmill is a different story.  From the SawMill FAQ:
>   5. How do I bind a key to execute a shell command?
>
>      This isn't yet possible through the configuration tool. However
>      these bindings can be defined in your `~/.sawmillrc' file. For
>      example:
>           (bind-keys global-keymap
>             "C-M-ESC" '(system "xterm &"))
>      binds the event `C-M-ESC' to the Lisp code to start an `xterm'
>      subprocess
>
> If you are talking about the console, then gpm might be of use, though
> that binds commands to mouse buttons.
