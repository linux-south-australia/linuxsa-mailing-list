Re: Task scheduler....

not sure if this has been answered already but if not, I think you need to look
at using the program cron, which can be set up to automatically run programs. Do
a man cron or info cron to find out a bit more. If you want to make this program
start up every time your machine is booted, you need  to play around with
setting up the program in a daemon, and initialising it in the /etc/init.d
directory, which starts all programs listed in there at boot (at least the ones
that start with 'S' I think).

Please correct me if I am wrong anyone.

Hope this helps,
