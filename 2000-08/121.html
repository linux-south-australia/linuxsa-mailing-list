Swat is an inetd program. If its installed (locate swat), you will
probably have to enable it. By default it is disabled but listed.

(as root) Check /etc/services and make sure there is a listing there at
port 901. Then edit /etc/inetd.conf to uncomment the line refering to
swat. Do a kill -HUP on inetd and SWAT should work.

I've assumed here you have some knowledge in inetd, and processes. If this
seems like white noise to you, reply, and I'll specify in more detail
about these items.
