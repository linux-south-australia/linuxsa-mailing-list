Re: Ethereal

Hi,

Ethereal dump can be in libpcap format (file:save as...), and can be 
accessed through libpcap.

It sounds like it would be a good idea to select the packets you would 
like in ethereal using the filter option, then use the "only save selected 
packets" option in the "save as..." to save the packets into a straight 
libpcap dump.

A good way to work out filter options is to select the field you are 
after in a packet, then use "display:match selected" from the menus.  
Examine the "Filter:" text at the bottom of the window to discover how to 
refer to the field.

Next, write a c program using libpcap to extract the data component of 
the packets.  Examine the tcpdump source for fairly straightforward 
introduction to libpcap, start with main() in tcpdump.c.

Does anybody know of a good tutorial/API reference for libpcap?

Sean
