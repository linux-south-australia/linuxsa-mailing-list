Hi.

Assuming that:

- You have just been using a pretty much standard RH6.2 install.
- The font files are missing from where you are sure they used to be.
- X is dying because it can't find those basic fonts.

You might want to try getting the install CD and reinstalling the
font packages (I haven't installed RH since 6.0 but on Mandrake 7.1
all the fonts are in an RPM of their own).  Either try to force an
update or uninstall then reinstall the font packages.

Just an idea.
