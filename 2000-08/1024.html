Several things:

1) Do you actually have FEATURE(mailertable) and FEATURE(access_db)
enabled?

2) I think RedHat actually supply a makefile; you could run make in
/etc/mail, or do an ls -l on the affected files to make sure that
the db files built correctly.

3) Did you restart sendmail after making these changes? (this one's
a bit iffy, as I don't think you need to restart sendmail to get it
to reread changes in external DB files).
