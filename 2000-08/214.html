Re: Free DB options

I've had good experiences with Interbase, both on Linux & Windows. It
compares well with Postgresql featurewise, and I've found it much easier to
install.

Source available (under a MPL licence) at http://www.borland.com/interbase/

Documentation at
ftp://ftp2.interbase.com/pub/products/beta6.0/ib_b60_doc.zip

A Zope driver is available, so I guess that means python (?)
http://www.zope.org/Members/RETierney/gvibDA

I know there is another python interface around, too.

See www.interbase2000.org for plenty of mailing lists and resources.
