Hey there,

It is probably because you don't have a program installed that the configure
script requires. This could be something like automake.

You didn't say which distribution you are using. If you are using something
like RedHat, Mandrake
or SuSE (which I have experience with), or Debian (which I don't), then
there is a DHCP server as part of the distribution, which you just install,
and it creates all the files, etc, and you can skip down the HOWTO until you
get to the configuring section.

Anyways, if you want to build it yourself, try running:

./configure > ~/dhcp-debug.txt &2 > 1
(I think I got that redirect correct. What it does is outputs STOUT and
STERR to the file dhcp-debug.txt in your homedir).

Then email that file to either myself, or the LinuxSA list, and we'll see if
we can track the problem down further for you.
