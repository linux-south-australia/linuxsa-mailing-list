Well I don't know if this will help or not, but according to the script that
I am using for my firewall the following comments include ICQ..

#MASQ timeouts
#
#   2 hrs timeout for TCP session timeouts
#   10 sec timeout for trafic after the TCP/IP "FIN" packet is recieved
#   60 sec timeout for UDP traffic (MASQ'ed ICQ users must enable a 30sec
firewall timeout)
#
#
ipchains -M -S 7200 10 60

from this I conclude that you can either include on the windows machine ICQ
settings a 30 second firewall timeout; or increase the 60 seconds to something
higher at the ipchains level.  If sorry I can't be of much more help, I'm still
getting used to all this firewall stuff myself.
