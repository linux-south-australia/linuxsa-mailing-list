perhaps you need to venture into "console land" :)

open a shell

become super user:

$>su

change directory to /etc

$>cd /etc

edit the file lilo.conf, using pico:

$>pico lilo.conf

edit the file as needed, exit pico.

run lilo to reinstall it to where ever you specified:

$>lilo
* adding <image1>
  adding <image2>

voila
