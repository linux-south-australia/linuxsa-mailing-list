Re: count open files

Sk wrote:
> 
> Is there any command/utilities, that can list all those files that are open ? also can it count number of files open  ??

lsof - "list open files", not included by default in most distributions.

To count the number of open files, lsof|wc -l
The other useful feature of this program is that it maps IP connections
to open files, and it's relative obscurity means that it's not part of
most rootkits, making it more useful and possibly safer than netstat, ps
and ls.

from the man page:

The latest distribution of lsof is available via anonymous
       ftp from the host vic.cc.purdue.edu.  You'll find the lsof
       distribution in the pub/tools/unix/lsof directory.
