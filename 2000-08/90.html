Paul,

This is due to not all the environment being setup for you when you su - to
a user.

The environment variable DISPLAY is not set for you.

xman -display :0 &

and this should now work.

Note this doesn't work for gtk apps (I think it's gtk!??!), you must
use --display.

Rob.
