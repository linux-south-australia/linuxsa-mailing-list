Re: multiple IP

/sbin/ifconfig eth0 1.2.3.4
/sbin/ifconfig eth0:0 1.2.5.4
/sbin/ifconfig eth0:5 1.2.6.4
/sbin/ifconfig eth0:100 1.2.7.4

notice the ':x" after the device name. As far as I know, the number of IP
aliases you can set up for a machine is pretty limitless (at least in a
practical sense). If it doesn't work, then you need a kernel with IP
Aliasing included.
