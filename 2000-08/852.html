Re: Matlab on Linux?

There is a Matlab version that does run under Linux.
It seems that all the UNIX versions come on the same CD, but you still need a
license key
to run the thing, and if you setup a network license (I don't know how else
to do it)
it's bound to the machine that your running it on.

[It uses a 'flexlm' as the license manager, which makes use of the id of the
host that the
license manager is running on.  Sun machine used to have a fixed hardware id,
which sometimes
corresponded to the ethernet MAC address. Be prepared to give the Matlab
distributers these
details in order to get a license. Email me if you need a contact for them..
their based in Brisbane.]

The 'free beer' program is called Octave.  It understands matlab '.m' scripts
but uses
gnuplot for graphical output.

**Asside: I recently had to add an 600Mb of additional swap space on a linux
machine
so that Engineer could run his Matlab data processing script (Hi John!).  I
was scrimping
disk space from any partition that had some spare, and ended up with 6
additonal swap
files, after going through several iterations of "Is that enough?", "No, it's
down to 25Mb,
can you find any more?"

PaulS
