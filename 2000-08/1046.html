Evan!

The most up-to-date howtos are at:

* http://www.linuxdoc.org or alternatively http://linux.senet.com.au if
you prefer a local site

The security howtos relating to ipchains et al are quite up to date. Try
searching for words such as "ipchains", "security" and so forth. In
addition there is a book called "Linux Firewalls by Ziegler, publ. New
Riders" that details a relatively easy-to-setup ipchains firewall.
Furthermore you might want to start taking a look at net filter which
will be released when kernel 2.4.0 is.

The steps I would take are:

* become familiar with ipchains' syntax and usage (also it's masquerade
usage as well) - see man ipchains
* decide what services you want to use or offer
* setup an ipchains firewall

* become familiar with tcp wrappers - see man hosts.allow
* set it up as strictly as you would have it

* become familiar with the security issues of any services you wish to
run

Naturally, you would check your distributors web-site on a regular basis
for any security updates.
