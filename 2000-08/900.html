Samba - web based administration program

Hey all :)
I have setup a linux / samba based system acting essentially as a
replacement for a NT server.. it validates passwords, personal files are
kept on a mapped drive etc...

I was wondering for ease of use, if there are any web based administration
programs for samba available.  The reason I want this, is so this person
[who is relatively unfamiliar with UNIX], can easily add users [using a

and with the same password], delete users, and add / delete / modify shares.

This would have to be in a REALLY (let me stress the word REALLY) easy to
use web administration package.

I have looked at SWAT and WEBMIN, I am unsure if these are user
customisable, but they are too "bloatful"... any more options than nessecary
will confuse and entice this person to play with settings and could be
somewhat disasterous :)
