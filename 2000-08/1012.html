Re: Binding a script to a key on a Keyboard

Hi Mike,

I have worked it out:

bind Key '\C-p:"ls -al\C-m"'

Will bind control-p to ls -al\n

The rest is on the readline man page.

Is this what you were looking for?

> > >      binds the event `C-M-ESC' to the Lisp code to start an `xterm'
