Re: default route dropout.

Andrew Pullin wrote:
> 

> When I reconnect either
> manually or automatically, the system drops the default
> route, which is really frustrating.

As a starting point, ensure that

  /etc/sysconfig/network-scripts/ifcfg-ppp0

contains

  DEFROUTE="yes"

This passes the "defaultroute" configuration to pppd.  Use
"man pppd" for more info.

Good luck,
