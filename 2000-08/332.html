Not being a committed gamer, I can't rebuke that 100%, but I can say that
I managed to create a connection to battle.net from a Mac on the home lan
via my RH 6.1 masq gateway box. I've got the details here someplace...

#!/bin/sh  
      case "$1" in 
       start) 
        /sbin/modprobe ip_masq_autofw.o
        /usr/sbin/ipmasqadm autofw -A -r tcp 6112 6112 -h 192.168.x.x
        /usr/sbin/ipmasqadm autofw -A -r udp 6112 6112 -h 192.168.x.x
        ;; 
       stop) 
        /usr/sbin/ipmasqadm autofw -D -r tcp 6112 6112 -h 192.168.x.x
        /usr/sbin/ipmasqadm autofw -D -r udp 6112 6112 -h 192.168.x.x
        ;; 
       *) 
        echo "usage: starcraft.fw {start|stop}" 
        exit 1 
      esac

Yea, that was it. I do not know if there is a way of doing this for
multiple boxes on the lan, but that .x.x box had a transparent battle.net
connection, and played Starcraft for bloody hours! :)
