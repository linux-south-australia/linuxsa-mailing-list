Apache Forbidden Problem - FIXED

Hi all,

I finally found a fix to that problem which I was having with Apache 
bringing up a forbidden page.

FYI - Here is the solution

in httpd.conf
the line

<Files ~>
AllowOverride AuthConfig FileInfo Indexes Limit Options
Order allow,deny
Deny from all
</Files>
needed to be changed to
<Files ~ "^\.ht">
to stop it from forbidding all files with a .ht ending ie. html htm, etc.

That fixed it


Thanks to all those who offered me their help.
