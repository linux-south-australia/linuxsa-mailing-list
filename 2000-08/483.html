Re: Netcomm Roadster

Hi Brian,

Just tell me if I'm preaching to the converted here -

1. Have you loaded the usb stuff as modules or as part of the kernel?
2. If you've loaded as modules then load the modules with modprobe as it will 
load any aditional modules needed for usb to work.
3. Have you looked in /var/log/messages? You can check out what has been 
started with your current session, and go to the end of the file and scroll 
up from there.
4. To get my usb mouse to work you may need to load the 'hid' (human 
interface device) module.

Robyn
