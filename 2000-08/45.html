Re: Samba used to connect EvERYTHING?

It depends how old the Macs are.

If they are recent then run AppleShareIP on the Mac and
connect to Samba on IRIX.  See
  http://www.macwindows.com/ASIP.html

If they are older then either:
 - install an commericial SMB client on the Mac
 - install an AppleShare server and AppleTalk network protocol on IRIX
     See http://www.umich.edu/~rsug/netatalk/

As always Google is your friend for these kind of questions.  See
  http://www.google.com/

As an idea of what is possible under Unix, my computer is
currently serving the same files using HTTP (web browsers),
FTP, TFTP , SMB (Windows), NFS (UNIX), AppleTalk (Mac), and
IPX (Novell).  You can print using lpd (UNIX), Apple, Novell,
SMB, HTTP, Telnet or FTP.
