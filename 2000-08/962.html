Re: count open files

Sk wrote:
> 
> Is there any command/utilities, that can list all those files that are open ? also can it count number of files open  ??
> 

lsof will show *all* open files. lsof |wc -l will count *all* open
files. If I want to see how many files are open by students over the
network, I use:

lsof | grep "/shares/students" |wc -l
