Re: adding isa ethernet card

Kim Branson wrote:
> 
> We are trying to add a 2nd interface to a redhat 6.2 box.  This second
> card is an old 3com ISA 10mb card, but the system is not detecting it,
> apparently.  Are there any tricks? 

just a little linux magic...
without knowing what 3com card it is I will give a little
about using a ne2000 card..
Redhat (kudzu) no longer picks up ISA so you need to do something
like

modprobe ne io=0x300 irq=10 eth1
ifconfig eth0 203.12.34.44 netmask 255.255.255.0 up
route add default gw 203.12.34.1 

substitute the correct 3com module in place of ne
substitute the correct IP address and netmask
substitute the correct gateway

then add the the corrected lines to the bottom
of /etc/rc.d/rc.local

reboot and the interface will come up
