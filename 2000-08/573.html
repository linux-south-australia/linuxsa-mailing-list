Kerberos! (and windows :/ )

Hello!

I'm thinking of designing a groovy network with Kerberos authentication
happening.

A possible issue is that of authenticating clients who run Windows 9x.

I've done some web-searching and found several clients, but all of them are
available only to either students of the university web site I found the
client on, or employees of the Federal Government of the USA.

Searching for "kerberos" on tucows only listed one entry, and it was a
telnet program :(

Can anybody enlighten me on the status of Kerberos authentication client
software for Windows?  Does any free stuff exist?  Does any costly stuff
exist?

My other option is to use Samba as a PDC and have it authenticate using
PAM/Kerberos - assuming that will work.  But that kinda defeats the purpose
of using Kerberos :/

I'm also looking into using NIS or NIS+ - I haven't researched those much
yet.

Does anybody have other ideas for network-wide single-logon security
systems that will run on all sorts of OS's and are open-sourced/standarded?
