Re: c to java conversion

Kim Branson wrote:
> 
> Hi
> 
> does anyone know of a c to java conversion utility...
> i heard there might be a gcc type thing but can't find it anywhere.

You want to turn C code into java source code, or C code into java
bytecode? Or do you want to put a java wrapper around some C code (ie
JNI)?

There is a gcc thing, but it deals with turning java source code into
bytecde and bytecode and source code into object (ie native) code,
which doesn't sound like what you're after.

The gcj stuff is at http://sources.redhat.com/java/ . There are rpms
available.
