HI Dan

have a look at the dmesg log and check the line near the top which report
the colour depth. General the is the maximum colour depth you can run the
Xserver at.

To change this these is a kernel boot parameter (which is set in BootX or
yaboot) which adjust the colour depth.

I do not have that parameter on me right now but I can find it for you.
