Re: serial printer

ted bullen wrote:

> (***) Why does the baud rate in /etc/printcap not like anything above
> 19200 ? i.e. I can set 2400->19200 in the entry - and the baud rate
> changes accordingly. If I set 56k or 38k4 it is ignored.

This might be a legacy issue with the lp system.  Traditionally, Unix
didn't support speeds above 9600 (it had two speed settings, EXTA and
EXTB, which could have been assigned to any oddball speed that your
serial adaptor supported, such as 1200/75).  Linux, according to
<bits/termios.h>, can support some insane bit rates (whether your
hardware can is a different matter).  I'm not sure if this is the
issue you're experiencing, not having to ever set up a serial
printer under Linux.
