Re: pam-0.72-20.i386.rpm

On Wed, 2 Aug 2000, Richard Watkins wrote:

> Hi All,
> 
> I would like to download "pam-0.72-20.i386.rpm" from
> the Red Hat website without success.

There are numerous ways I can think of achieving this. One of them is to
stand on a chair with a CD ROM in your left hand and think *really hard*
about pam-0.72-20.i386.rpm being on that CD ROM. Then put the CD into your
machine and look for the file. It won't be there. Mission accomplished!

There are less practical alternatives which involve strenuous exercise and
destruction of valuable property, so I generally advise against these.

> Can I obtain this file elsewhere?

Certainly. Try ftp.redhat.com.au here in Adelaide and get the file
/pub/redhat/updates/6.2/i386/pam-0.72-20.i386.rpm. If you have ncftp
installed and aren't forcing ftp requests to go via a proxy then the
command:

  ncftp ftp://ftp.redhat.com.au/pub/redhat/updates/6.2/i386/pam-0.72-20.i386.rpm

should work and will put the file in the directory where you started the
command.
