> >Hey guys. I have a samba server running on my box and i cant get it to use a

> 
> The default is security=user, which gives what you observe.
> 
> If you set 'security=share', you should achieve what you want. Check out
> 'man smb.conf' or books like "Special Edition, Using Samba'.

In addition to the global setting 'security = share', setup the share(s)
as 'guest only = yes' and make sure 'guest account' is defined to be a
valid user for the share(s).  In this scenario, no password checking
will occur for the share(s).  From the description below, that sounds
like what you're trying to do?

> 
> >eg it will only share home dirs and you have to use your username and
