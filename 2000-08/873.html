DHCP static-routes

Hello everyone.

I'm using a DHCP server to allocate IP information to Windows 9x and NT
hosts on several networks.

One of the hosts needs to have an extra static route in its routing table.

After reading `man dhcp-options` I discovered that there's a line you can
put into /etc/dhcpd.conf  --  option static-routes x.x.x.x y.y.y.y;  which
will configure IP destinations and their relevant routers.

After doing all the right things, my IP is allocated correctly, as is the
DNS server, WINS server, default gateway, etc etc - but the static routes
are not :(

At first I thought - aha!  Windows bug!  But my linux box, using pump, and
even trying dhcpcd, doesn't pick up the static route either.

How can I configure these static routes?  Can anybody help me?
