Re: lsof system call

On Mon, 14 Aug 2000, Ted Fernando wrote:

> I was wondering if anyone knew of a system call to list currently open files 
> by a paticular process. the lsof program kind of does this but lists all 
> files and all processes and also requires root. is there a more basic system 
> call i can use in my program? thanks

If you find the process id then you can do it yourself. Just cd into
/proc/pid/fd and you will find all files which are opened by that process
