Re: Cabling Rules

Robyn Manning wrote:
> Do these new rules apply to running pieces of cable between a few pc's and
> making a small lan.

Yes, if the LAN is connected to a carrier network. This would include
having one of the PCs on the LAN dialing an ISP via a Telstra phone
line.

However, if the LAN is not connected to a carrier network, and provided
the cabling is not fixed or concealed then there would be no problem.

(As Damien suggested, concealed cabling would be running cable through
walls, roof spaces, under the carpet, etc. and fixed cabling is fixing
it in place with staples (yikes! Not on Cat 5 I hope!), cable clips or
ties, etc.)


As an aside, I know that TAFE were looking to run quick bridging courses
for the Alarm techs that will now need licences, and I suggested that
Computer LAN techs may need a similar course. I'll try to remember to
ask TAFE if anything is going to happen when I'm there tonight.
