Re: kernel version

Sanjeev Jha wrote:
> 
> I have RedHat 6.1 with Kernel ver 2.2.12-20 (as shown by `uname -a`)
> Is kernel 2.2.16 is later version of the above ?

It is usually difficult to say.

2.2.12-20 is the stock 2.2.12 kernel with 20 Red Hat specific patch
sets added.  Some of these patch sets can add significant functionality,
perhaps more than a later version of the stock kernel.

2.2.16 is a stock Linux kernel.  The current Red Hat version is
2.2.16-3, suggesting that Red Hat doesn't vary from the standard
one too much.

Red Hat put their kernel build configuration files in

  /usr/src/linux/configs

This is installed by the kernel-headers RPM.

If you build your recent standard kernel with the Red Hat settings, your
machine should boot and work fine.

Remember to configure LILO to give yourself a way to boot
the original kernel as a fallback measure.
