Re: kernel version

On Wed, Aug 23, 2000 at 08:30:48AM -0700, Sanjeev Jha wrote:
> I have RedHat 6.1 with Kernel ver 2.2.12-20 (as shown by `uname -a`)
> Is kernel 2.2.16 is later version of the above ?

Sanjeev,

First, there is no such thing as linux kernel 
2.2.18, 2.2.19, 2.2.20 etc., as far as I know.

So your kernel has nothing to do with 2.2.20.

As I understand it, what you have is a variation of 2.2.12.
That's about what RH 6.1 would have had.
2.2.16 came out only in the last couple of months (to fix
a very bad OS vulnerability in 2.2.x for 0 <= x <= 15,
as I understand it.


Right now, I'm using a few machines with this sort of thing:

2.4.0-test1-ac21

This kind of thing comes from /usr/src/linux/Makefile, quid vide:

VERSION = 2
PATCHLEVEL = 4
SUBLEVEL = 0
EXTRAVERSION = -test1-ac21

This gives 2.4.0-test1-ac21.

Take a look at the first few lines of your file
/usr/src/linux/Makefile if you have the source of the OS on
your disk.
If not, you could install the kernel source RPM from the
source disk and have a look.
