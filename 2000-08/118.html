Terminal emulation problem...

Hello, all:

On a RedHat 5.2 system, I used to get around Microsoft's crappy telnet
emulation by specifying "vt102" in the /etc/profile file. It doesn't work on
a Mandrake 7.1 system. The lines that I used to include near the end of my
/etc/profile file are as follows (I nabbed it from an earlier post to this
list...)
USER='whoami'
if [ $LOGNAME = $USER ] ; then
TERM=vt102
EXPORT TERM
fi


Are things done differently these days, or have I missed a point
somewhere...?
