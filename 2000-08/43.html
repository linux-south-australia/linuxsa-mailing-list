Re: How to mount Linux extended partition under FreeBSD

On Wed, Aug 02, 2000 at 02:17:28AM +0900, Richard Sharpe wrote:

 > I am finally getting FreeBSD to install properly on my system, even had KDE
 > running and Gnome using Afterstep (could not find Enlightenment :-() ...
 
/usr/ports/x11-wm/enlightenment.

 > Anyway, I now want to mount my Linux partition so I can compile up
 > Ethereal, but the main partitions are extended.
 > Under Linux they are /dev/hda5 and /dev/hda7.
 > How do I mount these under FreeBSD. Are they /dev/ad0s5 and /dev/ad0s7?

Pretty much, yes.  You'll need your kernel with "options EXT2FS".

Note that there's a documented caveat for that option:

  # Add support for the EXT2FS filesystem of Linux fame.  Be a bit
  # careful with this - the ext2fs code has a tendency to lag behind
  # changes and not be exercised very much, so mounting read/write could
  # be dangerous (and even mounting read only could result in panics.)
