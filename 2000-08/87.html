>It would appear that the attributes you have set for the printer share
>are actually appended to the share for the home directories. This is
>because you haven't assigned a share name to the printer share
>([myshare] is commented out). If you still have problems, try setting
>the 'printing = bsd' line to say 'printing = LPRng'.

Actually, it should be simpler than that. If you create a [printers]
section, Samba will pull in all the printers in your printcap file.

So all you need is:

[printers]
  path = /var/spool/samba

and make sure that /var/spool/samba exists.
