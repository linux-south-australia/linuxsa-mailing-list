YES !

I have used Kicq, LICQ, and currently using GnomeICU. Kicq is very basic,
and was installed by default with my distro so I just gave it a shot. I used
LICQ for about 3 months, I found it to be fairly similar to the windoze ICQ,
altough I found that it was automatically deleting a great deal of my
history over time. A number of skins etc are available for LICQ. LICQ works
off the QT Libraries, so if your running KDE it's the best ICQ client to
use, altough I think a working GTK LICQ client is working, it still isn't
quite as stable as the QT version, so if your running Gnome LICQ can be a
bit more of a bastard to get working properly.

I am now using GnomeICU packaed with Helix-Code Gnome 1.2, and I have found
it to be very stable, very good and by far the best Linux ICQ client I've
ever used.
