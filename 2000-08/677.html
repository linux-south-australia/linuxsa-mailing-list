Re: small distro for 486 laptop

You can install Debian (the base) or Slackware without
the need for cd-rom. With slackware, you'd need to
download the packages from the internet and put them
on floppies, although I'm not sure on the installation
process from there on. With Debian, you need a few
image files from the internet (such as root.img and
some other images, check www.debian.org for more) and
that should put the base system on for you. 
From there, you can install from the internet or if
you have a network, do a network install..

There are some floppy linux distributions but they are
extremely limited.
-EITHER WAY-.. check
http://www.linuxhq.com/dist.html
there's a list of distributions there

As for the laptop installation guides, try
http://www.cs.utexas.edu/users/kharker/linux-laptop/
