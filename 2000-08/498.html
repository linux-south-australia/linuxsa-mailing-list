On 8/16/00, 1:00:55 PM, Matthew Tippett <mtippett@ticons.com.au> wrote 
regarding Re: Increasing Swap Space:


> On Wed, Aug 16, 2000 at 10:44:16AM +0930, Michael Neuling wrote:
> > > Does anyone know how I could go about increasing my swap space 
without=20
> > > messing with data on my other partitions?
> > > Any help hugely appreciated
> >
> > The following will create a 256 MB swap file and mount it.
> >
> > dd if=/dev/zero of=/tmp/swapfile bs=1024 count=262144
> > chmod 0600 /tmp/swapfile
> > mkswap /tmp/swapfile
> > swapon /tmp/swapfile
> >
> > Change the first line it you want a differnt size.
> >
> > You can do this with as many was files as you like.

> The one other useful thing which Michael didn't add is that to make 
this
> happen on boot, you should add the following to /etc/fstab.

> /dev/sda2       swap            swap    pri=0           0       0
> /tmp/swap       swap            swap    pri=0           0       0

> The only proviso is that the place where the swapfile is must be 
accessible
> at boot.  This means that it should be on the / partition.

> The other thing to be aware of is that without setting the priority 
assists
> the swap daemon working out what swap file to give preference to.
