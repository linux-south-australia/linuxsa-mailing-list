Open a command terminal (or if you are already in one) and "su -" to
root (if not already root) and run the command 'ntsysv' and arrow down
until your "cursor" is at the iptables box and then hit spacebar then
tab then hit spacebar again.  It will no longer start at boot up.  I am
sure there is an easier way but I don't know it / use it.

What it should look like in `ntsysv` Before:
[*] iptables

What it should look like in `ntsysv` After:
[ ] iptables
