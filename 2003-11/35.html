RE: Sendmail Aliases

Jim,

> If I try to add the above to my aliases file, I get this when I run
> newaliases...
> 
> /etc/mail/aliases: line 38: testme@company2.com.au... 
> Warning: duplicate alias name testme

This is because you are using the local aliases file...

Try using /etc/mail/virtusertable

testme@company1.com.au:  test1
testme@company2.com.au:  test2@exchange.company2.com.au
testme@company3.com.au:  test3@exchange.company3.com.au

First is aliased to a local user, second and third are aliased to
external users.

Looking at what you are trying to do this would be a better way of doing
it than using the "local" aliases file.
