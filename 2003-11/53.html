Howdy

>A neighbour of mine would like me to install Win XP on his machine.
>I have allready installed Win 98SE, and Mandrake 9.1 on this machine,
>but am not sure how the boot loader will accept WinXP.

The boot loader will accept it but most likely windowsXP will clear the boot
loader when you install it. The same effect as typing in "c:\fdisk /mbr"

>Win98SE and Mandrake 9.1 are already on separate partitions, 
>and have a 
>spare fat 32 partition

XP would be best to reformat that pertition - unless you really really want
that partition to be fat.

>XP could be installed on. Do I have to put another line in lilo?
>Any help would be appreciated, thanks.

Best thing to do would be to get yourself a linux boot disk... it will be
your best friend. Do your XP install and don't expect to see linux again
until you use your boot disk and re-run lilo (after modifying your
/etc/lilo.conf file).

Good luck.

Tip: Install windows first whenever you have the choice!!!!
