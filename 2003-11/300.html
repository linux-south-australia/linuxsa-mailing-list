Fellow Linuxians,
 
I have an 'interesting' problem I'd like to share with you ;-)
 
At our University (UniSA) we are trying to get wireless networking going at
various campuses.  You may (or may not ;-) be aware that we have implemented
wireless at the CityWest campus via a CISCO box.  This box authenticates
incoming users against a RADIUS server and if accepted sets up a VPN into the
University's wired network.  Sounds easy hey!?
 
Unfortunately Underdale campus doesn't warrant this expenditure and as such I'm
looking to emulate this functionality via a Linux solution.  Dan shearer
initially put me onto the FreeSWAN product as a possible solution here.  I have
now run this software up on a RedHat 8 system and obtained the relevant private
keys to access the Radius server etc.  I am not sure how to now configure
FreeSWAN to authenticate and then manage the tunnels.  The primary wireless
clientele will be Windows PPTP clients unfortunately.
 
Has anyone used FreeSWAN or tried something similar using another product??
Never done this before!
 
 
Many thanks,
 
Chris Trowbridge
