[OT] CSS Application

Hi,

Something I've always thought would be useful for a web designer would be an
application to easily modify CSS stylesheets (redundant acronym usage, d'oh)
on-the-fly, by 'calibrating' a live page displayed in a browser with a color
palette.

That is, some application renders your page, just like in a browser, and in a
panel or something elsewhere on the screen, it'll show all the CSS styles
used, and you can actually click on the page itself, which will select the CSS
style.  You can then change colours (and as an added bonus, font types, other
attrib's), which will reflect back on the page (& for any other objects using
that CSS style, too).

I've always done this manually, and given cleanly written CSS, it's not
terribly hard to do, but an application like the above could be damn handy;
especially considering right now I'm in the middle of trying to clean up some
pretty horribly authored stuff :-(

Anybody know whether something like this exists; Windows, Linux, Java, or
otherwise?
