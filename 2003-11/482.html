Re: Wireless card query

this is on redhat9.0 (so might be different for you)
not sure about wireless cards but this is how i add/change network cards

[user@hostname network-scripts]$ cat 
/etc/sysconfig/network-scripts/ifcfg-eth0 
DEVICE=eth0
BOOTPROTO=none
ONBOOT=yes
IPADDR=192.168.0.1
NETMASK=255.255.255.0
USERCTL=no
PEERDNS=no
TYPE=Ethernet
NETWORK=192.168.0.0
BROADCAST=192.168.0.255

these files are accessed when calling the /etc/init.d/network script
so you'll probably want to play around with or create the 
/etc/sysconfig/network-scripts/ifcfg-eth1 file which should do the 
/sbin/ifup on boot time
