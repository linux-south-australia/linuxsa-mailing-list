Re: Linux success story

Just want to publicly thank Glen Turner for his RH9: USB Memory Key
article.  That introduced me to devlabel which makes this so easy to do...

<quote who="Michael Davies">
>
> I have a JVC DV video camera (GR-DLV510EA), which wasn't supported
> under Linux 2 years ago, and so I needed to use Windows 98 to get
> (still) images from it (video support unknown - I didn't have firewire
> h/w).
>
> Tonight, I went to download images to another PC, this time one with
> Windows XP. To my surprise, it didn't work. Spent some time following
> this up - it appears JVC isn't supporting this device on Windows XP as
> it's 2 years old.
>
> Just shows another reason why OSS is important - to support devices
> that manufacturers don't want to.
>
> On the off-chance I tried Linux again.
>
> dmesg gave me:
>
> scsi1 : SCSI emulation for USB Mass Storage devices
>  Vendor: JVC       Model: DV CAMERA         Rev: 0000
>  Type:   Direct-Access                      ANSI SCSI revision: 02
> Attached scsi removable disk sda at scsi1, channel 0, id 0, lun 0
>
> Hmmm, looks _very_ promising:
>
> [mrd@lothlorien mrd]$ sudo devlabel printid -d /dev/sda1
> S:JVCDVCAMERA
>
> which only means it's missing a :
>
> sudo devlabel add -s /dev/jvc-dv-camera --partnum 1 -u 'S:JVCDVCAMERA'
>
> and we're set to rock and roll.
>
> Woohoo! My DV camera isn't supported under Windows XP, but it is under
> Linux. Very nice!
>
