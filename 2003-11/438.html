What statement exactly are you suggesting proves that there IS stolen code?

Nothing I have seen suggests that there is, except for the flunky
statements without any effective evidence from SCO.

The fact that the OSS community says 'show us the code so we can remove
it' is not evidence it is in fact in there, it's evidence that it's
_possible_ that there is code in the kernel that came from Unix, or
perhaps looks identical to Unix code but was independantly created. It's
a statement of good faith by the community, nothing more.

It is just as possible, perhaps more so, that there is code from Linux
in SCO Unix. This certainly happened in the BSD case.

The only 'proof' of anything is that it is not possible to emphatically
know either way unless you have access to unix code, and the kernel
developers don't want to see it for obvious reasons.
