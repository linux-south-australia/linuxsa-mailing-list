Re: Can I (externally) host my own domain?

Hi David,

another way to do this is to use your providers webspace.

1. get them to create a virtual host on their web server, which points
to your ~/public_html directory on their web server.

2. write a script that gets your dynamic IP address each time you
connect, writes an html file and uploads it to the ~/public_html
directory. The html file would have a redirect header so that anyone
requesting your domain ends up at your dynamic ip address.

script for automated ftp
~~~~~~~~~~~~~~~~~~~~~
#! /bin/bash

ftp -n  <hostname> <<EOF
user    <username>      <password>
<ftp commands>
quit
EOF
~~~~~~~~~~~~~~~~~~~~~
(parts of) script to get dynamic ip address
~~~~~~~~~~~~~~~~~~~~~
#! /bin/bash

# get the ip address
ip=`ifconfig | grep -A 1 ppp | grep inet | sed -e 's/^.*addr://' -e 's/
.*//'`
url1='http://'$ip'/~me/'
~~~~~~~~~~~~~~~~~~~~~

hope this helps
