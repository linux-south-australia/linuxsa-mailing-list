RE: Windows NT 4.0 + Linux Question

Quantum Fireball drives have a high failure rate.  When they fail, they
mostly exhibit clicking or clunking noises.

As I think someone else mentioned, the NTFS file system is probably
using a portion of the drive that FAT isn't using.

You could try creating a small dummy partition at the start of the drive
and creating your partition for NT further down the disk.  I don't think
NT4 needs to boot from the first partition like 9x does.
