> RedHat 8 system and obtained the relevant private keys to access the Radius server etc.  I am not sure how to now configure FreeSWAN to authenticate and then manage the tunnels.  The primary wireless clientele will be Windows PPTP clients unfortunately.

> Has anyone used FreeSWAN or tried something similar using another product??  Never done this before!

You're going to have a hell of a time trying to get FreeS/WAN (which is IPSEC based) to talk to Windows PPTP clients.  IPSEC is far a superior VPN solution, but PPTP is far wider supported.

I've done this plenty of times with wireless implementations; checkout poptop, it even has support to authenticate back against a radius server, which worked fine last time I tried it (albeit needing some source code hacking, but it's probably fixed now).
