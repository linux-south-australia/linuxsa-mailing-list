RE: [kinda OT] printing prn files under windows

I have had joy printing postscript files direct to a Windows network printer
with something like this

print <file> /D:\\<server>\<print device name>

If the printer you want is just the local LPT1: I think just print <file>
should do it.

If you can't get this going, check back with me. The above command is inside
a little batch file on my work computer I wrote long ago and I haven't
looked at it for a while.
