Hi,

What *exactly* needs to be done to the files?  You haven't been completely
clear whether you want this totally automated, or there's things that have to
be done uniquely to each individual file (which seems to be suggested).  What
*exactly* are you trying to accomplish?

If your concept to do the editing remotely via an X11 session is to make it
faster; it won't, it'll be godawful stupid slow to use if bandwidth is an
issue; this isn't a realistic option.

You're probably going to find most Linux Desktop Publishing tools are going to
render your Quark files wierdly, too.

If you want to do something along the lines of uploading the exported Quark
PDF files, resizing them all to a particular resolution, saving them as a
PNG8, and generating php files to refer to them (ack, why do you need to
'generate' a php file - doesn't this defeat the purpose of it?), you can
certainly do that with something like imagemagick.
