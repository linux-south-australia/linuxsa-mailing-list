Re: How to change the dir of a shell from a script

On Wed, 2003-11-26 at 19:14, Wiechert, Holger-Fa.IKS wrote:
> Typical newbie question, I guess:
> I want to use a "cd" as the last command in my bash script,
> such that the current directory of the shell starting the=20
> script is changed by that command.
> However, when the shell exits, the shell is still in the
> same dir.

That's right.  A child can't change the parent's
environment.

Perhaps you should be running an "include" script
instead.  These are run using the syntax

 . myscript

They are not as robust as shell scripts (as they
are not children of the shell, and thus can stuff
up the shell if they go wrong).
