Re: Vmware No Full Screen

I have it working on a geforce 4 mx420 with a standard install of
nvidia's drivers. 

XF86config parts:

Section "Module"
        Load  "dbe"
        Load  "extmod"
        Load  "fbdevhw"
        Load  "glx"
        Load  "record"
        Load  "freetype"
        Load  "type1"
EndSection

Section "Device"
        Identifier  "Videocard0"
        Driver      "nvidia"
        VendorName  "Videocard vendor"
        BoardName   "NVIDIA GeForce 4 (generic)"
EndSection

Occasionally after using full screen mode for a while it decides to die
and not go full screen again until restarting vmware, but I think thats
a different issue. Its VMwareWorkstation-4.0.2-5592 under redhat
8/xfree86 4.2.1.
