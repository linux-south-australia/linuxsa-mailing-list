Not necessary. Windows XP will eradicate the part of the boot partition 
that points to LILO anyway.

Install XP as usual, make sure it boots as intended, and then run 
Mandrake install again, this time in repair mode, to get it to write you 
a new LILO entry in the boot sector, and add WinXP as a boot option to LILO.
