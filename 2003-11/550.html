Ryan means (I think) that mutt uses the local mail server to
send mail but your kmail is presumably sending directly to
a remote smart/relay host.  Ie. I think the problem is your
local mail server is not configured right rather than mutt.

Dunno about suse, but debian (using exim or postfix) has the 
option upon install to set it up as a "smart host" meaning it 
just forwards non-local mail on to your ISPs mail server, which
sounds like what you want.
