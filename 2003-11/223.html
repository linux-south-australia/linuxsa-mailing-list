Re: Fedora - what else to download?

Go to freshrpms.net and install apt for Fedora.


Use freshrpms.net, fedora.us, and "Dag's apt repository for Fedora" in
your sources.list.

Install totem.  That's your MS Windows Media Player equivalent - MP3,
MPEG4, DVD etc.

Fedora comes with OOo 1.1 (Office) and Epiphany(Web).  Use them.
