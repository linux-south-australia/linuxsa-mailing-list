No need for a static IP with the solution I'm suggesting.

1. your domain name should point to your ISP's webserver. You will need
to redelegate your domain to your ISP webserver's IP address.
2. you get your ISP to set up a name based virtualhost. The entry they
make in their httpd.conf file points to a directory within your existing
space on their server (eg. public_html)
3. you create the scripts I suggested in the previous post
4. every time you receive a new IP you ensure the scripts are run
(automated)

Email me offlist if you are still unclear.
