[OT] RE: Windows NT 4.0 + Linux Question

 >How big is the drive ??? What Service pack revision is the 
 >install CD ?? the 
 >NT4 ide driver has some limitations on how big a drive it can 
 >handle during the install (yeah it's stupid I know). There is 
 >a work around ... if I find it I'll forward it on. I think 
 >you need to replace a driver with one from a later service 
 >pack or some such nonsense.

There is no limit (no limit anywhere near 4 Gigabytes anyway) for the
size of the drive.  There is a 4 Gigabyte limit for the installation
*partition*.  This shouldn't be causing the problem though, setup stops
you if you try to exceed this.

re: work around - There is an ide patch you can load a disk at the "F6
for additional device drivers" stage which increases the maximum to 8
Gbytes.
