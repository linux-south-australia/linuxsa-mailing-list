RH8 - gnome2 vs. mouse

howdy,

i've got a system running RH8.0 here, with pretty much all the recent 
updates & patches installed. it runs as an internet gateway and 
sometimes does duty as a workstation, running gnome 2.0, openoffice, 
mozilla, gaim, etc. as it's mainly a gateway box, all the non-essential 
services are turned off.

the peripherals are just a PS/2 keyboard, PS/2 mouse (generic MS 
compatible wheel mouse), and the modem. crossover ethernet cable to hook 
into the other machine.

the issue: every time i log into X (gnome), the mouse reverts to being 
very insensitive movement-wise. as soon as i run redhat-config-mouse and 
pick "Intellimouse (PS/2)", it goes back to it's normal sensitivity. the 
workaround is, of course, simply to run redhat-config-mouse and click OK 
every time i log in, but that's a little annoying. anyone have any ideas 
how to make it retain it's sensitivity setting?
