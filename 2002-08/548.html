Switching to iptables should merely be a matter of issuing as root the
following commands....

chkconfig --level 2345 ipchains off ; service ipchains stop ; rmmod
ipchains

Then enabling iptables is simply... chkconfig --level 35 iptables on

I use and recommend MonMotha's firewalling script from
http://monmotha.mplug.org because I've never liked redhat's firewall. To
use the MonMotha script you merely edit a few lines at the top of the
script (it's well commented), chmod +x it then run it normally (i.e.
./rc.firewall-2.3.8-pre7). Then "service iptables restart" should save
the rulesets for future boots.

Of course you can use any firewall you wish, even redhat's.  :)
