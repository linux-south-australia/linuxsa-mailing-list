Re: Linux Redhat + Bind + Apache = Webmaster

Take a trip to www.linuxdocs.org and start reading. Groups such as this
one will usually help if you have a go at working things out yourself
first, have a search on google for the answers you seek, then as a last
resort ask. When asking you need to include version of linux, version of
the software in question and enough info for people to be able to help.

Of course reading, while playing with a test server is proabably a good
idea too. Dont read and not play, you probably wont remember it. 

Oh, turning of HTML email will hopefully stop you being drawn and
quatered on the first day too :)

Hope this helps,
