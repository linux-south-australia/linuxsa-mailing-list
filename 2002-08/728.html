from http://www.firstlinux.com/articles/cards/permissions.shtml

-------------------------------------------
Permissions on directories

Permissions on directories are different to permissions on files. Here

    * r indicates the contents of the directory can be listed
    * w indicates the contents of the directory can be changed
    * x indicates that users can "cd" to that directory, read, write and
      execute programs in the directory. 
-------------------------------------------

apache can't read any files from the directory until the execute bit is
set....
