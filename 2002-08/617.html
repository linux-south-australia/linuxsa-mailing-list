Squid Entries

I have a little script that reads my squid access.log into an SQL
database (because I could). It shows:

mysql> select distinct clientAddress from accessLog;
+----------------+
| clientAddress  |
+----------------+
| 127.0.0.1      |
| 10.0.0.102     |
| 66.136.215.114 |
| 10.0.0.100     |
| 212.142.148.7  |
| 218.71.198.123 |
+----------------+
6 rows in set (0.45 sec)

mysql> select id, entryDate from accessLog where clientAddress like
'218.71%';
+--------+------------+
| id     | entryDate  |
+--------+------------+
| 134820 | 2002-07-10 |
+--------+------------+
1 row in set (0.21 sec)

mysql> select id, url, entryDate from accessLog where clientAddress like
'218.71%';
+--------+-----------------------+------------+
| id     | url                   | entryDate  |
+--------+-----------------------+------------+
| 134820 | http://www.yahoo.com/ | 2002-07-10 |
+--------+-----------------------+------------+
1 row in set (0.18 sec)

mysql> select id, url, entryDate from accessLog where clientAddress like
'2%';
+--------+----------------------------+------------+
| id     | url                        | entryDate  |
+--------+----------------------------+------------+
| 131027 | http://www.whatismyip.com/ | 2002-07-05 |
| 134820 | http://www.yahoo.com/      | 2002-07-10 |
+--------+----------------------------+------------+
2 rows in set (0.18 sec)
