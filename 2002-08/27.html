On Thu, 1 Aug 2002 20:20, Andy Zivkovic wrote:
> Does anyone know of any packages that can do this type of thing? I'd like
> something that isn't password protected (I'll set up iptables rules so only
> 10.0.1.x can use it), so I can set up the home page of IE to the web page,
> so it's really, really easy to connect/disconnect.

I have an effectively permanent ppp connection but a quick search on freshmeat 
for "ppp remote control" gave a few results which may be worth trying:

http://freshmeat.net/projects/pppremotecontrol/?topic_id=90

PPP Remote Control is a small Perl CGI program, perfect for a masqueraded LAN 
where a fully automatic diald (or similar) setup is not desired. It can 
start, stop, and give a status report on the pppd connections, all from a 
single control page. Any setup listed in /etc/ppp/peers can be started with 
the click of a button. 

http://freshmeat.net/projects/webppp/?topic_id=150

WebPPP lets you set up your web server to act as a remote control panel to 
start, stop, and view your Internet connection on an ipmasq server. Being web 
based, you do not need to install any client software on the other machines 
in your network (granted that they already have a web browser). It takes 
advantage of your already working PPP setup from the Red Hat Control Panel.

(I wouldn't let the redhat thing deter you, may still be a decent piece of 
software)
