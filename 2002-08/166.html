Re: Version Blues

Trust me, the easiest way to get wine to work is to go straight to the
CVS. I havnt tryed vanilla wine for a while, Ive been doing the cvs
thing with wineX, with the added advantage that you can play a lot of
windows games with it. I just checked the site, and it supports warcraft
3 100% now (among others).

<http://sourceforge.net/cvs/?group_id=17631>

Just go to that link, follow the instructions and do the ./configure ;
make && make install thing.

I have had troubles with one of the openGL header files missing, but
that stuff must just be somewhere else, because I put a // at the start
of the #include and then it all worked great. Ive heard about so many
problems with the pre-packaged versions, I wouldnt bother.

Ant
