RE: X Scaling?

>My laptop isn't fast enough to run counterstrike at any higher res than
>640x480, but on a 1024x768 LCD, that gives me a tiny window in a big
>box. One of my friends is using XP and his *640x480* CS game fills the
>(1024x768) screen. I'm guessing that there's some kind of software
>scaling the 640x480 game output to 1024x768 in realtime.

With Compaqs it's a function key combination, function key being the "Fn"
key,
Fn + T will stretch the screen to the entire display, looks pretty ugly but
its better than a tiny window.

Normally it's a hardware setting rather than software (drivers/X setting
etc),
so check up on the user guide/website etc, if its a well known brand the
online
support is usually pretty good, if its a no-name brand.....well...... ;)
