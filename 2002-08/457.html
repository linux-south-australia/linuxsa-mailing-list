Re: Any interest in IPv6 as a meeting topic?

John Edwards wrote:

> Just curious, is it looking like there will be any 'must have'
> applications for IPv6 in the near future, or even today?

No.   IPv6 is a network layer, so there are no applications
which run on IPv6 which won't run on IPv4.

IPv6 does have some better network-layer characteristics,
such as self-configuration, multiple addresses per interface,
and a nice address structure for wireless inter-cell handover.

> There still seem to be a lot of protocols and software that just don't
> care about IPv6 (by using things like a 4 byte value for address) and
> this lack of accomodation makes it look like not many people are taking
> it seriously.

I suggest you configure IPv6 and be pleasantly surprised.
All the major applications and protocols have IPv6 support,
mainly out-of-the-box in recent distributions.
