Re: Restricting FTP Access

Actually wu-ftpd is rather easy.

Just edit /etc/ftpaccess and add in something like:

guest-root /home/user user
restricted-uid user

Just change user with the name of their user and obviously /home/user as
their home directory.  man ftpaccess is also useful.
