RE: Up2date Question

Martin,

> How can I stop up2date from doing this?

Most probably the Redhat Network Services daemon is running, you can
stop it by doing:

service rhnsd stop

You can still run up2date manually, up2date -u , or if you want to leave
rhnsd running but with a longer time between checking for updates
configure it for 24 hours or something. 

rhnsd -i 1440 , to set it for daily updates, or you can edit the config
by hand, usually located in /etc/sysconfig/rhn/rhnsd .

If you definitely don't want RHNSD running fulltime, you might also like
to check that it isn't in the system startup.
