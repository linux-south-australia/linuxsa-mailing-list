David, Ben et al:

> > what distro do you think is best for a low-memory machine (i.e. 16 or
> > 32mb)? it's an old box, and i want it just to sit and serve files via
> > samba. no desktop or anything, i just want to run samba (and ssh, of
> > course). it's got a 6G HD in it, and i'll be adding at least a 30G, and
> > possibly one more.
> 
> > would freeBSD be better for this maybe? (i've never used *BSD, it'd be

Any version of Linux not running X and a lot of unnecessary services
would be satisfactory, including:

* RedHat
* Mandrake
* Debian
* Slackware

You might want to rebuild its kernel and take out as much as you
possibly can, but a default kernel -- especially if one gets the
architecture to match (my machine claims it's an i686 when it's really
an athlon) -- should run it.
