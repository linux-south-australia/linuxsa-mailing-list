Re: i845 chipset

Nah, they don't have 'em.
According to the xfree86.org team, support is present in CVS now, so the 
next release of XF86 will include support.

I managed to get the soundcard working (only 2 out of 6 channels function, 
but thats enuf for stereo MP3s).

Comment out the line like this:

break; /* I think this works, if not ready stop */ 

from i810_audio.c
This bypasses the error check which was triggered.

I know that seems like a bit of a kludgey thing to do, so don't try that 
on anything important and hold me responsible.
The machine hasn't crashed yet though, but its only been playing sound for 
20 minutes so i wouldn't call that a definitive test.

-Toby
