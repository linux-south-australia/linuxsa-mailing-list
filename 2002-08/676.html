On Tue, 27 Aug 2002, Andrew Lord wrote:

> Just wondering if anyone has successfully installed html2ps on Linux Mandrake 

Nope. 

But, just in case you found yourself up this particular gumtree in the
process of solving the wrong problem: If you are doing one-off conversions
and you don't want control over the layout, a really simple way to do this
is to open the file in a web browser and then print to a file. That's if
you want a result that looks a bit like it might on the screen. The reason
html2ps exists is that people sometimes want to hardwire in how the HTML
is to be rendered on the page in a way that makes more sense for hardcopy.
Just a thought.
