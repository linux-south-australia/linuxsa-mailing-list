Try running the ./install script without any arguments. It asks questions about where you want to install stuff:

By default all files will be installed in subdirectories 'bin', 'lib',
and 'man' in a common directory. Is this OK? [y]: 

Enter the name of this directory [/usr/local]: 
Directory /tmp/moocow does not exist, create? [y]: 
Created directory /tmp/moocow
Created directory /tmp/moocow/bin
Created directory /tmp/moocow/lib
Created directory /tmp/moocow/lib/html2ps
Created directory /tmp/moocow/man
Created directory /tmp/moocow/man/man1
Created directory /tmp/moocow/man/man5

You can see that it creates the $prefix/lib/html2ps directory (which is where it expects to find the html2psrc file). You'll notice during this process the following:


Installing html2ps in /tmp/moocow/bin
Installing html2psrc in /tmp/moocow/lib/html2ps
Installing html2ps.1 in /tmp/moocow/man/man1
Installing html2psrc.5 in /tmp/moocow/man/man5
Installing hyphen.tex in /tmp/moocow/lib/html2ps
Installing html2ps.html in /tmp/moocow/lib/html2ps

It's copied the sample html2psrc file (called 'sample' in the source tree) to the correct place (/tmp/moocow/lib/html2ps/).


If you don't want to re-install it this way, just copy the file called 'sample' from the source tree to whereever it expects to find it.

Hope it helps,
