Hi.

The puter doesn't have a ps/2 port, or you just don't have a ps/2
keyboard?

If the former, ( no ps/2 port ), you will need a boot disk that has usb
keyboard on by default.

It might be best to borrow a ps/2 keyboard if the latter, because if you
cannot type, you cannot fix. 

If you get to a console, and have a working keyboard, try:  insmod keybdev

I'm not 100% sure how to get to a console in Mandrake install, but you
could try Alt F2 or Alt F3  You are looking for a command line terminal
with a # prompt:

#
# insmod kebdev
