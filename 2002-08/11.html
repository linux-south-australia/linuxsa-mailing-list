It is quite likely that you are running a different version if you're
running a different version of RedHat (from memory, you're running 7.2?)
Every time the program is compiled it will change the md5sum, as I believe
the date/time/host it was compiled on is embedded into the end program,
which is obviously changing (just the time for example).

I just had a friend here sitting next to me do an md5sum on a RedHat 7.2
box and it came back with the exact same md5sum, so it looks like your's
is ok.

For your info, the sum I posted is from RedHat 7.3 on an i386 machine.
