RE: eth interfaces

The manual way to do it would be

edit /etc/modules.conf
hash out the line alias eth0 FOOBAR
add alias eth0 yourNIChere
write
reboot

where yourNIChere is a driver located in
/lib/modules/kernel-version/kernel/drivers/net/

u could also insmod and rmmod to do it. check the man pages out.

As far as using kudzu, rpm -e kudzu :)

Hope that helps
