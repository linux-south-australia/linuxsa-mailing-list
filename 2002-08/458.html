Re: Any interest in IPv6 as a meeting topic?

Michael Gratton wrote:
> 
> I'd say the killer app is having a publically accessable IP address for 
> all of your hosts - no more NAT.

Each subnet, including dialups, gets 16,000 addresses.

The important point being that there is no longer an
address shortage, so you can think of addresses in
different ways.

For example, there's enough address space for an allocation
for every meter on earth.  Packets for that address allocation
can be routed by GPS coordinates.  This solves a whole stack
of otherwise-tricky problems, especially for remote locations
and mobility applications.
