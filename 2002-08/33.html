RE: remote ppp0 control

you can can configure it so that

1. it turns green on all clients
2. it turns green on somme clients but not others
3. only your client turns green

the connection status is connect to a user name that the client uses (this
does not need to be validated but can be). If the clients have the same user
name then they will go up and down together.

the interface (ppp0) will stay up as long as there is a client that is
"green" but can be configured so that a user can overide and disconnect all
clients

> You send in
> a connection request with a client and it brings up the interface (in your
> case ppp0). The bonus is that it has a great windows client that has a
> pretty little picture in your task bar, red light for off, green light for
> on. To connect you just double click on the red light, to disconnect you
> double click on the green light, my parent have no problemss :-)
