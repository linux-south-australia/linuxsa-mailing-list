Thursday, August 1, 2002, 8:20:40 PM, Andy wrote:

> I'm setting up a gateway to control ppp0 (56k modem). It needs to be easy
> enough for my parents to use, so having to ssh in, su to root and do a 'ifup
> ppp0' is not really appropriate. I think it's possible to allow users to
> control ppp0, but I'd prefer something even easier.

> I was thinking a simple script on a web server would be good. It'd obviously
> be simple to make a PHP page to do the ifup/ifdown, but I'd like something
> smarter, if possible. Something that could detect if ppp0 was already up and

I used to have something like this setup for my parents. It was a
simple cgi script and a web page, which I set as the homepage of their
browser. Unfortunately I no longer have a copy.

perl or even shell scripting you should be able to whip something up
pretty quickly.
