[Fwd: [Xbox-linux] Xbox Linux Kernel Messages]

Just incase anyone is interested, the xbox-linux project is moving
forward. This is the most interesting post to read, but shorter messages
from a few hours later show that they have successfully mounted root
from a non-xbox hdd plugged in. This means the main hurdles now are
getting video to work, and/or getting some kind of console to work. I
dont think it'll be too long now. 

It looks like MS have changed the boot sequence on the nvidia chip
(where they hid it), so its also quite possible that a new method will
need to be found for new xboxes. The ones on the shelf now however are
known to work with current mod chips. For that reason, it might be
better for interested partys to buy one now.

Ant
