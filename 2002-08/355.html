I have used X-CD-Roast 0.98alpha8 with RH7.1 and a sony CDRW CRX140E and it
works fine. I have also use the command line tool cdrecord on the more
ambitious project of making a multisession recording with apple Mac HFS
info on it. This project basically has failed (so far - I'm ganna try again
some day), however cdrecord is more than adequate for the lesser task of
making a single session music record. Being a command line program you will
have to read the manual the requisite three times, however I reckon that
you could make a short shell script and that would be more convienient to
use anyway.
