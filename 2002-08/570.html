Re: Up2date Question

Try stopping the Redhat network daemon,

It doesnt actually download the packages unless you set it to, AFAIK all it
does is keep the profile of your system at Redhat up to date / checks it
against the current upgrades available.

So that when an upgrade comes out redhat know that you need that update...

If you stop the Redhat network daemon, then you need to run up2date manually
on a regular basis otherwise after a while Redhat will remove your profile
from their systems and you'll need to reenable it.

They do alert you to this before it happens.

I have several personal systems using Redhat network and up2date.
