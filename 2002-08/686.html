> BUT I do not use grub either. I let linux boot on its own
bootdis(partition)
> In my case not hda1 but on hdb1
>
> Having a startup floppy to boot into redhad 7.3 I have then to do the
> following thing:
> So after install Linux you need to create an image or file dump of the
Linux
> boot sector.You need this to configure the windows XP boot loader to boot
> into linux
> 1. Get a blank formatted 1.44mb floppy disk
> 2.Shut down and reboot with linux boot disk, login as root Then remove the
> floppy, and insert the formatted MS-DOS floppy in a drive and type the
> following command;
> mount -t msdos /dev/fd0 .mnt/floppy
> (if you forgot to format the dos floppy you can still do it in linux with;
> fdformat /dev/fd0; /sbin/mkfs -t msdos
>
> Next step is to write the linux boot sector to the floppy. So take the
> bootsector from the correct partition.
> This partition is the one that was installed in lilo during installation.
> Chech it by following command;
> more /etc/lilo.conf     with will give you output of lilo. The "boot="
entry
> at the top of lilo tells us that lilo is in /dev/hd?
>
> Use the dd comand and write command;
> /bin/dd if=/dev/hd?  of=/mnt/floppy/bootsect.lnx bs=512 count=1
> This command copy boot sector to floppy   Then unmount floppy
> umount /dev/fd0
> Now shutdown linux and start XP Go to the root directory of Xp and copy
> flopy boot sector on XP partation OR like I had to do on the first partion
> where I have installed MS M.E. So the bootsector must be copy to drive c
> Then go to Xp to modify boot.ini file.
> Open system control panel applet, view advanced tab, select settings under
> startup and recovery and click edit.
> Then add the following line at the end of the boot.ini file.
> c:\bootsect.lnx="Red Hat 7.3"
> Now save boot.ini, reboot and select red hat 7.3 as one of your boot menu
> options and it is easy going from there.
>
