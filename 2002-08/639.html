RE: Local Govt site

can you post the URL you were trying? it may be a tech evangelism issue
(i.e. that site are doing some fancy javascript checks that don't take
mozilla into acount).

also, it could be that mozilla was installed without the "Personal
Security Manager" module (which provides SSL functionality, among other
stuff).

there's more recent builds of mozilla than 0.9.8 out now. 1.0 was
released a little while ago, they've released 1.1alpha & beta and are
about to release 1.1 final and also 1.0.1 (update to the 1.0 stable
branch). so i'd probably suggest installing either a recent 1.1 build:
	http://komodo.mozilla.org/pub/mozilla/nightly/latest-1.1/
or if you prefer stability without some of the newest features, go for
1.0:
	http://www.mozilla.org/releases/stable.html
or even 1.0.1 RC2:
http://ftp.mozilla.org/pub/mozilla/releases/mozilla1.0.1rc2/

and make sure you install "PSM" if asked!
