Re: wireless card setup

it seems fedora lists wireless seperately to other ethernet cards -
hence wlan0 is obviously the first wireless device:) eth0 first ethernet 
card - wired, if you like:) debian doesnt distinguish:)

btw, i used a tool when i was starting up called wlcardconfig. its a
nice little gui to help set iwconfig parameters if they a bit confusing
- it ships with knoppix. kdew also has a nice gui to show whats
  happening. gkrellm has very nice wireless stuff too. 

try this:
http://www.tipsybottle.com/technology/wireless/RedHat8-Kismet-HOWTO.shtml

(it applies to fedora, kismet is a wireless program i use, but it
assumes you are starting from scratch with card in one hand, and puzzled
look on face;)  )

hth, r:)
