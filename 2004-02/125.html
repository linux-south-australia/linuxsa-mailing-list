First off, do you realise that you're upgrading a source rpm (rather than 
a binary rpm that may look very similar to pan-0.14.2-1.i386.rpm)?

Source rpms need to be built and normally you rpm -i the source rpm and 
then go into the /usr/src/redhat (or whatever - let me know if you need 
more assistance with this).

Honestly, it looks to me like you've picked up a source rpm instead of a 
binary rpm.  Perhaps try the binary if all you're wanting to do is upgrade 
the package.  If that isn't what you want to do, we can go through the 
rpmbuild stuff.  Perhaps let us know which distribution you are using as 
well.
