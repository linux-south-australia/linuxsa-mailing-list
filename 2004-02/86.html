Errrr, not quite. I would say that there is a huge difference between
advising someone to install Mandrake vs Slack. The range of software with
the default install, the preponderance for a distro to have GUI
configuration tools or insist you use the CLI, range of support for
hardware, etc. are all valid concerns which differ markedly from distro to
distro. I mean, if you're talking about RH vs Mdk well fine but in Deb vs
SUSE or Gentoo vs Lindows, the differences are HUGE. For example, if someone
said that they didn't have much time to bugger around, I'd hardly tell them
to attempt to compile and install Gentoo, would I?
