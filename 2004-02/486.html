Re: Penguin-ising a Toshiba 400CDT notebook

Ian Loxton wrote:
> If I cant boot from the CD how the hell do 
> I get a system on it?

you should be able to boot from floppy (boot then root floppies)
then the installer starts and you can now remove the floppy drive
and swap with the CDROM drive.  This would work for vector
linux 4.0 anyway.

vector is good cos another option is stick the hard disk in
another machine and copy the /veclinux dir off the CD onto
the disk, then put disk back into laptop and floppy boot,
now you don't need the CD at all cos the installer will find
it on the hard disk (assuming you have already partitioned
it and select the "do not partition" option this time).
