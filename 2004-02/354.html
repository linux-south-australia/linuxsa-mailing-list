Re: corrupt /dev/lvm - bizzare properties

David,

If you had an LVM setup that used these partitions:

 /dev/hda1
 /dev/hda2

...couldn't you do:

 dd if=/dev/hda1 of=/somewhere/that/you/can/write/to/A
 dd if=/dev/hda2 of=/somewhere/that/you/can/write/to/B

...and then accidentally stuff up, say, /dev/hda1. In which case you could
do:

 dd if=/somewhere/that/you/can/write/to/B of=/dev/hda1

...and then have another go. The only thing that might stuff this up would
be a real hardware fault in which case, well, there is a real hardware
fault.
