Knoppix is a good introduction to Debian. And maybe for a firewall it is ok.
But for a Desktop box in a small office it lacks the configuration 
utilities like Mandrake, Suse. 

How far has Debian come for a small office ? I find the support for Debian
very good. When Googling with software questions I frequently find debian
users providing lots of useful information. But the huge amount of packages
and navigating various releases is formidable.

Some things an office needs

Usual stuff like dialup, firewall, email, browsing the net, Office
applications. But you also need faxing, support for vmware, win4lin.
In my case support for Varicad, the linux drafting software is required.
