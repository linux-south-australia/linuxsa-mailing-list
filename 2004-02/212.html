Heya,

Ive had a good play with knoppix, including building my own version (for
work).
I haven't had any issues with mounting a USB key drive, I actually wrote
a script to automagicly detect and mount it :)
The only thing to be carefully of is if the system you are using already
has SCSI drives in it! I accidentally formated the /boot partition! But
funnily enough the thing still managed to boot off it! Thats lilo for
you :)

Cheers,

Ben.
