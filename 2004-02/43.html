Re: debian.org?

On 02/02/2004, at 4:31 PM, Schenk, Jarrad wrote:

> Hypothetical Question here:
>
> Say I have an offline network of 3-4 debian machines and I want to 
> keep a mirror of security/www.debian.org etc on one of them. Whats the 
> best way of transfering only the packages that get updated to the 
> offline network via some form of removable media? I know you can use 
> apt-zip etc to transfer the packages that one machine needs, but what 
> if every machine has a different set of packages installed?

You can always burn /var/cache/apt/archives/, which will have a cache 
of all downloaded archives, and dump this into the same directory on 
the other machine.  apt-get update && apt-get -uy dist-upgrade won't 
download the files if they're already there.

Not sure how you're going to update the apt database without a network 
connection, though.  Sounds like it's fun with dpkg and manually 
resolving dependencies.  Ouch.
