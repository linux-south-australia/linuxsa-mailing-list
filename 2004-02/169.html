Re: Sort of OT: Mozilla reusing windows

Sorry, I think the mozilla -remote option only works when running on
an X Window system. But it's useful to know anyway.

Oh, if you just mean opening a url in Mozilla's email client, then
just right-click on the URL and choose your option.

Even better, if you use Mozilla on X, write a script to accept a
url, then execute the appropriate command depending on whether
mozilla is running or not:
===============================================================
#/bin/bash

MOZ=/usr/bin/mozilla

$MOZ -remote "ping()" &> /dev/null

# Tests if above commant said mozilla is running or not then runs
# the appropriate command.
#
# Change "new-window" to "new-tab" if that's what you want.

if [ $? == '0' ]; then
	$MOZ -remote "openURL($1, new-window)"
else
	$MOZ $1
fi
