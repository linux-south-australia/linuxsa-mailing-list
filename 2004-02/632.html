> Ryan Verner wrote:
> 
> > On 12/02/2004, at 2:48 PM, John Edwards wrote:

> >> What MTU are you running on each end?
> >
> >
> > 1500.
> 
> 
> You probably want to set this to 9000 if the driver on each end allows it.

This is just not needed to get good bandwidth. We run plain 1500 byte 
frames because you simply cannot guarantee that switches will support it, 
and we need big switches to get the bandwidth we are.

Of course, you take a larger interrupt hit, but the drivers and cards will 
already be doing interrupt mitigation anyway.
