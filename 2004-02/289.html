For the record, IMHO --force and --nodeps are evil options and should never be used unless you are fully aware of what you are doing and deliberately want to damage your rpmdb or you are using a distro like slack which doesn't have an rpmdb.

> 
> Then I did rpm -e php-interbase-4.3.0-alt0.cvs20030207.i586.rpm to see if it 
> had installed and I got a report back saying that php-interas* is not 
> installed.

Exactly right. For that would mean you had installed a package called php-interbase-4.3.0-alt0.cvs20030207.i586.rpm.i586.rpm 

Instead try rpm -e php-interbase

> Then I did rpm -ivh --force --nodeps 
> php-interbase-4.3.0-alt0.cvs20030207.i586.rpm, again, and this time got a 
> message saying that php-interbase* is already installed.

Of course it did.
