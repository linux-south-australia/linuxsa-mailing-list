One way might be :::
You build the debian boot floppies (pick the latest set you have on a cd
some where) .. On another machine.
Or download the new installer which MAY work a lot better (haven't tried
it yet)

man dd

for disks 1 to 4 something  not unlike but not  this:
dd if=/cdrom///bf.24/disk1  of=/dev/fd0 bs=1024 sync

This from my memory only

Use the (4) floppies to :::
Repartition to taste.
(a 10 to 20 meg part from sector "0" is useful for /boot)

Format to ext3 (lappy and room enough)
Install to a base system..   ONLY.
Keep the pcmia modules. (later)

About here you are asked to reboot do so.
Set up hostname. guest user or somesuch,  answer 5 to the exim question
..this is traditional:) now you can use apt-cdrom or apt-setup or hand
edit/etc/apt/sources.list

hint: you can get a second terminal most of the time on F2.
Where you can mount stuff.

Change to CDROM or network and install
_only the things you need._

That RAM proc speed probably not good for any desktop environment.

of dependencies you can migrate up to latest stuff after you have tested
