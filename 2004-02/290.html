Re: What is mozilla doing while loading?

As in Mozilla takes a bit to drag it all off disk.

Netscape/Mozilla has been slow to load due to the binary size ever since 
Mosiac back in '94.

Run "vmstat 1" and watch the bi and bo (blocks in/out) whilst it loads. 

Whilst running vmstat do a "sync" and watch the bo go up.  "sync" 
flushes any unwritten blocks to disk.
