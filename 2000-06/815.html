Mini review on Mandrake 7.1

Hi,

I installed Mandrake 7.1 today. I didn't have a CD, all I had was an ISO
image.

So, I created a network boot floppy and used an NFS install.

The Mandrake 7.1 install is really impressive. It does a very good job, and
let me choose where I wanted things. It is a very good GUI/X install.

It set up X for me, and offered to let me override the current network
setup and offered to set up a dial-out connection.

It also uses a version of LILO (21.4.3) that overcomes the 1023-cylinder
boot restriction (only works with motherboards of the correct vintage).

So, all in all, a very good experience.

Anyway, without a CD, how did I install?
