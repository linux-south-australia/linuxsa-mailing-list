Brian Marr wrote:
> Aside from Netscape Composer what good software packages are available

For the installfest web pages I hand coded the html.  In vi.  
Drew the graphics in GIMP, cut 'em up, put 'em in a table.  
Re-used html by using PHP functions (need PHP on server 
though).

It might be slower, but a) you learn more about html, 
and b) you have more control over what you get.

And this isn't becuase Linux has any less html editors - before I
started using Linux I still hand-coded the html.
