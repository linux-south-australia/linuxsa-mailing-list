Re: Routers

At 09:17 PM 6/13/00 +0930, Steve Sloan wrote:
>I am looking at using a Linux box as a router with IPchains doing some
>filtering between two subnets. As the lan will be running at 100meg and the
>users tend to deal with quite large files ie. a lot of graphic files well
>over a meg on a regular basis, I was wondering how much power I would need
>to allow for to ensure that there is no drop in performance on the Lan.

The Linux Router Project is your friend here. Check the web site, but I
think they say you would need about a 200MHz Pentium to handle a couple of
100Mb links.

I have had some experience with LRP, and you can build a machine with no
hard disk if you need to. All it needs is a floppy, or LS120, or even a
DISK on RAM or whatever. 

You can also have SSH on the boot device so you have a very secure way of
getting into the machine from over the Internet.
