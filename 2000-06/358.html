Re: LILO

Hi.

Found this the other day, came in very useful for a dual-boot system I built
for a neophyte:

 Extended Operating System Loader (XOSL) is the world's only
   full-featured free boot manager with a real, easy-to-use and
   full-blown graphical user interface. Click the screen shots link to
   see what XOSL is like, and decide for yourself!

It installs from dos (!), but that does mean you can recover from a floppy or
an install directory on on the win partition. Once installed it seems to work
fine, and will call up a lilo prompt from the second HD without a problem.
Supports default OS's and password protects those you would rather left alone.
Also supports 'partition hiding' which is probably something those multi-win
users would appreciate. 

I found it on the freesco (that's a firewall as in cisco) homepage, try
http://www.xosl.org
