MIME types

People might find this Apache configuration extract useful -- it's
not an attachment to ensure that it hits the archive.

#
# AddEncoding: allows you to have certain browsers (Mosaic/X 2.1+) uncompress
# information on the fly. Note: Not all browsers support this.
# Despite the name similarity, the following Add* directives have nothing
# to do with the FancyIndexing customization directives above.
#
AddEncoding x-compress .Z
AddEncoding x-gzip .gz .tgz
AddEncoding x-bzip .bz2

#
# Apparently, modifying the MIME types file is the Wrong Thing
# and that additional types should be listed in the Apache
# configuration.
# GDT
#

# PKWare, ZIP archives are not an encoding
AddType application/x-zip-compressed .zip
AddDescription "PKWare ZIP archive" .zip

# UNIX archives
AddType application/x-tar .tgz .tar .tar.Z .tar.gz .tar.bz2
AddDescription "Compressed tape archive" .tar.Z
AddDescription "gzip compressed tape archive" .tgz .tar.gz
AddDescription "bzip2 compressed tape archive" .tar.bz2
AddDescription "Tape archive" .tar

# Microsoft products
AddType application/msword .doc .dot
AddDescription "Microsoft Word document" .doc
AddDescription "Microsoft Word stylesheet" .dot

AddType application/rtf .rtf
AddDescription "Microsoft RTF document" .rtf

AddType application/vnd.ms-excel .xls .xlt .xlm .xld .xla .xlc .xlw .xll .xl5
AddDescription "Microsoft Excel spreadsheet" .xls .xlt .xlm .xld .xla .xlc .xlw .xll .xl5

AddType application/vnd.ms-powerpoint .ppt
AddDescription "Microsoft PowerPoint presentation" .ppt

AddType image/x-MS-bmp .bmp

# AARNet only
# These aren't globally applicable, but they make
# our life easier
#

# MS-DOS
#  For netboot over HTTP!
AddType application/octet-stream .BIN .EXE .COM
AddDescription "MS-DOS executable" .BIN .EXE .COM
AddType text/plain .TXT .BAT CONFIG.SYS
AddDescription "MS-DOS text" .TXT
AddDescription "MS-DOS batch file" .BAT
AddDescription "MS-DOS system configuration" CONFIG.SYS

# Cisco Systems
AddDescription "Cisco router image" .bin
AddType text/plain .cfg -confg
AddDescription "Cisco router configuration" .cfg -confg

# E-mail
# This crashes Explorer, so it probably should be text/plain
# One of GDT's future tasks is to write a module to render
# mail and mailboxes into HTML (or XML).
AddType message/rfc822 .mail
AddDescription "E-mail" .mail
