Re: Squid - CONF FILE

Do not worry about it.
Unless you have some special reason to use a proxy for HTTPS don't. HTTPS is
encrypted so there is no benefit from using a proxy (every page is
different) and it is actually slower. The best solution is to disable proxy
for HTTPS in your browser.

If you're using RFC1918 address space just use ipchains & MASQ. This is how
my network is setup and IE5 works fine.
