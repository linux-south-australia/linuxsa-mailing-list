Re: Squid - CONF FILE

Ive been using the following basic conf to try to solve my problems (ive been playing about and im new so bag anything you like in the file)
Again - IE4 and Netscpae work, IE 5 does not work with https
Directly upstream is a netscape proxy

http_port 3128
icp_port 0
cache_peer the_netscape_proxy parent 1964 0 default no-query login=user:pass
acl QUERY urlpath_regex cgi-bin \?
no_cache deny QUERY
cache_mem  8 MB
cache_access_log /usr/local/squid/logs/access.log
cache_log /usr/local/squid/logs/cache.log
cache_store_log /usr/local/squid/logs/store.log
emulate_httpd_log on
log_fqdn off
#authenticate_program /usr/local/squid/bin/ncsa_auth /usr/local/squid/etc/passwd
#authenticate_children 10
refresh_pattern         ^ftp:           1440    20%     10080
refresh_pattern         ^gopher:        1440    0%      1440
refresh_pattern         .               0       20%     4320
acl manager proto cache_object
acl all src 0.0.0.0/0.0.0.0
acl SSL_ports port 443 563
acl CONNECT method CONNECT
http_access allow SSL_ports
http_access allow CONNECT SSL_ports
http_access allow all
icp_access allow all
miss_access allow all
proxy_auth_realm Squid proxy-caching server
cache_effective_user squid
cache_effective_group squid
forwarded_for off
acl external src 0.0.0.0/0.0.0.0
never_direct allow external
