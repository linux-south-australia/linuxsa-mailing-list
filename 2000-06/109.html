Abstract
--------

The FreeVeracity (freeveracity.org) integrity server is a free
implementation of Rocksoft's (rocksoft.com) recently developed
Veracity (veracity.com) TCP/IP integrity protocol, for which IANA
(www.iana.org) has allocated TCP port 1062. Following a symmetric key
exchange, which establishes a DES3-CBC encrypted TCP connection, a
FreeVeracity client requests information about a FreeVeracity server's
file system. The server returns a small snapshot containing filenames,
file attributes and the cryptographic hashes of file contents. By
comparing the snapshot with previous snapshots received from the
server, the client can identify unauthorized changes in the server's
file system. By repeatedly connecting to many such servers, a
FreeVeracity client can monitor the integrity of the file systems of
an entire network, making it a powerful security tool. FreeVeracity is
released under The Free World Licence (ross.net/fwl) which makes it
free (with source code) for all free platforms including GNU/Linux,
FreeBSD, NetBSD and OpenBSD. The same functionality is available for
commercial platforms in Rocksoft's Veracity product, allowing a
heterogeneous network to be monitored from a single client. The
presentation will provide an overview of the FreeVeracity software
and, if possible, a demonstration on Linux.

Speaker
-------

Dr Ross N. Williams, the author of Veracity and FreeVeracity, is
Managing Directory of Rocksoft Pty Ltd (rocksoft.com), a South
Australian software company that produces the Veracity (veracity.com)
and FreeVeracity (freeveracity.org) data integrity tools that are used
by organizations around the world for network security, document
notarization and many other applications. Ross obtained his Ph.D. in
text data compression from the University Of Adelaide and is the
creator of comp.compression and the FunnelWeb (ross.net/funnelweb/)
literate programming tool which he used to write Veracity. His
personal web is at ross.net.
