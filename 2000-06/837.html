I've setup a Squid server on Redhat 5.2 which is working just fine
accessing secure sites with IE5. Versions are Squid version 
2.2.STABLE4 and Redhat 5.2.

Make sure that in "Internet Options->(your ISP name)->Proxy Settings" 
that you have a valid proxy entry for the "Secure" proxy server, or 
that you have ticked the "Use the same proxy for all protocols" check 
box.
