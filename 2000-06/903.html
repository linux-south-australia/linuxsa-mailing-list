Hi,

> When I get to the login prompt and type in a login name it doesnt matter 
> whether I type a valid one or not, and I don't get a chance to enter a 
> password it just goes on with loading X, and I cannot stop it from doing 
> that.

Your pc is automatically loading to init state 5. You can change this by
editing /etc/inittab for default.

To get to the non-X login you can do one of several things...

At the lilo prompt abbpend a 3 to your boot command. Eg if you type linux
to boot linux, then you should type

linux 3

Or alternatly, login to X as root, and type the command

telinit 3

Hope this helps.
