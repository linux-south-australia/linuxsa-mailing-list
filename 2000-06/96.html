Mark Phillips wrote:
> Secondly, is there any way to easily write an application for linux,
> using gtk say for example, and then getting it to work for MS Windows?

I'm currently doing this myself.  If you go to http://www.gtk.org, and then
follow
the link to Gtk/Win32, you'll find a site with more information on it.

It works fairly well, but it took me a while to work out how to set
it up.  If you need some more information on doing that, feel free
to email me.

Another cross-platform approach that I've used is Java/Swing.  This works
fine, but is a little sluggish on slower machines.
