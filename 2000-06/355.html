FW: LILO

Win 98 will overwrite your MBR so your assumption is correct. Your Linux
install will most likely still be safely untouched. I'm not sure what disto
you are using (Linux 6.1???) but if that is SuSE 6.1 then just insert your
install floppy and you should be able to boot to the installed system. If
you are running RH then I'm not sure what the exact procedure is, as I run
SuSE but I'm sure that one of your install or boot floppy/CDs will let you
boot to the installed system. You might have to specify your partition
somewhere eg /dev/hda2. Once there, just log in as root and type 'lilo'.
This should restore the previous boot info from your /etc/lilo.conf file and
all will be as was. 
Failing this then locate a lilo config tool and re-run that. The tools
change from distro to distro but there are even some fairly standard x apps
like klilo.
