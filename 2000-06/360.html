Re: LILO

On your Linux installation cd (i'm presuming Redhat6.1, by Linux6.1 ?? ) you
should have an executable file called loadlin.exe. You can use loadlin to boot
your linux partition, otherwise using your Redhat CD, you can make a bootable
floppy which will allow you to boot your linux system, then use the lilo -v
command to re-install lilo into your MBR !
