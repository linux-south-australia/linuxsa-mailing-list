> Red Hat Asia-Pacific, is expanding its training programs.  In addition 
> to the RHCE, Red Hat will introduce three training modules. 
> 
> Introduction to Red Hat Linux(RH033) is for the users who want to 
> customise their Red Hat Linux system for common command 
> processes and desktop productivity roles. 
> 
> The Red Hat Linux System Administration (RH133) course will 
> enable users to 
>  perform system administration tasks such as 
> installation, configuration, and attachments to networks. 
> 
> The Red Hat Linux Networking and Security Administration (RH253) 
> program will prepare a system administrator to be able to setup a Red 
> Hat Linux server and configure common network services and security at 
> a basic level. 
> 
> The RHCE Certification (RH300) requires success on performance-based 
> practical exams, in which users actually demonstrate the ability to: 
> install and configure Red Hat Linux, set up common network (IP) 
> services, and perform essential administration, diagnostic tests and 
> troubleshooting, among other Internetworking and system administration 
> task.  Also available is the RHCE certification exam taken separately 
> (RH302). 
> 
> Course prices are as follows (Australian dollars includes GST): 
> 
>   RH133 Introduction to Red Hat Linux $2195 
> 
>   RH033 Red Hat Linux System Administration $2295 
> 
>   RH253 Red Hat Linux Networking and Security Administration $2395 
> 
>   RH300 RHCE (including certification exam) $2950 (courses taken 
> before June 30th - $2750) 
> 
>   RH302 RHCE exam only $950 
>   
> 
> All Red Hat training courses are centered around the latest software 
> release of Red Hat Linux 6.2. and are taught by experienced Red Hat 
> certified instructors with real-world experience. 
> 
> Training courses are conducted in Melbourne, Sydney, Brisbane, 
> Canberra, Hobart, Adelaide and Darwin.  For further information on 
> courses dates refer to 
> www.redhat.com/services/training/asiapac/index.html
