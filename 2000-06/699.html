> Boot from a windows 9x floppy that is the exact same version as the version
> of Windows on your machine and type:
>
> A:\> C:\windows\command\sys.com C:
>
> When you get the 'System Transferred' message, you should be able to reboot
> and Windows will start.
>
> If you want to boot linux instead (which you must ;-)), boot off a linux
> distro CD / Floppy and mount your Linux partition, run LILO using the
> lilo.conf on your linux partition and you should be OK (you may need to
> change some of the device names in lilo.conf due to some distros not having
> all /dev/* files).
