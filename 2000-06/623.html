Couldn't see another reply to this, so sorry if I repeat others'
comments.

Try PAM_NTDOM or PAM_SMB (one, NTDOM ithink, comes with TNG I believe).
These PAN modules will allow your Unix users to log onto an NT domain.

You could also try using the unix password sync option in smb.conf (see
your local man page).

I hope to move to a Samba PDC, have all workstations / member servers
members of the samba domain and have all the Unix accounts in an LDAP
database. I would then use the unix password chat line in smb.conf to
run ldappasswd for password changes.

Hope it helps,
