Re: Netscape and Imap problems

Jake Hawkes wrote:

> netscape is talking to cyrus imap on 127.0.0.1, so I tcpdump -i lo and
> find that when I click on the inbox icon, there is no traffic. when I
> click on any of the subfolders, there is traffic [1].

No traffic at all?  Not even an attempt?  Hmmm.  Try -

#!/usr/bin/env python

user = 'dt'              # Adjust as appropriate
passwd = 'yadayadayada'  # ditto.

import imaplib
i = imaplib.IMAP4('localhost')
print 'Logging in:', i.login(user, passwd)
print 'Selecting', i.select('INBOX')
i.logout()

You should get something like -

Logging in: ('OK', ['User logged in'])
Selecting ('OK', ['368'])

Anything else would tend to point the finger at IMAP.
