Speaking of the howto's, I've been using Linux for the last two years with
a generic 3-button mouse.  Until recently, the issue of the middle button
not working didn't really bother me as I didn't need it.  I recently
decided to check out Enlightment, which really forced the issue as it uses
the middle button for one set of its menus.  So, I dug out the how to, and
my goodness, it was one of the easiest changes to make.  I'd always
assumed the switch on the mouse had to be set for "Microsoft", which it
was and mouseconfig would pick it up as Microsoft.  What I learned in the
how to was that if the switch is set to the other setting, mouseconfig
picks it up as "Mouse Systems" and wow, I've got 3 buttons!  

It's amazing what you can learn in the Howto's.  :-)
