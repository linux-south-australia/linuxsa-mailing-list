> What is httpd?

the name of the Apache executable. Other webservers will also have this name
for their executables... it means "http daemon", in standard unixese. A
webserver is a http daemon, and vice versa, pretty much by definition...

> What is Apache?

the most popular web server -- for intra and inter net usage...

> Do I need to use Apache?

no, you could use any other web server, but Apache is the most popular, and
works very well...
