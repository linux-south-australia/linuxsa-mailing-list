Re: Filesystem Question

On Sun, 18 Jun 2000, Jeremy Ervine wrote:

> but, I was hoping there would be an easier way of doing it ... eg,
> converting Ext2 to Fat32... is this possible ???

Run explore2fs, http://uranus.it.swin.edu.au/~jn/linux/explore2fs.htm.
This gives you read/write to ext2fs from a Windows application.

If you want a driver for Windows rather than an application (which is
harder to do using free software for non-technical reasons) see
http://uranus.it.swin.edu.au/~jn/explore2fs/faq.htm for a list of
alternatives.
