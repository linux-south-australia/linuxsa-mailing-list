At 08:55 PM 6/23/00 +0930, Damien Wilson wrote:
>Hi...

>Just a quick "what if..." question.

>Can a linux box be set up to "log in" to an NT network ala 95/98

Well, yes, it can!

Now, to the details. You need the new WINBIND code that Andrew, Luke and
Tim have developed. It allows you, using PAM and Samba code, to have your
account info in the SAM on an NT domain controller, or in the smbpasswd
file on a Samba server.

Then, you log into the domain under Linux. That is, you enter DOMAIN/user
when given the login prompt, and when you do an ls, you see:

  -rwxr-xr-x  DOMAIN/user DOMAIN/group .......

You do not need any accounts on your UNIX system.

The problems are:

  1. Getting the source code
  2. Compiling it
  3. Understanding how to configure it :-)
