Hi.

I actually had Windows installed previously, and wanted a dual-booting
system.  For whatever reason, defragging didn't do
its job and there were "unmoveable" blocks towards the end of the disk.
The largest partition size I could make was 63 Meg.

Because there were some files I *had* to keep, I shifted those into the
newly created second partition (which was formatted
VFAT), installed Linux and overwrote Windows entirely (putting LILO into
the MBR).

So Windows actually doesn't exist on my machine, and it *looks* as if hda5,
hda6, and hda7 are logical partitions inside hda1
(which looks as if it's become and extended partition).

So:

1.  The presumption that I installed Windows on top on Linux is incorrect.
It was in fact the other way around.

2.  Windows *isn't* on an extended partition.  It just plain doesn't exist
anymore.

3.  "Not much to do with Linux at all, eh?"  Well, that's not an accurate
statement.  It *is* to do with Linux (not Windows),
since Windows isn't even on the machine, Linux was installed afterwords and
I didn't have to boot from a floppy when I was
using Windows!!!

Anyway, whilst I'm happy with Linux I really would like to boot from the
hard drive.
