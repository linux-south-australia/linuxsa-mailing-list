Re: Web Pages

Hi Brian,
	As a rule I much prefer to start with Composer or the Applixware design
tool and then just mess with the code in joe or emacs.  This way you can
'trim the fat' usually left by such tools as Composer etc.

For a tutorial introduction to HTML see:
http://www.ncsa.uiuc.edu/General/Internet/WWW/HTMLPrimer.html

For an introduction to forms within HTML see:
http://www.cc.ukans.edu/~acs/docs/other/forms-intro.shtml

For general information about HTML see:
http://www.w3.org/hypertext/WWW/MarkUp/MarkUp.html

For a list of some of the HTML editors that are available see: 
http://www.mdt.net.au/www/faq/htedit.htm

Good luck, and don't forget to let us know where to see your efforts. 
:)

Don
