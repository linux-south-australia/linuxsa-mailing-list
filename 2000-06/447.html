Re: Converting Outlook 98 to mbox format

At 11:07 15/06/00 +0930, you wrote:
>I'm sorry that this isn't 100% related to linux, but considering the
>expertise of many of ya'll on the list I was hoping that somebody would be
>able to help me.  I am trying to move all my email which is currently stored
>in .pst format from Outlook 98 to a mbox format or something similar for use
>in my linux email client. I've done it using Eudora before, but I don't like
>the way it handles attachments...

I haven't actually tried this, but I think it should work.

Set up the University of Washington IMAP server on your Linux box and 
configure Outlook to talk to it. Run Outlook and transfer your messages to 
the IMAP mailbox. You should now have your mail in standard Unix format on 
your Linux box. This could be tricky if you are dual booting a single box, 
but I suppose it could still be done through VMWare.
