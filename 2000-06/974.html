Routing betweek the networks is the problem??? Can you ping from the
gateway machine each way? If you can, then your routing is set up
correctly.

What you need to do is make sure IP Forwarding is enabled:

echo 1 > /proc/sys/net/ipv4/ip_forward

and see if that makes any difference.
