Tests in sh

I copied a script out of a book, but bin/test is unknown
How do I do following on Linux?

#! /bin/sh
## sh script to copy all files in one directory to another directory
## if older in target directory or not exist in target directory
#
#
FROM=$1; TO=$2

/bin/test $# -eq 2 && /bin/test -d $FROM && /bin/test -d $TO || \
{ echo usage: $0 \<from directory\>\<to directory\> ; exit 1;}

wd=`pwd`
cd $FROM
for i in *; do
    if /bin/test -d $i # a directory
       then continue
    fi
    if /bin/test -f $TO/$i
       then find . -name $i -newer $TO/$i -exec "cp {} $TO" \;
       else cp $i $TO
    fi
done
cd $wd
