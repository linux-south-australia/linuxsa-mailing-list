On Wed, 21 Jun 2000, Kim Branson wrote:

> this is a rather odd request but has anyone considered running linux on a
> nintendo 64. 

Quote from the Linux MIPS HOWTO (as found on http://linux.senet.com.au)

  The Nintendo 64 is R4300 based game console with 4mb RAM. Its graphics
  chips were developed by Silicon Graphics for Nintendo. Right now this port
  has pipe dream status and will continue to be in that state until
  Nintendo decides to publish the necessary technical information. The
  question remains as to whether this is a good idea.

The MIPS Decstation port was done in Sydney, and there are numerous MIPS
users around Australia that I have heard of (most seriously in the
embedded/scientific area, where MIPS-based VMEbus cards are common.)
