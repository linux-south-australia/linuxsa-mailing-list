Or the same machine. Richard's comments about initrd's are correct, but
I don't think necessary for a kernel which has the appropriate scsi
driver compiled in. Assuming you installed the kernel source, you 
could boot up with something like Tom's Root Boot disk :
  http://www.toms.net/rb/home.html

access to your programs, and then recompile the kernel. Then, 
*before* rebooting, copy the newly made kernel to your windows
partition.

Or, boot up as described above and copy the /boot/initrd-whatever to
your windows partition.

Or, boot up as described above and make a new ramdisk using 
/sbin/mkinitrd .
