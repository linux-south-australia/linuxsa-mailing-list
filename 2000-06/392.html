Ghostscript and PDF output

Hey all,

I have an interesting question and I am too lazy to read the docs
because someone else must have seen this!

I print (using Staroffice) to a postscript (level 1 or level 2, it
doesn't matter) file.  I then convert the file using gs to a pdf.

I use:
'gs -sDEVICE=pdfwrite -sOutputFile=body.pdf title.ps'

automagically and return to the command line when it is done, or stop on
error. the problem is I see the following:
