Re: Light web browser

On Fri, 16 Jun 2000, Alex Garner wrote:
> Does anyone know of a good _small_ web browser that I can use instead
> of netscape? The hope is that I can get some work done. :o)

KDE's combined kfm file / internet browser is more than adequate. It does not
handle java and has a little trouble in truly refreshing (reloading) the page
but apart from that, it's excellent and integrated into the general suite of
programs that accompany the kde desktop. Highly recommended.

One of it's nicest features is being able to drag 'n drop files from an ftp
site.
