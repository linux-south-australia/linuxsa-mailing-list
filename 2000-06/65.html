Re: PHP/Postgres/apache probs

Michael Davies <michaeld@senet.com.au> wrote:

>I have a bog ordinary RedHat 6.1 install with PHP (php-3.0.12-6), apache
>(apache-1.3.9-4) and postgresql (postgresql-6.5.2-1).  Straight from the
>box (well, netcraft burnt CD).

I have PHP/PostgreSQL for Apache working on RH 6.0 but I had to
install the mod_php3-3.0.9-1.src.rpm and then installed the .rpm files
it created after I rebuilt.


Before building from the source I had to edit
/usr/src/redhat/SPECS/mod_php3.spec

and add a line
  --with-pgsql=/usr

to the CFLAGS= section

and rebuilt with
 rpm -bb /usr/src/redhat/SPECS/mod_php3.spec


Working from my memory here as I didn't document my progress so hope
this at least puts you on the right track.
