> Can a linux box be set up to \"log in\" to an NT network
ala 95/98

Better. Windows 9x don\'t properly create a machine trust
with the server, whereas NT, 2000 and Unix/Samba will.
You can even do so quite happily with Samba 2.0.x.

Check out member.txt (I believe it\'s still part of the
samba 2.x dist) in the docs directory. The NT DOM FAQ
should still be on the samba site somewhere.

By doing this, there is no need for an smbpasswd file.
The username and password are sent to the NT server
instead of being stored locally. The user must have an
account on the Unix box, but not necessarily a shell
account.
