printers and Debian 2.2

Has anyone using Debian had any success in setting up a citizen 200 gx
printer?  Should be treated as an epson9 btw.  The doc's that come with
lprng include a shell script but that doesn't work.  The script, if you
are interested is here;

http://members.iweb.net.au/~conrad/files/printertest

When run it reports that $printer isn't a valid identifier.  When lpr -V
(anyfile) is entered it reports that it is sent to lp@localhost and the
proccess is complete but no action at the printer.  If lprm is entered,
it dequeues the instruction which seems to suggest that most of the
software is actually working.  The printer devices can only be lp0 lp1
lp2.
