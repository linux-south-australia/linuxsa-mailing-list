mount as non-root

howdy all,

i've got a RH7.2 desktop. i'm trying to mount remote smb shares on our 
windows network as a normal (non-root) user, and i can't get it to work. 
the error i get is:

[benw@orion mnt]$ mount docs
cannot mount on /mnt/docs: Operation not permitted
smbmnt failed: 1
[benw@orion mnt]$ mount music
cannot mount on /mnt/music: Operation not permitted
smbmnt failed: 1

the relevant lines from /etc/fstab are:
//foo/music /mnt/music smbfs user,noauto,username=benw,password=baz 0 0
//bar/Docs /mnt/docs smbfs user,noauto,username=benw,password=baz 0 0

smbmount is version 2.2.1a.

under /mnt, the relevant folders are:
drwxrwxrwx    2 root     root         4.0k Feb  7 13:21 docs/
drwxrwxrwx    2 root     root         4.0k Nov 30 11:16 music/

any ideas?

i had trouble before with mount reporting "smbmnt must be installed suid 
root for direct user mounts (500,500) smbmnt failed: 1" - i chmod'ed 
/usr/bin/smbmnt to suid root, which solved that problem. i can't mount 
still tho.

any ideas?
