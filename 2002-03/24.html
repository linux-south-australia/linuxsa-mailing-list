Re: OpenOffice

Sue Robertson wrote:
> After reading recent posts I've downloaded & 'installed' OpenOffice 641
> on my RedHat 7.2 Ximian Gnome system but...
> 
> Being a complete newbie, I expected that it would automatically appear
> on my Programs menu or somewhere similar.  I can open Writer from the
> shell, but would like to know how to add the OpenOffice apps as either
> menu items or icons.
> 
> Where do I look & /or what questions do I ask when searching forums. 
> (everything I've tried so far has failed!)

To create GNOME icons, run the GNOME Menu Editor, which you should find 
under System->Menu editor, or you can run it from a shell by typing 
"gmenu".  Remember that you'll need to be the superuser (root) to add 
icons to the menu for all users.

If you don't know what the OpenOffice programs are called or don't know 
where they were installed, you could do something like
"rpm -qa | grep -i openoffice | xargs rpm -ql | more"
to get a list of all files installed in packages that have openoffice in 
their name (assuming OpenOffice was installed via RPM).

If you know where Writer is installed, you could also do
"which writer" (or whatever the command for Writer is) from a shell to 
see where Writer was installed, then look in the same directory for the 
other programs.

I hope this helps.
