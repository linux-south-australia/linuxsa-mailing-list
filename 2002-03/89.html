RE: Laptop + Debian + no way to install it (help?).

I had the same issue on my 3110ct when installing Suse7.2.
From what I can remember, I think I copied most of the linux install files
(SuSE comes on a dvd or 7 cds so hopefully debian isn't as big) onto the c:
of my win2k by mapping the cd on an external box via the network.
I then booted via floppy, selected the install media as being from hard
drive and installed.

The bottom line is that you are going to need network access to a shared
cdrom AND A LOCAL FLOPPY (assuming debian has install floppies and can
specify to install from a hdd).
I think the floppy drive is the same for the Toshiba 7020 and a few other
models and you see them locally on ebay and sold quite often.

I'd lend you mine but you probably don't live in Alice Springs.
