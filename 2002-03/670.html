> Anyone had any experinces with real TIVOs in Oz (hey they run linux)?  I'm 
> thinking of getting one but I still can't work out what I _actually_ need to

I've had a Tivo for a while. I don't expect you'd have much trouble at
all getting it working (though if you buy one with a recent release of
their software some tweaks may be needed for the guide data). There is
a palkit module which contains a module you just load on boot up which
sets it to PAL input/output. Unless you're feeling adventurous and am
handy with a soldering iron you will need a VCR to do the actual
tuning for you.

One hassle is loading/generating the guide data every week, but its
well worth the effort.
