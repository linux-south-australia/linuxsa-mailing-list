Re: PC Card modes

Hi Richard,
My experience is with OS/2, but since this is a hardware issue, I think
I can help.
I suspect an IRQ conflict.
I've found if you start probing PCMCIA chipsets they really don't like
it.
You probably need to explicitly specify which IRQ is used by the
driver, or ensure the IRQ assigned in the BIOS isn't conflicting with
some-other I/O device.

The terms you are using suggest a Toshiba Laptop, could you elaborate?

PCIC =16bit ISA technology.
CardBus = 32bit PCI technology.

Using the wrong driver can lock-up the laptop :)

With my Toshiba 440CDX, in the BIOS you can set which IRQ is used for
PCI, which includes:
USB
CardBus
Video

Other Toshiba's setups are different :)
