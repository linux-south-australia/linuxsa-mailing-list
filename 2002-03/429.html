Re: Linux on Laptop

Hi All,
    In my relatively recent experience in installing Linux
on a Toshiba Laptop (a Satellite Pro 4280DVD), I found that
both RH 7.2 and Manduck 8.0 (I think, was a while ago), were
quite difficult to set up and very little auto detected and
it was a pain to configure. However, when I tried Suse 7.2
(and 7.1), I found that I had no problems at all. I
literally did a standard install and let it do it's thing
and had a fully usable system in about an hour plus a little
config and tweaking time.

    I found one of the best sources of info was the Linux on
Laptops Web pages, that had an extensive walkthrough for my
particular Laptop, and many other specific Brands and Units.
They recommend using the Linmodem Web pages to set up you
Modem, and I found this to be quite helpful also.

    My personal recommendation would be to try Suse. It is
advertised as the most up to date distribution of Linux,
basically meaning that they update their packages from the
development sites a little more often than some other
distributions do, and it certainly makes some difference.
Under the hood it is still about 99% the same as all the
other distributions, and it also comes with Star Office,
which is a snap to set up, but still a little memory and
resource hungry (I am waiting for SO 6 to be released, and
would probably even pay for it). I hope this has helped.
    Cheers!
        Andrew.
