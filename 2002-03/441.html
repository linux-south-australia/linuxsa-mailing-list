Re: Dumping a disk image

On Thu, Mar 14, 2002 at 12:15:28PM +1100, Daryl Pilkington wrote:
> I think this is due to the card being seen during POST as an slave IDE
> device.
> If it doesn't see it at POST, it never will.

don't know how your machine there is set up but in my laptop I
have a flash card in a pcmcia adaptor, this is not detected at
the start of the boot process only when the pcmcia stuff is loaded.
It appears as /dev/hde.  On another CPCI chassis I played with one
time that had a flash card and a 2.5" laptop disk, there was a
jumper on the CPU board to set which device was master/slave
(only one IDE controller).  It would boot off either device and
the flash was treated exactly as any other IDE disk.
