This is unlikely in two grounds:

1. The client, in this case, Linux, is supposed to do what the server 
tells it to do. If the server says PlainText passwords, then that is what 
the client is supposed to do.

2. Win98 does not have a SAM and really only understands share-level 
passwords, unless you have a domain controller, like Samba, that it can 
check with.
