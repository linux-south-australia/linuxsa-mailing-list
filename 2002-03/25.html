Hi Sue

You'll find an OpenOffice directory in your home directory, it'll have a 
program directory under that. At the end of that directory listing you'll 
find

scalc (spreadsheet)
sdraw (drawing)
setup
simpress (presentation)
smath (mathematical equations)
sweb (webpage design)
swriter (word processor)
spadmin printer setup

Hope this helps.

Robyn
