Linux on Laptop

Hi all,

It has been a while since I toyed with Linux on a laptop so I thought I had 
better do it again for the stress ;-) I mean fun.

Laptop is a toshi sattelite 2250 CDT 600 Mhz celery, 64 Mb RAM, 5.5 HDD, 12.1 
inch TFT screen.

I am trying manduck 8.1. Report so far... started 8:00pm, in windoze do a 
scan disk and defrag. Completed 11:00pm. Install Manduck, including standard 
config, finish 12:00 midnight. Re-start config (manual), set host name, set X 
params (600x800 24bpp, 1024x768 virtual). Setting up X with version 
4.something wasted about an hour. Each time I configged and tested, screen 
went melty and unreadable. Eventually reverted to X 3.something. Loaded 
applixware, added users, created data area and soft links. all done in about 
another hour after losing an hour to X. Now downloaded lucent ltmodem driver 
rpms for this version of manduck. will report later on how that goes. Then 
TA-DA finished.

Any comments/advice gratefully received, except the usual sermon about 
winmodems. Not into that discussion unless you are an expert on the subject.

Bruce
