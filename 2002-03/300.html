Re: No init found boot error

reading google groups they reckon;

'lilo boot=/dev/hdax' at the boot prompt

lilo boot=/dev/hdbx for the primary slave ide chain
lilo boot=/dev/hdcx for secondary master
lilo boot=/dev/hddx for secondary slave

where x is the number of the device (probably 5)
should fix the problem then reconfigure lilo once booted (edit lilo.conf and
then run 'lilo') to fix thereafter
