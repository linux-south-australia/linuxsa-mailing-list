your windows box gives you that error when you are trying to sync from the sun? or the sun says
that when you are trying to sync with the internet source....

if the former (or the later) check which version of ntp protocol your server and client uses. 
There are differences.

if the latter, make sure your not port screened from that:

engsun1:jacobh> grep ntp /etc/services
ntp             123/tcp                         # Network Time Protocol
ntp             123/udp 

make sure you can connect to these ports.
