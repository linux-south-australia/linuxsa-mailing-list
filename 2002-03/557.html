 > For all those who used to [still do] use the intense eye candy

I don't think it really matters, and I think the subject line is 
misleading at best.

In case anyone has missed it, E17 has, for all intents and purposes,
already been released -- As much as any open source project ever gets
released, anyway.  There are real humans using it as a real window
manager, that's all that counts.

Version numbers are arbitrary.  Raster can spend the rest of his life
calling whatever the current version is "e17 x.y" for increasingly 
larger values of "x" and "y", and it won't matter to anyone except
C-Net and anyone else who thinks there's something magical about "1.0".

Development is continuing, and people are using it, and the source 
is available -- so it really isn't dead.
