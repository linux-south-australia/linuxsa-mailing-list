RE: Virtual Mail Domains w/ sendmail

IMHO, for what its worth, dont waiste your time with sendmail.

Use qmail.

Have a look here : http://www.inter7.com/freesoftware/index.html

Theres a whole suite of utilities that run with Qmail once you have setup, are all configurable via a web interface.  With this setup you dont have to have a "unix" user (in /etc/passwd) for each pop/imap user you have a mail account for. 

Then you can stick a nice WEB interface on top of it all (http://www.horde.org/imp)

Very nice :)
