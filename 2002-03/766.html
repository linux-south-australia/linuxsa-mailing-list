Re: linux as a desktop (was Re: Transparent Proxying)

Jordan Noblet wrote:
> - browser address bars should automatically select the address when you
> single left click or right click on them (for copying to clipboard or
> overwriting).

This is a bad idea, because the text would automatically be placed in
the clipboard.

> If I want to paste an address in I have highlight the current address,
 > delete it, and hope that the address I want is still in
> the clipboard because (heaven forbid) I clicked the left mouse button.

The standard Unix/Emacs keybindings should work:
control-u = delete line
control-k = delete text after cursor

So, to delete all the text in the URL bar, click in the URL bar,
and press control-u.

Not so hard, was it? ;-)
