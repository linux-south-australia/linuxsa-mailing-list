linux on laptop - completion

Finished setting up the toshi laptop today. 

Bought a USB 'mini-mouse' and plugged it in. Easy autodetect install in both 
Winblows and Linux. Compiled the winmodem module and installed OK, and 
settled on X 3.something because it works. 

Manual configuration under Linux included edit XF86Config to declare a 
virtual screen size of 1024x768 (a simple one-liner), and manual mount CD 2 
of the distro to load the kernel source and headers for compiling the modem 
module. 

Under Winblows I still have one unsolvable (to me) problem - the system has 
recorded somewhere the name of the person who held the laptop previously. All 
attempts to remove references using the 'proper' tools have appeared to work, 
however any software that is bought new and installed is automatically 
registered to that person as the registered owner. Given Microsoft's penchant 
for writing 'phone-home' software, I am not at all comfortable with my 
purchases going down automatically and unchangeably to someone else. But it's 
Winblows, so I expect crap.

Bruce
