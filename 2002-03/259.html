Is this something to do with it?


-M hint 

Select Path MTU Discovery strategy.  hint may be either do (pro­hibit
fragmentation, even local one), want (do PMTU discovery, fragment
locally when packet size is large), or dont (do not set DF flag).


I can only assume that if you use "-M dont" and it does not set the DF
flag, then the other 2 do?

I found this by doing man ping and then searching for DF. (mandrake
linux 8.1 ping from iputils-20001110-7mdk.rpm)
