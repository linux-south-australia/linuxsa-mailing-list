Re: Matlab for Linux?

I recommend GNU Octave ( http://www.octave.org ) with Matlab-Compat (
http://users.powernet.co.uk/kienzle/octave/matcompat/ ) extensions.

So far I have yet to find any diffrences (except Octave supports
end/home keys).

Used it to check my Maths 1 (Linear Algebra) homework.
