Re: Community based wireless LAN in Adelaide - interest?

The repeater concept is what makes it all so interesting.  Pockets would 
  eventually conglomerate in to larger pockets and so on - it would gain 
momentem with enough interest.

Just file sharing, games, etc would make it worth it.  Put out a request 
for a large file on the local net before the internet - somehow 
automatically?  Say ISO images and the like.  It would lessen load on 
the greater internet and provide a faster way to download such files.

Perhaps some local ISP could provide a file server as a community 
service - and also to discourage Internet load.

Just rambling thoughts..
