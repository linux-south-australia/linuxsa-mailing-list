James:

It will be on one of the three disks.

1) Mount the disk

mount /mnt/cdrom

2) Do this

cd /mnt/cdrom/RedHat/RPMS

(or find the RPMS director)

3) Do this

ls | grep lex

4) It will spit out something like:

flex-blah blah blah.rpm

5) Install that:

rpm -ivh flex-blah blah blah.rpm



See:

man rpm
http://www.rpm.org/

If RPM says it wants other packages, you'll have to find them using a 
similar process and install them first.
