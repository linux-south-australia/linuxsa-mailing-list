Re: [OT] xDSL IP address change

Dynamic IP ADSL connections use PPPoE or PPPoA.

Both of them are PPP -the same protocol that drives dialup sessions.

So the same underlying situation applies - you -can- determine if you 
are online or not, because there is an active session layer (inside 
PPP) maintained when the connection is alive.

Connection drop is both (a) explained and (b) obvious in reasonable 
PPPoE clients, rendered in a window which is broadly similar to a 
dialup access dialog box.
