Thanks Andrew,
Desktop configuration is a PITA & getting everything configured so it
works smoothly is a never-ending battle.

Probably to the point where most people on this listserv don't care, &
let their users just "figure-it-out themselves".

I'm sure we've heard the saying, "Oh thats a desktop problem..." & left
it at that.
The reality is that the 1st interface for a user *is* the desktop, so
it needs to be simple, integrated & reliable.

So the challenge is to maintain the desktop & the integrated workflow
from one application to another, using portable Standards.

It takes a long-time to test & configure a desktop, so you'd like to
leverage that time by being able to deploy it quickly to many nodes.
