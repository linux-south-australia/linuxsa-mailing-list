Re: 'Taskbar' icons

Quoting Sue Robertson <srobertson@sr.rbe.net.au>:

> I've changed some config setting or other & lost my preferred desktop
> setting. Can someone please tell me which setting in the Gnome Control
> Centre to get it back?  (I'm using Ximian Gnome, with Sawfish Window
> Manager, RH 7.2 2.4.9-31)
> 
> Previously, when I minimized an application, it placed an icon on the
> bottom panel (taskbar in Windoze speak).  
> 
> Currently, all minimized applications hide behind a single icon in the
> far right hand corner of the top panel.
> 
> How do I get minimized applications to show as icons on the panel?

Hi Sue,

You need to run an applet in the bottom task bar called, "Desktop Pager" or 
similar.  Once that starts, your minimised applications will be visibile 
there.  Note that you'll need to save your desktop settings afterwards so that 
it is available on next log in.
