RE: [OT] Alcatel SpeedTouch Home to Pro

Matt Pearce has reminded me that I should mention that the Speedtouch series 
of ADSL Modems (are they modems?) are vunerable to remote attack (see:
http://security.sdsc.edu/self-help/alcatel/)

The ST Pro can be made relatively secure, but only if you login to it and 
issue the command "ip config firewalling on" which is NOT on by default 
if you do this upgrade.  This command doesn't actually do firewalling but 
stops the STPro accepting connections to the vunerable services on the 
WAN/ATM port which prevents the types of attack above from occuring.

This is only relevant if you login to the STH/Pro and modify it such that 
the IP address is NOT the default 10.0.0.138 (which of course isn't easily 
reachable across the net!)

(Whilst I don't accept any blame for people being hacked due to doing the 
STH to Pro upgrade as a professional Network and Security Engineer I feel I 
should let people know what the risks are)

If you have any apprehension about doing the upgrade then forgoodness sakes 
don't.  Any questions, direct them to me off list.
