> Compaq Prosignia 300 P75 with 64 Meg RAM, SCSI Hard disk / SCSI CDRom

Hi Michael,

Does the kernel say how much memory it found during bootup? If that is
shown, and it's less then expected, you'll need to lie to the kernel about
the amount of RAM to get this to work.

The kernel parameter is 'mem' if I remember right (google will tell you)
and you should be able to pass that through to the kernel via whatever
boot loader RedHat use for the installation these days :-)
