Richard Russell wrote:
> Why I still use Windows on the Desktop...

I moved my home systems to FreeBSD last month.

Here's some quick thoughts (mostly relevant to GNU/Linux)...

PROS
- more stable
- more powerful
- virtual desktops
- free
- easy filesharing
- remote X and remote sound
- less susceptible to viruses
- development tools
- can use cheap PC as a terminal

CONS
- mplayer is not as good as Windows Media Player
- XMMS is not as good as WMP/Winamp/RealPlayer/others
- people still send me Word attachments that I can't open
- some web sites will only work in Internet Explorer
- don't have a decent Java SDK (yet)
- few games
- no decent Flash plugin

If you're after consistency, I would strongly recommend you
choose KDE or GNOME as your desktop environment and stay in it!
