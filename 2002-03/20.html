Re: [Article] Sun to charge for next StarOffice

Daniel,
    Please try OpenOffice, Free Download about 65mg for Linux and 48mg 
for Windows, I use both, actually my kids use both I only use the linux 
version. This way they can do their homework on their dual boot machine 
in either OS.

For those that haven't picked it up OpenOffice 641c is the base on which 
Sun have built StarOffice 6. So if you want the free version use Open 
Office. (Similar to the Mozilla / Netscape arrangement)

Yes there are differences, StarOffice includes some proprietary PIM and 
filter software that Sun feels obliged to include so ensure 
compatability for their existing commercial costomer base. (Or perhaps 
to justify charging for it). However most of the inbuilt filters will 
read MS Office (pre 2000 stuffed files) and write to them. Office 2000 
files tend to be dodgy, (aren't they always). 

My kids use MS Office 97 Word, Excel and Powerpoint for school projects 
and all the files they use for these have been rendered properly 
modified at home and taken back to school. Actually the results are more 
consistantly rendered correctly on return to school, than those of 
children using MS Office products.(Go figure!) To the point that I have 
been asked, by their Teacher, which version of MS Office we used, and 
how we saved the Documents. As our chidren never have to reset up their 
homework when they bring it back to school ie. formats preserved 
correctly. (Huge time saver)

Startled him somewhat when I told him I wouldn't use dodgy MS product I 
was using beta software on linux.

David Delyn   
"My 50c worth! (Inflation)"

ps The only problem we have encountered, is that some of the gradient 
backgrounds in Powerpoint slides made on MS Office Print as giberish 
text, even to the point of locking up the printer system. Our work 
around is to remove the background and substitute one of the Presenter 
OpenOffice backgrounds which print fine.
