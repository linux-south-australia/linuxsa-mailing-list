>
> secondly - what's the best way of backing up a linux set to a dos
> partition?
> or would it just be easier to make a 2nd linux partition, and copy
> everything to it?

Depends what you want to do.

Probably the best way to do it is do a
      find / -newer file.date

Which will give you a list of files used/abused/created/whatever since
the date of the
file 'file.date'.  This way you can get a vanilla installation, copy
your edits off and save them.

So in short you could do a
       tar -cvzf backup.tar.gz `find / -newer file.data`

*note* the back quotes (under the ~) which executes the command in
quotes and lets
you use the output as command line arguments.
