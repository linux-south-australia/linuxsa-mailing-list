Re: MyFastModem Problems.

> To anyone who can help!
> 
> My Banksia MyFastModem (internal) is intermittant in it's responce to the
> computer.  Slow reponse time ~10-20 seconds (or more)

This sounds very much like an IRQ conflict.

If COM1 is using IRQ 4 and COM2 is using IRQ 3 (which is usual), then
you need to make sure that the modem is not using either IRQ 3 or 4.
If it is, then you will get the behaviour that you describe.
