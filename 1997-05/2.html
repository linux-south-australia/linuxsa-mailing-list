Any decent mail program should be able to let you specify that
messages from particular lists should go into particular folders.  The
header to look for for this list is:
X-Loop: linuxsa@liz.netcraft.com.au

If you use procmail and mh/exmh, then you could put this in your
.procmailrc file to drop LinuxSA messages into a linuxsa folder:

# linuxsa
:0 w
* ^X-Loop: linuxsa@liz.netcraft.com.au
| /usr/lib/mh/rcvstore +linuxsa -create

(I know you don't use procmail since you're not yet a Linux convert,
but you will be :), and maybe this will help someone else)
