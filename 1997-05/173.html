Please have a read of a document which I wrote, available on the web
at http://www.linuxsa.org.au/tips/.  It describes how to use floppy
disks and CDROMs under Linux.

As for your printer problem, login as 'root', start up X, and run Red
Hat's printtool program.  That is probably the easiest way to set up a
printer, although the printtool included with Red Hat 4.1 is much
improved over 4.0.
