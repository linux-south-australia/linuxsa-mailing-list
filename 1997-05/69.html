Re: MyFastModem Problems.

I have had this particular problem with a linux box and a banksia 
28.8/33.6 modem .... it is defininately a case of either incorrect 
interrupt (you have to set it manually in rc.serial, so that the lines 
about the serial portsa come up twice in the bootup, i.e. one from the 
kernal, near the start, and another one further down, from rc.serial.) or 
a shared IRQ, in which case you have to change the card's IRQ.  ( i have 
seen both cases give the same problem)
