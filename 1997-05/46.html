As before, the problem with the shared interrupt COM port 
issue is in the hardware not software.  In particular, the 
PIC (Programmable Interrupt Controller) is Level Triggered 
not Edge Triggered causing problems with sharing.  (Been a 
while since I did this).  I think I saw a hardware 'fix' 
(read open heart surgery) for certain IO cards.

BTW, Once the fix is done, Linux is reported to have no 
problems sharing!  (I believe Windows still does??)
