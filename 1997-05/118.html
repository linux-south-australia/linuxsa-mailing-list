servsurv@tardis.apana.org.au

This survey is for those of you using Linux as a server.

what they all do. We're more looking at induviduals than computers.

INTERACTIVE SURVEY

intsurv@tardis.apana.org.au

This survey measures interactive use of Linux.

No subject line.
One of more lines of body text, each line being in the following format:

OS:percentage,task:percentage:[,task:percentage]....

The OS percentages must add up to 100, as must the task percentages on
each line.

OS is one of the following words:

linux dos win31 winnt win95 freebsd 

[Any more OS's out there semi-commonly in use?]

task is one of the following words:

html	html editing
prog	programming 
games	playing games
inet	surfing the inet
bbs		Using a BBS
mail	Interactive mail
news	Interactive news 
wp		Word Processing
db		Database
ss		Spreadsheet
graph	Graphics editing/rendering
play	Just fooling with the OS
install	Still trying to get it installed

Example:

dos:20,games:100
winnt:10,install:100
win95:30,games:40,inet:50,bbs:10
linux:40,games:10,html:40,mail:10,news:20,graph:20

(This is a fake example :) I play games on DOS. I can't even get NT
even installed. I play a few Win95 games, and surf the internet and
local BBS's there as well. On Linux I do my HTML coding, play a few
games, read my mail and news, and do a bit of graphics work.
