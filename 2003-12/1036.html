Actually, probably not, because I doubt you'd be using tokenring, and
evidently 'pci bus error' isn't a string that lives in an actual network
driver.

Have you updated your kernel lately?  I've found updates to various
network cards, especially those not-so-popular ones, can actually break
things; I remember some earlier 2.4 kernels having this, and needing to
use the older kernel until the bug was fixed in a patch.

Failing that, do the test-for-hardware-failure-trick, as per David's
suggestions; move around cards, swap cards out, use one card at a time,
etc.  It's not all that uncommon for network cards to suddenly have
heart failure, especially considering how damn hot some chipsets seem to
get :-)
