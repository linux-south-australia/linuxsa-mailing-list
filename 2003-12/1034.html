DivX: mplayer... go to http://plf.zarb.org/~nanardon/urpmiweb.php, and setup 
the appropriate repositories for mandrake 9.0. Namely, contrib, updates, and 
plf. Grab mplayer from the plf repository using the software manager. You can 
click on "Maximum Information" to view the source of the package. Plf 
packages end in -plf anyway.

DVD: Download ogle. You can do this from plf, however I believe it is version 
0.8.5, which doesn't have support for 4 speaker surround sound systems (like 
mine). You can get the latest version of ogle from:

http://www.dtek.chalmers.se/groups/dvd/index.shtml

As to your TV Tuner card, you may have to specify an extra option to the bttv 
driver in /etc/modules.conf. When I set it up for a friend of mine I googled 
around and found a way to set the proper PAL setting for Australia, can't 
remember off hand what that was though.
