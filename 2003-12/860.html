RE: Second hard drive help

If you just want to use the disk and don't care about the Promise controller
(that's really for setting up RAID configurations I think) then just hook
the new drive into one of the existing IDE ports on your motherboard,
perhaps as primary slave. It should work fine like that.

If you want to connect it via the Promise controller, you will need to
install the drivers for that into Linux. A search on google for "Promise
ultra100tx2 linux" finds quite a few hits saying the 2.6 kernel does not
work with this controller. This may mean it used to work with 2.4.

John Bonnett
