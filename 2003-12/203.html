Mounting a DD hard drive image

Hi all

I wondering if any one knows the magic required to mount a dd dump of a hard
drive image.

As I'm all sure you know the problem is that a hard drive has a partition
table and boot record at the beginning of the drive. The normal mount loop
tool do not take that stuff in to account.

In my case the image only has one partition so maybe the easy way would be
to cut off the first part but I do not remember the byte of set.

It would be better if there was a tool which could make Linux see this as a
new block device which it would read the partition table from and then mount
in the normal way.

Any ideas would be great
