FWIW, the header files aren't considered part of the UNIX source
code.  They were distributed with the development system, and some
files were distributed with the base system.  My favourite one was
/etc/fstab, which as shipped consisted in its entirety of a copyright
notice.  Since it was there, I couldn't remove it (though I suppose I
could have renamed the file and created a new one), but it gives you
an idea how stupid some of these copyright claims are.
