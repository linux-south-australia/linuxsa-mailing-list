Re: Linux on Mac: Is there a point?

On 10/12/2003, at 10:05 PM, Daniel MacLeod wrote:

> I'm purchasing a new G4 iBook, and would like to know, is there a 
> point to running OS X and Linux on it? Especially now that Apple 
> supports X11 and most Linux programs have been ported.
>
> Is yes, can anyone recommend a distro? I have used RedHat, Debian, and 
> know of YellowDog.

I'm about to attempt a Debian install on my new 1.25ghz Powerbook G4 
(yay!).

Depends how you define a 'point'; problem is, a lot of this newer 
hardware isn't completely supported under Linux.  Things such as no 
airport extreme support (at all), no 3D accelerated video (which I 
doubt will change anytime soon), not being able to access your HFS+ 
partitions under Linux (which really shits me), make using it 
questionable.  I'd stick with OS X; it'll be far more functional, but 
dual booting can be interesting; Linux absolutely screams on PPC 
hardware.
