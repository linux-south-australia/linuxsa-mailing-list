Re: Linux VPN Server and M$ client

On Thu, 11 Dec 2003, Shane Warner wrote:

> Does anyone have any recommendations for the above mentioned scenario?

OpenVPN should work a treat:

	http://openvpn.sourceforge.net/

Some nice features it has:
- UDP based, so you can go through firewalls and NAT easily,
- Supports full TLS/SSL including CA signed certs.
- You can do bridging easily,
- Config file is platform independent,
- Implemented in user-space, so interoperability isn't a pain like
  IPsec.
