About:
Enhanced Linux Loopback Driver is a version of the Linux loop driver that
provides partition table support, and allows you to loopback mount
individual partitions contained within a single hard drive image file
(i.e. dd if=/dev/hda of=hd.img, losetup /dev/loopa hd.img, mount
/dev/loopa1 /mnt/stuff).
