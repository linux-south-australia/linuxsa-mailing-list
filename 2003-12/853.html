Re: Second hard drive help


On 21/12/2003, at 8:04 AM, Naota Hashimoto wrote:

>
> A non-linux friend told me the new hdd is going to be considered scsi 
> on
> my system because of the controller, so maybe it's about the new disk.

Yep, that's correct.

You should be able to see and use the drive under linux when connected 
to the Promise controller and it will likely appear as /dev/sd0, 
however, you're pushing muck up hill if you want to boot from it on a 
motherboard/bios so ancient.  If you're really really lucky, you may 
get the bios to boot from SCSI and the promise controller might load 
it's bios and the PC might use that for booting, but I've had no luck 
what-so-ever getting any OS to boot off one of these controllers.
