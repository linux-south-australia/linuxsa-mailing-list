Fr Tony Pearson wrote:

> Can anyone tell me if it is possible to change the password and make 
> it possible for us to access the programme.  Whilst it is operating 
> well at the moment, we are fearful of what will happen if there is a 
> fault.
>  


Hi Tony,

The easiest way is to type in 'linux single' at the lilo prompt.

This will then boot you into linux single user mode, which *may* let you 
proceed without a password, so you can then change the password for the 
root user to whatever you like.

Otherwise you may be able to use a linux rescue disk to gain access 
without a password.
