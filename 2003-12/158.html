Assuming you are looking at something to track bugs/features etc?

I've used 2:

keystone (not sure if it's still around - the company was sold, and it
was dying last time I went for an update. I probably have a copy in my
archive if you want to try it.

Recently, I've been using drupal for CMS and it has a project module.
Compared to the hoops I've had to jump through in the past to get a CMS
operational that I actually like the look of, drupal is a piece of cake.

http://drupal.org/

If you've ever visited debianplanet you have seen drupal. It's pretty
young, but I like the fact that apart from the basic modules, you only
load the code you want to use. A lot of stuff is possible, but
implemented as add-in modules. It's MySql/PHP/Apache, but it will run on
other platforms/DB.

http://drupal.org/project/project

(Yup, they use the drupal project module to keep track of the project
module)

If you are looking for something more sophisticated with timelines and
Gant Charts etc, look elsewhere...

Hope this helps,
