Perhaps you're making the mistake of equating this list with the "open
source community" (whatever that means) when in reality it's merely a
loose collection of like minded people who share a common interest. That
interest may well be working with open source software but most if not
all are simply trying to learn to use and improve their linux skills
with the software they've got. Although some few on the list are
involved with the development of various aspects of open source
software, it is most likely to improve rather than to innovate.

However to answer your "challenge" take a look at
http://fvwm-themes.sourceforge.net (begun in Jan' 2000) which has a new
and IMHO innovative approach to desktop themeing. Each theme component
can be combined with any other theme's components in any combination the
user wishes. What's so innovative about this you may ask? Well each
component also can have many variations and choices within it so that
the permutations of possible combinations increase exponentially. In
this manner I was able to add over 8000 desktop colour combinations to
my desktop in a mere 10kb tarball. You would ideally have to install it
to try it out to determine if this project meets your "innovation"
criteria, but I believe it does. Your opinion however may differ
completely from mine.

Perhaps you would be better served asking these questions on some
developers mailing lists where the concepts of developing new
technologies and software is the main interest and topic of the list?
