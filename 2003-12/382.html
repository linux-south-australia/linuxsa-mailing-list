Auditing the code yourself ...

This website: 
http://news.zdnet.co.uk/software/linuxunix/0,39020390,39118419,00.htm

Has an interesting quote:

<quote>
However, that model of software development comes with a hidden cost, 
critics say. companies that want to have a high assurance that an attack 
hasn't resulted in a security weakness will have to audit the code 
themselves, said Greg Wood, general manager of information security for 
Microsoft, a vocal opponent and rival to many open-source software 
projects, such as Linux and the Apache Web server. 
</quote>

Let's see. What if we change some of the words ...

Companies that want to have high assurange that an attack, or incompetence 
or market pressures have not resulted in a security weakness in Windows 
will have to audit the code themselves...

Oh, wait, you can't. What a bummer.
