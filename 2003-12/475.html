I haven't played with the NTFS version.  I spent a couple of years using it
under '98 and ME.  I looked into this for my IBM T30 but instead went the
shrink-the-windows-xp-partition-and-install-slackware-in-the-spare-space
path instead.

I downloaded this ISO http://www.sysresccd.org/ This is a Knoppix (SP?)
Based iso that is about 80MB to download.  I had to defrag the HDD A few
times to move files away from the end of the drive but then was able to use
QTParted from the install.  Having done that I was able to install
slackware.

If you want an open source alternative to Partition Magic Look at
http://qtparted.sourceforge.net
