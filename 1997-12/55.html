Re: Small Redhat install

> Hi guys,
> 
> The documentation for Redhat 4.2 suggests a root partiation of 50MB and a
> /usr partition of 200-500MB.  My question is, is it possible to get a
> minimal installation on to a 120MB hard disk? 

Yes.  A Red Hat 5.0 minimum install uses under 60MB, and with a few
minutes work you can get that down to under 40MB.  I'm pretty sure a
minimum 4.2 install uses less space.  The suggested values in the
documentation are intended for the average person wanting to do more
than just firewalling.

> The machine will just be a firewall.  I have one at the moment on a disk
> that small and it only uses half of it.  But that was a Slackware
> installation.  /usr/src is an NFS mount.
> 
> So is it possible to squeeze a Redhat installation in 120MB?  Does you
> have to install X with Redhat? 

You don't need to install X with Red Hat.  Just make sure that the "X
Window System" checkbox is not checked when it asks what components to
install.
