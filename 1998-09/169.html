Re: Wouldn't it be nice if...

> :every "Internet-connected" computer had
> its own domain name -and- the capability
> to create any number of e-mail addresses
> based on that domain name...?

If your mail server runs on sendmail (and most do), then you already can
create any number of e-mail addresses: You can add "+" and anything you
like after your account name (and before the @).  So, by way of example,
you can send mail to me at: davidn+linuxsa@rebel.net.au.  It will be
delivered the same as all of my other email, but when I look at the To:
address I'll know where it came from.  If you *always* quote your email
as user+something@your.mail.domain, and if you automatically trash any
email sent without +something, then I think it pretty much solves this
problem.  Of course that's if your mail server uses sendmail.  Sorry for
those who use Microsoft Exchange.  ;-)
