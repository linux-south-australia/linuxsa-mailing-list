Re: "shrinking" an ext2 partition

On Sun, 20 Sep 1998, Steven Bolbot wrote:

> Does anyone know if a utility exists that can reduce the size of an ext2
> partition without losing any of the data contained on it?
As far as I know this cannnot be done, the very design of the EXT2FS means
that dynamic resizeing is not possible. your best bet is to tar the data
on the partition and save it somewhere safe, then delete the it and create
a new smaller one in it's place. then just untar the data back on to it,
there are ways of doing this if / is the partition in question but you
nedd a boot disk.
