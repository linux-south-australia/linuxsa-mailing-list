Re: Netscape

> All initially works fine but after about five minutes use netscape  text
> in the web pages and the menu becomes garbled as well as the window
> manager text
> (window titles and pop up menus)
> 
> I am using metro x server and the graphic card is a  s3d virge.

I had the same problem with MetroX amt my s3d ViRGE card. It only really
started happening after I put extra memory in the card to up it to 4 megs.
After much deliberating and playing with settings to see what I could do
to fix it I decided that it was the metro x server. you will find that
other programs show the same symptoms if you SCROLL in large text windows
like in the borwser window.

the only solution that I found was to remove metro x and install XFree86
instead. IMHO, it works better anyway. 

Anyway, to cut it all down to 1 paragraph, the solution is to use xfree86
and you will find that the bug disappears.
