the number one rule of graphic design! if you can
leave it out then you probably should.
secondly use only one typeface from each family,
ie. only one serif (eg times) only one sans serif
(eg helvetica)  i don't think scripts have much
impact at all, and many designers consider them
the pinnacle of tastelessness and all that aint
modern :)
