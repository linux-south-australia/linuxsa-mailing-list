Anyhow, here is the scenario:

FTP and mail servers. As some of you might know, DNS on 95/98 isn't really

What I want to do is set up a linux box to run the servers and connect to
the net, then I can masquerade (did i spell that right?) behind the linux

Now, using my 'self-taught' method, I have managed to get the DNS and

complicated anyway... explain in a bit...) and the masquerading have got me
stumped... Anyone have any scripts or any other advice that will help?  :)

OK. The mail is kinda complicated... to all you linux experts out there

option...

Any help is much appreciated... I'm sure that to anyone who has been using
linux for a while will have no problems answering my questions... but i've
been using it for about 2 weeks....
