I may not be a SuSe user but in the days before I figured out how to use
chat scripts I used to do the minicom/manual pppd thing this is a good
generic solution and will give you the info to write a chat script that 
you can use in conjunction with pppd to automate your dialin's. I recomend
a good read of the pppd man pages and some of the mincom docs(although
minicom is pretty straight forward once configured), the only thing I
think it is nessecary to remind you is that you want to exit mincom
without hanging up (Ctrl-A-Q if memory serves me right) before running
pppd.
