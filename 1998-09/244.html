Re: ppp problems

Hi Tony,

It would be handy to check the debug output from firing up pppd with the
debug option (eg "pppd debug <other options>"), and the version number of
pppd (from "pppd version" or similar).

I suspect that the problem may be due to WinNT wanting to authenticate using
microsoft mschap80 encryption which pppd (at least the version I have) does
not support out of the box.  This may vary, depending on which distribution
you have.

If this is the problem, you can fix it in one of (at least) two ways:
1. The PPP-HOWTO has (some) information on how to get MSCHAP working with
pppd at item 13.1.  You can get PPP-HOWTO from
http://sunsite.unc.edu/pub/Linux/docs/HOWTO/PPP-HOWTO if it is not in
/usr/doc already.

2. Select the option (sorry don't have an NT box handy) that reads something
like "accept any authentication, including plain text".  I think it's in the
networking stuff in control panel somewhere, but it has been a while.  If
you need more accurate information, let me know.  From memory the default is
"only accept microsoft encrypted authentication" or some such.

Cheers,
Martin
