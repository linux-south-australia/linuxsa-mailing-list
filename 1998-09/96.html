Java

hi all.
between kernels 2.0.33 and 2.0.35, the default location for the java viewer
when 
the kernel is compiled with binary support has moved from
/usr/local/java/bin/java to /usr/bin/java.
just change the path in fs/binfmt_java.c for java and appletviewer.

it took me a while to find, i was nearly ready to format & start again.
(thank god i didnt!)
