Re: mounting limit

This isn't a warning message, it's just letting you know what's 
happenning.
The second extended file system keeps track of how many times it has benn 
mounted, and if this number is exceeded when teh system starts e2fsck is 
run on it ...  just as a precaution. 
It isn't a bad thing to leave going, but if it really annoys you, you 
can set this number to be higher by using tune2fs -c # device, where # is 
the number of mounts between checks.

Leigh.
