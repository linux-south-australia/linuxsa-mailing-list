Re: X and Redhat 5.1

> I have a basic install of Redhat 5.1 and am using the fvwm2 window manager.
> 
> When I first start X it works fine but after a few hours (this machine is on 24 hours a day) thinks stop working.
> 
> The minimise and maximise buttons do nothing and to swap between applications/windows I need to click on the task bar as clicking on
> the window will not activate it.
> 
> Is there a cure for this ?

Try Start->Exit->Restart to get it working again; I don't know of a
cure.  You're using click-to-focus I presume?  If you switch to
point-to-focus is it any better?

> Also the time function is screwed up (it won't keep time), I know this is known problem but cannot find the fix. any suggestions?

The only time I've seen Linux not keep time is when the BIOS had power
management turned on.  If the problem is just that your time is set
wrong, see http://www.linuxsa.org.au/tips/ ('Linux, Clocks, and
Time').
