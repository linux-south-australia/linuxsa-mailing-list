I have contacted them about CBA NetBank under Linux also.

The reason I received was that the Crypto software is licensed
from overseas only for the Windows platform.

Not letting that deter me...

I have had varying amounts of success using NetBank under WINE.

Up until the most recent snapshot netbank would install, register and 
do everything that is needed as part of the install.  When trying to
run netbank after that, the software authenticatses itself but 
when it tries to bring up the tab based window it dies with a 
'Cannot open empty document'.

In the most recent version of wine (980913) looks like they
have started coding the support for the 'Tab Control' which
looks promising.  Unfortunately the install program doesn't work 
for the downloaded version.

They are getting there.  But aren't there yet.

BTW BankSA internet banking appears to work well under wine.  
Haven't used it though.
