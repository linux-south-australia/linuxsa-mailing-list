Re: Quality of Red Hat

On Tue, 29 Sep 1998, Matthew Golder wrote:

> I have 2x32MB + 2x8MB = 80 MB
> Linux recognises 64 MB,
> Win 95, plus CMOS memory initialisation recognises 80MB

Linux/x86 only detects the first 64M automatically, you need
to use a command line argument such as mem=80M.

For instance, in my /etc/lilo.conf, I have the line:
 
append="mem=128M aic7xxx=verbose aic7xxx=dump_card"

which gives command line arguments to the kernel specific
to my configuration.
