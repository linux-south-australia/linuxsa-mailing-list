Re: mail-all alias?

On Thu, 17 Sep 1998, Timothy Aslat wrote:

> hi all,
> can anyone help me figure out how to do a mail-all to every user on a
> linux system
> I think it's got somethign to do with sendmail or aliases but I can't
> find any documentation to help me figure out which.
edit /etc/aliases and create a an entry that reads along the lines:

mail-all-users:		root,user1,user2,........,userN

and then run newaliases as root. Then just send mail to
mail-all-users@your.machine.com it's that simple, no .aliases file like
some inferior mailers have ;)
