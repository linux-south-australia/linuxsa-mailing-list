RE: xterm's

I have found that if I define how the [color-x|nx|x]term in the
.Xdefaults file that I can not get the ctrl menu on right click and all
I get is a partial menu or just a beep. However, if I remove all of the
information for the terminal from the .Xdefaults file and specify it all
on the command line everything works ok and I get the menus back.
Try typing in a command line the following.
xterm -fg green -bg black -ls -sl 1000 &
and then try to use ctrl-<rightclick> you should then get a menu that
will allow you to change font size.

I can only go by what has happened to me. If it doesn't work for you
.... at least I tried. :D

Good Luck
