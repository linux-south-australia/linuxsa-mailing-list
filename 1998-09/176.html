I had similar problems with sound on Redhat systems when trying to
create a kernel with sound built into the kernel and in fact didn't
succeed until I tried the modular approach.

My sound card is a non-Plug'n'Play Soundblaster Pro which worked fine
after using these settings in the kernel config.

#
# Loadable module support
#
CONFIG_MODULES=y
CONFIG_MODVERSIONS=y
CONFIG_KERNELD=y

# Sound
#
CONFIG_SOUND=m
CONFIG_AUDIO=y
CONFIG_MIDI=y
CONFIG_YM3812=m
# CONFIG_PAS is not set
CONFIG_SB=m
CONFIG_ADLIB=m
# CONFIG_GUS is not set
# CONFIG_PSS is not set
# CONFIG_MPU401 is not set
# CONFIG_UART6850 is not set
CONFIG_UART401=m
# CONFIG_MSS is not set
# CONFIG_SSCAPE is not set
# CONFIG_TRIX is not set
# CONFIG_MAD16 is not set
# CONFIG_MAUI is not set
DSP_BUFFSIZE=65536
# CONFIG_LOWLEVEL_SOUND is not set
