Re: Key Bindings in bash

On Tue, 15 Sep 1998, Alex Garner wrote:

> I'm trying to bind the cursor keys in bash using the bind command (and if
> it works eventually ~/.inputrc)
the only bind I could find was the nameserver and the tk commans so I have
no idea about this.

> There seems to be reserved words in readline for things like DEL, ESC and
> RETURN, but nothing for arrow keys. I'd like to bind control-left arrow to
> backward-word and control-right-arrow to forward-word. It would also be
> nice to bind the home and end key.
> 
> Any ideas?
I am using a package called keymaps to remap parts of my keyboard (I
prefer the old SUN and NCD bindings with left Ctrl where Caps Lock
normally is) it is really easy to use, you just edit the default keyboard
mapping in /etc/kbd/default.map.gz and bob's your uncle you have a new
mapping next time you run the keymaps.sh which is in  /etc/init.d and is
run at boot.

Hope that helps
