My presentation on Tuesday night

Ladies and Gentlemen,

My presentation on Tuesday night is going to be titled :

"Getting Linux Connected, using Ethernet and TCP/IP"

And will cover the following topics :

Setting up the Hardware
  Overview of Ethernet
  A 2 Node Connection
    Connecting using Thin Coax
    Connecting using a 10BaseT Crossover
  A Multi-Node Connection
    Connecting using Thin Coax
    Connecting using 10BaseT
  What types of ethernet cards are there, and which type should I buy ?
  Configuring Linux to talk to your network card
  Can Linux see your network card ?

Configuring Linux to Network
  TCP/IP Overview
    TCP/IP Model
    Common TCP/IP Protocols
      IP
      TCP
      UDP
      Telnet, FTP, HTTP
  Linux IP Settings - IP Address, Subnetmask, Default Gateway
    IP Addressing
    Subnetmask
    Default Gateway Setting

Testing & Using your Connection
  Using PING
  Telnetting
  Using FTP

I am looking for some feedback on the above topics. If there are any I
have missed, or you want me to add, please feel free to email me, or the
list.

I am trying to keep the presentation at a fairly low level, ie. the
essential knowledge and information you need to get two linux boxes to
talk via ethernet, so please do not ask me to add a topic such as "So
now my boxes can talk, how do I set up Sendmail ?" These in-depth topics
can be covered at other meetings.

Thanks,
Mark.
