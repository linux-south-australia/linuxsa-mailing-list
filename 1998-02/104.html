My guess is that XCalc is wanting an a.out libX11.so.6 and not your
ELF libX11.so.6.  Do you have a libX11.so.6 existing in
/usr/i486-linuxaout/lib/ ?

The easiest solution might be to recompile the program (and if you are
not using any other a.out binaries then it will run more efficiently
too :).
