Re: [LIOZ] How do u set up LILO to optionally boot a Win 95hard-d rive...

Continuing the floppy idea, I always set my Linux systems to boot using
lilo installed on a floppy but Lilo is configured to boot from the Linux
partition, hence the system effectively boots from your hard disk but
using a lilo 'key disk' type setup with only 1 sector on the floppy (not
the whole kernel).  This makes for a quick&simple boot (disk in = linux,
out = windows) plus a simple security 'key disk' system.
