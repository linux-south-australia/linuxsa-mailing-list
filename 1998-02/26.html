Hey, I know you've already been given the answer, so I shan't repeat it
here.  You are obviously a newbie and so are excused for not knowing
standard netiquette.  The fact is you should try to find the answer
yourself, instead of expecting everyone on the list to be your personal
manual readers.

The Unix command to read about a command is "man".  If you had done a
"man chmod" you would have found your answer.

To help you with any next question, the way to find the command relevant
to some topic is to use the "-k" option to man.  For example, to find the
command relevant to mounting a disk you could say, "man -k mount".

Remember: try to find the answer yourself or you're gonna get a reputation
for being lazy or selfish.
