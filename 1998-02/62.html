Re: How do u set up LILO to optionally boot a Win 95 hard-drive...

Here we go again.  Windows 95 sees that it is on drive d: and is quite
happy being there.  If you were to install win95 to a second drive,
while the first is linux only, it would see itself as c:, and would be
unable to see the first drive, although as this is linux's drive, it
does not matter.  Windows would go around thinking it owns the computer,
while good old linux sits back and looks at both drives, and is able to
access both easily.  You can get a driver for win95 so that it can then
see the linux drive as d:.
