My computer has Two drives, with linux, and windows 95 both on the
second drive.
Look at the file /etc/lilo.conf
in it you will see something like this

image=/boot/default
        label=default
        root=/dev/hda1
        read-only
To boot Win95 off the second hard drive, add something like this

other=/dev/hdb1
        label=win95
        table=/dev/hdb

then run 'lilo' to change the settings, when you next restart your
computer, press tab at the LILO prompt, to get a list of the what lilo
can boot, and type win95, if you have installed win95 correctly it will
all go fine.

Watch out when installing windows 95, it does not like other operating
systems, and keeps giving me trouble when reinstalling things.
