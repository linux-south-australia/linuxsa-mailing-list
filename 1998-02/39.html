RE: Blocking telnet access (Port 23)

The ipfwadm I am not sure of, but you can try using hosts.allow and
hosts.deny in the /etc directory.

The hosts.allow file has a line similar to:
ALL : 203.42.3.0
or addresses of allowed hosts or networks seperated by comma's

The hosts.deny file has a line similar to:
ALL : ALL EXCEPT 203.42.3.0
or addresses of allowed hosts or networks seperated by comma's

I believe you will need to specifically allow ftp if you want to give people
ftp access (not annonymous) from outside of the networks listed.  Some other
people may be able to help you out with that.

I am not sure if it needs to be done with both allow and deny files, but
thats how I got shown.
