Re: IP forwarding problem (redhat)

> I was trying to set up a dialin line. It seems to be working execpt that
> it doesn't do IP forwarding. I have recompiled the kernel making sure that
> I have included IP forwarding, but when the system boots I get:
> 
> kernel: sysctl: ip forwarding off
> 
> I have checked that it is using the right kernel and according to the date
> from the uname -a output.
> 
> Ideas?

Hi Horst,

Presuming you are using Red Hat 5.0, you don't need to recompile your
kernel; the one that Red Hat supply will work fine.  All you need to
do is run netcfg and click the "Network Packet Forwarding" checkbox on
the "Routing" screen.  Either that or set "FORWARD_IPV4" to "yes" in
/etc/sysconfig/network.

Of course, if you had been at the last meeting you would know this :).
