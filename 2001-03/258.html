Re: Perl Formats

On Wed, 7 Mar 2001, Andrew Hill wrote:

AH> I'm using formats to make a report all nice and pretty, and this
AH> works nicely if I output the report to STDOUT. However, what I'd
AH> like to do is to write each line of the report to a variable in
AH> Perl, so that I can then email the report off.

formline and $^A are your friends here:

formline <<'FLOBB', $var1, $var2;
Some header line
@<<<<<< @>>>>>>
FLOBB
# $^A now holds the formatted output
