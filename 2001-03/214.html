Re: how to dissable ping in linux ...

howdy,

in RH 6.2 you can add the following line

net.ipv4.icmp_echo_ignore_all = 1

to the file
/etc/sysctl.conf

to disable ping response.

i found this in a book called securing and optimizing RH linux,
you can download it in pdf from
www.openna.com
