Re: tar '.*' files

> is there a way to include '.*' files in a tar archive without including '..'
> etc as well?

cd ..
tar zcvf /tmp/uni.tar.gz uni

tar under Linux is a GNU command and consequently uses evil info pages instead
of man pages.  Try 'info tar' and use emacs-like key bindings to navigate.

Shar and compress!
