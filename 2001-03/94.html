Re: Mozilla 0.8 - Java anyone?

On Thu, 1 Mar 2001, drwho@firstlinux.net wrote:
> Has anyone managed to get Java working in Mozilla 0.8 yet?  What is
> general consensus on a solution?

These bugs in bugzilla are tracking the problem:

http://bugzilla.mozilla.org/show_bug.cgi?id=66840
http://bugzilla.mozilla.org/show_bug.cgi?id=68356

Summary is: It's a broken install issue.

Some people have had some success by installing the java plugin (which
then does not work) and, after installation is complete, executing this
from their plugins directory:

~/mozilla/plugins$ ln -s java2/plugin/i386/ns600/libjavaplugin_oji.so .
