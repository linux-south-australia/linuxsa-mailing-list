>Are there any Linux alternatives (open source) that are compatible with MS

Outlook98 and 2000 can accept appointments in the vcalendar format, which is
open and is already in use by a number of open-source applications. Palm
Pilots also support this standard.

When I go looking for "linux groupware", I see about 50 different projects
all with the same goal, and nothing beyond the "beta" stage on sourceforge.
Rather than compatable, all we really need is an "equivalent" that is well
supported and works on a number of platforms including windows. I guess
exchange-compatability to the point of making outlook happy might be useful.

Browser based equivalents are cool (it's like 50% of the aforementioned
projetcs are PHP + mysql), but not for everyone. Can anyone point to a
project like I've just described?
