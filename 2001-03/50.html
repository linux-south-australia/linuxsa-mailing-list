Re: Print Queue Control

Hi...

> I want to use a SAMBA printer to give access to a group of users to a Unix
> printer. I want a subset of these users to have the ability to control the
> queue (pause and delete jobs). The generic users will be printing from a
> remote machine, but the administrators will have local Unix and Samba
> accounts, but most don't know unix.
> 
> How do I make an ordinary user have Administritive access ofer a printer
> without using sudo, or similar ? Can this be transfered to Samba then ?

Here's one possible solution:
Install LPRng on the machine.  LPRng has quite flexible access
controls, and you can give particular users access to various features
(like queue control).  It's not too hard to install, and it's got some
other nice features as well.

If SAMBA is configured correctly, the LPRng access controls will still
work, as SAMBA should run as the user that's connected to it.
