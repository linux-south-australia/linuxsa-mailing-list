I have found the Kodak DC265 ok. It uses a flashcard with commonly
supported chip (tcic). With a card reader it is possible to read this
with the pcmcia facilities in Linux.

I have been using a small command line program called ks to download
files. It talks to the camera via Usb.

I have the module Kodak dc2xxx enabled in the Kernel.
