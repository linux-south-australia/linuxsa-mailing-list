Re: Change the file systems size???

> I have Red Hat 6.2, can I change the size of my file systems???

I think you can only do that by recreating your filesystem; although
maybe I heard of a commercial program that resizes an existing ext2fs
partition.  Have you searched the web?

If you have more than one filesystem, and one is full and the other
emptyish, you can move directory hierarchies from the full to the
empty and use a symbolic link to point the old location to the new.

  $ su
  # cd /full
  # tar cf - dir1 | (cd /empty; tar xfp -)
  # rm -rf dir1
  # ln -s /empty/dir1 dir1

(That's another recipe: standard administrative procedure.)
