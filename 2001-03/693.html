GNOME 1.4 almost there

Last night Mel made mention that GNOME 1.4 would soon become available.  Well,
sooner than even he thought perhaps.  

GNOME 1.4 Release Candidate 1 has just been released, and if no serious bugs are
found it will become GNOME 1.4 in a short while.

This release includes Nautilus 1.0, the graphical shell from the guys at Eazel,
which is really cool.

If you want to grab it early before the mirros get flattenned, or want to
contribute bug fixes, go ahead, download now, from http://download.gnome.org/ in
the GNOME/stable/betas/gnome-1.4rc1/ subdirectory.

The other interesting thing is that Sun have contributed a test regression suite
for GNOME 1.4.  Testing is something that hackers don't typically like to do, so
Sun contributing this is a GoodThing(tm).

Anyhow, off to flatten my link :-)
