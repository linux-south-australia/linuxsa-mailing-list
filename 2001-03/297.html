Re: samba stopping

On 07 Mar 2001 21:45:16 +1030, Stephen Donaldson wrote:
> After an installation of samba on a friends machine I found that samba
> was stopping automaically, ie I started samba and green OK was seen,
> however, when I checked with samba status is was stopped!

My Development server runs Samba (RH7). Probably check your log files
etc for any hint of reason to die.

> This was on a RH 7.0 install and although I did not do the actual
> install I believe all of the samba RPM's were installed (common, server
> and client)   Would installing the client have anything to do with it.

See what you have installed by using rpm

rpm -qa | grep 'samba'

Should show you all the samba packages. Look on the RedHat CD for other
Samba RPMS using similar method

ls /mnt/cdrom/RedHat/RPMS/ | grep 'samba'

RedHat 7 is still a bit funny. Maybe you should consider installing
RedHat 6.2 or some other distribution that has a better track record if
its a mission critical box. Of course, no matter which distro you
choose, be sure to get the updates.
