Re: file copy prevention

> probably what I'll do is have a cron job which removes all said mpegs,
> jpgs which have the same filenames as those in the funnies directory

Don't do that, it will only encourage your users to rename the file.
Instead create a cryptographic hash (e.g. MD5) of every file and remove
the ones that are identical to those in your funnies directory.  Better
yet, replace those which are identical with hard or symbolic links to
the file.
