The documentation for the mod_ssl (when I last compiled it, at least) is
has two different methods of installation.
The first is the 'general person easy way' method which does the Apache
configure script as well. The second is the 'hacker-only' (or so they
call it) method.

I found that the second method was easier, read the INSTALL file about
half way down. It was a lot easier to follow for me.
