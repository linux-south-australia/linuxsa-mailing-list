RE: DNS Question (Kind Of)

I use the Apache Redirect command to accomplish this, assuming you desire
to actually change locations; www.original.com would forward you to
www.another.com

Here's an entry from my httpd.conf file

<VirtualHost professional.uvm.edu>

ServerName professional.uvm.edu>

<snip>

Redirect /highered http://www.uvm.edu/~dceweb/profprog/higherEd
Redirect /healthcare http://www.uvm.edu/~dceweb/profprog/healthCare
Redirect /pubadmin http://www.uvm.edu/~dceweb/profprog/publicAdmin/
Redirect / http://learn.uvm.edu/profprog/

</VirtualHost>

Its minimal, but gets the job done.
You would probably want to add something like

Redirect /website/ http://www.anotherdomain.com/website/

So if you entered http://www.originaldomain.com/website/ into the netscape
bar, it would immediately (after being processed by your webserver) be
replaced with http://www.anotherdomain.com/website/
