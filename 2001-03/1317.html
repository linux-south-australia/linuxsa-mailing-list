FYI: SINA Proxy Servers

Hey All,

A little while a go, someone asked about how you'd go about setting up a 
proxy server beind the SINA router.

I found that it is a simple matter of setting up a peer cache in 
/path/to/squid.conf . You can even make it logon under one username if you 
want it to, so theres no authentication presented to students or staff.[1]

I've got a squid.conf file that works for anyone that's interested. Mail off 
list if you wanted a copy.
