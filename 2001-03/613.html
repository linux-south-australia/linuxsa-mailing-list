Matt,
	Is it possible that the libraries are installed, but not in the location
expected?  If so (try `locate library-name`), then recreate /opt/KDE as a
symbolic link to the directory that actually holds the libraries the install
is looking for!  This is where Linux distro's really need to get together
and standardise.  Different distro's put stuff in different places, and then
we sometimes complain about Micro$oft's refusal to adhere to standards....
hmmmmmmmm; the rebellious become the same as that against which they rebel
sometimes.
Don.
