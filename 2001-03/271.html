Re: Perl Formats

Andrew Hill wrote:

> However, now I tell you that I want to use a format
> that looks like this:
> 
> @<<<<<<<<<   $@####.##   ^<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<
>                         ^<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<
>                         ^<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<

Yikes. This is getting messy, but I have a solution:

-------------------------------------------

use Text::Wrap;
$Text::Wrap::columns=45;

@lines = split /\n/, (wrap('', '', "$var3"));
$^A = "";
formline <<'END', $var1, $var2, $lines[0];
@<<<<<<<<<   $@####.##   @<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<
END
$mail .= $^A;
for ($i = 1; $i <= scalar(@lines); $i++) {
   $^A = "";
   formline <<'END', $lines[$i];
                          @<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<<
END
   $mail .= $^A;
}

-------------------------------------------

I guess a write to file would look cleaner, but this seems nicer
to me - I hate temporary file :-)

If anyone can think of a cleaner solution....?
