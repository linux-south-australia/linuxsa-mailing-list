usb problem

Mandrake 7.0 kernel 2.2.18

Following instructions from the Handbook I have connected two new usb
slots to the appropriate pins on top of an older Aopen Motherboard. Usb
support has been built in to the kernel, along with usb-uhci. 

Bios has usb enabled.

Enable usb legacy support......yes
Enable usb irq release........yes

Modpobe will load a number of modules ok,

Modprobe acm (modum) works
Modprobe dc2xx (camera) works

The module usbcore loads outomatically with the above. 

However the core module usb-uhci causes a fault

The problems appears to be no IRQ assigned. As this is not a pci card,
the system does not seem to be able to set/use an irq.
Can anyone suggest how this may be done ? It would appear that with this
sort of hardware it may have be done in Lilo or at the command line. 

Brian Marr
