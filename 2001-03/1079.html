Re: SQL syntax

> so i want to search for records with $name occoring in them, but the
> field may contain something like "fred" or "this persons name is fred".
> i want both to be displayed as matches.

Note that the correct answer to your question, which everybody else
already gave you, does not help find, "the name is Fred".  In Oracle you'd
say, "WHERE lower(field) like '%fred%'"  (Note that Oracle requires single
quotes around strings.)  I don't know how relevant this is to MySQL.
