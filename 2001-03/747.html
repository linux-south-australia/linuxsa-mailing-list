RE: Samba Drive mappings

Looks a bit on the weird side of ordinary!

You may want to post your smb.conf & netlogon.bat.

Run the following at a console prompt on the 
server:  "testparm >smbconf.txt" (no quotes obviously)
you'll need to hit enter after executing the command as 
testparm waits for this if you run it normally.
(also you may want to edit the file before posting to remove
any site-specific details if necessary)

E-smith is a little peculiar in setting up smb.conf in that 
it uses templates for everything, i had a few hassles with it
because i was used to hacking up the smb.conf file on its own,
in E-smith that can break things
