Re: Compiling Apache 1.3.19/1.3.12 with mod_ssl

Matthew Geddes wrote:

> After running the configure script for mod_ssl (with the extra apache
> options), it says to change directories and run configure. If I don't, I
> can't make, as there's no Makefile.

Oh, er, uhm.

Can I just check that you are doing something like this in the mod_ssl
source?

./configure --with-apache=../apache_1.3.19 --with-ssl=../openssl-0.9.6
--other-apache-options --enable-module=ssl

It should say to after doing that:

Now proceed with the following commands:
  $ cd ../apache_1.3.19
  $ make
  $ make certificate
  $ make install

I assume you've got the right mod_ssl source for the right version
of Apache?
