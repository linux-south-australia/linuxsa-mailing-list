> > P.S. How would I send a ping for example through a non default
> > interface
> > to an unknown host?

[root]# route add -net ${MY_IP_AT_ISP} netmask 255.255.255.255 ppp0
[root]# route add gw ${MY_IP_AT_ISP}
[root]# ping ${AN_IP_ADDRESS}

Which does:

* add a route to your ISP via ppp0
* make it the default gateway
* ping some IP address

You may need to:

* use ifconfig to determine what your MY_IP... is
* use route, followed by route del to get rid of a conflicting default
GW

And don't forget man route :-)
