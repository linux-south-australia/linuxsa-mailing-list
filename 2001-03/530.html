COMMERCIAL: Samba courses possibly available

Hi,

Because someone has expressed an interest in Samba courses, I am posting
this in this mailing list and one or two others.

Basically, is there any interest out there in 1-day or 2-day Samba course
before the end of the FY?

Course format would be:

Day 1: Intro to Samba, which would cover the basics and get you up to speed
on many aspects of Samba and accessing Linux/UNIX systems from Windows.

Day 2: Advanced Samba. This would focus on Samba 2.2.x and would take you
through setting up Samba to allow NT and Win2K to join a domain and
handling login scripts, profiles, and so forth.

More detailed outlines available if there is any interest.

To put this course on, cover my time, update the course materials, hire a
venue, hire machines, etc, I am going to be charging $395/day, and
delegates can choose to go on one or both days. I will probably also throw
in a copy of Special Edition, using Samba as well :-)

Please don't bother to send me flames about how it is so expensive. There
is only so much of my time that is free.
