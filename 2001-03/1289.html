> I'm trying to compile ssl support into apache again, but this time, I'm
> having a little trouble. I have successfully compiled and run
> apache/mod_ssl on FreeBSD 4.2 and RedHat Linux 6.0 and 6.2 before. I
> have a different RedHat 6.2 machine, but I'm having a few problems
> (well, actually one big one).

Using Dynamic modules in apache makes adding stuff like this really
easy. Just compile apache with mod_dso, and then use apxs to add new
ones without having to recompiling your existing setup each time. Also
handy for weird module combinations that don't always play together
nicely when compiling.
