<h2>Tech Tip of the Week</h2>
<p>   
If you've started using Mozilla, and find that it makes fonts too
small, just create a user.js file and set the minimum allowed
font, like this:
<p>
<pre>
user_pref("font.min-size.variable.", 14);
</pre>
<p>
More useful tweaks to get rid of other web annoyances such as pop up
windows and animated GIFs are at the:
<a href="http://www.mozilla.org/unix/customizing.html">Mozilla web
site</a>.
