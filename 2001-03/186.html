Re: Mozilla, Java and Fonts

Hi David,

If it's a Java problem you should be able to edit font.properties in the
lib directory where your java install is.  This can be in various places
depending on how you have installed java.

"find / -name font.properties" will probably find it eventually.

In here you will find X logical font descriptions for the standard fonts
that java uses (serif, sansserif, dialog etc).

Note that the IBM JDK 1.3 has the following warning in it:
--snip--
# Note: modification of this file is risky and not supported
--snip--

I think it is possible IBM are being painful and pedantic here.  The Sun
JDK 1.3 font.properties has no such warning.
