There is two tools I know of which can do this with a linux ext2 file
system. The first Partition magic. This is commercial software which runs
under windows/dos. The second is a program called GNU Parted. Which is
slowly trying to duplicate the text based interface to partition magic.
Both work, it's just the expensive one has a pretty progress bar :)
