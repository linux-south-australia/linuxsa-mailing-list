ipfwadm was the linux 2.0 version, ipchains in the 2.2 version, and ip 
tables is the 2.4 version. wether these are installed of not, should not 
really matter, you should still be able share the connection. ip chains 
is primarily a firewalling/accounting tool. provided that IP 
masquerading is enabled and you have the correct routes set up, it 
should work fine, security will be non existant!
just set up your other machines with the IP address of that machine as 
the gateway.

read the firewall how-to @ linuxdoc.org
