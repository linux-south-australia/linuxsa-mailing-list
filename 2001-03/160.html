Apparently ReiserFS is a bit quicker. I would also consider looking at
software RAID if hardware RAID is too expensive. I have had a bit of
success running IDE disks with software RAID 0. It seemed a fair bit
faster, but I haven't done any benchmarks. Also, look at the squid FAQ.
It has a fair bit on performance of proxy servers if I recall correctly.
