Re: Netwrok problems -Red Hat 7.0

> The address space is the 192.168.0.x class and the server is 192.168.0.1
on
> both eth0 and eth1. the hosts are 192.168.0.3 and 4 with the gateway set
to
> the server.

Herein lies the problem...

The two interfaces _must_ have different IP's otherwise you will end up with
all sorts of wierd stuff happening

So, change eth1 to 192.168.0.2, modify your route so that traffic can flow
between them and all should be well

Oh, the machine hanging off eth0 needs the gateway set to 192.168.0.1 and
the machine off eth1 will need a gateway of 192.168.0.2
