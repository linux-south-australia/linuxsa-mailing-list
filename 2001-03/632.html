Re: installation problem

Hi there,
    I don't know if this will work or not, but try setting
the global path in /etc/profile and see if that makes a
difference. Usually .profile in a user's directory adds
whatever modifications to the global path, but sometimes it
overwrites it if you have done it wrong. There is quite alot
of stuff that runs with Qt AFAIK so you probably want to
make it available to everyone on your system anyhow (O.K.
you may have a single user system, but you might want to
create a second account some time later). The modifications
you have to make to /etc/profile are fairly obvious when you
edit the file. Hope this helps.
    Cheers!
        Andrew.
