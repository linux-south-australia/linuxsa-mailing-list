Re: Title Case in PHP

Andrew Reid wrote:

> > What is "Title Case"?
> 
> Like "lower case" or "UPPER CASE" or "MiXeD CaSe", "Title Case"

There must be about 12 ways to do this, even more in perl :)

Here's my attempt:

function tc($string){
	$tok = strtok ($string," ");
	if (!$tok){
		$tok = $string;
	}
        while ($tok) {
        	$ret.=
strtoupper(substr($tok,0,1)).strtolower(substr($tok,1))." ";
        	$tok = strtok (" ");
	}
	$ret = trim($ret);
	return $ret;
}

strtok() skipts multiple instances of the same delimeter, you might want
to use explode() or split() if you need to keep spaces intact.
