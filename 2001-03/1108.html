It gets easier every time you do it :-) Make sure with PHP that you read
the INSTALL file thoroughly, it makes it a lot easier. Seeing as Apache
is already installed, you'd probably want to install it using
`--with-apxs=/usr/local/apache/bin/apxs' which compiles apache as a
module, rather than into the core. It is marginally slower than having
it hard-compiled into Apache, but I daresay it won't make too much
difference.
