Re: how to dissable ping in linux ...

> Rakesh wrote:
> 
> I have written one program which can respond to any icmp ping request
> and it is working fine ..
> 
> now i want to run it on the linux machine in such a way that whenever
> the ping request comes only my program should respond to it and the
> linux ping program should not ........

One way is to re-write your program to use "divert sockets" for linux,
and setup appropriate ipchains rules to funnel ip packets of type ICMP
(0x01) to your software.

This link might help:

http://www.google.com/custom?q=linux+divert+socket

Divert sockets are a handy way to write IP-mangling software for a
specific application without re-writing the kernel.
