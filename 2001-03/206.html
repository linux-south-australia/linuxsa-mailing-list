Now, I'm not entirely certain what you want to do here. However, ping is
a program that sends ICMP echo requests and interprets ICMP echo
replies. Note that it does not _respond_ to ICMP requests - that's left
to the kernel to do.

I think what you're trying to say is:

"I want to write a program that responds to ICMP echo requests (and
maybe other ICMP messages as well) in lieu of the kernel's default
behaviour..."
