Solaris gets a little cranky, but works. QNX 4 appears to work. SCO and
plan 9 don't work well at all. I've also had troubles with OS/2 warp 4,
but I've only tried that on one machine. Also, for some reason, many
operating systems don't like being installed on anything but the first
SCSI disk. This includes Windows 2000, Windows NT, Novell server 4 and
Solaris. They're all quite happy to use other disks, but won't boot from
them (even with a boot manager). I think I once had BeOS running off the
primary slave IDE channel.
