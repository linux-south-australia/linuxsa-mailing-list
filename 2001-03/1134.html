You can convert it to html if you wish with pod2html and man2html.
man pod2html and man man2html for details.
POD in case you are not aware is "Plain Old Documentation", which is the
"standard' perl documentaion format and can be embeded in code.

And yes if you want to program much perl, a book will be useful, though
the perldocs and man pages are very useful and quite complete.

The definitive perl text is "Programming Perl" by Larry Wall  et al,
published by O'Reilly.

man perlbook for details.

There is a companion book "Learning Perl" for beginers, but Programming
Perl is _the_ langauge text as K & R's "Programming C" is to C.

If you know any programming at all, then "Programming Perl" will serve
you well.

If you dont already program tehn you might want "Learning Perl" as well,
but eventually you will want "Programmng Perl".
