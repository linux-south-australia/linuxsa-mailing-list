Re: Throttling bandwidth on the WAN

well, you're in luck...I have just spent the past couple of days working
on getting rate limiting working on a slackware machine for a client so I
have been saturated in information on the topic. :)

It works rather well, I'm quite impressed with it considering it only took
a short while to figure out how to get things working.

It uses iproute2 which is just basically a linux version of what cisco
routers do, albeit a bit cutdown i think.

In iproute2 you'll be using a utility called tc (traffic control), it
handles all the setting up of containers and the rate limits across
individual ip's or entire subnets.

Here's a couple of links which helped me out:
Advanced Traffic Control HOWTO
http://www.ds9a.nl/2.4Routing/HOWTO//cvs/2.4routing/output/2.4routing-9.html

TC Command List
http://linux.bergqvist.se/tc/index.html

Have fun. :)
