Re: Sendmail quick query.

Timothy Aslat wrote:
> 
> I know this isn't strictly Linux, but I'd like to know if there's a simple way I
> can bind Sendmail to a specific IP address or group of IP addresses on a machine
> using multiple IP's.  I've scanned through the man pages and even dragged out
> the Bat Book but to no avail.  Maybe I'm checking under the wrong headings.

See the Addr option in the DAEMON_OPTIONS() M4 macro.  It is
documented in the Operations Guide under the DaemonPortOptions
variable.

  FEATURE(`no_default_msa',`dnl')dnl
  DAEMON_OPTIONS(`Port=smtp,Addr=127.0.0.1, Name=MTA')dnl
  DAEMON_OPTIONS(`Port=smtp,Addr=1.2.3.4, Name=MTA')dnl

This option is also used to set authentication requirements
(eg: local delivery and messages from local machine don't
require authentication, everything else does).

If you use Red Hat Linux you should install the sendmail,
sendmail-cf and sendmail-doc RPMs to get a full sendmail
installation.  You will then have the Operations Guide
(/usr/share/doc/sendmail/doc/op/op.ps) and the M4
configuration macros available.
