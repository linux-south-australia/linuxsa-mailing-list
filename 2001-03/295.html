Re: pppd

Martin,

If you read the documentation for pppd, I believe it will tell you that the
defaultroute options does NOT replace an existing defaultroute. To do that you
will need to write a short script which deletes your current defaultroute and
adds your new one, which is run from the ip-up script. Then you will also need
to write another one which is reverses the first, and place it in the ip-down
script, to replace the defaultroute on your LAN when the ppp connection comes
down.
