RE: Netwrok problems -Red Hat 7.0

> I have installed two ethernet cards in the server to avoid having to use a
> hub. Unfortunately I am having problems with each of the nodes seeing each
> other (except for the server that can see both nodes). I was wondering
what
> steps I need to take to facilitate a working network. 

I assume both workstations & the two cards in the server are on the same
IP range, so I won't ask the obvious ;-)

You may need to enable forwarding between interfaces on the server.
Try the following at a console:

echo 1 >/proc/sys/net/ipv4/ip_forward

(or if you're into the gui-based ways of doing things, use linuxconf to
enable
routing in your client network settings, the above method is much easier
tho)
