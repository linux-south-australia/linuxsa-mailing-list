Andrew Reid wrote:

> I just wanted to ask some of the Perl gods out there two questions.

I know you directed this question at perl gods, but I hope you don't
mind a mortal answering :)
 

> In PHP, if I want to join a string to another one, I can do it like
> this:
> 
> $foo = 'Your name is ' . $name ;

This works just fine in perl too, exactly as you've written it.
 
> Is there a site that has consolidated the perl documentation into a
> nice, friendly format like PHP.Net has with there 'quick reference'
> feature? I understand that to document all of Perl is near impossible,
> due to the mamoth amount of modules that people have written, but the

It's all at: 

http://www.perl.com/pub/v/documentation

or 

http://www.perl.com/pub/doc/manual/html/pod/perlfunc.html 

if you just want the closest thing perl has (or had, since they
integrated it with search) to PHP's quickref.

You'll also find it on your own machine with the 'perldoc' command. I'm
sure all the PHP docs are on the system too, but I always find myself
back at php.net :)

ie: perldoc perlfunc
