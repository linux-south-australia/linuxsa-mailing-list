linux support for Cyrix?

Can anyone tell me what the situation is for
support of Cyrix CPUs in linux?

I'm trying to install SuSE linux 7.1 on a
Cyrix 333 MHz processor of some sort.
Every time, the machine hangs at the same place
for each of various installation strategies.
After about 9 attempts, I'm wondering if this is
a CPU problem.

The SuSE site says that they support
150-200 MHz Cyrix chips.
The /proc/cpuinfo thing shows the CPU as 250 MHz,
although it has a label syaing that it is 333 MHz.

I got on OS onto it the other day, but then I wiped
it and started again, with a new 30 GB disk.
It tends to stop while installing the glib library.

This problem occurs with YaST and YaST2 in different places
in the installation procedure.
The problem is always a complete freeze of the machine.

kernel.org lists architectures, but is not specific about
which kinds of x86 chips exactly are supported.

I just have this faint memory in my mind that there's
a problem with the Cyrix of some sort.
