Re: database backup

> Essy Ren wrote:
> 
> Can you give me some simple code about making shell scripting using
> bash to backup database ?

It really depends on the database. Most DBs allow you to dump the
database to a text file of SQL statements. You can then dump that to
tape. You would need to check out the manual for the database engine you
are using. Most of them will have a section on backups.
