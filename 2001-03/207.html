Re: how to dissable ping in linux ...

> I have written one program which can respond to any icmp ping
> request and it is working fine ..
>
> now i want to run it on the linux machine in such a way that whenever
> the ping request comes only my program should respond to it and the
> linux ping program should not ........

As I understood your program was written not for Linux originally. For
windows system?

Anyway it's hard to understand how you made such program. You need to hack a
stack to intercept such packets or use some well documented hooks (like
tcpdump do) which depending on OS which in turn means that such program
cannot be transferred directly from one platform to another just by
recompiling because it's platform dependent.

Also I cannot understand the reason to intercept ICMP packets - I think that
kernel cope with them well! :)

And finally: as David mentioned - ping program not responding to incoming
ICMPs. It only sends them and listens for replies. If you want to disable
this program do

rm -f `which ping`
