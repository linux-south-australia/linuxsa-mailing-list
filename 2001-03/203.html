UPS software

for the info of those looking at UPSes for their linux/unix systems:

SOLA are a very reliable brand, they are actually made by BestPower in the 
US which is now owned by Invensys.

They have their own custom software called CheckUPS which works on pretty 
much all platforms including unix, here is a blurb from their website:

------- Start copied text ------------
CheckUPS II for UNIX is available for monitoring UPS models with either 
Advanced (RS-232 communications) or Basic (contact closure communications).

Jump to the CheckUPS II Basic documentation by clicking here.


---------------------------------------------------

For users with UPS models capable of Advanced communications, CheckUPS II 
Advanced for UNIX provides the user with software capable of being used in 
several scenarios.

Possible Scenarios:

UPS monitoring to be done by a UNIX system connected locally (via RS-232) to 
its UPS, where additional, networked servers and/or workstations need to 
monitor the same UPS remotely (via TCP/IP). Click here to view a diagram of 
this scenario.

UPS monitoring to be done by a UNIX system connected remotely (via TCP/IP) 
to a separate server or workstation running CheckPS II Data Server software 
(UNIX, NetWare or NT/2000 version).

UPS monitoring to be done by a single UNIX system connected locally (via 
RS-232) to its UPS (No remote monitoring possible by other servers or 
workstations because integral Data Server is not started).


------- end copied text --------
