Re: Wine....

On Sat, 20 Feb 1999, Mike Carter wrote:

> Hi Matt and Linux sa,
> I read your note on using wine.  How do I get it to see the win ini?
> I have wine setup.  The latest version on debian and it complains it can't 
> see it. TIA,
> Mike

Have a look for wines config file. It should be in one of the following
locations:

/usr/local/etc/wine.conf
/etc/wine.conf
/usr/share/wine.conf

In it you can specify where your windows directory is, which sounds like
your problem.
