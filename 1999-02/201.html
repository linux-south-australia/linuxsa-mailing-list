Re: Checking Network Status

>What would be the best way from a shell script to check the status
>of a network connection (PPP in this case) so that I could either 
>bring it up if needed, or just run the apps that need the connection
>if the connection is already up.

ifconfig <device> will give you what you want, or netstat -i. Alternitevly
you could ping a known host on the other side of the link, but just greping
the results ifconfig would be quicker.
