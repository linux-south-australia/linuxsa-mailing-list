future linuxSA seminar topics

Here's a short list of topics which I think would be suitable
topics for future linuxSA seminars:

1.	Electronic commerce (how to set up e-commerce for a web site
	using linux).

2.	Device drivers (how to get started writing device drivers software
	under linux).

3.	Internet API programming (how to write one's own software
	to communicate with TCP/IP and UDP/IP sockets).

4.	X window system programming (summary of styles of X window
	system available with linux, and APIs, GUIs etc. for
	programming them).

5.	Mini-Perl course, especially for CGI scripts.

6.	TCL/TK introduction.

7.	ext2 file system intro (i-nodes, hard links, directories etc.)

Is there a form somewhere where people can submit
topics for seminars, and view what other people have submitted?
Could this be done in PHP?
Will skippy find true love?
