>When the computer boots up or shuts down, (when it changes to run level 1, i
>think) it says:
>
>modprobe: can't locate module net-pf-4
>modprobe: can't locate module net-pf-5
>

I have been fascinated by these messages since I first installed
kernel 2.0.30 a year or so ago. I have a perfect 'work-around' for
it but I'm just as curious about what causes them.

Write the following lines to "/etc/conf.modules" to
get rid of the messages on boot:

#########################
alias net-pf-5 off
alias char-major-10 off
alias net-pf-4 off
#########################

Anyone got a better solution for this one? :-)
