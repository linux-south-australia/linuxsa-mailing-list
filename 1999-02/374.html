Re: Linux on 2nd-ary master.

On Tue, 16 Feb 1999, Malcolm McGrath wrote:

> Is it possible to install lilo (in a manner that works) on the secondary
> master IDE HDD?.
> My system has
> HDA - 3.2G Quantum (Win98 partition)
> HDB - LS-120 UHD Floppy Drive
> HDC - 1.2G Linux
> HDD - CDROM

Certainly is, reverse the ls-120 & the cdrom and you have my setup -
different HDD's though.

> 
> So far i think tyhat i have done something wrong, when i install lilo, it
> will boot linux but not windows, and if there is no disk in the LS120 drive
> it spits errors.

Strange. Here is my lilo.conf

boot = /dev/hda
timeout = 2
compact
prompt
  vga = 4             <-- you may like to leave this line out or change it
  root = /dev/hdc1
  read-only
map=/boot/map
install=/boot/boot.b
image = /boot/vmlinuz
  label = linux
other=/dev/hda1
  label = dos


> I have a file with them in but i cant mount my windows drive because linux
> says it has a bad superblock on it. (Only happened after i installed lilo.)

Boot off a win boot disk. If it doesn't recognise any C: drive something
is really stuffed.
