> The module I downloaded says to apply a patch called ip_masq_icq_ker.diff
> (which is with the module source) into the kernel and to rebuild the kernel.
> My question is how do I apply it? There is no man page for diff on my system
> and the help makes no sense to me at all. I have the 2.2.1 kernel source in
> /usr/src/linux
> 
> Can anyone help? Thanks if you can...
> Dazz

Diffs are used with patch.

To apply it try:

patch -p1 < ip_masq_icq_ker.diff

If it asks for a file try -p0 instead.
If it still fails you may have to find the directory with the file that
needs to be patched. Have a look in diff and the top line will indicate
any file.
