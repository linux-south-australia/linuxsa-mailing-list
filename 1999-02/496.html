I think you should remember that jpeg compression is not reversible - it is "lossy".
i.e. it is not like say gzip where the file you get back is exactly the file you put in.
So when you "load" a jpeg image you have a different file from the one you started with.
When this gets compressed you will get different results. The only way to test jpeg
compression is to start with the original file in both cases.
