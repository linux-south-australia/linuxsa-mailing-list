Reversing the boot order in lilo.conf

I've been trying to reverse the boot order in my lilo.conf file, but
haven't been able to get the syntax right. Could someone show me how to
change the lilo.conf below so that NT is the default operating system.

Regards, Steve

========== begin lilo.conf ============

boot=/dev/hda
map=/boot/map
install=/boot/boot.b
prompt
timeout=50
image=/boot/vmlinuz-2.0.36-0.7
	label=linux
	root=/dev/hda9
	read-only
other=/dev/hda1
	label=nt
	table=/dev/hda

========== end lilo.conf ============
