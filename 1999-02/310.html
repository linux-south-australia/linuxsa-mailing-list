Re: FTPd mail & logs

On Tue, 16 Feb 1999, Adam Crawford wrote:

> G'day all :)
> 
> I was just wondering is it possible to have ftpd mail me daily logs?
> and perhaps make it log to another file other then /var/log/messages for FTP
> useage only? :)
> 

Hmm, Yes it is possible.
Put the following in a daily cron job.

#!/bin/sh
ftp << EOF
open somehost.somewhere
cd /somedir
put /var/log/messages             # file(s) you want to send 
close
quit
EOF 

Then make a .netrc file in root's dir with the login and password to the
other machine. Make sure it is chmod 600

man ftp 

for more info on .netrc
