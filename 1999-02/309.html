Re: Linux on multi-processor computers

On Mon, 15 Feb 1999, Joao Campos wrote:

> Will linux use all the processors available?
> What I want to know is: if we have a server with a dual pentium, four
> xeons, or even a multi-alpha, will Linux use them? What about
> UltraSparc?
> 
> This is pure curiosity . . .

Linux has (S)ymmetric(M)ulti(P)rocessing support. If you really want to
make good use of both processors then use 2.2.x as 2.0.x didn't support it
well. 2.2.x has had MUCH code added for SMP support and it will kill NT
with its speed. 

> Where can I learn more about this?

In the kernel source dir...

Documentation/
	smp.txt   
	SMP.txt
	IO-APIC.txt

On the web:
	http://www.irisa.fr/prive/mentre/smp-faq/

The above are from the kernel config help.
See:
	Processor Type & Features ->  Symmetric multi-processing support

under: make menuconfig
