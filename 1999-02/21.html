Re: New User: help needed SB AWE32 Setup under Redhat5.2

On Mon, 1 Feb 1999, Matthew Mills wrote:

> Also do I always have to log back in as root to shutdown?

You have to be root to run shutdown, however this doesn't mean logging out
and logging back in. The 'su' command allows you to temporarily become
another user, and the -c flag runs a single command in this state. If you
don't give the username, root is assumed.

So to shutdown from a user account, type

	su -c '/sbin/shutdown -h now'

and it will ask for the root password before running the command. You have
to specify the location of 'shutdown', because a user account doesn't
usually have /sbin in the path, and su doesn't change the environment
unless you tell it to.
