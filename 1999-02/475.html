Re: sharing via samba

You can turn off the (now enable by default) windows password
encryption by making a change in the windows registry.

There are files and documentation on how to do this in the Samba 
Documentation, and the samba site has some regedit files
which you can use to make these changes.. they are..

--8<---(begin) NT4_PlainPassword.reg --------------------------------
REGEDIT4

;Contributor:   Tim Small (tim.small@virgin.net)
;Updated:       20 August 1997
;Status:        Current
;
;Subject:       Registry file to enable plain text passwords in NT4-SP3 and late
r

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Rdr\Parameters]
"EnablePlainTextPassword"=dword:00000001
--8<---(end) NT4_PlainPassword.reg ---------------------------------------

and

--8<---(begin) Win95_PlainPassword.reg ---------------------------------------
REGEDIT4

[HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\VxD\VNETSUP]
"EnablePlainTextPassword"=dword:00000001
--8<---(end) Win95_PlainPassword.reg ---------------------------------------
