Re: Questions about upgrading kernal

RH 5.2 upgrades to 2.2.x pretty easily I believe.
There are some RPMs to do it for you, I forget the url, but I saw it on
slashdot.org, and if you look back on the LinuxSA list, it's probably
there too.

The only trouble I have had in going from 2.0.x to 2.1.x/2.2.x on my
home machine is that smbmount has been broken. I suspect I need a newer
samba version, and I downloaded samba 2.0.2 last night, and I will see
if the newer smbmount in there fixes anything.
(Has anyone else had this problem, and how did they fix it?)

I started using 2.1.x kernels a while ago, and upgraded various parts of
the system to work with it at the time, so if you can run a late 2.1.x
kernel, a 2.2.x kernel won't really require anything new at all.

The main things that come to mind in upgrading from 2.0.x to 2.1.x/2.2.x
would be the pppd, and ipchains replacing ipfwadm.

-Toby
