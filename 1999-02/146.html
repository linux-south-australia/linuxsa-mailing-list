> How can I determine the identity of 129.127.134.80 ?

The machine is...

Name:    athena.itd.adelaide.edu.au
Address:  129.127.134.80

In the DNS there are two forward and reverse mappings.  The forward mappings
are the general ones that we see, as in given a name ...

	www.slashdot.org. 

that resolves to one or more IP addresses.

The reverse (which is what we are looking for here) is actually

	80.134.127.129.in-addr.arpa.

Which resolves to a PTR record to athena.itd.adelaide.edu.au.  This relies
on the system administrators of the domain putting this information
in the DNS.

Although the actual mechanics are a bit complex, in short most IP addresses 
have a reverse mapping to a human readable name.  The reverse lookups can
be done very easily using nslookup or dig.

Simply type in using shorthand 

	nslookup 129.127.134.80
or
	nslookup 80.134.127.129.in-addr.arpa.

Hope this helps.
