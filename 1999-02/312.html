There are two variables you need to set which are: lpr-command and
lpr-switches.  

To print a postscript file in my shell I normally do something like

cat file.ps | lpr -Phplj -h

Therefore I have something like 

(set-variable 'lpr-command "lpr")
(set-variable 'lpr-switches (list "-Phplj" "-h"))

in my .emacs.  You will need to put what is a propriate for your
machine and printer.  

If you bring up a file in emacs and then do 
M-x ps-print-buffer-with-faces under emacs, it will make a postscript
version of what you see and pipe it to the command set by
lpr-command and lpr-switches.  You will also get all of the nice faces
if you have font lock mode on.  You need to have a postscipt printer
or some way to print a postscript file though.
