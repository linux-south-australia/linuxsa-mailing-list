Re: Routing in a PPP network

> At the hub, however, there will be lots of PPP connections out to the rim
> networks, so a default route will not be suitable. What I need is to be
> able to insert lots routing entries, one for each rim network.
> 
> I tried adding stuff to /etc/sysconfig/static-routes for each ppp link, but
> this firstly does not work, and I could never ensure that a particular rim
> node comes up on a particular pppn device.

Hi Richard,

What you want is to add the routes when the link comes up.  From the
pppd man page:

  /etc/ppp/ip-up
    A program or script which is executed when the link is available
    for sending and receiving IP packets (that is, IPCP has come up).
    It is executed with the parameters

    interface-name tty-device speed local-IP-address remote-IP-address
    ipparam

Those parameters should be enough for you to determine which route to
add where.
