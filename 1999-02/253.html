you need to set the environment variable called KDEDIR to /opt/kde.

eg: in bash (i think)
 set KDEDIR=/opt/kde; export KDEDIR
in (t)csh
 setenv KDEDIR /opt/kde

or something. check the man page for your shell (probably bash) to see if it
will work. sorry i cant check, but im forced to '95 at school.
