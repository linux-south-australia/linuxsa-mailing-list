Re: netscape problems

> does anyone know how to stop netscape from adding www and com when I try
> to contact localhost. This is a problem that has only just started and
> is a real pain as it stops me from accessing my apache server.

Um, it only does this if it cannot resolve the domain name you
type in. It tries to find localhost first and obviously it cannot.
Try adding an entry to your /etc/hosts file that reads

127.0.0.1	localhost.localdomain	localhost

This should do the trick nicely.
