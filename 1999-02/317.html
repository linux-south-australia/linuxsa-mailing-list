Re: Linux on multi-processor computers

Hi there,

Have a look at the smp faq below, it may answer some of your questions.

http://www.phy.duke.edu/brahma/smp-faq/smp-faq.html

We are using a duel processor machine as a quake 2 server and it works
brilliantly.  From my (very) limited understanding I think that rather than
share (for eg) 1 quake process across the two processors, if I start another
quake process one is likey to be handled by each processor.  I think it
works this way unless you have software especially written for multi
processor machines.

Hope this helps,
