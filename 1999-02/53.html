Re: FTP upload: default permissions

> 
> Hi all,
> 
> Is anyone familiar with where/how you set the default file permission
> for a file that is uploaded or dir created via FTP.

The assumption here is that you are using wuftpd.

There is an undocumented (well some say that the ultimate documentation 
is the code so it is documented) command line switch which allows you
to set the default umask for connections.

It is in the source :)

So if you are running ftpd from inetd then you need to add the '-u'
switch.  From an inetd that I have worked with...

ftp             stream  tcp     nowait  root    /usr/local/ftpd/bin/ftpd ftpd -u027

Sets the default umask to 027 - user rw, group r other -
