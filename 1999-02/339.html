Hi.

I dunno, I never have tried it, (hate floppies), but there is a
package called mactools. It may do what you want. The filesystem is hfs
or (more recently) possibly hfs+. Ancient macs use mfs, but you are
unlikely to see that.

If you have access via ethernet between the linux box and the mac, you
can run a package called netatalk on the linux box that allows file
transfer (and print serving) between the two. The shared directories on
the linux box appear in the mac chooser. Works well, needs appletalk
enabled in the kernel.

The issue must have been dealt with in the maclinux and linuxppc
distributions, that's where I'd start looking otherwise.
