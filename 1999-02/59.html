easier to use

cd /opt

tar cvzf - . | ( cd /newdrive/opt ; tar xzf - )

Doesn't need any intermediate storage

>

> tar xvf bigtarfile.tar

WARNING WARNING ****
ln -s /newdrive/opt /opt
