Re: Command-line editing

Hi Brenton,
    For the least painless method of editing a file from the command line,
and without having to learn vi the hard way (No vi emacs wars here please),
try typing just

emacs <filename>

on the command line, or

emacs -nw  <filename>

and this will launch emacs with the No Window option. Editing the file
should be straight forward then. To save the file, hold down

<CTRL> x, then s for save.

To exit hold down

<CTRL> x, c.

You can use vi if you want, but for an absolute beginner, unless you have a
manual it may be a bit overwhelming. Also if you get stuck, there is an
online tutorial that comes as part of emacs.

    Cheers!
        Andrew.
