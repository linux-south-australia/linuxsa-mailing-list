On Redhat systems that boot into full multiuser runlevel (init=3), the
pertinent files are found in /etc/rc.d/rc3.d.

The method I use is courtesy of Geoffrey*.

For every service that I do not wish to start at boot time, I'd change
the 'S' in front of the filename to an 's' like so:

mv S80sendmail s80sendmail
mv S60nfs s60nfs

Then, if I ever need to start a particular service, I'd type:

/etc/rc.d/init.d/sendmail start

When I don't need that service anymore, I'd type:

/etc/rc.d/init.d/sendmail stop

Hope that helps.
