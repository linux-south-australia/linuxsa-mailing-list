Hi.

Do you have your printer correctly configured? To find out if so, try
printing something with lpr, or just use Red Hat printtool to print a test
page.

emacs uses the printer configured with the environment variable PRINTER.
If none is set, it uses the default lp printer, if there is nothing
attached to lp, it won't do anything.

To set this up try:

export PRINTER="the_name_you_gave_your_printer_here"

before starting emacs.
