Macintosh floppy drives used a SIWM ("Super Integrated Woz Machine". Honest!)
controller, which write tracks and sectors in a totally non-standard format.
The drives can also read and write in the standard 18 spt format that we all
know, but it's not guaranteed.  In particular, if it's an 800K or 1600K floppy
you are out of luck.

Are high density Macintosh formatted floppies ever anything other than 1600K
format?  I thought not.

Note that if you happen to be running Linux on an Amiga, the Amiga floppy drive
*can* read and write Macintosh 1600K format.  I don't know how.  I don't know
if Amiga Linux retains this feature.
