fvwm task bar problem

On the subject of the fvwm task bar,
is there some really easy way to get either:
1.	the X applications to recognize it's there and
	avoid using that area, or
2.	the task bar to disappear when I'm not using it?

I've done a fair bit of X programming, and I know that
when you size dinwos etc. etc., you go looking around for the
full screen size and use that.
But with the fvwm window manager, somehow applications try to
use the full physical screen, and that means that
windows get cut off below the task bar, which is
a real and continuing nuisance.

I'm sure it's mentioned in a manual somewhere, but
life is short, and manuals are long.
I've `read through' the man fvwm page, and found nothing.

Is it possible that this is an unfixable problem
caused by applications not realising that there's a taskbar there?
