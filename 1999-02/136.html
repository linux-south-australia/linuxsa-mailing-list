I haven't been able to get my DC50 to work under linux.. And I think the
DC25 uses a similar system..

You could probably get the .kdc files off ok, the problem is that the
format that KDC uses, is a super secret proprietary (patented) jpeg like
format.. that Kodak refused to release specs for..

The best we can hope for is a wine that will run the decoder software..
Which I have yet to make happen.

(BTW.. I was able to get a PCMCIA card reader to work under linux, and
using a FLASH card on the DC50.. it was MUCH easier to transfer files out
this way)
