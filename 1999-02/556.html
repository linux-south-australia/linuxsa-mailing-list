Re: GCC Crosscompiling?

Yes, it is possible.
You'll need to download the gcc or egcs sources, plus various assorted
gnu goodies (they're mentioned in the main readme or install in the main
package).
What can be slightly harder, is getting an appropriate library file from
the uni that contains some functions already made in the destination
format, but yet in a gcc-readable form.
Since the uni uses gcc though, you should be able to find the
appropriate bits in their /lib directory.
Basically, you need to read the documentation a whole lot, and note down
all the associated parts required, before you start ./configuring and
compiling the gcc.
