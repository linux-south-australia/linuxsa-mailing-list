Re: Real Player 5.0

On Fri, 12 Feb 1999, Joao Campos wrote:

> Hi.
> 
> Has anyone had success running real player streams with the real player
> plugin?
> I installed Real Player 5.0, following all the instructions. I had
> already Netscape Communicator 4.5.
> 
> Everytime I visit a page that contains RealPlayer content, netscape
> tries to start the player and crashes with "Bus Error".

Netscape is very touchy with plugins. It may be because of using a libc
plugin in a glibc system or vice versa.

Your safest bet is to set it up as an application in the navigator
preferences.

Ie Mimetype:    audio/x-pn-realaudio
   Extensions:  rm,ra,ram
   Application: rvplayer %S
