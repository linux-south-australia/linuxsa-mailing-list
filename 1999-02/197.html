-----BEGIN PGP SIGNED MESSAGE-----

Hi All.

What would be the best way from a shell script to check the status
of a network connection (PPP in this case) so that I could either 
bring it up if needed, or just run the apps that need the connection
if the connection is already up.

I do not want to use DIALD because I do not want dial-on demand all
of the time, just in this case where I run a cron job to toss mail,
and download news to my local spool once a day.

I am using RedHat 5.2 off the Pocketbook CD if that is any help, and
I use ppp-2.3.5-1 off that CD.

Also, thanks to all who replied to my question about grabbing screens.
It turns out that the latest KDE RPMs have a nice utility called

but it wouldn't grab some windows (namely, Staroffice was just NOT
visible in the screenshot).
