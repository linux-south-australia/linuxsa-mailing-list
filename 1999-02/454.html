Radical

Myself and a few other like minded people have been disappointed with
the
amount of GPL software for the Linux ISP community in .au and so
have embarked on a project to create a GTK app capable of Adding
/Deleting
users from both the system and a Radius user file. Also incorported will
be the
facility to run a billing program based on the content of the user file.
This will
probably be done in perl. Allso the ability to email users accounts and
have a
html tool which users may query their account status.

A requirement of the end package will be that it run on Standard Redhat
installation
with qmail and gnome installed.

not wishing to reinvent the wheel, we have looked at other programs
trying to achieve the same
goal and seen much good work by freeside and acua, but no releases and
no real content for the
Australian community, A web site will soon be available at
http://radical.oceania.net
for more information. If you have any tips, or know of any like minded
projects we may
link up to, or even any small apps that may be incorporated, or anything
you would like
to see in such a package, please let us know.


Kind regards

Kevin Waterson
