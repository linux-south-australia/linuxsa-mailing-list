Re: a few things

On Mon, 15 Feb 1999, Daniel Callan wrote:

> I have been fascinated by these messages since I first installed
> kernel 2.0.30 a year or so ago. I have a perfect 'work-around' for
> it but I'm just as curious about what causes them.
> 
> Write the following lines to "/etc/conf.modules" to
> get rid of the messages on boot:
> 
> #########################
> alias net-pf-5 off
> alias char-major-10 off
> alias net-pf-4 off
> #########################
> 
> Anyone got a better solution for this one? :-)

This is not a better solution but it explains what each is. This is taken
from my conf.modules - some other useful one are also included.

alias net-pf-1 off                # unix sockets 
alias net-pf-4 off                # ipx module
alias net-pf-5 off                # appletalk module
alias net-pf-17 af_packet         # raw sockets module
alias ppp-compress-21 bsd_comp    # Enable bsd compression
alias ppp-compress-24 ppp_deflate # protocol number from draft RFC
alias ppp-compress-26 ppp_deflate # real protocol number.
