hi,
make your smb.conf as simple as possible to start with and then make it tighter
as you go. try these settings...
[global]
workgroup = workgroup
security = user
allow hosts = your ip address here
[your share directory name here]
create mode = 0665
path =  /your/share/name
writable = yes
valid users = your login name on the linux box
write list = your login name on the linux box
public = yes

even some of these can be loft off.
restart the smbd
