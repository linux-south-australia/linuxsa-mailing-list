links to linux simulation software

Linux simulation enthusiasts:

In the last few weeks, I've gradually put together a list of
simulation software links at

	http://www.topology.org/sim.html

This is principally focussed on free, linux-capable,
discrete-event simulation libraries, but also includes other kinds
of simulation software (circuit simulation, network emulation etc.),
and some lists of links prepared by other people.

Request:  If you know of simulation software that is suitable for
supporting simulations under linux, and which is not in my list,
could you please let me know?

Currently, I get abnout 4-5 accesses to this links page each day,
and not a single user gives me any feedback.

Cheers,
Alan Kennington.

PS. I've also put together a list of general linux links from
the point of view of an Adelaidean at
http://www.topology.org/linux.html . I've be interested to know
of any serious gaps in this list too.
