Re: Command-line editing

On Mon, 22 Feb 1999, Brenton Pettitt wrote:

> Hi,
> As a very inexperienced new user I have only worked in X windows. Having
> just installed WP8 I decided in X to edit the path statement in
> /etc/profile to avoid having to use the command ./ all the time. I
> obviously messed up and now I can't start X windows.
> 
> I added :. at the end of the path statement but inside "..." (crash!)
> 
> Could someone please help me on the command line statement I need to use
> to edit etc/profile to rectify the error (emacs/vi?) and the proper
> placement of :. in etc/profile.
> 
> I'm using the bash shell with RH 5.2.

My path statement in /etc/profile is:

PATH="/bin:/usr/bin:/usr/X11R6/bin:."

This has the required dirs for startx (/usr/X11R6/bin) and .

To add it try using pico which is a nice little editor 
(CTRL and X exits and saves)

If you don't have that try emacs.
to save/exit = CTRL and X  then CTRL and C
