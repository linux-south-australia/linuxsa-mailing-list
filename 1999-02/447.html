Re: Writing to a Win partition as a normal user.

On Sat, 20 Feb 1999, Adam Hawes wrote:

> Hi.
> 
> How would I give a normal user WRITE access to a Windows95 partition
> mounted at /mnt/win.
> 
> I can write to it as root, but when I try and write to it as myself
> I get errors. I only really want one user to have the write access
> too, but I don't know how to do it. I have tried mounting it with
> the UID=500,GID=500 on the command line but it still defaults to
> root ownership.
> 
Try this:

gid=500,umask=007

This gives root and  anyone in group 500 read/write & execute access (to
change dirs). Everyone else can't accesses it. If you don't want anyone to
be able to execute file then add noexec.
