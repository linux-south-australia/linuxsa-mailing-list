FTP upload: default permissions

Hi all,

Is anyone familiar with where/how you set the default file permission
for a file that is uploaded or dir created via FTP.

When a normal user uploads a file it is being created with a 664 perm
and likewise any dir they create via FTP is created with a 775 perm.
That is, in both cases, the "Group Writable" flag has been set on them
(which isn't too cool IMO). The system standards of 644 and 755 would be
much preferable and I cannot seem to find where/how this can be set.

Any ideas?
