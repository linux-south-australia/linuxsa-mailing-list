Re: how to move kde without breaking it!

Ben Kramer wrote:

 > I asume you have another drive with a good amount of space
 > on it... tar up the tree and place it on a second drive
 > 
 > tar cvf /newdrive/bigtarfile.tar /opt/*
 > 
 > untar on to the second drive and then make a link from /opt
 > to where it now sits on the new drive
 > 
 > cd /newdrive/
 > tar xvf bigtarfile.tar

You can do both of these in one step:

  tar cf - opt | (cd /newdrive; tar xvfp -)

You'll want to run it from the root directory, too.

 > ls -s /opt /newdrive/opt/
 > (check on the ln as I quite often get the file names around
 > the wrong way)

Mnemonic:  When you link you're creating a "virtual copy", so the
args are in the same order as the args for "cp".
