Re: Squid performance question

Joao Campos wrote:

 > Hi. A friend of mine asked me advice on this:
 > Would a Celeron, 64 MB RAM, be apropriate to run squid web/ftp/... proxy
 > server for about 250 people? (over Linux, of course)

I assume you mean "250 simultaneous users", not "250 entries in the
passwd file of which about 15% are using it at any one point in time".

Get more memory (as much as you can afford).  Even if you get enough
for 250 people, it'd be ludicrous to think that the system won't grow
as more people recognise its usefulness, so you should plan for 
expansion.

Other than that it'd be more than adequate (assuming a 100BaseT NIC).
