Re: Changing default font face in emacs

> What do I need to write in my .emacs to change the default font face?

This is what I have.  I believe that the bits at the end set things like bold,
italic and underline but I have never looked into it much.

(font-lock-make-face '(font-lock-comment-face "pale green" nil nil t nil))
(font-lock-make-face '(font-lock-function-name-face "sky Blue" nil t))
(font-lock-make-face '(font-lock-keyword-face "Blue" nil t nil nil))
(font-lock-make-face '(font-lock-string-face "light Grey" nil nil t))
(font-lock-make-face '(font-lock-reference-face "cornflowerBlue" nil t))
(font-lock-make-face '(font-lock-variable-name-face nil))
(font-lock-make-face '(font-lock-number-face "tan" nil))
(font-lock-make-face '(font-lock-type-face nil))
