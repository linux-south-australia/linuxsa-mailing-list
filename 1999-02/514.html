Hi!

I'm wondering if somebody have achieved setting up Matrox Mystique G200 8MB 
under Linux?

I'm running RedHat 5.0, and have upgraded the "XFree86" drivers to version 
3.3.3.1. The monitor that I'm using is a old HITATCHI 21" (unknown 
model/type". The machine is a Compaq Deskpro EP 6400/10 EA5.

I'm able to get Xconfigurator recognizing my video card but it is still 
impossible to get the video resolution correct.

What I've been doing is the following:

I log in as root.

Start Xconfigurator

Choose video card Matrox Mystique G200 8MB

Choose monitor "Custom"

Choose Super VGA, 800x600 @ 56 Hz (Does anybody have an better

Choose 50-90 Hz (Vertical sync)

Choose Don't Probe

Choose 8 meg (The Xconfigurator ask for how much memory I have on my Video 
card.)
