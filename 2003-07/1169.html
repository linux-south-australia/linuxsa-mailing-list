3.9 seems to translate to:

if( program specifies $version ) {
    if (program specifies "any later version") {
        GPLversion >= $version applies //user chooses
    }else{
        GPLversion == $version applies //user has no choice
    }
}else{
    GPLversion == anyVersion applies   //user chooses
}

The file COPYING at the base of the Linux source tree clarifies what
is intended in their case quite clearly, but notes that exceptions
can occur where explicitly stated (eg, the modules you mentioned).

 Authors who include "and any later version" make that choice,
perhaps not realising what the implications (as you stated) could
be, but they clearly have the choice to limit the valid GPL to a
single version.
