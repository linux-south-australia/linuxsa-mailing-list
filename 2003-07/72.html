RE: Samba Squid and another Proxy

>Our network is run off a Samba PDC, is it possible to send the 
>username and 
>password from Samba, to Squid so it can authenticate the 3rd party 
>firewall/proxy?
>
>Instad of us having to enter them in everytime.

In theory it may be possible, but it won't help you with authenticating
your browser session with Squid as the proxy authentication is handled by
Squid,
not Samba. 

How it works is as follows (more or less):

Browser --> Squid --> Auth Source (Samba) --> Allowed, Y/N? --> Squid -->
Browser

What you'd need is some hack for your browser to automatically pass your 
Windows logon username and password to the proxy without physically entering
it yourself.  There was a thread discussing this a couple weeks ago, I'm not
sure if anyone came to a satisfactory conclusion...
