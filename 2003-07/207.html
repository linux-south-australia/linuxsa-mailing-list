> Secondly, my son was playing frozen bubble & it locked up, in 
> full screen mode. What is the ctrl alt del equivalent in 
> Linux? We just turned it off in the end.

If this is in X windows, you can usually Ctl-Alt-Backspace to kill X.
Otherwise, Ctl-Alt-Del usually causes a reboot (see /etc/inittab, IIRC, for
setting this). Also, try Alt-Fn (or Ctl-Alt-Fn if in X) where Fn is one of
the Function keys (F1, F2, F3, etc) to get to a different virtual console
and try logging in and killing the offending process. If that doesn't work,
then it's probably frozen well and good (my test-of-thumb is to see if
caps-lock works -- if it doesn't, then I know it's pretty much locked). You
could look into the magic sys-req key if you found this happenning too
often...
