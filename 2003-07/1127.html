Re: Fwd: Re: XF86config mess

First make a backup of the file (eg cp /etc/X11/XF86Config
/etc/X11/XF86Config.bak)

I think RH has some utility called Xconfigurator or something, you could
try that.

Otherwise, you could also try deleting the config file, and creating a
new one fromscratch, using xf86config (having made the backup first
ofcourse). After the new one is created, you can start tweaking it, and
maybe even find out what caused the problem to begin with.

Assuming that it is your XF86Config file that is at fault though....

Goodluck,
Daniel.
