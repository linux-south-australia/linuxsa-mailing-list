bsd- easier than you think

On Wed, Jul 23, 2003 at 10:15:55AM +0930, David Lloyd wrote:
> 
> Its installer is a little too swiss knife like for one thing. It is a good
> system though...but is there a nice, GUI interface to fiddle with the
> ports collection?

Not sure, I have never found the need to use one.
Want to find say lynx in the ports: whereis lynx
Want to install it: cd /usr/ports/www/lynx; make install clean
Want to remove it: cd /usr/ports/www/lynx; make deinstall
Want to make it even easier:
Install portupgrade (from the ports) and then portinstall <app name>
To upgrade portupgrade <port name>

Want to keep the ports tree updated, run cvsup -g ports-supfile.
Want to see which installed ports have newer versions in the tree:
pkg_version -v

You can use a browser to look around the ports tree, or you can do
searches with: cd /usr/ports; make search key=<search_keyword>

If ports aren't your thing, you can always use packages:
pkg_add -r lynx
pkg_delete -x lynx

Copy a few of these commands into a text file so you don't forget them
and take a look around the man pages. I am sure there are better ways of
doing everything above, but this all works well enough for me.

The real fun is when you use cvsup to upgrade your operating system and
it only has to download cvs patches instead of complete binaries.
