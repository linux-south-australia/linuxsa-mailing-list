Hi Shane..

I can't say for certain, by try running the twinstall.sh script included
with tripwire.  You might also want to edit this script and change the
parameter CLOBBER="false" to true, so it will overwrite the site keys. 
This should completely reset your tripwire install and re-prompt you for
new passkeys.
