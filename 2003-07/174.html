Re: data - does network card record it all?

jonathan soong wrote:

> The line reads:
>    RX bytes:1509937488 (1439.9 Mb)  TX bytes:3869741880 (3690.4 Mb)
>
> What i was wondering is if this value is the correct value? (the 
> machines have not been reboooted).


interface.h uses 'unsigned long' to define the byte counter values for 
ifconfig.

On an ordinary x86 system, this means that the maximum value it can 
store before looping back to zero is 2^32-1. This is 4,294,967,295 
bytes, or 4.3 gigabytes, so your counters have probably looped more than 
once to arrive at your 12G usage.
