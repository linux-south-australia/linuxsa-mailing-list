It's called Konqueror. Both Safari and Konqueror use the same HTML rendering 
engine, the difference between them is just the user interfaces. There are 
still a lot of fixes that Apple have made to the HTML engine that haven't 
made it back to KDE, but all those should be merged by the time KDE 3.2 is 
released.

As far as Windows ports goes, I read that the GPL version of QT is being 
ported to Windows and the KDE libraries are also being ported. Not sure how 
much interest there is in that so it may take some time or get abandoned, who 
knows.
