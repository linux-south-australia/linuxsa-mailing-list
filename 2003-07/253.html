Have you tried using PPPoE over a multipoint wireless lan? I've seen a 
few people try it out and it hasn't been something I would hail as a 
total success. Dropping a PPPoE session will cancel any connection to 
the network on the other side of the concentrator (presuming that you're 
terminating said session on the same host you're working on), whereas 
plain wireless ethernet + ip is a bit more forgiving should you suffer a 
momentary loss of connectivity.
