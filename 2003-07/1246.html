[OT] FastTrak 100 TX2 (NON-RAID) Booting?

Hiya,

Anybody here used a Promise FastTrak 100 TX2 card at all?  I had this running
on a system doing RAID1 on two 80gig hard drives on Linux - problem is, the
machine would randomly kernel halt - apparently the chipset isn't well supported.

I've swapped this over with software raid, which is working fine, but it seems
the card won't let you boot off a hard drive unless there's a RAID Array
defined.  I've tried it in 3 different machines, all with the same result.

I've got other ATA cards here that will boot, but they're not supported under
Linux at all, so the kernel dies when it goes to mount the root filesystem.

I've upgraded the firmware on the card in hope that it'd let me boot off it...
but to no avail.

Any ideas?
