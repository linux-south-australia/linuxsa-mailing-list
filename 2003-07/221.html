Does it only happen for PDF's? If so it probably isn't a problem with 
Moz, but maybe the helper app or plugin it's using to display the file.

If it isn't already enabled, go to <about:config> and set 
"plugin.expose_full_path" to true. This causes <about:plugins> to 
display the full path to the plugin file, which is useful.

Go to <about:plugins>, if there is one listed to handle the 
"application/pdf" MIME content type then move the plugin elsewhere, 
restart Moz, and see if it is still a problem. If so, go to your Moz 
prefs and see if a helper app is assigned to PDFs, try changing it to 
"Always save".

If all that fails, b.m.o and the n.p.m news groups are your friend.
