Dynamic DNS, DHCP & DNS Search fields

Hi All,
    Just wondering if anyone has a solution to this problem.

I use isc-dhcpd to issue leases to various clients. The client passes 
back their name. isc-dhcpd passes this name and the domain to dns 
dynamically updating it.
The problem I have is I want to add an additional search field to 
clients so typing: ssh fred would not only lookup fred.domain.com but 
also fred.someohterdomain.com.
If I specify multiple domains in isc-dhcpd.conf the client is able to 
search both domains however, dns then refuses to update as it gets 
request for things like:
