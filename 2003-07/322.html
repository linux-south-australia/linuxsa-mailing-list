I can say that Drupal is very nice from a programmers point of view, and 
i've been running www.dirtbike.ws on it for quite some time now.

Is quite easy to administer once you get your head around how it works 
with the module systems, taxonomy etc.

The contributions to drupal are available via CVS. There are about 100 
contributed extra modules (including using htmlarea for wysiwyg html 
submission), 20+ extra themes, and it can be very .css ified.
