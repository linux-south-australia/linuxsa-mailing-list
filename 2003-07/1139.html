> I have a list of things I want to do in FreeBSD:

> 1. Run startx automatically before I log in
Why, just run xdm
change:
ttyv8>."/usr/X11R6/bin/xdm -nodaemon">.xterm>..off secure
to
ttyv8>."/usr/X11R6/bin/xdm -nodaemon">.xterm>..on secure

in /etc/ttys

> 2. Upgrade to kde3

cd /usr/ports/x11/kde3;make install clean

> 3. Add Reiser support
> 4. Get the lpd to start automatically
you say later that you want cups, so bsd printing or cups ?

> 5. "Port" over my Netwhere IPX client
> 6. Run a Samba client

> 7. Install Winex
> 8. Install Cross Over Office
> 9. Figure out what's wrong with my screen colors - they are slightly off

.bashrc
if you are using tcsh, then you know how to alter this
> 10. Set up a network printer

but it was a no brainer

> 11. Set up cups & autostart
> 12. Learn how to easily upgrade without having to download a CD

> 13. Replace vi with emacs for an editor - I am a horrible typist.
