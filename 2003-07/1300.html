 SF.NET Project of the Month: Tiki
 ---------------------------------------------
 Keeping a busy web site up-to-date with fresh, engaging
 content is hard work. There are timelines, multiple
 contributors, editing of content, and the graphical layout
 of the site. Content Management System (CMS) is a class of
 software that aims to make all these chores easy.

 SourceForge.net's July project of the month, Tiki, is a
 project that executes on this promise and sports the latest
 content management features including: Wikis, Forums,
 Article/Submissions, Journals, templates and 370 other
 features. Tiki has been on SourceForge.net since October,
 2002 and currently enjoys an activity ranking in the top ten
 of our 65,000 projects. The code is licensed under the LGPL
 and will run on any web server supporting PHP. The SF.net
 team is proud to make Tiki project of the month.

 Project page: http://sourceforge.net/projects/tikiwiki/
 Home page:  http://tikiwiki.sourceforge.net
