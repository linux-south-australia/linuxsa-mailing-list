Printing to a Mac, from Linux, across the net..

Hi -

I'm trying to print from a linux box to a printer attached to an old
Mac, across the net. It looks like this:

[linux box] --adsl--> [internet] --adsl--> [MacOS 9 box] --cable-->
[printer]

Much googling has determined that a package called 'netatalk' provides a
utility called 'pap' to do this. But Appletalk is not TCP/IP, and it
looks like I'll need something called 'Appleshare IP' to encapsulate the
appletalk packets in TCP/IP and allow them to be routed over the ADSL
links. Appleshare IP is commercial software that I don't have.

Is there a better way? I considered writing a simple listener to run on
the mac that would receive print data from the linux box, but I don't
know anything about mac coding, and it seems like a bit of a hack.

Gwyn.
