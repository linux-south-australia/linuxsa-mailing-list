PROBLEM

If you use sylpheed-claws (development version) and then switch back to
the normal version the fonts mysteriously start to play up. You'll know
it when you see it.

SOLUTION

Find your:

 $HOME/.sylpheed/sylpheedrc

...file.

Back it up.

Edit it and DELETE any line that has the sequence of letters "font" in
it.

Restart sylpheed.

OUTCOME

Your fonts end up looking sensible as opposed to plain annoying.
