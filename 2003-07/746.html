Re: unwanted pop ups in windows

Whilst I have to agree that Safari is a great browser, I still think it
has a little way to go before it comes up to the level of some other
browsers...

There's still quite a few little glitches, even in Safari 1.0

I've noticed several layout issues that are apparent in Safari but not in
other browsers on the MAC such as IE and Mozilla.

But considering that Safari really hasnt been around anywhere near as long
as some of these others, Apple have done a really great job with it so
far.

I also love the fact that they include a very easy to use Bug Report
system so that one can report any bugs or layout issues to Apple.

It seems that Apple is doing such a good job with it that Microsoft has
decided to stop further development of Internet Explorer for MAC,
admitting that Safari is a far better browser, or something akin to that.

p.s. I use both Mac's and PC's, when in Windows I choose IE, *nix I choose
Mozilla, and OS X, I choose Safari.

It'd be great one day to see a port of Safari for Linux/FreeBSD and also
for Windows. If there was a port of Safari for Windows, I'd definitely use
it.
