Using ssh through Http tunnel

Hey all,

  There has been a couple of e-mails lately and i just got this working so 
i though i would share. We only have http access at work and as such i 
have never been able to use SSH. I have tried a few times and never 
suceeded as all traffic has to go through the proxy server and this 
filters out all traffic that is not http. I finally got it working using 
http tunnel from http://www.nocrew.org/software/httptunnel.html. I ran the 
command 
hts -F 8080 servername:22 
from the server box that says listen on port 8080 and forward to port 22. 
Then i ran the command
htc -P proxyserver:port -F 22 servername:8080
This told the client to listen on port 22 and send all trafficvia the 
proxy server to the server on port 8080. I could then use Putty to ssh to 
localhost:22 and i can ssh home.

I hope this helps some people.
