SA govt's main website runs on open source software

By Sam Varghese
Sydney Morning Herald
July 17 2003

While the South Australian Minister for Administrative Services, Jay
Weatherill, feels that "generally open source software is not yet seen
by the marketplace to be suitable for fundamental business functions",
his own government's main website is quietly humming along on Linux
and serving pages using the Apache web server.

For the uninitiated, both Linux and Apache are as open source as they
come.

The website, South Australia Central, desribes itself as being the
first point of call for worldwide access to online information and
services in South Australia.

The site has been running since 1997 and underwent a makeover in
2000. It is "maintained using Dreamweaver. The underlying technology
includes: IBM DB2, IBM Websphere and Java Server Pages."
