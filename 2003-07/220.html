message from bigpond...

> Received: from My Computer ([144.135.24.69]) by 
> mta02bw.email.bigpond.com  (iPlanet Messaging Server 5.2

> ilox@airnet.com.au; Mon, 07 Jul 2003 12:31:23 +1000 (EST)

The above line (it's all one line) was added by a bigpond server when it
recieved the message from another bigpond server (which appears to be
calling itself My Computer, which seems somewhat odd).

> Received: from cpe-144-137-199-162.sa.bigpond.net.au 
> ([144.137.199.162])  by bwmam01bpa.bigpond.com(MAM REL_3_3_2c 
> 8/11184621); Mon,  07 Jul 2003 12:31:02 +0000

Here's where we see where the message actually came from.
cpe-144-137-199-162.sa.bigpond.net.au is probably an SA bigpond dialup
account. At 12:31:02 +0000 (which is the time that the server recorded -- it
appears to be running with GMT time set to local time, which indicates to me
that it's probably an MS machine), this dialup account sent the mail...
So... if someone was connected to bigpond at that time, and is subscribed to
this list, and they can check to see what their IP address was at that point
in time, then they may discover that it was actually them...

> Date: Mon, 07 Jul 2003 12:31:00 +1000 (EST)
> Date-warning: Date header was inserted by mta02bw.email.bigpond.com
