RE: How to get current Apache configuration options

This is probably too little too late, but there is something else that
might help you here.

Enable the server-info handler in the Apache config, with the following
lines:
	LoadModule info_module modules/mod_info.so
	<Location /server-info>
	    SetHandler server-info
	    Order deny,allow
	    Deny from all
	    Allow from #your_subnet_or_"all"#
	</Location>

Then point your browser to http://apache_host/server-info.
