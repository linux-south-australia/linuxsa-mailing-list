Re: Mounting/unmounting query

"Paul Malcolm Bailey" <pmbailey@senet.com.au> wrote:

> I booted up my Linux box recently, and amongst the list of messages that
> usually come up was one that read something like:
> 
> WARNING:  check time reached, running e2fsck is recommended

And goes on to describe the classic catch 22.

If you use the -F option to shutdown, it touches a flag in the root filesystem
that the boot scripts check for, and force a filesystem check on ReBoot[tm].

"shutdown -Fr now" can be your friend (unless people are still logged in :-).
