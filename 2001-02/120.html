Re: ISDN vs ADSL

>so does ADSL support static IP or not? i have heard its the hardware that
>telstra give to customers, and have heard its adsl that doesnt support it.
so
>which is it?

Most IP-enabled devices support a static IP, as it's definately easier (from
a programmer's point of view) setting an IP address for the network
interface from a static config than it is to use an allocation system (ie:
DHCP).

Even if for some strange reason they require an automatic allocation, the
system doing the allocation can still set the IP staticly, whether it be
determined by username/password, caller-id or mac-address.

Telstra have chosen to not allow static IP addresses with their ADSL
service, presumably because they needed something to differentiate the
~$100/month service from their $1000+/month services that are rated at
similar speeds.
