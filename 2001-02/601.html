First of all turn off HTML when posting here.

Problem which you expiriencing easy enough: you put your kernel too far away
from disk beginning. It means that you windows have very big partition and
there no chance to boot your kernel. Only solution is to move your existing
Windows partition back slightly (for 1-10-100 mb) and then add linux boot
partition there...

Also NT boot loader will not install Linux anyway - it can only boot it...
:) And it's not good idea to do so... Much easier to boot NT through LILO...
