RE: Remote access - Accessing a Private network.

Thursday, February 08, 2001, 3:54:46 PM, Adam wrote:

> Hi,
> i'll get straight to the point :)
> I wanna be able to dial into my linux box - fire up ppp (this is already
> configured) and have access to my windows boxes which are connected to this
> linux box.

> Firstly, is this possible.

Yes.
But the question lies, what sort of access do you want to your windows
boxen?
If you only want filesystem access, smbmount them on the Linux box. If
you  want  to remote control them, run WinVNC on the Windows boxen and
port forward the VNC ports through the linux box to the Windows boxen.

> Secondly, is this called Virtual Private Networking.

Bit   of overkill really, unless you need secure (encrypted) access to
the Windows machine(s).
