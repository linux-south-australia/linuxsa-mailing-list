Hi.


I've been tinkering with an email scanning package called inflex. 

It's not too bad, you can integrate it withvirus scanning software, and it
seems easier to implement than some of the other packages. Files can be
excluded based on name and type, so it is suitable for simple requirements as
it is. The blocked files are kept on the system, so if you exclude an mp3 for
the boss (!), you can easily dig it out for him...
