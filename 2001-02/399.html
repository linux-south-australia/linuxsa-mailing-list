ncurses is, well, ncurses. It's an abstract window GUI toolkit that runs
on terminals. Things like the RedHat text based installer use it; ntsysv
uses it and linuxconf on a kernel use it.

I can't recall it's main page but I think that is either saying:

1) You don't have ncurses installed

In which case, install it (using RPM, apt-get, dpkg or from a tar ball -
if you don't have your distro handy there's always
http://freshmeat.net/)

2) You have an older version installed

In which case, update it (with the same caveats as above).
