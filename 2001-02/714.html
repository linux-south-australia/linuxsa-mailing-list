Try using uneeda.telstra.net or 139.130.4.4 this is the one that TI 
recommends BPD customers use for name resolutions. I think that the others 
are more like to server domains, ie. ns.telstra.net tends to be used for all 
reverse delegations of TI ip addresses. ns1.telstra.net tends to be the 
secondary used by a lot of BPD customers. ns0.telstra.net is the primary 
assigned to BPD customers that choose to use it through custdata. These may 
not be the true uses of these nameservers it is only what I have 'seen' 
through my own experiences.
