Re: Login Problem

> "bash: fork: Resource temporarily unavailable"

This means the system is unable to create ("fork") a new process.
If it's not a transitory problem then the chances are the user's start
script (.profile, .bash_login or .bash_profile) is invoking a program
that repeatedly creates processes until the system runs out of process
slots, or memory, or some other resource.  Here's a hint: Look for the
& character.
