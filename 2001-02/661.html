Re: Modem Speeds

> I just love sweeping statements, such as "Every modem ... will work with
> ATW command", especially when it's followed with a justification like,
> "because it's defined in Hayes extended AT command set."  Let me just
> check the manual for a genuine early model Hayes.  Nope, doesn't support
> extended AT commands.  Hey, could it be that extended commands aren't
> part of the basic set that all Hayes compatible modems support?

I checked few different modern modems before posting - all of them claim
that they are Hayes compatible and all of them understand ATW command (I
didn't compared whole extended set). So statistically "every" modem will do
so. It really depends on "What can be called Hayes compatible" -
compatibility with basic command set or compatibility with extended command
set too.

I don't know what Hayes model you're looking but as I remember my old Hayes
SmartModem 1200 (alluminium case - good style :) of 1989 this command was
supported.
