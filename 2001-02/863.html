Depending on how your PHP is compiled and installed, you may have a
'php' binary available so that scripts can be run from the command line:

[justin@hostname justin]$ php -q
<?
echo "Hello World!\n";
?>[CTRL+D]
Hello World!

More generally useful is the fact you can put together a text file like
so:

#!/usr/bin/php -q
<?
  echo "Hello world!\n";
?>

chmod it, and run it from the shell.

It is available in my RedHat binary RPM by default, my recollection is
that a stock ./configure ; make ; make install from source does not
compile this program. I've never really needed it so I haven't tried
to get it going.
