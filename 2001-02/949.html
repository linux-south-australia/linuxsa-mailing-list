Re: Apache 1.3.14 httpd.conf problems

If you used an rpm, I would check and see if its using one of
/etc/httpd/conf/httpd.conf
or
/usr/local/apache/conf/httpd.conf

/etc/httpd/ is usually the RPM install location.  It seems strange that it
would overwrite your old settings, check and see if you have a
httpd.conf.rpmsave file.
