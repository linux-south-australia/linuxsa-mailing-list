Samba & Group Policies

Hi Everyone,

I have just recently installed a new network of 30 pc's, running windows 98,
connecting to a linux box running samba and a whole heep of internet
services. My one problem which I have just come across is none of the Win98
boxes seem to be following any of the group policies I have set. I have
created a config.pol file, placed it in /home/netlogon and all the machine
policies have taken effect but nothing else. I even created a user policy
and nothing happened.

The procedure I followed to setup policies were :

* Create Unix Groups, add Users to groups. (this is working because the
logon scripts are reliant on groups and these work perfect)
* Installed Policy editor, Group policy support from /tools directory on the
Win98 cdrom.
* Enabled User level access control (also tried shared level access control)
* Created policy file with all appropriate groups and machine policy and
placed config.pol in /home/netlogon

After all this each PC followed the default computer policy, however if I
update the default computer settings in config.pol none of the win98 pc's
change.

Has anyone had any experiences in this field or knows what step(s) I have
missed or forgotten ?

Any help greatly appreciated.
