Re: Modem Speeds

> There does not seem to be a standard for the command used
> to select which of these two speeds is reported, so you will have to
> read your modem manual to find the AT command to use.

Every modem which claims that it's Hayes compatible (most of them do so)
will work with ATW command because it defined in Hayes extended AT command
set - http://www.modem.com/general/extendat.html . And because Hayes command
set is standard de-facto this command will probably work for you.
