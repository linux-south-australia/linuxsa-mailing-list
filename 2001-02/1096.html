Re: Forgotten command/package

Dave Lovett wrote:

> I downloaded and used a package last year but I've forgotten the name of it.
> It allows you to specify an IP address and it guesses the OS shows the ports
> used and level of security. I think it begins with the letter 'o' but I'm
> not sure. Any help greatly appreciated.

One such package is 'nmap'. While it is cool to be able to see what
ports are open and guess the type of OS from remote, scanning someone's
computer on the Internet is generally considered a hostile action and
will almost certainly be interpreted as a precursor to a hack attempt.
Use it with caution.
