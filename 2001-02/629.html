Re: Modem Speeds

Hi Mark et. al.,
	I think you'll find that M1 is actually to tell the modem to use the
speaker for output noise until a connection is made, then turn the
speaker off.  This is commonly used in conjunction with an L setting,
eg. ATM1L1 sets the modem to use the speaker until a connection is
established, and to only use the speaker in low volume.  The setting you
need to change is probably W0 or W1.  ATW1 will cause the CARRIER
response to return the line speed, the PROTOCOL message to return the
error correction mode, and the CONNECT message to return the terminal
speed.  ATW2 simply sets the CONNECT message to report the line speed. 
ATW0 will simply cause the CONNECT message to report the terminal
speed.  I believe ATW2 is the default initialisation setting for
many/most modems.
regards,
Don.
