Matthew!

> Does anyone have any suggestions as to how I can find the solution to
> this problem?

Stop the compression on the AMANDA side by adjusting which tapetype you
use. In the default AMANDA configuration these are the "nocomp" style
dumptypes. Furthermore, AMANDA will compress on the client side by
default - are your versions of gzip exactly the same from where you're
restoring? You might have stumbled across a gzip format incompatibility
issue...
