Re: newbie kernel installation question

> 1.    unpackage the following into /usr/src directory using following
> command
>         # bzip2 -cd linux-2.4.0-test11.tar.bz2 tar xvf -
>
> when I do this I get screen output, but when I check the usr/src/linux
> directory it seems to be linked to a directory /usr/src/linux 2.2.14-5 and
> when I look at the changes documentation it refers to kernel 2.2.
> What am I doing wrong here??

It must look as
# bzip2 -cd linux-2.4.0-test11.tar.bz2 | tar xvf -

and you will get linux-2.4.0 in your current directory.

> 2.    Using RPM to install the package.  I have the following version of
> RPM -         3.0.4.  Using the command:
>         rpm -u kernel-2.4.0.26.i386.rpm
> it then comes up with an error something like older versions <=3 not
> supported

Because it is using another type of data (9). You need at least RPM 3.0.5.
Probably you running RH 6.2 then go for new RPM at ftp://updates.redhat.com
