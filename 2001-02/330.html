How did you run lilo?

You need lilo -r /mnt/mnt-point-for-hdb ...

You could do the following if you have an install CD, especially if it is a
RH6.2 or thereabouts:

1. boot the CD

2. Enter to the boot prompt:

   linux initrd= single root=/dev/hda1

3. When you get into single user, run lilo -v
