RE: CD Burner

Hi Martin,

The base unix packages you use to write to your cd burner is cdrecord. This
is a command line utility which you may have to use in conjunction with
makeisofs. If you have an IDE writer goto linux.nf and read the step by step
guide for getting ide writers to work under linux.

If your not a fan of the CLI, you can download X applications such as
Xcdroast, very similar to adaptec easy BSOD Creator (cd creator ;-), which
is just an graphic front pretty much to all the command line writing tools,
however a few more features implemented such as writing on the fly, and last
time i checked writing mp3's to cd audio was being implemeted too.

Hope this helps,

Jeremy
