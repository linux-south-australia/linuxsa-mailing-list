RE: lilo and redhat 7

Fixed it! Thanks to all that replied. The problem was that red hat was using
linear mode for lilo when lba32 was needed. Red Hat anaconda must
  not detect the fact that lilo couldnt install the MBR due to the 1024
cylinder problem. I used my boot disk, edited lilo.conf and replaced linear
  with lba32, ran lilo and it worked!
