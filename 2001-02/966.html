Re: Apache 1.3.14 httpd.conf problems

Have you checked your /var/log/messages file
for error messages?
I think that's where the go to.
Or.... the error log in your httpd's log directory.

E.g. here's a quote from one of my web server error_log files:

============================================================
[Sun Apr 23 21:14:16 2000] [notice] SIGHUP received.  Attempting to restart 
Syntax error on line 604 of /usr/local/apache/conf/httpd.conf:
Invalid command 'RefererIgnore', perhaps mis-spelled or defined by a module not included in the server configuration 
============================================================

With that, you should be able to debug the httpd.conf file
line by line.

I don't know how you would send a file via hotmail,
apart from using the copy/paste method.
And I don't receive mails from hotmail any more - 
because more than 50% of hotmail mails are spam,
to me anyway.

Anyway, debugging other people's httpd.conf files is not
something one ordinarily does for pleasure at 1:30 a.m.
on Sunday morning!
