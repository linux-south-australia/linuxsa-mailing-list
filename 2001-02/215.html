Assuming you can't use static LMHOSTS files and if your client can log
into their NT PDC, try setting up semi-dynamic lmhosts files on each
workstation (not the ideal way) using the lmhosts file and nbtstat -R.

If the machine dialling the Linux box is a Windows machine, WINS
*should* work over the modem link. If the link is being pushed hard, you
may experience timeouts for WINS lookups.
