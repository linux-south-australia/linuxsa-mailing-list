Re: Mylex AcceleRaid 170 & RH7.0

At 05:52 PM 2/3/01 +1030, Steve Sloan wrote:
>Hi all
>
>I am trying to build a Raid 5 box using a Mylex AcceleRaid 170 & RH7.0 and
>it would appear that RH7.0 doesn't support this card but there are drivers
>available from http://www.dandelion.com/Linux/DAC960.html. I think the
>DAC960 driver in RH7.0 isn't quite up to date.
>
>This is great except it involves recompiling the kernel, which brings me to
>my problem of how do I recompile the kernel if I can't install the operating
>system in the first place.
>
>Is it possible to setup a system on another drive, upgrade the kernel and
>install it from there.

This must be a FAQ question ...

Here is the way to do this ...

Copy the driver file to a floppy

Boot the installation and before it gets to the section about the drives
available and partitioning etc, swap to console 2 (AltF2) and then mount
the floppy (which may take a bit of effort :-) and then do:

  insmod /tmp/floppy/driver.o
