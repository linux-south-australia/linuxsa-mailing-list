Re: The "From" Escape

"David Lloyd" <lloy0076@rebel.net.au> wrote:

> Does anyone know where the "From" escape gets added? Is it at:
> 
> * MUA or MTA level?

It's for the the "benefit" of mailbox-style message stores, so the MTA/MDA
adds it (typically, MDA), and the MUA maintains it.

> Better still, is it really that useful?

It's used to distinguish an accidental From from a real From, and therefore
mark message boundaries.  If you don't use it, and I stick a 
From 
just there, and it wasn't escaped, you wouldn't see the remainder of this
message (probably a good thing), because your MUA would think it was
the start of the next message.  If you use some other message store format
(eg, ./Maildir, IMAP), then it's a non-issue.  It's a "typical" in-band signalling
kludge.
