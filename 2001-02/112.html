I've used e-Smith on a P133 (32MB RAM/10GB/3c509) at work, and it worked
well.

It was for a client, and it seems quite reliable. Probably the biggest
plus for e-Smith is the web based management interface. It's quite
intuitive, and allows easy account creation etc.

Does email, smb, pop-3, imap, www/php. Based on RedHat.

One thing that I noticed is that on a `slower' box it seemed to take a
while to load the web management pages. I assume that this is because of
the perl that it runs to generate output...
