with a NAT , generally u should have an internal address for other 
machines to use and u can also supply DHCP service from the NAT.


All you need to to is set up the wondows box under the network settings 
to 'point' to the NAT, including DHCP if needed (WINS should not be 
required). the NAT should have details on its address etc, then you 
should be able to use the win98 box to surf.  Test the new config by 
opening a command session on w98 and 'ping' your NAT address to see it 
the TCP/IP stack is working and talking to you box (assuming the link 
layer is ok)
