Re: Mailing list scripts

Andrew,

> Does anyone know of a good mailing list script I could use to give people
> the ability to 
> subscribe/unsubscribe through the web browser? Any good links to scripts in
> general?

I use Majordomo, which seems to be extremely stable and easy to use. I'm
not sure, but maybe even this list is running Majordomo. It's quite a
popular package. If you use Webmin you can make use of the web-based
configuration interface.

All subscribing/unsubscribing etc. is done through an email to the
majordomo program. ie, to subscribe to the list linuxsa, you might send
an email to:

linuxsa-request@linuxsa.org.au with "subscribe" in the body.

Majordomo's website is http://www.greatcircle.com/majordomo/

If you're a QMail fan, I beleive that EZMLM (an program written by Dan
Bernstein, the author of QMail) does mailing list work as well. I'm not
sure how it compares to Majordomo features-wise or the reliability of
it.

The EZMLM website is: http://cr.yp.to/ezmlm.html/
