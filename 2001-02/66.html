Re: Adaptec 2940 Drivers for linux

* Prashanth N V (nv@sasken.com) wrote:
> Hi,
> 
> Can any one let me know the source for  Adaptec 2940 SCSI
> Controller Drivers for linux and also i'm installing a Segate 4MM Dat
> Drive so pls help me with this.

It seems to be included in linux 2.4.1. Look at:

SCSI Support->Low Level Drivers->Adaptec AIC7xxx support

The help for that option contains:

: This is support for the various aic7xxx based Adaptec SCSI
: controllers. These include the 274x EISA cards; 284x VLB cards;
: 902, 2910, 293x, 294x, 394x, 3985 and several other PCI and
: motherboard based SCSI controllers from Adaptec.

I'm not sure about the DAT drive though... it should just "work" if 
SCSI tape support is enabled.
