I've got one of these babies working nicely using 2.4.1 and the standard ac'97 drivers.
Make sure the soundcard isn't setup as a Soundblaster in the BIOS, and then use the kernel drivers for it. Should work fine. If you had problems with the drivers in 2.2.18 you should probably have a look at a newer kernel.
This codec is relatively new.
