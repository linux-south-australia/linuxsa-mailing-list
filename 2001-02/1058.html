Star Office 5.2 and it's network install.

Greetings,

[I haven't seem my original mail on the list, but here's a summary..]

I send a query this morning about a problem that we were having
getting Star Office to install with the /net option...

We were getting an error
"Failed to load necessary components"
(after installing for a user by running
/opt/office52/programs/install)

The solution (after browing SunSolve etc.) is that there is a file
~/.user52.rdb
which, if corrupted (through multiple installs) stuffs everything up.

The solution is to delete this file, and restart Star Office.
