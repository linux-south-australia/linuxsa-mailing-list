Re: keystroke monitor ?

dantree@adam.com.au wrote:
> I was told that there is a file with in Linux (REDHAT v 6.2) that records
> all keystrokes in a session ?
> Any info on it ?

Not exactly.  Some shells (bash, etc) have history files which enable you to use
the same commands from session to session without needing to type them in
again.  This is especially useful if you accidentally log-out and forgot the
exact command you typed 30 seconds before you did so.  As far as it being a
keystroke logger, I believe this is an error, although I'm not infallible and
someone else may have evidence to the contrary.

> If so how would it be able to be edited ?
> Is it available with in other distro's , including Mandrake v 6.0 ?

for bash shells, the file you need is in the user's home directory and called
.bash_history
