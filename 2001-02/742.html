Furthermore, if you want to test whether a connection is still open
(presumably from the machine that is supposed to be connected):

1) Why web suck a web page?

You'd save more bandwidth by using ping or ifconfig.

#! /bin/bash

# I am going to check this host
$HOST=127.0.0.1

# I am going to send 2 pings to the host, be quiet about it and discard
the output
ping -c 2 -q ${HOST} &> /dev/null

# I am going to save the status (not really necessary, but it will help
if I want
# to get more complex later
$STATUS=$?

# ping returns a 0 if not OK, so I'll do something if it does NOT
if [ ${STATUS} -ne 0 ]; then
 echo "I am going to do something in this if block!"
fi
