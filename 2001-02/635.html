Music CD problem

Hi folks,

I have this weird CD problem that I can't put my finger
on, so hopefully someone has stumbled over this too
and can help....

I have an Acer 32x8x4 CD/RW ATAPI drive. I use SCSI
emulation so that I can use cdrecord and cdparanoia,
both of which work well with the CD/RW drive.  I can
also mount data CDs just fine, and have installed my
Mandrake distro from it.

Problem is that I can't play music CDs with it. Every
CD player app that I have found basically cannot access
the CD device. /dev/cdrom is a symbolic link to 
/dev/scd0 (the SCSI device for the CD), and permissions
are correct on /dev/scd0. I even did chmod 777 on
/dev/scd0 just to make sure. As far as I can tell, the
kernel is configured ok as well.

Any ideas? I configured the SCSI emulation as per the
CD writing how to guide. I'm currently using 2.4.2 pre 3
kernel, but this was happening using the bog standard
Mandrake 7.2 kernel as well (2.2.xx).
