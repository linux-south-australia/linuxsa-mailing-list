Comm ports are dafaulted to 9600 and Linux will not change speed
automatically for you. So if you're not used ppp yet (or another program)
which set speed explicitly then your modem must detect port speed.  I had
expirience with GM56flexE-V and this modem cannot detect speeds (dunno what
about another your modems).

> I found some
> serial port reader program and watched the output while I turned the modem
> on and off and some values changed so I'm assuming that its not a hardware
> thing.

That really means that you have good but misconfigured hardware. Try to use
minicom and set different baud rates. Minicom probably good for you because
it using "windows" interface. :) As I remember GM56flexE-V locks on 57600.
