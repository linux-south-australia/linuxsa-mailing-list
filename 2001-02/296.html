Re: open same file at the same time

Prashanth N V wrote:

> Two users are trying to open the file (same file name ) at the same time.
> It displays a error for the second user as Found a swap file by the name
> "/tmp/test.swp".
> 
> Can you pls let me know you suggestions for the problem.

Yes - please be more explicit in your problem definition.  I'm
inferring that you are attempting to use vi to edit/view two
files simultaneously, as very few other applications that I
know of display "swap file errors when opening files of the
same name" (paraphrased).

If you _are_ using vi, and it _is_ on a linux box, chances are
it's vim[1].  "vi -R filename" opens the file in read only mode,
which doesn't create the swap/lock file (the program "view"
should do this as well, as it's a symlink to vi/vim).
