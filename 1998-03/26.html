First and foremost, I tended to prefer the Slackware release, mainly for the
Ncurses setup and pkgtool utilities, I wasn't very keen on using X for
everything as RedHat seems to prefer.
I would be interested to know if RedHat have remedied this situation in
release 5.

As for Metro-X from what I can work out it's an X-server with a few extra
accelerated features.  I personally am comfortable using the standard server
for my card (Diamond Stealth 64 S3) as it seems to work flawlessly.

Infomagic release a set of CDs every 3-4 months for a cost of about $40-$50
and it generally includes slackwarte, RedHat and the Debian releases.  Plus
a near complete copy of the sunsite online file archives.

I'f I'm wrong on any of the above information I would appreciate correction
as I'm still reasonably new to Linux.
