At last months meeting, Jim (etal) demonstrated a diskless PC booting
Linux via the network. A number of applications for such a box were
raised. One such application was as a telemetry station that could
perform I/O via the parallel and serial ports.

In the Februrary 1998 edition of Dr Dobbs Journal (has "Scripting
Languages" in large print on the cover) there is an article on
building an I/O driver for the parallel port under Linux.

    "Our authors present a general-purpose data-acquisition system for
    Linux that can be connected to the PC's parallel port to record
    eight channels of analog voltage."

Some details are available at:

    http://www.ddj.com/ddj/1998/1998_02/

search for "A DATA ACQUISITION SYSTEM FOR LINUX" - about 1/3 down the
page. Source code is available for downloading.
