Re: Can't mount msdos partition

> 
> I am using vfat 
> I use the command mount /dev/hda1 /mnt
> it tells me that /dev/hda1 is not a block device
> I am running version 3.2.1 but i have 3.3.1 on my dos drive....

Can you do the following

So we can see the exact error message:

        mount -t vfat /dev/hda1 /mnt

So we can see what /dev/hda1 is:        

        ls -al /dev/hda1

So we can just confirm that vfat is working properly (it should
fall back to msdos if that fails):

        insmod vfat;lsmod

And send the output to the list.

The error (on the surface) appears to be a problem with your 
/dev/ directory.  My 'ls -al /dev/hda1' gives me

        brw-rw----   1 root     disk       3,   1 Sep  8  1994 /dev/hda1

Where the b at the start of the line gives me an indication that it
is a block device.
