What sort of CD-ROM is it ?

If it is a standard atapi (IDE) cdrom try the following from a linux prompt

If you do not have a cdrom dir first enter

mkdir /cdrom

then use mount ie.

mount /dev/cdrom /cdrom

and then it should mount any cdrom that is in the drive..

To operate it under DOS you need the device drivers...
