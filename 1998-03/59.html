Re: Fat-32

> Hi all:)
> 
> Got a quick one for you... where can I download a patch that will let me
> mount FAT32 Fartitions in Linux?

http://www-plateau.cs.berkeley.edu/people/chaffee/fat32.html

> I've installed the RedHat 5.0... does that have support by default? If so,
> please tell me how I can do it.  I've tried mount -t vfat /dev/hd?? but it
> told me it couldn't.  Just tell me if I was doing it wrong or not :)

No, the kernel supplied with Red Hat 5.0 does not have support for
FAT32 filesystems.  The FAT32 filesystem code was incorporated into
the standard kernel in version 2.1.60.

At the web page mentioned above there is also a modified version of
FIPS that will work with FAT32 partitions.  Has anyone out there tried
it out?
