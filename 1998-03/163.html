Dont know what you are trying to achieve, but I suspect a few simple
shell scripts wrapped around the 'unzip' command would solve you
problem.

ie -
# list filename.zip
      - unzip -l filename.zip

# view filename.zip file1.txt file2.txt
     - zipfile=$1
     - switch         (bump arguments along)
     - for i in $*
       do
           unzip $zipfile $i
       done > temp_output_file
     - less temp_output_file
     - rm temp_output_file

Is that what you had in mind ?
