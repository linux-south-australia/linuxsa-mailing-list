Helpdesk web tools

Hi...

I'm looking for a simple web-based helpdesk tracking tool. It doesn't
need to have email interaction -- I'd prefer email notifications, but
web-based updates. I've seen rt (Request Tracker), and I dislike it.
It's too complex for my needs.

All I need are:

* requestor name, email, phone
* urgency
* priority
* due date
* subject/summary
* description
* comments (each comment should get emailed to me and the requestor)

Additionally, it should be trivial for people who have never seen it
before to submit requests, view their own requests, and make comments on
their requests...

I'd also like it so they can't see others' requests, and so I can see
all requests, and sort them by whatever.

Ideally, I should be able to apt-get it :)

Does anything better/simpler than RT exist?
