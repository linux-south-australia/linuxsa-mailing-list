Re: Can it be done?

On Tue, 2003-03-04 at 19:35, terry the wrote:
>     
>    I know it is not practical but is it possible to install xwindows on a 386 DX 40mhz PC with 32 megs of memory and a one gig SCSI drive?

It's not necessarily impractical. If you get a copy of the X-Windows
source, it's possible to build a framebuffer-only X server that is as
small as 700kB. It's possible to have an entire X-Term (plus some extra
stuff that's handy to have) fit in the around 2-4MB of disk. That's
without any X clients (applications), but that's what networks and
servers are for :-)
