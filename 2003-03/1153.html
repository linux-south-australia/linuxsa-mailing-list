For redhat 8's default firewall type into a root prompt the command
lokkit. Choose the customise option and open any ports you need/want.

Or use a firewall script like generated by
http://www.freshmeat.net/projects/quicktables or the script from
http://monmotha.mplug.org (which is what I use). Both of these let you
specify which ports you want to open and those you want closed, but the
quicktables is written especially for those who dont know much about
firewalling. Both will let you setup a working LAN in just minutes.

You can use google to search for "firewall testers" or other keywords to
find free online sites to test the security of your firewalls, but none
of these is probably going to be as effective as a friend with nmap.
