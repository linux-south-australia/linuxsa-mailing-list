RE: Promise PDC 20267 (On-Board) RAID Controllers

I don't know the answer to your questions but the following link has quite a
bit of info on setting things up.

http://www.tldp.org/HOWTO/ATA-RAID-HOWTO/

I have managed to install RH7.3 with a RAID mirror using a board (Gigabyte
GA-8IEXP) with a Promise 20276 chip but the system freezes on bootup,  just
after it has initialised the USB devices. I don't know if the problem is
related to the RAID setup but I have Win2K on another partition which works
with the mirror OK. I can also boot off the Linux CD into rescue mode, load
the RAID drivers and look around on the installed Linux system.

By the way, I did not use the instructions in the above link to set things
up but followed the instructions which came with the drivers I got from the
Promise web site.
