Re: Evil Email/Playing a .wav file when local mail arrives.

On Friday, Mar 7, 2003, at 21:42 Australia/Adelaide, James Leone wrote:

> Ok, mr. desktop is at it again.
>
> Tonight I decided that I wanted to run a cron job every minute to check
> my /var/spool/mail/james file every minute, and if it was larger than 0
> bytes big, have it play a .wav file of Dr. Evil saying "You've got
> fricken mail!"
>
> So I learned Cron finally. (Is there a way to do this without VI?)
>
> At the shell I typed crontab -e         (This got me into vi)
>

Yes, set your EDITOR variable to your favourite text editor before 
running crontab -e.

e.g.

(in a /bin/sh derivative)

EDITOR=`which emacs`
export EDITOR
crontab -e
