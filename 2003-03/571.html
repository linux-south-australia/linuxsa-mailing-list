Re: Samba user management

On Wed, 2003-03-12 at 09:00, Andrew Galdes wrote:
> What GUI are people using to manage user accounts on a Samba server?
> 
> Ones i know of are "command line, SWAT and Webmin". I'm after something that has the ability to restrict the administration to JUST user management, not server management.

Not SWAT. It was hacked up at very short notice and isn't really up to
the job. Webmin is a little better, but I've also heard bad things about
it regarding security.

It also depends a great deal on:

 a) which password backend you're using (smbpasswd, LDAP, TDB)
 b) who the user account administrators are

I know a lot of people that use 'User Manager for Domains' with a great
deal of success. I've never seen it run under WINE, but if you've got a
Windows workstation...

I use gq if it's an LDAP backend or User Manager and sometimes install
'Directory Administrator' for others to use.

Most people find it very advantageous to use the command-line tools, due
to their flexibility and scriptability.

Hope it helps,
Matt
