Re: SMS Gateway

On 21/03/2003 3:01 PM, steve@lansol.com.au wrote:
> Just wondering if there is way to setup an Email to SMS Gateway on a Linux box.
> I need to be able to send messages to Telsta mobiles from a RH8.0 mailserver
> running Postfix

i reckon <http://www.gnokii.org/>; would be the place to start if you 
want to do this yourself. i've never done this myself, but i've heard 
one of the ways you can do it is via gnokii + a prepaid nokia 5110 with 
a datacable, and some sort of script to pipe any mails sent to 
sms@your-mail-server.com.au into gnokii and then into the 5110.[1]

the other way is to pay for credits at blueskyfrog.com - you then get an 
email address of the form yourmobilenumber@blueskyfrog.com that you can 
send mail to and it automagically sends to your mobile. possibly not the 
cheapest way to go, but certainly the easiest to set up.
