apt-cache search <topic>
apt-get install linuxcookbook
>>>> apt-get install debian-reference-en <<<<<
(and debian-reference-common)
apt-get install doc-linux-html 
oh and the Debian Site has the Debian manual which gives 
you a rather daunting (in quantity at least) path to getting
on line.

pppconfig (as root) may be enough to get you connected.
There are a few mpg3 players (apt-cache search mpg3 )
apt-get install  the following:
lynx,
dpkg
galeon(-snapshot)
Will also grab Mozilla

One small note with debian and xservers is that the fonts
are not forced dependencies so make sure you install at
least xfonts-base (and probably  xfonts-100dpi) or you will
be wondering why you can't start your xserver.
(Not forced as they may optionally be served from another
machine).
Jump straight to xserver-xfree86 (XFree864.2xxx).

You can also use task-select for less "fine-grained"
control.
(personally don't like that but many do)
or dselect (which is VERY comprehensive) .
 There are other "interfaces" as well
perhaps aptitude will suit you more than dselect initially
at least.
 

The default Debian Apache is reasonably straight forward but

add php4/perl/( and python) for a little more "fun".
