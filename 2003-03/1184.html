Re: Adaptec 2120S U320 raid controller

On Wed, 26 Mar 2003, Steve Sloan wrote:

> Just wondering anyone has used a 2120S in a Linux box ie. booting off 
> the raid.
> I know the 2100 works OK and I am looking at building up a new box and 
> the 2120 seems to
> be go at the moment

You will need a kernel on the install CD that understands that device. The 
Adaptec I2O driver should do the trick. The same one as the 2100 driver.

If current versions of RedHat do not have support for the 2120, try a 
driver disk and check the Adaptec web site.
