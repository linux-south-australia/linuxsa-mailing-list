Ah,
should have added.

if you have:

@mmc.com.au		mmc

then it's used as a last resort.  So, if anything else matches it then it'll 
use that first.  So:

@mmc.com.au		mmc
foo@mmc.com.au		foo
notfoo@mmc.com.au	error: no foo

Will send email to "foo@mmc.com.au" to localuser foo, bounce notfoo with 
error "nofoo" but everything else to mmc
