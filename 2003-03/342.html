Evil Email/Playing a .wav file when local mail arrives.

Ok, mr. desktop is at it again.

Tonight I decided that I wanted to run a cron job every minute to check
my /var/spool/mail/james file every minute, and if it was larger than 0
bytes big, have it play a .wav file of Dr. Evil saying "You've got
fricken mail!"

So I learned Cron finally. (Is there a way to do this without VI?)

At the shell I typed crontab -e         (This got me into vi)

Once in vi I pressed every key imaginable. But I think pressing "a"
allowed me to see letters on the screen rather than a string of beeps
when I typed.

Then I typed:
1-59 * * * * sh $HOME/mailcheck.sh      (Runs the script mailcheck once
a minute)

Then I figured out that if I pressed the escape key and shift ZZ, it
would save the file and create the cron job.

Whew!

Here is $HOME/mailcheck.sh:

#! /bin/sh
TERM=vt100
if [ "`ls -s /var/spool/mail/james | grep 0`" != "   0 
/var/spool/mail/james" ]
then
play /home/james/Documents/frknmail.wav
fi
exit

When I finished making the script,  I was happy that I actually got cron
to work, and it sounded funny to hear Dr. Evil tell me I had mail.
