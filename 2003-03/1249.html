Problem with first Mandrake/Redhat GUI installs

Hi,

I was wondering if anybody have had this problem or could say what the
cause is.

The problem is a lock up upon booting off ReadHat 7.3, RedHat 8.0,
Mandrake 9.0 and Mandrake 9.1 installation CDs. The user sees the
initial splash screen then if you proceed with a graphical installation
(generally the default), the machine displays a blank screen and
proceeds no further. The Cap and Scroll Lock LEDs on the keyboard then
blink.

It may manifest itself with other distros, but I can say it doesn't
occur with Knoppix 3.1.

I reckon it might be some reason related to the graphics adapters, both
machines that do have this problems use Nvidia Quadro 4/GeForce 4 Ti
chips. Swapping them out with a Quadro 4 NVS/GeForce 4 MX or even a
GeForce 1 makes the problem go away.

On my home machine I drop back to the motherboard's ATI Rage adapter
then after installation configure it for the Quadro 4. It's annoying -
has anyone else had this problem?
