OpenOffice 1.1 beta available

"A beta release of version 1.1 of OpenOffice is available now from 
OpenOffice.org. 

Significant additions in version 1.1 include new formats for importing 
and exporting documents, including the ability to save files in PDF 
(portable document format)... 

OpenOffice also includes enhanced support for XML (Extensible Markup 
Language) and expanded support for devices such as text readers that 
allow the disabled to take advantage of software."

http://zdnet.com.com/2100-1104-994264.html

Download from;
Windows: 
http://public.www.planetmirror.com/pub/openoffice/stable/1.1beta/OOo_1.
1beta_Win32Intel_install.zip

Linux:
http://public.www.planetmirror.com/pub/openoffice/stable/1.1beta/OOo_1.
1beta_LinuxIntel_install.tar.gz

Solaris:
http://public.www.planetmirror.com/pub/openoffice/stable/1.1beta/OOo_1.
1beta_SolarisSparc_install.tar.gz

Note that it is a stable but beta version so you shouldn't install on a 
production machine or over a current OOo installation.
