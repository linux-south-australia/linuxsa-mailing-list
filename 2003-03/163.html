RE: blogging

Luke,

In a nutshell, you're got it.  Typically a blogger uses some software like
MovableType, Blogger, Radio Userland to make the posting easier (typically
browser based).  The software takes care of posting new entries to the home
page, creating/updating archive pages, and generating the RSS feed.  The RSS
feed is an XML version of the n most recent posts which is then able to be
"read" by various News Aggregators. 

Most bloggers are of the "link and comment" variety (take a link to
something and add your own opinion).  Some keep a diary this way, while
others use it as a way of publishing original content.  I tend to use my
blog (currently http://humboldt.kendle.com.au/weblog/) to collect links to
useful articles I've found elsewhere, and to record bits of wisdom I learnt
the hard way (i.e.. I've posted on Port Forwarding with IPTables a couple of
times, Building Apache 2 with PHP, and so on).  The "it took me ages to work
out how to make this work, so I'll write it down for future reference" type
of stuff gets blogged, in the hope that it'll help me (or someone else) out
in the future.

I'm also considering setting up a weblog for a work diary sort of thing, to
record config changes on servers, and general "what I did today" stuff.

Jim.
