OpenOffice 1.1Beta

To save others the pain:

  - "Export to PDF" works wonderfully.  There is little size
    difference between "optimise for screen" and "optimise for
    press".  Works for presentations too, which goes a long
    way to solving client problems (eg: can give presentation
    on any machine with Acrobat/xpdf).

  - The import of DocBook is a farce.  It interprets SGML
    entities and strips markup.  Leaving the document as
    a mess of unformatted text.

  - menus look nicer. It might be exploiting XRender and
    the new X font system.

  - changing from Inches to Centimeters is painfull, the
    option is repeated for each OO module.

  - no help text for new features

  - export options for PDA file formats

Things that haven't changed:

  - still slow

  - still its own user interface style

  - font importing and printer setup still sucks (although
    apparently it will accept a wider range of PPD printer
    description files).
