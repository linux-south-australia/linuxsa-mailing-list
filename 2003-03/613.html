Re: Samba user management

Andrew Galdes wrote:
> What GUI are people using to manage user accounts on a Samba server?

I suspect that most sites use the command line, as it's
a pretty rare scenario that you want to add accounts
in multiples of one very frequently.

> Ones i know of are "command line, SWAT and Webmin". I'm after something
 > that has the ability to restrict the administration to JUST user management,\
 > not server management.

It also depends on your authentication infrastructure.  For
example if you use LDAP than any PAM-aware tool that updates
what we'll nominally call "/etc/passwd" in reality updates
the LDAP directory.

You can set up some LDAP servers to update the LDAP and NT
passwords when a client requests a LDAP password update.

If you use this sort of infrastucture than there are
oddles of GUI front-ends for Unix account maintanenace
that can be used.
