Re: how to's for noob

Fabian,

> i finslaly got debian installed, just wondering if there are any
> tutorials on how to connect to the internet, set up a network, load x
> windows, load the games that come with it, and some sort of web
> browser and maybe even an mp3 player,

There are a lot of HOWTOs at:

 * http://www.tdlp.org/

...but they're not always easy to understand if you are a beginner. You
may have a copy of these on your system. Look in /usr/share/doc/HOWTO/.

To get you started, try:

# dpkg-reconfigure etherconf
# dpkg-reconfigure xserver-xfree86
# apt-get apt-cache
# apt-cache search games
# man apt-get
