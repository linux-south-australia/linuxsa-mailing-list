Re: Bash & Script

On Fri, Mar 14, 2003 at 03:19:47PM +1030, Glen Turner wrote:

> >Hello all,
> >
> >Just wondering if anybody could point out if there is a way to include a 
> >command in a Bash script to cause the "Script" program to terminate ?  To 
> >explain further:
> >
> >If in shell (bash) I do
> >
> >$ script SomeScriptName
> >$ ./myBashScript
> >
> >then I want the script to automatically terminate at the end of 
> >myBashScript.
> 
> $ SHELL=./myBashScript script SomeScriptName

Am I missing something? I'm sure it's not necessary to work that hard.
'script' accepts two command line arguemnts, script file to write and
program to run (instead of a shell):

hostname:~ > script test.log ./test.sh
Script started, output file is test.log
Hello, world

Script done, output file is test.log
hostname:~ > cat test.log
Script started on Fri Mar 14 16:11:16 2003
Hello, world

Script done on Fri Mar 14 16:11:16 2003

Oh OK, Linux 'script' doesn't support that. Maybe one day they'll bring
the basic utilities up to date :-) I guess you'll have to stick with
Glen's ingenious solution.
