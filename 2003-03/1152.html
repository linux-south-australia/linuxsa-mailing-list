run this as root:

iptables -L -v

(list your iptables rules verbosely)

you should see a line that mentions "tcp dpt:pop3"...

make sure this line is set to ACCEPT, and it's got the right interfaces
set for "in" (eg ppp0, or any). Check the source and destination -- if
either the interface or the source excludes the internet, that would be
your problem.

Also run "netstat -ap --inet | grep LIST" as root and see if your pop3
server is actually listening on the rigth IP addresses...

If neither of these tell you anything, check hosts.allow and hosts.deny,
or the pop3 server's config itself...
