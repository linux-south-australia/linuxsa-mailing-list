You've given me an idea. I think Chris should try the following steps:

1) Make a verifiable backup of what's already on the USB drive
 - use windows
 - make sure the backup is rock solid

2) Insert the drive into a Linux machine

3) Do the following:

# cfdisk /dev/sda
 - make one big vfat partition as the first partition
 - it should be reasonably obvious how to do it
 - make sure type is set sensibly (either vfat or msdos)

# mkfs.msdos /dev/sda1

4) Then try mounting the device under both Windows and Linux

That is to say, create an MSDOS partition, initialise it using Linux
tools and check it out.
