Re: Grid Computing

GRID computing

just high speed computing using shared resources..
e.g. super computers linked up by fast connections, and more fine 
grained control (API's etc) so you can write programs that use programs 
(and cycles) on far flung machines..

see Globus for one of the earlier incarnations.. http://www.globus.org
