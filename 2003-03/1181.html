I use an IPCop firewall and it works fine. On the dialup screen in IPCop for
my connection I have specified the IP address of my ISPs DNS (I'm not sure
this is actually necessary as I think the PPPoE connection handles this
automatically) and on my internal network (Green) IPCop provides IP
addresses via DHCP which I believe passes on the DNS info to the clients.
