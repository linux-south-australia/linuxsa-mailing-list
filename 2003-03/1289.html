Re: Printer Sharing

On Mon, Mar 31, 2003 at 02:21:05AM +0200, rbrandt@tumourbiology.com wrote:
> I've installed on my SUSE server box a printer (Epson on CUPS) I'd like to
> share with client boxex also running linux. Can somebody tell me how to
> configure the server to make that printer public and how to access it from
> the client boxes
> 
Ralf,

You need to have a look at the file in /etc caled printcap.
There should be tools on SUSE to configure this, tho...

$ man printcap 

the entry needs to be on both the client and the server.
the server entry connecting to the local printer and the client
connecting to the remote server... so both client and server will need
entries in the /etc/printcap for the printer.
