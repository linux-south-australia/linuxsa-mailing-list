Re: Text formatting -- *roff, [La]TeX, etc

Hi

I have been following this discussion about *roff and (La)TeX and nowhere
have I seen the package auctex mentioned. Auctex runs under (X)emacs and
provides pulldown menus for most (La)TeX commands. In addition it helps
format the tex in much the same way that C-mode under emacs helps format C
code - with multiple colours, etc. It provides shortcut keystrokes
for commands like \section{<title>}, prompts and other tricks
to make (La)TeX easy to write under emacs. In fact to my mind faster and
with better results than any word processor, though I admit that it takes
a while to learn.

There is another package - reftex - also an emacs addon - that makes it a
joy to insert references (internal to the document) and citations
(external).

I don't really know anything about the *roff packages, but for technical
documents (especially with math in it) I recommend (La)TeX written in
(X)emacs with auctex and reftex - and I write of the order of 20 such
documents a year. INcidentally I also use it (with the package texpower)
to produce slide presentations with all of the special effects of
powerpoint.

It's hard to have full wysiwyg in TeX though some packages try hard -
Scientific Word for example and the MacIntosh package TeXtures (I guess I
used this a long time ago and don't know its current status) - and those
that do try tend to sacrifice something else. This is because it optimises
the layout on a page by page basis. If you want to have partial wysiwyg
try using x-symbol in (x)emacs.

To see something of the capability of LaTeX look at
The LaTeX Companion by Goossens, Mittelbach and Samarin.
and
The Latex Graphics Companion
by Goossens, Rahtz and Mittelbach
