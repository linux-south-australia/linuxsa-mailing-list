On Friday, Mar 28, 2003, at 17:31 Australia/Adelaide, Rhett Kipps wrote:

> Hey everyone =)
>
> I've got a small problem...  Unfortunately, QUT fail to recognise 
> Linux as an operating system, and only support Windows with their 
> available VPN software.  I need a Linux VPN client that can login to 
> QUT's VPN (vpn.qut.edu.au), but it seems they use Nortel Network's 
> Connectiva software (as far as I can tell -- anyone have any 
> experience with this?).  Login involves a username and password, in 
> addition to a group username and password.
>
> I believe it is not possible to login to this VPN using Linux, but I 
> only have limited experience with VPN software.  I'm hoping someone 
> here may have helpful hints with what has worked in the past?
>

There is, unfortunately, no Nortel client for Linux (or any OS other 
than Winblows).  However, Nortel boxes use IPSEC and ESP/IKE for vpn 
protocols, so it *should* be possible (providing they're not using 
Entrust software) to use freeswan to connect to the Nortel box.

However, I have not attempted this.
