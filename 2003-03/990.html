How to use a Troff based Grapgical User Interface in Linux

As of March 19, 2003, the only troff based GUI word processor available 
is the EZ Word Processor, a component of the Andrew User Interface 
System, developed by Carnegie Melon University up to 1996.

Why is this important?

For those that know and understand Troff, they say that it is the most 
powerful word processor available.  However, it is extremely difficult 
for a new user to use,  so its power remains untapped if he or she does 
not have a set of training wheels to help get started.

What's the plan?

The idea is to download and use the EZ Word Processor to write up a 
document and explore the features available to troff. Once you have 
marked up a document, take a look at the file you saved with a text 
editor such as emacs. This will allow you to see how Troff works. As 
time goes by, perhaps you won't require EZ at all.

Procedure:

1. Log in as root
2. Go into the console
3. Type this: cd /usr/local
4. Type this: mkdir andrew
5. Open up your favorite web browswer and get this file:
    ftp://ftp.andrew.cmu.edu/pub/AUIS/bin-dist/ix86_linux/auis75lw.tar.gzip.
6. Type this in the console: gzip -dc auis75lw.tar.gzip | tar xvf -

Congratulations, you installed the EZ Word Processor

Now to use it!

Go into the console, type:  ./usr/local/andrew/bin/ez
OR
Make an icon and place that command in the execute tab.
