Re: port mapping

Run reverse proxying.

eg. Run Squid or Apache in reverse proxy mode on the firewall so that 
when a connection comes in they connect to the machine in the backend 
with the actual content.  Make sure you setup ACLs so that YOUR websites 
are the ONLY ones that can get reached via a proxy.  

This is sometimes called httpd-accelerator mode: 
http://www.squid-cache.org/Doc/FAQ/FAQ-20.html#what-is-httpd-accelerator
