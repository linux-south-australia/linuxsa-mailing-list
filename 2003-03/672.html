Re: remote build/install of kernel

Alan Kennington wrote:
> Has anyone figured out what things you have to worry about
> when you compile a kernel on one machine and then install it
> to another machine?

ObDebianPlug: Are you running debian?

If so, install ther appropriate kernel source package. Configure it and 
apply any pacthes you want/need. Make sure the "kernel_package" package 
is also installed. Then in the kernel source's root dir (usually 
/usr/src/kernel-source-x.x.x) do:

   make-kpkg --revision=mykernel.0.1 kernel_image

You'll end up a custom kernel .deb which you can copy and install onto 
other debian machines.

Read the docs in usr/share/doc/kernel-package for more info.
