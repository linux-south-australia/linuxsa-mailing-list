Hi Trent

When compiling a kernel there are a few steps

as root in /usr/src/linux

1. make mrproper
2. make xconfig
3. make dep
4. make bzImage
5. go to /usr/src/linux/arch/i386/boot and copy bzImage to /boot
-   go back to /usr/src/linux and copy System.map to /boot
-   go to /lib/modules and delete the directory (if there is one) of 
    the kernel you are building ie 2.4.20
-   this will delete modules from a previous build that you may not
    want this time.
6. make modules
7. make modules_install
8. edit /etc/lilo.conf
-  add a section using bzImage as your kernel image
9. enter the command   lilo   at the command line, this updates
10.restart
