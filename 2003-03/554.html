Re: Impress Player / Star Player

The StarOffice player included in SO 5.2 was built using a proprietary 
3rd party module so it could not be included in the fully OpenSource 
OpenOffice. I am not sure if anybody at OpenOfice.org is working on 
one, I hope they are. If you get a copy of the SO 5.2 player (I have 
one if you want) and then save your Impress files out as StarOffice 5 
Impress files, extension .sdd they will play fine in SO 5.2 player. 

You may need to match your fonts to the ones that the Player handles.

Test and double test before using this in a presentation situation.

Hope this helps.
