Re: XHost +

David,


As I understand it you've not executed this:

# xchost +

...but rather this:

# xhost mincepie

...which I assume works. This means that any machine that can convince
the server that it is "mincepie" can use the server's display. I don't
particularly see this as a security risk because the clients would be
running not on the server in the first place.

In fact I'm not sure why allowing people to place their windows on your
desktop (i.e. an X Window) is such a security risk apart from taking
bandwidth and/or being a nuisance...
