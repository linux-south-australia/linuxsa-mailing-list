RE: SMS Gateway

Steve,

We've been using gnokii and a nokia 5110 to send SMS messages here for a
while.  We use it for network monitoring, where it sends us an SMS whenever
Netsaint determines that something has broken.  Download and install gnokii,
and get yourself a cheap data cable for the phone (you can get them for
about $40 at Tandy's and places like that, don't pay $200+ for the Nokia one
unless you really want the windows software as well).

Once that's done you should be able to hack up a quick shell or perl script
to take the message and send the SMS.  Call the script from an alias, or
.forward.  We get Netsaint to run gnokii directly and bypass the email step,
so  don't have such a scrip to hand unfortunately.

Some ISP's also run an email to SMS gateway, and we originally used
Internode's service, but ditched it and set up our own because an external
service was no good if our internet link failed and the email couldn't get
out.  Their service was excellent, and we had no complaints, it just wasn't
ideal for our circumstances.

Jim.
