Re: Text formatting -- *roff, [La]TeX, etc

Daryl Tester <Daryl.Tester@iocane.com.au> wrote:
> The first formatter program (i.e. the program that the
> Unix guys convinced Bell Labs to buy a PDP-11 in the
> first place) was called roff.

Troff was based on roff, which was based on a program of the same name,
standing for "run off".  Since early versions of troff only produced
output on a CAT photo-typesetter, I presume troff stood for "typesetter
run off."  Later came ditroff, which despite the oxymoron, stood for
"device independent troff."
