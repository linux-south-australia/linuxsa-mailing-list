Re: full duplex w/ Tulip driver

"Freeman, Peter (ERHS)" wrote:
> 
> Just wondering if anyone knows the option for switching to full duplex for
> NICs using the Tulip (new one, not old one) driver.
> Theyre running at 100 ok, but only at half duplex.

Someone else has already provided the answer.

But be warned that auto-negotiation to half duplex is often the
result of misconfiguring the switch at the other end of the cable
to run at full duplex with no negotiation, rather than setting the
switch port to auto-negotiate (when it will then agree on 100Mbps
full-duplex).

If possible, you should always set ports to autonegotiate, this along
with DHCP allows "plug and ping" networking, which is useful for
notebook users and very handy when moving hosts around to cope with
a switch failure.
