TCP Port Forwarding Success

I have finally managed to get port forwarding TCP Port 110 working. It
actually does work exactly as the man pages in fetchmail but you _must_
use openssh - I was using the free for personal use ssh2.

poll mail.rebel.net.au via localhost port 1025
no dns
preconnect 'ssh -C -L 1025:mail.rebel.net.au:110 rebel sleep 20'
password as_if_i_would_tell_you
