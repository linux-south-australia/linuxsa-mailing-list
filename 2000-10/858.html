RE: Bandwidth Throttling

> Is there any way to stop an active [1] ftp get from hijacking 
> almost all
> of my bandwidth? Currently almost everything else simply goes awfully
> slow, whilst the ftp transaction appears to be going as fast 
> as my link
> (over a modem) will allow it.

I use "Network Downloader" which sort of emulates Gozilla!
for WinXX by providing throttle buttons. You can even define 
how much bandwidth each throttle button will restrict data 
flow down to.

It works reasonably well, although it's not perfect. The
neat thing about Network Downloader is that you can right
click on a URL and select "Copy link" and it will intercept
it (when using Netscape that is). No need to cut and paste 
down load links into FTp applications or command line utils.
