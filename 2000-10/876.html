Alan Kennington wrote:
> My guess is that as someone else commented, you need to get a better 
> C compiler.

'omake CC=kgcc whatever' uses the version of gcc which is supplied with RH7
which is able to compile the Linux kernel successfully.

It's all documented in Red Hat's manuals that you'll find on their documentation
CD :-)

On a side note, it appears that 2.96 of gcc was included so that big business
customers of RH developing in C++ could have a reasonable compiler. 
Unfortunately some of the optimisations in the kernel always break bleeding-edge
versions of gcc, hence the older version supplied as kgcc.
