More strictly speaking, it is the index of the last argument, because, as
you suggest below, $0 is passed and is the [file]name of the command.

The reason for making this distinction is that C does it slightly different
...
and passes the actual number of items on the command line, which can be
thought of as the number of argument, with argument 0 simply being the
command.
