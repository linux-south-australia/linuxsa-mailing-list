Re: Safe way to move/copy between different partitions

> I shall shortly be moving around bits of my filesystem from one
> partition to another and then later back again.  Some of this will
> occur over NFS.  I am worried that corruption might occur in the
> process.  Are there any ways to guard against this.

Before you copy the files create a checksum file in the root directory
of the files you want to copy.  ie. goto root directory and type

	find . -follow -type f -exec md5sum {} \; > md5sum.txt

Then copy all of the files to the required location (include the
checksum file) and use the checksum file to test that they are
correct.  ie. goto the root directory of the copied files and type
	
	md5sum -c ./md5sum.txt

This will tell you if there are any differences between the files.
