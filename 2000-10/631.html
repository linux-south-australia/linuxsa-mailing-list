> 
> What can I do ? Some solutions for me ???

Your friends are "top" and "strace".  Firstly, use
top to see if the five or so soffice processes are
starting, if not then it's crashing very early.

Use "strace" to make sure that it's not waiting for
some unavailable resource (such as doing a DNS name
lookup when you are disconnected from the Internet
or a font that is no longer served by the font
server).

The other way is not to care.

I hope you did a network install of StarOffice.

If so, just rename the soffice52 directory (or whatever
it's called) in your home directory and start SO again.

Copy across the files you care about from the old soffice52
directory.

You can also kill the splash screen from the Tools | Options
menu.  Since the splash screen is modal, but has no window
manager controls, turning it off is a good idea.

If you have no luck, as it look like you speak German, try
a posting to a German language StarOffice newsgroup.  They've
been using SO for a lot longer than English speakers and
tend to work out issues sooner.
