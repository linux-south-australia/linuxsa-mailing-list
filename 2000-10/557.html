Re: Interactive Perl example needed.

How 'bout using  the <> operator?

Eg:

#!/usr/bin/perl

print "1 - something\n";
print "2 - somthing else\n";
print "Enter your menu choice: ";

while ( $command = <STDIN> )
{
        if( $command =~ /^1/ )
        {
                print "chose something\n";
                goto OUT;
        }
        elsif( $command =~ /^2/ )
        {
                print "chose something else\n";
                goto OUT;
        }
        else
        {
                print "Invalid selection - please choose again: \n";
        }
}

OUT:

print "valid select made\n";

Pardon my goto :-)
And of course you can get rid of that silly explicit variable $command and
use $_ etc, but you get the idea.
