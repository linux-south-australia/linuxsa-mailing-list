assign is source for cdb by running 'qmail-newu'. If this cdb file
exist, then qmail no longer using qmail-getpw to find user's homedir,
which is slower.

Seems that the guy is using virtual user account. There are several
patch/add-ons for qmail. One that I know is checkcdb from Chris
Johnson @ palomine.net.

The most popular add-ons for qmail to administering virtual
users/domains are vpopmail (http://www.inter7.com) and vmailmgr.
