RE: Exchange Type Mail server.

They've probably outgrown Teamware.
Teamware is marketed as sm-med business, and from using both TW & Exchange,
I'd say that Exchange does scale to the larger enterprise much better than
TW ever would, but for the small-medium sites (such as ourselves), is the
price u pay for Exchange (+ NT + fat server to run it on + CALs) really
justified.
After doing the math on a 300 user scenario, we came to the conclusion that
it wasn't!
