Re: LILO boot interrupted <newbie>

Schultz, Peter (DIT) [Schultz.Peter@saugov.sa.gov.au] wrote:

> my problem is that boot routine is stalled on my PC.  I have both
> Win97 & RH 6.2 with dual boot.  On start up normal processes apply
> but then stall with at LI (not LILO, which automatically takes me to
> a Win97 install if I don't log on for RH).
>  
> Any ideas how I can fix the problem?

A "LI" rather than "LILO" suggests that something is probably wrong
with your boot sector.  Have you got a boot floppy, so that you can
get linux started?

If so, boot Linux using the floppy.  Then run lilo to reinstall the
boot sector.

If you haven't got a boot floppy, you will need to use a rescue disk
to do the same kind of thing.

Cheers,

Mark.
