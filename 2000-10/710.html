Re: Wingdoze fonts & Debian

See the font beautification howto, but briefly:

* copy the fonts into a directory
* execute ttmkfdir > font.dir in the directory
* add the directory to the xfs path or X path

Look in /etc/X11/XF86Config or /etc/XF86Config or /etc/XF86Config/fs (or
something like that - I refuse to use a font server)....
