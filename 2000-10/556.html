Interactive Perl example needed.

Greetings,

I would like to have a perl script prompt for input, so users(me) can
select from a menu of options.

The following works (almost):

$ch='';
while($ch !~ /^q/){
  menu();
  $ch = getc();

  if($ch =~ /^1/){
    ...
  }elsif($ch =~ /^q/){
    ...
  }else{
    ...
  }
}

The problem is that getc, only reads one character at a time,
and the input from stdin is buffered.. so typing '1<ENTER>'
runs through the loop twice.

1. Is there a better mechanism in perl to do this?

2. How do I get unbuffered output from the command line?
   (This will also be useful for solving some C problems that I have.)
