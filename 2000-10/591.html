> Hey All.. I Was wondering if I could add a user with super user rights
> eg su -

Of course, there are a number of ways.

1) Tell then the root password

   You may also need to add them to the wheel group,
   edit /etc/group and append the user ID to the line

     wheel:x:10:root,gdt

2) Get them to use their password

   Install sudo (it will be on one of the application CDs
   of your distribution, as it is a popular program).

   Set the sudo configuration file /etc/sudoers to

     # This file MUST be edited with the 'visudo' command as root.
     root	ALL=(ALL) ALL
     %wheel	ALL=(ALL) ALL

   Add the users that need superuser access to the wheel
   group, as before.

   Instruct your users that the command

     sudo -s

   and supplying their password will get the root access, if
   this is allowed.

You can set up sudo to do keystroke recording, but really
you shouldn't be giving anyone root access to people you don't
trust and running CVS across the configuration files is
probably better for working out who changed what and why.
