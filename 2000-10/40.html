Re: Fake Cd's and Samba

Steve Sloan wrote:
> 
> List
> 
> Does anybody know if it is possible to run fake CD's on Samba, I believe
> there is software around that will work on NT ok.
> I am looking at this option as opposed to using a CD stacker, which should
> workout cheaper and give quicker access.
> 

How about using dd to grab an ISO image of the CD and mount it using the
loopback block device. You can then share out the mounted directory
using samba ( and even set the SMB file system type to CDFS, thus giving
the complete illusion of a shared CD-ROM ).

Hope it helps,
