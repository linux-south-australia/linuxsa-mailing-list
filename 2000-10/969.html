linux point of sale software for 20,000 Tesco points

Someone asked a while back if linux could be used
for point-of-sale applications. Well....

http://www.theregister.co.uk/content/1/14317.html

now it seems that 

-------------------------------------------------
Tesco tills are going Linux, following an agreement with
Israel-headquartered retail food industry software specialist
Point of Sale Limited. The deal, announced today, will cover up
to 20,000 checkouts in eight countries. 
-------------------------------------------------

They previously used DOS and NT.

Alan Kennington.

PS. 
By the way, on the subject of MIDI transmission software,
I've tentatively decided that the ALSA software must be
where the centre of gravity of linux MIDI development must be.
http://www.alsa-project.org/
And a simple MIDI transmittion utility using the ALSA
software for the drivers etc. is pmidi:
http://www.parabola.demon.co.uk/alsa/pmidi.html
This software is part of the SuSE distributions, but
I changed the kernel on my main workstation.
So it was necessary to get the latest software
and re-build it for the new kernel.
