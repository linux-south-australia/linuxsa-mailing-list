RE: [OT] who wants it anyway!!??

NO!!  Definately NOT!!!!  Don't even THINK about it.

Accessing this kind of stolen source code would be a legal nightmare
and probably death of ANY Open Source (of Free Software - GNU/Linux,
GNU/Linux - Hiya Stallman) project!  To see a very well laid out
description of the issue here's a link to "Say "No, thanks" to Offers
of Illegal MS Source Code" from linux journal:

http://noframes.linuxjournal.com/articles/currents/0024.html

For the lazy the most damning quote from the article:

"Microsoft, he said, would be in a position to seek damages from
anyone trafficking in misappropriated trade secrets, which can
include merely reading the Microsoft code and then contributing to a
free project. "

So remember kids, JUST SAY NO!  (And GNU/Linux if you want to stay on
RMS's good side).

Cheers
