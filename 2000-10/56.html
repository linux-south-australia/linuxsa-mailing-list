any procedural language that is... and most Object-Oriented languages, but
you will be programming in a procedural style... One of the biggest problems
with people moving from C (or Pascal or Perl or any other procedural
language) to Java (or C++ or any other OO language) is that they generally
program in a procedural style anyway... So they get the disadvantages of
both prcedural (lower maintainability and modularity) and OO (lower
efficiency) without the benefits of either. You really need to learn how to
program in an OO style to use an OO language effectively.
