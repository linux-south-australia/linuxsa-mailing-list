> Quick question anyone had an "easy" upgrade to PHP4? I have a few goes and I
> couldn't apache back up, so I'm still on PHP3 and have put the upgrade in

If you're using Apache with apxs (shared object support) it couldn't be
easier. The default apache installs I've seen for Linux don't do this,
but I think the FreeBSD one does. Check for it by trying to execute
/path/to/apache/bin/apxs. For some reason, compiling PHP4/Zend crashes
on some AMD CPU's that have a rare bug.
