Re: RPM specs

> Does anyone know how to re-create a spec file for an installed RPM?

If you mean an installed binary RPM, then I don't think that is
possible.  You can obviously get the files and scripts sections from
rpm queries, as well as the relationship between the binary packages
and the source package.  However, the actual commands used to prepare,
build, and install the source code will not be part of the binary
package.

I can furnish you with a skeleton spec file that is consistent with
most autoconf/automake controlled source.  You will be in a good
position to complete it, because you will have an idea of which files
you need to install, but there are _always_ wrinkles.  (People just
aren't anal enough about there makefiles ;-).  email me if interested.


Cheers,
