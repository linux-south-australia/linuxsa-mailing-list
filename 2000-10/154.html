On Tue, 3 Oct 2000, Ian Hogben wrote:

> I had a similar problem when I moved from one ISP to another. Try the modem
> init string. Reading through the MODEM HOWTO (linuxdoc.org), I found a good
> string that is very reliable. Try something like this:
> 
> Located in /etc/sysconfig/network-scripts/chat-ppp0
> OLD: "ATS0=0Q0V1"
> NEW: "AT&K3E1S0=0Q0&C1&S0V1"
> 
> A description on the various AT commands follows:
> &Kn - Select Flow Control Option
> En - Select Command Character Echo Option
> Sn= - Write To An S-Register
> Qn - Select Result Code Option
> &Cn - Select Data Carrier Detect Option
> &Sn - Select Data Set Ready Option
> Vn - Select Response Format Option
> 
> HTH.
>
