smb authentication for Squid

Hi all

I've got Squid successfully running & authenticating users through our NT
PDC using the smb_auth module.
However I'm having one small hassle.

If I use a valid username on the PDC to authenticate it will accept/reject
if the password is correct/incorrect.
This is great, and exactly what I want, however if I enter a user that is
unknown to the PDC, Squid lets them through, thus defeating the point of
having authentication in the first place!  

I would really like to get this working with smb usernames/passwords so our
poor old users don't have another set of passwords to remember.  (and, more
to the point, I don't have to maintain another set of usernames and
