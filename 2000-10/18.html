RE: C

I started with "The C Programming Language" by Kernighan & Ritchie.  It has
an easy intro tutorial.

Another good tutorial can be got from http://www.cs.virginia.edu/~lcc-win32/
.  The compiler there is for Windows (it's free and quite good, with an IDE
and all) but if you just download the documentation it has a good tutorial
in it.  There are many more around the web.  Once you get into C you may
find the need for some advice on some of the trickier things.  There is a
good document I got from somewhere which was a sort of "tips & tricks".  I
will see if I can track down a reference for you and let you know.
