RE: Setting Variables in BASH

Hi Andrew, 

It's pretty straightforward.

You would call your script like this:

$ ./myscript myarguments

Inside your script you would have something like this 

echo "The argument you typed was $1"

if the user types in for example 3 arguments, they then are $1 for the first
argument, $2 for the second argument, and $3 for the third. 
An interesting one is if you did something like:

.....
echo $0
.....

it would echo the scripts name. 

you can also count how many arguments have been passed at the command line
with $#

if you did:

.....
echo $#
.....

and the user gave 3 arguments, then echo $# would echo 3

For lots more info on BASH scripting see the BASH-HOWTO's
http://www.linuxdoc.org/HOWTO/Bash-Prog-Intro-HOWTO.html
http://www.linuxdoc.org/HOWTO/Bash-Prompt-HOWTO.html
