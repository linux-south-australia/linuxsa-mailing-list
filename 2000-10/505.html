Re: RH 7.0 : nVidia TNT2 32mb

Hi Neil

One of the rpm's you list here is an smp (Symmetric Multi Processor) binary. 
Do you have more than 1 cpu in your box?

There is a RH7 version of the Kernel file but it's only Release Candidate 2, 
but I haven't had any hassles yet on my machine.

You need to create a new XF86Config file as this file is different in XF86 
3.3.x, use xf86config and manually make the new config file. I couldn't get 
xf86cfg to work so I didn't use it.

If you need some help with the settings for these files leave another message 
and I'll try and help out.

Robyn
