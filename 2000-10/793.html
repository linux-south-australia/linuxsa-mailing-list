Grab the latest sendmail. I'd look into using sendmail's LDAP routing
feature (there are some articles on www.sendmail.net and the sendmail
documentation is very thorough). This will allow for better scalability
and you won't need a setuid root CGI script to create the accounts. You
can use any programming language to create the CGI part. I'd use perl
and the CGI modul, as you won't have any buffer overflow problems and
the CGI module appears to do a good job at escaping nasty characters.
