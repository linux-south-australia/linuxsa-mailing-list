Re: Setting Variables in BASH

Andrew Reid wrote:
> 
> I am trying to write a BASH script that takes a variable from the
> command line
> in -- form
> 
> ie: ../command --argument1=asd
> 
> How do I go about this? Any help would be appreciated.

Dealing with non-trivial command line arguments (in shell, C, perl etc) is
something that has been done many times before, so don't do it yourself, use the
appropriate library function - getopt or getopts.  See the man/perldoc pages for
details.
