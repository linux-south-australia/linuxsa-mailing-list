recommended CD-RW drives

I'm considering the purchase of a new CD-RW drive.  It will
be going into a dual-boot RH 6.1 + Win98se box, and I would
like the drive to function no matter which is booted.

The CD write speed is not an important factor for me. The
box does not have any SCSI adapters and if possible I would
prefer to avoid this additional expense.

HOW-TOs and other Net sources I consulted include lists of
older CD-RWs, but don't mention currently-selling models.
One would expect that a more recent sibling of a
known-to-be-ok older model would also work...but I've been
caught out before :(

Any comments/opinions on which new CD-RW drives I should
consider (or avoid) are welcomed.
