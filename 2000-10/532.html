Re: Support for modem?

Hi Daniel

There isn't really such a thing as a modem driver for the KTX, you only need 
to tell Linux that it is there and it will use the default settings which 
should work fine. I have a KTX 56k and it's fine.

Use su and go to root, the program to configure your modem is called 
modemtool.
