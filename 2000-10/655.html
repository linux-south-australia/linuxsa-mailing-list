Re: Slow response to POP3 server

Check your DNS setup. Is it slow to telnet / ssh into the server as well? It
sounds like it is trying to find the IP address of the mail server.

Outlook Express won't complain about the mail server taking longer than 60
seconds to reply, because that's not the problem, the problem sounds like it
_can't find_ the mail server for over 60 seconds.
