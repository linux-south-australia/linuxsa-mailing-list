Hi Michael

Yes, as you said junkbuster is a proxy so just set netscape to use it.

Say your PC with the ISP connection has an ethernet ip address if
192.168.1.1 and you Notebook has 192.168.1.2

You may need to adjust the /etc/junkbuster/aclfile to have the line "permit
192.168.1.0/24"

Then go into the Netscape setting and set the http proxy to 192.168.1.1 on
port 8000

It will just work.
