Re: Monitor resolution using Gnome

Peter Wade <pwade@posword.org> writes:

> How can I change my monitor resolution using Gnome under RedHat 6.2?
> Is there a similar utility to the Display control panel of Win98, or
> do I have to force it from the command line?

You can set up your xserver to be able to use several resolutions
by putting multiple modes in the screen section:

Section "Screen"
   Driver          "SVGA"
   Device          "Primary Card"
   Monitor         "Primary Monitor"
   Defau   EndSubSection
   SubSection "Display"
      Depth        24
      Modes        "1024x768" "800x600"    <--- Here
   EndSubSection
   SubSection "Display"
    
It should be possible to swap between then using <Ctl><Alt><+> and
<Ctl><Alt><->.  This keeps the same virtual desktop (you can scroll
the view port around) and the color depth.  I'm not aware of being
able to change from "1024x768" 256 Colours (8 Depth) to "800x600" (24
depth) this way though. (eg. swapping between 'Display' definitions).

There is a Gnome applet which can do the above magic key stoke thing for
the user.
