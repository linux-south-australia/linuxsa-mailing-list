Re: linux for mobile phones - supported?

> > On the topic of connecting a linux computer to
> > a mobile phone (e.g. Nokia 6210 or 6150),
> > is there anything different between this and
> > connecting up to a standard vanilla modem?
> 
> I'm not entirely sure how the "modem" works in the Nokia phones, but
> it's not like a Hayes AT interface. I don't think it's like a winmodem
> either, so the phone has some smarts. You still need to buy the windows
> software to get the cable for the phone <-> computer interface, and I
> don't think the Infrared stuff works. If someone knows where to get the
> cables cheap, speak up!
> 
> http://gnokii.org/ is attempting to address this, but I don't think it's
> yet ready to be a substitute for a modem/gsm card. It is, however, great
> for sending SMS messages and tweaking the settings on your phone :)

You need software drivers for the 6150 and 6210.  Gnokii will allow
you to do data calls but it is still beta.  

I suggest you forget about the 6150 and the 6210 and get an 8210
instead.  Much nice phone and the IrDA is far better.  Also, the
prices have dropped a lot in the last few months.  

I have an 8210 and you *don't* need any drivers.  The phone has a
fully IrDA compliant serial port which accepts AT commands.  

Once you get the Linux IrDA stuff working it's really simple.
