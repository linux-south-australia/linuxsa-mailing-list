RE: RedHat/Apache/PHP/PostgreSQL/SSL/FrontPage

I have done many installs of Apache/PHP4 (simultaneous Apache module + CGI)
and Postgres 7 and have not had to many grey hairs doing so.

The only thing that I would recommend is installing the penultimate version
of PHP4 first including, then installing the latest over the top. for some
reason this clears up the dreaded file not found message when using the cgi
version of the PHP inetrperater.

Postgres is easy.

Just follow the instructions from PHP to get it apache/php installed 
(typical php configure is

 ./configure  --with-pgsql --enable-calendar --enable-track-vars
--enable-trans-sid --enable-discard-path

This supports sessions, and does postgres 7
