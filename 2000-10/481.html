RE: DNS & Squid problem.

It's almost certainly an DNS problem. 
Does it happen with other websites such as www.yahoo.com ? or is it only
your server's web address that this happenes on. 
If it's only happening on your servers, then you need to add an entry in
your DNS that will look something like

www.clarehs.sa.edu.au  	IN 	A 	208.10.196.10
 The IP I've given is bogus (obviously) you should put in the correct IP for
your server. 

I'm guessing there is a file in /var/named called something like
clarehs.sa.edu.au 
This is the file that you need to put the above entry in. Don't forget to
increment the serial number in this file, and then once you've made the
modification, you need to restart named
