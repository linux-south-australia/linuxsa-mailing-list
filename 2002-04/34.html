If you're running on SuSE, SuSEFirewall2 is a script and config file
front-end to iptables. I haven't played with it a great deal, but it
seems to be pretty straight forward to setup ( no pretty GUI, just a
conf file in /etc/rc.config.d ).

If you want, I can post a copy of a config file to see if it's what
you're after.
