I am using a program called NTOP (http://www.ntop.org) to monitor our
Internet usage on a per-IP basis.

It's pretty good, does graphing and is very thorough, although I'm not
completely satisfied with it yet, because there are a few bugs and
sometimes I'm not confident the information I'm being shown is actually
correct.  BUT that could be because I'm still playing with it and I just
have to play some more.  But it might be worth investigating.

It also has a web front end, so you can use any Browser in conjunction
with it.  It also uses DNS to display hosts by hostname.
