RE: IDE geometry detection & getting to BIOS

> Some Sun BIOSs use "Stop-A", I believe, but I guess you were 
> only talking about IBM-compatible AT/ATX style PCs.

Yes -- that's a little different though... BIOS is a PC only thing,
AFAIK. Suns use OpenPROM, which is functionally similar, but more
sophisticated (yet less pretty). Suns, by default, boot into OpenPROM,
then you type "boot", and they boot into the OS (unless you have them
set to auto-boot). You can also drop into OpenPROM at any stage, by
pressing Stop-A, but of course, this actually stops the running OS and
programs, so normally you don't want to do that... :)
