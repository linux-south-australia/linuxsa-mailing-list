Re: squid

On Tue, Apr 16, 2002 at 01:41:10PM +0800, jason pernito wrote:

> just asking if it is possible to configure squid cache/proxy
> to always point at a particular URL whenever a client first 
> logs in to the cache?

That's pure evil.

Client: "I want http://www.google.com/"

Cache: "No. Piss off. You'll get what you're given"

It'd probably need to be built into redirector_program. Have a
poke around, or write something yourself. 

What I'd suggest you do is set the default homepage of the
browser that you are using to the page that you want to
redirect to. Make sure this is a site that is accessed via the
cache (i.e., not on the "Do not use a proxy for this address"
list).

That way, as soon as they login, they'll get your page.

Returning a page different to the one that the client requested
will probably cause havoc with browsers own caches, too.
