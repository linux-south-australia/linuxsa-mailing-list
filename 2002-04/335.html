>  > A windows window opens and tells me that I don't have the tahoma font
>  > installed.  What is the best way to add fonts under Mandrake?
> 
> I believe Linux Mandrake has a font management tool.  Going by
> Mandrake's scheme, it is most likely called "FontDrak" or something
> similar.
> 
> If you also have Windows installed on your machine, FontDrak will
> be able to find your Windows fonts and install them on your Linux
> system.  If you don't you can download Microsoft's font pack from
> the Microsoft web site.  I think the standard pack is called
> "WebFonts", and was available from microsoft.com/typography last I
> looked.
> 
> If none of these helped, you could copy all the *.ttf files from a
> friend's Windows system and use FontDrak or a command line tool like
> ttmkfdir to install them on your system.  (Be aware that doing so may
> violate some licenses.)
> 
> Hope this helps.
