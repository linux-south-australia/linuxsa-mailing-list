Re: [NEWBIE QUESTION]: Fonts and Mandrake 8.2

Daniel Falkenberg wrote:
> This is probably a pretty basic question for most of you!  I am
> currently trying to run outlook.exe under wine (I know! :) ).  When I
> try to run the executable OUTLOOK.EXE A windows window opens and tells
> me that I don't have the tahoma font installed.  What is the best way to
> add fonts under Mandrake?

It seems the Tahoma font isn't available in the font pack.  It does,
however, seem to be available here:
http://support.microsoft.com/default.aspx?scid=kb;EN-US;q188081

As others have already stated, there are some good Linux mail clients
that would be worth looking at.  Evolution is quite similar to Outlook,
and seems to be a great program.

If you wanted to install Evolution, it would be easiest to install
Ximian GNOME and let it handle all the other libraries that need to
be installed to make Evolution work.
