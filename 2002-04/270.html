Re: ssh query

If web access is available, there is a java ssh client called 'mindterm'
that can tunnel through a http proxy server to connect to port 22 ssh hosts,
assuming the proxy server isn't behind a similar firewall. Google will get
you straight to a page that allows a browser-based demo of mindterm, where
you can enter an arbitrary host to connect to.
