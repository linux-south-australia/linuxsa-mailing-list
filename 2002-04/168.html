RE: Mandrake and terminal services

> Does any one know of any software that will run on my 
> Mandrake 8.2 machine that will allow me to terminal services 
> into Win2K machines?

If you're actually using Windows Terminal Server, then you want Citrix
Metaframe for Linux. I know there's a RedHat version, and suspect it
should be ok under Mandrake with minimal frobbing.

If you're just using a plain old vanilla Win2k, then you want VNC or
something, which is actually quite different to Terminal Server.

VNC lets you control the single disktop, as if you were in front of the
machine. It's equivalent to remotely controlling the console on a Unix
box that only has one console. You can't have more than one person doing
it. Terminal Services is a kludge to make Windows Multiuser. In
practice, it works quite OK, apart from a bit of slowness (on a 100Mbit
network), and a bit of instability.
