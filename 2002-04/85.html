Re: displaying non-printing characters (tabs, spaces, ...) in Emacs

Quoting Michael Wardle <michael.wardle@adacel.com>:
> Somebody on an Emacs newsgroup pointed me to a collection
> of Emacs Lisp modules:
> http://www.anc.ed.ac.uk/~stephen/emacs/ell.html
> 
> It includes a "blank mode" which apparently shows
> whitespace in a distinctive, visible manner.

Or indeed for vim users you can add the following to your ~/.[g]vimrc for the 
same effect:

set list
set listchars=tab:>-,trail:-

[And while I'm at it - get rid of those pesky tabs with:
set expandtab
:-]
