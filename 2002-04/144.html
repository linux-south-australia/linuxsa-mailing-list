Re: Xserver Problemo :-(

Brian,

One thing you can try is using the xhost command.  From the shell as
root if you type xhost it should tell you if access control is enable
for the Xserver and hosts who have access.  You can turn access control
off with the command "xhost +" and see if you can connect.  But leaving
it off is one less layer of security...

Mark
