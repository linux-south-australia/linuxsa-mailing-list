Re: Tricky install for the clueless...

Well, I havnt done it but hopefully I can give you some pointers. The
accton pcmcia cards (at least that I have seen) are like almost all of
the cards of brands that dont/cant make their own chipset and use the
intel i82365 chipset. That should be the main bit of info you need. 

Presumably you'll need to make the network install boot disk, tell it
what card you have and either tell it to get an IP via DHCP if you have
a DHCP server, or else manually assign an IP. Under linux this is all
done via the text mode installer that you get when you boot the network
install boot disk. I doubt you'd need to modify any conf files before
the system is installed. I havn't tryed this with bsd (this is linuxsa,
after all :). Of course you'll need an ftp server with the installation
files from the cd on it, and you'll need to point the installer at it.
One you have the network card going though, it should all be straight
forward enough. If its not, use linux! :) </troll>
