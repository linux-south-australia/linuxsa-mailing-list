Re: [OT] Partition Images

Hi Guys

I've been setting up partimage to allow me to image 40 new PC. I had to add
support to the boot CD-ROM for the rtl8139 network cards, this turned out to
be easy. Just mounted the 2.88 Mg boot image and copied in my new kernel.
Then used mkisofs to create a Eltorito boot CD-ROM

I was also able to adjust the initrd image to auto mount the CD and run
partimage to restore the image.

I highly recommend this software, as long as you are willing to learn how to
building your own kernels, editing boot scripts and creating CD-ROM's.

Cheers
   Mike O'Connor
