> Basically looking to by a second hand unit that has what it 
> takes to run the above using Mandrake 8.2.
> 
> What processor speed, Ram and harddrive size do I require.

Star/Open Office and GNOME will eat a lot of RAM. Get lots. Try for
256Mb. As much as you can, really. CPU speed -- well, I'm pretty sure
you'd see people using 200-300 MHz, but I'd suspect it gets a little
painful at times. Aim for 500+. HDD -- again, big is good, but I suspect
a few Gb would be sufficient, if you're prepared to be a little careful
about not installing things willy nilly...
