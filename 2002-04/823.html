Hello

Recently I setup ipac-ng http://sourceforge.net/projects/ipac-ng/
It uses iptables (or ipchains) to setup packet accountting rule to
monitor taffic and logs into a postgres database.  It can then be used
to create a webpage showing traffic and because it uses ip tables you
can log traffic via port or ip or interface etc.
You can then write cgi scripts to generate the graphs when you want them
or cron jobs to generate the graphs automatically.
