Matthew Moyle-Croft wrote:

> "Partition Image for Linux" http://www.partimage.org/
> 
> Which supports most O/S AND supports network images and multiple clients
> simultaneously.  Doesn't support Multicast nor some of the more advanced
> features, but looks good enough to backup and restore a whole image.

Partimage works well, especially the network mode. If you ignore the
warnings and copy mounted partitions, it still works to produce working
replicas of data. Because it only copies actual data and can use
compression, it's a LOT faster than dd on large disks.
