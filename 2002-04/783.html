I've only ever seen this problem on bad or corrupted install media. It 
most likely means that the Mandrake installer can't read the disk or 
something's just not right about the format of that file.

When you were installing, try using "ctrl+alt+f1..4". One of them might 
contain errors about the CD drive (likely to say something about seek 
read error or some such obscure message). Press "alt+f7" to get back to 
the graphical installer.
