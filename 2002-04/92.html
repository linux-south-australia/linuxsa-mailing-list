Re: DHCPD events - Linux or Win Problem ??

The requesting machine requests an IP. What is going on should be logged
to /var/log/messages.

EG:

Mar 12 10:44:13 adsl dhcpd: DHCPREQUEST for 10.3.255.252 from
00:d0:59:2e:56:b7 (c1844) via eth2
Mar 12 10:44:13 adsl dhcpd: DHCPACK on 10.3.255.252 to 00:d0:59:2e:56:b7
(c1844) via eth2

However, if you want a particular IP for your machine each time,
consider using static IPs via DHCP. A clause like so in your
/etc/dhcpd.conf should do it:

# User Name
host hostname {
    hardware ethernet 00:50:BA:76:eb:30;  
    fixed-address 10.0.1.51;
}
