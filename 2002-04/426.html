Its not through networking as far as I can tell, for instance you can
access a drivers directory from a basic DOS drive (Just boot files).But
I have to say I havent read enough of the documentation but my belief is
that its emulating a  drive to the OS (Not to the BIOS, as our systems
boot the Windows image will out having any drives set in the BIOS).

An example is that to install Windows, you create a basic drive through
the software (It includes DOS boot files). Once thats created, you boot
it and then (Without using MSCDEX drivers), access setup.exe thru an
emulated drive.

Of course, this could mean that the version of DOS (Caldera OpenDOS
7.01), could be hacked aswell.
