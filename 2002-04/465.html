Re: SAMBA

Daryl Pilkington wrote:
 > Hi, I've got to install a Samba server with DB2 v7.2 by Saturday. It
 > will have W2k & Win98 clients. Printing will be lpd. I have an Axis
 > Win9x lpr client.

You can print using "Windows Printer and File Sharing" (or similar)
over Samba.  There is no need for lpr on the Windows side unless you 
want it.

Do you plan to use the Samba server for anything other than printing?

 > Could people suggest some good URLs for HowTos for newcomers.

There is supposed to be much information in the latest version of Samba,
including a copy of a SAMS book IIRC.  Once it's installed, you could
look in /usr/share/doc/samba* and see what's there.

Until then, www.samba.org will have plenty of information, and 
www.linuxdoc.org has lots of Linux HOWTOs.

I find Samba is quite easy to configure, and it is definately more 
functional that I would have first expected.  The one thing that 
probably requires a few brain cycles in the authentication method. 
You'll need to decide whether users need to enter a password, or whether 
you will allow or deny access depending on their username, and similar 
issues.  (This is not particularly difficult, but it's one of the things 
I'd read up on if I wanted to do some prereading.)

Hope this helps.
