Re: is there a FAQ on the BEST MODEM

Hi Peter,

I have been using an external original SwannSmart modem (in nice
translucent blue :) )
for about 3 years now. This is the best modem I have ever used altough
they are
not cheap. The SwannSmart II modems on the market now have the same
external
design but I'm not sure about what has changed inside. The only problem
is that 
they set you back about $180 - $200. I'm sure that you could get one
cheaper but I don't
know where.

The SwannSmart modem in my opinion performs excellently on all line
conditions and 
because it is external works with all versions of Linux.
If you really want to get an internal and still want a SwannSmart II
be careful which one you choose as there is one based on a Conexent
chipset
and one based on a Lucent chipset. I'm not really sure but I assume the
Lucent
may be a winmodem which will not work with Linux.

Hope all of this helps.

Nathan Millhouse
