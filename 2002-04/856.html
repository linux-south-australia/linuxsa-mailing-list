RE: some basic help please

>Could you please give me some detailed help on what I need to do to get
>it working, short of setting up DNS,  if this is possible.

All you *should* need to do on the Windows PC is add an entry to the hosts
file with a hostname to match your linux box + its ip address.
(edit c:\windows\hosts, if it doesn't exist, create it in notepad, just make
sure it doesn't have a .txt extension after you save it)  

(You probably want to change your hostname on your linux box to match just
to keep it simple, run "netconfig" from the CL on your Redhat box, this 
will allow you to setup the hostname/ip etc)
