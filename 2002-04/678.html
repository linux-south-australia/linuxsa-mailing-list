> This is set in /etc/bashrc:
>
>     case $TERM in
> xterm*)
> if [ -e /etc/sysconfig/bash-prompt-xterm ]; then
> PROMPT_COMMAND=/etc/sysconfig/bash-prompt-xterm
> else
>     PROMPT_COMMAND='echo -ne
> "\033]0;${USER}@${HOSTNAME%%.*}:${PWD/$HOME/~}\007"'
> fi
>     ;;
>
>
> So, there is a default ...
>
> You can override it in /etc/profile.d with a file called bash_prompt.sh if
> you want to. Or, just put it in /etc/bash_profile.
