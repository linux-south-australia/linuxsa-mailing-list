RE: Title bar and Domain name

I believe the way this stuff works is by having your prompt set to include
some ansi escape sequences to set the xterm title bar.

I use the following in by bash_profile:

PS1='[\W]$ '
export PS1

if  [ "$TERM" = "xterm" ]; then
    export PROMPT_COMMAND=
    echo -ne "\033]0;@${HOSTNAME}\007"
fi


This is specifically because i DONT like having the title change as i move
around - i just want it to become @myHostname and stay that way.

The default on redhat is to have PS or something set it on every chdir
though.. so thats why i mentioned it might be in the bashrc or profiles
somewhere..
