COMMERCIAL: Linux course in SA 19-June-2000

Hi,

Because some interest has been expressed, we are looking to run another
Linux course in Adelaide on 19-June-2000.

More details about the course can be found at www.iit-training.com.au, but
briefly it is a 5-day Hands-on Linux System Admin course. 

It gets fairly intense at time, and all delegates get a PC. Each PC has two
ethernet cards in it, and current planning is that we will install two
different versions of Linux during the course. We also plan to have two
different video cards so delegates can get some experience at setting up X
on different cards.

If you have any interest, please send me mail, or contact IIT training on
1-800-654-103
