The GNUish port of GNU awk is probably the most compatible
with the one on your work Linux machine.

Home page: http://www.dms.auburn.edu/gnuish/

Mirror:    ftp://mirror.aarnet.edu.au/pub/simtelnet/gnu/gnuish/
           gawk304h.zip   GNU awk 3.0.4 manual in Windows hlp format
           gawk304s.zip   GNU awk 3.0.4 [src]
           gawk304x.zip   GNU awk 3.0.4 [exe]

Alternatively, you can always run Linux at home :-)
