PS. On the subejct of warnings about sending multiple
e-mails, that's not so far fetched.
With MS-DOS, if you typed "del *.*", it would ask
something like "This will delete all files. Is this okay."

Andyes, people often post to multiple people, but most often
if the number exceeds 5 or 10, they will use a distribution
list name - some sort of pre-defined destination grouping.

Therefore it would be practical to have a default question
sort of thing that asks "do you really want to send to N
recipients" when N > N_MAX.

But it would be much easier to just stop programs
initiated from e-mail having any ability to send e-mail at all.
In the integrated, innovative win32 environment, this should
not be so difficult.
