Re: VM Ware help

Footnote to the previous reply, if you have the latest version of VmWare, u dont
need to download the tools, its already provided.
Once you have the VM up, go to one of the menus (cant recall as I'm on a different
PC) and enable VMware Tools Install or something to that effect.
Its quite cute actually, it "inserts" a floppy in your virtual PC containing
the tools, usually a readme file will pop up on the next reboot, follow the
steps & bingo u will have your own VMware SVGA display adapter.
Dont try to install your own video drivers, just use the generic VMware ones.

You also get a couple other things like mouse grabbing when u move the mouse
over the window, stuff like that.

Have fun
