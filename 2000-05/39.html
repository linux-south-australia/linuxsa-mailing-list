Re: how do you stop site-downloaders?

Alan Kennington wrote:
> 
> From time to time, more and more frequently,
> people browsing my site decide to download the whole site,
> perhaps not realising that there are some 55 MB of files,
> about 1000 html pages and so forth.

Are you sure it is not a robot indexer?

Try adding a file

  http://www.example.com/robots.txt

containing

  User-agent: *
  Disallow: /

and see if the downloads continue.   If they stop, alter the
robots.txt file to allow as much indexing as seems appropiate
(maybe just the top-level page).

Ref:
  A Standard for Robot Exclusion
  http://info.webcrawler.com/mak/projects/robots/norobots.html
