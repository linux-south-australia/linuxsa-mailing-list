> I have finally powered up my BP6 dual celery mobo, with only one celery,
> and went to install RH6.2, only to find that it could not find the UDMA66
> drive that I have attached to the UDMA controller.
> 
> Does anyone know if Linux 2.2 can understand UDMA66 controllers/drives, or

There is a patch at http://www.highpoint-tech.com/ and other good info
at:

http://www.bp6.com/

a fuzzy memory, or FUD.
