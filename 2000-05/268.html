CD-R

Okay, recent threads got me inspired to get my CD-R working under
GNU/Linux. So, I downloaded and installed cdrecord-1.8.2. No worries.
Made sure that the CD-R (SCSI) is working - yup, it mounts CD-ROMs okay.
Found out the SCSI Bus, ID and LUN (0,5,0). I've configured
/etc/cdrecord.conf with the following lines:

  CDR_DEVICE=/dev/scd1:0,5,0
  CDR_SPEED=4
  CDR_FIFOSIZE=2m

(It's a 4x writer, with 2MB of Cache). So, now I should be ready to
record. But, when I say:

  [root@slave-i /root]# cdrecord -v -dummy file.iso

To test that everything is okay, I get:

  Cdrecord 1.8 (i686-pc-linux-gnu) Copyright (C) 1995-2000 Jörg
Schilling
  TOC Type: 1 = CD-ROM
  scsidev: '/dev/scd1:0,5,0'
  devname: '/dev/scd1'
  scsibus: 0 target: 5 lun: 0
  Using libscg version 'schily-0.1'
  atapi: 1
  cdrecord: Cannot do inquiry for CD/DVD-Recorder.
  cdrecord: Success. test unit ready: scsi sendcmd: fatal error
  CDB:  00 00 00 00 00 00
  cmd finished after 0.000s timeout 40s

Hmm, that's no good. So I try:

  [root@slave-i /root]# cdrecord -scanbus
  Cdrecord 1.8 (i686-pc-linux-gnu) Copyright (C) 1995-2000 Jörg
Schilling
  cdrecord: No such file or directory. Cannot open SCSI driver.
  cdrecord: For possible targets try 'cdrecord -scanbus'. Make sure you
are root.

Hmm. Anyone who's used cdrecord before able to point me in the right
direction?
