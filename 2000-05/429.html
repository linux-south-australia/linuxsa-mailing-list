RE: deleting files names "-bx00"

the usual way is `rm -- -bx00`

the '--' usually tells the program to interpret the rest of the arguments as
non-options... ie files... it's pretty common among Unix programs...

Note that this isn't a shell thing, as the program is getting passed the
'--' as an argument, just as it was getting '-bx00' as an argument...
