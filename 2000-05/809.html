Yes... it might be worth having some installation notes kicking around so that
the installees can follow what the installers are doing, and so that they can do
it again themselves if they have to.  But I think that a quick reference sheet
is more important - when they get home they'll be saying "how do I see a
directory listing again?"

> What do we install?

I can understand why you'd like to make this an "if it's open-source, we'll
install it" event, but I don't think that's realistic.  Installing multiple
distributions will increase the complexity of the situation by orders of
magnitude.  It'd be much easier if we just install one.  RedHat is a good one
for beginners.  And although my experience with Debian is not that flash, I imagine
that if you have to have someone install it for you then you probably shouldn't
be using it.
