VMWare

VMWare 2.0 comes with VMware Tools ready to install. 

Earlier versions required a slightly different install. 

One method was to download the file, unzip, and transfer to a floppy.
Boot the VM and load the file into VMWare. Follow instructions to
execute and install it. It may be necessary to go into Control Panel,
Display, Settings, Advanced, Adapter, and change the device by following
the prompts and pointing toward the new driver. 

The VM will then emulate the resolution of the Host O/S.
