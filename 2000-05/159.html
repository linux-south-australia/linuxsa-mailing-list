I may be wrong here, but if sendmail is set-up for the local domain
(sendmail.cf), you have two options.
1/ Use the /etc/alias. (i.e. fred:	alan,root), this way you do not need to
have a user called fred.
2/ I think there is a mail alias file that can be used for this in the newer
versions of sendmail..
