Mike Zbierski <stellaris@chariot.net.au> wrote:
> I was just wondering whether anyone knew when/if Samba is going to handle Windows2000
> authentication/logons?

I found the following in samba-2.0.5a:

$ cat Win2000_PlainPassword.reg
REGEDIT4

;Contributor:   Herb Lewis (herb@sgi.com)
;Updated:       16 July 1999
;Status:        Current
;
;Subject:       Registry file to enable plain text passwords in Windows 2000

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\LanmanWorkStation\Parameters]
"EnablePlainTextPassword"=dword:00000001
