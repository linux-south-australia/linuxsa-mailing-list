Re: USB Modems

I've been using a Netcomm Roadster USB for over 12 months now.
Havent had any problems with it at all.  The main reason I went for the USB
version was to cut down on interrupt usage under Windows.  (i had both my com
ports disabled in the BIOS to save  a couple IRQs).  And as u said the other
adv. is less power packs hanging around (my power board is bad enough as it
is!!)

As far as speed, I havent seen anything noticeable, as with all modem use, it
all comes down to line speed/quality.
I've just started using the same modem with Linux, tho had to send it back to
Netcomm for a USB firmware upgrade.  Have to say it still works great.

I havent had any probs with interference either, not sure if this would be an
issue with multiple USB devices or not.  I know someone using a USB hub with
modem, scanner & printer hanging off it, & they dont seem to have any problems.


HTH
