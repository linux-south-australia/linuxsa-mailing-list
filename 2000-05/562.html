Richard,

I have a BP6, and use the UDMA controller for access to the hard drives. I
was in a hurry, and just wanted to use it without installing any of the
kernel patches, which at the time (>6 months ago) where somewhat dubious.

By using this line:

append="ide2=0xd800,0xdc02,11 hde=1123,255,63 hdf=1123,255,63"

in your lilo.conf file, you should be able to install linux (and boot off)
any disks attached to the UDMA controller, although they will only be
running in ATA/UDMA-33 mode.

Obviously the C/H/S settings are unique to my drives, but I found they were
neccessary or I would have problems with the dreaded "LI" LILO lock. I use
the software RAID on my system and it seems to work fine with both drives
running off the UDMA66 controller. I use the UDMA33 controller for a CDROM,
tape etc.

Although this does not fully answer your question in that the drives are
only running in UDMA-33 mode, I hope it helps you at least see your drives
in the short term.
