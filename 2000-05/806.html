Re: Linux + laptop

Running Linux on both of these machines is possible. It 
will be quite slow, but we all knew that. Whatever you 
do, go minimal. Things like using sawmill instead of 
enlightenment, cutting the number of virtual consoles 
from 6 to say 3 or 4 and making sure that you only load 
the daemons/services that are absolutely necessary.

As for the laptop-specific stuff, check out the 
Linux+Laptop page (someone posted a message not too long 
ago that had a link, if not, use a search engine).

Hope it helps,
