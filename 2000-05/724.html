Benjamin Close <linux@senet.com.au> wrote:
>> There may also be a maximum limit ont he number of allocated sockets
>> on a system.
>
> The limit does exist and is the same on all machines. 65535 sockets. The
> first 1024 are reserved for programs run by root. I don't think you need
> to worry either :)

Flippant, but wrong.  You are talking about the number of ports available,
but don't forget that there's more than just TCP; remember UDP, ICMP, raw
sockets; and then unix domain sockets.  The total number of "ports", if
you will, is effectively uncountable.

I don't know how many sockets you can have.  Early SCO systems were limited
to 256, which then got expanded to 64K (via clone devices.)  Look at the
kernel if it matters.
