Re: Where do I put my scripts?

Hi Jim,
    If the scripts are system scripts then probably
/usr/local/bin, or create /sbin/local/bin and put these in
the appropriate path variables. However, I found that at
times I write useful scripts for my own use also that I
don't want everyone else to use and so I created a ~/bin and
put this in my personal path. Path variables are set in
~/.bash_profile and /etc/profile I think.
    Cheers!
        Andrew.
