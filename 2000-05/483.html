Re: Retrieving mail from an Exchange server

Don Collins wrote:

>     Just had a query from someone in Perth regarding mail on an M$
> Exchange server.  He's wanting to be able use Linux on the network but
> has to be able to retrieve local mail from the Exchange server.  He's
> not sure what they're using as the protocol, but the server is
> configured for SMTP, MS and X400.

Fetchmail can handle both POP3 and IMAP4, but the Exchange server may
not be set up to allow it.  This usually needs the relevant connector
installed, thus the SysAdmin must be amenable[1] to this.

Regards,
  Daryl Tester
