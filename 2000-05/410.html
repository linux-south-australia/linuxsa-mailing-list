I would say this would only be so for an ATX power supply. 

The difference between an AT and an ATX power supply (simply) is that a
motherboard can "talk" to an ATX power supply, and tell it to do things
like power off etc. WOL (wake on LAN - where you can ping a machine and
it will start up) is another function of ATX power supplies.

Old 486's would have AT power supplies.
