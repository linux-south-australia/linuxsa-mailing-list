Re: USB Modems

Hi Mike
There are actually quite a few USB modems working under linux right now, my
Netcomm Roadster II USB being one of them ;-)
(By working, I mean full comms functionality that I had under Windoze)
Check out the working devices link from the linux-usb site and have a look at
the Comms devices.
Admittedly I had to return my modem to Netcomm for a USB firmware upgrade, tho
one has to do this for W2K also.
Its been working great with the 2.3.99 backport to 2.2.14.
AFAIK the only requirement for a USB modem to work with the current kernel USB
code is that it conforms to the relevant CDC & ACM protocols/standards.

Regards
