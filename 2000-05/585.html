Re the calendaring bit, 
see http://members.xoom.com/_XMCM/mtsirkin/outlook/index.html.

   Outlook2Ical is a simple perl script that converts mail messages
   produced by MS Outlook Calendar to .calendar file used by the
   ical program. It handles appointment messages and task requests
   (with -task flag).

   ical is not compliant with any calendaring standards, but it is a 
   simple program that works for smallish numbers of people and a lot
   of people use it. I don't use it, nor do I particularly recommend it,
   but it might help some people.
