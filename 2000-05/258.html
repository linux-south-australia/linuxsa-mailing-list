RE: Data Accounting for a system running IP Masq

There's something called IPAC (I think), that will let you check how much
data was sent over a link, to specific IP addresses, over specific
protocolss, and perhaps a few other things.

I'm sure I found it on freshmeat.net, so have a look there if you want it. I
don't think it'll be able to tell you the origin of where the data is coming
from, unless you add a rule for every IP address that exists.
