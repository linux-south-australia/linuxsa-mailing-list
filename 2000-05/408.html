I've never done this before, so I guess you can assume that in a lot of
cases I know less than you. :) But I guess some extra information might
help:
1)	Do the other linux accounts you're trying from have the same
username/password for linux and netware?
2)	Would there be a necessary entry for /etc/fstab? If so, is the syntax
correct for allowing a normal user to mount the file system?
3)	Are there permissions set on ncpmount that would restric others from
eexecuting that program?

I've also made an assumption or two. The "admin" account was created by you
manually, and it isn't a superuser. You've probably set the admin password
as the same as the novell password. Is this correct?
