Use mh-e under emacs.  Able to run in a console or under X, excellent
MIME support (with the standard Debian install you can even click on
an attached Word document and mh-e opens the word document in wordview
for you), supports folders, mailing list digests and many other
things.

There is an O'Reily book availible for mh with an excellent section on
mh-e for emacs.  It is avaliable on the web for free.

If you use emacs normally, then you will be confortable with mh-e.
