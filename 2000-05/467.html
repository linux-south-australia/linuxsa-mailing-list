RE: tcp /ftp port disabled

* I just installed RedHat 6.2 and both the tcp /ftp port are disabled
* because of these i am not able to telnet  /ftp to this machine from remote
* host,Can you please let me know how to enable tcp/ftp port,
*
* Please let me know if you have any querys in this mail,

Make sure that the telnet and ftp entries in your /etc/inetd.conf file don't
have # characters at the beginning of the line. An example of the inetd.conf
file is as follows:

ftp stream  tcp nowait  root    /usr/sbin/tcpd  in.ftpd -l -a
telnet  stream  tcp     nowait  root    /usr/sbin/tcpd  in.telnetd

Also, make sure that the /etc/hosts.allow file has the IP of the machine
that you're trying to connect to. An example of that is:

ALL: 10.1.1.1
telnet: 10.1.1.1/24
ftp: 10.1.1.1/255.255.255.0

syntax correct for hosts.allow...?
