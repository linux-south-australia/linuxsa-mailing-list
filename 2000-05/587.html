Re: how to install linux

On Fri, 19 May 2000, anang harianto wrote:

> can you show me and tell me how to install linux redhat 4?

The simplest way is to print off the installation
guide (on the Red Hat website) and follow the
instructions carefully.

If you get stuck then e-mail this list.  If you get
really stuck bring your machine to the next LinuxSA
meeting.

Install Red Hat Linux version 6.2 rather than version 4.
Version 4 is far too old to be secure and 6.2 has much
better support for recent PC hardware.  If you are short
of cash, then present yourself at L7, 10 Pulteney St,
Adelaide and we'll loan you the CD for three days.  Walnut
Creek sends us CDs for free as long as we loan them to
members of the public as well as to university staff and
students.

If you have a CD burner, you can download the CD image from
http://mirror.aarnet.edu.au/ and burn your own CD.  If you
have an ethernet connection to the Internet, then you can
use the HTTP install option to install directly off the
AARNet mirror site.
