In regards to SAMBA this sounds like you need the fix to make windows send
the password in plain text. There is a doco (somewhere around /usr/doc)
which will detail the change to be made to the windows box, otherwise there
is a link to goto to get the .reg file which does this as well.

http://us1.samba.org/samba/ftp/docs/textdocs/Win95.txt This link is about
Win95 but may help.
