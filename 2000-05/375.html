Re: Gnome and the Welcome message in GDM or XDM or whatever

On Sat, 13 May 2000, Richard Sharpe wrote:

> When you use Gnome, and configure the system in run level 5, GDM, or XDM,
> or whatever, comes up with the logon dialog window, and it says Welcome, etc.
> 
> How can one change that message to include the usual draconian message
> about jail and hard labour and no drugs to ward off evil hackers? :-) :-)
> 
> That is, what file is the equivalent of /etc/issue and /etc/issue.net?

For a picture see "Logo=" in /etc/X11/gdm/gdm.conf. Currently it should
be the Gnome foot.

For the background as a whole see /etc/X11/gdm/Init/Default, in particular
the xsetroot and xsri commands. xsri can be used to put up an image with
anything you like on it. You can also run a screensaver here (usually with
-root) which of course can display messages in all sorts of whacko ways.
If you do run a screensaver then you have to remember to turn it off
before the X session starts. There's a special directory for scripts to
run (a bit like init scripts).
