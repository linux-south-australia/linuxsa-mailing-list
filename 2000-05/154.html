Re: Mailiases

You should be able to do it using a .forward file in the users home
directory. They can turn it on by creating the file, and turn it off by
deleting it (or moving it somewhere else).

Syntax is, I believe:

----------------------------------------

account@other.place, \local_username

----------------------------------------
