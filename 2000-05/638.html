I believe you can set up squid as a transparent proxy 
too, which would allow you to use the proxy without 
needing to change the workstation configuration.

We are running a single squid proxy (which also acts as  
a DNS, DHCP, router, firewall, test NT PDC and 
Fileserver) on a machine that MS Proxy server would not 
enjoy.

Your boss should know that most proxy servers in the 
world (I think it\'s around 75%) are running squid. I\'m 
sure that many of these are Linux based. I would also be 
happy for her (or anyone else) to contact me for 
specific answers to specific questions.
