Re: Getting XFree86 into 16-bit or 24-bit mode

On Sun, May 14, 2000 at 12:27:48AM +0930, Sam Silvester wrote:

 > In the file /etc/X11/XF86Config look for the appropriate screen section
 > and add the line "Depth xx" where xx is either 16 or 24.
    [ ... ]
 >     Subsection "Display"
 > 	Depth	16
 > 	Modes	"800x600" "1024x768"
 >     EndSubsection

It's a bit more complicated than that.  What the clause above is saying
is basically, "If the color depth is 16, permit modes 800x600 and 1024x768".
It isn't actually saying, "Use color depth 16."

You can start the X server with the "-bpp 16" parameter to manually set
16-bit colordepth.  Alternatively, if you always want to use 16 bit
colordepth you can change the default in XF86Config:

  Section "Screen"
      [ ... ]
      DefaultColorDepth 16
      Subsection "Display"
	  Depth 8    # define modes when we start with -bpp 8
          Modes       "1280x1024" "1152x864" "1024x768" "800x600" "640x480"
      EndSubsection
      Subsection "Display"
	  Depth 16   # Define modes for default color depth
	  Modes      "1600x1280" "1280x1024" "1152x864"
      EndSubsection
  EndSection


Hope this helps,
