Re: How can I get the Ip address of server's external IP address

> Now, I would like one of the internal server to get the ip address of the
router's exterenal interface.
> Are there any c APIs/linux utilities that can be used to get the external
ip address of the router, using
> the http connection to it through its internal address.

    I use something like this -:

$IF0="ppp0";
$IP_IS= `/sbin/ifconfig $IF0 | grep inet | cut -d : -f 2 | cut -d \\  -f 1`;

    to read the IP Address from ifconfig, and then using LWP I send the
contents of $IP_IS to the web server of the target machine.

    That all you need?
