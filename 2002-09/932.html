> I think I should just wait for Mandrake 9.0 - maybe that has better
> configuration tools for firewall / proxy and caching name server..

Ok, what is wrong with:

* a few HOWTOs
* a few BEERs
* vi
* a Mandrake XYZ distribution

???

You will, trust me, learn a lot more by doing all of this by hand and in
text. Unless you've been to a well designed and presented course, and
actually understood it, you won't learn the concepts you are looking for
simply by knowing what a few point-and-click buttons do.

I don't want to sound harsh, but consider that most people whom I know
who setup firewalls as their living generally don't install nice GUI's
to administer them. Why?

* bloat
* X can be seen as a security issue

You might want to start out with the "Iptables HOWTO" (whose URL again I
forget, but it's been mentioned on fresheat and I'm sure it's part of
http://www.tldp.org/ -- http://www.linuxdoc.org/ if I've stuffed the
first address up). I'm almost certain that will be a little confusing
because it's more of a reference than a tutorial but it will get you
asking questions.
