Re: Simultaneously window manager sessions

Actually, I don't see why you couldn't get another XDM running by
cloning and fiddling with this line from inittab:

# Run xdm in runlevel 5
# xdm is now a separate service
x:5:respawn:/etc/X11/prefdm -nodaemon

...prefdm is simply a wrapper script to choose GDM, KDM or XDM.
