Red Hat 8.0 out!

Red Hat 8.0 is out!  Well, out on rhn - mirrors syncing as we speak :)

New features include:
  LVM on installation
  Personal Desktop option available on install - including 
    unified Gnome/KDE look'n'feel (controversial)
  Gnome 2.0.x
  KDE 3.0.3
  GCC 3.2 (which includes C99 support)
  Perl 5.8
  Apache 2.0
  a whole bunch of new Gnome-based user friendly system config tools
  etc etc etc

Woohoo!
