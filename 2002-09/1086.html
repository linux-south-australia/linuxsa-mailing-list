Re: USB flash drive

Do they work? The answer is - it all depends!

They are presented as scsi disks. So you need scsi support as well as
usb turned on in the kernel. In particular you need the usb-storage
module, which basically works as a kernel interface between the USB and
SCSI subsystems. You also need the hotplug scripts available if you want
the device auto-detected.

There are a number of companies producing USB flash drive's. The Apacer
ones are supposed to work with Linux. I tried one from Transend the
other day which almost works ie it doesn't work.

Hotplug see's it, usb-storage recognises it and generates a scsi device
at the proc fs level:

Host: scsi0 Channel: 00 Id: 00 Lun: 00
  Vendor: GENERIC  Model: USB Mass Storage Rev: 1.00
  Type:   Direct-Access                    ANSI SCSI revision: 02

But sadly /dev/sda doen't seem to exist (can't open). Which I guess
means the scsi driver can't talk at the device (using a 2.4.19 kernel).
