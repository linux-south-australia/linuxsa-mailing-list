Re: Simultaneously window manager sessions

On Wed, Sep 18, 2002 at 01:44:09PM +1000, Mcleod, Ian wrote:

 > For the benefit of us all - would you mind explaining how to allocate RAM to
 > a process?  That sounds like a good idea - throw it at Mozilla, X, etc - the
 > ones that can really benefit.  I never considered that.
 
Mozilla has a field in Edit -> Preferences... -> Advanced -> Cache which
tells it how much memory to use.

UNIX-style VM systems give processes as much memory as they ask for,
so you'd get negligible benefit from "reserving" memory for a process.
