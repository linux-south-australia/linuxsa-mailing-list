Re: Ghost

If you can remember:

1) the setting for initrd
2) the partition for root
3) the name of the kernel

...then on a RedHat 5.X system you can do:

linux initrd=[path to initrd] root=[path to the root partition] 

...and add:

linux initrd=[path to initrd or blank] root=[path to the root partition]
init=/bin/sh

...if you can't get init to behave itself.

That can be quite useful. RedHat 5.2 also came with a rescue mode.
