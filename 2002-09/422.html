RE: Mouse configuration problems

Press Shift + Num Lock to activate the mousekeys

arrow keys (2,4,6, and 8) move the pointer
5 behaves like the default pointer button
0 locks the default pointer button for dragging
. unlocks the default pointer button to release a drag
+ double clicks
/ sets the default button to button1
* sets the default button to button2
- stes the default button to button3

if shift + numlock doesn't work try alt+shift+numlock
