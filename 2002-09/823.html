I read the ypinit man page and it reads (relevant parts quoted) :-

SYNOPSIS
     ypinit -m [domainname]

OPTIONS
     Ypinit supports the following options:

     -m [domainname]
             Set up a master server.  By default, the script sets up a server
             for the system default domain.  The user can override this
             default by specifying domainname explicitly.  Maps are con-
             structed from scratch using local files as templates using the
             yp_mkdb(8) command.
</quote>
