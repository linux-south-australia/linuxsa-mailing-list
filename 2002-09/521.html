Please note that you're not even getting to the point where
you can authenticate - this is expecting a 'CONNECT' string
from the modem after it's dialed out, and it's not getting
it (hence the alarm - it's timed out).  Either the modem is
failing to connect to the ISP, or the modem hasn't been set
to give verbose responses to commands (although it did get
an OK response to the ATZ, so that's probably out the
window).
