Re: (MacOSX Browsers)

Have many of you guys running MacOSX tried out Chimera 
<http://www.mozilla.org/projects/chimera/>;?

It's a Gecko-based, native browser for X. Effectively a Galeon for 
MacOSX. It has all the advantages of being a Gecko-based browser 
(excellent rendering, standards support, tabs, plugins, etc) and is a 
native app so it looks and feels like every other real X app.

I'm just curious to see if anyone on the list uses it, how stable/good 
they find it, etc, etc.
