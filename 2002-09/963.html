RE: best firewall - proxy software / reporting tools?

Monday, September 23, 2002, 8:57:18 AM, Ian wrote:

> I want to set up a dedicated firewall / proxy - cache on an old machine to
> run via wireless to my laptop.

> Any ideas on the best way to go about this?  What is Smoothwall like?  Can
> it produce pretty graphs and lots of statistics?  I want to take the
> opportunity to learn more about networking.  I know Mandrake firewall can
> produce a lot of graphs and statistics and looks quite good but it is also
> quite dated.

I use OpenBSD+ipf for my firewall machine and SuSE 7.2 with Squid
2.5.4BETA (hand compiled) as my cache.

> Does anyone have any ideas for good reporting software for Squid?  I have
> Calimaris installed and this works great (if it were not for Squid emptying
> my access.log upon reboot) but it is all in text - how can I produce a
> graphical representation of all this?

If you want really pretty reports, and are not averse to Winblows,
WebTrends will report on Squid logs.

Otherwise, configure snmp and use mrtg or rrdtool to generate pretty
graphs.
