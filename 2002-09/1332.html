Apache, PHP and MySQL will all run rather slowly on that system but they should
work. You'd need to checkout all the documentation for Apache and work out how
to stop it from taking up too many resources. For example, I'd only spawn two
servers on that system NOT the normal 8 [from memory].

There are ways to stop MySQL from using as much memory for its buffers. The
MySQL tuning section of the MySQL manual will be more help here.
