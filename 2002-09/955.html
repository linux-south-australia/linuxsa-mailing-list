Kendall, I worked out a solution for the problem that I was having at 
startup.  

If I first:  cd ~/cxoffice/support/dotwine/fake_windows/Program 
Files/Macromedia/Dreamweaver MX> and then executed the program from 
within that directory with this comand, /home/james/cxoffice/bin/wine 
Dreamweaver.exe, it works.

If I don't cd to the directory, it doesn't work.

I also tried it under cxoffice 1.2, works the same as 1.1.

So,  what I did:

1. downloaded cxoffice 1.2 from codeweavers
2. Installed Internet Explorer 5.5
3. Downloaded Dreamweaver 6 from dreamweaver.com
4. Started the program by following the steps outlined above.
