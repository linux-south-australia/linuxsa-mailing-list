Re: help with grep

On Sun, 29 Sep 2002, dcd wrote:

> Hi,
> The Problem
>     Using mandrake 9.0 I have an unusual PATH setting and I am 
> attempting to find where it has been set
> 
> So I thought this would be an opportunity to learn a little grep
> 
> The path variable $PATH has /usr//bin as it's first setting
> 
> so I tried
> grep -iR /usr//bin /etc/*
> or
> grep -iR /usr//bin /etc
> or
> grep - iR /usr//bin /  (This one ran out of Memory)

What you want is the Swiss Army Knife of commands: find.

find / -type f -exec grep -l /usr//bin {} \;
