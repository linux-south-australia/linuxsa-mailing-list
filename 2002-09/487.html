Re: Internet Usage Stats

On Fri, Sep 13, 2002 at 11:05:58AM +0930, Adam Smith wrote:

 > > If your ADSL router is a Cisco, you will probably be able to use 
 > > Netflow, which will deliver exactly what you're asking about.
 > 
 > I'd assume that although my FreeBSD system running NAT is acting as a
 > 'pass-through' for NAT requests, this wouldn't affect reporting, because
 > they're only pass-throughs, right??
 
It doesn't matter, but not for the reason you're thinking of.

Netflow runs on the router, which is outside your NAT gateway.  So
it can see the various source/destination IP-address and port 
combinations.

The collector software you'd run on the FreeBSD box merely receives
netflow accounting records (via UDP) from the router and stores them
somewhere (database, flat-file, blah, blah).  

You can then use various tools on the archived netflow accounting
data to perform traffic pattern analysis, getting answers to questions
like, "How many bytes have I transferred from source-port 80".
