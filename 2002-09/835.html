Re: How to install an additional harddrive in Linux

There is a file called /etc/fstab that stores information about your 
hard disks. If an entry is not in that file for a hard disk, I think 
that you can't mount it.

You also need to decide where to mount the drive and create a directory 
that will act as a mount point.

Hope that gets you started.

Ask for some more opinions though....because I wouldn't want to stir you 
wrong.

James Leone
