[OT] Uni SA hack attempt

FYI

> -----Original Message-----
> From:	Stephen Stone 
> Sent:	Friday, 20 September 2002 3:59 PM
> To:	UniSA-All Staff
> Subject:	Important information for staff who connect to the University
> from the Internet
> Importance:	High
> 
> This message is intended for staff who either
> *	use an Internet Service Provider to connect to the University or
> *	use a workplace or an outside institution's Internet connection to
> connect to the University
> 
> On Friday 20, September a hacker made a concerted attempt to guess a number
> of passwords for accounts on a wide range of Windows based servers. As a
> result of this attack ISTS has had to take steps to block access to shared
> resources and services from the Internet to ensure that this type of
> disruption does not continue.  These blocks will have the following effect
> for staff who fit the criteria described above.
> 
> It will not be possible to connect to any of the following from outside the
> University's network:
> *	a shared file system on a University workstation or server, using
> Windows networking
> *	a linked drive letter to a University server or workstation
> *	a print queue on a University server using Windows networking
> *	browsing the University's Windows network and workgroups
> *	the full Outlook email client to access an Exchange mailbox
> 
> The following access is still available from outside the University's
> network:
> *	access to University based web sites, including password protected web
> sites
> *	access to the web interface for email  at http://www.mail.unisa.edu.au
> *	other lesser known services such as FTP, TELNET and SSH
> 
> Please note that none of this applies to staff who dial up via the
> University's Enhanced Dialup facility.
> 
> At this stage there is no practical solution to this problem however if these
> changes cause you significant problems, please contact the IT Help Desk on
> 8302 5000.
>
