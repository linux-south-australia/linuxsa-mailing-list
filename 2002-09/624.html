Aw,

The line "localhost login: " is display on the screen by login... The
localhost is set (sometimes only untill you reboot) by 
"hostname newhostname" (as root). The word login:, i beleive is
hard-coded into login it self. I just did a grep for the string
"login:" and came up with this result in login.c 

#define DEFAULT_PROMPT          "login: "

If you are wanting to display mycompany login: on all , you would need
to modify the login source code. I had a quick look thorugh the source
and couldn't find it... 

But really we are second guessing what you are trying to do.. :)
