As Anthony Symons said, the power output isn't everything.

On a vaugely related note, there was an article on ./ a few months ago 
about a hack for a generic brand of wireless cards which upped the 
card's power output from 30mW to 100mW, or somthing like that. The end 
result however, after someone took the time to analyse exactlt what was 
going on, wasn't a better signal, but one which performed pretty much 
the same but also bled large amounts of noise into neighbouring 
frequencies. This of course would cause a lot of interference for anyone 
using those frequencies.

A much better idea is to use better antennas. Using point-to-point 
directional antennas you should be able to get excellent ranges (at 
least 2-3kms) without needing any amps.
