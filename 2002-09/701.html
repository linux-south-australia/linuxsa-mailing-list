RE: ftp with tcp wrappers

Aw wrote:

> How do we Deny the user  abc access to ftp with tcp wrappers

Short answer: you can't.  tcp wrappers doesn't know about the
underlying ftp protocol to determine who the user is logging
in as.  Choose an FTP server that allows you this sort of
control, and configure it appropriately.
