RE: Alpha installation

Hi Sam,

We have a few AlphaServers of varing ages about, ones of interest are an
A/S-800 and a A/S-1200.
On both of these machine there is a button [in or out] which needs to be
depressed before powerup in order to get access to the SRM console. (almost
identical to the STOP-A with SPARC's)

Note: you should expect output such as "polling dra.12.1.000.0" or similar
before the SRM console appears, I've had them take up to 15 minutes before
the SRM console prompts.

Following is a link to DEC/compaqs/HP's documentation on the AS-1000

http://www.compaq.com/alphaserver/archive/1000/

Regards Dale Butterworth
