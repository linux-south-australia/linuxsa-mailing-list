Re: CDW and redhat 7.3

If its currently using the ide-cd module, I dont think you can swap to
ide-scsi on the fly. You need to use a kernel option like (in grub)

title linux-2.4.18
kernel (hd0,1)/boot/bzImage-2.4.18 root=/dev/hda2  devfs=mount
hdd=ide-scsi

Damn that wrapped. Obviously its a 1 liner.

When you boot it'll then use ide-scsi properly and you should be able to
burn.
