The Westpac site partially works with Mozilla (1.1), but not perfectly.

The menu on the left hand side doesn't show the pop up sub-menus, like on
IE, and the table used to format the account information doesn't resize
with the browser window, like on IE, so if you have a smallish display or
don't have the browser window maximised, you need to scroll to the right
to see the transaction amounts and totals.

It's a bit annoying, but you can still get your Westpac banking done over
the web with Mozilla.
