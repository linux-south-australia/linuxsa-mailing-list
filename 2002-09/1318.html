Re: ICEauthority

On Mon, 2002-09-30 at 12:17, Wilkinson,Alex wrote:
> Can anyone tell me what the $HOME/.ICEauthority file is used for ?
> 
> I have searched extensively to find out vaguely that it has something to do with
> gnome,kde,X and .Xauthority.
> 
> I have found nothing concrete though.

X authorization records are stored and removed removed from the
~/.ICEauthority file

When you start an X session a new record is appended to it.  If you have
a crash, stale records can hang around ie authorization records are not
removed, and if you restart a new session, there could be a chance  that
you would use the same socket id and be denied access
