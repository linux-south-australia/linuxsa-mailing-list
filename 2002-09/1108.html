Hi,
     VideoLan, The main implementation is for video streaming and
therefor takes a variety of inputs, including DVD and TV card, decodes
and encodes on the fly and outputs it for streaming.
Therefor can be used as a player, output to x or converter output to
file or network streaming.
     There were howto's available on the site for the main usage and
links to howto's for some of it's other uses.
Until xine improved I used to use this for dvd playing on linux.

Main Site is www.videolan.com
Mandrake files are available from ftp://zarb.plf.org/

Iv'e started using arson for burning, includes tools for zone unlocking
cdplayer and appears though I havent used them to have tools for
vcd/svcd burning included, uses cdrdao and cdrecord backends
