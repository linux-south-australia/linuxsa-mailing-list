Re: Internet Usage Stats

On Fri, Sep 13, 2002 at 10:36:22AM +0930, Adam Smith wrote:

 > I have an ADSL router at the edge of my LAN, and a FreeBSD system
 > running natd.  I'd love to be able to view a port-by-port analysis,
 > which I could look at and would be able to work out for example "In the
 > last month, we did 5Gb of TCP traffic over port 80, but we did 3GB on
 > port 1214 [KaZaA or something.]"  Then it could do a host-by-host
 > analysis, or whatever is required.
 
If your ADSL router is a Cisco, you will probably be able to use 
Netflow, which will deliver exactly what you're asking about.

You'll need to run a Netflow collector on your FreeBSD system;
Check /usr/ports/net/flow-tools/ for a basic set of command line
tools, or do a Google search for "Netramet" for something a bit
more full-on.
