Re: Where does apache-1.3.26 keep its html files by default?

Hi Damar

In the /usr/local/appache/conf directory there would be a file called 
'httpd.conf'

It contains the directive 'DocumentRoot'

This is where the "webspace" is defined
In your case this would probably be
/usr/local/apache/htdocs

kind regards,

Luke
