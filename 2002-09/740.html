RE: Memory use / application optimisation

> I think using Squid and a caching name server is a good idea 
>- would this cut down bandwidth use by much on a ADSL connection?

It will cut down the bandwidth usage, but by how much depends on how many
sites you access repeatedly.

We've been running a squid proxy here for around 2 years, on average we
save about 15% of traffic, this is across a PC base of 80 or so.  (For less
users you would expect the percentage to increase as the number of different
sites accessed decreases.)

(and its still running squid v2.2.x as I've been too slack to u/g it to 2.4,
I'm uncertain whether there are any improvements in the caching department
with
current versions, someone may be able to shed some light on that....)
