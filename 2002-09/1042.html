Most of the students at TV-TAFE wouldn't bother about attempting to
crack the systems, but would rather study towards their course or play
FreeCiv because the materials being presented was too easy. Steve
Donaldson has already pointed out getting too strict with security could
impact learning, or "leaning" as he put it.

We need to remember that a system is NOT secure if it's NOT useable. If
it's so SECURE it's UNUSABLE then people WILL find a way to use the
system which often exposes them to more security flaws. If they can't
find a way around the security then your system is UNUSABLE which is a
wholly different concept to NOT SECURE.
