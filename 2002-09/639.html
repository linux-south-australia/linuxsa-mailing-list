Ok, looks pretty much like a stock GNOME 1.4 install (I don't have a
debian install handy - not until I get a bigger hard drive on my laptop
:-P

The software for configuring the sound properties is "sound-properties"
(to find this out right click on the menu item and select 'Properties' -
on Red Hat 7.3 it's found at GNOME Foot / Programs / Settings /
Multimedia / Sound).

strings `which sound-properties` tells me that the gconf location for
enabling sound events is "/sound/system/settings/event_sounds=true".

So looking in ~/.gnome/sound/system you'll find an ini file with a
"settings" section which has 2 fields - start_esd and event_sounds.  The
second one is the one you want to change to "false".

I don't have 1.4 source lying round, so give that a try and see how you
go.
