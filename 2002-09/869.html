It deals with it quite well actually,

It will download and install a second set of libraries 
I.E. ncurses 2.0.1 and ncurses 2.1.3 will be installed

When a "Make world" is performed all programs are recompiled for the Latest
libraries installed (if possible).
Then there is a clean command with will remove all unused library's.
