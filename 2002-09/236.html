Re: Installfest 2002

Martin,

> I'm willing to be persuaded otherwise but at present believe I want to:
> Run Win98, Mandrake 8.2, & someother Linux OS like Redhat (suggestions
> welcome).

Don't run RedHat if you're running Mandrake. They're very similar and if
your aim is to have a diverse experience I'd suggest you run Mandrake
for a few months and then tackle Slackware or Debian.

> I need to know if there is an optimal installation order so LILO or GRUB
> recognises all OS for a multiple boot.

Whatever you do install Windows 98 first. The Linux'es don't matter.

> I have an internal ATAPI/IDE Zip (100Mb) drive

One can generallly get them working. What brand is it?

> a cheaper KODAK camera which
> requires USB

I have a cheap Plustek Opticam which requires USB. In your spare time
you might want to find out what model it is and search on google for
"usb [model number and details] linux" and see what turns up ;-P 

> an Epson ~720 (colour) printer (USB or parallel option)

I'd hazard a guess that that will work a treat. Mandrake uses a printing
system called CUPS (forget the URL, look for CUPS at
http://freshmeat.net/) and I'm certain I know at least one person who's
gotten full colour working.

> I was also planning
> to buy a Liteon CD-RW.

Linux will normally recognise most modern CD-RW's - but I'd advise you
to try and buy a quality one rather than a cheap one depending on the
number of CDs you want to burn.
