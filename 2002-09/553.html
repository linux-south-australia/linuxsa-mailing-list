Hi David,

not really to do with curriculum, more to do with where the site is to be
hosted. The host organisation, also the project sponsoring body, run their
site on XML therefore we were given a template and a tool to help build an
XSL stylesheet for the site.

I rejected their suggestion and put the responsibility back onto them to
find a way around the problem. There isn't any way I can take these newbies
to HTML and swing them over to creating the site in XML, and using XSL
stylesheets, and still come close to the anticipated October 30 launch of
the web site.
