samba again

Hi all

More questions

In Network Neighbourhood on Cliff's computer if you click on the Samba share 
it asks for the password for IPC$, and therefore can't access the printer 
share either.

- I have made cliff, robyn and root Samba users and set their passwords. 
- I used the Win98....reg file to set plain passwords on Cliff's computer.

Following is my smb.conf

[global]
        workgroup = STARFLEET
        server string = Samba Server
        log file = /var/log/samba/log.%m
        guest account = guest
        password level = 4
        security = share

[homes]
        comment = %U's Home Directory
        browseable = no
        read only = no
        writable = yes
 
[robyn]
        comment = Robyn's Home Directory
        browseable = yes
        path = /home/robyn
        writable = yes
        read only = no
        valid users = robyn, cliff
 
[cliff]
        comment = Cliff's Home Directory
        browseable = yes
        path = /home/cliff
        writable = yes
        read only = no
        valid users = cliff, robyn
 
[printers]
        comment = %p Printer
        path = /var/spool/samba
        printable = yes
