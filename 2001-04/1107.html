Re: Partion size for 40 gig drive

Hi Danny,

First off you need a partition for Win98 and then install it, make the 
partition yourself or Win98 will use the whole drive and make an extended 
partition using the rest of the drive. Linux will reside on smaller 
partitions inside the extended partition.

Win98 first, Linux second allows Linux to write to the boot record and set up 
its dual boot stuff.

When you start your Linux install it will ask you about partitions, you need 
a '/boot' partition, a '/' partition, and maybe a '/home' and '/usr'. The 
reason for so many partition is if (heaven forbid) you have a crash you'll 
hopefully be able to resurrect some of your stuff. I didn't have separate 
partitions and a huge crash took out everything.

Make a boot disk on installation as you may need to boot from it and change 
the line in lilo.conf from 'linear' to 'lba32', if your system doesn't 
recognise the larger drive size.

XFree 4.x supports the usb mouse but you may need kernel 2.4 for the ATA100. 
Let me know if you need any help with 'X'.

Robyn
