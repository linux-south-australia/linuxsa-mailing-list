Re: colours

Hi All,

I decided to track it down:
~/.bash_profile 
which calls
~/.bashrc 
which calls
/etc/bashrc
which then calls all .sh scripts located in 
/etc/profile.d/
there is a file called colorls.sh,
rename this to colorls.sh.bak and you will no longer have color aliased
to your ls.

Now if you just wish to change the colour and not totally disable it,
yes, /etc/DIR_COLORS looks like the correct place.

Personally I just disable them all together as I often dont have access
to a Terminal that supports the colours any how.

Ben.
