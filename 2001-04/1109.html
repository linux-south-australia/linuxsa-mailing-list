> I have a new 40gig WD 7200rpm ATA/100 which I need to install a WIN98
> platform and also a LINUX System aswell..
> I have Mandrake 7.2, will this support the ATA/100 standard ?
> Does Mandrake 7.2 have support for a USB MS intelli/optical mouse?

You'll need to add a few switches to your install prompt when installing 
Mandrake 7.2 so it will recognize the ATA100 device.  This is the howto 
which I followed to install 7.2 on my A7V mainboard.

	http://www.geocities.com/ender7007/

My advice would be, if possible, to install Mandrake 8, as it uses 
kernel 2.4 by default, should solve the detection of the ATA100,
it will also help with detection of the mouse, tho
Mdk 7.2 has no problems with usb mice generally.

 
> I Use this setup for games as well, which is why I need a 
> dual boot drive...

Depends which games :-)
I also have a Win98 partition for games, but I find that Q3, SOF, Descent 3
all run as well on Linux.  Nvidia 3d drivers are simple to install for your 
card.  Make sure you install Xfree86 4.0.x tho (again would recommend Mdk 8
for that purpose), download & install the *.tar.gz drivers, not the RPMS.

http://www.nvidia.com/Products/Drivers.nsf/Linux.html

see the Readme file for install instructions, but summarised:

run from a console (ctrl-alt-f1)

untar the nvidia kernel module, run "make install" from the directory
created
ditto for the GLX module

modify /etc/X11/XF86Config-4
	change the Device "nv" to Device "nvidia"
	add Load "glx" in the Load Modules section

restart your X server (ctrl-alt-backspace)
