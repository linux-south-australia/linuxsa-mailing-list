> Can I somehow say start the ppp0 script from 10 am and stop it at 2pm
> and then the ppp1 script to start at 7pm and stop at 7 am?
> 

Sure you can. Depends a bit on the distribution, but in RedHat there is
/usr/sbin/usernetctl, so your personal or system crontab could look like
this:

# Handle ppp Connect Schedule
00 10 * * * /usr/sbin/usernetctl ifcfg-ppp0 up
00 14 * * * /usr/sbin/usernetctl ifcfg-ppp0 down
00 19 * * * /usr/sbin/usernetctl ifcfg-ppp1 up
00 07 * * * /usr/sbin/usernetctl ifcfg-ppp1 down

I'm sure there are similar commands available if you have some other
distibution.

I usually put the interface control stuff in a shell script. (saves typing
later), so I need only type or have my crontab execute a single command to
bring up the link.
