The  trick is to not login to a Windows NT domain, but tell Win2k it's
a  workgroup. Then, under:

Control  Panel  ->  User  and  Passwords -> Advanced [Tab]
 -> Advanced [button] -> Users [folder]

 Right click on the user, go to Properties
 Click  on  the Profile tab and set the profile path to where you want
 your profile and the Logon Script to the name of your logon script.
 You can also set your Home Folder here too.

 Below is a copy of the very simple Logon.cmd logon script I use:

@Echo Off
net time \\eric /set /y
net use H: \\loki\homes
net use I: \\loki\share
net use S: \\eric\share
net use W: \\loki\www 

Which basically sets the time off of my Samba server, eric
and maps some drive shares to eric and the dev server loki.

I can provide a copy of my smb.conf file if needed, but I think you've
probably got that bit worked out already.
