Re: Mailing list software

I've been using mailman <http://www.gnu.org/software/mailman/mailman.html>;
for almost two years now with no problems.  I found the install easier
than Majordomo (not exactly sure why I had problems with that).  Its got a
fairly intuitive web interface, although I've never bothered to look at
the mailing options (eg, creating a new list via an email).

-corey

> Personally, I use majordomo for my mailing lists. Web based
