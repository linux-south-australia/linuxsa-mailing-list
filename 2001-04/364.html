Re: Mailing list software

On 13 Apr 2001 16:50:46 +0930, Dieter Jansen wrote:

> What's the concensus on the best modern, free list management
> software?  Preferably with integrated web archives and admin
> interfaces.  It's for a small list of only 30 - 40 people, so
> the implementation language is not critical (perl is fine).

Personally, I use majordomo for my mailing lists. Web based
administration can be achieved through Webmin and tools such as
Majorcool. As far as archives go, you're really on your own. There are
tools available such as wmf (Web Mail Folder) as used by LinuxSA which
provide the archives.

Also, consider using GNU Mailman. It's written in Python and has all the
web based administration bits and more. Sourceforge use it for the
hundreds (probably thousands) of mailing lists that they host. I think
Sourceforge use something called Geocrawler to to the web archive of the
list.
