There are some other sources:

http://minnie.cs.adfa.oz.au/Unix_History/index.html describes the
evolution in nauseating but somewhat uneven detail.  Further input is
solicited (contact Warren Toomey <wkt@cs.adfa.edu.au>).

The file /usr/share/misc/bsd-family-tree on any FreeBSD system shows
pretty much the same thing as David mentioned (from "The Design and
Implementation of the 4.4BSD UNIX Operating System") with the
exception of commercial AT&T UNIX (System III, System V).  I'm
attaching it.  Note particularly the break in the Research UNIX family
tree: all current UNIX versions can trace their ancestry back to 3BSD.

http://perso.wanadoo.fr/levenez/unix/ has one of the biggest displays
of the history, currently 13 pages.
