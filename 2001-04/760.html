>> The card isn't working. The card is a 3Com 3c905-tx. Windows reports it as

That card is a PCI card. You should not need to add anything to
/etc/modules.conf, and indeed, I think that if you modified conf.modules,
it has been ignored, since I think Mandrake 7.2 uses modules.conf first.
 
>> The module loads (/proc/modules), and won't unload either, and the lovely 
>>little green OK appears, but ifconfig only shows the loopback. messages has 
>>the message "ifup:Network is down" (or similar) (it may have something else 
>>too, but I can't remember if or what).

Well, what does 'ifconfig -a' show. If the module will not unload, that
would tend to indicate that it is in use. What does 'lsmod' list as the
in-use count?
