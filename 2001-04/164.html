Re: Routing and Mgetty problem

On 05 Apr 2001 10:34:44 +1000, Andrew Pullin wrote:

Just a suggestion that I found usefull when setting up a similar thing
on my RedHat 7 box.

Read the 'ISP-Setup-RedHat' HOWTO. It takes you through the whole issue
of setting up mgetty in inittab, the mgetty config files etc.

The other thing that I'd suggest is to use a seperate, private IP range.
For example, in /etc/ppp/options.ttySx have the IPs something like:

192.168.10.1:192.168.10.2

Keep it seperate from the rest of the network and/or world.
