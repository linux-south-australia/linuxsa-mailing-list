Here's some salient advice on looking for things like this:

1) Disconnect said machine from all networks (Internet, local networks)
2) Do NOT turn the machine off
 - leaving it on can leave useful information in the memory
 - naturally, turning it off erases that information
3) Do NOT trust any commands on it
 - people often trojan things like `ls' to hide specific files and so
forth

You can look for things like last time update (ls -alt, if ls hasn't
gone west, will list all files, sort them in order of update and so
forth). There's tools you can find at http://freshmeat.net/ which can
help you too...
