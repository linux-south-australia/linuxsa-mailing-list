Re: ipchains and accounting

> Shane Gelven wrote:
> 
> Hello,
> 
> Does anyone have any experience with using ipchains to do data
> accounting?
> 
> I have added the following to my ipchains rules
> 
> /sbin/ipchains -N site1
> /sbin/ipchains -I forward -i eth0 -d 111.11.111.111 -j site1
> I type the following: ipchains -L forward  -v ,this will display the
> data used, however it also shows everythings for the entire forward
> chain.
> 
> Does anyone know how I can get ipchains to do a list for just the user
> defined chain that I created. I have tried various combinations of:
> ipchains -L site1  -v
> which only displays:  Chain site1 (1 references):

ipchains -L site1 -n -x -v

should show you heaps of data. You could also use the data in
/proc/net/fw_* inside a perl/bash/csh/python/tcl script to grab the info
you want and display it in the format you desire.

Hope it helps,
