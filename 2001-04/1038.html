Re: samba again

Hi Robyn, I would check to make sure the registry patch worked. It really looks
like the enablplaintextpassword reg has not been implemented.  I know you
applied the patch but that IPC$ request is a classic encryption problem.

To confirm this, enable encryption in the smb.conf file and add the user to the
smb password file and try to log in again.

Remember to restart samba after each change

If you can its a reg problem if not let me know and send me the complete
smb.conf file.
