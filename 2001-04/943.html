Re: Why use DocBook versus Word

Hi,

Well, I found one good reason for using DocBook ... I can modify things ...

DocBook produces PDF by converting the SGML to TeX and then going to PDF
using a tool called pdfjadetex ...

There is a problem with TeX: any << or >> on the text gets turned into
chevrons because the living god, Don Knuth, requires that you code them as
<{<} or >{>} if you do not want a ligature between them.

So, whips out sed, runs it over the TeX and all is well :-)
