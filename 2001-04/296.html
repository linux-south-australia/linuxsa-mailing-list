Re: telnet connection refused

On 11 Apr 2001 11:10:48 +0530, Bala Krishna H V wrote:
> Hi All,
> I installed linux redhat6.2 workstation on my PC.But I could not telnet
> to this workstation.I get an error "connection refused...".

This is because in.telnetd is not running on the RedHat 6.2 box. You
have to enable it in /etc/inetd.conf (probably just uncomment it as
it'll already be there). Then killall -HUP inetd to restart it.

> But I can telnet to other systems from this w/s.

That's because the other systems have a telnet server running on them.

You should consider installing SSH as it improves the level of security
offered for remote access.
