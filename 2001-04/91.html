Linux Gaming & Linux Business Models

This link,

http://www.gamespy.com/interviews/march01/gavrielstate/

Is to a interesting interview on GameSpy.com 's web site titled:

"Transgaming Technologies
A talk with TransGaming's CEO Gavriel State concludes how the companies 
"WineX" will allow you to play off-the-shelf Windows' games on your Linux 
desktop."

Quite interesting for avid (rabid) gamers, but far more interesting is the 
CEO's business model for this Linux product.
They are hoping to get 20,000+ subscribers to the software and release the 
code to them. Questions are then asked by the interviewer about what will 
happen if this level is not reached or if this level is reached and code 
given out the subscriptions drop will the project die etc etc.

Nice lunch time read, even if your not a gamer.
