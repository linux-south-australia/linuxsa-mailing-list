Re: Linux and Ethernet Devices

Hi.

Oh, Ok, I see where you are coming from.

Given that this will be used in your absence, and there is a _risk_ that
the network connections will change, I think you may need to write
something to 'discover' what is attached to each network card before they
are configured on the rebuilt system. Maybe something that brings up the
first card with it's expected addressing, and then checks that it is so -
(ping or whatever to another box on the network) If that fails, drop the
config and try the next config until all cards are mapped.

It might help idiot proof the machine too, if it was part of the boot
process rather than the build process. What happens if someone switches a
couple of the net cables on the box?

Michael
