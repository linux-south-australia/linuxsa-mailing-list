RE: LILO / Grub

The lilo.conf file is as follows:

boot=/dev/hda
map=/boot/map
install=/boot/boot.b
prompt
timeout=50
message=/boot/message
linear
default=linux

image=/boot/vmlinuz-2.4.2-2
    label=linux
    read-only
    root=/dev/hda6

other=/dev/hdb1
    optional
    label=dos


Windows drive is Primary Master (hda)
Linux is Secondary Master (hdb1)
