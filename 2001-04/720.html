Re: CNET 650/Freesco/486 setup

Evan Bourlotos wrote:
> I am trying to setup freesco on a 486 with limited resources (8mb ram 120
> mb hdd 1.44 and 1 cnet650 ISA card) and ABC 56k modem.

Depending on your serial card you may have trouble using any modem faster than
9600bps (quick warning).  486 machines (pre DX2-x) generally have either a 16450
or 8250 UART chip on board, this severely limits what you can do as far as modem
speed.

The easiest way to test this is to type 'dmesg | more' in a shell and see what
shows up.  Under Linux (mandrake 7.2) this is the sort of thing you are looking
for:
  ttyS00 at 0x03f8 (irq = 4) is a 16550A
  ttyS01 at 0x02f8 (irq = 3) is a 16550A
I'm not sure what FreeSCO uses as it's basis, but it should be similar to this.

> I have it connected to my other machine atm
> I configured freesco to boot from floppy it dialled out etc but I casn't
> tell if my network card is working.

this is where having a shell is convenient, log in as whichever user is most
useful (not necessarily root) and type 'ifconfig'
this should show which network interfaces are currently configured and running.
