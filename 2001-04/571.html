PAM Authentication in PHP

List,

Someone wrote an extension for PHP to provide a PAM authentication
function.

Basically, it works something like this:
http://www.math.ohio-state.edu/~ccunning/pam_auth.html
if (!pam_auth($username,$password,&$error)) {
    echo $error ;
    } else {
        echo "Success" ;
        }

You just add it in when you compile PHP.

The Author's website is:

http://www.math.ohio-state.edu/~ccunning/pam_auth.html
