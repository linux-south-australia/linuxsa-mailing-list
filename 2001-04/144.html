Re: hostname and defaultroute questions

Hi Michael!
	The thing to remember here is that IP addresses
are assigned to interfaces, not machines.  This therefore means
that names are also assigned to interfaces (e.g. eth0, eth0:0,
eth1, eth1:0, eth2) not machines.  In the case of a machine
without physical interfaces - eth*, ppp* etc., then any name
given will apply to the internal loopback interface -
127.0.0.1, still an interface all the same.

For your second question, you need to set the default gateway on
MachineA to the IP address of eth0 on MachineB.  This assumes
that MachineB is acting as a gateway/router to the world outside
via the PPP interface using gated, routed or IP Masquerading.
