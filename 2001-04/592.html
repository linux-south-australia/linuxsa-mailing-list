Re: Search engines

On 19 Apr 2001 12:11:45 +0930, Richard White wrote:

> I would like to register a web site with search engines.  Having never
> attemtped this before, I'd be grateful if someone could give me some
> pointers on how to go about it.

Although this is something that you should probably be searching Google
for, I'll give a few pointers.

While the registration process for each Search Engine is fundamentally
the same, they all tend to vary slightly.

Some you have to pay for, others you don't, for example.

You can register with Google by going to
http://www.google.com/about.html and clicking on "How Google Adds
sites".

Follow the prompts and (hopefully) presto! You're site is in the index.

Yahoo has a similar procedure. Make a list of all the search engines
that you use, then go about finding out how to register with them.
Common ones include:

    1) Google
    2) Yahoo
    3) Altavista
    4) Hotbot
    5) Go Eureka
    6) Lycos

Do your own research as well -- I'm sure I've missed plenty of others
that we all use frequently.
