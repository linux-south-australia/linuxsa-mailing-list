> There is this port 27374 (Sub Seven) on my linux box. Now I wanted to know
> how I can like say disable that port as well as 10008 .ipchains would be a
> good start no??

Well:

* man ipchains
* /usr/doc/HOWTO/ipchains (howto)

Essentially something like:

ipchains -A input -i ppp+ -p tcp -s any/0 -d ${MYIP} 27374 -j DENY

That is to say, append to the input chain for any tcp packets from
anywhere destined for my ip addres a deny rule.

All ipchains rules look like that. If you're using 2.4.0 you may need to
either investigate netfilter or get the ipchains --> netfilter module.
