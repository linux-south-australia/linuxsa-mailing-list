Re: Giving users reboot permission

SUDO is what ya need.

www.freshmeat.net, search for it there.

Read about the config file, add a line for your http user to execute
shutdown as root (nobody/www-data/httpd/apache usually.. do a ps auxw and
find out) to execute /sbin/shutdown
then in your cgi-bin folder something as simple as

#!/bin/sh
sudo shutdown -h "now"

will probably suffice.
