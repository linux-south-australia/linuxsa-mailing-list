RE: UPS & Linux (Sola 325) software interface

Network UPS Tools came accross my path a little while ago, may be of help to
you.

Quote from their site: http://www.exploits.org/nut/

"Welcome to the support page for Network UPS Tools. This project is setting
out to monitor every kind of UPS with sufficient interest behind it. Many
models have serial ports that allow monitoring of some kind. While the
feature sets vary wildly, there is a compatibility layer that lets them all
get along. Useful features like safe shutdowns and live web monitoring have
been built on top of this foundation. You can also have many systems running
different operating systems monitoring a single physical UPS thanks to the
network communications layer. "

Hope it helps.

Patrick
