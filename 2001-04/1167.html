> I have tried two separate downloads now of the iso file and each (when CD is made) resulted in failure to load stage 2 ramdisk. Installation aborted.

Go to the mirror site, download the MD5SUMS file, and compare it with
the MD5SUM generated when you run:

md5sum ./mandrake-8.0-CD1.iso

Note that for ~650MB, it might take a while to generate the string.

That way you can check whether or not the image is funny. If the MD5SUMS
check out OK, you might have a hardware-related problem.
