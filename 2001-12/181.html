RE: chvt as normal user

You can use leftAlt-Fn to get from a text VT to any other VT (including
X at VT7), but to get from an X VT (7) back to any other VT, you need to
use ctrl-leftalt-Fn. I've never used the chvt command, but you may find
that it would work if you were to bring up a console window within X
(rather than just an Xterm...) I'm not entirely sure how to do that
though.
