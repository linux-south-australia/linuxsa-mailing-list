Re: Stopping all disk activity?

Gwyn Morfey wrote:

> I'm running Mandrake 8.1 on my 64mb laptop, and I'd like to be able to
> do simple word processing (or even text editing) without having the
> hard disk spinning. I'm finding that, even using vi on a single shell
> without X running, the disk is accessed every 20 seconds or so.

It's definitely vi that's causing the problem? (ie, it doesn't occur
when you're not running vi?)

vi writes a temporary file to /var/tmp, unless you're using vim, in
which case its temporary file usually resides in the same directory
as the file itself (with a '.' prefix and a '.swp' suffix).  You
could probably get vi/vim to construct its save file elswhere, like
to a RAM disk (probably through the use of TMPDIR), but I can't remember
the gritty details of RAM disk construction...
