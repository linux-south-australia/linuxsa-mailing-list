RE: Macro expansion with newlines with CPP

Michael Neuling wrote:

> and I can't work out how to get CPP to insert newlines in
> expanded macros.

CPP is a line based macro tool, so this is/will be fraught with
difficulty.

>  #define my_macro \
>	  mov a b \
>	  add c d \
>	  bra end

>  my_macro

> The output I get is:
>   mov a b add c d bra end

C doesn't care about this sort of stuff, which is why it doesn't
matter to C.

> Otherwise I'm going to have to insert my own line separator and
> post process the output of CPP with sed.

As long as that character doesn't occur elsewhere (eg, in strings
(yes, I know this is assembler - you can get strings in assembler)),
it should work fine.
