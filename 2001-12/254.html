Re: Blocking Ports

On Wed, Dec 12, 2001 at 12:24:29PM +1030, Adam Hawes wrote:

> iptables -A INPUT -p tcp -i ppp0 --source-port telnet -j DENY
> iptables -A INPUT -p udp -i ppp0 --source-port telnet -j DENY
> 
> I would expect that for both TCP and UDP connects on the telnet port of
> ppp0 (my net connection) the firewall would deny the connect, but I can
> telnet to the box from work, so I dare say that these rules don't do the
> job I want.

No, because you're saying that you want to deny connections on the PPP 
interface that come from port 23 on the client machine. You want
--destination-port[1]. You're trying to filter based on the packet's
destination.

   - andrew

[1] I don't normally do IPTables, so check the man page.
