Re: How to write ftp client program, using C socket?

<snip>

> I would like to automate the editing process at boot time, writing a 
> socket client program in my system.

It's not hard to do in the network startup script.  You need a base HTML
page (not hard to make), some simple shell code and /sbin/ifconfig.  A
little Perl should let you extract the IP address from the interface
config line.  You can then mangle this into the HTML page and use a
pre-configured FTP client to upload the page.

It's probably easier in Perl/Shell/ftp than it is in C.  An FTP client
isn't exactly an easy thing to write.
