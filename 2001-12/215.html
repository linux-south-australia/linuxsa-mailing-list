Re: Something hiding in my OS

Jason,
devices are treated as system files and as such exist in every
directory.

try this test for instance:
C:\> if exist c:\con echo CONsole device exists in root of C drive
C:\> if exist c:\prn echo You have a printer port in root of C drive

C:\> if exist c:\Windows\con echo CONsole device exists in the Windows folder (directory)
C:\> if exist c:\Windows\prn echo You have a printer port in the Windows folder (directory)
