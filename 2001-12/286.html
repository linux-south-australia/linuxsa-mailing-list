Re: Website blocking for single dialup PC

On Thu, Dec 13, 2001 at 01:05:09PM +1030, Freeman, Peter (ERHS) wrote:

> What I'd like to do is restrict access to certain websites (ala squidguard),
> however as its only a standalone PC, and low specs (P166,32MB), I figure it
> wouldn't be a great idea to set up a proxy on the localhost, else it'll run 
> like a dog.

Install Squid[1] and squidGuard[2] on the machine, and implement ACLs
to restrict people to certain sites.

You can make Squid take up very little resources by not caching, not
keeping logs and turn all the memory usage options down really low.

It's probably not going to be particularly noticeable if you setup
Squid properly.

   - andrew

[1] http://www.squid-cache.org/

[2] http://www.squidguard.org/
