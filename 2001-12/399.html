RE: MYOB Premier on Linux

Richard,

I can confirm to you that MYOB Premier only needs a File Share. I have done
this before and it works quite well.

The only suggestion I have is to provide a fast network. The MYOB Premier
datafile I commissioned was 277Mb. It takes about 7 secs for the client to
look up a part. This is is using 100Mb Full Duplex and a P166 with 64Mb RAM
and IDE HDD.

If you want to make them impressed make sure that you have a fast server and
network otherwise MYOB will slow as the data file grows.

Hope this helps.

Brett Hales
