Re: Evolution 1.0 Out!

On Tue, Dec 04, 2001 at 10:25:44AM +1030, Ben Williams wrote:

> anyone know if evolution will run on a standard KDE desktop? or do i need 
> ximian gnome? i much prefer kde.

[ The answer to this question -was- on the Ximian website, but they've 
  reorganized it and I can't find it any more. ]

You need the GNOME libraries for Evolution to work. Things like GTK,
Bonobo, OAF and a myriad of others.

To summarize the content on the aforementioned vanishing page, you can 
install Evolution without having to install Ximian GNOME, but doing so 
is quite painful (there are so many dependencies to worry about). They 
recommend (funnily enough) that you use RedCarpet, which comes with
Ximian GNOME.

So, install the Evolution dependencies and you'll be set. It's a
character-building exercise :-)
