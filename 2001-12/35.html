I've been extremely impressed with Evolution, but they changed the
format in which they quote the date. Because Evolution uses all the
GNOME/i8n stuff, patching it requires changes to a gazillion files. I
can't be bothered at this stage.

If you can get over the emetic date format, it's quite a nice
client. It's slightly wonky on FreeBSD, but I think that's got to do
with OAF and FreeBSD not cooperating.

However, I think Mutt is the best of the lot. I can access my mail
from anywhere there's an Internet connection[1], I get nice pretty
colors so I can tell which list each message is from, nice threading,
and I can use Xemacs as my mail editor. I can't think of anything else 
that I'd need in a mail client.
