re: MYOB Premier on Linux

> What I want to know is if MYOB only needs a file share, or if it needs
> a Windows server to have some server component of it's own there


A simple Windows file share always worked whenever I've set up MYOB...

By default, MYOB keeps its data files (and backups) in its installation 
directory. Any species of MYOB on a client computer can also connect to data 
files in a host computer's shared folder. But only MYOB Premier stops Bad 
Things Happening when multiple users are changing the data simultaneously.

Doesn't matter if the host is Win 4 Workgroups, Win 9x or Win NT/2000; 
therefore I see no impediment to using a Samba host.

MYOB seems to work better in this scenario if you "Map Network Drive" the 
host location for each of the clients. Plus it's easier to explain to users 
that the company financials are living in eg: "S:\MYOB" rather than 
"//megacorporation-server/company/data/headoffice/financial/MYOB"

BTW the MYOB setup program is known to break Paintshop Pro 6 & 7, Netscape 
(sometimes) and MS Access (sometimes). Possibly other programs too. It 
replaces a couple of Windows system files MSCVIRT.DLL and MSCVRT.DLL with its 
own versions. Restore the original versions to c:\windows\system if this 
problem crops up.
