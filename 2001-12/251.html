Blocking Ports

Hi all.

I'm trying to block ports on my Linux box at home so that I can run some
services internally without having them available to the greater
populous of the Internet.

I tried IPTables rules like the following:

iptables -A INPUT -p tcp -i ppp0 --source-port telnet -j DENY
iptables -A INPUT -p udp -i ppp0 --source-port telnet -j DENY

I would expect that for both TCP and UDP connects on the telnet port of
ppp0 (my net connection) the firewall would deny the connect, but I can
telnet to the box from work, so I dare say that these rules don't do the
job I want.

Anyway, does anyone know the way to do what I want to do with IPTables?
