David,

if you just typed the "php -v" on the cli then you have the CGI version of
PHP installed.  If so then you need to add a bang line to the beginning of
your php script for it to execute correctly.

just do:

"which php"

When you get the location add the following to the top of your php script

#!/path/from/which/php

You can also add further options to that (see PHP docs for more info)
