Re: MYOB Premier on Linux

Richard,
	I haven't done this on Linux, but it really only needs a
shared file path for the data files (*.prm).  The application
itself is installed on all workstations and access the data file
from a common share.  The only hassle might be reminding the
users to select TCP/IP as the transport protocol rather than
NetBEUI when MYOB loads up.  Just create a share on SAMBA and put
the data file(s) in there, accessible by a common mapping.

eg. M: = \\linuxserver\myobdatashare on each workstation

Don.
