Re: Samba Authentication

Hi

The reason why samba does not use the standard Unix password system is
because windows uses a password system with a different encryption format.

There is a system were you can keep both the Unix and windows password in
sync by using the smbpasswd scripts.

I have heard of a PAM module which makes the password system in Unix the
same as windows, but this would be less secure.

The programmer of samba would not have made it complicated and different if
they did not have to.

Cheers
