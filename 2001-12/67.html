Re: Evolution 1.0 Out!

Quoting Ben Williams <benw@webmedia.com.au>:

> anyone know if evolution will run on a standard KDE desktop? or do i need 
> ximian gnome? i much prefer kde.

Both KDE and GNOME applications will run on each other's desktops, as long as 
you have the libraries installed.  I used to use KWord on my GNOME desktop 
before OpenOffice came along.

Evolution, like all GNOME apps, aren't dependant upon having a Ximian GNOME 
desktop.  They'll run on a Red Hat GNOME desktop, a KDE desktop, or just vanilla 
X-windows as long as you have the right libs installed (thanks rpm/apt-get).

And don't worry, you should grow out of KDE one day :-)
