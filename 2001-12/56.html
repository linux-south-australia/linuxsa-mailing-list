Macro expansion with newlines with CPP

Hi all,

I'm using GNU CPP to expand macros for a assembler I'm using.
Unfortunatley there doesn't seem to be a line separator in the
assembler I'm using (eg. like ";") and I can't work out how to get CPP
to insert newlines in expanded macros.

For example, if I have the below input to CPP:

  #define my_macro \
	  mov a b \
	  add c d \
	  bra end

  my_macro

The output I get is:
  mov a b add c d bra end

But the output I want is:
  mov a b
  add c d
  bra end

Can anyone tell me how I can write my macros and run CPP so that I get
the desired output?

Otherwise I'm going to have to insert my own line separator and post
process the output of CPP with sed.
