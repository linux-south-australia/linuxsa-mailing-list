RE: developer bad attitude [was: Re: suitable OS ?]

Smoothwall is a good product and is VERY easy to set up, even with no Linux
experience. I'd recommend it in a flash to home users that wanted a stable
secure firewall product.

I agree that Richard Morell ( Lead developer ) is a very abrasive person,
but given the time he must spend on the product I can see why he might be.
He was personally going through all the posts on the mailing list ( Don't
know how he found the time ) and jumped down on anybody that hadn't RTFM.

If you are a corporate there is now a Support option that can be purchased
for Smoothwall.

I use a Smoothwall box at home for my ADSL connection and highly recommend
it. Use it at work too on remote site for dial in and modem sharing.....my
10c worth
