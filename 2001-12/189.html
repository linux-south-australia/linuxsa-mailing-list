Most M$ file formats are definately NOT published, hence why the open sourcing 
of star office was so important.  If for nothing else, it gave the community 
some stable and robust M$ file format loaders.

In the past decade file formats were a key part of Microsoft locking in 
companies to their products.  This has been recognised and exposed, and now they 
are trying something different - using XML as a file format, but with 
proprietry, undocumented DTDs.  They can now claim they are using an "open" file 
format, but without the DTD it still makes it difficult for other software 
products to extract the information from the data.
