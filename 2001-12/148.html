Re: MSAcess to MySQL

>Is there anyone who could export an MSAccess table to MYSQL?

I've often found that this is best done by exporting MS-Access as text, and
then importing into Mysql, possibly using a php script.

Field types like currency and date don't quite match the Mysql equivalents
and cause the sorts of problems that you're seeing.
