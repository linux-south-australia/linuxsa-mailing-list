Re: Stopping all disk activity?

On Mon, Dec 10, 2001 at 11:29:40PM +1030, Gwyn Morfey wrote:
> Hi -
> 
> I'm running Mandrake 8.1 on my 64mb laptop, and I'd like to be able to
  do simple word processing (or even text editing) without having the
  hard disk spinning.

You should have a command called hdparm which will allow you to spin down
the disk after a set period of inactivity e.g.
"hdparm -S 18 /dev/hda" will spin down after 90 seconds inactivity
(ha! the 18 refers to 5 seconds increments 5x18=90, man hdparm has all this).
You can insert the command as above in /etc/rc.d/rc.local so it is in effect
everytime you boot.
