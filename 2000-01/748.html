Re: Rebinding Delete to delete-char for editing

Hi - edit the ~/.xinitrc file (ie: /root/.xinitrc  OR 
/home/(username)/.xinitrc and replace the exec gnome or exec
$WINDOWMANAGER or exec whatever line at the end of the file to read exec
startkde
If .xinitrc doesnt exist in the users directory create it & add the line
"exec startkde".
