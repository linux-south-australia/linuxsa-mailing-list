A more RH 6.1 - compliant way to do it is to edit the contents of
/etc/sysconfig/desktop (I think.. correct me if I'm wrong)

echo kde > /etc/sysconfig/desktop

will set X to run kde as the window manager.

Phil's way will work also :)

The /etc/sysconfig/desktop method is probably a cleaner way to go though?

Keep in mind, that's a system-wide default.
