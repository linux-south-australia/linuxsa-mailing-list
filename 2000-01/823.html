What you can do is go here: http://206.31.99.95/masq/ and download
winMasqDialer. You need to then run a masqdialer server which you can also
download from there. It is probably better to so it this way as diald in
Linux is, for lack of a better word, kludgy. It is much better in
FreeBSD. I am using winmasqdialer and it runs fine with my redhat machine.
