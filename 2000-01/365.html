Re: sending data through a dummy device

Use Squid to proxy all requests and point you browsers through here. It
should handle whatever ports you want.

139.x.y.z --> www.bigpond.com.au --> SQUID via 203.a.b.c <><><><><><><>
203.a.b.c --> SQUID --> 139.x.y.z

This is the concept I think you need. Not sure on the details of setting up
SQUID (having never done this so I will not try to bamboozle you on that
point!).

There maybe a way to make SQUID transparent so you can just send out a
request and the server picks it up and re-directs it for you without
resetting the browsers.

Have a look at the HOWTOs (I think SQUID is in there), otherwise the man
pages and finally LDP.

Best of luck.

Rob.
