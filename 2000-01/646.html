Re: forgot root passwd

It's really quite simple. Catch lilo and type in [tab]. Type the one
that loads your linux (it is probably called linux) and type single.
You'll boot into single user mode. You won't need a password. Then use
passwd to change the root passwd and you're up and running.

IE

[BIOS STUFF SHOWS ON SCREEN]

lilo: [you press tab]
linux dos

lilo: linux single [you type this]

[LINUX BOOT STUFF]

#
#passwd
(and so forth)
