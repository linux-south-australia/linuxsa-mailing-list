Re: Help please! Kernel upgrading

1st.... you do not do "make install"
2nd... you missed the step "make zImage" or "make bzImage"
3rd... you must then copy the file from the arch/i386/boot/zImage to
/boot/vmlinuz-2.2.14
4th... edit /etc/lilo.conf to use the new image

BTW you cant have looked to far for the info on how to do it... it is in the
README file that comes with the source.
