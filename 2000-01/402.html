Why is that not a problem? The BIOS needs to know about FDD / HDD to be
able to boot. Without, it will usually do the NO ROM BASIC SYSTEM HALTED
thing. Once you get past LILO or DOS (to run loadlin), Linux will take care
of the rest.

A different (read weird) approach would be to obtain a Network card with a
boot PROM and boot Linux over a network (from another Linux box of course).
Now No hard drives in Bios = no problem.
