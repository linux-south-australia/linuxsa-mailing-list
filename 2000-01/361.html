Re: Lilo Help

Lilo hides in the MBR (when you tell it to). All the MBR does when LILO doesn't occupy
it is load the first sector of the ACTIVE partition. So when fdisk /MBR replaces the
MBR, and you reboot, it will boot the active partition. If (for some reason) Windows
WON'T boot, I think you can fix that with the three Windows boot disks. When was the
last time you backed up?
