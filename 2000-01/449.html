Re: X mouse troubles

Tim wrote:
> 
> Following the installation of both RH 6.0 and 6.1 on a p2-350, the mouse
> failed failed to work in X.
> 
> The mouse pointer would display...but not move  During the graphical
> installation of 6.1, it wokred fine.  Kudzu detected it and seemed to
> configure at fine.  The mouse is generic microsoft.  Tried both RS232
> ports to no avail.
> 
> Any suggestions?
> 

check /etc/X11/XF86Config

in there is a section that looks like this (for me):

Section "Pointer"
   Protocol        "MouseSystems" <-- put "Microsoft" here
   Device          "/dev/mouse" <-- see below
   BaudRate        1200 <-- can leave these lines out
   Resolution      100  <--  "    "      "    "    "
# for a 3 button mouse
   Buttons         3 
# for a 2 button mouse
   Buttons	   2
   Emulate3Buttons On   <-- i think this is right (?!?)
EndSection

/dev/mouse is a link, like this (for me):
lrwxrwxrwx   1 root     root           10 Dec 17 17:42 /dev/mouse ->
/dev/ttyS3

Remember:
com1 = ttyS0
...  = ...
com4 = ttyS3

To make the link:
ln - /dev/ttyS3 /dev/mouse
