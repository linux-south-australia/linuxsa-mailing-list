> (2). If I telnet from my Windows machine to my Linux machine I can't use
> the host name.
>         ie I have to telnet using the IP address. However, telnetting
> from the Linux Machine to the Windows
>         machine using the host name rather than the IP address works

>         Can anyone point me in the right direction as to how to get the

Sounds line name serving isn't configured properly.
Does your TCP/IP config in windows have DNS enabled, and pointing to the
IP of your name server ?

If so, is your name server set up so that the names define IPs as well as
IPs defining names (look at the DNS HOWTO for information on this).
Also is your name server set up so that it will respoint to requests from
your Windows box ?
