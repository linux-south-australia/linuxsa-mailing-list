I can only help with part 2 at moment.

There is a file in your windows directory called hosts.sam.
This allows you to specify hostnames for IP addresses.
The format of the file is explained in hosts.sam. 
You then just save the file as c:\windows\hosts and you can connect to
your machine by hostname.

We use this a lot as we have several unix boxes to connect to.
