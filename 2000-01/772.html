VMS has a remarkably consistent and good text interface, so much so
that it is invariably copied by vendors that need a command line
(such as on routers).

I've written large technical documents on both VMS and the
Mac, and there's no comparison.  The graphical component of the
Mac UI adds a lot of value.  From changing text to a heading
with a mouse stroke, or actually seeing an included diagram,
or having both the document and the endnotes pages open.
And to be actually able to draw diagrams, not describe them
programmatically.

I wouldn't to back to the command line for serious work in
authoring or drawing diagrams or doing overhead slides.
And this is a lot of everyday people's work when they
deal with computers.
