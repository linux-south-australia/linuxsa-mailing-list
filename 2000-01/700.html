Which won't help if it hasn't been installed in the first place, which
is David's problem.  When I get a new RPM distro, the first thing I do
is run the following shell script -

    #!/bin/sh

    ##  List all files contained within RPM packages

    find /mnt/cdrom -name "*.rpm" | while read file
    do
        echo "==== $file"
        rpm -qpl $file
        echo
    done > ${1?}

This will build a file which lists all the files within an RPM.
Then searching is just a matter of lessing the result for the
appropriate file, then typing ?^== for the package.
