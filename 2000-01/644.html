There are two things you can do to reset the root password:

1    Boot linux into single user mode (type 'linux single' at the lilo
prompt, where linux is the name of the lilo image) and edit the /etc/passwd
file
2    Boot off a linux floppy disk and mount your root filesystem. Edit the
/etc/passwd file.

Note: if your system is using shadow passwords, you will need to edit
/etc/shadow instead. There are a number of fields in these files, seperated
by colons (:). You want to delete the contents of the second field on root's
account. Make sure you don't delete the colons. This will give no password,
so when you reboot, you can log in as root without a password and live
happily ever after. When you successfully log in the first command that
should be run is: passwd.

looking at the man page for passwd will show you the format for the passwd
file. It's probably a good idea to look at it.
