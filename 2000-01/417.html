Re: multiple monitors

Hi this isnt exactly whats being discussed BUT I just wondered if
everyone knows that you can run multiple XWindows (as different users)
on the same pc.

1. Login as user 'me' & startx - up comes KDE etc
2. ALT+CTRL+F2, log in as root or another user and type this :   xinit
-- :1
3. Up comes ICEWM (my other self likes ICE:))
Not incredibly useful but certainly a nice party trick...
Oh, btw.... ALT (+ CTRL)F7 gets u to the first XWindows and ALT (+CTRL)
F8 to the second one.
Dont know if you can do more than 2... hang on will try :)
OK... got 1 KDE & 3 ICEWM running - F7 has KDE, F8=ICEWM#1 F9=ICEWM#2
F10 is skipped & F11=ICEWM#3
PC is a Pentium-75 with 32Mb RAM .. CPU usage = 1.1% Load=0.7 Swap=26Mb
used.
Anyway... I am sure someone didnt know this ;0)

ONE thing though - having 4 XWindows up uses 8 "consoles" (ie/ F1+F7,
F2+F8, F3 & F9, F4 & F11)!!
