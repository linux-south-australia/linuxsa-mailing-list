Re: Partitions

Note the Logical bit next to all partitions after (and including) hdb5.
These logical partitions are partitions inside an extended partition
(because the partition table only has enough room for 4 partitions).

I think if you type 'fdisk -l' from linux, it should tell you that
partition 3 or 4 is an extended partition that takes up the same amount of
disk as the 3 linux ones combined.
