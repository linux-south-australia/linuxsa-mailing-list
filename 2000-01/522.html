Not useful unless the uuencoded portion is at the start of the file.
A little-known fact about uuencode is that uudecode knows how to 
find the "begin 600 filename" sequence anywhere in a file;  this means
that if someone sends you a uuencoded file, you can decode it by
simply running the entire mail message into uudecode's stdin (i.e.:
you don't need to edit the saved mail message first;  in most
mail user agents, it's possible to decode a file by typing |uudecode).

 > file core.uue
 > core.uue: uuencoded or xxencoded text

If that was in a mail message, file(1) would say it was a mailbox file
(and you'd want to kill the bastard who emailed you core dumps :-)
