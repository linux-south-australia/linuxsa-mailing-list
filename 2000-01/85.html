G'day,

Have you tried using SMART, the web based Mars Admin tool. It probably won't
let you do all you want to do but it's a good start. We've used SMART to get
MARS working and then edited the NWSERV.CONF once it's working.

We've used MARS because it is not possible to use SAMBA for our application.
While MARS it isn't as easy to configure as SAMBA we've had it running as a
server at one of our sites for two months with no problems.
