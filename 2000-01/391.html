Re: Lilo Help

> I then go to run lilo to intialise the canges using the following
> command. HDb1 is my linux partiton, and mounted on the linuxcare disk as
> /mnt/0
> 
> lilo -C /mnt/0/etc/lilo.conf -m /mnt/0/boot/map -r /mnt/0 -i /mnt/0/boot/boot.b

'-r /mnt/0' sets lilo's root directory to /mnt/0.  That's the only
option you need (ie. you should run 'lilo -r /mnt/0').

The other options you've got there (-C, -m, -i) are what makes it not
work, because they are interpreted relative to the new root directory,
making the overall path become eg.: /mnt/0/mnt/0/etc/lilo.conf
