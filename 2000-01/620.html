Re: printer

What I've noticed (along with speculation):
RedHat 6.1 has the following problem.. it doesn't load
the parport_pc.o module, which is required to use lp0 or lp1.

After loading parport_pc by hand,
  # /sbin/insmod /lib/modules/2.2.??/misc/parport_pc.o

it looks like the original modules need to be reloaded, in order to
pick up the dependencies (or something).

In order to solve this permanently, some thing like to following
should be adde to /etc/conf.modules (but I can't confirm that this
will fix it). 
  
  pre-install lp /sbin/insmod parport_pc
 
Any comments?
