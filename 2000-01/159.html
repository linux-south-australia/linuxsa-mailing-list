Weird POP problem

Greetings,

This problem has been resolved by other methods, but if someone can point
me in the direction of information I would be most greatful.

Server Setup... RH 6.0 box, running the standard ipop3 deamon.

Client Setup...

Widowze Box at home running Eudora with a POP connection. Email is
downloaded but not removed from server.
Windowse Box at work running Eudora with POP connection. Email is
downloaded and REMOVED from server.


Symptom...

On home box, get an email that has a corrupted attachemnt. Problem clears

User returns to work, and downloads mail. Same message downloads and

Here is the problem, his mail spool in /var/spool/mail was )kb in size
