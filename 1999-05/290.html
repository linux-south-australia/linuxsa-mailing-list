Re: Multiple CPU's under Linux?

Timothy Aslat wrote:

> Is there a way I can test to see if Linux is using both CPU's or have I
> got a spare CPU that I could put in another machine?

A quick check of the man pages for the keyword cpu:

	man -k cpu

brings up xcpustate.  man xcpustate states that it shows one 
bar per CPU... so give that utility a try.  

cat /proc/cpuinfo will tell you if Linux sees the second 
CPU -- you get information for processor 0 and processor 1.
