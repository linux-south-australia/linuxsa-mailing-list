Greetings,
You should be able to set these cards up as an lpd server.
The documentation says this is possible.
Please note you will probably have to update the firmware on the JetDirect
card to support lpd printing unless it is a current model JetDirect card.
This information is available, (or was), on the HP web site, as I had to
research compatability with OS/2 Warp on this piont.
