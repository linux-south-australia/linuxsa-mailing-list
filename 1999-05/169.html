Jake Hawkes wrote:

 > > I'm sure you noticed that you can set the file to be line buffered even if
 > > the output is to a file...  (Please read up on stdio before you ask how.)
 > 
 > darn. I thought that was the answer.
 > setbuffer() only works on FILE*.
 > I need something that works on int file descriptors.

Why?  

 (a) you're using stdout and stderr, which are FILE*'s; and
 (b) integer file descriptors aren't buffered.

If you want to buffer them, you'll have to do it yourself.  Or use
fread()/fwrite() to do your I/O
