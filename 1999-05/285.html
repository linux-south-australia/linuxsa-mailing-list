Re: XFree86 - i740 - MS Intellimouse

According to my documentation on page 325 of the Red Hat Installation Guide
for RH 5.2

`It ahs been found that the 2 button Microsoft Mouse of Version 2.1A or
greater is a `smart' [intelli] mouse...This caus X and or gpm to not work
with Linux because the mouse is not responding in a way these programs are
expecting. To `reeset' the mouse to work with them you can use several
methods:

use mouseconfig as
  mouseconfig --kickstart --device cuaX

where cuaX is where the mouse is (cua0 = com1, cua1 = com2)

another solution is to get gpm1.13, run as gpm -t pnp -R and configure
XFree86 for
MouseSystems Protocol with /dev/gpmdata as the device
'

These are direct quotes.

I found RedHat took ages to recognise my mouse properly. After about thirty
restarts and installs the mouse now works perfectly!!!!!!
