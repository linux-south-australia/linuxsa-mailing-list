Desktop wise, it does not fit in with what the students are expected to 
be experienced in.  Microsoft currently is the dominant platform, it
is appropriate that it be in the desktop.

As time goes one, Linux will be more appropriate for that market.  Small
steps will end in the results, GNOME/KDE are great steps forward, but there
still is a large void that needs to be filled.  Windows based products 
do fill that void.

The real strength for Linux is the Server area.  No one sees it, people
just use the services offered by it, Mail/DNS/Samba/FTP/etc...

So there is still opportunities, just not at the desktop :)
