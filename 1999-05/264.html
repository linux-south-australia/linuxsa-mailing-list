Re: CONNECTING TO THE NET

i've noticed that the 5.2 out-a-tha-box distribution hasn't got the
modules.dep file correctly configured (or it never was, and I'm supposed
to set it up, or whatever)

i find that I have to hand insert the ppp module, before anyting will
work. I know i should learn how to correctly utilise kerneld, but.....
(interestingly, a stock standard SB16 soundcard wont auto load modules
either.... RH5.2)
i manually insert the modules for ppp like this: (as root)
$> /sbin/modprobe ppp
if that fails saying some dependancies are broken, try
$> /sbin/modprobe slhc
if that fails, you need more help :)

here's what i do when fiddling with Linux.

  o  start up about 4 terminals (virtual, X, telnets whatever)
  o  in one, do a tail -f /var/log/messages
  o  in another, do more /usr/doc/HOWTO/ISP-Hookup-HOWTO
     really, with a bit of patience, and _alot_ of reading, you can get
     what you need going from this file.
  o  in another, edit the various files
  o  in the last, run the commands.
