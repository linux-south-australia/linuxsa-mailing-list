Re: Installing Multiple copies of Linux on one machine?

Well, I don"t know laws -rules - conventions etc.....but
Having one machine with two harddisks in it. Harddisk numero one works with
windows98 using 50% of the disk.The other 50% is formated ext2fs with linux
5.2 redhad (It had have three others before like dummy and Caldera ) On the
second harddrive I have a 1/4 reserved for dos/windows with zipfiles
stored.The rest 3/4 is ext2fs with redhat 6.0. BEFORE I had lilo boot from
drive one (hda1) and while startingup had to be very very quick to choose
between windows or linux bootups. I you are not quick enough....you get
windows if you want redhat or get redhat if you want windows. SO I deleted
this lilo boot thing and make each linux now bootup with its own floppy boot
up to boot its own redhad kind. SO that is now peaceful for me ( maybe for
all persioners anyway, who are not so wild goose any more ) Jacob.
