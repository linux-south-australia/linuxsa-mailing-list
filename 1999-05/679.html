NTFS writes may corrupt disk

There are reliable reports on the linux-kernel mailing list
that the new NTFS file system for Linux 2.2 may corrupt the
file system when writing files.

Patch available in a few days.

Note that the NTFS file system isn't the same as the VFAT file
system used by most Windows NT and Windows 95 users.
