Even easier again - setup samba on the Linux box and keep all your work on
a samba share. No copying at all required, you want to access a file from
Linux, it's already there.

Quite useful (for windows anyway) is the feature to change where 'My
Documents' (ugh) lives. Create a samba home share, each user who logs into
the Windows box gets say a T: drive which is their home directory on the
Linux box. 

Mkdir t:/doc and then point 'My Documents' at that. 

Other advantages:

o Much easier and more reliable to backup
o Can do cool Unix-style things on your windows files (like auto-signature
  changer etc etc, disk quota checker - sky is the limit)
