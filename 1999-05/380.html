Why not a "Machine & Version ID"?

I see lots of questions without the obvious info on
Linux version & features included in it...

Wouldn't it be nice to have a small program that
would (i.e. if the kernel couldn't be persuaded to)
present a nice, short string that - when decoded -
indicated all that a would-be answerer might like
to know about a asker's configuration, e.g. Linux
version, etc.

Perhaps this string could be formed into a .sig
file for the would-be asker's e-mail client to use
e.g. in asking-for-help mode...  ;)

The Radio Amateurs' Packet BBS systems tell
would-be Users' software this kind of info in less
than 30 characters, if memory serves...
