Re: User Accounts

On Tue, 18 May 1999, Adam Rumbelow wrote:
> is possible to make a user equal (or close to it) to root?

Of course.  There are two halves to the user root: There is the
name, "root", which is not magical; and there is the uid 0,
which is.  You can add as many users with uid 0 as you want.  So
to create a user which is exactly equal to root:

  # echo "davidn:DbuDJMxFRJoE2:0:0:David Newall:/:/bin/ksh" >> /etc/passwd

Simple as that.  ;-)
