The Debian Woody install CDs (at least) include a floppy image "sbm.bin"
which you can write to a floppy with dd (Linux) or rawrite.exe (DOS). SBM
stands for "Smart Boot Manager". You boot off that floppy and it then gives
you a menu of other devices from which to continue booting, so you can boot
from CD even with a non-CD-aware BIOS.
