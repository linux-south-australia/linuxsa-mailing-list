Re: limiting man to sections

On 09/01/04 10:58, Klovekorn, Patrick wrote:
> Hi
>  
> I need to know more about the send() command in sockets, however when I 
> type in "man send" I just get the mail help, not the library function. 
> How do I limit the man search to the C library files?

Hi Patrick,

If you know the section the man page you're after is in, you can tell 
man to look in that section by putting it before the keyword, like this:

man 2 send

You can probably find out the section that the relevant man page is in 
via `apropos send` (or `man -k send`, which I think does exactly the 
same thing).

> Also in general I find man pages describe things nicely but fail to give 
> examples on how to use a command. Is there a better source of info than man?

Some programs maintain documentation in gnu info; try `info send`. Other 
than that, try grepping through the kernel source for examples maybe?
