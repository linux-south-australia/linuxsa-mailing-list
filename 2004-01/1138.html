Ditto to that.

If you haven't got an updated virus scanner installed, I'd recommend
Trends free online scanner, on a 1.5mbs link it should run rather well :)

Run it from here: http://housecall.trendmicro.com

Another suggestion, viruses/trojans are just one of the many
problems you can have on any net-connected Windows box.  Spyware, adbots
and other nasties can take advantage of many holes as well, especially 
in IE...

A useful tool to detect any ads/bots etc is Spybot, just make sure you
run the online update before you do the first scan!  You'll be amazed and
maybe a little horrified at how many things this tool finds...

http://www.safer-networking.org/index.php?page=mirrors
