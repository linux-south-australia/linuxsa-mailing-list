On 28/01/2004, at 5:41 PM, steve wrote:
> So, I have done the right thing and started rolling my own distro. Its
> rally easy once you start -  All you need to do it write a monster bash
> script for the installer, and then write another bash script to replace
> the antiquated SysV init process.

Without being partisan about it, those monster bash scripts are a basic 
version of what Gentoo is all about. All Gentoo does is provide scripts 
to compile the original packages.

Boot Gentoo, compile what you want, strip the Gentoo stuff out, and you 
have a Linux-From-Scratch system without the pain.
