not enough googling :(
thttpd is VERY picky about file permissions.  Data files should be mode 644
(rw-r--r--), directories should be 755 (rwxr-xr-x) if you want to allow
indexing and 711 (rwx--x--x) to disallow it, and CGI programs should be mode
755 (rwxr-xr-x) or 711 (rwx--x--x).  See the man page above for more
details.  IF YOU HAVE THE WRONG FILE PERMISSIONS, YOUR WEB PAGES WILL NOT BE
ACCESSABLE!
