Oh, you are kidding?

Mount the ISO image using the loop device:

   mount -o loop /path/to/image/of/iso /mnt/your-distro

then delete the offending file and replace it with the new copy.

As long as the new copy is not too big, you will probably be OK.

However, it is also possible to build a larger file system if need be.
