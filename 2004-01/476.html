> I assume the USB ones use the sg driver so appear as scsi devices?

Yup, probably something like /dev/scd0
You may also need to use the 'ehci-hcd' (USB-2.0) USB controller module
for the sake of some pretty serious data flow/bandwith.

> you don't use dvdrecord?

Nope :)
'dvdrecord' is part of the dvd-rw/dvd-r toolset, I use dvd+rw/+r discs &
so use the dvd+rw/dvd+r toolset ('growisofs'), it can write -rw/-r also.

I much prefer the +rw/+r format as it's multi-session, -rw/-r is not.
