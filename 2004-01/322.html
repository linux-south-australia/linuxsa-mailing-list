The CVS version of k3b works fine with the latest kde_beta & kde_cvs.

# touch ~/.cvspass
# cvs -q -f -z4 -d ":pserver:anonymous:@anoncvs.kde.org:/home/kde" login
# cvs -q -f -z4 -d ":pserver:anonymous@anoncvs.kde.org:/home/kde" co
-dk3b kdeextragear-1/k3b

cd k3b & compile/install

Though you may lose package management on this package if running
Mandrake as I don't think KDE provide .rpm CVS snapshots.
