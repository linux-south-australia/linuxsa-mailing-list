Peter
re Q4 - Smoothwall has a good reputation but  seems to
require a dedicated box. I've been using shorewall
(shorewall.net), which as an easier-to-config wrapper around
the Linux kernel IP filtering framework (iptables). You
still need to edit config files in /etc/shorewall but the
documentation is good and the ideas are intuitive. Worked
for me anyway.
