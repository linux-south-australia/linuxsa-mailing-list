Re: Introducing Next Office

David Fitch wrote:

>		Next Office has the Open
>                file formats that are international standard set by W3C,
>                the worldwide organization that also set the standard in
>                Internet such as "www." in all Internet addresses.
>

I don't believe that there is any such standard. Tim Berners-Lee 
intended for the web to be driven by the hostnames of servers, and 
called his own host 'www' as he couldn't run his then-experimental web 
server on the main server at Cern. Others apparently picked up the 
naming convention from there, and the rest is history.
