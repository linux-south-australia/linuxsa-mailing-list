Howdy Daryl,

I'm currently using the base model Logitech USB optical wheel mouse on 
my Fedora Core 1 desktop, and I like it. I don't mind using the middle 
button (i.e. pressing down the wheel), but I got used to it on windows 
first (the logitech drivers allow you to set the middle button to 
something at least slightly useful, like double click). It got detetcted 
properly by the FC1 installer, the mouse wheel sorks as expected, it all 
works fine.

I'm not sure about the advanced buttons on some of the fancier 
multi-button models (MS explorer, Logitech MX500 etc), as I've never 
tried them, but I'm sure at least the main 3 + wheel would work. You 
could always buy them from somewhere like officeworks that will let you 
return them for full refund if they don't do what you want.
