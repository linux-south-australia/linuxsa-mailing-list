Networking question: multiple MAC addresses, same IP address

Hello all,

I'm inside a switched LAN (ethernet) and facing the following:

With a tool called 'arpwatch' I noticied that a same IP address
(10.2.200.110) is related to many MAC addresses, and it changes every
time (almost every second). This server is direct connected to a
switch, with just 2 other machines with it.

Let me show you a syslog message:

Jan 14 18:10:39 gerenciamento arpwatch: reused old ethernet address
10.2.200.110 10:0:5a:75:56:d8 (10:0:5a:79:88:dc)
Jan 14 18:10:47 gerenciamento arpwatch: reused old ethernet address
10.2.200.110 10:0:5a:79:81:1c (10:0:5a:75:56:d8)
Jan 14 18:10:48 gerenciamento arpwatch: reused old ethernet address
10.2.200.110 10:0:5a:75:55:f2 (10:0:5a:79:81:1c)

As you can see, all the MAC address are from the same vendor (IBM, in
this case).
