Re: Sendmail - modifying "from" address based on "to" address for outgoing email

On Wed, Jan 07, 2004 at 05:59:31PM +1030, W. Sierke wrote:
> Hi,
> 
> I'm looking for a way of changing the "from" address in outgoing email
> (sendmail), based on the "to" address (both envelope and headers).
> 
> >From what I've read so far, it looks as though sendmail's milter facility is
> probably a good candidate for achieving this with a custom-written milter.
> I'm wondering whether anyone happens to know of any existing solutions
> (preferably with a FreeBSD port available) that only require a configuration
> file - since any custom solution I implement will almost certainly have the
> configuration embedded (which I'm anticipating from a combination of
> laziness and ineptness^Winexperience^Wineptness).

Exim can do this; see the following excerpt from my /etc/exim/exim.conf
which I use to send mail to Yahoo mailing lists, primarily the BJH (Barclay
James Harvest, an excellent group) discussion list, from "bjh.pigeon" and
everyone else from addresses looked up in /etc/email-addresses:


######################################################################
#                      REWRITE CONFIGURATION                         #
######################################################################


pigeon@droitwichbox    "${if match {$header to:}{.*@yahoogroups.com}\
                        {bjh.pigeon@ukonline.co.uk}fail}" frFs

*@droitwichbox    ${lookup{$1}lsearch{/etc/email-addresses}\
						{$value}fail} frFs

end
