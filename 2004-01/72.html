Hi Kishor

If you install lynx and cups you can use the web interface of cups.

Lynx is a text web browser, the command line you want is

lynx kanga:631

where 'kanga' is the ip address or name of your box.

The navigation keys are ?=help  <- = previous  -> = next   q=quit
