ajith kumar <ajithkumarbolla@yahoo.co.in> asked:
> lsof lists open files.
> Is there an API available?i.e,

There is no standard for determining what files are open in unix.
You can produce a list of open files for recent versions of Linux from
/proc/*/fd/*.  A general solution for all unix is to look through the
user table in kernel memory (/dev/kmem).
