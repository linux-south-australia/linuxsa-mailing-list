Re: Fedora Core 1 vs. Adobe Acrobat

On 07/01/04 16:26, Ben Williams wrote:
> Howdy all,
> 
> has anyone had any luck getting adobe's acrobat reader (available from 
> http://www.adobe.com/products/acrobat/readstep2.html) to work with 
> Fedora Core 1? i've tried installing it and running it, but it just 
> exits with the somewhat enigmatic "Aborted", thus:

wow, how embarrassing when you solve your own problem 5 minutes after 
posting to a mailing list. if anyone's interested, the solution was to 
set LANG=C :

[benw@mirage bin]$ printenv LANG
en_AU.UTF-8
[benw@mirage bin]$ ./acroread
Warning: charset "UTF-8" not supported, using "ISO8859-1".
Aborted
[benw@mirage bin]$ LANG=C
[benw@mirage bin]$ printenv LANG
C
[benw@mirage bin]$ ./acroread
[benw@mirage bin]$
