Re: Peoples experience with Clarkconnect?

Clarkconnect / ipcop / smoothwall are all dedicated firewall dists, any
left over space is used for logs and squid proxy. They do not co-install
with any other OS, so don't try and install it onto a machine with
anything important on it ;)

The specs are fairly low, any machine that is pentium or higher, with a
couple of gig HDD and 32mb+ of ram will make a good box.

To install you burn the ISO to a cd, and stick in the cdrom drive and
reboot. If you machine is set to boot off cd the install will start and
you get asked a few questions and it should install fine. One installed
you useally have to use a browser to access the web interface, but check
the dists manuals.

I would install Smoothwall v2, I've always been impressed with the speed
they release patches for exploits etc... Never used ClarkConnect.
