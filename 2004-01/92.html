Hi Kishor,

An alternative to cups.
LPRng & apsfilter.

Install both, back-up & delete your /etc/printcap file then run the
apsfilter setup & answer the questions, very easy. Apsfilter will write
a new /etc/printcap file based on your answers.

Apsfilter uses either the native printer drivers contained in
ghostscript (which most good distros come with standard these days), or
the official HP drivers which you can download seperately.
