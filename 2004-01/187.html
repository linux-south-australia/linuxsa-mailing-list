On 05/01/2004, at 4:21 PM, vijay purohit wrote:

> Lsof is used to list the open files.Is there any API
> available? Can it be called from C

 From the man page for lsof:

SEE ALSO
        Lsof  versions  2  and 3 have been tested under older UNIX 
dialects.  They are available via anonymous ftp from
        vic.cc.purdue.edu in the pub/tools/unix/lsof/OLD directory.

        access(2), awk(1), crash(1), fattach(3C),  ff(1),  fstat(8),  
fuser(1),  gethostname(2),  isprint(3),  kill(1),
        lstat(2), modload(8), mount(8), netstat(1), ofiles(8L), perl(1), 
ps(1), readlink(2), stat(2), uname(1).


Thus, you'll have to roll your own using fstat, access, and other 
system calls above. I'd recommend reading the source for lsof and 
determining how the utility does it, then coding your own routines (or 
checking the licence of the original code and including some of it).
