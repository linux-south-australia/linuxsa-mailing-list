Hi David

I've noticed people are talking about your Billion as a possible 
security problem.

Ok, even if the Billion IS the problem, they have to get through that 
AND Windows XP.

So: (here is what i would do)
 Presuming they DID get thru the Billion easily, then they have to get 
through Windows XP.
 Two ways to do this:
    1. Vulnerability in the OS
    2. Trojan/Virus/Malicious program on WinXP letting them connect.

I would initial suspect a Trojan/Malicious program - these are probably 
a lot easier to use than an OS vulnerability (depending on how 
up-to-date your Windows install is). There are plenty of the so called 
'Back Door' programs out there (e.g. Back Orifice).

However, you may as well fix both these possible problems:
  - Running Windows Update should take care of OS vulnerabilities and 
running an updated virus scanner should get rid of Trojans...


After i had done that, i would start looking at my Billion or the 
possibility of installing another firewall..

J
