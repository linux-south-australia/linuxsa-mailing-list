> I found the perl-Net-CIDR-Lite-0.15-1.noarch.rpm file

>         perl(Carp)   is needed by
> perl-Net-CIDR-Lite-0.15-1
>         perl(strict)   is needed by
> perl-Net-CIDR-Lite-0.15-1
>         perl(vars)   is needed by
> perl-Net-CIDR-Lite-0.15-1

as root, type in the terminal/console:

perl -MCPAN -e 'install Net::CIDR'

This will install the Net::CIDR package which you require, and automatically 
grab all it's dependencies, and install them for you.  It'll take some time 
if you're on a modem, and if you've never run CPAN in perl before, it'll ask 
you some setup questions.  I'm using Mandrake 9.1 too, and it works.  If you 
ever need to install more perl modules, substitute Net::CIDR in that command 
for the module's name which you want to install.
