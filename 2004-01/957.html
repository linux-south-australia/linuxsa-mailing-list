RE: Peoples experience with Clarkconnect?

Yep..Been running CC for the last 18 months. Currently running version 2.0
Out of the box it comes with FTP server, Apache, PHP etc

I installed MySQL as well. My current install of CC is in Standalone mode as
I now have my modem/router doing the authentication, but is equally at home
as a Gateway device
Official and unofficial updates are here
http://www.clarkconnect.org/webapp/updates.jsp

Pretty much anything that you can run on Redhat will run on CC and it works
well out of the box. It's VERY easy to set up. Friend of mine that knows
NOTHING about Linux got his CC box up and running.
