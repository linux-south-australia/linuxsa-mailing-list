RE: DVD rippers

>Looking for a DVD ripper. Any recommendations for those under 
>Linux and what libraries/dependecies I might need (if possible)?

Dvd::Rip is without doubt the best one I've used (Windows or *nix).  

The website lists a bunch of binary packages for different distros, YMMV
depending on how dependencies are handled ;)  I use Mdk 9.2 at home and
using the PLF urpmi sources makes it pretty simple...

http://www.exit1.org/dvdrip/

Also I recommend K3b, though I've managed to break this at home due
to installing the latest beta of KDE :)  It needs a few extras like
transcode (as does dvd::rip) but all of these can be found in the 
above PLF or contrib sources for Mandrake.

http://k3b.sourceforge.net
