Suse Linux Wine Rack

Hi all

Here's a new offering from Suse.

http://www.everythinglinux.com.au/item/winerack

SUSE LINUX Wine Rack


SUSE LINUX without limits! Ever thought of using your Windows
applications on a stable, secure operating system?

We make it possible - with SUSE LINUX WINE RACK!

SUSE LINUX WINE RACK allows you to install and utilize numerous
Windows-compatible applications and games in Linux. The well-known
CrossOver Office and CrossOver Plugin from Codeweavers and WineX from
Transgaming enable you to use software such as the following in SUSE
LINUX 9.0:

* MS Office 97, 2000, and XP:
o MS Word
o MS Excel
o MS PowerPoint
o MS Outlook (only 2000)
o MS Access (only 2000)
* Adobe Photoshop 7
* Lotus Notes
* Intuit Quicken
* Apple QuickTime
* Internet Explorer
* Top games such as:
o Half-Life: Counter Strike
o Diablo 2
o EverQuest
o Warcraft III
o Battlefield 1942
o Starcraft
o Civilization 3
o and many more

The fascinating 3-D arcade game MarbleBlast from Garagegames is included
as an extra treat. 

Content of the CD:

* CrossOver Office 2.1
* CrossOver Plugin 2.1
* WineX 3.2
* Marble Blast


If anyone installs it, I'd like to hear how it goes.
