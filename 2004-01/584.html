Grouse Grep 3.00 released, GPL2

G'day,

I've been preparing for the talk this afternoon on "Grouse and
Grep".

Re-reading the sources, my own assesment was that I was not
comfortable that the copyright statement and related licencing
terms and conditions in the sources fulfilled the requirements
of the Open Source definition required by the conference:

	http://linux.conf.au/cfp.cgi
	http://www.opensource.org/docs/definition_plain.html

I've decided to re-release Grouse Grep as Version 3.00, with
the main change that the copyright statement in the sources has
been changed to conform to the description advised by the GPL2,
and the file LICENCE is now the GPL2 document.  I've also
removed the statement "All rights reserved" from each copyright
line.

This new release of Grouse Grep is available at:

	ftp://ftp.grouse.com.au/clients/grouse/ggrep-3.00.tar.gz

cheers,

behoffski
