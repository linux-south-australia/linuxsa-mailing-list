If you read this far, you're in for it :-)

Download *pilot* from my site, then:

	mkdir ~/.xcopilot
	ln -s /your/dir/pilot.rom ~/.xcopilot
	xcopilot   (this works, try it!)
	xcopilot -serial (this doesn't work, quite.)

There's an reasonable set of instructions at
http://www.snapgear.com/tb20020807.html, although you shouldn't need
them. Creating your own build environment is fun but requires time and a
little debugging ability to make it all work.

Some ppp instructions are at
http://patriot.net/~jcaple/uclinux_faq/install_howto.html . I have been
using somewhat different parameters in my testing (noauth, local,
-detach, crtscts) but basically as soon as I see characters echoed in
minicom I'm sure it will work with pppd.
