> I dont know VB - how close is the code to VB ?   (I could read the FAQ,
> but Im being lazy, and thought you should share the answer), since I
> know plenty of people who would not touch Linux since all they know is

Its pretty close, has a GUI designer, and interfaces to Mysql as well. That 
required installing the mysql-devel libraries and rebuilding once I realised 
i'd missed them, but the demo mysql thing worked first time as well once it 
was built correctly.

The guy wrote it to be very close to vb syntax, because:

"I took from Visual Basic what I found useful : the Basic language, the 
development environment, and the easiness to quickly make programs with user 
interfaces."

Luckily, however:

"But I dislike the very bad level of common Visual Basic programmers, often 
due to bad pratices imposed by the bugs and strangeness of this language. So 
I will try to make Gambas as coherent, logical and reliable as possible, and 
I hope that Gambas programmers will make effort in return ! ;-)"

And note that:

"The development environment is written with Gambas itself, so that I can show 
the abilities of the language. And it is very useful for debugging !"

All from:
http://gambas.sourceforge.net/introduction.html
