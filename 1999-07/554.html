On Tue, 20 Jul 1999, Robert Mibus wrote:

> does anyone know how to set up (or can point me in the direction of docs
> to) a dumb terminal using a 286 or 386, preferable via the parallel or
> serial port. also, whatever happened to the person with those $1 vtys?
> 
> mibus
> 

its quite simple, setup the dumb terminal in say, vt100 mode, attach a
null modem cable to it and to your serial port of ur linux machine, then,
edit /etc/inittab and put a line in similar to this

s1:12345:respawn:/sbin/getty 19200 ttyS1
   
12345 <-- runlevels 
/sbin/getty <-- software to run
19200 <-- baud rate that u have set the dumb terminal to run (19200 is a
good one, no packet loss and useable)
ttyS1 <--- com 2

then as root
type "init q"

and bobs ur uncle, u should get a login prompt to ur linux box

have fun
