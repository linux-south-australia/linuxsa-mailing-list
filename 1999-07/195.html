> Is anybody else noticing these extra >'s that appear sometimes?

The traditional Unix mailbox format separates messages by a "mail envelope"
which is a line of the form "From sender's-address date".  A consequence
is that any line in your mail message which starts with "From " must be
escaped, and this is done by prepending a ">".
