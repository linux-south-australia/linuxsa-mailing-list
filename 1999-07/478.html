Presentation:

    FreeBSD        by Mark Newton
    =======

	Mark will be discussing another free Unix operating system that
	runs on x86 based architectures.

	Discussed will be the following :
		o History
		o Development Methodology
		o Features of FreeBSD
		o Available software (including the 'ports' collection)
		o Some Myths about FreeBSD
		o Future directions
