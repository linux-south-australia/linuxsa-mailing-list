Des,
  Window's problems aren't as easy to corect but here's a possibly cure.
If your running win98 run the System Information Program (somewhere under
accessories - if installed) One of the options within it is a system file
checker. This should check for any corrupt files.

There are a few other possibilites on why this error may occur. 
	1> is it really an mpeg file?
	2> Can you play it by starting MediaPlayer then opening the file
           in that.
	3> Does renaming it to .mpg fix the problem.
	4> Do other mpegs play ok? If so it may not be an mpeg
	5> Last but not least (this normally solves any windows problem)
		Reinstall windows.
