Re: Adding swap space

On Thu, 22 Jul 1999, lai kokcheong wrote:

> 
> I have 40 Mb swap partition and I only have 90 Mb free space on my Linux 
> partition.I've plenty of free space in my Windows partition (FAT32) .Could I 
> create a swapfile in the windows partition or mount another swap directory 
> to my windows partition w/o affecting the data intergrity of windows 
> partition ?

There is no reason you can't use a normal file as a swap partition. The
only downside is if you forget to mount that partition you have no swap.
Also it is slower than partition access.

To make a swap file:

echo > swapfile
dd if=/dev/zero of=swapfile  count=100k    (This makes a 100k file)
/sbin/mkswap -v1 swapfile                  (make it a version 1 swap file)

Then just add it to /etc/fstab.
Just make sure that you mount the partion it is on in fstab before the
swap file.
