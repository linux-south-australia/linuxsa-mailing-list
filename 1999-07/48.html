Re: Loop device

> And, if the file for the loop device resides on a normal partition, then
> the file cannot exceed that size either. (I found this out when
> recording videos to harddrive, but the harddrive was fat32 mounted under
> linux...And fat32 has a 2gb filesize limit, which was annoying because
> it was a 10Gb drive that was blank, and I couldn't understand why things
> weren't working.)

Actually that is wrong, FAT16 (including win95, and win95a) has a 2GB
limit, and a 4GB limit under Windows NT. Fat32's limit is much larger, I
don't have how large to hand, but I believe it will handle a 10GB HDD.

The problem is that Win95B's install boot disk only has the old Fat16
FDisk, and Format, the Win95B ones are on the CD. So you have to install
Win95B, create a boot disk with CD drivers then Re-Install. (or find
another win95B instalation prior to this)

This has been a public service announcement :)
