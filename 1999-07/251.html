I have had similar experiences to Nick, on some of my dual boot
systems...particularly when I split a single hdd... occassionally win95 could
see there was a partitiion there, but couldnt read it so presumed it was
unformatted... as Nick did, I ignored windoze request to format it!! I would
guess that having now blanked the partition with half a BSD install, windoze
cant make any sense of the partition so has decided its not there anymore :)
sort of win95_ostrich syndrome I suppose !

In all cases I fixed this problem by deleting both windows & linux linux
partitions and repartitioning for linux only !! This is, IMHO the best cure for
any of windoze more bizarre attributes!! ;)
