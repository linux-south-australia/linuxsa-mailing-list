Re: Removing Linux

1. Beg, borrow, steal or even make a DOS/win95/98/2000 boot disk.
1.a copy fdisk, format, sys, copy, and any other utilities you may need
including CD-ROM drivers onto floppy

2. boot system with boot disk
3. run fdisk and delete all non-dos partitions
4. reboot with boot disk
5. run fdisk /mbr

6. reinstall Linux,  oops, OS of choice  :->

7. remember important data that WAS on HDD and panic cause you forgot to
back it up*

*Feel free to miss this step and backup first.

That be how I do it and the more I do it the less I need step 7.


If I missed a step, I am sure someone will correct me :->
