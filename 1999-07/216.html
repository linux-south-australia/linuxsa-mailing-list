This is not accurate in the case of the KDE packages.  I have already
posted to this list the URL of a web site that contains plenty of KDE
stuff ready for a debian system.  They have core KDE 1.1.1 packages,
KDE 2.0 development packages, and kde-devel (constantly updated from
CVS (1.1.x strain)).

To obtain the debian packages, add one of these lines to
/etc/apt/sources.list :
      deb http://snowcrash.tdyc.com potato kde contrib rkrusty
if you're running "unstable", or
      deb http://snowcrash.tdyc.com slink kde contrib rkrusty 
if you're running "stable" (Debian 2.1)

For more information, see http://kde.tdyc.com/Debian/
