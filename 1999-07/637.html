Did you set the link up for the new server? 
	Ie: ls -l /usr/X11R6/bin/X    should show a link point to the
					xserver you downloaded

If you did.

try putting: Option "noaccel" in your XF86Config file 
(type: find / -name 'XF86Config') to find where it is located.

Ie this is it inplace with my videocard

Section "Device"
    Identifier  "Tseng ET6000"
    VendorName  "Tseng"
    BoardName   "ET6000"
    videoram 2048
    Option "noaccel"      # when problems with accelerator
    #Option "hw_cursor"    # Use hardware cursor (see docs for
correct
EndSection

> but it's late, I'm tired! Any help would be appreciated...!
