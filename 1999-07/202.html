HI

Postmaster must have a -i switch

and the file pg_hba.conf for you database must have a line allowing
connection to you machine something like the following

local        all                                         trust
host         all         192.168.50.0     255.255.255.0   trust
host         all         127.0.0.1     255.255.255.255   trust
