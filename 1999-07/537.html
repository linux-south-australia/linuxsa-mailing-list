>My specific queries are as follows:
>1) What packages are available (preferably .rpm and/or .deb, but not
>necessary), and which should be avoided?

I've set up an IRC server on RH5.2. I used a server called Elite 1.3. Was a
dream to set up, the scripts pretty much do it all for you. Its a case of
untar it, run the ./Config script, run make, run ./makeconf and then run the
program. You don't need to do it as root (in fact, it won't let you)

>2) Where to get recommended packages from?

I'll put the tar file on my public ftp directory. (Done, the URL is
ftp://ftp.bachblue.com.au/pub/Elite1.3.tgz)

>2) What requirements (real .. not specified) are there?   ie. PC
>requirements etc.

I'm using a 6x86-166MMX with 64Mb SDRAM and a 1.2Gb IDE hard drive. No
problems at all.

>3) What warnings are there to heed?

I haven't had any problems with it.

>4) How much administration time will be required once up and running?

1) Create (Update?) the ircd.MOTD file and that's about it. I haven't had to
make any changes at all
