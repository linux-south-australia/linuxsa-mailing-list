Re: uninstall

Hi,

> I'd like to know, if there's any good way to uninstall s/ware that have been 
> installed using makefile. e.g scenario:
> let say I am trying to install a s/ware, & i've done these:
> ./configure
> make
> make install
> Let say after that, I would like to uninstall the s/ware. Is there any best 
> way to do that?

I don't know about a 'best' way, but when I install the package (do a make
install), I always direct the output to a file (since make install always
outputs when files it's copying and where). Once you have this, it's as
simple as going through that file and deleting all the files it mentions.

that's my way, anyway :)

good luck,
