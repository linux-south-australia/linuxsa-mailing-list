Re: Adding swap space

lai kokcheong wrote:
> 
> I have 40 Mb swap partition and I only have 90 Mb free space on my Linux
> partition.I've plenty of free space in my Windows partition (FAT32) .Could I
> create a swapfile in the windows partition or mount another swap directory
> to my windows partition w/o affecting the data intergrity of windows
> partition ?

Only if you can shrink your windows partition to make room for another
(swap) partition. You can try to do it with fips, that comes with every
distribution CD, in RedHat under dosutils/, I think.

FIPS usually allows one to shrink VFAT partitions without losing data.
This is not a warranty, read the FIPS docs.
