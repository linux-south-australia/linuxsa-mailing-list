Hi Mike,
Find someone with Windows '98.  You will find a utility in \WINDOWS\COMMAND
which I think is called cvt.exe which can be used from a DOS session (boot to
command prompt..don't uses a DOS shell from Windows) to convert to FAT32
without loss of data .  Alternatively there is the Windows based tool as well
called cvt1.exe in \WINDOWS, which also requires cvt32.dll.  Hope this helps.
regards .. Don
