Could you give us some information on your system?

(The first thing to check is the obvious one: If it is saying you're
running out of memory, then how much memory do you have, and how much is
used up? If you have less than 64mb of ram, and no swap, then I can
imagine you having trouble getting a default RH6 install to run X and
netscape and gnome simultaneously.)
