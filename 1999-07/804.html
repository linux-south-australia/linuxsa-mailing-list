Re: Vmware

Win 98 works ok.

Vmware is very handy Small Office/Business tool. Once set up Win98 will
run easily as well as it does on its own Hard Drive. And it runs windows
apps just as well. A system with a lot of grunt is needed. My 233mzP11
and 128mg ram is very slow. Once you have allocated space on your disk
(setup asks for 1 Gig) it will need to be formatted (a bit hairy).
Vmware prefer an upgraded X server to be used (available at their web
site). Install this first. While running it will use approx half your
ram, but this can be adjusted. -Good product.
