Re: mounting a cdrom

/dev/cdrom is a symbolic link to your cdrom

check that the link now points to the correct device
(assuming you are using ide)
primary master is hda
etc.
secondary slave is hdd

if you are using SCSI then check the link points to the correct SCSI
device

I can't suggest anymore than that without more info!

hope it helps!
Jarrad
