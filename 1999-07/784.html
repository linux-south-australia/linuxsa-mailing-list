rlogin problems with RH6 fixed !

Hi All.

I managed to fix the problem myself...but here's what it was in case any of
you run into this as well:

There were a couple of problems with the file /etc/profile.d/kde.csh. I had
to replace the line 'kdepath="${KDEDIR}/bin"' with 'set
kdepath="${KDEDIR}/bin"'.
There was also a problem with the 'if' statement, so I changed it from the
default 'if ( echo ${PATH} | grep -q ${kdepath} ) then' to 'if ( `echo
${PATH} | grep -q ${kdepath}` ) then'.

Now it works fine !
