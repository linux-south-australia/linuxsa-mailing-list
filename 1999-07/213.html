Re: Linux SA mailing list... Send ONLY to Linux SA

On Wed, 7 Jul 1999, John wrote:
> |"/bin/sed 's/Resent-From:.linuxsa/Reply-To: linuxsa/' >> $MAIL"

Well no, this is terrible for three reasons.  Second, it fails to lock
your mailbox and there is a window of opportunity during which multiple
messages can get intermingled; thirdly, if the sender placed a Reply-To
header in the message this will result in two headers (with unpredictable
results); and first of all, it is improper for a mailing list to direct
replies back to the list.  This first point has been specifically discussed
before, and for much the same reasons it is inappropriate for a user to
insert a Reply-To header.

Also, of course, it totally fails to stop multiple replies.
