RE: KDE state AND Star Office

Just a hint on installing so 5.1

when running setup for the first time run: setup /net
This installs a network copy.

Then once installed run: setup as a user to install about 1mb to the that
users home directory. It's very handy when more than 1 person uses the
same pc :)

I also like how SO integrates with kde (view -> integrated desktop).
If only they could make it fast :(
