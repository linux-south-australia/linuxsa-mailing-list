AARNet used to provide a e-mail/fax gateway for Australia
via tpc.int.

We no longer do so simply because offering the gateway
no longer provides any value to AARNet members.

Internet e-mail is now globally and cheaply available
so there is little point in gatewaying it to the fax
network.  People wishing to recieve e-mail from the
Internet can, in the worst case, go to an Internet
cafe and use hotmail.
