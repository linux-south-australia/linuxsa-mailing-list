TTHE RIGHT TOOL

Try to type "df" at the command prompt. You should be able to do this
whether you are root or a normal user. If you can get into X and GNOME,
you may be able to use the GNOME disk space reporter tools (in the main
system menu under utilities).

However, I'd suggest that "df" is the simplest. Once you've worked this
out you'll get a display such as this:

Filesystem	1k blocks	Used	Available	Mounted On
/dev/hda2	9000		9000	100%		/

This should be enough info to work out which one is not working
properly. Emil me again if you need any further info than what I've
said...
