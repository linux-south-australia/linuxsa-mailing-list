HI Brandon

I have RH6  allowing psql and ODBC access, so there is no problem with the
version in the Distro.

There is a file in your database directory call pg_hba.conf this file
needs to have a line added to allow you local ethernet to connect to the
database.

I suggest you use the postgres account and its password until you have got
a connection

Hope this helps

Mike
