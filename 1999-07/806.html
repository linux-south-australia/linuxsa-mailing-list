Re: Vmware

You can also use VMWare to run windows instaled on an existing partition.

This allows the following 'problem'..
If you set VMWare to boot from your Linux/Windows partitioned hadware
it will loyally start lilo (wow!) and then if not caught in time, will start to
boot the default OS, linux (isn't it for everyone?) which will then crash your
system.

The documentation warns about trying to run VMWare inside of
VMWare.

There is also the lesser problem of getting the screen resolution etc.
to work in both as you need different drivers.. but thats a windows
profile problem (and another prompt at the windows boot!)
