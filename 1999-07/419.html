RE: Gnome

Redhat 6.0 seems to do things differently..  after following thru some
script or other, I found that the X startup looks for a file called
/etc/sysconfig/desktop, which should (if it exists) contain a single line,
with the name of the window manager you want to use.

echo kde >/etc/sysconfig/desktop
echo enlightenment >/etc/sysconfig/desktop

or whatever..  check the scripts for the exact options :)

I'm not sure I like this..  RedHat's becoming horribly 'customised'...  but
I suppose it's all in the name of progress :)  New users should all be aware
of these small but significant idiosyncrasies, and the fact that they are,
in fact, RedHat standards as opposed to any Unix-type standards.

Not that I profess to know much about Unix-type standards..  except the
filesystem has to include /etc, /usr, /bin, /var etc etc :)

But as for the *order* in which to install gnome's packages?  ummm there is
probably an install script somewhere that will do the job for you.. there is
for kde 1.1+... check the gnome ftp site, wherever it may be :)
