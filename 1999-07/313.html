Re: Cd Burner

Hi there,

Have a look at the cd burning howto on any LDP mirror or the redhat 6.0
cdrom, I am currently using a Panasonic 7502 (8 read 4 write) on an adaptec
2940 scsi adaptor with xcdroast on Redhat 6.0 - this setup is so much better
than it was with win95 its not funny, no more buffer underrun errors !! Yay
Previous to this I was using a Sony 926, this was ok but I changed for a few
reasons.  If you have a scsi adaptor already definately go this way,
although some of the newer ide burners are suposed to be quite good as well.

Have a look at:

http://www.fokus.gmd.de/research/cc/glone/employees/joerg.schilling/private/
cdwriters-1.8.html

for supported cd burners.  Hope this helps!!

Cheers,

Michael.
