gdm Login Screen

RedHat in 6.0 have included their logo in the gdm login screen Red on Blue
black ground which isn't my choice of colours. I found that this is easily
changed by editing the /etc/X11/gdm/Init/Default file. I have changed the
colour code too "000000" not knowing the colour code for black but producing
the same result none the less, and I changed the
/usr/share/pixmaps/redhat/redhat-transparent.png to the rhad.png in the same
directory, all looks pretty hot now! So if your wanting to change it I
thought I would save you some time.
