Re: Multiple Mail Servers on One machine

Hey There,

Not sure if anyone else has replied to this yet or not (been in Melbourne
for week and a half, 354 emails on this list alone when I got back!) but
there are two ways you can do this, depending on how you need it to work.

1) There is not going to be the same username on any domains, ie, you
**don't** need to have fred@domain1.com.au and fred@domain2.com.au to be
completely different email addresses.
2) You *do* need to have the same username in more than one domain.

In the case of 1), it is fairly easy to set up. Just add the domain name to
the /etc/sendmail.cw file and restart sendmail. My sendmail.cw file looks
like:

########## START /etc/sendmail.cw ##########

# sendmail.cw - include all aliases for your machine here.
bachblue.com.au
ross.wattle.id.au

########## END /etc/sendmail.cw ##########

Then you just have to either create users for each person you need. If you
need 2) I'm afraid I can't help you, but have a look at
http://www.sendmail.org for starters.

Hope that helps
