I too run a Voodoo2, alongside my S3V - the problem you're talking about is
to do with the windows 3dfx -drivers-, and a conflicting report of memory
usage from the S3.  Glide works fine, as does 3dfx mode when you've
configured it correctly under windows (not difficult).  If you're going to
use the Voodoo2 under linux, at the moment Glide is what you'll be using
(I've got it successfully working, with no problems installing drivers etc).

I'm not sure what to tell you to help you fix your Alliance Promotion..
just go to the xfree website and read about your card (they tend to have
detailed information on each supported card).  You may need to disable some
acceleration options, or something.  Maybe the Voodoo2 is conflicting?  When
the Voodoo2 / S3 lockup happens, the system suddenly and completely freezes.
