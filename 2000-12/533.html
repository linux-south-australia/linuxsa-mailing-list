Re: fat32 partitions read only

> /dev/hda5               /mnt/d                  vfat    defaults        0 0
> /dev/hda3               /mnt/win95              vfat    defaults        0 0
> 
> both partitions come as read only, i'm sure i had this configuration before
> with the petitions having write permissions, has anybody got any ideas on
> where to look?

man fstab (or mount), or:

/dev/hda9	/mnt/crap		vfat	users,rw,exec	0	0

users: allows users to access mount point
rw   : read write access
exec : allow executable files to be executed as programs

there are many more helpful options, you really should check the man pages.
