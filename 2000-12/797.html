Re: Formatting tapes?

Martin Stacey wrote:
> 
> Is there a command to format a tape?

Modern tapes aren't formatted.

There's a convention that the first file on the tape has
a special structure and is called the "tape label".

But UNIX breaks this convention and will happily
write to an unlabelled tape and treats the first
file on the tape no differently than any other file
on the tape.

If you are seeing a complaint about a lack of a tape label,
then you need to use the utility that came with the software
that is complaining to label the tape.  For example, the
Amanda backup utility has a program called "amlabel".
