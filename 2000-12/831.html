> Can you tell me more about this "make menuconfig". In which directory can I
> find this or isn't it standard Linux? What must I do? etc. 

First, find the Kernel-HOWTO. Go to www.linuxdoc.org (or the nearest
mirror to you listed there) and look under 'HOWTOs'. Read it all, but 
section 3.3 "Configuring the kernel" is what you need. NTFS is under
Filesystems on the main menu that 'make menuconfig' gives you.

All of the above assumes you have a version of Linux installed and running
in a partition on the same machine. I'm not sure if this is the case for
you.

> I'am desperate
> to install this NTFS file system to save information which is on my Windows
> NT partition before re-installing a new operating system

Careful, desperation often leads to losing data, I've seen it many times :-)

Depending on your setup the easiest solution might be to access the data
over a network. Or maybe you could consider burning a CD.
