Re: Background of redhat login screen

On Tue, 12 Dec 2000, Harvey.Davis@dsto.defence.gov.au wrote:
> Does anyone know how to change the background screen 
> that is displayed at the login window under red hat 7.0

Not sure about RH7, but on RH6, the file /etc/X11/xdm/Xsetup_0 contains
a line or lines that looks something like:

/usr/bin/xsri -geometry +5+5 -avoid 300x250 -keep-aspect \
	/usr/share/pixmaps/mccorp/mccorp-transparent.gif

I have changed the above to include the logo I want - usually it would
be the file redhat-transparent.png.  You can either delete this line for
no logo, or replace the path to the .gif with a file that you like.

PNG and GIF are supported, and maybe JPG.
