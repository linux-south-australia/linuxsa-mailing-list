Re: Staroffice & Mandrake7.2(newbie)

Darrin Ritter wrote:

> Does any one know how to batch-convert truetype fonts into type 1 fonts?

Search for "ttf2pt1".  There's also a "ttf2pfa" but this seems
less capable.

You can run any command in batch like this:

  for i in *.ttf
  do
    j=`basename $i .ttf`
    ttf2pt1 $i $j.pfa
  done
