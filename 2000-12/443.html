RE: SAMBA question

Brandon -

I think I've found what you were looking for.  If you run "smbstatus -b" on
the server, you'll get both the usernames and the machine names.  Thus, if
you know the username, you can use the output of this command to provide you
with the machine name necessary for using "smbclient -M".

(I still have had no success with the syntax for referencing IP address
instead of machine name when using "smbclient -M".)
