Corel Linux uses Debian packages and packaging system and layers an
installer and some pretty desktop work on top. They have gone very quiet
on the development front over the last few months however that isn't so
serious because nearly all the value is in the packages, which is Debian.

One personal experience of mine is that I was stuck wanting to install
Debian without any CDs, but I did have a Corel CD. I installed Corel and
then apt-get'd my way to an updated system. After getting rid of a couple
of GUI bits and pieces (eg the fancy VGA splashscreen that is displayed
while the kernel boots, covering up the messages) I can't recall seeing
any vestigas Corel for a long time.

So, Corel is interesting and worth a try for the experience, a bit dated
(eg the installer might not know about recent hardware and its an old
kernel) but easily updated to Debian.
