Re: Perl and MySQL problem

> I'm trying to install the DBD::MySQL package and am having a problem.
>
> My comments are marked by **
>
>
>[root]#  perl -MCPAN -e 'install DBD::mysql'

Try this instead:

perl -MCPAN -e 'install DBD::mysql::Install'

(basically, DBD::mysql has an 'install' optional port that seems to work
better on linux)
If you're using FreeBSD, install it from the ports collection instead of
CPAN.
