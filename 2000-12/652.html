G'day,

First of all, Eggdrops are an IRC bot (robot)
they manage channels mainly, but can do pretty well anything via/on IRC.

Secondly, mIRC is a windows program, so hence BitchX mIRC does not compute.

Thirdly & finally, An Eggdrop is a stand-alone program, it does not go with
mIRC or BitchX
It is compiled in C and runs via TCL scripts and some modules.

check out these
http://www.eggheads.org
http://www.eggdrops.com
http://irc.themes.org

Hope this helps,
