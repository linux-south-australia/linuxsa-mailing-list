Unfortuantly libstdc++ is part of the gcc compiler so any RPM upgrade for
the libstdc++ packages means gcc (and its other packages) have to be updated
along with it. There is no way outa this (its why i hate package managment
sooo much).

So either your stuck with getting all the packages OR you go downaload a new
version of the GCC source (which libstdc++ cmae with) and compile/install it
yourself. This is also a good thing if your running RH7 as it fixes the
whole gcc/kgcc RH7 screw up.
