Re: Mail attachments from command line

On Tue, 5 Dec 2000, bmonten wrote:
>     is there a way I can attach a file to a mail message using a command
> line ? For instance, if I just wanted to include text in the body of the
> message, I could do something like:
> 
> mail backups@monza -s "Backup Status from Hockenheim `date`" <
> /var/log/dailybackup.log

mpack [-s subj] [-d file] [-m maxsize] [-c content-type] file address...

nail [-iInv] [-s subject] [-a attachment] [-c cc-addr] [-b bcc-addr]
to-addr...

Or interactively:
pine [-attach file] etc...
