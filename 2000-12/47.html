You can use dpkg 
But you don't need to compile your own ( unless that's the way you really want
to do it ) as the package was corrected in the disto so you can just do

apt-get update ; apt-get install tcpdump

and you will get the updated version from the proposed-updates 
Assuming your sources list includes something like

    deb ftp://mirror.aarnet.edu.au/pub/debian/ stable main contrib non-free
--> deb ftp://mirror.aarnet.edu.au/pub/debian-non-US/ dists/proposed-updates/  <-----------
    deb ftp://mirror.aarnet.edu.au/pub/debian-non-US/ stable/non-US main contrib non-free
    deb http://security.debian.org stable/updates main contrib non-free
