Re: printing from Star Office 5.2

Darrin Ritter wrote:
> 
> Ive installed Star Office 5.2 as a /net setup and am having trouble
> getting it to print
> to my printer through the CUPS printer driver in Linux Mandrake 7.2
> 
> does anyone have any Ideas?
> 
> regards Darrin Ritter

As root you must run 'spadmin' (from memory).  If you run soffice as
root then you can run the printer configuration tool from there.  You
need to configure Staroffice printers separately to the Unix printers. 
The soffice printers use the underlying printer system, but staroffice
doesn't know about that until you tell it.
