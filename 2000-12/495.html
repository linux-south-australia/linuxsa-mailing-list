> by
> a five letter random string. I was wondering if anyone knows what creates

they could be anything :)

ways to find out more:
fuser <filename> - will tell you which user/process has that file open. 
                   if you get a number from this command, do
                        ps -eaf | grep <number>
                   and see what process has it open
file <filename>  - might tell you what type of file it is. However, it
might
                   also tell you `data' `ascii text' `socket' which
isn't
                   too helpfull.
