* I come from a Solaris background and are new to Linux. In 
* Solaris they have
* a command "boot -s" to boot a machine into single user mode 
* from a halt
* position. Does Linux have a similar command?

LILO Prompt:
linux 1 
Hit tab for a list of boot names, linux should be the default one. There
might also be a linux-secure if you're using mandrake, for example.

Command Prompt:
init 1

There is a list on the man pages of the different init levels, but I can't
remember which page. So if you type "init 1", you go into single user mode
(I think that's the right level...).

Under /etc/rc.d/ are a bunch of directories. There's an init.1 directory,
for example, that controls which programs must be turned off (start with an
S), and which programs must be started (start with a K). I think the
runlevels are as follows:
0:	Halt
1:	Single User
2:	Don't know :]
3:	Multi User
4:	Not used...?
5:	Multi User with X
6:	Reboot
