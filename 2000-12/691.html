Re: NTFS file system

On Mon, 18 Dec 2000, 752304 wrote:

>  Can anybody tell if it is possible to access data on a NTFS partition
>  within Linux???

Accessing an NTFS filesystem from Linux has been possible for some time
now.
Unfortunately it requires a kernel recompile to do so. (Well at least it
does with Redhat, 
I don't know about others.) Currently the read-only access to NTFS
partitions is safe but
read-write access is not recommended as it has not been perfected as
yet. Even with the 2.4-test-11
source which I compiled it did not recommend read-write NTFS support. 

If you haven't done a kernel recompile before find some docs on how to
do that (sorry can't give you 
any addresses offhand) and read them well, then read them again. Know
all your system hardware exactly 
and from there on you should not have any problems (I hope) :)

Compiling a kernel can be daunting at first for ex Windows users. Just
read all the help information
provided by the Xconfig program.
