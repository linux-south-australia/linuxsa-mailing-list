Putty Updated

Hi all

Just to let you know that putty has had a significant update, have a look a
the list below
  a.. PuTTY (the Telnet and SSH client itself)
  b.. PSCP (an SCP client)
  c.. PuTTYtel (a Telnet-only client)
  d.. Plink (a command-line interface to the PuTTY back ends)
  e.. Pageant (an SSH authentication agent for PuTTY, PSCP and Plink)
  f.. PuTTYgen (an RSA key generation utility).
All this and more can be found at
http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html

Cheers
    Mike O'Connor
