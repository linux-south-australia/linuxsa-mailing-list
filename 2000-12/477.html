Re: Background of redhat login screen

If u r using gdm and u also want to change the greeter, edit the file

/etc/X11/gdm/gdm.conf

in particular: 

[gui]
GtkRC=/usr/share/themes/Default/gtk/gtkrc

Change the path to whatever Gtk theme u wish to use.

eg use: ee /usr/share/themes/Pixmap/gtk/gtkrc to view that theme
