Re: introduction

Jerry McAlister wrote:

> Not. The 286 couldn't do multitasking.

It certainly could.  Xenix ran on a 286, as did Coherent (a very
Version 7 style Unix).  It had a "protected mode" that was very
reminiscent of PDP-11 (64K code + 64K data/bss segment).  There
was even a version of Xenix that ran on 8088/8086, but without
any form of memory protection it was trivial to crash the entire
box.

Minix ran on 8088/80286 (and above) - I believe Linus used this
to bootstrap Linux into being, which is why the first file system
format was Minix compatible.  But Linux itself (until recently,
with the embedded stuff) required the minimum of a '386.
