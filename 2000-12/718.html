Re: printing from Star Office 5.2

> As root you must run 'spadmin' (from memory).

Also:

spadmin will ask for a PPD file for your printer if it is a PostScript
printer.  You can find these in the Windows NT driver from the printer
manufacturer.

There is a generic PPD, but the results with the real one are
nicer (eg: tricks like first page on letter head, subsequent
on plain).
