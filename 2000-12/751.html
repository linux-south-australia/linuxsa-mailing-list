Not related to the original question, but ps2pdf is the best way to get
documents produced on Windows systems into PDF format.

Simply install a windows printer driver for something like the Apple
LaserWrite II/NT, or a HP Laserjet PS, print to file instead of parallel
port. When you use this as a printer, windows makes a PS file for you.

Then by using Ghostscript's ps2pdf you can turn obscure junk like MS
Publisher and Powerpoint documents into PDF files that everyone can
enjoy.
