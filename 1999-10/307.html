Emacs/Xemacs has a setting similar to this, in that it doesn't display
the ^M 's but re-writes the file back to disk on save with them.  

It was a "feature" that I thought I thought I could do without, and
luckily it doesn't come pre-set for this option. 

I mention it because if it's been turned on and the user forgets about
it (because you're not always aware of things you don't see), it can
cause the same problem/symptoms/head-scratching and head-banging.
