Re: Telnetd problems....

On Sat, 23 Oct 1999, MWP wrote:

> Hi....
> 
> The problem is simple... when i try and telnet to the linux box i get the
> error "telnetd: All network ports in use".
> 
> This seems to have started when i installed a newer kernel verison (2.2.12).
> Im using RH6
> Any ideas???

Did you remember to add support for pts?
There is two options:  Character Devices->Unix98 pty Support
                       Filesystems->/dev/pts filesystem for Unix98 PTYs 

These are used by rh6.0 and forgetting them would cause the problem
mentioned, both should be enabled.
