I suspect this has nothing to do with the switches. My SureStore gave this
message when I first did a dump to it (straight out of the box and without
reading the manual :) ). 

Try telling the dump command that the tape is really long:

      /sbin/dump -9u -c -s 100000 -f DEVICE FILESYSTEM 

If you read the manual you can work out the precise number - I just throw
in a big one because I know I won't crash the tape.
