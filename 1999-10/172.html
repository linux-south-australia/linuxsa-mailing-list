The default kernel only supports the old raid tools (0.40 to 0.50). This
is only raid 0 and linear. If you want support for raid 4 & 5 you must use
raid tools 0.90+ and this requires a patch.

The patch also makes raid 1 much more stable so I recommend it.

* The Redhat 6 kernel comes prepatched with the raid patch. *

For 2.2.12 you must use the 2.2.11 patch (none exists for 2.2.12). You
will get 1 reject, just say no to both questions. (The reject is an
already fixed bug).

If you want raid support for swap it's already there. The kernel has in
built code for raid 0 for swap (been there since 1.x ). Just set all
swap partitions to the same priority.
