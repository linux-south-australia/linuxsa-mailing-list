Re: modem speed

On Sat, 2 Oct 1999, Andrew O'Callaghan wrote:
> I ... set the line speed to 115200 (?), but [the modem] still isn't
> lightning fast.

How curious.  This question just came up on another list.  There is only
one satisfactory answer, which is that you have to read the manual for
*your* modem.

Just as putting a thick AC power cord on your computer doesn't make it
run faster, so talking to your modem using a faster serial speed doesn't
make it talk to the remote modem any faster.
