Re: telnet

I can maybe help with Q2.

Check /etc/securetty (I think RH6.0 uses this). This file says which tty's
are allowed to have root log in through them. If you put in the /dev/tty
that you come in on (use tty when logged in) into this file and save it,
then logout, and relog in as root this works.

Have fun...
