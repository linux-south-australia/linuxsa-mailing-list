G'day,

Just a quick note - I was able to get the answer to my question at the 
PHP Knowledge base I posted the URL for.

 > Unfortunately, PHP doesn't seem to "know" about the /global/ alias. Is
 > there a way to set the directory as a variable in PHP so I could have
 > the same functionality?

The solution:

  --------------------------------------
    In your httpd.conf file, put this:

    php3_include_path .:/some/path/phpinc

Then simply do:

    require "vars.php3"

and PHP will first look for this file in . and then in /some/path/phpinc
  --------------------------------------

I tested this and it works a charm.
