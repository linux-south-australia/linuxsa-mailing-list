Yeh... ECM's in cars do run relitivly slow, but they are very very well
programmed. It would be possible but very difficult to get a linux box to
replace a standard ECM. The hardest bit would be getting timing and IRQ's
acting the way the ECM would.
As far as i know, its not possible to control a standard ECM via PC because
the memory maps it uses are normaly in EPROM. Some people have come up with
replacements for the EPROM which are actualy flash ram that can then be
changed on the fly via a PC.

If you wish to do this, the best way would be to get somthing like a Haltech
ECM installed. They have an external RS232 port for monitoring and changing
of parameters. As far as i know, Haltech only have software for windows9x
machines and may not be willing to release info on thier RS232 interface
coding.

BTW... Haltech are an Adelaide based company.
