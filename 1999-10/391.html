Brian Marr wrote:
> 
> When booting RH 6.0 I get the following error messages.
> ...
> NFS statd neighbor table overflow
> ...

i didnt get this with nfs (as i dont use it), but i had a 'neighbour
table overflow' when the network service wasnt started.

run '/etc/rc.d/init.d/network status' as root.
if it doesnt com up mentioning 'Configured devices' or if it has an
error...

run 'ntsysv' as root, move down to 'network'. If there isnt a [*] next
to it, press space. Press [TAB] then [ENTER]. reboot (or type in
/etc/rc.d/init.d/network start)

hope it helps.
