Re: questions

On Fri, 8 Oct 1999, Robyn Manning wrote:

<SNIP>B
> Do I need to set interrupts and i/o for the card as in win98?
> 
> If so where do I do this?
> 
> Currently the card is set to eth0 but this doesn't initialise on startup it
> comes up 'failed'.

If you need to setup i/o for the card you would put it in
/etc/conf.modules

Ie this is my setup (eexpress will be replaced with 3c509 for you):

alias eth0 eexpress               # alias the network device
options eexpress io=0x300


Also from the kernel help for your card:

"If your card is not working you may need to use the DOS                 
setup disk to disable Plug & Play mode, and to select the
default media type."
