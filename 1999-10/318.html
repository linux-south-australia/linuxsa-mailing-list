Re: dosemu

On Mon, 18 Oct 1999 alloy@smug.adelaide.edu.au wrote:

> hi, i have recently installed rh 6.0 and am trying to get dosemu, the ms-dos
> emulator up.
> 
> i make it to the C:> prompt, but when i try to lredir the c: (or d:) drive to
> a directory i get an error.

How are you trying to use it? 
You must use in on an unassigned letter for it to work ie:

c:> lredir h: linux\fs/usr/local

will make drive h: in dos be /usr/local in linux

If you are trying to do this with drive d: at boot time you could also put
the following in the config.sys

DEVICEHIGH=C:\UTILS\LINUX\EMUFS.SYS /mnt/dosdrived

This will emulate the file system as a networked file system
