ssh-agent

Hi,

In the bash shell, I start ssh-agent, and it produces what looks like the
right output:

$ ssh-agent
SSH_AUTH_SOCK=/tmp/ssh-andrew/agent-socket-13296; export SSH_AUTH_SOCK;
SSH_AGENT_PID=13297; export SSH_AGENT_PID;
echo Agent pid 13297;

However, when I then try and run ssh-add to add my ssh identity key, I get
the error:

Could not open a connection to your authentication agent.

(after correctly typing the required passphrase for the identity key, of
course :-)

Interestingly, trying to echo $SSH_AUTH_SOCK doesn't bring up the socket of
the ssh-agent like I'd expect (it shows a blank line). However, a quick ps
aux shows that the process is indeed running.

Then, if I type:

$ export SSH_AUTH_SOCK=/tmp/ssh-andrew/agent-socket-13296

the variable is set correctly, and ssh-add now works.

Starting ssh-agent with the -s option makes no difference. Any ideas why
ssh-agent doesn't start up correctly, and what a newbie can do about it?
