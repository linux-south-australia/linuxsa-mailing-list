Re: Caldera Open Linux is slick

I have to agree with you totally too, Don! It is the end result
stability along with flexability and security that is wanted. So it is
slick as a greased pig to install, but nothing works properly (Am I
suggesting Microsoft? hmmm) doesn't make it... In most cases the
average person isn't going to use Linux. As it stands now, Linux has
come a long way since version .95 install! I think stability has
gotten to the point where Linux is a very viable product. With more
support from the commercial world M$ has a BIG scrapper in the oppsite
corner.

The only thing that is wrong is now Linux is getting extremely
commercialized. RedHat has done a good thing to make it easier for the
savy 'GEEK' to have a system up and configured is no time. The thing
that does both me is the pricing. It seems to me to be growing into
the Windows market pricing. If this keeps up where is the limit? The
price of NT??

Sometimes there is only so much that can be done to a program to make
it work the way it was intended. This is to say that I have NO
derogatory feelings toward Linux! Security is my main concern!!! Linux
is filling the bill here!

I love the ability to put a server in and office and have it do
everything that is needed AND stay working for a long period of time!
I haven't seen an NT server do this as of yet!!! One *NIX person can
control and service many many Linux machines, while it normally takes
one person to service no more than 2 NT machines. To me this is
negitive productivity and quite positive for a complete Linux
network...

I guess when I compair Linux pricing to Windows 9x pricing I am doing
an apples to oranges comparison and this should not be done. I should
do this to NT, which is still not the same in actuality. Compairing a
big ole Granny Smith apple that is sweet and juicy to a little bitter
crabapple still isn't the same even though they are both apples or
both servers...

Now that I have gotten up on soap box and spewed forth my bubbles of
ranting with the raving, I should step down. This is MY oppinion and
only MY oppinion.

Is there any takers that want to discuss this?
Frank W. Holden Jr. - Holden-On
