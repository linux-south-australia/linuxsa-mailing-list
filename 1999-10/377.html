As it turned out, under RH 6.0 it was absolutely trivial.  IIRC Mandrake
is derived from RH so it should be simple.

IIRC, it is easy, the following steps should get you going.

	1) Get your TT fonts (I put them under /usr/share/fonts/tt).

	2) Go to that directory.  Run the ttmkfdir program (part of the
		freetype package).  For more information, go to the
		freetype pages at http://www.freetype.org/ (note that
		there are some nasty patent issues on the horizon).

	3) Edit the file /etc/X11/fs/config and add the directory (I added
		it first) to the catalogue configuration file option.

	4) Restart X and the font server.

	5) Run xfontsel and select the 'Microsoft' foundry and you should
		find that Verdana and Wingdings will be there.  (Arial is
		in the 'Monotype' foundry.

So have a go, and good luck.
