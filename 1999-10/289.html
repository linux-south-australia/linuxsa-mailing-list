Re: Uninstalling Compiled Packages

lloy0076 wrote:

> Does anyone know of any good programs which can automatically remove a
> "make install"ed package?

Not so much a "good program", but one of the techniques I use is:

    1) find / -print | sort > /tmp/before.image
    2) make install
    3) find / -print | sort > /tmp/after.image
    4) comm -12 /tmp/before.image /tmp/after.image

It's then very easy to spot what's been added (and deleted) by the
install process.  Another method is run "script" just before you
do the make install, so that you have a record of what actually
was done (or conversely, you could look at the Makefile itself,
but some of these are pretty convoluted).  Although this doesn't
help you after the fact, one option that springs to mind is that
you could "touch" a file to a time just before the install, and
then use find to locate files with mtimes more recent than the
file you created.
