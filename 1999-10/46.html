RE: Concentrator Box

There are a few other hints that may tell if it is token ring or not:
- Perhaps if there is an indication anywhere that may hint to speed, either
4 or 16, instead of ethernet's 10 or 100.
- The presence of "ring in" and "ring out" ports (one on each end)
- If you're really keen, the RJ-45 jacks soldered to the mainboard will only
have use for pins 1,2,3,and 6 if ethernet.
- Any reference to MAU (multistation access unit) would indicate TR.
- Any db-15 jack on the box would indicate ethernet
- any 9-pin female jack would indicate either TR or an RS-232 programmable
(do not use, either way)
