Re: Samba and logon scripts

At 10:17 PM 4/2/00 +0930, Sam Silvester wrote:
>Hi All,
>
>I've just got around to building myself a 'test network' so I can play
>with Samba and Windows NT & 95, but I've got very stuck when I tried to
>get login scripts working on my Win95 boxes.
>
>Under NT, I simply specify in User Manager which login script I want to
>run for each user. Under Samba I have absolutly no idea where to set these
>sort of options. Can anyone help?

You need the following in the global section:

  logon server = yes
  logon script = somescript.bat ; or something like %U.bat

and then you need a netlogon share and you need to put your scripts in the
netlogon share:

[netlogon]
  browsable = no
  path = /home/netlogon

>Also, I wasn't able to find much documentation on the samba website. Where

it in nauseating detail :-)
