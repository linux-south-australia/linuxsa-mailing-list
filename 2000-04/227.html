Re: Telstra Big Pong

Hi All,
    Before I set up my own site I was guilty of having a Big
Pong Account. I had no trouble setting it up under Linux.
You already know what your login name is, you already know
what your password is, so the only thing holding you up is
the DNS from Telstra. Just ring Telstra and ask them, I
forget since it was so long ago, but Tech support will tell
you. Ring 131 282 (I think). Once you have the DNS IP
addresses, just run the dial up setup, or Kppp or something.
I set up a friends One-Net dial up account the other day and
it takes about 5 mins (unless you have a win-modem). BTW
Telstra "Officially" DO NOT SUPPORT Linux, unofficially more
and more customers want to use it and so some of the Techs
are slowly learning the tricks (they even have an unofficial
cheat book). Good Luck.
    Cheers!
        Andrew.
