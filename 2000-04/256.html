squid problem

Hello, all:

I've got a weird little problem. I can only start squid from the squid
account. If I try start it from root it returns without an error, but it
doesn't start. I had to hack the squid account so that I could su to it, and
it ran okay. But it won't run from root. As I understand it, the default for
cache_effective_user is to run as the squid user. So there shouldn't be
anything special I need to add to the squid.conf file. But I've included it
it case. Any ideas?

http_port 8080

acl all src 0.0.0.0/0.0.0.0
acl safcol src 10.1.1.1/255.255.255.0
acl manager proto cache_object
acl localhost src 127.0.0.1/255.255.255.255
acl SSL_ports port 443 563
acl Safe_ports port 80 21 443 563 70 210 1025-65535
acl CONNECT method CONNECT

http_access allow manager localhost
http_access deny manager
http_access deny !Safe_ports
http_access deny CONNECT !SSL_ports
http_access allow localhost
http_access allow safcol
http_access deny all

icp_access allow all

miss_access allow all
