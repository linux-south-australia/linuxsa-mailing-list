Re: virtual network

Hi Jasmin,

  Jasmin> Hi All, Is it possible to create multi-homed host (more then
  Jasmin> one network interface ) on host that has only one physical
  Jasmin> network interface (network adapter ) and it is not connected
  Jasmin> to any kind of network and if so how?

Yes.. :-)
try the following, to create a clone of the eth0 interface with
another IP address...

ifconfig eth0:0 new.dotted.ip.address 

You may also need to play with the routing table (route).
Have a good read of the man pages on 'ifconfig' and 'route'.

  Jasmin> system so that I can test distributed application that I am
  Jasmin> writing in Java.
