Re: Sendmail - slightly off topic

Steve,
  The easyest way is to setup the lowest MX record holder as ie.
booth1.telstra.net for your domain or whatever the Telstra link ID is, and
setup sendmail to accept all mail for that domain in the sendmail.cw
file and to either forward all mail to an internal Notes, Exchange
or other SMTP server or use the Linux box as a POP3/IMAP server by
delivering all mail locally.

	Steve
