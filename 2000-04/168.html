Re: Linux Virus Scanners?

Alex Wilkinson wrote:
> 
> errr....I don't think linux/UNIX can be infected with viruses. Although
> there was one occasion...I think it was called "bliss" from memory.

Why not?  Linux users and application writers are currently more
clueful, but nothing in the operating system itself excludes
e-mail applications that automatically run attachments, or
word processing documents that allow arbitrary code to executed.

There's also the issue of protecting others.  It's doubtful that
an e-mail server should accept a mail without scanning it for
viruses.  And a Samba file server should be scanned for viruses.

The Computer Associates' virus scanner was announced for Linux.
They promised sendmail integration too.  So it might be worth
a look.
