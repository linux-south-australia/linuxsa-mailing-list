>
>Not sure what you mean by this??
>Another option is using the "encrypt passowords" option in smb.conf rather
>than using the reg patch on every win9x machine.

Well, perhaps I was being too brief. You will need to add:

  encrypt passwords = yes
  smb passwd file = some-file-in-the-file-system

and then add entries to it with the smbpasswd command. All in all, for
simple home or office use, the registry hack is easier, but does mean
plaintext passwords travel the wire, where programs like Ethereal can
snaffle them :-)
