Re: intranet solution

Hi all - sort of been following bits of this thread... someone recently
asked/commented on "WHY would anyone want to have web based email?" or
wors to that effect. Thought I saw a few negative-type answers in
amongst the 3million messages in this folder :)

I have just set up AtDot on my freebsd firewall (it also runs on linux
btw ;)) - its a web based mail app which allows for various
configurations including linking it your smtp/pop3 server for selected
users (my setup uses this option)- hotmail type operation etc etc etc.
Its brilliant, took 5 minutes to install and it is very useful! 

The setup I used allows various family members to have an email account
on my unix box which they can access from anywhere (Uni, work, public
libraries etc). They can send and receive email, send attachments etc.
It is also very useful when I am out on site doing a job and waiting for
info to be emailed to me etc etc.

Anwyay, my 2c worth.
