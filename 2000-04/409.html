Re: PHP>MYSQL Connectivity

Personally, I've found that because of the version dependencies
and incompatibilites, it's best to get all of the 3 chunks
of software and recompile the lot:

Apache
MySQL (or PostgreSQL, in my case)
PHP

If you get the latest PHP source, it's likely to need
the latest Apache and db source for it to compile.
So you might as well create everything new -- and
install it in a new place, so that you can revert
to the old software if the new stuff doesn't work
as planned.

By the way, the whole business of re-making everything
took me about 12 hours to get it all correct
(on a 486). Recompiling all of PostgreSQL on a 486-DX2/66
gives enough idle time for a gallon of coffee and a big
tub of ice cream.
