lloy0076 wrote:

> i have the setup of /etc/resolv.conf of
> 
> nameserver 0.0.0.0
> #nameserver 203.20.69.66

I'm unsure if 203.20.69.66 is the IP address of your server or not.
Also the use of 0.0.0.0 looks odd for the local nameserver address.

> i.e. will my nameserver cache ANY valid reply OR will my nameserver only
> cache replies IT has gotten itself....

Your name server will cache any entries it's been asked to resolve,
because it's the middle-person of the process.  If you bypass your
local name server and resolve names via some other server then your
local name server has no opportunity to cache those results.
