Re: Enable SNMP

Install snmp/snmpd
run snmpd &
type snmpwalk (ip.of.pc.to.monitor) public
eg/
snmpwalk 192.168.1.199 public

if that gives you pages of numbers & stuff its working.
Then set up mrtg to create a pretty web page with graphs.
(hint: use cfgmaker to suck data of the boxes you want to monitor - it
creates the necessary html)

Read up on snmpd/snmp & mrtg.
