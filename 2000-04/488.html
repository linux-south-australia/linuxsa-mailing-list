Re: IDE CD Burners under Linux

> What are people's experience with IDE CD-Burners?
> Are there any unit out there that people are happy with?


I bought a Panasonic IDE CD-R (8x Read and 4x Write) some time ago and
have had no problem with it.  There are some hassles setting it up to
work with cdrecord that concern SCSI emulation, and some of the
mkisofs command line syntax baffled me for a while.  Nothing that a
little friendly advice could not overcome.

The kernel reports it as

    hdc: MATSHITA CD-R CW-7582, ATAPI CDROM drive

if that helps.  However, I presume that it is already old enough to no
longer be a current model :-(
