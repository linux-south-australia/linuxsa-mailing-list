> from a M$access .mdb file? It is a once-off task, so expensive/difficult

> 
> Bruce

if you have access to a windows box, there are plenty of solutions.
here's one.

1. open the file in MS access.

2. before you do this though, :), update the windows MDAC to the latest
version (the file is called "mdac_2.1.2.4202.3 (GA) typ.exe" and is 6meg
!!)

3. install the Insight PostgreSQL ODBC driver for windows from here:
ftp://ftp.postgresql.org/pub/odbc/index.html

4. configure a postgres database on a linux box, and configure the ODBC
system (thought the windows control panel) to point to your
linux/postgres server

5. in access, link to the tables on your postgres server

6. write a query to shovel the data accross, like
select * from table_in_access into linked_table;



if you dont have access to a windows box, time to hit google and
freshmeat. :)
