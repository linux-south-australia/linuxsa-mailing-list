Re: file conversion/import - M$access .mdb file

Bruce Moffatt wrote:

> Is anyone aware of a Linux tool for importing/extracting/whatever data
> from a M$access .mdb file? It is a once-off task, so expensive/difficult
> solutions not really appropriate.

Last time I looked, there weren't any.  It's a proprietory file format,
and it's unlike the "serialized OLE" format that Word documents use
(for which there is minimal Microsoft documentation).  Your best bet
is publish it as an ODBC source (like Jake suggested), or export it
as a CSV (or some other) text format, and import that instead.
