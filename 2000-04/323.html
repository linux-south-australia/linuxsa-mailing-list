Latest Ethereal (0.8.7) up and now runs under Windows

Hi,

The latest version of Ethereal is avaliable at www.zing.org. The version is
0.8.7.

Ethereal is a GTK+-based packet capture and decode package.

It now runs on Windows, and a binary is provided for those bereft of a
Windows development environment. You do, however, have to pull down a
couple of libraries from various places, like GDK for Windows, and so on.

I have used Ethereal for Windows in a TCP/IP course I give, and it was
quite useful, although it would crash on occassion.

The latest version has even more protocols.
