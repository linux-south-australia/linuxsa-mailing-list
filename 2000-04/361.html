RE: samba-homedir_access

>If you are using some versions of 95,98 or NT SP3 or above it might be
>trying an encrypted password. In which case you need -
>
>[HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\VxD\VNETSUP]
>"EnablePlainTextPassword"=dword:00000001
>
>in your registry (windows 95) to use plain text.

On a NT system it is better to use "regedt32" than "regedit". The key is a 
bit different too.... try

[HKEY_LOCAL_MACHINE\System\CurrentControlSet\Services\Rdr\Parameters]
"EnablePlainTextPassword"=DWORD:00000001

Cheers
R

NB: if you use "regedit" to edit a NT registry, and you stuffed it, MS tech 
support will tell you to re-install NT
