RE: install Red Hat

when the first screen appears (whether booting from floppy or CD) it should
give you 3 choices. 1. Enter to install in graphical mode 2. type 'text' at
the boot prompt and then hit enter to install in text mode and 3. Expert
mode (I think).

Use option two - type 'text' at the boot prompt before is starts booting and
press enter.
