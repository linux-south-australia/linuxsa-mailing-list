>Q
>How do I use search or find to bring up the file?
>
A. Cd to the / directory with...
       cd /
   then use ....
       find ./ -name <the-needed-file-name>
   or try ....
       find ./ -iname <needed-file-name-with-wildcards>
   For all of this and more, try
       info find
>Q
>How do I get the dir listing to lick past page by page?
>
A. Try the more pipe, eg..
       ls -al | more
   where "|" is the pipe symbol, <SHIFT> <\> (ie shift, back-slash
   on my keyboard).  Try     info more     for insructions.
   Also try   less   instead of   more!  Hope that does not confuse :^)
>Q
>Where is the simplest place to print a summary of the unix commands?
>
A. Try "UNIX for Dummies" 2nd edition (or later, don't bother with the 
   1st edition).  SSC, Inc. (The 'Linux Journal' people) do some 
   rather nice command reference cards, including a bash one,
   however, I have yet to find a satisfactory Oz reseller of SSC things.
   You could try ...
         info bash
   but it is rather 'nurdy' for raw beginners.  The gurus in LinuxSA
   will know of better sources of information  - let's hear from them.
Hope this is a help, Haydn.
