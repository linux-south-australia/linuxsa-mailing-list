This is not really the job of your MUA. What distribution do you have?
Most new RedHat distributions now by default use procmail for delivery (I
guess assuming you install it!). Try this:

* edit a file in your home directory called .procmailrc
* put in it:

:0:
* From:.*<youremailaddresswithouttheanglebrackets>
mail/myemail

Now send an email to yourself. It should go straight into that file
mail/myemail.

Then in pine you can use the L to go to that folder.

Even cleverer Pine has the concept of 'Incoming message folders' (or
something), different folders it checks for new mail. Look in your .pinerc
file.

If this doesn't work, procmail is not filtering your mail. Make sure it's
installed, if so look at the man pages; you can make procmail filter your
mail by using a .forward file which is a bit trickier but I have done it
and it does work just as well. This method is neat because it requires no
root access.
