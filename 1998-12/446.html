XF86 on a laptop

I've been given the task of installing Linux on a friend's laptop.  I had 
no problems installing RedHat 5.2, but the X configuration proved to be a
bit trickier.

I don't have access to the manuals and stuff that came with it, but the
existing 'doze 95 setup tells me that it has a Cirrus Logic 7548 PCI video
adapter, and a generic 800x600 LCD display.

Xconfigurator detects the adapter fine, I tell it the monitor type, and
everything looks fine until I try to start X... the display looks "wierd"
(different pattern each time, unchanging), and slowly fades to white.
When I switch back to a different VT, or quit X (ctrl-alt-backspace), the
text mode is also stuffed up - text is readable, but sync is out.  Top
left of the virtual screen appears almost in the middle of the physical
screen.  Only way I know of to fix it up is a reboot.

Does anyone have any experience with this sort of problem?  Any ideas on
where to look for help?
