 The Oracle Beta release, released as the EAP (Early Adopters Program)
 Release, has some problems. It was released on CD Rom to people who
 subscribed on the Oracle Technology Network homepage. The Oracle 8 for
 Linux developer's  release is available from the Oracle Technology
 Homepage, but it is definitely not the Beta release. It is a full
 commercial version of Oracle 8, with a restricted license for a
 developer to use for development uses only. i.e. not for loading and
 running a working commercial database. It comes with the real Oracle
 installation util, and you should absolutely download the installation
 instructions from the OTN page first.
 
 I don't have the otn URL, but you can get there easily from
 www.oracle.com.au
 
 I would be happy to assist if you have any trouble loading it up.
