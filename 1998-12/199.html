> After rebooting, MS-fdisk would read the disk, but continued to report the

I had the same problem with a computer when I installed a new HDD.
M$ fdisk expected to see LBA turned on for the drive, and consequently
would report no more than about 500Mb for the disk. Maybe it is that
you have not set LBA in the CMOS setup utility that you cannot see
the full size of the drive.
