Re: Tar Help!! Please

> The file I downloaded is called qmail-1_03_tar.gz and I am trying to
> untar it by:
> 
> tar -x -z qmail-1_03_tar.gz
> 

You need to tell tar that you are using a file for the input and not
standard input by using the 'f' switch, and I also use 'v' to verbosely
list the files as it proceeds.

the line:

% tar zxvf qmail-1_03_tar.gz

should work just fine and untar the file. Be careful because if it was
packaged up from inside the intended directory, you may get a heap
of files in your current directory.
