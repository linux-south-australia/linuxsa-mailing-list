I had a similar problem Stephen and I'm not sure I fixed it the best way but
what I did was used NT's format program in the installation procedure to delete
all partitions and then allocated 1.7Gb to NT and 1.5 I left unformatted. I then
finished installing NT and then went on to reinstall Linux. I tried to folow the
HowTo for an NT Linux Dual boot but I couldn't get it to work so I just use a
Boot Disk for Linux.  I hope this helps, has anyone had more success, If so
please send details.
