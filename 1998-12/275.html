I might aswell help you out here - compiling source is usually -very-
easy. Just download the lilo-21.tar.gz file, do:

tar -xzvf lilo-21.tar.gz

Then cd into lilo, and run the QuickInst script by typing:

./QuickInst

Should be fairly easy from there onwards.

I urge you to get used to compiling stuff - download some simple
utilities, and try compiling them. There's always a file named 'INSTALL'
in the directory which will give you instructions on how to compile it.
Also, check out the 'README' file there, too. Btw, the above files are
always in caps, so they show up first when you do a directory listing. :)
