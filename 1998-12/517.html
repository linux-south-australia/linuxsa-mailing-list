Re: PRINT PROBS

Brian Marr wrote:
> 
> Has anyone had experience configuring a Canon BubbleJet BJ 4550 or similar
> printer under Linux?
> My Redhat 5.2 will print a test page of ASCI text ok-however attempts to
> print a test page of postscript have resulted in "unrecoverable error" and
> several lines of junk.
> When I go to the Word Processor (Applixware in this case) and print a
> Postscript file, the output is the all the Postscript instructions and not
> the actual text.
> I think that RedHat is supposed to invoke some filtering automatically via
> the magic filter.
> Have not had any luck trying to use Ghostscript.
> Any suggestions?
> 
> Brian Marr

Try this:
edit the /var/spool/lpd/lp/postscript.cfg
find the COLOR=-dBitsPerPixel=1, and put a # at the start so it becomes
#COLOR=-dBitsPerPixel=1
save & quit

/etc/rc.d/init.d/lpd restart

try again...

Hope it works.
