Re: Windows 3.1 Under DOSemu

On Wed, 2 Dec 1998, Adam Hawes wrote:

> Has anybody had any luck in running windows 3.1 or 3.11 under the
> linux dos emulator? I really need to run a windows based program,
> but Wine will not run it without crashing yet.

The only way you could run windows in dosemu would be in the x86 mode (not
2/386, standard?). Windows (along with some braindead games) use ring 0
privs, which isn't possible. 

When most people talk of running windows in dosemu they mean running the
winos2. To download that you need an OS2 license.

There was talk of trying to emulate the required privs/instructions to get
things working, but afaik nothing significant has happened.
