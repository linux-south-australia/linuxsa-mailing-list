Re:Catch 22

With the help of the Linux Pocketbook I worked out what the problem was.
The chapter on configuration files described /etc/resolv.conf. I had a
look at this file with Midnight Commander and discovered that only root
had read permission on this file. So I added read permissions for group
and others and, dadaa! ping worked, netscape worked, and my e-mail is
working.

I should also withdraw my earlier criticism of the Linux Pocketbook. I
do have reservations about it's assumption that if we don't use Linux we
use Win95/Win98, and it has some failings regarding NT and large disks.
But it has proved an invaluable reference for me. I would not be
cruising so easily with Linux without the help of the book and this
mailing list.
