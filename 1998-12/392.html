>> Eudora Pro:
>>
>> - Click on the 'Tools' menu and select 'Filters'.
>> - Click 'New' and check the 'Incoming' and 'Manual' boxes.
>> - From the 'Header' drop down list, list select 'Any Header:'
>> - Underneath this, ensure 'contains' is selected and to the right of
>>     this enter the text 'linuxsa@linuxsa.org.au'
>> - Underneath this, select 'or' from the drop down list.
>> - Underneath this, ensure 'contains' is selected and to the right of
>>     this enter the text 'linuxsa@Elizabeth.netcraft.com.au'
>> - Select 'Transfer To' from the first 'Action' drop down box and click
>>     on the destination button immediately to the right of this
>> - Select a mailbox from the list, or create a new one.
>> - Close the filters window and answer 'Yes' to the 'Save Changes?'
>>     dialogue box.
>>
