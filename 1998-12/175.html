Re: Sendmail MX query

Hi Steve,

If your ISP is using a ETRN compatible mail transfer agent (ie Sendmail -
not quite sure which version support started, but 8.9.0 has it), you can use
ETRN to get it to send its mail to you.

If you have Sendmail 8.9.0 sources (and possibly earlier), there is an
etrn.pl perl script in the contrib directory that will do this for you.  If
not, I can send this to you (about 8k).

Amusingly, this perl script is both executable and a man page.  You can do
this:
./etrn.pl mymailhost
And the thing executes as expected.
You can also go:
man ./etrn.pl
And you get a man page. A worthy and highly amusing hack IMO.

Cheers,
Martin
