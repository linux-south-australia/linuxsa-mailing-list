Re: Fetchmailrc

> I think I need to add the -a command and/or the -K command to my
> .fetchmailrc.

-a on the command line will fetch all. In the .fetchmailrc file, you
need the keyword 'fetchall'. I cannot remember exactly where it goes
in the file but i do remember having some trouble with placing it at
the end of the line.

The -K option is the default to REMOVE messages once downloaded. The
respective .fetchmailrc command it nokeep. It is a default option 
in fetchmail, and so shouldn't be necessary anyway  :-)
