Steer clear of ftp.redhat.com and use a local mirror. For the contrib stuff
(like ircd), use the shiny new contrib.redhat.com.

For what it's worth I'd get the ircd source and compile from scratch. The
source tree has *heaps* of documentation, which the rpm doesn't and you
should compile it anyway, as you would be surprised how many includes are
not 'included' in the birnary dist. Also, the source version will install
nicely in your /usr/local tree if you want it to.
