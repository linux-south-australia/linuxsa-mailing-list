Re: Red Hat 5.2 Sparc install failing

I installed RH5.0 over FTP from a Win98 box a while ago, if I remember
correctly, it was because the 2x CDROM drive I had in the machine I was
installing RH onto wouldn't read any disks that I had burnt.

I remember that I had some fun trying to get it to work, and what I ended up
doing was copying the entire CDROM to my hard drive (not a problem on my 6Gb
drive..) and then installed it over FTP from there.

If you are putting 5.2 on, the install has an option to do it over a samba
share. That would be a much better option than the FTP option, I would have
thought anyway. If you need any help setting up the samba options, let me
know and I will run through it with you.

Cheers
