the 3com 509 tp card is works great and is auto detected fine by a RH5.x
install. In my experiance they are a great value for money card. The
software to setup them though runs under dos, but then if you are going
to use only one of them and it only has the TP port on it then you
proabably dont need to set it up. I have setup three in the one machine
and am using it as a gateway/router/firewall and Linux picked them all
up spot on, with out any kernel messages :-)
Just plug it in and boot up, then check the boot messages and it should
find it fine.
