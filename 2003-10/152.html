Re: Setting up a gateway

Ok all done. Thanks to everyone who helped.

Basically all i needed to do was in the smallest steps possible (if anyone
else was interested):

echo 1 > /proc/sys/net/ipv4/ip_forward

IF1=eth0
IF2=eth1
NET=192.168.1.0/24

iptables -A FORWARD -i $IF1 -o $IF2 -j ACCEPT
iptables -A FORWARD -i $IF2 -o ! $IF2 -j ACCEPT
iptables -I POSTROUTING -t nat -o $IF2 -s $NET -j MASQUERADE

apt-get install ipmasq

Hehe.. now i can read about iptables with this out of the way.
