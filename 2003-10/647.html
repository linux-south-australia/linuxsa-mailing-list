>  (We should also note the ability that exists in Michael's Lindows
>  Click-N-Run service to access a subscribers hard drive and delete
>  programs that no longer are licensed for use etc.)

Ian, you're a bit incorrect on this. Lindows has Click-N-Run and 
Click-N-Buy, which are functionally the same except, of course, you buy 
apps from the latter. The CNB service includes for application developers 
an API which they can OPT to use. This is a piece of code which requires 
the CNB(R) to check with the Lindows servers that the app is currently 
licensed, It doesn't delete anything but it will stop the app working if it 
is not listed as licensed. There is a debate going on at present as to the 
merits of this and the ? of what happens if you are 
disconnected/standalone. PLs note, Lindows have stated they had to work 
with developers to provide a way to stop application theft. THe use of the 
API is opt in however. It is up to the developer.
