Re: metaframe on linux

Greetings,
As far as this is a Linux related question on a Linux list its relevant. 
Probably more than some topics discussed here. 

Please note I've never used Citrix under Linux, so I am aplying my skills 
from the Windows platform. 

You do not mention whether this is a hardware upgrade, or a new install, or 
a software and hardware upgrade. 

The first qestion is "Is sound working on the server ?" If it isn't it will 
never work on any client. Same question about the client. 

Is the ICA client on the client configured for sound. Is the ICA connection 
on the server configured for sound ? 

If you have upgraded (or have a new install) do you have all relevant 
libraries, and other packages installed on the system (I don't know what 
these would be). Are you running the minumum required Kernel version, or a 
known working kernel version. 

Finally, sometimes there is a requrement of some packages that sound be 
either compiled into the kernel or be a loaded kernel modual. Check if this 
is a requirement here. 

If none of these work, check the web, there are some good Citrix resource 
sites out there, when I return to work Monday I can check my bookmarks if 
you can't find any. 

cya
