Re: html forms and bash scripts.

I've never used bash for a CGI script, however I would say you would
make sure the form is Posting the data ("METHOD=POST" in the  FORM tag).
I would then expect the data to be available as environment variables.

I would suggest you do use PHP or Perl though, bash is not really
designed for CGI and there are a lot of security concerns etc... that
are better handled under one of the other languages.
