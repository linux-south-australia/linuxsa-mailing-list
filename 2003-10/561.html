Re: mouse wheel in kde konsole

On 20/10/2003 9:09 AM, Luke Vanderfluit wrote:
> I'd like to be able to use it when editing in vi,
> actually... maybe this query is more vi related, 
> geez, sorry about the confusion....

Hmm, this could possibly be achieved by binding "ESC, j" to the 
mousewheeldown event, and "ESC, k" to the mousewheelup event, while in 
vi, while in konsole...which is probably harder than it seems, and more 
trouble than it's worth. (Might be a fun way to learn about scripts and 
binding under KDE tho ;-) )

The other (easier, I think) option is to edit with gvim (the graphical 
vi), given that you're under KDE. It'll mean you'll have to switch out 
of konsole to actually edit a file (you'll still be able to navigate etc 
in the shell etc tho). I don't have a KDE install handy right now, but I 
think you invoke gvim via:
	$ gvim <filename>
or
	$ vim <filaneme>
(instead of
	$ vi <filename> .)

Gvim *should* give you mouse wheel support without you having to do any 
heavy lifting (he says, optimistically).
