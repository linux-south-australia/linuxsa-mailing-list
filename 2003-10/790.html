Re: Gentoo Linux - any good?

Once upon a time Taylor, Corey (AHA) tapped upon the keyboard...

> Just got a copy of Gentoo 1.4 and I'm curious as to opinions on
> it. Has anyone here ever used it? Anything I should know about it?

Gentoo is good. If you're not used to playing with the contents of
/etc/ then you soon will be. It has a steep learning curve in that
respect, but once you've started to work it out you discover that
Linux is no longer the black box it used to be when you still used
all the "walk through" graphical and console based configuration
tools as with RedHat/Mandrake/etc.

 Tends to work faster since you can set make options to have
 -O3 -march=pentium4 -msse -mmmx -msse2
 and all the funky things that use all the new instructions
available in your CPU.

 I've been using it for almost 2 years now (after RH7) and I find it
good. Also have it running on a server and stability is no problem.

 Zynot (www.zynot.org) is a fork of the Gentoo distribution which
looks interesting too, but so far not much has been done.
