RE: ftp - restrictions

Trevor,

> Was wondering if someone could lead me down the right path,
> on how to restrict users to there own directory when the
> log in using ftp.

> System is Redhat 9, using vsftpd.

Edit the config file, usually located in:

/etc/vsftpd/vsftpd.conf

Add a line like this:

chroot_local_user=YES
