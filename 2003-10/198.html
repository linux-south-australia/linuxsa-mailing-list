Re: html links to local file systems

I think the link to your file is wrong from the point of view of a web 
server. The link can either be absolute to the document root of the web 
site or relative to the location of the file containing the link. Unless 
your local link exactly matches the website path then the browser can't 
find the file. I also don't think file:/// is right. I believe that refers 
to local files. <a href="path/to/file/css.html"file</a> would be more correct.
