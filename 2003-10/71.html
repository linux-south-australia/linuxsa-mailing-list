Advanced Server probably has a firewall going.

Try doing:

 # iptables -L -n

...and if it comes out with LOTS of information then there is a firewall.
This could be the first problem.

The portmapper also looks for entries in:

 # /etc/hosts.allow
 # /etc/hosts.deny

Obviously if it's listed in hosts.deny and not in hosts.allow then you
have a problem.

Also, do you know if the other interface is running at all?

What is the output of:

 # ifconfig -a

(You may want to do ifconfig -a | less)

???

Obviously portmap can't bind to an interface that isn't running.

Also, check to see if the interface has the correct IP address.
