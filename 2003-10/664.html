Re: ssh with public and private key

<quote who="Luke Vanderfluit">
> Hi,
>
> I wish to ssh from a local machine to a remote machine without having
> to enter passwords.
>
> I have created a private and public key on my local machine with
> 'ssh-keygen'. I have the private key file in $HOME/.ssh on the local
> machine( 'myKey'), then I have copied the public key file-contents to
> the $HOME/.ssh/authorised_keys file on the remote machine,
>
> according to the man page this should work,
> what am I doing wrong,


Three things to check:


1) Are you running sshd on the remote machine?

2) Are the permissions on remote machine for authorized_keys -rw--r--r- ?

3) You're specifying the remote username if different from your local
username?

i.e. localUsername@localhost>  ssh remoteUsername@remoteHost
