David, David, David...

Would you like me to paste the contents of the website in here? Maybe I can 
give you a quick summary so you don't have to do the hard task of pasting a 
link into a web browser.

It is not meant to replace the packaging format of a distribution, but rather 
be complimentary to it. It is intended to be distribution independent, i.e. a 
single package works on any distribution. Tell me how I can configure a tool 
to let me download packages from the Debian distribution onto my Mandrake 
box?

It is intended for third party applications, games, proprietary software, etc 
(like the loki game installer, but more general). On the TODO list they are 
planning "self-contained" packages, that include the entire installer, not 
just a stub that downloads the installer, as well as interactive installs 
that allow the user to choose where they want the package to go, and various 
other options.

So... what's the apt command for installing something as non root into a 
directory of my choosing, for a package I've already downloaded (that may or 
may not have all of its dependencies met)? Keep in mind it should be 
interactive so that Mr/Ms former-windows-user has no problems with it.
