As for Nero, the following settings should work:

  Data Mode 1
  Block size: 2048
  Image header 0
  Image trailer 0
  Scrambled unticked
  Swapped unticked

I'll be the first to admit that Nero can be picky with images.  If
you're using XP (gasp!) you could try this:

http://isorecorder.alexfeinman.com/isorecorder.htm
