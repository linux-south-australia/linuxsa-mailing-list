Re: ssh with public and private key

I have read throuhg some of the replies to this thread. So here's a
couple of gotchas. The primary one is if you have used the paraphrase to
enter a password it will always ask for a password. This is common.
Do not use paraphrase, eg just press enter and move on.
Make sure you made the keys as the user on both ends and not as root in
a users folder.
Make sure authorised_keys is a file and not a folder

Other than that you should have no difficulty.
