Re: Quake with Linux

On Sun, 12 Oct 1997, John Spence wrote:

> I have the registered MSDOS version 1.06 and was wondering if there is
> any kind of a Linux package that I can install to use the DOS
> version's data/game files?

The data files for both linux & dos Quake are the same, you only need to
download the binaries you want.
Quake binaries for linux:

squake		SVGAlib quake. Highly recommended.
xquake		needs an 8bpp X server, no mouse.
xf86quake	fullscreen in X, sorta buggy.
unixded		dedicated quake server that you can background.
qwcl		QuakeWorld client
qwsv		QuakeWorld dedicated server

Just put the ones you want in your quake directory and it should work
fine. You may have some problems with permissions since squake, qwcl and
xf86quake need to be setuid root if you want to run them as an ordinary
user (unless you write a wrapper). You could also copy all your quake data
files to /usr/local/games/lib/quake, then you would be able to set the
perms however you liked. If you do this put a wrapper script under
/usr/local/games like:

-----
#!/bin/sh

cd /usr/local/games/lib/quake

if [ "$DISPLAY" ]
then
	exec ./xquake $*
else
	exec ./squake $*
fi
-----

I think the urls to the linux quake binaries are something like:

http://www.planetquake.com/linux
or
http://www.threewave.com/linux

There should be a linux link under http://www.planetquake.com as well.

If they don't work just do a search for 'linux quake'

BTW you may be interested in the upcoming crack.com game which will be
released for linux called 'Golgotha'. For info have a look under
http://crack.com/.
