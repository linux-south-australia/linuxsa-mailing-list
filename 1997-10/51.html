> But I'm not sure where to untar it. 
> The general question I want to ask is: 
> do you have to untar a file in a directory where you want to install it, 
> or /usr/local or /tmp and then move the files to the correct directories.

It depends on the tar file, and there isn't any way to tell without
first looking in the tar file and/or reading any instructions there
might be for it (therefore, it's best to unpack it to a temporary
directory :).

Also, you don't need to gunzip and then untar.  If you use the z flag
to tar it will untar from the compressed file.  For example, instead
of:
  gunzip foo.tar.gz
  tar xvf foo.tar
you can do:
  tar xvzf foo.tar
