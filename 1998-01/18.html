> My questions are:
> has anyone had any problems with TX motherboards and Linux

I'm using an Asus TX97 mb with Pentium 200MHz non MMX. A great mb, no 
complaints. 

With RH4.1, the boot up message complain about not recognising the 
bridge, USB controller, IDE interface, ISA bridge and the host bridge but 
Linux still runs fine.
I supposed the complaints were because the above devices have not yet 
been registered in pci.h as of kernel 2.0.27. Should have no problems 
with the newer kernels.
