Sounds like sendmail trying to find out who it is.  If you don't need
sendmail
get rid of it, or make sure sendmail can find out who it is (someone
else might
be able to point you in the right direction - sendmail and I have not
been on
speaking terms for a while :).

>After logging in Xwindows won't start properly, I get the grey
checkered
>screen but it won't go any further.

That is *technically* X-Windows.

There is an errata in the RH upgrade about a .XClients file.  It doesn't
work
after an upgrade.  (Check out http://www.redhat.com for more details)
That got me
when I installed and maintained my user directories.

>
>      I have reinstalled the system and at the prompt "do you want to
>install networking" I have answered no and the system instals and works
OK
>including Xwindows but no network card obviously.

Check out your dns config.  If you have a dns server that is used on
dial up
and you have a network card which installs a default route the DNS
queries
that sendmail sends (which are UDP) disappear into /dev/null.

Hope this helps.
