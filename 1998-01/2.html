RE: Help with RH Installation

Mael,

I've done a few NFS installs and they can be tricky.  A few things I
learnt the hard way,

Check your network card is actually being recognized and used.  Do this
carefully, sometimes it may look OK but is not correct.

ALWAYS specify the IP address of the computer to mount via NFS.  You can
check if your computers 'see' each other by pinging from the already
installed computer.

Check that you have set your exports file correctly on the 'NFS server'.
simplest way is to set it to allow *anyone* access to that dir/CD
whatever.  I would have to dig up the exact line in the file if you
needed it.

If using a CD, check that you have it mounted ;-)


Hope this helps..


Chris
