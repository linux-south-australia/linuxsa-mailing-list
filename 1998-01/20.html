Re: IBM PR200 CPU

Hi Bruce,
I'm not using a TX board, I'm actually using a VIA chipset board, but I am
using an IBM PR200+ CPU and have no problems at all with it.  I've got RH
4.2 installed and it runs *VERY* well with 64MBytes RAM.  Bogo's sit around
150.

Don't try to overclock it though...you'll almost certainly cook it and get
some real interesting kernel errors.  I accidentally had it running at 200
*REAL* MHz (remember, it's a 150MHz internally - bus = 2 x 75MHz) and it
lasted about 5 minutes before it really got interesting at the console. 
This was thanks to a badly marked m'board and poor (another flaming
Taiwanese) manual.
regards..Don
