thanks all re upgrade info

Thanks ppl for the good info (which is still coming
in) about compatible hardware for my upgrade.

The upgrade was performed Friday evening, with
virtually no pain. I only made one minor mistake,
which was overcome quickly.

The system now runs like lightning. I recompiled
a kernel to match the new hardware, and it took
me more time to select the appropriate items at
the config stage than it took the computer to
compile the kernel binary. Approx 6 mins for the
'make zImage' stage... I didn't even have time
to make a coffee. In short it steeeeenks!! :))

For reference the system runs an IBM (Cyrix) PR200
MX chip and 32 Mb RAM in a VX motherboard. The 
board is set up at bus speed 75, multiplier factor
2, and no it doesn't hang, spit or gurgle.

I appreciate all the good advice.

Bruce
