*HUGE* win for the Free Software Movement

Hello people.

Late breaking news.....

Netscape is doing the following....

1) Giving it's browser away for free...

2) More importantly allowing free access to source code for the 
	the next release of communicator.

	Currently the licensing model for the next releases is
	'building on the heritage of the GNU Public License (GPL)'
	so stay tuned to find out more.

Both press releases are available on Netscape's homepage at

	http://home.netscape.com/news.html?cp=hmp01nnwi

A quote from 
	http://home.netscape.com/communicator/free_faq.html?cp=nws01flh1

"Will Netscape make Netscape Communicator 5.0 source code freely available? 

Netscape has announced that Communicator Standard Edition 5.0 source code 
will be freely available for modification and redistribution. Netscape 
believes that harnessing the creative abilities of all Internet developers 
will be an unprecedented way to advance the features and quality of Communicator 
for all customers. Detailed license terms, additional information, and a developer 
version of Communicator Standard Edition 5.0 source code is scheduled to be 
available by March 31, 1998, on the Netscape Web site."

I say let the paradigm shift to free software.

Matt
