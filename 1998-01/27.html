Re: Viewing messages

>>>>> "Mael" == "Ismail Ibrahim <mi-mi@sbs.itc.usm.my>" writes:

  Mael> When using Xwindows, I would like to have a spare xterm opened
  Mael> that would show the /var/log/messages file as it is being
  Mael> written. This is so that I don't have to keep 'tailing' the
  Mael> file every now and then.  Anybody know how this can be done?

It depends on what system (RH, Debian, Slackware, etc) and what
version you are running. 

For example, the simple way to do it using the above xterm method is
simply to use "tail -f /var/log/messages", but there are other tools
which can do the job.

For example, Debian package "xbase" which provides the basic X11R6
distribution (e.g. xfree86 Version: 3.3.1-1), also includes a utility
called "xconsole" which has been obviously designed to display the
console messages inside an X-window.

If your system doesn't have it and you'd like to try it, you could FTP
xconsole over from a X11 mirror.
