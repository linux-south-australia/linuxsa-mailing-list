Hi All, I've just installed MySQL for use in a molecular genetics database
project. However, though following the tutorials at
http://www.devshed.com/Server_Side/MySQL/ and trawling through the MySQL
docs and deb install scripts, I can not figure out why I can't set the
MySQL root password with the described commands eg:

# mysqladmin -u root password new_password
mysqladmin: connect to server at 'localhost' failed
error: 'Access denied for user: 'root@localhost' (Using password: NO)'

# mysql -u root mysql
