Re: Sendmail config question

Sam Silvester wrote:
> 
> What can I do to allow sendmail to let me send mail from any machine but
> keep it from relaying spam? I can't seem to find the right balance!
> 

AFAIK, the only good way to do it is to allow all relaying for 
authenticated SMTP over SSL/TLS (which use use for for sending mail), 
but deny relaying for normal SMTP (which gets used for local delivery).

The alternatives are to allow relaying by IP address, host address or 
possibly by the sender's email address, but all of these options are 
vunerable to spoofing of some sort and so aren't a good idea.
