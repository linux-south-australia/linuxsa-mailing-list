Re: Founder of "Enlightenment" thinks Linux on Desktop is dead

Well there's the third, unmentioned location of linux on the desktop, 
which is upon die-hards' PCs ;)

Also, I believe that very large organisations with limited numbers of app 
requirements may realise the savings to be made from using Linux instead 
of Microsoft Windows + Apps.

If a single Linux OS + required apps can be imaged and rolled out en 
masse for a given purpose, you could get away with lower hardware costs, 
avoidance of expensive yearly per-seat licenses, etc..

First example that comes to mind would be call centres. If you could have 
a setup involving one custom app with everything else driven via a 
web-browser, then Linux would be a certain winner.

Toby
