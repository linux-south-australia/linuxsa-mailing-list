Re: whois help

On Wed, 2002-07-24 at 21:34, heidi@internode.com.au wrote:
> my guess is you're querying the now defunct aunic database. the new whois
> server (whois.ausregistry.net.au) won't actually provide you with a
> renewal/ expiry date.

the aunic db might be defunct but as I understood things you
are still supposed to use whois.aunic.net for queries as it
redirects them to the correct registrar.  A quote from Kim
Davies:
<quote>
There is a problem with the RIPE whois db which is supposed to do
referrals which we are trying to sort out. When it is working again,
"whois.aunic.net" should be the best place for .au whois queries. This
is because ausregistry is only the registry for 5 2LDs, and AUNIC still
handles some others. Additionally, there is a potential for other
registries down the track (for new 2LDs etc.), and whois.aunic.net
should always provide the correct referral.
</quote>
