WinXP Problem Connecting to Samba Servers?

Hi Everyone,
Here's a setting that might fix problems connecting to samba servers from
XP.  Control Panel, Administrative Tools, Local Security Policy.  Local
Policies, Security Options and under there is a 'Microsoft Network Client:
Send unencrypted password to third-party SMB servers...

Probably newer versions of samba fixed this prob, but this may get people
out of a bind temporarily...
