>3.  At the console only 640x480 pixels of my screen are being used.  how
>do I set the console to display at 800x600?  This is the native resolution
>of the flat screen.  In effect the console has about an inch or more
border
>around it.  When I start X it uses the whole screen at 800x600x16 bit
>color.

You need to enable VESA frame buffering, but there is a catch.   IBM does
not enable VESA in the Video Card by default, you have to activate a small
DOS program first before it work (vesa.exe).   If you can make a small FAT
partition, install just enough DOS to get your system running, run the
driver and boot a linux kernal (loadlin).  There is no linux only based
alternative that I have found.   Also if you intend using the serial port,
disable the IR port, by default they share the same IRQ which causes Linux
a few problems, can be done with the DOS utils that some with the laptop.

>anyone has any experience getting sound to work on an IBM laptop from the
>console, I'd like to hear from them =)

*extremely* lucky !   A majority of my IBM thinkpads all have MWave sound
>:(    You will find the information on the site listed quite useful.
There maybe a howto install linux on the site that gives some additional
info.
