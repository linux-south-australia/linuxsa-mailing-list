This is pretty explicit - apparently you're trying to send an email
with a null sender address, and the SMTP server that you're talking
to (ozemail) is not allowing you to send to multiple recipients in
that case.  A nice anti spam method, as typically the only type of
message sent with a null header is a bounce message, and that'd be
to one recipient.  I'd wager that your "10% unsuccessful" would be
when multiple recipients were involved.  (mind you, some mail system
might be balking at "mick@dave.2001" if that's leaking into the
outside world.
