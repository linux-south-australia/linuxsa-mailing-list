Re: G-Force 420 & Likom Monitor for seawolf

Hi.

Have a look in /var/log/XFree86.0.log for error messages, usually down the
end if X is not getting started at all. As usual, run an eye over
/var/log/messages as well.

Geforce Issues I've seen or heard about:

* AGP. apparently many agp cards are flaky at 4x, and some agp chipsets
are not supported with agpgart. Agpgart will throw some useful hints into
/var/log/messages if this is the case. The NVidia drivers support agp on
their own, so you have a choice of using agpgart or NVidia drivers.

* Botched nvidia driver install into the XF86Config-4 file - double check
against the instructions on the website.

* Incorrect or mismatched NVidia packages

* Someone else suggested irq's for VGA.

Best strategy for working on the problem is to set the initdefault to 3 in
/etc/inittab and test the X config using startx from the commandline. It's
impossible to diagnose the problem if X won't run reliably at all, and
your system is booting X by default.

If you cannot resolve the problem yourself, try searching google for
whatever error messages you are seeing, and also post a more detailed
description of the problem to linuxsa.
