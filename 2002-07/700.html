"Unix" is a trademark of the Open Group. If you want an operating system
to be a true Unix operating system, you have to pay money to the Open
Group and if the OS passes their standards tests then it can officially be
called "Unix".
With the growing popularity and acceptance of Unix-like OSs like Linux
that are not official Unix OSs, it's not really important if Linux has the
officail Unix stamp or not. Linux can stand on its own reputation: it
doesn't need the Unix brand to be accepted.
