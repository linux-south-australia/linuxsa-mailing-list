Re: Founder of "Enlightenment" thinks Linux on Desktop is dead

No-one ever said it was going to be easy :) There seems to be another person 
every week saying that Linux on the desktop is dead, yet the apps and 
improvements keep coming regardless. Also the profile of Linux is getting 
better, it is being taken seriously in mission critical environments. Sure 
it's being perceived as a cheap backroom server that'll just keep running, 
but before that it was perceived only as a toy. 

People keep predicting Linux will fail here, there and everywhere, yet they 
keep getting proven wrong. Where there is a will there is a way, and there is 
a *lot* of will out there. 

I think the key to getting Linux on desktops and in peoples minds, is schools. 
Kids will want to have what they use at school, and they get to take a copy 
home, legally. According to the K12 project, setting up a terminal server 
classroom with a beasty application server and some recycled machines turned 
into thin clients, reduces the cost of the _hardware_ by half. Then comes the 
fact u can stick a copy of redhat on the server for nothing, or if u want 
support for that server, buy a single copy. 

Typical high school applications are certainly plentiful (word processing 
etc), and of course it's a fantastic environment for computer programming (we 
learnt toolbook and logowriter at my school, um.. yay, how about some C?? or 
some scripting languages, or Java, ie something useful). I'm not too sure 
about educational programs for little kids though, but the various Linux for 
schools projects are working on that, and the new addition in KDE3 includes 
some educational programs - although rather boring, but that can be fixed.

Also, Tux is cute. Linux is fun, it can appeal to kids. It's a stable, easily 
administrated server. Beige windows environments are BORING! Where are the 
negatives here people? Linux in schools is cheaper, centrally administered, 
easy to update with no downtime (except in kernel upgrade, but why do that if 
the current kernel works?), linux can be fun and cute, it sends a good 
message about sharing and choice and to not just follow your corporate 
masters blindly, provides a better learning environment  - kids will be more 
competent with computers, etc etc.

I'm not giving up on linux until Microsofts marketshare has dwindled 
significantly (and i'm talking like 50% or more). I know things will keep 
improving, regardless of what the naysayers(?) say.

Damien.
