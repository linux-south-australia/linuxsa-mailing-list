[Fwd: Re: SCSI]

That is normal. Every linux burning prog I know of supports only scsi
under linux. Linux uses a kernel module to translate the IDE to SCSI
(ide-scsi). So you should find your ide burner shows up as a scsi device
on 0,0,0. From there, it should all work.
