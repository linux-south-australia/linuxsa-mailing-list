Debian can be installed with to not be heavy on disk space, just do a
net install (30mb iso download for the setup cd) just install the
packages you need with apt. My installation is at around 700mb currently
-- and I have a lot of server apps installed that I am going to get rid
of soon (since I just got a spare box setup as a server)

My server:
Filesystem            Size  Used Avail Use% Mounted on
/dev/hda2             121M   27M   94M  23% /
/dev/hda1              27M  5.3M   21M  20% /boot
/dev/hda5             1.5G  134M  1.3G   9% /usr
/dev/hda6             121M  128k  120M   1% /tmp
/dev/hda7             1.9G  8.1M  1.8G   1% /home
/dev/hda8             183M   26M  158M  14% /var

As you can see it has hardly used any space -- /usr is only 134mb...
Granted, this is a *very* minimal install with only a few server apps.
