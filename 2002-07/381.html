A useful tool for cleaning up a Debian install is debfoster (apt-cache
show debfoster for more info). This allows you to selectively remove
anything you explicitly requested rather than those packages added as a
dependency.	

You can pretty much take your system back to the base install with
debfoster if that's what you want. I guess this wouldn't work if you
installed X as part of the debian setup, as the packages weren't
explicitly requested by you via apt, although if this is the case then
tasksel might do the trick.
