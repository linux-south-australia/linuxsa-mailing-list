Re: Installation question

You can upgrade your whole redhat installation while online by using
apt-get for redhat from http://apt.freshrpms.net but the downside is
(unless you edit your /etc/apt/sources.list) it uses a lot of bandwidth
and you have to do it incrementally AFAIK. That is from rh7.1 to rh7.2
then to rh7.3
The upside is no downtime at all except for a reboot.
But you will have to go through all the configs.rpmnew files and update
all your old config files with the new features.

Alex

P.S. You will need both the apt- and apt-devel packages for your
currently running version of redhat, and to run apt-get update first. Do
man apt-get of course for details.
