OK here we go - from memory - it has been several years now so it's a bit
scratchy.

Before installing NT go to the Micro$oft web site and get the drivers that
enable large disk support - sorry I can't remember names or where but I can
search around if you need me to, I think I found a link off an article on
the site that described the problem and how to get around it - not sure now.
Anyway copy these drivers onto a floppy disk.

When installing NT, at one point it will ask if you have any other special
hardware.  Say YES and point it to the drivers on the floppy disk (there are
a number of steps in this process but I cannot remember them all).  You will
then be able to see the whole 40GB from go and partition it as a single
drive.

Scares me that I know this!!
