I find Photoshop's interface irritating and I can almost never quite
work out what I want to do. This is because I'm used to using The GIMP
and how it thinks.

> Its an excellent program that can do way more
> than they have to but even selecting an area and doing a cut and past

The only thing that is not altogether too obvius is that unlike
Photoshop, The GIMP doesn't use an MDI style interface so you have
separate windows all over the place. These windows don't have menus, but
if you're used to X then you'd probably work out that the right mouse
button brought the menus up quite easily. And if you don't understand
what "edit->select all" and "edit->paste" does, well, what can I say
without displaying my infamous sarcasm?

Adobe's Photoshop has been a de facto program for the work it does and a
lot of people are used to it. The GIMP does not use the same "thought
process" as Photoshop and hence for a lot of people it seems to be "more
difficult" and "less intuitive" to use. But that's only because they're
used to Photoshop -- not because The Gimp is inherently
"non-intuitive"...

BTW, I can use Photoshop, it just takes me a while to work out where
things are. The basics between Photoshop and The GIMP are really
similar. For example, their concept of "layers" is essentially the same
but the place where you can change, alter, merge or wotever with the
layers is different and the dialogs don't look the same.
