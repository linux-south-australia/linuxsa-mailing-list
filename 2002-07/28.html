Well, I'll make it easy for you.  Here's a summary:  

1) The directory with TTFs needs to be in your font path.  That will depend 
upon if you are using xfs to serve fonts.  If you're not using xfs, have a look 
in your /etc/X11/XFree86[-4] file for fontpath.  If you are using xfs, 
see "chkfontpath".
2) In your TTFs directory you need a fonts.scale file.  See my previous thread, 
or just do a "ttmkfdir -d . -o fonts.scale" in it.
3) In your TTFs directory you need a fonts.dir file.  Use "mkfontdir" to 
generate this.
4) Restart xfs if you are using it, otherwise restart your X server.

If you are using Red Hat, and "ttmkfdir" doesn't barf on some of your fonts 
(Grrr:-( you don't need to do 2, 3 or 4 because /etc/rc.d/init.d/xfs does it 
for you (unless you add fonts to a directory that already has a fonts.dir file 
in it).
