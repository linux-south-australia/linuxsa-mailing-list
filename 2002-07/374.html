RE: *STUPID* NT Workstation 4.0

David,

I had the same problem as you've experienced when installing NT Server on a
20Gb drive...  The problem it would seem is that the atapi.sys driver on the
NT installation CD's/Disks has an 8Gb limit to it.  This is fixed in SP4 and
later, but that doesn't help you at all when you can't even get Windows
installed!  The workaround is to create a floppy with an updated atapi.sys
driver, and use that during install, the install SP4 or later once
installed.

The MS KB article explaining the problem and workaround is here...

http://support.microsoft.com/default.aspx?scid=kb;EN-US;q197667

Also useful...

http://is-it-true.org/nt/atips/atips44.shtml

Hope that helps.

Jim.
