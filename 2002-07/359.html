Re: *STUPID* NT Workstation 4.0

yeah I've had a bit of a search round the web and found that NT4 needs
to be service packed up (as previously mentioned) before it can create
partitions greater than 2GB/4GB and access beyond 8GB (1024 cyl).
 
Its funny cuz there is a consensus that 2GB will certainly work, 4GB
might work, but greater than 4GB will not work. The reason being that it
creates a FAT16 partition before converting to NTFS. But all the service
packs address these issues..including a new IDE driver Atapi.sys to
power on thru the 1024 cylinder issue..praise the lord!
 
yeah its crap..lets get back to Linux.
