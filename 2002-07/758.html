Re: Reading "Smart Tags" from PHP

$file= fopen('filename.html',r);
$data = fread($file);

$begin = strpos($data, '<xml>');
$end = strrpos($data, '</xml>');

$text = substr($data,$begin,$end);



this will give you the text between (and including) the xml tags as an
output as $text, providing they are the only xml tags in the document.

didnt test it though, cos time is short, but it looks about right
