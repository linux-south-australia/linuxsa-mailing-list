Re: Squid proxy server

I cant help you with webmin, but if you edit /etc/squid/squid.conf by
hand and find the section relating to parents and siblings, and add the
equivalent of:

cache_peer adelaide.cache.telstra.net parent 3128 3130

That should do it. There should be doco on the command in the conf file.
Its been a while since I did it though. You might also need to find the 

icp_port 3130

line and remove the # from it as well. 

Ant
