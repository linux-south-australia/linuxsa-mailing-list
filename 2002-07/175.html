Re: Postgres and Perl::DBI

David Lloyd wrote:

[Snippities follow]

> I am attempting to get a timestamp from Squid (the first field in the
> access.log) into Postgres.
> 
> create table(
> ...
> timestamp float;
> )
> 
> ...but DBD::Pg complains that:
> 
> "pg_atoi" fails because it can't conver the "integer"

Well, the above is a create table, not an insert statement, so
I'm not sure how the above is failing.  Without seeing your
insert statement, I can only suggest you try casting your
float expression.  The following worked for me:

TEST=> create table test (timestamp float);
CREATE
TEST=> insert into test values ('123.45');
INSERT 90404042 1
TEST=> insert into test values ('123.45'::float);
INSERT 90406190 1
TEST=> insert into test values (123.45);
INSERT 90410571 1
