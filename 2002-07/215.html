Re: Suitable MB for Linux?

Dale Long wrote:
> 
> Does linux run well (or better) on any of the following mother boards?
> 
> Asus A7V333 or A7V266-E.

I've always had a great deal of success with Asus's A7V mobos, I'm 
running 2.4 on a A7V266E ATM. They're good boards, but can be quirky 
when it comes to IRQ assignment. Provided you're not doing anything too 
weird[0], and make sure you have the latest BIOS version, you shouldn't 
run into trouble.

USB works fine, as does the Promise RAID variant. I've never used the 
onboard sound variants (I really hate onboard sound), nor have really 
used the more advanced stuff, like hardware sensors, IrDA, WOL, WOR, or 
ACPI, so I couldn't comment on those areas.

Good boards, I like them, and will probably buy more (for use with 
Linux, FBSD and Windoze).

/mike

[0] - more weird than the GeForce2 + SBLive + EEPro + PCI/PCMCIA bridge 
combo that I'm running now, which admittedly is at least slightly weird.
