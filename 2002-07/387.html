RE: [OT] RE: *STUPID* NT Workstation 4.0

The 8GB limit is not to be trifled with. If any files that are required for
successful boot reside
beyond the 1023 track barrier you will be up that well known creek. This is
a known issue
that Micro$oft will not fix. (they say go to 2000!) Recommended practice is
to have a 4GB
(or 8GB if you must) boot partition and a large D drive.
 
HTH

At install time 4 GB is the maximum.  As someone mentioned, this is because
the drive is converted to NTFS on first boot.

There is a patch (floppy disk, loaded at the point where you insert device
drivers in first stage setup) that allows to you create an 8 GB boot
partition from the start.  I did a quick search but couldn't find it.

NT4 is not officially supported, however, so you REALLY should upgrade to 2K
or XtraPoo :-)
