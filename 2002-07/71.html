I don't quite follow what you're trying to do but, from what I've seen
of Glade it simply creates the GUI widgets for you, makes the code to
create the Gnome/Gtk+ Window, place menus and so forth.

It doesn't seem to:

* be component based like Borland's Builder, Delphi or Kylix (or VB)

I reckon you're actually asking something to the effect of "Why can't I
see a Glade component that has a property that links automatically to
[My|Postgre]SQL"; I *suppose* you could write one but I would suggest:

* investigating Kylix
* seeing whether you can use a Web Server, CGI/PHP and HTML forms
