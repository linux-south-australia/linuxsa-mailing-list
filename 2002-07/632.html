I was at Panorama TAFE (about 2 years ago now) and they taught intro to
operating systems on RedHat 5.(something). Although I was up to speed on
what they were teaching (file permissions, using Vi (good to see they
arent teaching Emacs :P ), installing etc), I found the course to still
be informative. For some of the Windows-centric people in the class,
they could have done some clarification on the more in depth basics like
how X windows relates to Linux, versus how the Windows GUI related to
the base OS. As an example, one of the students yelled out "i crashed
linux, i crashed linux", it took 10 minutes to explain the difference
between windows and linux, and that it was X windows not linux that died
:). However, i guess it was not directly related to the course topic,
being how the operating system works, cpu management, memory management,
file management etc rather than a linux class.

My $0.02
