RE: thumbnails, mogrify and large filesizes

Thursday, July 11, 2002, 9:31:42 PM, David wrote:

> What do all you people that create thumbnails do?  Got any ideas?

Well, I use a php package called mig (check freshmeat) to create
galleries of piccies for my site, and it comes with a utility called
mkGallery which creates thumbnails.

mkGallery is written in perl and uses ImageMagik.

Using the default quality settings, I get 1 - 3 kb thumbnails from
jpeg images around 100 - 300kb.
