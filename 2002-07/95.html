RE: [OT] MacOS for PC

Wayne Simes wrote:
> Does anyone know if there is a version of MacOS that runs on
> an intel based
> pc ?

the essential answer is there isn't (and in all likelihood, never will
be) a genuine macOS that runs on intel hardware. in theory it could be
ported; there is ports work going on for the mach kernel of OS X i
believe, but that's not all the gui etc that really makes the macOS
experience.

it's not really in apple's interest to run macOS on intel hardware; why
would you buy a $7k mac if you could run the same OS on $4k hardware?
it's another selling point for them.
