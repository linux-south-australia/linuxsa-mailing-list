Re: java support in mozilla?

David Fitch wrote:
> 
> I'm trying to get java working in mozilla (0.9.9) without much success.
> I've got the 1.3.1 jre and copied the "javaplugin.so" file into
> /usr/lib/mozilla/plugins

I think you want to symlink the lib instead of copying it, and make sure 
Java is enabled under Edit > Prefs > Advanced, of course. If it still 
doesn't work after that, you *may* need to get a newer JRE. Which vendor 
and bugfix version (i.e.: blackdown/1.3.1, sun/1.3.1_02, etc) do you have?
