Re: mounting dos (win95) file system

> fdisk tells me this about my first (windows95) partition:
>    Device Boot   Begin    Start      End   Blocks   Id  System
> /dev/hda1            1        1      254  1024096+   b  Unknown
> 
> What is Id type `b' (I guess it is DOS, since it is less than 10), and
> what mount type do I need to make Linux mount it?  So far all I get is
> this (for type = msdos and vfat):
> 
> kernel: VFS: Can't find a valid MSDOS filesystem on dev 03:01.

Id 0B is FAT32.  FAT32 is not supported by the standard Linux kernel,
but you can get patches to support it at:
http://www-plateau.cs.berkeley.edu/people/chaffee/fat32.html
