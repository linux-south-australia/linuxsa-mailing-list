I gather (I think I lost a few messages) that the original poster wanted
to sort the linuxsa messages into a seperate folder. Easy. It doesn't
happen automatically, but it's easy enough to do.

Press ';'
Press 'T' (select by text)
Press 'A' (all text)
Type 'X-Loop: linuxsa'

This will select all the linuxsa messages. Now just press 'A' to apply a
command to all tagged messages, and then press 'S' to save them to a
different folder.

Then type in the name of the folder (which you have pre-created of
course), or just press CTRL+T for a point and shoot selection.

Provisos:

By default my pine came with aggregate commands turned off. This means
when  you press ';' it will say Unknown Command. To fix this, from the
main menu type 'S' for setup and 'C' for config, scroll down to
enable-aggregate-command-set and toggle it on.
