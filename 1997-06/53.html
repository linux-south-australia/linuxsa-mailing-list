Re: Fingers!

Alex Garner wrote:

> I want to make fingers to my site a little more interactive and
> intuative.
> Where do I get info on how that is all controlled/set-up?

  Unfortunately the finger protocol is REAL dumb.  You can write a
15-odd  line
C program to implement a dumb finger daemon.  (Even less in perl i'd
imagine).

Anyway, check out RFC1194 at (http://ds2.internic.net/rfc/rfc1194.txt)
That has the RFC that describes the finger protocol.  A 12 page RFC!!!

The program which is used for finger is /usr/sbin/in.fingerd.  It is in
the
NetKit package, an the rpm is NetKit-B-0.08-13.i386.rpm, there should
be the source on sunsite or on your RedHat CD (if you have it :).

There are also many versions of fingerd around. On sunsite, checkout
http://sunsite.anu.edu.au/pub/Linux/system/Network/finger/

Happy Hacking.

Matt
