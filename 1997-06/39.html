On Fri, 6 Jun 1997, Chris Trowbridge wrote:

> Regarding the attempting to telnet to the Linux machine 
> from Windoze 95.  Check in your Windows dir. for a file 
> called LMHOSTS.SAM (i.e. sample LMHOSTS file similar to 
> /etc/hosts on Linux).  Rename this to LMHOSTS and edit it 
> to hard code your linux box into the file.  Now should be 
> happy?
That's true if you want to use samba. For telneting there is a number of
things you can try. Check your network setting for tcp/ip in the network
control panel, make sure that your ip addresses are on the same subnet.
(That usually means that the first three number are the same.
(xxx.xxx.xxx.whatever). Make sure your subnetmask is the same (usually
255.255.255.0). run ifconfig on your linux box.

Use the linux box as your default gateway in win95. DNS wont matter much,
but be sure that you use the same domain on both machine Thats all I can
think of right now, but try it and let me know.
