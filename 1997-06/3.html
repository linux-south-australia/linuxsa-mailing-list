Re: netscape

> ok, so i can now set up a PPP connection, and i've downloaded netscape, how
> do i install it into X ?  thanks muchly

Log in as root.
Create a temporary directory to unpack Netscape into and move into it:
  mkdir netscape; cd netscape
Unpack the netscape-*-linux-elf.tar.gz file:
  tar xzvf /path/to/netscape-*-linux-elf.tar.gz
Move the netscape program into your /usr/local/bin directory:
  mv netscape /usr/local/bin
That should do for starters (to run it, just type 'netscape').
You can also remove everything in that temporary directory as it's not
needed anymore:
  cd /root; rm -ri /root/netscape (then answer 'y' lots of times)
You could do rm -rf /root/netscape instead, but that could be
considered dangerous (-f means "I know what I'm doing; ask me no
questions").

If you want to run java in Netscape, you need to install another file
or two as well as some other stuff to work around a bug in Netscape.
Ask later if you really want to do that.
