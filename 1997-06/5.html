> > ok, so i can now set up a PPP connection, and i've downloaded
> netscape, how
> > do i install it into X ?  thanks muchly

The way I do it is (as root)

cd /usr/local/lib
mkdir netscape
cd netscape
tar -xvzf [path to netscape]netscape*.tar.gz
cd /usr/local/bin
ln -s ../lib/netscape/netscape .

Done!

That will allow java, plugins, etc to all run.
