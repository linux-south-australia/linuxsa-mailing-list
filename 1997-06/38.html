Telnet to Linux from Windoze95

(My apologies for deleting the orig message)

Regarding the attempting to telnet to the Linux machine 
from Windoze 95.  Check in your Windows dir. for a file 
called LMHOSTS.SAM (i.e. sample LMHOSTS file similar to 
/etc/hosts on Linux).  Rename this to LMHOSTS and edit it 
to hard code your linux box into the file.  Now should be 
happy?
