SuSE 8.1YOU fails - Solved

Just to save some headaches for anyone else in a similar situation...

When I was working on the SuSE 8.1 upgrade from SuSE 8.0, I found that 
the Yast Online Update on all 9 workstations did not work out of the 
box. The new updates would not download or install.

APT was what saved me. I used it to install the newer version of 
yast-packager, yast-core and yast-packagemanager to get YOU to work 
properly.  Installing the newer version of these packages through APT 
solved the problem for all 9 machines.

I tried using the RPM patch from the SuSE update web page, but it seemed 
that the very tool used to install packages needed to be upgraded, so it 
did not succeed. It was at that point that I tried APT.
