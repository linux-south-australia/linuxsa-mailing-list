Re: 1 DNS Name 2 IP

Other question:

Yes, it'll round robin, bind etc will and have for a while - do a dig on 
the address and the first entry will change.  However, I suggest you think 
about it carefully:

- clients will cache for the ttl of the entry - this can lead to a problem 
with one proxy going down, then the client seeing no response.

- might want to investigate other options to hide the fact that there are 
two.  (eg. transparent etc)
