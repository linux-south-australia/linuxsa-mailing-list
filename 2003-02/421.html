Redhat 8.0 uses iptables. So, try doing this:

First of all, make sure that "#route -n" shows routes in both direction. I understand that you can use the internet from that (linux) computer, but can you "$ping 192.168.0.2" your windows computers?

Second, make sure that "#iptables -L" isn't blocking forwarded packets. 

Third, do a "#echo 1 > /proc/sys/net/ipv4/ip_forward" to make sure that forwarding is is turned on.

Lastly, if you want to, install and configure (not much to configure) squid for web caching. Save you alot of downloads.

* Go to your trusty windows computers and try "c:\>ping 192.168.0.4" and then "c:\>ping <a class="moz-txt-link-abbreviated" href="http://www.google.com">www.google.com</a>"
