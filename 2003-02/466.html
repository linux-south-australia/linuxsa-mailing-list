The correct place to set these variables is in the /etc/sysconfig/i18n
file.

In fact redhat 8.0 shipped with broken UTF settings so you have to edit
that file to fix it so all the man pages display correctly too. I'm sure
redhat has some mention of this on their website but I don't happen to
have the url handy. Perhaps their search engine will reveal it?
