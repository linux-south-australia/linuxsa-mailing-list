Re: Printing problems

On Thu, 2003-02-06 at 10:44, Edge, Martin (SSABSA) wrote:
> I'm new to Linux, and have just got a new machine, installed RedHat 8.0 and
> had a few problems.
> Printing (I have an Epson 440, which RedHat found). I've installed CUPS but
> can't get openoffice or mozilla to print to it. CUPS test page works, and
> the RedHat test page works, but can't find any details on getting openoffice
> or mozilla to print. I am assuming it is a simple setting somewhere.

I believe CUPS doesn't come with an "lpr" command (unless
you install some extra package) but most things use "lpr"
as the print command.  CUPS appears to use "lp".
So try changing mozilla etc to use "lp" instead of "lpr" (on
the print window, print properties).
