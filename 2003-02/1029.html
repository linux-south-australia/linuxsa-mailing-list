[ANNOUNCE] LinuxSA March 2003 Meeting

Hi all,

Time for the LinuxSA March Meeting.

Here are the usual details:

  When:   7:00pm (doors open 6:45pm) on
          Tuesday, 18th March, 2003
  Where:  2nd Floor
          Compaq Computer
          139 Frome Street
          Adelaide SA
  Cost:   FREE
  Who:    Anyone and everyone.
          No pre-registration necessary.

Presentation:

  Apple Australia and Tellurian Pty Ltd will be presenting a talk on
  Clustering for Beginners.  While it might sound like "Building your
  own nuclear power station at home", this presentation explains how
  ordinary systems administration skills are all that are required in
  order to build and run a useful, heterogenous cluster using open
  source software:

    We start with an explanation of MPI (Message Passing Interface)
    technology and where it came from, and how batch processing
    systems lead into the current Open Source GRID project. Then we
    define an example of a real-world problem which can be addressed
    with grid computing and solve the problem using a demonstration
    cluster.  The cluster consists of several Apple machines and
    others, all running a mixture of Unix-like operating systems.

    This presentation does not cover programming for clusters, or
    other advanced research-related topics. It will explain to you how
    to put together the infrastructure and deploy cluster programs
    that others have written.

    Further details will be advised closer to the time.

Extras:

  As per usual, if you are having a problem with your Linux
  installation, feel free to bring your machine.  Make sure to bring
  everything necessary to make your computer work (eg. monitor,
  keyboard, etc).
