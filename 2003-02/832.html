Re: problems with multiple X sessions

On Tue, 18 Feb 2003 16:03:54 +1030
Daniel Burke <burito@airnet.com.au> wrote:

You need to set you're XAUTHORITY environment variable to null
ie change you're line from this

export XINITRC=~/.victim;xinit -- :1

to this:
export XINITRC=~/.victim XAUTHORITY="";xinit -- :1

then it will work :)

Cheers,
