Todd, as others have suggested, if you can connect using M$ (Yuk! how COULD
you?) it is highly unlikely there is a fault with the ISP.

Using FBSD/Mozilla I find that turning my modem off, waiting ten seconds, then
on again, waiting for it to reinitialise, then trying again, seems to clear
problems nicely and thereafter I have a trouble-free run for as long as I need -
until I ignore ADSL for a prolonged period, when it might or might not need
resetting as described.

Worth a try.
