Re: Konqueror in Debian Woody

~/.kde/share/apps/konqueror/profiles
or
/usr/share/apps/konqueror/profiles
should have
filemanagement - this is one u're after.

Another option is to remove everything from 
konqueror profiles dir (both above), start konqeror
setup everything you like
and then go to
menu "window" -> "Configure view profiles...",
here enter "filemanagement" (without ""), and press Save.
should work, works for me at least ;P.

If you want to hide menu bar from defalut profile, you have to
bind keyboard shortcut to Save View Profile, then
hide main menu bar, and save profile using shortcut.
