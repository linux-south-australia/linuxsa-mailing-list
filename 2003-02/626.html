Re: kde error message

Romana,

>  i have an error message on one box : "couldn't start ksmserver. check
>  your install". 

The "session manager" didn't start.

> put error message into google, no obvious solutions, lots of people have
> this error in conjunction with others, and using kde 3.1beta, but noone
> has had it suddenly happen.

1) Log in as root
2) adduser romana (or some appropriate name)
3) rm -rf /tmp/*
   - and make sure that ls -la says nothing but . and .. are there
4) rm -rf ~romana/* ~romana/.*
   - i.e. get rid of any file in the new user's home directory
5) vi ~romana/.xinitrc
   startkde
   [i.e. make sure that romana's .xinitrc contains just one line

# cat ~romana/.xinitrc
startkde

6) chown romana ~romana/.xinitrc
7) su - romana
8) startx -- :1

..and see what happens.
