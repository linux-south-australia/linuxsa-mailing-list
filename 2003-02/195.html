Re: Adam Internet - DSL

I was having similar issues today with ADSL in attempting to connect to 
Airnet using RH7.3/rp-pppoe-3.5. The connection would simply time out. This 
box had been working normally with a basic setup using 'adsl-setup' until 
yesterday when SA had this national ADSL problem. 
http://forums.whirlpool.net.au/forum-
replies.cfm?t=61525

In the end I managed to resolve this issue by manually checking all ADSL and 
Network configuration files and correcting any irregularities. I am a little 
uncertain on the file I actually changed that eventually fixed the problem 
but your best hope in resolving this is somewhere in there.

/etc/ppp/pppoe.conf
/etc/sysconfig/networking/devices/ifcfg-*
/etc/sysconfig/networking/profiles/default/ifcfg-*
