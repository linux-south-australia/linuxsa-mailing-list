Re: cp help

On Monday, February 17, 2003, at 06:28 PM, Cameron Murray wrote:
> is there a way to copy and keep the existing directory structure?

If you just want to copy the directory, you can use the GNU extensions 
to cp which is "-a" for archive. Or possibly "-pR" for 
preserve/resursive, but I think that's OSF/1.

If you want to merge directories together, "tar cvf - blah | (cd bloop; 
tar xvf -)" works a treat.
