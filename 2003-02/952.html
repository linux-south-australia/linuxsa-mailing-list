fetchmail generally takes mail from a POP server and hands it, via SMTP to
a mail server which then works out how to deliver it. It doesn't deliver
directly to mailboxes in general (and I'm reasonably certain I've read
somewhere that if you do get it too then that's an unsupported feature and
you may lose mail).
