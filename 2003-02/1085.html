Kim,

> i have, usually by force set my window manager, etc
> but now would like to use gdm or kdm on my desktop
> (for a change)...

Heh

> so whats the best (read: standard) way to be able
> to choose how to set this up? (config files/init.d things)

There is a file in /etc/X11/ called default-display-manager. Mine says
this:

"/usr/bin/gdm"

..if you have installed a Debian version of KDM or GDM then you just need
to change that line to the output of "which kdm" or "which gdm".
