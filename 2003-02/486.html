Congratulations, the default printing system is enough for your needs
:-)

I think many distributions (Mandrake being the most notable exception)
default to using LPRng. This is fine for most users (particularly home
users), but in a heterogeneous network environment, CUPS has a bunch of
cool and useful features (like support for the IPP protocol, which is
also used by Windows and Netware). There are also a number of printers
which are best supported by CUPS and gimpprint+cups.
