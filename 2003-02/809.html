Big time. G2 is pretty damm good, and from what I hear about 2.4 (with 
the exception of smooth-scrolling), is only going to get better.

If anyone using G2 are looking for a decent G2 web browser, the Mozilla 
GTK2 port is looking pretty good. Whilst it is still experimental, after 
using it for a few months it seems to be only slightly less stable than 
nightly builds from the trunk.

It uses XFT and (using the classic theme) picks up my GTK2 theme quite 
well - it looks just like a normal GTK2/Gnome app. With stuff like the 
native filechooser getting hooked up, the GTK2 port going to become much 
better as well.

RPMs of semi-nightly builds are available from 
<ftp://ftp.mozilla.org/pub/mozilla/nightly/experimental/gtk2/>;. Doing an 
"alien -d *.rpm && dpkg -i --force-overwrite *.deb" installs them fine 
on a Debian/unstable box.
