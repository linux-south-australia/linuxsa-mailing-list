> 
> then when i get into a prompt i type
> service pcmcia start

When you say "crashes", do you mean:

 a) machine locks up
 b) kernel panics

You can tell the latter by the "kernel panic" message that appears on
the screen (if you're not in an X session) or by the keyboard lights
flashing. If it's the latter, you could try posting that info.

I have a Toshiba laptop that kernel panics when RedHat and/or Slackware
pick the wrong PCMCIA controller. I've also heard of people having
problems when their Toshiba laptops have their PCMCIA controller type
set to "AUTO", as opposed to setting it manually.
