I copy the comment block below into the top of my system crontab to
remind me what the fields mean. Saves making mistakes when I'm in a
hurry:

#minute (0-59),
#|      hour (0-23),
#|      |       day of the month (1-31),
#|      |       |       month of the year (1-12),
#|      |       |       |       day of the week (0-6 with 0=Sunday).
#|      |       |       |       |       commands
#0      2       *       *       0,4     /etc/cron.d/logchecker (eg)

Knowledge of the /etc/cron.daily cron.hourly cron.weekly etc directories
is handy for system admin type cron jobs that you can prepare a script
for and drop into the appropriate dir.

and of course, man cron, man 5 crontab
