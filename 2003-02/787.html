only up date what you need to ..

man apt

(i can supply more detail if you like ... just did something

I run testing/unstable and gnome 2.2 so it can be done ...:)
The quick answer _may be this_

#######
// /etc/apt/apt.conf


// set the preferred level of debian dist

APT::Default-Release "testing"; 

// Always show packages to be upgraded

APT::Get::Show-Upgraded "true";

#####
You can also fine tune per package in /etc/apt/preferences..
Though I haven't.
