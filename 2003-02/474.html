This may have been covered already, but have you checked that you're
using the correct lpr? RedHat distributions in particular ship with both
LPRng as well as CUPS. Have a look in /etc/alternatives to see which one
you're really using (/usr/bin/* or /usr/bin/*.cups). Without a bit of
manual tweaking, you'll have trouble using the LPRng 'lpr' command with
a CUPS backend.
