This is what is does AFAIK....

1. It puts a text file ending with a .1 into the proper format for a man 
page.

2. It copies the gzipped file into the same directory that both Lindows 
(I would assume Debian) and SuSE Linux 8.1 use for man pages

3. It updates the manpage database.

* All the user has to do is make a text file and plop a .1 on the end of 
it, and they get an  ----> instant man page.


***********************Personalize your computer and have fun! :-)

The script can be used to quickly personalize your computer . Yeah, I 
know its not kosher, but it could be fun. :-)


For example, someone might want to have an easy way of creating and 
accessing a Lord of the Rings encyclopedia on his computer based on the 
content of this page:

http://www.thecabal.org/gurps/rareitems/me_races.html


Anyway, I just wanted to clear up what the script does, albeit not much.
