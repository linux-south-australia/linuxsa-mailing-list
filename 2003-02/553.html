If you already know a bit about a linux system, give Linux From Scratch (www.linuxfromscratch.org) a go, to really learn linux inside out. I've learnt a lot from it, and now, wouldn't run anything else on my desktop :)

Also, just play around, installing/configuring a web server, ftp, mail etc...  Thats one of the best ways to learn. Start by following some of the many HOWTO's on the net.

I recently completed Certificate IV in IT - Network Management, and the linux modules (read "Red Hat" :P) left a LOT to be desired. But they weren't too bad as a _basic_ introduction to linux.
