Re: cp help

On Mon, Feb 17, 2003 at 06:28:27PM +1030, Cameron Murray wrote:
> is there a way to copy and keep the existing directory structure?
> 
> example:
> 
> contents of /test
> directory /a
> file /a/b
> 
> i want to copy /test to /temp which already has a folder called /a but no file called /a/b..
> 
> however the folder im copying from is a bit more complex than that!
> =P
> 


Cameron,

You could use two tar processes back to back.
The simplest way is something like this:

cd /test
tar cf - . | tar xvf - -C /temp

This has the advantage that all symbolic links are copied correctly,
which cp does not do (at least not by default).
