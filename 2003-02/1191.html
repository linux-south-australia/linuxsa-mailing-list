Lindows documentation project.

I am working with a couple of other Lindows users and we have started on 
a users guide for Lindows.  So far, all we have done is mine out 125 
detailed hints and tips, and put them in book format.  Anyway, a lot of 
useful information in the book, written by users that were hungry for 
details on how things work.

This 172 page compilation is going to be put into logical order, edited 
for context and such, but the current version of this thing can be found 
here:

PDF: http://www.morton2001.org/LindowsDocsHintsTips.pdf
LyX: http://www.morton2001.org/LindowsDocsHintsTips.lyx
Text: http://www.morton2001.org/LindowsDocsHintsTips.txt

Just thought I would pass on the info.
