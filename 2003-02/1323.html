Re: Tape backup capacities

Hi Michael, 

Compression ratios always depend upon the data your compressing. I've found 
for a random set of user and program data you get a 1:2 compression on 
average, but much higher if you are compressing mainly text, or much lower 
if you are compressing binaries/already compressed files/ 

Are you sure compression is turned on, I have found on some tape drives, 
despite an indication to the contrary that compression isn't turned on. Some 
backup programs aren't smart enough to tell the tape drive to turn on 
compression. 

Hope this helps a little. 

cya
