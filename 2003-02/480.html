Re: Printing problems

mgeddes@tellurian.com.au wrote:

>This may have been covered already, but have you checked that you're
>using the correct lpr? RedHat distributions in particular ship with both
>LPRng as well as CUPS. Have a look in /etc/alternatives to see which one
>you're really using (/usr/bin/* or /usr/bin/*.cups). Without a bit of
>manual tweaking, you'll have trouble using the LPRng 'lpr' command with
>a CUPS backend.
>
Well, if this works, I learned something else. :)

All I could do to get the printing to work on SuSE 8.1 was to uninstall 
CUPS, then install and remove lprng, then use apt-get to install CUPS 
1.18 instead of 1.15. If I didn't do this, the printing would suddenly 
stop working at a number of desks. I also had to install kde2-compat 
because Wine and Netscape print jobs would not work without it.
