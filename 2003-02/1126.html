RE: PHP Gurus? page link to change global registered variables ?

This may not be the best solution or answer, but...

I'm fairly sure that the session "a" is different to the HTTP_GET_VARS["a"] that you get via the address line. Try this:

<?php
   session_start();
if (!session_is_registered('a')) {
        $a = 1;
        session_register('a');
        print "Session New<br>";
}
else {
	
	if (isset($HTTP_GET_VARS["a"])) {
		$a = $HTTP_GET_VARS["a"];
	} else {
		$a++;        
	}
	
        print "Session Registered Already<br>";
}
print "<a href='test1.php'>Increment $a</a><br>";
print "<a href='test1.php?a=60'>Change $a</a>";
?>

Good luck.
