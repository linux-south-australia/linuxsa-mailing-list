RE: PPP Accounting?

If you "man pppd", there is a section on the auth-down script. This script
is run when the link is disconnected (if auth-up was run). It provides, Peer
name, Username, Connect time, Bytes sent, Bytes received and link name. A
simple script could then do what ever you need to do with this information.
