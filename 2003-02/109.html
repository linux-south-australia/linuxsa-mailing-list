If you only have installed standard redhat packages they come with all
the hooks you need to run them from the command line. No need to hand
edit any files, as you will most probably just break something anyway.

As root, the command....

chkconfig --list

...will list all the available installed services, and what run level
they are all started from. For ftpd the standard package for redhat7.2
was wu-ftpd so to start it the commands.....

chkconfig wu-ftpd on ; service xinetd restart

....is all you need to do. Now at every reboot or change in run level,
wu-ftpd will be started.

Do man chkconfig for more details.
