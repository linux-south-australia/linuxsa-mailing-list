Re: Commentary: The Greatest Flaw In Linux

On Tuesday, February 18, 2003, at 06:35 PM, Francesco Peeters wrote:
> Now if Apple would only make and sell MacOS/X for i386...   :-D

On a generic PC platform, if you have an issue with the sound card, the 
software has to be hacked and fudged to get around the problems. Apple 
just replace the sound card with something that works.

Microsoft are already doing the best job possible with supporting lots 
of different hardware, so what you see from Apple is only possible when 
the manufacturer builds and controls the hardware too.

If OS X was available for PC hardware, it would have all the same 
problems and quirks as Linux and Windows. The bottom line is that the 
idea of a generic OS for different hardware is an engineering 
compromise that doesn't quite pay off in all situations.
