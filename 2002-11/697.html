Re: interesting discovery

"> " is Bash's secondary prompt.  You can change the prompt by setting
the shell variable PS2.  The secondary prompt is displayed whenever a
shell command is incomplete, for example "if" before "fi".  In your
case it was displayed because you had an incomplete string.
