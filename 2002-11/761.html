RE: PHP -> Printing Cards?

The best way, is through PDF. PHP and pdf work very well together
because of this project http://sourceforge.net/projects/pdf-php There
are also some others around, just search at google and you'll find
heaps.

What ultimately will happen (if you write it nicely) will - it will
dynamically generate the PDF and then write it to the screen, so the
user can print the PDF out straight away... There are also other ways of
saving the PDF, emailing it etc. as you probably know.
