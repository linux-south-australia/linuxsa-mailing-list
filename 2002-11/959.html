Probably not for you (I don't think it does emacs key binding) but a nice java 
(and C++ and ada and VHDL) editor is jgrasp 
(http://www.eng.auburn.edu/grasp/).  It provides a "control flow and data 
structure diagram that is designed to fit into the space that is normally 
taken by indentation in source code. The intention of the CSD is to improve 
the comprehension efficiency of source code and, as a result, improve 
software reliability and reduce software costs. " (from the site).  I have 
found it quite good, especially from a student point of view (helps to 
visualise the code better).
