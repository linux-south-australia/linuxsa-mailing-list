Hi.

On RedHat derived systems, there is a file:

/etc/sysconfig/network-scripts/ifcfg-ppp0

There should be a line in it that says:

DEFROUTE=yes

I think you can set this through the gui, but I haven't used it for so
long that I forget where it is.
