Program to list NTUSER.DAT and potentially change it

Hi,

For those situations where you want to copy profiles from a Windows NT 
server and you cannot set the SID of your Samba server (Samba 3.0 alpha21 
allows that with net setlocalsid S-1-5-21-x-y-z), I have developed a 
utility that can list all the SIDs in the Security Descriptors in 
NTUSER.DAT. 

The eventual goal is to be able to replace those SIDs with new SIDs for 
the user on the machine, which will allow you to copy profiles from one 
server to another.

My next step is to implement the code that changes the old SIDs to a new 
SID, and then test that it works. The issues that I have are:

1. I don't know if there is an overall checksum on registry files like 
NTUSER.DAT, but there does not look to be one.

2. I don't know if there are any SIDs in the values of keys. I have found 
one KEY in the NTUSER.DAT I have and don't plan to do anything about it 
yet.

Some of you might be interested in such a program.
