I have used dyndns.org and although I don't have my own domain, I think

> The hosting company provides a small program you compile and then include in
> your boot scripts or run manually. It only needs to be run once and perhaps
> cronned if you're paranoid. This updates your IP on the hosting DNS servers 
> every five minutes so if your connection drops out (I get kicked every 6

I would think it more efficient to put this in /etc/ppp/ip-up.d/ that
way it only gets run when you get a new IP.
