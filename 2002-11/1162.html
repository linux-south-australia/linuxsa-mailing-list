Ralph,
I agree with Andrew, you need a firewall script that will block all 
incoming connection (new ones from the internet) and allow all outgoing 
requests (from the local network to the internet) and also allow 
incoming related and established connections (the reply packets your 
network send to the internet) as well as masquerading and IPforwarding 
(which is the parts it doesnt seem to have right now)

this should then fix the problem.

you can generate a firewall script here;
<http://morizot.net/firewall/gen/>;

if you need further help on how to make it work then let me know, but it 
is very well documented on that page and reading it will help you learn.
