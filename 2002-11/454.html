Re: Downloading Non-ISO OS?

I just installed SuSE 8.1 on my laptop over FTP.  Was really easy,

1.  make boot disk ( I used bootdisk.iso ~18MB) and boot from that and 
start install.

2.  during the install select the install from FTP option.  It will 
prompt you for you basic network parameters.

3. enter FTP address ( I used the internode mirror at 192.231.203.130)

4. enter directory of /pub/suse/i386/8.1/

5. it will install directly from mirror and only download and install 
packages you select.

I wouldn't want to do it on a dial up though ...


for more info go to 
http://sdb.suse.de/en/sdb/html/lmuelle_suselinux_internet.html


Good luck!
