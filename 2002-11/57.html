Re: Webmin

When I use webmin to access my server, I can do it at my workstation 
desk or at the server itself.

What is neat about it is that it doesn't make a difference where I am 
at, I still use the same procedure to open webmin.

But first, I have to know what the ip address of the server is.

So, I will usually log into the server as root and use this command at 
the shell prompt (The thing that looks like the windows command prompt...)

ifconfig

Weird name huh? ifconfig.

Now write down the ip number and open up your web browser.

See the second line, and on the left where ot says 192.168.7.133?

That's where your ip address will be.

Lets pretend that's what it is.

Go to the top of the web browser where you usually put 
http://www.yahoo.com (or whatever) and type the following:

http://192.168.7.133:10000

The 10000 means port 10000, and that's where Webmin usually runs.

What? You didn't find it?  You did? ok, skip down to the ***** area, 
otherwise read on.

Go into Mandrake Control Center/System and make sure the system is shown 
as running and that it is checked off to run at start up.

Still no luck?  Is the Bastille firewall running?  Yes...ok, that could 
be it. You will need to modify the firewall.

*****

When it asks you to log in, just use the same root password you already 
have and root as a user.

Hope this helps.
