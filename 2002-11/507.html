Hi Andrew, I recently just happened to be doing this.

Ok, I have a suggestion for another approach...

Save the file as text..

Import that text into LyX

Insert any pictures or graphics as objects....

Format in LyX and then export as PDF.

Looks really nice.

I wouldn't suggest this if the document had a lot of graphical drawings, 
but not pictures however, well at least I don't know much about that....

I don't know what format MS Word embeds its pictures in, but all I knew 
was that the print outs in other formats were very blurry, while the 
printouts from LyX were very sharp.

I wish LyX picked a different name because it is actually user friendly, 
even though it has some limitations.

Best,

JL
