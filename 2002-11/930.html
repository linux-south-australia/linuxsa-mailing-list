Re: [NEWS] United Linux 1.0 to release

Dan Shearer <dan@tellurian.com.au> wrote:
> I think that by the grossly abused metric of copies shipped -- which as I
> say seems to be what the media likes to rely on -- that Red Hat ships
> considerably fewer boxed server editions than SCO does. 

When I last did serious work on SCO, which was many years ago and they
might well have changed since then, you got no server software with the
desktop version of their Unix.  SCO could therefore accurately state
the number of servers they sold.  Red Hat, by contrast, provide almost
everything with their most basic edition, and so number of boxed server
editions sold is likely to significantly under estimate the number of
Red Hat servers installed.  I really can't think how to determine the
number of Red Hat servers 'out there.'
