Re: Array sort procedure using asp or jsp

I have done this exact same thing in one of my ASP projects.

All you have to do is post back the field name to search by. You can do some 
fancy ASC DESC depending on what the sort is at the moment. e.g if its sorted 
by ID ASC and they click sort by ID again, it'll order by ID DESC - its 
pretty simple logic work.

But the basic thing u do is post back the field name, the page reloads and 
you'd run something like this

orderBY = request("orderBY")
if orderBY = "" then
   orderBY = "table1.field1"
end if

sql = "SELECT * from TABLE1 WHERE Blah = 'Blah' ORDER BY " & orderBY

set RS = cnn.Execute(sql)

and voila, you have your "array" sorted by whatever you like :)
