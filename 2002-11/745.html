Printers ! <shudder>

Try CUPS there are web interfaces (that are actually quite good) and
command line as well.  I started using it this week. It does a great job
with my Epson Stylus Color but so far I have had little joy with an
Epson LX 300 .. (line printer -- well no piccies) .. Still looking into
that. (Me dumb not CUPS) This printer would be just as difficult using
lpr -- (I tried that too :( )

It also brings web and other standards to Unix printing which is
probably a good thing. You do not lose ghostscript or gimp-print drivers
either.(and there is an online database at linuxprinting.org that is
usefull)

I think it would be reasonably practical to write your own PPD files as
well... (though I have yet to even attempt this).

Oh and I am reasonably sure you can still cat to the device if you like
(with the right permissions) ...

Just some random thoughts ..
