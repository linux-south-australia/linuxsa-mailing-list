Re: Weather station under Linux?

>What interface card would be required etc.

Serial Port (9600 baud 8-N-1), search the web for wx918 and you will find
all of the relevant protocols and example 'c' code of how to decode the
stream.

I have three such units setup across Adelaide as remote weather stations,
each has its own embedded processor / battery / solar panel and 5W VHF
radio link.   Each remote WX node periodcally transmits weather updates to
a linux box at home which gets displayed on a GIS package on the X console.

They work very well for a cheap Weather station !    I also have the
designs for the B.Metorology white 'tardis' boxes in which to house such
instruments if you are interested.
