Re: mounting novel fs

Wilkinson,Alex wrote:
 >
> Is it possible to mount novell filesystems on Linux.
> If so, pointer ?

Select IPX support when building kernel and then install
user space tool ncpfs from <ftp://platan.vc.cvut.cz/pub/linux/ncpfs/>;.

RHL has IPX built by default.  Put "IPX=yes" into the
/etc/sysconfig/network-scripts/ifcfg-eth0 file.  You
can optionally indicate the frametype and if this is
a primary interface.  Best to set the frametype to
whatever the server uses (ask the admin, they'll know
as a mismatch is a common error) and set the primary
to be the eth0 interface for a client (servers with
multiple ethernets need to set a primary on a loopback
interface for service stability).  See the RHL doco
for the full list of sysconfig varaibles, their GUI tools
no longer seem to support IPX or AppleTalk.

Install the ncpfs RPM.  Then you can use the mount command
as per usual.  See the ncpmount manual page for options
that can appear on mount and in /etc/fstab.

The RHL printing system will also use nprint to print
to a NetWare printer, which is why I bother with IPX
at all.
