Re: This distro goes with that CPU?

Ian,

> Ok, thinking about distribution of distros and also about recommendations
> about the basic CPU/RAM/HD to suggest per distro.

That is quite a difficult thing to suggest because it can be one of those "it
depends on what you're using it for". For example, if you're not going to use X
(the GUI), then any distribution (even BloatDrake 9.0) might be useful.

> For starters, what current distro is suggested for a 386 or 486 home system?

If you've got all the bandwidth and time in the world, Gentoo. However, Debian
might also be a good choice here. Libranet would be useful because it comes with
a nicely setup IceWM "desktop", however trying to work out how to configure the
panel thing took me a little manual reading.

There doesn't seem to be any obvious tools to edit it with a GUI or such.
