Re: c shell containment

On Sat, 2002-11-23 at 16:33, Darryl Ross wrote:
> Can you set their shell directly to your script so when they close it they
> get dumped back at a login prompt?

Yes you can.  People used to disable accounts by setting the shell to
/bin/false (which is a Bourne shell script.)  If a command is executable
then it can be a shell.  To make a csh script executable you need, aside
from ensuring it's mode is a+x, to put "#! /bin/csh" in the first line.
