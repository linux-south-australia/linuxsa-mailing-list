Hi.

There's a package called uudeview that I use and which invariably 'just
works'. From the package info:

Smart multi-file multi-part decoder (command line) This is a command-line
decoder and encoder for files encoded with the following formats:
uuencode, xxencode, BASE64, quoted printable, and BinHex. This package
used to contain an X11-based GUI decoder called xdeview. That program is
now in the xdeview package.

It's included in most distributions, but I don't think I've seen it in any
default installs.
