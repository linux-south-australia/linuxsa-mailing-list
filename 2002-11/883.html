Re: This distro goes with that CPU?

>> Mandrake.  The whole distrobution has been compiled
>> for i586 for years now.
>
> What does that mean anyway? Isn't the whole i586 thing
> just a marketing tactic used by Mandrake. It is really
> just i386, but branded differently.

It means that the kernel is compiled for i586s and above, which is another
name for a Pentium. (Going back I think Intel wanted to trademark their
CPUs but you can't trademark a number, so they renamed the 586 to Pentium,
or something like that).

I can tell you from experience that Mandrake will not install or run on
anything less than a genuine Pentium. It doesn't run on 386s, it doesn't
run on 486s and it doesn't even run on Pentium Overdrive Chips for 486
boards.

> It is interesting how other i386 distro's run faster
> than the special "compiled and optimised for i586"
> Mandrake.

Not really done any 'speed' testing, so I can't comment about that.
