Re: Filename manipulation

David Sexton wrote:

> This is probably a really n00b question - I need to find a way of quickly
> automating changing some file names.
> Here's the story - I have a directory of 1700 odd images, and as many
> corresponding thumbnails. At the moment the thumbnails are named
> t_artx_xxx.jpg where x and xxx are numbers.
> I need to do two things in a hurry - I need to strip out the leading t_ from
> all of these file names, but I need to add -th to the end of the file name
> (but before the file extension).

Under sh:

for file in $(find . -name "*.jpg" -a -type f -print)
do
    mv $file $(echo $file | sed -e 's/^t_//' -e 's/\.jpg$/-th.jpg/')
done

Try it with an echo in front of the move to make sure you get the
results you want.
