Try doing this from the command line:

# ln -s /dev/psaux /dev/mouse
# X -configure
...
# XFree86 -xf86config /root/XF86Config-4.new
  [ it may be called something else; instructions will be given on
    screen ]

...and see if the mouse will move about on the gray background. If it
does then:

# mv /root/XF86Config-4.new /etc/X11/XF86Config-4

...and startx.

You may end up in 640x480 mode. E-mail me at dlloyd@microbits.com.au if
you do.

I'd suggest you turn off Kudzu. I don't like hardware autodetection
because it tends to stuff things up (whether you're using Windows or
Linux).

To turn off kudzu:

* ntsysv --level35
  ...and deselect kudzu.
