Re: c shell containment

On Sat, 2002-11-23 at 14:51, Axel Lingstadt wrote:
> How can I
> prevent these users getting access to the command line, either through
> control C or any other means.

Look, you really do need to read the man page before embarking on this.
You can prevent users escaping from your menu using onintr and disabling
job control, but they might still be able to get a shell from one of the
programs within your menu.  For example, suppose you give them a "vi"
menu option: they can use the ":sh" command from within vi.
