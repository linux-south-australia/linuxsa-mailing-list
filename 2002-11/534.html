Re: Html email gets a reply

Just to add my $0.02, and because i have nothing else to do, I 
created an experiment to see what the file sizes of these actually 
are.

I copied the text of this message into "todavid.txt", downloaded the 
attached PDF file "todavid.pdf" and created a HTML email 
"todavid.html" in Outlook Express, changing the formatting to look 
like the PDF file, and copying the source to a text file.

Then I used the Mac's "Get Info" to check the respective filesizes.

todavid.txt  -- 5,759 bytes

todavid.pdf -- 10,776 bytes

todavid.html -- 11,922 bytes

My initial theory theory for this was if the HTML email was even half 
way between the .txt and .pdf files, it was too big, but O M F G ! Let's 
see the HTML advocates get out of this.

P.S. I will only accept that Outlook was to blame if someone else 
sends me the same thing, using a different program, in which the 
source is not edited by hand ( i.e. source is created by the email 
app as it is edited, not altered afterward)

just a little bit of home science for the list.
