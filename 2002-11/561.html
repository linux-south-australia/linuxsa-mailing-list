Font install script

Someone was nice enough to share how he gets his Windows fonts installed 
in Linux last summer.

I just made a small script to automate the process.  The script has only 
been tested in Lindows
but should work in Debian or any other Linux system that uses apt.

It requires that your windows partition be mounted and accessable.

It is available at:    http://www.morton2001.org/fontcopy.sh

After downloading, set the execution bit, and type

sh fontcopy.sh

Enjoy!
