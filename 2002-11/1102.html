In my experience, SOLA + Linux sucks, don't bother, I think this has 
been discussed on the list before, check the archives....

All our production Linux boxen run off APC UPSes, theyre usually dearer,
but worth it.  There's a few free utilities out there too which work with
them,
for instance most of ours run the APC webmin module which gives you
the same type of monitoring graphs that you get with Powerchute, the APC
software.  That along with the apcupsd monitoring daemon and all the 
notification etc is taken care of.  

(The only thing I have to do now is get the script working for notifying
all connected samba users when the power goes down....)
