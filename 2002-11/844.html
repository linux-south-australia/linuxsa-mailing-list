Actually Eclipse is pretty impressive.  It uses non-portable Java for the GUI 
(ie JNI calls) so it is all Java, but optimised for the platform (currently 
includes Win98/ME/2000/XP, Linux [RH,SuSE], Solaris 8, QNX, AIX, HP-UX).  
Choice of JDK, a pluggable open-source architecture, support from every 
toolmaker except Sun and Microsoft.  What more could you want?

What _IS_ very interesting is that Eclipse forms the basis of Web Sphere 
Application Developer (IBM's premier web application development environment) - 
they just take drops of the Eclipse code, do some value-add and support that.  
Once you couple that with IBM's premier web server (Websphere) being based on 
Apache, you see how IBM is very much in the OSS camp nowadays.  Good to see.
