Grub problem in RH 8

Has anyone else experienced a problem with booting Grub after a fresh
install of Red Hat 8?

I'm getting following message at boot before hang:

	GRUB Loading Stage1.5ReadError

which is reported in the manual as:

	"Read Error"
	This error message will occur if a disk read error happens while trying to
read the Stage 2 or Stage 1.5.

extra info:
machine has old P3 smp mobo. Used before with Grub so bios has been
compatible in past.

New 40GB hdd. I had to use the 32GB restriction jumper before bios would
recognize this drive.
Maybe this is the problem. Compatibility between bios and new hdd in chs
mode.

I have used a separate /boot partition, it is 100MB at /hda1.

things I've tried:
machine boots fine from floppy created during install.
refreshed Grub installation with /sbin/grub-install /dev/hda
