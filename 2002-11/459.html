Re: Downloading Non-ISO OS?

Hi All,

Simon wrote:

> On the ftp site, there is a folder that contains a few
> hundred files. Is there a method to avoid having to go
> through every single file/rpm and downloading each one
> individually?
>
With command line ftp, you can use "prompt off" and "mget *" in order to
retrieve every single file in a folder (directory).

You can also use a mirroring tool, like wget, to do the same.

You'll end up with a copy of all those files on your hard disk - you then
have to build some sort of image to boot and install from (with Slackware
you use rawrite - dont know about your distribution).
