Hi,

Here's some rough code I made up for someone one time to do just that. This
will use a TIFF file as a background image for a business card, and put it
on a page with crop marks for easy cutting. Should be easy to modify to do
what you want, but getting pdflib installed will be the hardest part!

John Edwards


<?

$file = './test.pdf';

# pdf test.

$pdf = pdf_new();

pdf_open_file($pdf, $file);

pdf_set_parameter($pdf, "resourcefile",
"/usr/local/pdflib/fonts/pdflib.upr");
pdf_set_info($pdf,"Author","system");

pdf_begin_page($pdf, 595, 842);

$image  = pdf_open_image_file($pdf, 'tiff', '300dpibizcard.tiff');
pdf_place_image($pdf, $image, 170, 343, 0.24);


# some text?

$font   = pdf_findfont($pdf, "LuciduxSans","host",1);
pdf_setfont($pdf, $font, 10);

pdf_set_text_pos($pdf, 190, 363);
pdf_show($pdf,"Test Text");

# crop marks

#horizontal
pdf_moveto($pdf, 150, 343);
pdf_lineto($pdf, 100, 343);

pdf_moveto($pdf, 150, 499);
pdf_lineto($pdf, 100, 499);

pdf_moveto($pdf, 445, 343);
pdf_lineto($pdf, 495, 343);

pdf_moveto($pdf, 445, 499);
pdf_lineto($pdf, 495, 499);

# vertical

pdf_moveto($pdf, 170, 323);
pdf_lineto($pdf, 170, 283);

pdf_moveto($pdf, 170, 519);
pdf_lineto($pdf, 170, 569);

pdf_moveto($pdf, 425, 519);
pdf_lineto($pdf, 425, 569);

pdf_moveto($pdf, 425, 323);
pdf_lineto($pdf, 425, 283);

pdf_stroke($pdf);

pdf_end_page($pdf);
pdf_close($pdf);

print "Done.";

?>
