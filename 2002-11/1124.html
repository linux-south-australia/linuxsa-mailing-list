This package adds checks for and skips symlinked browsers.  Thanks to 
Troy Engels for this report.  I also re-arranged some of the symlink 
code in order to get rid of /etc/flash-plugin-uninstall.list.  There was 
one report of 6.0.62-1 being broken on SuSE 8.1 (desktop, not server) 
but I couldn't figure out the cause yet.

This package should work on any recent RPM distribution.  If you find 
any problems on any RPM based distributions please report.  I haven't 
had a chance to test this on PLD or Conectiva, so feedback from those 
distributions would be greatly appreciated.

(This is a beta installer for Macromedia's beta Flash 6 plugin.  Please 
use only for testing purposes.  Production use not recommended although 
it may work.)

* Tue Nov 26 2002 Warren Togami  6.0.62-2
- TODO: Cleanup redundant code

- Safety improvement

   from the installer.  This functionality can now be found in:
