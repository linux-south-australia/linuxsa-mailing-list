Re: Desktop plea

On Sun, 2002-11-24 at 07:53, Sue Robertson wrote:
> I"m running RedHat 8.0 & have a problem trying to change my default X
> environment back to Gnome.  (I 'desktop switched' into KDE in an attempt
> to find where to create a boot disk.  The "desktop switching tool' seems
> to only be working one way, as any attempts to change back to Gnome have
> failed.) 

Hi Sue,

a) Regarding the bootdisk, try "man mkbootdisk".

b) Regarding getting your GNOME desktop back, "man switchdesk" suggests
that your personal desktop choice is stored in ~/.Xclients.  Remove this
file from your home directory, and you should get the default, which
should be GNOME 2.  If there is no ~/.Xclients file, and you're still
not seeing GNOME 2, change the contents of /ets/sysconfig/desktop to be
DESKTOP="GNOME".

Hope this helps,
