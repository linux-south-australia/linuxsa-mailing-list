Re: [OT] Sun V100 boot net/disk problem

During the very first part of the boot routine hit stop-A to get into
OBP.
With 'printenv' you'll probably find that 'boot-device' is set to 'net'
Change it using 'setenv boot-device disk' 
If that doesn't work refer to Open Boot PROM Commands at

http://lios.apana.org.au/~cdewick/sunshack/data/sh/1.0/wcd00007/wcd007e2.htm

Examples are given and it's a great reference.
