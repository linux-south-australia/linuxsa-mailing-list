Re: Filename manipulation

Daryl Tester wrote:

> for file in $(find . -name "*.jpg" -a -type f -print)
> do
>     mv $file $(echo $file | sed -e 's/^t_//' -e 's/\.jpg$/-th.jpg/')
> done

D'oh.  "find" prefaces files with "./", so the t_ stripping won't
work because I've anchored it to the start of the line.  Try the
much clearer  -e 's/\/t_/\//'  instead (as Larry Wall calls it:
"Leaning Toothpick Syndrome").
