Re: MRTG

On Fri, 8 Nov 2002, James Collins wrote:

>
>     Have any of you folks ever managed to get MRTG to monitor aliased
> Ethernet devices? i.e.: eth0:0, eth0:1 etc

MRTG doesn't have to monitor via SNMP. If you can provide a shell script
which simply outputs the number of octets in and out of the
interface, MRTG can graph it.

See the documenaton for more details.

Also, I highly recommend using the RRDTOOL backend options of newer
MRTG, and the 14all.cgi script to show them.
