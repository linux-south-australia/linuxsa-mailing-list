Re: Filename manipulation

David,

> I've tried a couple of things, none of which have been successful. Any
> ideas?

This regexp should find what you want:

/(^t_)(.*)(\.jpg)$/

Put that into a perl script:

"transform.pl"


 #!/usr/bin/perl

 while (<>) {
	 /(^t_)(.*)\.jpg$/;
	 print "mv $_ $2-th.jpg\n";
 }

NOW, do:

# ls | perl transform.pl

IF the move statement look like they are behaving themselves, then
change this line:

... print....

to:

 `mv $_ $2-th.jpg`

Get it?

1) Get a REGEXP and save some results
2) Check the REGEXP is working
3) If it is, perform the action
