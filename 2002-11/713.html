Re: Distinctive Ring with Fax Modems

On Wednesday 13 November 2002 11:47, Michael Lindner wrote:
> hiyas,
> how do i get mu external swann speed demon fax modem (standard type  modem)
> to answer distinctive ring (telstra duet)  calls? redhat 7.2  with fax
> software (efax, mgetty etc)
>
This Netcomm manual shows AT-SDR=4  as the command code for setting up Telstra 
Duet distinctive ring. It may be different for the swann. 
It also says do not set Autoanswer for less than two rings so S0=2 should be 
enabled. Minicom can be used for testing. 
Disable mgetty and ring the fax number with minicom running. Look for the ring 
ring response, and see if your commands take effect.
