SuSE comes with a firewall script that is a) quite adequate and b) 
fairly easy to configure (it's done in config file and/or maybe via 
Yast2).

To do it by hand, edit:

/etc/sysconfig/SuSEfirewall2

READ THE COMMENTS and change things as necessary.

Then issue (as root):  rcSuSEfirewall2 stop ; rcSuSEfirewall2 start
