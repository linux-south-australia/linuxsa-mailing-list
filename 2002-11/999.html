Re: c shell containment

On Saturday, Nov 23, 2002, at 14:51 Australia/Adelaide, Axel Lingstadt 
wrote:

> Hi All,
>
> I am writing a menu system in c-shell (customer request) to allow
> application users quick access to various bits of information. How can 
> I
> prevent these users getting access to the command line, either through
> control C or any other means.
> These users will login as c-shell users, pass through .cshrc and then
> straight into the menu.
> Any ideas?
>

I'd rewrite the menu in C and compile it. Put your compiled program in 
/usr/local/bin (or somewhere else suitable), add it to /etc/shells and 
then make it the shell of the user(s).
Then when they exit the program, they get logged out.

You can execute other programs from inside your pseudo shell by using 
the system() or exec() functions.
