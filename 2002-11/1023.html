Mouse problems

My mouse absolutely refuses to work.  In both RH 8.0 and Mandrake 9.0, the 
graphical install and X config runs flawlessly, but on first boot it does not 
work.  I have a generic ps.2 wheel mouse, and I install it under generic 2 
button (because I got weird bugs when I tried installing generic 2 button 
scroll).  In RH 8, Kudzu informs me that my mouse has been removed on first 
boot.  In mandrake, it just refuses to work in X or gpm. I have tried disabling 
gpm, to no avail.  I read the error messages for X, and it said this:

xf86OpenSerial:  cannot find device or file /dev/psaux

It gives me the same error with other devices (/mouse, /psm0, etc).  In 
FreeBSD, my mouse worked with the mouse daemon enabled and X used 
BSD's "sysmouse" port with the "SystemMouse" protocol.  Unfortunately, neither 
RH nor Mandrake supports this scheme.  I am out of options.  Has anybody else 
had problems with this?  I don't understand why X is trying to open a serial 
port for a ps/2 mouse, but I can't find any documentation on why it does this 
or how to fix it.  Thanks in advance for any help.
