Much as I hate to say this:

Windoze: Run something called setup.exe. Click Next a few times.
Reboot (probably). Fire up new application. Works. (except you can't
read the help files if your system background colour is black, or some
similar fault which you just have to put up with)

Linux: Unzip tarball. Bugger about with config files, Makefiles etc.
Compile. Compilation breaks. Hack something. Compile. Breaks again in
different place. Hack again. (repeat...............................)
Install. Try it. Doesn't work. Hack something. Try again.
(repeat.................) Write tear-stained post to list begging for
help...

Installing Windoze 98 takes about an hour and a pot of tea. Installing
X 4.2 has taken me two days so far and the mouse doesn't work (which
is fairly vital) - incompatible with the supplied protocols, but it
works in Windoze as a "Standard PS/2 Mouse".

Yeah, it's personal preference. I prefer the Linux way because I _can_
hack things if I want to, but I could never pretend it's easier.
