Handspring Visor Automatic Setup Bash Shell Script for Debian & Lindows- USB Only.

I just finished making and testing a script that will automatically set
up your handspring visor by typing "sh visorsetup.sh" in bash.

USB Only!

It uses apt to automate the installation of jpilot and coldsync.

There is going to be an annoying scalability problem that I have to work
out because Lindows is a "root only"

version of Linux, and I used the $HOME variable to store the visor's
backup files, so the user will have to go
in as root to retrieve them. Maybe not so bad in this case??

It is GPL and available for download at:

http://www.morton2001.org/visorsetup.sh

This should work if you have a Palm Pilot. Make a back up first if you
want to give it a test.
