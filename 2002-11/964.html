RE: Bash variable

>In more detail, i am adding users to linux from a text file 
>list. There are
>thousands of entries. So i have started doing it by starting 
>letters; "a"
>first and then all the "b's". But still it doesn't work. It 
>only works with small numbers of entries.

Maybe you should doing it a different way.
If the text file is in delimited format it should be very 
straigtforward.

if textfile is in the following format:
username,f2,f3,f4

for f in `cat textfile |cut -d, -f1`
do
useradd <params> $f
done


Possibly this will still complain about too many entries, so
maybe try using awk instead.  Below is what I use to add users
from delimited text dump out of an NT box (using pwdump).

awk -F , '
{
	# Field 1 = username, Field 2 = UID
	system("useradd -n -d /dev/null -s /bin/false -u " $2 " " $1)
}' /home/user/ntpasswdfile


HTH
