Further to my previous post:
I'd like to create users from the smbpasswd file, format is
[username]:[uid]:[hash pwd]:[hash pwd]
Creating the users is no problem but they need to have the same uid as in
the smbpasswd file.
Is there an easy way of using two variables in the following, one for
username & one for uid:

For f in `cut -f1 -d: smbpasswd`
Do
Useradd $f -d /dev/null -s /bin/false
Done

Ie: something like

For f in `cut -f1 -d: smbpasswd` and g in `cut -f2 -d: smbpasswd`
Do
Useradd $f -d /dev/null -s /bin/false -u $g
Done
