Re: Web based control panel

"Charles Williams ( CEO ACNS )" wrote:
> Finally got the web server up and running hehe.  anyway, was looking at
> offering some folks hosting on the server.  But was wondering if anyone knew
> of a web based control panel that I could install for them to control thier
> accounts.  You know so they could change thier email info and alias' and
> such.  Any ideas would be appreciated.

If you are using qmail (http://www.qmail.org/) with vpopmail
(http://www.inter7.com/vpopmail/) to provide virtual accounts for e-mail
users, then the qmailadmin package (http://www.inter7.com/qmailadmin/)
is great for giving a "postmaster" total control over a domain name's
e-mail accounts, aliases, autoresponders, etc. via a web interface.
