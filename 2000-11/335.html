Nathan Millhouse wrote:

> I had added the line label="RedHat Linux 7" and on non floppy boots
> my PC was saying INIT: Entering runlevel 7, and would then stop

Never used a label with spaces in it before.  My advice is don't.

>     Further investigation of this uncovered that if I had a label of any
> length ending in a number with one space preceding it, INIT would enter
> that runlevel rather than the one specified in inittab. In addition to

Is this the default label, or are you entering this via the LILO Boot:
prompt?  Entering a numeric at the boot prompt tells the kernel what
init level to run to, so I'm not surprised that LILO is getting confused
(bear in mind that LILO is a boot loader - it can't afford to waste too
much program space, so it's possible that some of the program logic is
peculiar).
