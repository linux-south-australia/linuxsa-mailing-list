G'day Adam, 

This is works for me. The original text below was sent to someone reading Linux
Journal who noted I didn't mention MS Office in one of my articles. 


{Using StarOffice}
When I need to send out a document, I print using "Print -> Print to File" as
this creates a Postscript file.  I then convert this to a PDF file using
ImageMagick's convert program - it's a command line program and all I have to
type is "convert file.ps file.pdf" and it creates a file I can send anyone.

The reason I use PDF is so that it *won't* be changed by any other software
along the way, and it can be viewed/printed by any platform - Linux, Mac,
Windows, etc.

If you use StarOffice and have ImageMagick installed this should do the job for
you.
