	I think you should try using the back quote in the for command (note
use of "'" instead of "'"):
	for f in 'cut -f1 -d: smbpasswd'
	do
	useradd $f -d /dev/null -s /bin/false
	done
	This will evaluate cut -f1 -d: smbpasswd and pass the stdout to the
for
	command as the list argument. What your command doing is running a
for
	list of cut, -f1, -d: and smbpasswd
	cheers
	Dan
