DIA Problems Kinda Fixed

Well, I've kind of fixed my DIA printing problems. The steps I took
were:

1) Get the source
2) Install libpng (see below)
3) ./configure --disable-libarttest (see below)
4) Export as PNG works wonderfully but PS is still awful

However, GIMP etc will render PNG quite nicely.

Now, the reasons why I had to install some stuff and fiddle with the
configure options is probably because I've rebuilt everything in site
(except for Qt) with optimised compiler flags without bothering about
doing an RPM --rebuild; hence, there are bits and pieces lying all over
the place and configure scripts and even RPM can get quite confused.
