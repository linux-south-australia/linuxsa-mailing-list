Re: Converting Windows-style mail to Linux

On Thu, 30 Nov 2000, Richard Sharpe wrote:

> Hi,
> 
> Seems that there is plenty of interest in converting/recovering all that
> mail that people have locked away in Windows-based mail clients.
> 
> There are some hints up on LinuxWorld, but in addition, there are some
> tools up on FreshMeat that help convert Outlook Express mbx/dbx files as well.

Another very simple way if you have the infrastructure is to use network
mail standards. OE can talk to IMAP servers, so with a bit of configuring
you can go click-drag and your files end up on an IMAP server. That IMAP
server could be quite a few open source things. If it happens to be UW
imapd then all the mail will be in bsd flat file/mbx/whateveryoulike
format.
