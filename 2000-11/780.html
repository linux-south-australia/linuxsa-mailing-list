Re: iHUG Ultra and Linux

Andrew Reid wrote:

> I am looking at getting iHUG Ultra (Satellite) installed at my home.
> 
> The rates seem quite reasonable, but I wonder about linux compatibility.
> 
> Is anyone currently using the service?
> 
> I would like to know how you have gone running the PCI Card under linux,
> and also what your download speeds have been like.

I've used some of their older stuff in some linux boxes for satellite.
The only real problem would be that you need to use one of their modules
to make it work, matching their supported kernel versions (something
like 2.0.34 last time I looked..), and no access to source code.

The reason being that a lot of the satellite data stuff is insecure. If
you had access to source code you could use the dish/receiver in a kind
of promiscuous mode to monitor anyone else's data, or worse, spoof
packets using their IP, and get the data back anyway!

Keep in mind that the same data sent to you via satellite is also
radiating nearly all of Australia's surface, just like a TV broadcast.
