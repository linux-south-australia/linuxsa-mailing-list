Re: Perl Script... Stuck on the passing parameters from the command line

> I'm trying to write a Perl to preform the following commands.
> 
> if (! defined($ARGV[0]) {
>        die "Please type in the form of command user domain\n";
> }

use @ARGV in a scalar context to get the number of elements, ie:

die "usage $0 <user> <domain>" unless (@ARGV == 2);
my ($user, $domain) = @ARGV;

> And then do the following commands which are bash script...

use system() for the commands. Look at perldoc system, and perldoc in
general, there is lots of documentation available, and a book is a good
investment.
