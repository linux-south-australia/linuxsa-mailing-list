Re: Telnet Not Working under RH 6.2

Try these things.

1. Check that inetd deamon is running...
Running the command :-

ps ax |grep net

You should get the following results

  429 ?        S      0:06 inetd

2. Check that you haven't commented out the line that starts with
telnet...
it will look similar to this...
#telnet stream  tcp     nowait  root    /usr/sbin/tcpd  in.telnetd

Note that there is a hash # symbol in frount of it... In this case I
have commented out the telnet service...

Part from that we need more information... Is there anything that is
informative in the logs such as /var/log/messages...

3. Try grep telnet /var/log/messages


I hope that thoses pointers are clear...

If not, post back the results of the entries from those logs...
