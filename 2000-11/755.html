Re: Permissions

At 02:07 PM 11/20/00 +1030, Ian Hogben wrote:
>Hello, all:
>
>I am using Samba to share a fairly deep hierarchy of files, and I need to be
>able to force some default attributes on new files and directories. I have a
>strong feeling that this is possible, but I can't figure out how it is done.
>
>I'd like to set by default that new files and directories be created with
>the gid of the parent directory. Is this possible...? How do I do it if so?

What you need is the new 'inherit permissions' parameter in Samba 2.0.7.

It is documented briefly in Special Edition, Using Samba (on inside back
cover I think) and in the smb.conf.
