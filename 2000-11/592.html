I did a fresh Mandrake 7.2 install a few days ago from a
GPL CD set that I got from www.lsl.com.au, and it worked 
flawlessly. I did an "expert" install instead though, so 
perhaps give that a go (it's not that hard).

I also noticed that the 7.2/updates directory on Mandrakes'
web site already has about 150Mb of packages in there, so
perhaps check that your distro CD is current. The guys at
www.lsl.com.au usually update their distro CDs with the 
latest updates before they ship them, as my Mandrake 7.2 
CDs has all available updates (as of last week, anyway).

BTW, I am *so* impressed with Mandrake 7.2. This is a 
really polished distro, even more so than 7.1. If you're
considering changing distros, put this on your short list.
