Re: how to check possibly damaged cd?

Andy!

> Is there any way to check a CD for errors?

Use it and if you get errors, it has errors [1].

You may try the following:

* mount -t iso9660 -o ro,loop=/dev/loop0 cd_image.iso /mnt/cdrom

where:

-t gives the type
-o
 -- ro is read only
 -- /dev/loop0 is the local loop device

cd_image.iso is the image of the cd (obviously change this to suit your
image)

/mnt/cdrom is some mount point (can be any directory but don't use a
directory that's already in use!)

If you haven't fiddled with your kernel this should work. If you've
fiddled with your kernel then you might have uncompiled the loop devices
out though that's unlikely.

This command mounts the ISO image on your harddrive as though it were a
CD. If that doesn't work, or if you still get errors then your ISO image
itself is dead.
