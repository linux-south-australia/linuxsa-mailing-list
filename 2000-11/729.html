Re: html to PDF

Hi All,
    What exactly do you need? Acrobat Reader is _supposed_ to read PS files
with no problems, and so far I have not found any problems doing so. If all
you want to do is convert HTML to a file format that can be read by Acrobat
or some such, then html2ps should be fine. I am not aware of a tool that
converts HTML directly to PDF, but if it is a problem then I am sure someone
has done it out there on the net. Another option is to get ghostview and
read PS files directly, or double filter it as someone else suggested
(html2ps then ps2pdf). I would not recommend the later method if you want
high quality pdf files since there are often some fudges in converters for
various things, and in some cases you may end up fudging a fudge, which can
lead to some very strange results if you double filter. Hope this helps.
    Cheers!
        Andrew.
