> DHCP Served IP Address
> 
> Problem:
> THe linux box is served an IP address. The linuxbox has no hostname.

I think this might be the source of the problem. I think Linux likes to
have a host name assigned to it, even if all other Internet info is
served by DHCP, because that is only server at init level2 when Networking
starts, and some services like to know the host name prior to that.

Alot of startup scripts initialise using the "uname" or "hostname" command
to gain the hosts name. In RH Linux this is one of the first steps. The
HOSTNAME file contins the hostname, and this is set in /etc/sysconfig
and /etc/rc.d/sysinit scripts, called befor /etc/rc.d/rc2.d/...

Try using the linuxconf utility to set the hostname field, and see if that
fixes the problem.
