work for is looking at providing some of this GPL software (ecommerce)
in a server product.  

They seem to typically use MySQL as their backend (which I am not all
that familiar with).  Some of them are offering the option of
PostgreSQL.. (prefered).  I haven't had any success installing any
yet, but it's currently a "when I have the time" project.

Here are my notes so far on available GPL Software
- ------------
Symphero
	
  Ecommerse package based on Perl and Apache
  license: Artistic and GPL
  version: 3.0.3

  Release notes: 
    http://sourceforge.net/project/shownotes.php?release_id=7530

  An open-source e-commerce suite for Linux and other Unix-like
  operating systems, supporting multiple databases (currently MySQL,
  PostGRES and Solid 2.x), distributed object storage (in beta), and
  advanced marketing and affiliate tracking capabilities.

  Download:
     http://download.sourceforge.net/opensymphero/
     symphero.3.0.3.tar.gz 
     sos.1.0.b4.tar.gz  (108735)
     design-draft.sdw (96768)

  Additional Packages required:

    DBI-1.13 
    CGI-2.53 
    Business/CreditCard.pm 
    Mail::Mailer (available in MailTools) 
    libnet-1.07+ (particularly, Net::SMTP) 
    
    ...and you will need the DBD module appropriate to the database
    you are running. For example, if you wish to store your data in
    MySQL, you would need Msql-MySQL-2210. if you wish to use Solid,
    you need DBD-Solid, etc.
