oh, and one thing I forgot -- it's generally advisable to write scripts as sh scripts as opposed to bash scripts... to do this, put #!/bin/sh at the top, and make sure you only use sh constructs in the script, rather than bash-specific ones. This is so that if you ever need to port your script to another system, and it doesn't have bash, it will still work (sh is a POSIX standard, and always exists on all Unices...)
