Yep, the best distro is the one your used to :-)

I started with Slack as it was the only one that would run on my PC at the
time (kernel thing with large drives). RH 2 wouldn't install for me. But
then there were these RPM things everywher and they were easy, and RH was
easier to set up on the net, so I went RH4.2. Then the KDE thing came and
I used the Beta's but got sick of the D/L so I went Mandrake. So I've been
getting mandrake coz I'm used to it and know what I'm doing. I've
installed RH 6.2 and it's similar, tho the KDE desk isn't as well set up. 

It really doesn't make a big difference, you get used to what you
have. But this is my main machine for work, so I like it stable and I like
stuff to be easily available for it... I'm not a RH7.0 cutting edge type
person :-)  With RH I always waiting for the .2 release :-)

with Mandrake I've never figured the number scheme... 7.0 works for
me. Have 7.1 on the other machine and will upgrade to 7.2 soon.
