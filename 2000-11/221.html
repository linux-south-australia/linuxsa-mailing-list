Try 

Kernel-HOWTO

In my system it is in /usr/doc/HOWTO. It might be in a different place on
your system or not installed at all, in which case you should be able to
get it off of the web.

I personally strongly advise you read the howto befire doing a kernel
recompile if you have not done one before.

If you want short dirty instructions see below but I strongly suggest you 
read the howto it is not necessarily a trivial thing and migh tleave your 
machine unbottable if you muck some of it up - also I might have made
a mistake so please read the howto first.



Quick and dirty guide to kernel recompilation.
This assumes you want to use the kernel version you already have and not a
new version.

read  Kernel-HOWTO

cd /usr/src/linux
makemenuconfig

This will brign up a text absed menu program to create a config file.

Go through and choose your settings.

make dep
make clean
make bzImage
make modules
make modules_install
cp /usr/src/linux/arch/i386/boot/bzImage /boot/
edit /etc/lilo.conf
Add soemthing like:

image=/boot/bzImage
        label=newkernel
        read-only
        root=/dev/hda5

What exaclty you need to add depends on how exactly your system is set
up.
Basically you can copy an entry that is already in there and point 
image=
at your new image and 
label= 
as something unique.
label is what you will type in at the lilo promt to choose that kernel.

/sbin/lilo

You might want to make sure tht your old(currnet)kernel is not called
bzImage beacue you don't want ot replace the current kernel until you knwo
the new one works ok.

Reboot.

When you get the lilo prompt type in "newkernel".

I think that will do it.

When you are happy the new kernel does all you wnat it to, you might wnat
to add a default clause to /etc/lilo.conf or make your new entry your
first entry, which I bleive makes it the default kernel if there is no
default clause.
