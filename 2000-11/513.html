RE: Samba Sharing

Setup the share on the Samba server, give it guest access, & make it
writeable.
Then in your NT logon script run the net use command to map a drive to your
share.
Or alternatively if you cant get to the NT logon script, run the following
from a command prompt.

net use z: \\<SambaNetbiosName>\<Sharename>
