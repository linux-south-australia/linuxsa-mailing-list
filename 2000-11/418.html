Hi, guys,

it's advisable to have a separate firewall(s) on corp site - FW then DMZ
then FW. Before deployment check file and directory permissions, check
that all services are off, run scanning tools like Nessus, SAINT and the
like. To see that you firewall is running correctly run ethereal and see
that your ip chains are correct and the proper services are being
allowed and denied. Run Tripwire and tcp_dump on the Firewall and have
this util email you reports. 

YOu don't need much as a firewall, a P100 with 16Mb RAMs heaps. So it's
fairly inexpensive.

Get a book called Maximum Linux Security, Publisher SAMS, very good book
on cracking, malisious code, spoofing, etc. It show you how it's done
and preventions.

hope this helps
