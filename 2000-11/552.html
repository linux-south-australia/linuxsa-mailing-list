Andrew Pullin wrote:

> ---------- Forwarded message ----------
> Date: Tue, 14 Nov 2000 16:14:13 +1100
> From: Mail Delivery Subsystem <MAILER-DAEMON@serverXX.domainYY.com.au>
> To: postmaster@serverXX.domainYY.com.au, root@serverXX.domainYY.com.au
> Subject: Returned mail: see transcript for details
> 
> The original message was received at Tue, 14 Nov 2000 16:14:12 +1100
> from root@localhost
> 
>    ----- The following addresses had permanent fatal errors -----
> userZZ@domainYY.com.au
>     (reason: system config error)
> 
>    ----- Transcript of session follows -----
> 553 5.3.5 domainYY.com.au. config error: mail loops back to me (MX problem?)
> 554 5.3.5 userZZ@domainYY.com.au... Local configuration error

If possible, buy a copy of the O'Reilly bat book (sendmail). It has a
section on this problem. I believe that a solution is to add
domainYY.com.au to /etc/hosts, to let the mail server know that it is to
deliver mail for domainYY.com.au locally.

Hope it helps,
