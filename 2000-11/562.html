> Is there a man page on the passwd file as man passwd doesn't seem to tell me

There is a man page on the passwd file: passwd(5) (use 'man 5
passwd'), but it's irrelevant because '/./' in a home directory path
is a wu-ftpd thing, which is documented along under 'guestgroup' in
ftpaccess(5).

> When I got the personal web page going I found I had to add public_html
> under the home dir. I doc's I have read they keep refering to a web dir
> instead, is this a setting in apache as I had trouble find any referance for

That's an Apache thing.  It's the 'UserDir' directive in the
httpd.conf file.
