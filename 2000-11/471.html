Re: ip masq

> I've just setup a new firewall / masq box for home, running red hat 6.1
> which interfaces to my satellite/modem and acts as a router for my two
> desktops running win2k and linux. On one box I use napster, and i want to
> setup port forwarding for port 6699 so that it goes to 192.168.1.2. How do I
> do this ?

The command/program you need is 'ipmasqadm'

There is a howto on the www.linuxdoc.org that deals with ip
masquerading, and it has a very specific section on ICQ.  The ICQ
example is similar to the napster configuration but you change the port
numbers.  I'd get the HOWTO as it was an excellent help for me, and I am
a fairly experienced Linux user (nearly 4 years now).
