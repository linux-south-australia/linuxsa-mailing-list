Re: Mandrake update from existing installation?

Hi there.

A few days late but, if you haven't already upgraded, there is
another option to do so.

Rather than booting from the install CD, boot into Mandrake 7.2, go
to the install cd and there is a file in the root dir of the CD
called live-update.  This is their new application for updating your
OS.  It is meant to do a better job than doing an upgrade off the CD.

Although despite what the documentation says, be prepared for a full
reinstall.  The reason for this is that 7.2 has a small but
significantly different directory structure, in part to start
complying with the LSB released protype structures and partly to get
ready for Kernal 2.4.

Anyway good luck, and I hope all goes well.

Damien Byrne
