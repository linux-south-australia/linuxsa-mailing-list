Re: user storage limitation

D Thapa said:
> Hi,
>
> Some ISPs limits the user storage limitation to certian size, say
> 20M-30M, (one account holder can store files upto that size, or pay
> extra if exceeds).
>
> How can we set that limitation in Linux ? I am using RH7.1.
>
> Any pointers would be highly appreciated.

You can limit user storage by enabling disk quotas on the file systems that
the users have access to. You can also use disk quotas to limit their email
inbox size if you are using procmail as your local mailer (it obeys disk
quotas).

Take a look at the quota man page and its related man pages: "man quota"

Cheers.
