RE: Adding Daemons

Firstly I think it's Ben Not Andrew.

To check if the daemon is running you can do a command
ps aux|grep smbd.

If you are running the latest version of Clark Connect then it is based on
Redhat 7.2
If you are lazy you can go to the Clark Connect admin pages though a Web
browser on another PC and then set the default startup behaviour of each
Daemon.


P.S.  There is also the command
chkconfig  which allows you to turn on/off services default behaviour under
various run levels.
