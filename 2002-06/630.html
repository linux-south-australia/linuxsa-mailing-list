Hi Andrew

The following command converts a ps file to a pdf and emails it back to
the user who sent it.


print command = ( (/usr/bin/ps2pdf %s %s.pdf) ; (echo '' | /sbin/biabam
%s.pdf -s 'PDF of Printed Document' %u) ; (rm %s %s.pdf) ) &

Hope this helps

Mike O'C
