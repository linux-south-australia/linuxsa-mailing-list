United Linux - a good thing?

just reading about United Linux from my Tech Republic newsletter.

 Four of the largest Linux distributions are forming a
 new coalition. This group, composed of SuSE Linux,
 Caldera, Turbolinux, and Conectiva, will collaborate
 to define a common base for Linux servers, which each
 will then extend to create their own personalized
 distributions.

 The goal of UnitedLinux is to provide a single
 distribution for hardware and software vendors to use
 to ensure compatibility with their products. Each
 distribution must use the same base as that provided
 by UnitedLinux; components such as the kernel and file
 system layout will have to be identical between all
 four distributions in order for UnitedLinux to work.

 The aim is to have more hardware vendors certify their
 products for Linux. 


While they only seem to be focussing on server distributions one 
wonders just what effect this will have on the Linux desktop scene?

I think it is positive news, what are your thoughts?
