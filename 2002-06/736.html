Re: DHCP Server basic setup

What you have below will work, except you will need to change the

option host-name "pedarecc.sa.edu.au";

to

option domain-name "pedarecc.sa.edu.au";

And you will only need the server-name and next-server if you are booting
diskless clients. I would also put the default-lease-time and
max-lease-time at the start of the file, in the global section.
