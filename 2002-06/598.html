At Adam Internet, we have started using Redhat 7.3 for most of our tech
support and sysadmin desktop machines. Out of the box it's been more than
adequate for the desktop and doesn't produce the same sorts of time-wasting
problems windows does. I think it's the lack of random failures that make it
more pleasant in this role. ie: it still has time-wasting problems, but
they're consistent :)

Some of the power users aren't quite ready to switch yet, but for the
limited set of mostly web-based applications tech support need to use, linux
is ideal.
