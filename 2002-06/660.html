The following is some backgound information as well as the method I used 
to print from bochs. Since I am not a programmer, I created a shell 
script to print the parport.out text file created by bochs when the 
print command is executed.

Printing from bochs is a two step process. If you were inside of bochs 
you used a program to print, bochs will create a file called parport.out 
and place that file in your home directory.

To enable printing, bochs must be configured to print by editing the 
bochsrc file to contain a line that reads:

parport1: enable=1, file="parport.out"

You can edit the bochsrc file with any text editor such as emacs or kwrite.

After this file is created - you can run a command at the shell to print 
the contents of the parport.out file with the default printer in Linux.

Here is the print command:

cat $HOME/parport.out | lp.

Just type the command in and don't worry about what it means.

To automate this - create a new link to application icon and give it the 
following properties:

under execute type this in the command line box.

sh $HOME/bochsout.sh

CHeck off the box to run in terminal.

You should now create a shell script with a text editor and call it 
bochsout.sh

type the following in the file with a text editor:

#!/bin/bash
cat parport.out | lp

Make sure that the script properties are set as executible.

When you click on the icon it will activate the script causing the print 
command

cat parport.out | lp to send the parport.out file contents to your Linux 
default printer.

The bochs file is not erased until deleted or overwritten by another 
bochs print job.
