STLabs/IWill/Promise IDE Cards

Has anyone toyed with the ST Labs IDE controllers under linux ?  They also
have a RAID controller (0,1,0+1) which looks interesting.   They look to be
reasonably priced in the $55-70 bracket depending on where you purchase
them from.   These cards look to use the CMD 0649 controller.   Another
option is the IWill RAID controller ($90-$110) which looks promising, uses
a Highpoint HT370 chipset.

I believe that these chips are supported in the 2.4.x kernels, but am
concerned with the comments in the kernel code about CMD0649.

Apart from these cards the only other choice is Promise TX2 and TX4 RAID
controller but that is significant more $$$...

Basically I want to upgrade from an onboard ATA66 controller to ATA100 and
add an ATA100 RAID for some critical data.  All runs on a Slackware 8.0 box
with 2.4.18 kernel.

Comments ?
