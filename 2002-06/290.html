Hey All,

I'm trying to get procmail working system wide basis while still using
vdeliver for actual mailbox delivery on my server I have here at home.

Twin PIII-500
512Mb Ram
RedHat 7.2
Qmail 1.03
VMailMgr 0.96
Procmail 3.21
