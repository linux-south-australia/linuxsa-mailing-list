Re: Spam Assassin/Prevention

On Mon, Jun 17, 2002 at 11:52:45AM -0700, James Leone wrote:

> Is Spam Assassin the right tool for a linux workstation that uses 
> Netscape email - or is it just for sendmail? If Spam Assasin is not, 
> what the correct tool to use? Or is there one?

SpamAssassin works with Procmail. If you get your mail delivered to
your box via Procmail (one way or another), then SpamAssassin'll
probably do the trick.

If you're just using the POP-3/IMAP functionality built into
Nutscrape, then you might want to consider changing it around to use
Fetchmail or Getmail[1], which can use Procmail.

   - andrew

[1] I think Getmail can use Procmail, but don't quote me :-)
