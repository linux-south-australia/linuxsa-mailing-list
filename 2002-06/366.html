> the best I have read about so far is using apache httpd for 
> php and forwarding jsp requests to the tomcat server, but not 
> tomcat doing both. is it possible?

You need to use apache with php and tomcat, you can't just do it with
tomcat and php -- tomcat is jsp and Servlet specific, but also does some
extra straight flat-file stuff. I don't think many people would want to
use both together, as they are really alternatives...
