As others have written, your best bet is probably "wget". If you really want 
to use OMT for a particular reason though, it sounds like the problem is Perl 
related.

It would appear that the OMT package cannot find the Perl URI module on your 
system. Perl programs often use library packages that may not be installed on 
a system as standard. You can remedy this by looking for a URI module in you 
system distribution, and installing from there if possible.

If not, you can get it from CPAN, the "Comprehensive Perl Archive Network" (I 
think !), by doing the following:

perl -MCPAN -e 'install("URI")'

If CPAN has been setup correctly, it should download and install the module 
for you. If you have not setup CPAN before it will ask you some questions, 
and *then* download the module.

Unfortunately chances are if that module is not installed, other modules that 
the program may want will also be abesnt. You may have to repeat the above 
several times to get all of the modules needed.
