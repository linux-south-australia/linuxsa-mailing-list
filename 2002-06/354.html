> To check if the daemon is running you can do a command
> ps aux|grep smbd.

With Samba, you often want to see if all [two|three] daemons are running. 
If you are using winbindd, then it is three.

The following will do it for three:

  ps ax | grep '[bd]d'

while the following will do it for smbd and nmbd

  ps ax | grep mbd
