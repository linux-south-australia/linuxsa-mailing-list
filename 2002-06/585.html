LDAP 'problem' / query

Hey All,

Trying to implement an LDAP database here to centralise our system
accounts and staff address book.

directory, so that part works fine, but I have a query about the address

What I would like to do is put the division specific details into the
DivisionX entries and the site specific details into the SiteX level.
Things like the site address, phone number, fax numbers, etc, would
obviously be at the site level. What I would like to be able to do is set
it up so that if the site phone number changes I don't have to change 15
or 50 entries, only 1.

Is it possible to set OpenLDAP up so that if an entry for

cn=dross, ou=NorthAdelaide, ou=ICT, o=SYC, c=AU

is asked for, it will return the entries for that, as well as the entries for

ou=NorthAdelaide, ou=ICT, o=SYC, c=AU
