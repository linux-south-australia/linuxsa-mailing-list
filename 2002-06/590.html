er... no it's not... not according to www.debian.org, anyway...

AFAIK, it's available, but not released -- it's still in testing, and
hasn't made it to frozen yet, let alone stable. When will it get
there? ... when it's ready :)

> 
> You should read the manual for apt.
> but it can be as somple as apt-get upgrade dist (or similar)

apt-get update && apt-get dist-upgrade

sometimes you need to do this more than onces.

> You may need to edit /etc/apt/sources.list

Indeed, if you are presently running potato, you'll need to change all
instances of "potato" (or "stable") to "woody" (or "testing"). It's
a good idea (IMHO) to use the real name of the distro (potato, woody,
sid, etc) rather than the state of it (stable, frozen, testing,
unstable), so that you don't inadvertently get upgraded without
knowing... also note that the sources are not neccessarily the same -
there are no security sources for woody yet...
