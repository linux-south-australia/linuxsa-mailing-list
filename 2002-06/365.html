RE: Adding Daemons

Hi B,

Look in /etc/init.d/* (or sometimes /etc/rc.d/init.d/*) for a script
that looks like it starts smbd. Make sure it's linked into
/etc/rc.d/rc<n>.d/S<xx><name> or similar to make it start, where <n> is
the runlevel (or runlevels) you want it to run in, and xx is a number
that defines the order it starts in, and <name> is the name of the
script -- look at other files in /etc/rc.d/* to see what I mean re
linking...

alternatively, use inetd or xinetd to set it up.
