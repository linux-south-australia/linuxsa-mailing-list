I need to have a basic DHCP server setup ASAP to provide something like this
when you type ipconfig on a Windows workstation:

IP Address.................................: 10.60.110.123
Subnetmask.................................: 255.255.248.0
Default Gateway............................: 10.60.110.1

I only have want to assign IP's in the range 10.60.110.60 to 10.60.110.255

Below is my dhcpd.conf file, have read the docco but I still unsure if this

I'm just after confermation/corrections from the list 'coz I have to pull
the plug on the current novell based DHCP and turn on the Linux DHCP server
and have it work first go! :-)

Thanks in advance.

option routers 10.60.110.1;
option host-name "pedarecc.sa.edu.au";
option subnet-mask 255.255.248.0;
option domain-name-servers 202.174.40.33, 202.174.40.106;
