Re: AMD XP1800

The 1533 is correct - AMD went for an "equivelent speed" rating, because
there's quite a big difference in performance "per mhz" between the
Athlon(XP) and its competitors.
See <http://www.anandtech.com/cpu/showdoc.html?i=1543> for more
information.

As for the 1150 reading - thats definately wrong. I'm not 100% sure why,
but depending on how old the rescue disk is, you might want to try using
something more recent.
