> Any ideas on getting postgress to listen for connections?

PostgreSQL doesn't ship with TCP support enabled by default. From what I 
remember, you'll need to add the -i parameter to your postmaster.opts or 
postgresql.conf file.

This does present a slight security risk, however. Make sure that the machine 
is either firewalled off or on a private IP address.
