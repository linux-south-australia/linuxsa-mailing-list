> Just wondering if there is any way to read-mount the fat32 file system

Support for FAT32 isn't in any of the distro kernels yet. Many people
don't like the way the charsets are supported by it. If you need to access
to FAT32 partitions you can get a patch for 2.0.30. Go to
http://www.linuxhq.com/ and search for fat32. You should find a page with
various revisions of the fat32 patch. Unless you are running one of the
development kernels you will want version 2.7. I have been using
(read/write) this patch for a few weeks now and haven't had any problems.
If you can't find the patch, mail me and I'll give you the url.
