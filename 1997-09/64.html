Re: X and linux virtual terminals

> While on the subject of X -
> 
> Is there a way to switch back from a text-based virtual terminal to one 
> that was running X. The virtual terminal handler doesn't reestablish the 
> graphics mode setup for the X session (I guess it probably doesn't know 
> about X (or graphics) at all?). 
> 
> Is there some magic key sequence or something that can be configured?

Hi Mark,

The simple answer is "ALT-F7".

The slightly more robust answer is: when you start X, it doesn't start
on the text virtual console that you run "startx" from (eg. ALT-F1),
but rather it starts on the first free virtual console.  Since most
init setups have getty (your "login" prompt) running on the first six
consoles, X gets the seventh.
