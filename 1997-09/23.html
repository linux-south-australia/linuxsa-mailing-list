As an aside, in general root should have a *greatly*
reduced path.  ie only /sbin /usr/sbin /bin and maybe
/usr/bin.  This is for similar reasons to what Andrae is
alluding to.  It is usually easier to subvert a non-trusted
directory (/usr/local/bin) if users are allowed to write there,
especially if you are using NFS.  

It gets annoying, but for those that are truly paranoid it is
considered an essential part of life.
