Re: BackupDrives

Hi Michael,

The parallel Zip drive certainly works with Linux, having run it with
RH4.1.

To mount a Zip disk (DOS, VFAT or EXT2FS with reformat) is
straightforward:
- load module with 'insmod ppa'
- mount the disk with something like 'mount -t msdos /dev/sda4 /mnt/z'
for msdos.

To create ext2fs on Zip disk, use mke2fs.

I've also added the auto option to /etc/fstab to force dos, vfat or
ext2fs detection when mounting disk.

Use 'eject /dev/sda4' after umount to eject the disk if wanted.

There's doco in /usr/dos/HOWTO/mini on the Zip drive.
