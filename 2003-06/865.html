RE: Squid Auth. question

IF you want to do a lookup of the name of the machine (from an IP address)
and IF they are windows machines the following works really nicely.
nmblookup -A <ip>   (requires samba)
This does a node status and returns the name of the machine the name of the
current domain/workgroup name (username) of the person logged on the
machine
Hope that helps.
