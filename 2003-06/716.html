Re: ext2/ext3 Partition Recovery Tools?

Ryan Verner wrote:
> Hiya,
> 
> I've had a strange problem with my Debian Sid development box; something odd 
> happened to the hard drive, and nothing will no longer see the partition table.  I'm not 
> sure what exactly's gone, or why, but I sure can't get to my stuff.
> 
> Googling around shows there's a whole bunch of commercial/GPL'ed recovery 
> applications, but none that I can actually boot from a CD (there's no current Linux 
> install).  Anybody know of any that work well?  Ideally  I'd like it to repair what it could, 
> and make it bootable, but even if I can just access crucial files and pull them off, that'd 
> be enough.

If you know what the partition table said, then use "dd"
to pull off the filesystems.  Note the "noerror" option.

If you don't know the filesystem start/end then dd the
entire disk into a file and go superblock hunting.
