Re: Mysql access over network

if the '--skip-networking' option is passed to mysqld then tcp will be 
disabled for mysql. You'll probably have to edit the mysql init script to 
remove this option from where the daemon is called (or from some option 
variable somewhere - i don't use redhat so i can't check for you :-))

the problem could also be what some of the other replies described - 
insufficient privilages for remote access as defined in the 'mysql' database 
on your mysql server

hope this helps
