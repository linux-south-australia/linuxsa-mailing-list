James,
FreeBSD is not as "generous" with su and sudo access as Linux is. First
check to ensure that that you are in the wheel group. I should read
something like 
> id
uid=1001(george) gid=0(wheel) groups=0(wheel), 5(operator)

This will allow you to successfully  su to the root user. For sudo, you
need to edit the config /usr/local/etc/sudoers and add your user to that
file.
