Mozilla Firebird--which was once named Phoenix--is quite a good browser,
quite fast and it is more familiar to Mozilla/Netscape users than
Galeon.

I've had no issues with the version that is contained in Debian Sid
(0.6 which is the latest version) and have found:

 1 That the java plugin works quite nicely
   - I require one built with gcc 3.3 (I think) but other than that it's
     fine

 2 That the non-free flash html utility works

 3 That mailto: links are now working

It is also very, very fast unlike that piece of clunky rubbish that it
is slated to replace.

Its only drawback at the moment is that the personal security manager
which manages such things as TLS (or is it still SSL) certificates
hasn't been implemented yet. It is able to visit secure sites but if you
need to present a certificate to authenticate yourself you're stuck.
