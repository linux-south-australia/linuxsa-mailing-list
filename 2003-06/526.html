I use OpenBSD for my home firewall (and any other home firewall I 
deploy), because, although Theo is a pratt, the OS is pretty solid. pf 
(or ipf if you want to swing that way) is vastly easier to understand 
and configure than ipchains or iptables.

You don't need an iso, just download the branch you want, burn it to CD 
and use the cdfs file for your el Torito boot sector.

However, if you want a more general purpose box, possibly for desktop 
or dedicated server, go with FreeBSD, it's easier to install/configure 
if you've never seen a BSD before.
