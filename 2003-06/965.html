I got RedHat 9.0 for my birthday and I couldn't get to install on my main
computer unless I unplugged my 80 gig drive. To install the 80 gig drive in
the first place I had to set the jumper so that it reports itself as 32 gig
and use Ontrack's overlay to install it to it's full capacity.
I tried switching it off in the BIOS but Redhat boots and then crashes when
it is trying to change a partition to read & write mode. If you actually
physically unplug the main drive then it is O.K. I feel It is something to
