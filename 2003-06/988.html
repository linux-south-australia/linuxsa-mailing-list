Kieran,

> Anyone got anyideas why I can browse to http://localhost:631/ locally
> but can't remote into http://10.86.44.56:631/

Look for the cups configuration file - it only listens locally by
default.

- From my Debian, in /etc/cups/cupsd.conf

<Location />
Order Deny,Allow
Deny From All
Allow From 127.0.0.1
</Location>

It's very much like the Apache configuration file.
