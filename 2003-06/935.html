Re: File Logging

Mark,

> Is there a way to find out what user deleted a file on a linux system?

Sort of.

How do you think they deleted it? With something like konqueror or
nautilus?

> Is there a logging mechanism that I need to install to do this?

If it's a system file, find the package called sudo (it may already be
installed) and type:

 man sudo

It uses BNF format to describe how to setup the sudoers file - a format
designed to confuse anyone except logicians - but you'll get the
picture.
