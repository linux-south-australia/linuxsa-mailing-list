By "the drivers" did you mean X? If so, you should be able to run a more
current version of X. The FreeBSD version is not out of date (it's about
the same vintage as is found on many current Linux distributions), but
there are more recent versions. My laptop's running a stock FreeBSD 5.0
system with XFree86 4.3.0.

You could also try subscribing to BUGA (the BSD user's group), which I'm
sure Greg has some instructions for around the place.

It's probably also worth noting that parts of the BSD crowd may not see
the funny side in calling FreeBSD a distribution - it's not related to
Linux in any way and is it's own operating system.
