> But did either of them notice the faulty hard disks? My point is that

When the disk in the main machine actually did die, we were notified
that it had and that we had to fail the machines over. We didn't quite
trust an auto-fail over so we'd basically get a "Oh my god, I'm about to
or have died" e-mail.

When we performed the manual failover we discovered that the backup
machine was working perfectly but that the disk images were corrupted
because they had mirrored the corruption as the first machine started to
fail.

The code has probably improved.
