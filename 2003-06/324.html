Re: K3b device setup

Hey,

I think there are problems with your permissions for cdrecord, et al.

I can type:

cdrecord -scanbus

on my system as a normal user and it works. Try logging in as root, loading 
k3b, and running the setup wizard. Don't let it modify your fstab though 
(i.e. just leave your cdrom mount points alone), just let it setup the 
permissions for all the cd burning tools it needs. If this doesn't work, try 
setting up the permissions manually.

cdrecord has the following permissions for me:

-rwsr-sr-x    1 root     cdwriter   277932 May 17 04:24 cdrecord*

and I am part of the cdwriter group. Not sure what other permissions need to 
be set (certainly mkisofs, etc).

Hope this helps,

Damien
