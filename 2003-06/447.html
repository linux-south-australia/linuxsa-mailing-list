Anyway, a computer isn't terribly useful without an OS.  I can see
Microsoft's point of view by wanting to bundle an OS with the computer.  The
vast majority of buyers still want Windows with their computer, and not
providing any OS is an open invitation to piracy.  If you dispute this then
you haven't bought a 'white box' computer in the 90s, where it was very
common for the salesperson to ask "you have Windows at home, right?  *wink*"
to which the customer replied "why yes, *nudge* *nudge*".  $200 off the
price of a new PC.

The Linux community is going about this the wrong way.  We shouldn't be
fighting to unbundle Windows from the laptop, we should be suggesting they
bundle Linux with it instead.  The tightening of the EULAs has made the
first case very difficult to achieve.  What is Microsoft going to do?  Force
laptop makers to modify their EULAs stating that it's only legal to install
Windows on the machine?

Whether there is enough customer support for Linux/whatever else to be
installed instead of Windows is a different case altogether.
