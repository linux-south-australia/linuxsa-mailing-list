Using Belkin Power Supply Software (Bulldog Plus) in SuSE Linux 8.2

I recently moved our server over to SuSE Linux 8.2 after the server 
seemed to stop without explanation.  I come to find out that our APCC 
power supply was running out of juice, and it was not really a software 
issue with Mandrake 9.

In the process of changing the battery for the APCC power supply, the 
lid broke, so I went out and got a new battery box. I decided to get a 
Belkin F6C800:

http://web.belkin.com/support/download/download.asp?download=F6C800-UNV&lang=1&mode=

I did not know how feature rich it was until after I bought it, but here 
is a review of the software:

http://www.envynews.com/rev_print.php?ID=192

 From my experience, the software is easy to install, with one 
exception: it assumes that you will know the name of Linux's version of 
com1. It does suggest the name as /dev/ttyS0 however, so it shouldn'y be 
that hard. Before installing the software make sure that you connect the 
cable to the battery box and the server before starting the install. 
 Also on mine, it seemed to make a difference that I used the serial 
cable provided by Belkin and not my own.

Anyway here is what did to install it:

1. I mounted my cd by clicking on the cdrom icon
2. I opened up a shell console by clicking on the small blue 
monitor-looking icon on the bottom task bar
3. I typed --> cd /media/cdrecorder  [1]
4. Then I typed ----> mkdir $HOME/Belkin
5. Then I typed-----> cp /media/cdrecorder/Unix/Linux/linux.tar $HOME/Belkin
6. Next I typed -----> cd $HOME/Belkin
7. Next I typed -----> tar xvf linux.tar
8. Then I typed -----> su
9. It asked me for the root password, so I entered it.
10. Then I typed ./install
11. Then I typed y
12. Then I pressed enter
13. Then I entered 1 which means the battery box is connected to my computer
14. I then typed----> /dev/ttyS0
15. Then I typed ----> y

That's it!

Hope this helps!

James Leone







[1] Most commonly you would type  ---->cd /media/cdrom.  However the 
name of the directory is available in the window that pops up after you 
click in the cdrom icon in KDE. Just use that instead of /media/cderom. :)
