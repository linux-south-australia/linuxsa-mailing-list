Why do you think I'm slobbering over Gentoo so much? It has portage! 
Portage. Portage. Portage. It's the same as BSD ports, but better.

It's also not just Redhat with ports slapped on. The distribution is 
elegantly put together with a level of integration that results from 
having one build system for all the source code. It is a command line 
installation, but the installation procedure also teaches you more 
about how your machine is put together in 10 minutes than any Linux 
article, because you type the actual commands that perform the work.

Gentoo also runs on PPC... with no extra effort! So I'm interested in 
trying Mac on Linux so I can run Gentoo and OS X at the same time. Add 
in Virtual PC and User Mode Linux, and I'll be running a little network 
of machines all on a little laptop.
