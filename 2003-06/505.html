RE: Internet as user

Brian,

You should use "#groupadd" if you have it to add groups to the system. You should use use "#usermod" to add users to that group.

#groupadd mygroup
#usermod -G mygroup1,mygroup2 myuser

Each time you log in, this file is checked. So you don't have to do anything other than log out and back in.
