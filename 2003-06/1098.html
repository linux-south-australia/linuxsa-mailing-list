You might find that "apt-get dselect-upgrade" is an excellent choice.
apt-get upgrade will not remove currently installed packages
New versions of currently installed packages that can't be upgraded without 
changing the install status of another package will be left at their current 
version. If you've found this frustrating then "apt-get dselect-upgrade" is 
your friend, it used the simple apt-get interface but has the power and 
"intelligence" of dselect beind it.
Give it a go, you wont regret it.
