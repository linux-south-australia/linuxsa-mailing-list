>> Nope,
>> Aint a dongle. As someone else stated, a dongle is purely a copyright
>> protection device. CISCO call a similar device a tranceiver, which is an
>> acronym for transmitter/receiver. This tranceiver converts the signal to
> use
>> another medium, ie BNC to RJ45. So, if you had a NIC, Network Interface
> Card
>> with a BNC socket on it, and you wanted to connect a cable using an RJ45
>> plug, you would use the appropriate tranceiver for the task. They are a
>> layer 1 device, except if they are onboard the NIC they are a layer 2
>> device. So, a change from PCMCIA to RJ45 would be possible.
