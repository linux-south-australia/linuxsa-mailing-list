> What are people using for displaying squid statistics? PHP 
> preferred but all suggestions are welcome.

> Goal: Display usage stats based on IP and/or UID.

Personally, Calamaris does what I want.

By the way, I doubt you'll find a PHP solution, because the process of
generating these statistics, for a busy proxy server, can take quite
some time, and is usually run by cron regularly. However, it might make
sense to have a script that runs regularly, and dumps its results in
some sort of database, from which a PHP script could generate a page. I
don't know what advantages that would really have over static pages, but
it's a possibility.
