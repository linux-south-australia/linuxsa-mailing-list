Vinum root file system (was: gentoo vs freebsd)

On Tuesday,  3 June 2003 at 12:47:57 +0930, Richard Russell wrote:
> Greg 'groggy' Lehey writes:
>
> <snip>
>
>> Vinum now supports the root partition as well.
>
> Does this mean that in 5.0, I can boot off a fully vinum'd drive?

Yes.

> Any caveats?

Well, restrictions.  The boot loader doesn't know about Vinum, so it
needs a real partition to boot from.  The trick is to make that
partition overlap with one of the root file system's subdisks.

> Any restrictions (RAID1 only, etc)?

This means that at least one plex must be concat with one subdisk
only.  It's probably a good idea to make them all like that so that
you can still boot if something fails.

> Any doco? (I checked http://www.vinumvm.org/vinum/wishlist.html, and
> it still says vinum root is on the wishlist)

It doesn't any more.  You also missed the pointer to the docco:
there's the complete text of the Vinum chapter of "The Complete
FreeBSD" there for downloading.
