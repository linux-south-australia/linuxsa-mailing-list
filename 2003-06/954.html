I got RedHat 9 for my birthday

I got RedHat 9.0 for my birthday and I couldn't get to install on my main computer unless I unplugged my 80 gig drive. To install the 80 gig drive in the first place I had to set the jumper so that it reports itself as 32 gig and use Ontrack's overlay to install it to it's full capacity.
I tried switching it off in the BIOS but Redhat boots and then crashes when it is trying to change a partition to read & write mode. If you actually physically unplug the main drive then it is O.K. I feel It is something to do with this boards BIOS, as I didn't have this problem on my old board. I wanted to put a BusLogic flashpoint PCI SCSI host adapter that I had on the old board but the computer crashes straight after finding the adapter during bootup.  I've checked to see if it is an IRQ problem but I can't see anything wrong. The board is a PC chips M748MR.
