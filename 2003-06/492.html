Boot a rescue disk:

dd if=/dev/zero of=/dev/hda bs=512 count=1

should clear things out good.  from there you can make a new partition table 
with fdisk and see how it works.

Windows98 boot disk

fdisk /mbr

followed by MS fdisk- doesn't it read the partition table?

That said, doesn't the RH8 fdisk read the partition table?
