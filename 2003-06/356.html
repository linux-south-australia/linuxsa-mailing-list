Re: OpenLDAP / ADS / SUN One directory

Michael,

To sum things up simply, I'd suggest you spend some time investigation the
LDAP option. I am familiar with at least one organisation (with multiple
sites) that has successfully deployed a replicated LDAP directory that
supports authentication to a central samba PDC, allowing various Win clients
access to network shares etc (across sites). BDC's had not yet been
implemented, awaiting the release of samba 3.0.

The beta release in the last few days of samba 3.0 includes additional ldap
functionality (according to the release notes).

One downside of such an LDAP configuration, off the top of my head, is the
lack of some kind of client so that "non-tech" user's can say, add people to
the orgnisation's global address book. (Has anyone out there seen something
like this??)

Cheers,
Damian.
