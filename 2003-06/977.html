This is because your server is not "parsing" the file and executing the command you have in the index.html file. You need to setup server side includes on your webserver, generally this also means files with SSI would be defined with a .shtml extension.

Possibly you could renamer you .html to .shtml as a start, if it still doesn't work, google should be your friend...

http://www.google.com/search?q=enable+server+side+include+apache
