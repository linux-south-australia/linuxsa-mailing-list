RE: ATA raid

Here is a newer version of the HowTo http://www.murty.net/ataraid/

I had a go with RH7.2 and the install seemed to go well but the system froze
on first boot-up. I used the drivers I got from the Promise web site. I was
also able to boot from the RH7.2 CD in rescue mode, load the driver disc and
mount the installed partition on the RAID array and poke around in the files
but I'm too much of a newbie with Linux to be able to figure why it doesn't
boot completely.

I poked around in the kernel source to see what it said about my Promise
chipset (PDC20276). It seemed to be supported. I tried installing RH9 and I
saw  hde and hdg like you. I emailed the supporter of the relevant code at
RedHat arjanv@redhat.com and he told me the installer has no support for
Promise RAID but it is probably possible to build a kernel which supports it
once you have a Linux system. He also directed me to the ATA RAID mailing
list at RedHat and I found a HowTo http://www.tldp.org/HOWTO/ATA-RAID-HOWTO/
which look hopeful. I added another HDD (non RAID) to my machine and
installed RH9 on that. I can boot into Linux or Win2K by using my BIOS as a
boot manager. I plan to try the HowTo later.

John Bonnett
