RE: Samba

Yes, though possibly not securely.

You set the queues up on Win 98, however you want, and if it's possible
(I don't use window print queues), you restrict access to them so that
only the PDC can print to those queues. You set the PDC up so it has
printers that print to queues (again, I don't do this myself, so you may
need to fiddle -- or you could set lp up to do that for you, so samba
prints to the local lp, which prints to the remote print queues). Then
you tell everyone about the printers on the PDC, and not about the
queues on the windows boxes, which, if there is any security available,
you should be able to keep relatively hidden anyway.

Hope that helps (I don't use Windows printing at all, so it's rather
vague, but the general idea is sound)
