Re: Wireless Access Point - 802.11b

On Monday, Jun 16, 2003, at 13:53 Australia/Adelaide, Brett Hales wrote:

> I would like some advice regarding purchasing a Wireless Access Point
> (802.11b). I notice that many WAP's have a USB interface and Ethernet
> interface. I have been advised that the USB interface can only be used
> in a Windows environment. Is this correct? If so how do we get around
> this?
>
> Secondly, what make/models give the most 'bang for buck'?
>
> Thanks for the advice.
>

I bought a SVEC WLAP AP. It uses the ATMEL chipset and thus works with 
ap-tools (see freshmeat).  Thus, you don't need to use the USB/Windows 
config port at all.  :)

I also have an Apple Airport basestation, but for this you do need 
either MacOS / MacOSX or Windows.  I haven't found a linux config 
client for it yet.
