Re: Evolution and emacs

Hi Bill,

I've got an earlier version of Evolution (1.0.5) so maybe this is of
no help, but...

If you go to compose an email, in the window that comes up there is a
menu item "Edit->Properties" which brings up the gnome control centre.
In particular it puts you in "HTML Viewer".  In here you can set 
"Keyboard Shortcuts" to "Emacs like".  I don't know if Evolution 1.4
is the same, but perhaps it is.

I think when I last tried this, it didn't take effect immediately.  I'm
not sure if I had to restart Evolution.  Or perhaps I needed to restart
my X session.  I can't remember.

Hope this helps.

Cheers,

Mark.
