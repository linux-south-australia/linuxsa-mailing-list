Hi,

when you say 'downloaded', do you mean that you have installed it or
have just downloaded the setup.exe from the cygwin site?

After running the setup.exe and following the prompts to complete the
install, which may take some time over a modem connection, you will
usually be left with a shortcut to cygwin on your desktop and certainly
with a menu option under start button -> programs.

My advice is to do the smallest install possible unless you have
broadband. You can install more packages later.

By running the 'cygwin linux' program you get the bash shell. So if you
are getting the bash prompt what you really want to do is learn about
the bash shell.

Try http://www.gnu.org/manual/bash/html_chapter/bashref_toc.html
for information about using bash and then after that I would suggest
google.com for more info.

Packages that you may find useful in time are: cron, gnupg, tar, cvs,
zip, ssh, make, perl... and the list goes on.

goodluck, Stephen.
