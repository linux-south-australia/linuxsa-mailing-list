[ANNOUNCE] LinuxSA June 2003 Meeting

Hi all,

Time for the LinuxSA June Meeting.  This month we're going to have a
GnuPG key signing session.  This means that you will have to do some
work before the meeting.

If you don't already have a GnuPG key, you will need to install GnuPG,
and run 'gpg --gen-key'.
http://www.cryptnet.net/fdp/crypto/gpg-party.html and
http://people.debian.org/~stevenk/keysigning.html will be useful
reading.  For more general GnuPG info, see
http://webber.dewinter.com/gnupg_howto/ and
http://www.gnupg.org/(en)/documentation/

Once you have a GnuPG key, if you run this command:

gpg --export -a your-key-id
(eg. I would do: 'gpg --export -a g@netcraft.com.au')

and then email the output to keys@linuxsa.org.au, I will collect them
all up for distribution at the meeting.  If you can mail them before
the end of this weekend (15/Jun), I will also make the list available
for people to download on 16/Jun.
