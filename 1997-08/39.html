me
> where I can find out what you are doing with these lines? Maybe some
> information on the .inputrc file, what it is, when it is used? What does

> be able to know what the actually do!

You need to check that your console/vt220 generate these escape
sequences for these keys.

Try typing Control-V followed by each key, and see what it
prints. Note that escape prints as ^[, so `Home' should print ^[[1~
and `End' should print ^[[4~.

The inputrc file basically maps key sequences to commands.  You can
find some info on the .inputrc file, and the available commands, in
the manpage for bash.
