Re: Lost files

On Thu, 7 Aug 1997, Andrew Allison wrote:

> A colleague of mine has lost a file in his hard drive ( under 
> Slackware 2.0 ) ... I fear the worst.... In my experience there is no 
> "Norton's Unerase" for LINUX... because of the way in which the file 
> system works.... Is this true?

In the past there has been no 'undelete' for *nix systems because they are
multiuser/multitasking. Normally the space that is freed by deleting a
file is assigned to some other file pretty quickly. However if you have a
good knowledge of the filesystem structure you can recover some or even
all of the data, provided it hasn't been reused. Someone is working on an
'undelete' program for the ext2 filesystem but it is alpha.
