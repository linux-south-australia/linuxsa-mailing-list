Re: Adding memory

> Hi folks,
> 
> I just added 32Mb RAM to the already present 16Mb in my machine.
> The problem is linux wont recognize the xtra ram.
> i am wondering if I am supposed to recompile the kernel or anything. The
> memory is OK cos DOS and OS/2 recognize it.

You don't have a Compaq do you?  Some Compaq BIOSs apparently only
report 16MB with the standard BIOS call.

Try booting linux at the lilo prompt with 'linux mem=48M' instead of
just 'linux'.  If that helps, add a line to your lilo.conf file that
looks like this:
append="mem=48M"
(and don't forget to rerun lilo)
