Re: Linux & win95

> Unfortunately I have been put in a position where I am virtually forced to
> install win95 on my machine so I can test some portable software. I have
> heard that there were/are problems getting win95 to work with Linux
> because win95 does whatever it feels like with the partition table. In
> particular, is there anything to watch out for with lilo and win95?
> Ideally I would like to have lilo on the MBR and let it boot both Linux
> and win95. Will win95 boot from lilo-0.19 or 0.20?

Hi Mark,

The only problem should be that when you install Win95 it will erase
LILO from the MBR.  So... before installing Win95 make a LILO disk
which you can do by running "lilo -b /dev/fd0" with a formatted floppy
in /dev/fd0.  Use this disk to boot Linux when Win95 erases LILO, and
then run "lilo" to reinstall LILO on the MBR.

Win95 should be bootable from any LILO version AFAIK.
