Re: Directory listings

Paul Malcolm Bailey wrote:
> I'm a Linux newbie and need some advice on how to list directories --
> *only directories* -- in any directory in my filesystem.  

Hi Paul,

In Unix every problem can be solved multiple ways.  The Perl community says
TMTOWTDI (There's More Than One Way To Do It!).

I find the easiest way of listing directories is:

find / -type d

where / is the directory you'd like to start recursing from.  Of course this can
be the current directory, '.'  (That's just a dot by itself).
