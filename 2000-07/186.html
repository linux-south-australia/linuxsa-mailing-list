Re: Installing GNOME

Richard Watkins wrote:
> Everything has been working well, but now I want to also install GNOME.
> 
> I have succesfully installed the required XWindows rpm files, but are having
> all sorts of problems with dependencies for the GNOME files.
> 
> Is their a set order for the installation of these files? Also once
> installed how
> can I point "startx" to run GNOME instead of other desktops.

Hi Richard,

The best thing to do is to install them all in one big rpm command, ie:

rpm -Uvh first.rpm second.rpm .... three-thousandth.rpm

and let rpm handle the dependencies.  Need to know what goes together?  Go and
have a look at the download section of the Helix Code web page (specifically
http://www.helixcode.com/desktop/packages.php3).  Here under the "Required
Packages" section is what you need to install to get "Core GNOME" functionality
(whatever that is).  Also note that their RPMs will have slightly different
names to the standard Red Hat ones (like the addition on 'helix')

You could download their RPMs which are even cooler than std Red Hat GNOME RPMs
(or come to installfest to get them a lot quicker :-) 

Have fun - GNOME rocks!
