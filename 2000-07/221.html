> Can anyone recommend a good starting point i.e. books tutorial etc. that

I have also decided to learn python.  From the python web site

    http://python.org/

I started by downloading the html version of the documentation from

    http://python.org/doc/current/download.html

It is worth downloading this rather than attempting to build it from
the LaTeX source.  It is also well worth reading --- I have, and that
is as far as I have gotten ;-).


Otherwise, there is an O'Reilly book that I have considered buying

    Learning Python 
    By David Ascher and Mark Lutz
    O'Reilly & Associates
    ISBN: 1-56592-464-9
    Paperback, 300 pages, March 1999

It is favorably reviewed on Amazon with the caveat that it is most
definitely a book for beginners.
