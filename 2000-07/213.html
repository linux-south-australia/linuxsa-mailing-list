Stephen Donaldson wrote:

> I have decided that instead of trying to add/learn  bits and pieces of
> programming piecemeal that I will put all my time and energy into
> learning python and using that as the basis for any programming I want
> to do.
> 
> Can anyone recommend a good starting point i.e. books tutorial etc. that
> they might have used or seen with a good review.

Yes.  The HTML tutorial at www.python.org (which is available for download)
is very good, if a bit sparse with examples.  I use Mark Lutz' Programming
Python a fair bit for writing C extension modules, and it has good examples
on Python and program construction style, although it's out of date in that
it uses Python 1.4. (but the changes are minimal wrt 1.5.2 - I haven't kept
up though, so I'm not sure what 1.6 features will be introduced when it comes
out of beta).  One of the Fredrik's (Lundh?) has a PDF "eMatter" book out
chock full o' examples, and I'm keen to get my hands on that (he's one
of those regular posters to c.l.py that posts snippets of code that point
out language features/quirks).
