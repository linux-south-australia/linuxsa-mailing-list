Just to fill in a gap here...
The reason that the 3Gb drive couldnt be seen in Windows is because you only
created at extended partition.
In the extended partition you must then also create logical partitions which
you then see in windows as a drive which you can then format.

> Hi Ben,
> The disk can only be seen by DOS/Windows after it has been formatted
> eg: format d: /u
>
> What you appear to need to do is:
> 1) create a primary partition of 1.5GB on the second Hard Disk
> 2) reboot and then format that partition in DOS ->  format d: /u
> 3) reboot and begin the installation of linux and select the unallocated
> space on the second drive as the place to put Linux.  The installation
> will create the partitions as needed and then format it as an ext2 file
> system for you.
>
> *NOTE*  Don't forget to create a swap partition for Linux.  It will
> almost certainly ask you to do this anyway, but you need to leave space
> for it.  If you don't intend doing a whole lot of work on the Linux side
> ie. only for exploring and learning purposes, then you can usually do
> with just a 64MB swap, but maybe more if you don't have much physical
> RAM (< 32MB).
>
> regards,
> Don
