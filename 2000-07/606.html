IBM WebSphere Application Server

Greetings,
I've just gone to a IBM training course called Making Web Content Real.
Some of the tools / products included:
WebSphere Studio (runs on Windoze only).
WebSphere Application Server (Intel platforms: NT, OS/2, Linux).

Has anyone used WebSphere Studio as a content authoring / assembly /
management tool?
What products are available for Linux that author / assemble / archive /
manage websites?


The 2 products as a authoring -> server solution look good, but
cross-platform server support is patchy, (The current version of WAS is
not available on all platforms).
Never-the-less the technology is standards-based, which is A Good Thing.

Has anyone used WebSphere Application Server on Linux?
It essentially consists of DB2, Apache & an Apache plug-ins for running
Java Servlet Pages.
Since its Apache with some IBM enhancements, it still supports CGI, but
IBM's longer-term strategy is to go with Java Servlets, which are
cross-platform.
