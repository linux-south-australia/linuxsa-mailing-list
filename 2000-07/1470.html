Robyn Manning wrote:

> I've downloaded galeon and mozilla.
> 
> When I try to run galeon it says segmentation fault core dumped.
> 
> What does this mean and what can I do about it?

Means it's done a bad thing and needed to be killed. The core dump
contains the state of the application when the crash occured for those
who are that way inclined to debug it.

Try compiling from the source. I have found that with applications that
are heavily under development, it is often better to compile it on the
machine you want it to run on. Different systems have different stuff
installed and different versions of other stuff and it can get quite
messy.
