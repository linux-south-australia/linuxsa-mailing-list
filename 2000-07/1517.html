Re: Windows after Linux.

Hi Derek,
  I'm assuming that Linux is on /dev/hda1.  If this is correct
(which is what you seem to be saying) then all you need to do
before you install the others, if you wish to not touch Linux at
all but just leave it there, is to boot to Linux, place a floppy
in the A: (/dev/fd0) drive and type in the following:

    /sbin/lilo -b /dev/fd0

This will write the LILO to the floppy.  Once you've installed
the Winsux versions, boot off the floppy and you'll be left at a
familiar LILO prompt.  Type in Linux (or whatever you've named
your Linux image in /etc/lilo.conf) and once back in Linux run:
/sbin/lilo and then you'll be back whewre you wish to be.

Don.
