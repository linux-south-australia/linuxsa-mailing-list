> ne2000 NIC 33.6 modem 36x cdrom and umm anything else?

Shouldn't be a problem machine wise, compaq are notorious for using
proprietary hardware though.

700meg is very low these days. A full install of most distros is around 1.3 to
1.5gig. From memory, a redhat 6.1 installs in about 230 meg with NO extras
(absolute minimum install). Caldera is around the same figures, slightly less.

You'll need to set aside a minimum of 32meg for swap, but you probably been
there dun that.

700 meg is above the magic 512meg barrier and there are issues on 486's with
lba vs large addressing. Lilo will throw a wobbly if the reported c/h/s is
lying.

Which part of the install fails? Which distro? Or does it get thru but cannot
reboot?
