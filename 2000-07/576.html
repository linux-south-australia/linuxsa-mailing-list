Hi!

If I might jut in here:

It looks like I might be doing a reasonably sized doc project myself.
Whilst it's a little arcane, the tools I'll be using are:

* DocBook V4.1

At the moment I'll probably write XML compatible SGML

* Vi or Emacs

I prefer Vi whilst others prefer Emacs. It's a personal preference.

* CVS

This allows multiple people to work on the document all at once. In
addition, it allows me to roll back documentation because I envisage the
documentation being updated on a relatively regular basis. This means
that we'll actually be able to see what the system was like 5 months
ago, two months ago etc
