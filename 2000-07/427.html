"Jose N. de Vera Jr." wrote:
> I'm having dificulty on configuring it to use Maildir.. I just don't know
> where to put the command
> 
> % makedirmake $HOME/Maildir
> % echo ./Maildir/ > ~/.qmail

What this is saying is that if you have configures QMail to use Maildir,
then every user that intends to receive email needs to have a Maildir,
and a .qmail file that tells QMail where to put the mail.

For example, if my username is "andrew", then my home directory is
somthing like "/home/andrew/".

So, I need to make a directory called "/home/andrew/Maildir/", and
ensure that user "andrew" is the owner of this directory.

I also need a file in "/home/andrew/" called ".qmail", and in that file,
there must be the line "./Maildir/" (assuming that you want the mail for
user "andrew" to be delivered into the Maildir, and not forwarded on to
another account, etc.)
