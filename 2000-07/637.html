Re: How: mv x.tom x.dick x.harry -> y.tom y.dick y.harry ?

Alex Garner [alex@netcraft.com.au] wrote:

> There is something very similar and I have used it to do what you are
> describing. The pattern matching part of these operators is only bash
> 2.0.
> 
> echo ${example##/*/}
> echo ${example#/*/}

I've got it.  The following script works for the example "x.x.tom",
"x.x.dick.old" and "x.x.harry":

--------------------------------------------------------------------------
#!/bin/bash
#
# mmv: multiple file move.
#
# Usage: mmv <current prefix> <new prefix>
#
# Description: changes all files with prefix <current prefix> to file names
#   with suffix unchanged, but with prefix changed to <new prefix>.

for f in $1*; do mv $f $2${f#$1}; done
--------------------------------------------------------------------------
