RE: Backups

> I should have access to the /home dir structure on the kanchi server
> from the melkote workstation. Except using Bill Ball's example of of the
> fstab file for melkote seems incomplete. I believe that my entry should
> be

> kanchi:/home	/home	NFS	defaults	0 0

The man page for fstab mentions that if you miss of the entries 0 0
at the end of the line then 0 is assumed for both fields. I am not
sure what happens if you miss of the defaults field as well. Maybe it
assumes the value defaults, although this is not mentioned in the fstab
man page.

Look in the man pages for fstab, nfs and mount. Lots of useful information
in there.
