oops, are you using ipchains or ipfwadm? If ipfwadm, then you will want
something like ipfwadm -I -l -e while if you are using ipchains, then you
want ipchains -L -n -v.  In addition, depending on whether or not you have
forwarding and output rules, you might need to give additional commands.
