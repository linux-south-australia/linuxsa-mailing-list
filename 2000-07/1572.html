Re: ppp-compress module

On Tue, 25 Jul 2000, Michael Fewings wrote:

> Hello people,
> 
>     I was glancing through /var/log/messages (RedHat6.0) and noticed
> that when I connect to my ISP and pppd begins it trys to load a module,
> namely ppp-compress-[21,26,24] but modprobe can't locate it.
> 
> 
> So is pppd using the module bsd_comp?
> 

Yep, add these lines to /etc/conf.modules :

alias ppp-compress-21 bsd_comp
alias ppp-compress-24 ppp_deflate
alias ppp-compress-26 ppp_deflate

and you should find it working. Oh, BTW, from memory one distribution
had the same file renamed as  /etc/modules.conf (or similar). Try
looking for that if you can't find it first off.
