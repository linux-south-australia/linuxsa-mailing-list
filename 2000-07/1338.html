I personally feel that a group of individual taxpayers (corporate taxpayers
that is), would possibly add more weight than a User Group coming forward
demanding that a Linux version be produced.  With appropriate interested
parties (accountants, known business people, etc), we could probably go
a lot further.

I also feel that rather than 'give us a Linux version' we should push for
something more like 'let us work with you'.  Assist the ATO defining something
like a BAS XML schema to allow *us* and others to be free to develop software
with output that can be easily validated by the ATO.

So who is interested in actually seeing if we can achieve something here?
