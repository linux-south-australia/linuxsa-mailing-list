Directory listings

Hi.

I'm a Linux newbie and need some advice on how to list directories --
*only directories* -- in any directory in my filesystem.  I've checked
the man pages and info documents on "ls", but the closest thing I could
get was:

ls -R

which lists all the directories in your working directory (one by one),
along with all the files (including sub-directories) underneath every
single directory.

I tried:

ls -d

and:

ls -Rd

but they didn't prove useful.

They only way I could get what I wanted was to write the following bash
script:


#!/bin/sh
echo
num=0
for somevar in $(ls); do
  if [ -d $somevar ]
    then
      echo "$somevar/"
      num=$[num+1]
  fi
done
if let $[$num > 0]; then
  echo;
fi
echo -e "($num directories)\n"
exit 0


- which is not a bad effort given that last Friday night I'd never done
any bash scripting at all (and didn't know the language), and had the
above script working by the following Monday evening (i.e. last night). 
(Had I read man bash?  You better believe it!)

Anyway, the above script is what I'm happy using list out directories,
but is there an equivalent command to do the same thing?  Also, is my
approach in the above script the best approach for what I want, or is
there a more elegant way to do it?


R,
