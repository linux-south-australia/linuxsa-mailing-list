Re: Setting up a router...

On Thu, 20 Jul 2000, Dan Shearer wrote:

> Switching on ip forwarding should be done last of all, otherwise someone
> could sneak in before all your rules are set up.

... assuming the network interface is already running, of course! As Daryl
Tester pointed out here not so long ago it often makes sense to leave the
firewall rules in place all the time and bring the interfaces up and down
as required. It doesn't matter how insecure the rules are, if the
interface is down there is no risk :-)
