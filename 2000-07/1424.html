Re: Directory listing bash script

> for somevar in $(ls); do

Prefer: for somevar in `echo *`  (although this won't include dot files)

>  if [ -d $somevar ]

Put quotes around it:  if [ -d "$somevar" ]

> for anothervar in $(ls -ad .[a-zA-Z0-9]*); do

Prefer: for anothervar in `echo .?*`, but remember to weed out .. thus:

>  if [ -d $anothervar ]

Use:  if [ -d "$anothervar" -a "$anothervar" != .. ]


> is it true that to make a hidden file or directory, all you have to do
> is prefix the file name or directory name with a full-stop?

That's a convention, but it's not very hidden.
