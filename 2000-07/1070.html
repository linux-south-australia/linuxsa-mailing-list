Re: Gui vs Cli

I read about some guy years ago who build a "pipes" system for the Mac. Each
pipe was set up by changing it's name, sort of like the cut pipe you would
use cut -c3,5 as the name. Each pipe would send it's output to the pipe to
it's right in the folder it was in. It could handle sending the entire file
in one hit or character by character.

Not sure how well it worked, but the whole system did have a glowing write
up.

Rob.
Just my 2.2c worth.
