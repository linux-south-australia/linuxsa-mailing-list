Yo...

Everything you need to know is a HOWTO which youll find here
"/usr/doc/HOWTO/mini/IP-Masquerade" (assuming you have the HOWTO's
installed).
It explains the setup of the linux and win98 boxes.
I also recomend that you look at setting up a caching DNS server and web
caching (squid) so you get the most internet speed. For info on setting up
the DNS server look at "/usr/doc/HOWTO/DNS-HOWTO"

And yes, using those IPs is fine.
