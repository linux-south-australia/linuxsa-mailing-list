Friday, July 28, 2000, 9:28:43 AM, Ian wrote:

> Hi..

> I have downloaded and compiled the 1.9 version of the CD Record package.  I 
> am stumped if I can work out how to use it.

> Under Windows (yes, I am still trying to break out of the GUI 'press button 
> here' dependency forced upon me by Windows) it was easy, I just pressed copy 
> or moved a directory into the Adaptec CD burner software Window and Bob 
> became my uncle.

> I have tried using XCDRoast - but it complains that I can't mount the image 
> partition as 'root'.  I am not sure if I am supposed to have a seperate 
> partition for the image.

You  don't  need a whole partition for your image, just any partition will
do,  as long as you have at least 700Mb free (better to have 1400Mb ie 2 x
size  of  a  cdrom). Then tell XCDRoast to use an image FILE rather than a
partition  (ie  go to Setup and specify the file system to write the image
file  to  and  make sure you've selected the FILE option not the PARTITION
option or you will have the potential to overwrite an existing partition).

Hope this helps a bit...
