Try...

	rpm --verify $(rpm -qa) | more

Also, a useful thing to have around (I keep one in my home directory on
most machines I work on) is a listing of all files in all packages.
This is great when you need to find out what package file
"/usr/bin/blah" is in. Use this...

	rpm -qlip /location_of_redhat_cd/RedHat/RPMS/*.rpm > filelist

This gives you a file with each package description and a file list
handy. That way you can pretty easily track missing files. The options
are 'q' - this is a query, 'l' - list all files in a package, 'i' - show
package information, 'p' - query an uninstalled package.
