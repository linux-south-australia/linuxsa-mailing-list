Re: Alternative to Netscape

> > Friday, July 21, 2000, 6:48:00 PM, Nick wrote:

> > > learn something! My message is targeted to those people that have
> > > experinced with browsers. I am looking for someone that can advice an
> > > alternative browser to Netscape, or a version of Netscape that does a
> > > better job. Any suggetion would be appreciated!

you might consider kfm, the internal 'file' browser of KDE. Its a lightweight
non-resource hog that copes perfectly well with most html. It won't do Java nor
sound type plugins, and that, is no loss to me. It is very similar in
functionality to windoze in that it seemlessly integrates the local file view
with the internet view. Importantly, it handles drag 'n drop and through it's
inbuilt mime-types opens external file types automatically (such as a gif
editor, a zip file or whatever).

Most people would agree that kfm is very good at what it does. What it doesn't
do is surroundsound schlockwave or java encoded script as it doesn't consider
that functionality actually belongs in a browser. Your opinion might differ.
