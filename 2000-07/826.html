Hi Robyn,

With the 2.2 kernels I use this command.

	make dep clean bzImage modules modules_install install

or if I'm in a hurry (off to bed)....zzzzz

	make dep clean bzImage modules modules_install install; shutdown -r now

All one line.
This will ensure the new kernel is installed correctly.
