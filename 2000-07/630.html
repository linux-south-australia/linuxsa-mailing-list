If you are looking for documentation on a RedHat system (or an rpm
derivative), first of all find out what rpm the binary came from...

	[alex@Lisa alex]$ rpm -qf /sbin/lilo
	lilo-0.21-15

This tells us that /sbin/lilo came from the 'lilo' package, so lets
query it for all its files...

	[alex@Lisa alex]$ rpm -ql lilo
	/boot/boot.b
	/boot/chain.b
	/boot/os2_d.b
	/sbin/lilo
	/usr/bin/keytab-lilo
	/usr/doc/lilo-0.21
	/usr/doc/lilo-0.21/CHANGES
	/usr/doc/lilo-0.21/COPYING
	/usr/doc/lilo-0.21/INCOMPAT
	/usr/doc/lilo-0.21/QuickInst
	/usr/doc/lilo-0.21/README
	/usr/doc/lilo-0.21/doc
	/usr/doc/lilo-0.21/doc/Makefile
	/usr/doc/lilo-0.21/doc/README
	/usr/doc/lilo-0.21/doc/Technical_Guide.ps
	/usr/doc/lilo-0.21/doc/User_Guide.ps

	*Snip*

We can see that there are some pretty .ps files with tricky diagrams
and cool stuff. There doesn't appear to be any man pages in the
output, but that's beacuse some man pages are in the 'man-pages'
package...

	[alex@Lisa alex]$ rpm -qf /usr/man/man8/lilo.8.gz 
	man-pages-1.28-6

There is also a shortcut with rpm to tell us only about documentation
related files. The '-d' switch will help here...

	[alex@Lisa alex]$ rpm -qld lilo
	/usr/doc/lilo-0.21/CHANGES
	/usr/doc/lilo-0.21/COPYING
	/usr/doc/lilo-0.21/INCOMPAT

	*Snip*

Have a look at the query options with rpm (man rpm). rpm can tell you
a lot about the location of stuff on your system with very little
fuss.
