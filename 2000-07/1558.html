Re: Apache Logs

Because polite search engines are hitting your site.

robots.txt, is if not a standard, then a conventional way of telling
search engines and other similar programs which parts of your site you
dont want to be added into search engine databases or to be otherwise 
published.

I am not sure of the format of a robots.txt. file or if it has been rfc'd
or other wise standardised.

But if I wanted more info I would try www.w3.org orietf and search
the rfcs.
