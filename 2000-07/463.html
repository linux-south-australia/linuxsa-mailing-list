Computer Security

I came across this website which relates to Computer Security, but is also a
host to a rather comprehensive array of security resources.

It is a good site for an archive of various security exploits, programs used
to exploit the exploits, and all that kind of stuff.

I thought it might be a good reference for sysadmins etc - in fact you'll
probably find that some of the hacking tools are useful for legitimate
purposes :)

It also includes a large section of discovered exploits grouped by Operating
System, Distribution & Version.

The website URL is http://www.ussrback.com/ (Underground Security Systems
Research)
