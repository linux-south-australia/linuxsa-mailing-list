Re: Windows after Linux.

--- Derek Grocke <dgrocke@ctel.com.au> wrote:
> Hi all,
> 
> I know this has been asked before but I need to ask
> it again.
> 
> All on the same /dev/hda
> I have RH 6.2 installed on the primary boot
> partition. Total of 20Gb.
> I have a FAT32 partition and a NTFS partition..
> 
> I don't want to reinstall Linux but would like to
> place win98 and W2K Server
> on the other partitions.
> 
> I am aware that installing M$ software will kill
> Lilo..
> So how do I get around it or recover if it does?
> 
> Derek

ither make a boot disk when you are in linux, before
you install windows, with "mkbootdisk" I believe in
redhat (type "apropos mkboot" to see what the command
to make a boot disk is, if it isn't mkbootdisk), or
you can use loadlin through dos to load up linux that
way, either making it your default way to boot into
linux, or to reinstall lilo. If you intend on using
loadlin, then you should know where your root
partition is (eg /dev/hda2).
