Re: SCSI Disk Hard Drive speed

On Sat, 1 Jul 2000, Tim Ansell wrote:

> Hello, i've been playing around with my hard drive lately so i thought i
> would test out compared to the SCSI disk in at school. 

<SNIP>

> I was wondering if hdparm is lying, if so is there a better tool to
> check the speeds of these hard drives. And if hdparm isn't lying what
> can i do to make these drive run and something close to there rated (or
> even half there rated) speed.

For strong testing of drive speed use a program called bonnie. It produces
the most accurate results available. The thing you MUST do is make sure
any tests you do use at least twice the size of your ram. If not then
chances are linux will cache it and the listed speeds will be corrupted by
cache interferance.
