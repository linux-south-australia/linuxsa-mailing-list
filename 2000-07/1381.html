Re: GNUEmacs

G'day Neil,

On Tue, Jul 25, 2000 at 06:47:47PM +0930, Neil Walter wrote:
> Hi,
> 
> I've recently installed Redhat 6.2, and I have a trivial question. When
> logged on as root, emacs comes up in an aesthetically pleasing dark
> green background with cream coloured text (and a raft of colours for
> syntax highlighting). I'm used to this interface in earlier versions of
> RH.
> 
> However, when I invoke emacs as a user, it comes up in a large window
> with a white background reminiscent of some early X applications. There
> is only one emacs executable, and nothing in particular in the .emacs
> files.
> 
> Just one of those little things that niggle. Any suggestions (it's
> probably something obvious that I've overlooked).

Compare the .Xdefaults file for root with the .Xdefaults file for the
user, the answer is likely there, as many of the settings for Xemacs
are actually set here, and possibly as well for Emacs.  

Generally though, you should only use X as a user then su to root as
and when needed.  It keeps you out of (most) trouble that way.  :-)
