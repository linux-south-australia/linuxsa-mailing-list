The following instructions work for Mandrake. Small variations should work
for others. 

http://www.s3planet.force9.co.uk/sections/files/drivers/x/xf86_1011.tar.gz 

Save it somewhere useful, eg /tmp

Unpack this archive:

   cd /tmp
   tar zxvf xf86_1011.tar.gz

This will create a directory within /tmp called s3savage-1.0-11. Now:

   cd /tmp/s3savage-1.0-11
   ./install.sh

and you should get a message that says "Installation complete".

Now run Xconfigurator. Select any S3 Savage card (I used "S3 Savage 4").
Press Return.

For Monitor it should automatically highlight "High Frequency SVGA,
1024x768 at 70Hz". If not, select this. Press Return.

Under resolution select "16 bpp", which is 64 thousand colours. It can
probably do a little better, but leave it for now.

Select "Test", and the test screen should come up, filling the whole 14.1
inches of your screen.
