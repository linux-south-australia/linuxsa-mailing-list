Re: Rumour: MS uses apache

This thread can be pretty definitively answered by Netcraft UK, who list
every month what each web server developer runs in their own domains. The
report for Microsoft is at:

http://www.netcraft.com/Survey/Reports/200006/developers/microsoft.html

and this month only shows one Apache server, download.microsoft.com
running on Solaris.  (microsoft.org.za is not a za Microsoft domain, as
one might guess :-) There is also one Cougar server running on
netshow.microsoft.com.

Netcraft also do a section in their monthly written report at
http://www.netcraft.com/survey/ called "dogfood", where they round up
whose is using what relative to what they are selling. It is often
amusing, and they also pass comments on what they have observed being
swapped in and out of large web server pools.
