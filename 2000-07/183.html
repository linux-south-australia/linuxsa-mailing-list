It should ask you where you want to have the root (/) partition.
/dev/hda is the first IDE disk, /dev/hdb is the second, /dev/hdc third,
etc. So the first partition on the second drive will be /dev/hdb1.

Try checking out the documentation that should be on your CD. redhat.com
will probably also have some redhat specific info. linux.org is a very
good source of info for general linux stuff.
