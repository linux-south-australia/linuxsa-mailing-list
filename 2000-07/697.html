I'd say definately give Slackware a try.  I installed version 7.1
for the first time two nights ago - although the install is a little
utilitarian, it should be fine for anyone with a bit of experience (read,
easy for a 16 yr old with a year of use behind him), and it was the
//fastest// install I have ever seen.

I have not encountered any problems so far - generally, it's a
well-rounded, fast and stable distro.  Just don't expect nice integrated
GUI config tools (linuxconf / drakconf etc.) - still, editing config files
gets things done a lot quicker anyhow.  The BSD init system was something
new to me, but it seems a very simple and effective method.

The only downfall I can see is that lack of a robust packaging system -
however, slack's pkg tools are compact and easy to work with.  But I don't
think you would encounter many other apps packaged in this format on the
net.  Anyhow, installing from source is fine.

Just my thoughts :)
