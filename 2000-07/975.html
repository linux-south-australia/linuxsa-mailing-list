Re: can you copy files directly from win98 to linux?

Try explore2fs.  It allows you to access your ext2 (ie
linux) partitions through windows explorer (for local
linux partitions, like on a dual boot system).  You
can find it here:

http://uranus.it.swin.edu.au/~jn/linux/explore2fs.htm

Otherwise, if it's on a network, Samba is the way to
go.
