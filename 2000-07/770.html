Re: Dialup and RH6.1

> I just installed RH6.1 and now cannot work out how to change the scripts to
> achieve this again.
> Anyone know where i can find the code that does detect the loss of
> connection and where it re-exec's pppd?

I've been using RH6.1 for quite a while also.  The grahpical tool that
ships with RH (netcfg) will allow you to turn off the redial of the
connection by checking (or unchecking) a box.  

The first line in mt ifcfg-ppp0 file os:
"PERSIST=no"

without the quoutes.

Setting that to yes made it redial the connection when it went down.
