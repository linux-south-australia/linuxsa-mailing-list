FW: Anyone interested in this hardware?

The Adaptec 1510 scsi card does work with the aha152x driver (as will the
1505).
The trick is to known the irq and memory address of the card and pass this
as a boot option in lilo so the system picks up the card on boot. For the
exact string to use, check the scsi howto.
