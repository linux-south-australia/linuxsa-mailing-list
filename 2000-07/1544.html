Re: Samba question...

At 09:00 AM 7/30/00 -0400, Andrew Williss wrote:
>I am an absolute samba newbie beginner!
>Hi all,
>I have a RH6.2 box and a win98 box tied together via ethernet so I can
>modem share which is working great...
>I am now starting to have a play with samba and have installed
>samba2.0.7.
>I have set a smb.conf file and at this stage I can see the linux samba
>server from the windows box
>( it appears as \\will\)
>when I try to open the location I get a windows box that asks for a
>password...
>
>"\\will\IPC$"
>"password "___________________________
>
>I haven't a clue what password it's after as I haven't set one!!!!!

Common problem. You are using plaintext passwords on the Samba server,
which is the default. However, Win98 is insisting on encrypted passwords.

Either apply the plaintext password registry hack for Win98 (a .reg file
for it is in /usr/doc/samba-2.0.7/docs, I think), or move to encrypted
passwords.
