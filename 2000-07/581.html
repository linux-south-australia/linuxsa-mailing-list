SGML stuff

Hi,

For those who are interested - there is available a module thingo for Microsoft
Word 97 that appears to turn it into an SGML editor.  It, apparently, allows you
to specify a DTD, and verify and edit and graphically view the SGML as you go.
It's supplied my Microsoft, it's called "SGML Editor for MS Word 97" or
something similar.

However, when I tried to install it, it said "Could not locate the path to Word
in the system registry.  To fix, please run word from Windows Explorer and exit,
and try installation again." - which I did, to no avail.

Which stemmed my next question - .. I hope I've not asked it before .. - what
are some good GUI editors for SGML?  I know that they're not cheap,
necessarily..  but.  I'll even compile a list of ones I've tried and do a li'l
report type thing if people are curious.  (I added that last sentence because I
