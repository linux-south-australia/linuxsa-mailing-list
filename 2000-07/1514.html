Re: Windows after Linux.

At 04:13 PM 7/28/00 +0930, Derek Grocke wrote:
>Hi all,
>
>I know this has been asked before but I need to ask it again.
>
>All on the same /dev/hda
>I have RH 6.2 installed on the primary boot partition. Total of 20Gb.
>I have a FAT32 partition and a NTFS partition..
>
>I don't want to reinstall Linux but would like to place win98 and W2K Server
>on the other partitions.
>
>I am aware that installing M$ software will kill Lilo..
>So how do I get around it or recover if it does?

Install Linux first. Then win9x then Win2K.

After Win2K is installed, boot using a boot floppy, or the install CD.

Assuming that you have

   /dev/hda1  - Win98 (2 or 4GB)
   /dev/hda2  - /boot (20MB)
   /dev/hda3  - Win2K
   /dev/hda5  - Linux swap
   /dev/hda6  - Linux root

Then boot from the RH 6.2 install using:

   linux initrd= root=/dev/hda6 single

although you can boot all the way up if you want to, but will have some
problems with drivers loading.

Once in Linux, edit /etc/lilo.conf to reflect the above. Actually, what you
want is:

   other=/dev/hda1
     label=crap
     table=/dev/hda

and then rerun lilo with

   lilo -v
