Galeon.  http://galeon.sourceforge.net/

It's a GTK / GNOME ( i guess ) browser, that uses Mozilla's gecko
rendering engine.  I've heard it is //very// fast, and would be a good
alternative if you don't like Mozilla's  clunky XUL interface.  It
requires gtkmozembed.h from Mozila M16.  Due to lisencing issues, this
cannot be distributed with Galeon itself, so you have to get it from the
mozilla-devel rpm or from the source code tarball.

Note, I haven't actually used it myself yet, I am just suggesting it from
others' recomendations.  However, seeing as it uses Gecko, it should be
pretty good, and will improve.
