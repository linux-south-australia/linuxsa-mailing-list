Re: Dialup and RH6.1

> > I have a normal modem dialup connection to my ISP.  I have been
> > using RH6.0 for quite a while now and for several reasons, i
> > changed the ifup-ppp script so that it would not redial on loss of
> > connection.  I just installed RH6.1 and now cannot work out how to
> > change the scripts to achieve this again.  Anyone know where i can
> > find the code that does detect the loss of connection and where it
> > re-exec's pppd?
> 
> Ummm, as I recall, there is a variable in ifcfg-ppp0 that controls
> whether or not ifup-ppp redials ... Why not set that variable to no
> or whatever.  You should be able to see it if you edit ifcfg-ppp0
> with vi etc ...

Yes thats the one. The option is PERSIST=yes. Put it in there rather
than in /etc/ppp/options as you will won't break any tools like netcfg,
linuxconf, rp3 etc etc etc.
