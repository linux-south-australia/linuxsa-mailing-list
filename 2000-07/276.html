Re: Client's IP in telnet

On Thu, Jul 06, 2000 at 02:49:18PM +0930, Nick Morrison wrote:

 > My host has a dynamically-assigned IP address.  Could be anything.
 > (realistically, it's usually the same, but)
 > X applications use the "DISPLAY" environment variable in order to
 > determine where to send X clienty messagey things.
 > I'm probably overlooking something really, really simple here.. but
 > is there a program, or an environment variable type thing, that I can
 > use in a li'l script that sets my DISPLAY to wherever I'm logged in
 > from:0.0?
 
If the UNIX host you're logging in on supports ssh, use that instead of
telnet.  Not only is it more secure, but it also sets your $DISPLAY
automatically (X sessions get "tunnelled" over the same encrypted session 
which carries your keystrokes).

Apart from that, you should be able to parse the output of the "who am i"
command (which includes the IP address you've come from) to work out a
rational value for $DISPLAY.
