> Why?  You need to substantiate that claim.  It doesn't correspond to

Mainly because of the fact that SCSI drives on the same "bus" can read and
write at the same time, where IDE drives on the same controller cant.  If
you have 4 SCSI drives in RAID 0, the speed should be 4x that of one single
drive. With IDE, using two controllers (prim & secondary), in the same
situation, youll only get 2x the speed.

> > Although... compare the price diff between a 2 drive SCSI RAID
