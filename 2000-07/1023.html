The Gnome people were working on something like this, using CORBA to
share objects between applications. Bonobo I think it was called. Drag
and drop requires changing focus to an application, performing the op,
dragging the data to another app, perform op, drag etc. A pipe can be
setup with commands so that you set the batch processes off, walk away,
come back and it's done.

The REAL power of the command line or script is that you can work out
how to do something once, put it in a script and have it execute when
you want. CLI's scale much more than GUI's, though perhaps at the
expense of initial learning curve.
