Hi,

I use Outlook 2000 with our Exchange server here...  The rule I use looks
like this...

"Apply this rule after the message arrives 
with linuxsa@linuxsa.org.au in the message header
move it to the Linux SA folder
  and stop processing more rules."

This has caught every message ever thrown at it, including Alex's demo
message.  It doesn't explicitly filter using X-Loop, but if X-Loop is all
there is containing linuxsa@linuxsa.org.au in the header, it'll find it.
