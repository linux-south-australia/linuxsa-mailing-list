There is a command that allows you to clone the floppy version onto a hard
disk dos partition, and an option to use a swap partition (which is
recommended if you have 8MB or want to run a few more services than normal
if you have 16MB or there abouts). You can use 'hdparam' or your PC BIOS
to set the hard disks to go to sleep once freesco is loaded (for those of
you who want a quiet router). 

Most of this information is listed in the freesco manual or on the web
site.
