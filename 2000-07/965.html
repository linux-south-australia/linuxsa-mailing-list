Can anyone recommend "embedded" hardware able to run Linux ?

Hi,

I would like to use Linux in a small, personal project in an "embedded" type
device. Can anyone recommend hardware which may meet the following basic
specs:

1 .. Runs Linux.
2 .. Supports boot from Flash ROM type device, ie no hard disk needed.
Alternately can boot from CD-ROM
3 .. Supports a CD-ROM drive.
4 .. Ethernet support nice, but not mandatory.
5 .. LCD support nice, but not mandatory.

fitting this description ?
