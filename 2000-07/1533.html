Re: FTP Error

Mick Collins wrote:
> 
> Hey everyone. I am having a problem with ftp. I currently have a server
> running DHCPD as a ip dedicater and ipchains. Ok when i try to Ftp to eg.
> ftp.redhat.com
> i get a "Data Socket Error"
> What could this be and if anyone could help i would appriciate it.

FTP uses 2 connections. A control and a data connection. The data
connection I think is initiated by the remote host, so if you are using
IP Masq (aka NAT) then you have to have some special stuff on your NAT
box to handle this. Typically done by loading the ip_masq_ftp.o module
(in /lib/module/<kernel>/ipv4/.
