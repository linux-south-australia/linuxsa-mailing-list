Re: Trace Route

Michael Kratz wrote:

> When I try to trace route out of my network through my linux box it won't go
> past the linux box, does anyone know what port has to be enabled for trace
> routing? and whether it is tcp or udp?

Windows, from memory, uses ICMP echo.  Most Unices (definitely Linux)
uses UDP.  If you want to know which port, man traceroute(8) provides
the details.
