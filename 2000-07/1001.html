Re: can you copy files directly from win98 to linux?

No worries!

Do you know how to 'mount' a MS file system?  I don't know how much you 
know, but I will start from the top.

MS is a 'vfat' file system that must be mounted, and then you can copy files 
to and fro as you wish.  The filesystem difference is transparent, it is 
only the file formats and types of files that matter.

Also, as an added precaution, I mount the MS filesystem as 'read-only', as 
linux treats everything so transparently, it will also delete everything 
transparently without much consideration of filesystem boundaries if you 
issue the wrong command under root.

Email me directly if you need help with anything..
