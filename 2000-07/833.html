Second Class e-citizens

Hi,

I have just had one of those soul-destroying moments. My wife suggested
we lodge our tax return electronically, using the newly available ATO
system 'e-tax'. Sounds great, just log in and follow the promts on the
ATO web page... which include download the ATO e-tax software (20-30
mins) download the help files (20-30 mins) download the private key
encryption yadda yadda yadda, then the killer, system requirements,
including 'operating system Windows 95 or later'.

So, I thought I'd try to use the online suggestions box provided on that
page to gently request a Linux version... typitty tappity type, press
the magic SUBMIT button (but I'm darn well NOT going to submit, no
way!!!) and after the appropriate period of waiting...

An error has occurred which has prevented your feedback from being
posted. Please try again later.

              Error:-2147467259
              Microsoft OLE DB Provider for SQL Server;
SiteFeedback.AddResponse 
              Timeout expired

What a useless waste of time. Good job I didn't actually do the
downloads before reading the instructions. After five years of Linux I
have actually been trained (at long last) to RTFM first.
