Linux has a port of the kernel to its own system call interface, meaning
that you can have an entire Linux system running in user space. It works
completely differently, but think of it as roughly similar in effect to
vmware - you go 'ps' on the host OS and see a single process, switch to
that process and go 'ps' and you see everything you'd expect to in a Unix
environment. Much lower overhead than vmware (and 
architecture-independent, too.)

http://user-mode-linux.sourceforge.net/
