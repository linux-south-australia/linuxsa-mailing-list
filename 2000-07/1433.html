> I am trying to print from SO printing works from other apps like

Just load SO as root and run the print config tool.  You can run the
tool from the command line as root but you'll have to find it and I
can't remember the name.  If you just configure your generic printer as
postscript and tell it that the print command is 'lpr blah' or whatever
print command you use you should be right.
