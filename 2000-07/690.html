FINAL InstallFest things

OK, last minute things that I returned to work late at night pre-installfest
for... please read them all and help where possible...

Posters -- can anyone who has any spare posters please bring them -- we will
use them for signage, etc.

Power -- Bring ALL your extension cords and powerboards, preferably labelled
(labeling will be available at hte InstallFest, but pre-labelled is better) 

Network gear -- Bring any spare Cat5, and if you happen to have any spare
100BaseT hubs lying around, bring them also :) ... Again, label everything
-- we can label on hte day, but pre-labelling is better...

Software -- if you have any software that you think may be interesting
(including commercial games that we can install on demo machines for the
day), please do bring them.

YOURSELF -- without you, none of these things are going to get here :)
Seriously, this InstallFest is only going to be as good as the people who
make it happen -- and that's all the installers... Be there no later than
9:30 if possible (we start setup at 7am, and you're ALL welcome :) ... Be
friendly, useful, co-operative, friendly, helpful and friendly, and we will
have at least 100 new Linux users by the end of the day (there's 102
registered installees).

7AM -- yes, we have a 7am start for enthusiastic organisers.. all are
welcome, and the earlier we get everyone, the better prepared we will be.
