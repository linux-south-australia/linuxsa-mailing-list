Re: syntax to find process id for username

"Freeman, Peter (ERHS)" wrote:

> I'm wanting to know the easy way to locate the pid for a user's session.
> I know this would involve the grep command but am unsure of the exact
> syntax.
> For arguments sake I want to know the pid of username fred.

Not exactly sure what you want here.  PID is the process ID, which is
owned by a user.  "ps -fu fred" will show you all processes running
under fred's _UID_.

If you want fred's userid (UID), then egrep '^fred:' /etc/passwd
will return the matching record from /etc/passwd; fred's uid is
the third field, or, using awk:

awk -F: '$1 == "fred" {print $3}' /etc/passwd
