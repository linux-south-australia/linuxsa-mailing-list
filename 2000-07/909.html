I have a document called "The Exchange Server Replacement HOWTO" which details
how you can set up an IMAP/LDAP/etc-based server which does the same job as the
mail part of Exchange Server - but I haven't seen anywhere the calendar and
schedule sharing part replaced succesfully.  I've no idea how it works with the
real Exchange Server.  I can't remember where it's from but I have the document
in postscript format, I can email it you if you like.

There are several open-source web-based and database-backended
calendar/scheduling packages available for linux which may be worth a look by
now ;)  Check http://www.freshmeat.net/.

So - in a nutshell - the Exchange Server Replacement HOWTO can help you replace
the mail server part of exchange, with a central address book etc etc, but the
calendaring bit is god knows how easy/hard.

When you find out, please let us know :))  I know for a fact there'd be more
than one person on this list interested in your research.
