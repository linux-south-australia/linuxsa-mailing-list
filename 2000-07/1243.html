Re: Trace Route

At 06:57 PM 7/22/00 -0600, Michael Kratz wrote:
>When I try to trace route out of my network through my linux box it won't go 
>past the linux box, does anyone know what port has to be enabled for trace 
>routing? and whether it is tcp or udp?

Windows traceroute uses ICMP echo requests with the TTL in the IP header
changing.

UNIX/Linux traceroute uses a UDP echo request ... 

You need to figure out which one you are using first.

Second problem is that traceroute through the internet does not work all
that well these days as lots of people filter.
