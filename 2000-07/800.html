Re: command to list users

Michael Kratz wrote:
> 
> sorry I didn't word my question properly,
> 
> What is the command to list accounts on the machine, not those that are
> logged on.

depends on what sort of directory services you are using.

If it's a default RH install, it's probably just using /etc/passwd and
/etc/group.

try 'less /etc/passwd'

or if using OpenLDAP, use:

'ldapsearch -b [baseDN] 'uid=*'' Where baseDN is the Base DN you would
have supplied if you set it up.

If you are using nis, ask someone else, I haven't used NIS for ages...
;-)
