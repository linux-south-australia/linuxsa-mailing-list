Other Modes:
I meant using other network protocols such as:
IPX/SPX, (Novell)
NETBEUI, (Microsoft, IBM, incorrectly called NETBIOS)
TCPBEUI, (Microsoft, IBM, incorrectly called NETBIOS over TCP/IP)
SNA,
(IBM, gosh can't even remember what this stands for, ultra-reliable,
hi-performance, mainframe protocol)

Protocol can be likened to the language used by the actual network
packets.

lpd is actually a printer daemon that uses the TCP/IP protocol.
NETBIOS is an Application Programming Interface, (API), that is usually
pushed around a network using the NETBEUI protocol.

Note NT supports lpd printing but Windoze 9x does not support, but have
a look at:
http://www.uk.axis.com/techsup/prtsrv/printmon/

Hope this helps. Errors & ommisions accepted.
