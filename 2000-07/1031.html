> and I can't ftp...

Make sure you have wu-ftpd installed (use the command 'rpm -qa |grep
ftpd')

> 
>         When I created the new user (ftp user) I configure the root directory as
> the default directory.

Why do this? There will already be a user ftp up for anonymous FTP and
this is restricted to the /home/ftp directory (this is good, because you
don't want just anybody having access to your whole hard disk). If you
want specific people to use FTP on your machine, just create a user for
them using the default home directory and they can then access
everything (assuming the permissions allow this).

>
