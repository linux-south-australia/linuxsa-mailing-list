HI Chris

Depends on what you are trying to do. i.e. what programs and what type of
data and over what media (internet ?).

Using ssh you can create a simple encrypted tunnel using the 127.0.0.1
interfaces at each end. You just bind ssh to a port at each end.

If you need real ip connectivity you could use IPSEC, but this is hard to
configure.

hope this helps
