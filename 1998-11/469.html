Re: SB AWE64 Configuration Problems (newbie)

Hi duder, 

With the new(er) Redhat kernels (assuming you use Redhat! :-) you don't
need to compile the sound support into the kernel! (And there was much joy!)

You just need to load the module...I love modularity...

To achieve this, first make sure you have the latest kernel - 2.0.36-7 at time
of writing. (This, BTW, is always a good idea...can solve a lot of probs.)
 
Next, (and this is the nice part) run 'sndconfig.' It will DETECT!!! yes
DETECT!! your card, and it's all downhill from there. (I have an AWE32 with
wave blaster daughter board (aren't they great?) as well.)

Then just pick your irq/dma/port numbers etc etc, and off you go! It should
play you a sample.

NOTE: if it says "An error occured running modprobe" (Like, thanks for ALL the
detail!!), the you probably have the kernel from RH5.1, which happened to have
the sound card module missing, accidently I assume.

So there you go!! that easy (And you should hear Quake...)

Cheers!
