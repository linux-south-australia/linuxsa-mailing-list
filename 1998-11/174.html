Re: Getting connection stats from ppp

> 
> Hi all
> I use pppd (via ifup ppp0) to connect to my ISP. I'd like it to log to a
> seperate file when connections are made, when they are lost, and also, the
> amount of data transmitted and received for each session.
> What's the best way to go about accomplishing this?

I would hook in some ipfwadm rules.  Where whenever a connection is made 
or dropped the amount of data transferred is kept.  Using the glib library
and perl/php you should be able make some reasonable looking web pages plotting
your ISP usage.

ipfw isn't needed for just classic firewalling, but is very useful for local
security and accounting in a normal network or ISP environment.
