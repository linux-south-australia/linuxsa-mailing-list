I don't know about RH 5.1 (I run all slackware servers), but as far as
RADIUS is concerned I don't think it matters.

I've had varied experiences with RADIUS auth (using Merit and Livingston
dists)
when I was beta testing the Easy Server II (a terminal server) for Stallion
last year.

I found Merit to be the most complete package, ie: in docs, installation, etc.
However, one of my techs is currently testing with the Cistron distribution
of RADIUS and he has found that to be quite functional too. 

I don't know if you have a particular distribution downloaded already
or if you are just starting to look into it, so I have just included 
all RADIUS links have (not dist-specific):

http://www.iphil.net/~map/radius/
http://www.n2h2.com/radius/
ftp://ftp.cityline.net/pub/radiusclient/
http://nis.nsf.net/aaa/documentation.html  (Merit ver)
ftp://ftp.future.net/RADIUS

Sorry if any of these are completely out of date, I didn't have
time to check them before copying them from my bookmarks.
