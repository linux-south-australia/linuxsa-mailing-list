hmmm, shouldn't do - try getting the latest S3V server.

> I tried using the SVGA server - it uses a 'generic' driver and does
> 320x240 only - should the SVGA server drive a Virge card? How do I tell it
> to do so?

You can edit the /etc/X11/XF86config file and change the settings
manually. Or, you can use the command 'xf86config' as root to set
the video card to standard VGA and then select all the resolutions/
depths that you want to do. Make sure you back up the file mentioned
above before running xf86config though, as it will overwrite it 
otherwise and that is not necessarily a good thing if you make an
error in the configuration. :-)
