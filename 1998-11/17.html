Re: larger drive needed (two drives?)

Andreja Zivkovic wrote:

> >  Is there a way to create a virtual parti-
> >  tion... i.e. spread over several drives,
> >  so that - e.g.- 3 (or 4)  250 Mb drives
> >  could "look like" a 1 Gb drive...
>
> This is possible using RAID (i'm pretty sure that's what it's called). I
> have no idea how to do it though :)
>
> You will need to enable it when you compile your kernel, but this should be
> no problem anyway. I'm pretty sure there's a RAID-HOWTO (or a mini), so try
> that.

There is also a way to do it with standard hard disks and no raid unit. The
option is
CONFIG_BLK_DEV_MD in the kernel  (or Block Devices->Multiple devices )
There is info about it in the Documentaion directory of teh kernel - md.txt
or in one of the HOWTO's
