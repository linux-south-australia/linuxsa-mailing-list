Re: PPP Server

Hi Sam

I'm not 100% sure what your problem is, but a few months ago I also
became frustrated with setting up my PPP Server for friends.  However,
it's all working now *phew*

In /etc/passwd, you will find that some users have a /bin/bash
account, others have a /bin/tcsh and yet others still have a: -

/usr/local/sbin/ppp-start

account.

Additionally, I had to muck around with the actual
/etc/mgetty+sendfax/mgetty.config file to ensure that it wasn't
allowing fax transmissions.  I hope this helps you out... My next
challenge is to enable those persistent Win95/98 users an automatic
login.  We'll just have to see how we go...

Have fun

Kirsty
