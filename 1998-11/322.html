> Would someone be so kind to point me in the right directions to find
> info/howto so that I can access my Win98/fat32 drives from linux.
> I read somewhere that this can be done.

Kernels from 2.0.35 onwards have fat32 support.  You can get the latest
kernel (2.0.36) at ftp://ftp.kernel.org.
You'll need to untar the kernel into a directory somewhere (preferably
/usr/src).  Then configure the kernel (read the README file).

In the Filesystems options, select Y or M to DOS FAT fs support, select
Y or M to VFAT support, select Y or M on Codepage 437, and select Y or M
to NLS ISO 8859-1.  

Compile and install the kernel, reboot and you should be able to mount
your fat partition with something like:
mount -t vfat /dev/hda1 /mnt/hda1
assuming you have created a mount point.

There's some info at
http://www-plateau.cs.berkeley.edu/people/chaffee/fat32.html but it may
be a little outdated now.

If you get stuck feel free to email me direct.
