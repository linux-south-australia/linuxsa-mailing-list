Re: Configuring X Windows for S3 Virge GX2 AGP

On Thu, 5 Nov 1998, Tim Jones wrote:

> Has anybody been able to configure X Windows to run successfully on an
> S3 Virge GX2 AGP video card?

The S3 X server doesn't support the GX2. There is some support for the GX2
in the latest SVGA XFree86 X server. It may be a bit tricky to install.
I think I had to run "xf86config" to get a working config file. The next
release should have much better support for the GX2. The current version
is extremely slow, but it is better than no support at all.
