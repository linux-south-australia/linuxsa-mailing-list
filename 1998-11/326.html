Re: read fat32 partitions

> 
> Would someone be so kind to point me in the right directions to find
> info/howto so that I can access my Win98/fat32 drives from linux.
> I read somewhere that this can be done.

Just download the latest 2.0 kernel from ftp.kernels.org. If you can't
stomach a recompile, I think RedHat will have RPM'ed it by now.

Once that's installed, make a directory where you want your HD to be, ie
/mnt/fat32. Then find out the device it's on. It usually goes something
like this:

/dev/hda1 = c:\
/dev/hda5 = d:\
/dev/hda6 = e:\
etc

hda1 is your primary partition, and hda5 and 6 and your logical partitions
which are -inside- an extended partition (which is usually hda2). 

Once you've done that, add the following line to your /etc/fstab:

device directory vfat defaults 0 0

As an example, mine is:

/dev/hda1     /mnt/fat32      vfat   defaults      0 0
