Re: Damn Puters!

Under RH you can get it done through the normal configuration of 
the ppp0 interface through netcfg.

In /etc/sysconfig/network-scripts/ifcfg-ppp0 there wil be entries
that include

PERSIST=yes
ONBOOT=yes

Make sure these are set, then '/sbin/ifup ppp0' and you are done.

This is true for RH5.1, not sure about 5.2.
