Re: how do i remove file???

On Mon, 9 Nov 1998, Michael Schwarz wrote:

> On my 5.1 redhat quake 2 server I have a file that has filled up my root
> partition.  Is is called kcore and I cannot change permissions, change
> ownership or delete it in any way when logged in as root, including going to
> single user mode and trying to delete it.
> Any help would be appreciated as the system is starting to do some screwy
> things.....

None of the files under /proc take up any space on your disk. "kcore" is
basically a copy of your memory moulded into a special form for use with a
debugger.

If you ever find a real file which you can't delete it may have the
immutable bit set. You can remove the bit with "chattr". This has been
known to happen after not shutting down properly, but it is very rare.
