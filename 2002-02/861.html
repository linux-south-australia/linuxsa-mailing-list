Re: Auto start squid with parameter

Sia wrote:
> Anybody know how to auto start squid (Red Hat 7.2) with parameter 
> during OS bootup e.g. squid -N -D -1.

The preferred way would be to edit the init.d entry.  On a Red Hat 6.2 
system, there is a variable "SQUID_OPTS" in /etc/rc.d/init.d/squid -- 
you could just add the desired options there.  Of course, you'll need to 
"chkconfig squid on", as well.

Alternatively, you can add the full squid command line to your local 
startup file (something like /etc/rc.d/rc.local).
