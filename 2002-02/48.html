Re: WebCams

On Fri, 1 Feb 2002 12:46, Matthew.Cook@tekelek.com.au wrote:
> Howdy !
>
> Does anyone have any experience with setting up a webcam under linux
> ?   I am thinking of grabbing one of the cheap webcams (ie $50
> special) probably USB and connecting it to the box.   I would like to
> start off capturing the frames and storing them as JPEG's and
> updating a webpage with a timed meta-tag, possibly in the future
> trying real time streaming.   Has anyone done this ?  If so could I
> pinch a few pointers ?
>
> Matthew

Yep lots of experience! Make sure you know the web cam is supported, 
there are a couple of places on the net you can find this sort of info 
out.

There are heaps of Linux webcam streaming programs out there, some 
quite powerful, at least for the task you seem to want them for. There 
are motion detectors (http://motion.technolust.cx/), streamers 
(http://www.ee.up.ac.za/~justin/bttv/) and many more.

Any reasonably cheap webcam will have been around for a while, and 
should have some decent info about it, though if it is new you will be 
taking a lot more chances.

The webcam I have most experience with is the Philips USB webcam, I 
wrote a frame grabber module for a PCV680K as part of a robotics 
project. The Video4Linux API is not bad, it does have some annoying 
features but these should be ironed out with V4L2 if it ever becomes 
the standard. A good example of code for Video4Linux devices can be 
found in CamStream (http://www.smcc.demon.nl/camstream/).

Linux works very well with webcams (those that are supported anyway). I 
had two USB webcams (352x288 at 30 fps) running on a Linux box 
simultaneously, without dropping a frame for at least 8 hours.

good luck
