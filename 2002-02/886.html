RE: Email to Fax Gateway

Hi.

IMO, yes HylaFAX is the most robust fax server on linux, and probably more
robust than the commercial offerings on NT.

No idea what problems you expect with MIMEDefang, so I cannot comment.

HylaFAX modems can be configured to be send only, send and receive, allow
data calls in and out as well as send and receive, or any combination of
the above.

Earlier versions of HylaFAX had some local security exploits. I have not
heard of any security exploits with the current version.

Head over to www.hylafax.org and have a look at the faq. HylaFAX takes a
bit of an effort to get your mind around what is going on, but it rewards
the effort in spades! Our setup consists of a network of 5 HylaFAX servers
around the country connected to a big name ERP package that spits out
nearly a thousand faxes a day. There are cost savings in being able to use
existing linux boxes and no license fees, and we have also picked up lower
comms costs by routing faxes to the closest fax server thereby eliminating
inter-capital std fax calls. The prices offered by the NT fax server
brigade on this sort of setup is scary!

Regards,

Michael
