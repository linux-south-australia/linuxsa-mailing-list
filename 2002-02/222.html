Hi Ben,

Don't know if this is too late or not.....I'm a bit behind in my email.

Last week I decided to move grub to my mbr. A quick 'man grub' told me
to type
'info grub', and lo and behold....the full manual!  :-)

You need to edit the /boot/grub/grub.conf file.

Clues on how to do this are in the Naming Convention section: 

http://www.gnu.org/manual/grub-0.90/html_chapter/grub_2.html#SEC6

..this will tell what partitions are called what. 

The 'Installing GRUB natively' section:
http://www.gnu.org/manual/grub-0.90/html_chapter/grub_3.html#SEC9

..will tell you how to set the root partition and use the setup command
to write
to the mbr from the grub shell.

It surprised me how easy it was!
