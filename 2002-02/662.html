To make sure the rpm can satisfy all the dependancies in it it would be
better to use alien to convert the equivalent debian (.deb) package to
an .rpm, although rpm2tgz will work too, it probably wont have the
correct specs for your rpmdb.

I use alien a lot and it works flawlessly if you get all the correct
parts to it. Use http://rpmfind.net to locate the alien rpm's.


Another cool tool is apt-rpm from http://apt-rpm.tuxfamily.org which
works like apt-get for debian but only for redhat. Mandrake has urpmi
which does the same thing. I don't use mandrake so you should refer to
its man page for details.

Hope these tips help.
