Re: FreeBSD 4.2 Networking/ Masquarding

In the immortal words of "Mick Collins" <mickwell@bigpond.com>...
> Ive finally updated kinda hehe to FreeBSD 4.2. All I need this machine
> to do is have bind, squid and dhcpd running pretty much.

ok, bind 8 is part of the base system anyhow, so that's relatively
simple to set up, in fact I believe it is already configured for caching
DNS server.

Both squid and dhcp are available in the ports collection
/usr/ports/www/squid24
/usr/ports/net/isc-dhcp3

both come with config file examples
/usr/local/etc/dhcpd.conf.sample
/usr/local/etc/squid/squid.conf

Although I would suggest upgrading to FreeBSD 4.5 instead of 4.2 but
that's just a suggestion.
