Re: PDF on the 'fly'

> I have recieved a job where I need to convert ANY image (JPEG, FH10,
> PSD etc) to a PDF file from a form submission.
> It will be hosted on a remote server that is currently running a
> standard UNIX virtual hosting system.  It runs PHP, Perl, etc....
> 
> Does anyone know hoto do this? or has anyone done it?

pdflib and php is one way to do it, but it's not common to find this
compiled into ISP servers with PHP. PHP does have most of the PDF
creation functions available if you do the right ./configure option.

If you want to do 'any' image format, you'll probably need to run
uploaded files through something like imagemagick first.
