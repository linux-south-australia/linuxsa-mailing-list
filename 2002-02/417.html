According to freshmeat, PGSQL uses the BSD license and MySQL uses the GPL. In
a usage context there's little difference.

As far as I can remember, the BSD license allows you to nick the code as long
as you keep the copyright notice, and the GPL allows you to nick the code as
long as you keep the resulting distributed product under the GPL.
