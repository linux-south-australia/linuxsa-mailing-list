Re: any downside to USB keyboards?

>Is there any other non-obvious reason why I might not choose a USB
>keyboard?  Conversely, is there any reason why a USB keyboard might be
>preferred over a PS/2 keyboard (perhaps for the builtin USB hubs if
>they're good)?

KVM switches that support USB keyboards are rare at this point in time, and
you might want to check that they're compatable with DOS boot disks for
things like BIOS upgrades. If you have a favourite old dos application that
looks for keyboard scan codes, it *might* be affected.
