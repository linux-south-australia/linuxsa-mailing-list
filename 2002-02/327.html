> > I wanted to restrict Telnet access to this server.
> > But, I need only administrators to do a telnet to this
> > server only.
> > For my other Servers (Proxy and Socks server) I used
> > hosts.allow and hosts.deny files to restrict Telnet
> > and FTP access. 
> > My hosts.deny file is like
> > ALL: ALL
> > 
> > This would deny access to all the clinets 
> > and my hosts.allow files is like
> > ALL: localhost
> > ALL: 172.27.10.11,172.27.10.20
> > 
> > This would allow only those 2 machines to gain all
> > access.
> > IS there any way that I could Restrict Telnet Access
> > and allow only FTP?

> 
> There is.  Replace the second line in hosts.allow with:
> 
> in.tftpd: 172.27.10.11,172.27.10.20
> 
> This will allow all services from localhost, but only ftp from those two
> IPs.
>
