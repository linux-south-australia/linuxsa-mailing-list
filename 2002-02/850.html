Re: Flash drives

CompactFlash cards actually emulate an IDE disk.  You can get adapters which 
allow you to use it as exactly that.

This is probably the easiest as it requires basically zero drivers.

eg:

http://www.flashmemory.com.au/shop/shopdisplayproducts.asp?id=3&cat=FlashMemory+Disk+Drives
