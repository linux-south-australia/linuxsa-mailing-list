RE: Proxy_auth

We run Proxy Auth and if you open up a new window within the current browser
( File - New - Window ) to get to another web site it does NOT ask you for
username/password again.

However if you fire up a new session of IE even if you already have one open
then it will ask you for username and password again.

We have a mix of IE5, 5.5 and 6 and it is the same for all of them.
