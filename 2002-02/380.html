It sounds like a failed dependency check. There's no need to continually
reinstall RedHat, as it's the ancillary package that seems to be causing
grief.

Try the following:

# wget http://wherever.com/bpalogin.rpm
# rpm -Uvvh bpalogin.rpm

That's putting rpm in "very verbose" mode, resulting in a whole heap of
fairly boring information being pushed out onto the screen. However, it
will ensure that you are getting all the information that you could
possibly want to ensure that the cause of the problem is found.

Can you run that command (as root, as indicated by the '#') and send the
output (if you can't make sense of it yourself)?

> I can't help but think that I'm sniffing around it but there's probably
> something obvious I can't see for looking.
