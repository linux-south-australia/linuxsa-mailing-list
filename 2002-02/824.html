Re: Proxy_auth

On Sun, 2002-02-24 at 13:41, Mick Collins wrote:

> I finally got ncsa_auth running with squid
> when i put the username and password in it works
> but everytime I open another window its asks again.

That's odd -- proxy authentication shouldn't have to happen again and
again.

> The client machines are running Windows 98 SE
> Internet Explorer 6

My immediate feeling is that this is a problem with IE6, but I cannot be
sure. It sounds like you're opening another copy of IE that's got its
environment variables stored some place else.

I assume that you don't have to logon again and again if you're opening
up different sites in the one window?

Have you tried this with another browser? Proxy authentication with
Squid definitely *should not* ask for a password on a per-window basis. 

The fact that you're opening a new window is somewhat irrelevant, it's a
"deeper" issue that's causing you grief.
