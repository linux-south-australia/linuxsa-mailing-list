Re: how to stop "AUTH" request from pop3

> every time I connect to my pop3 server, in netstat (on the server)
> it says that the machine is attempting to open a connection with my
> machine on the AUTH port

It sounds like you might be invoking pop3 via tcpd.  Perhaps you have
a line a little like this in your /etc/inetd.conf file:

  pop-3 stream tcp nowait root /usr/sbin/tcpd /usr/lib/ipop3d

If so, you could change it so it invokes pop3d directly:

  pop-3 stream tcp nowait root /usr/lib/ipop3d ipop3d
