Re: Video Conferencing with USB WebCams

On 2002.02.07 13:10 Daryl Pilkington wrote:

> What video-conferencing software for USB WebCams is available for
> Linux?
> Anything that will connect to M$ NetMeeting?

I use gnomemeeting, which works well.  You can get better
results over a modem link if you install a optional MS-provided
GSM codec into NetMeeting.  You can get excellent results if
you hack with openh323 to run a G.723.1 codec (this has
software patent issues attached).

gnomemeeting will apparently work with any V4L camera.
I use a Philips Vesta Pro USB web cam.  It's probably
the best of the USB cameras.

If you have not already purchased a webcam then look
at FireWire webcams too.  USB doesn't have enough bandwidth
for full-rate large-image video conferencing, so the
cameras have to compress to get the image down the USB,
which in turn causes the video codec to underperform.
Propbably not a problem for a modem-based Internet
connection.

An alternative if you have a video camera is a real video
capture card.  These are very well supported.
