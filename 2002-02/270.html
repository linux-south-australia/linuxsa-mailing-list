Re: Internet connection

On Sat, 9 Feb 2002, Robert Thomas wrote:
> I am trying to find out if there is a relatively simple (& secure) way of
> connecting to a file server/internet gateway machine via the internet so
> that Windex machines can access files etc from that machine. Can anyone
> point me in the direction of some information.

CIPE, IPSec, PPTP.

Windows 2000 supports IPSec out of the box. Windows 98 supports PPTP out
of the box. Windows 95 needs diaulup networking, patches for that and
patches for those patches, and teh correct ammount of rebooting.

Most versions of pppd and the linux kernel need to be patched to support
PPTP using MS-CHAP2 'secure' authentication and an encrypted tunnel via
MPPE.

Most recent kernels have CIPE and/or IPSec (FreeSwan) compiled in 'out of
the box'.

CIPE does have a Windows componet, but I cannot comment on that as I have 
not used it yet.

I don't know how well redirecting smb traffic (tcp and udp) accross an SSH
connection would work, or even if it is a recommend thing to do.
