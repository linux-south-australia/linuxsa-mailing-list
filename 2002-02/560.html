Re: PDF on the 'fly'

On Fri, 15 Feb 2002, kane wrote:

> HI yowl,
> I have recieved a job where I need to convert ANY image (JPEG, FH10, PSD etc) to a PDF file from a form submission. 
> It will be hosted on a remote server that is currently running a standard UNIX virtual hosting system.  It runs PHP, Perl, etc....  
> 
> Does anyone know hoto do this? or has anyone done it?

You'll need to get the cooperation of the hosting firm.

Look at netpbm (any format-->PBM), cjpeg (JPEG-->PBM)
and GhostScript (PBM-->PDF).

Paste it all together using Perl.

Watch the MIME types carefully, if you've got a range
of users you might need a drop box optionally asking for
the format as some user's MIME types will be misconfigured.
