Shane, put simply, Internet data transmission is all done with IP addresses.
The names are just aliases to the numbers (so that humans can deal with them).
Once your server's IP address is derived, then the data connection is
established. So if the IP is local, then the connection will be local [1].

[1] Extraneous routing/NAT scenarios [2] excluded
[2] It doesn't sound like you would have one of them yet anyway :-)
