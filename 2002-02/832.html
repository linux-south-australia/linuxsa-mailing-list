Troy

> hi all is there a way i can comment out my modem strings so i can send 
> them to the group to check out as every time i send the email containing 
> the strings the connection to the net drops./ cheers troy


I thought I'd already told you about "uuencode"...

(man uuencode starts off)

----

uuencode(1)                                           

NAME
       uuencode - encode a binary file

       uudecode - decode a file created by uuencode

SYNOPSIS
       uuencode [-m] [ file ] name

       uudecode [-o outfile] [ file ]...

----

All you have to do is:

uuencode logfile > logfile.uuencode

(You can call "logfile.uuencode" whatever you want -- I just decided to 
call it that)

That shouldn't bomb your modem.
