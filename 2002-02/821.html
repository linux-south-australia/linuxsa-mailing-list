any downside to USB keyboards?

Hi.

I'm looking at a buying a new keyboard, and have always bought DIN-5 
serial (old AT-style) or PS/2 models.  Now that my box supports USB, a 
USB keyboard seems to be a viable option.

I'm wondering if there's any drawbacks to having a USB keyboard.  For 
instance, is the keyboard supported from boot, or do I have to wait 
until USB support is initialized on my system (usually FreeBSD or Linux) 
before the keyboard can be used?  If I remember correctly, my BIOS has 
an option to enable USB keyboard support, but I'm skeptical whether this 
is merely for the BIOS or for the entire boot process.  If it's not 
available for the first few stages of booting, is this likely to cause 
any problems (e.g. selecting a boot option at the boot loader)?

Is there any other non-obvious reason why I might not choose a USB 
keyboard?  Conversely, is there any reason why a USB keyboard might be 
preferred over a PS/2 keyboard (perhaps for the builtin USB hubs if 
they're good)?

Thanks for any advice.
