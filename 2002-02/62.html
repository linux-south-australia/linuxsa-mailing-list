Hello all,

I have just installed Redhat 7.2 for the first time. I have noticed that =
Sendmail is only listening on localhost. The version of Sendmail is =
8.11.6,

After doing some searching at the Sendmail Website, I found a small =
mention that Redhat has configured Sendmail this way as most users that =
install Redhat don't require Sendmail to be listening for incoming =
connections from outside, which is fair enough I suppose. The =
documentation then goes on to say, to enable sendmail to listen on "all =
interfaces", to comment out the following line in sendmail.mc

DAEMON_OPTIONS(`Port=3Dsmtp,Addr=3D127.0.0.1, Name=3DMTA')

I have done this, and it doesn't make any difference.

What else do I need to do ?
