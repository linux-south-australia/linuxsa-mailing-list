Re: CD burning: read drive not recognised: 3

Are you using devfs (eg mandrake 8.1, and possibly others)? I have
problems with it creating the symlinks from /dev/cdrom to where ever the
cdrom is. Normally I work around it by using /dev/cdrom<number> where
number is the device the cdrom has picked up today :( Do an ls -al
/dev/cdrom and I suspect you'll see its a broken sym-link. If this is
the case rm it, and re-create to the correct device.

Ant
