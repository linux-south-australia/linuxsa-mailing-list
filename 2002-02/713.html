Re: Printer Drivers

Sean.Chivers@OriginEnergy.com.au wrote:
> Now in any x gui, use the 'printtool' application to setup the 
> drivers... been a while, my apologies if this is incorrect. Yes, I think 
> it's printtool.

printtool was recently replaced by printconf.  Which is available will 
depend on which version of Red Hat is being used.

> You will need to stipulate the port of your printer. something like 
> tty01, something like that, but right now I'm not near my PC to have a 
> look.... Anyone?

If it's a local printer, the port name will be something like "/dev/lp0" 
(that being the first parallel/printer port).  If you're running a print 
server, it's probably best to get the information from your network 
administrator, but I'd try using JetDirect if your printer supports it.

FWIW: Mandrake's GUI printer configuration is much nicer -- it is based 
on CUPS rather than LPRng.
