Re: Flash drives

Matthew Moyle-Croft wrote:
> CompactFlash cards actually emulate an IDE disk.  You can get adapters
> which allow you to use it as exactly that.

Apparently the quality of the IDE emulation differs by brand.  IDE
emulation is only one of the modes that CompactFlash can run in and wasn't
that popular in the beginning.

There are also 5V IDE/3.3V CompactFlash kits from TAPR, should you need
something cheaper and know which end of a soldering iron is hot.

Daryl Pilkington wrote:
> What is required for Linux to boot to it as a replacement for a HDD?
> i.e. *no* HDD in the PC.

Flash memory has a limited number of writes, so you'll need a strategy to
limit large numbers of accidental writes (think /var/log).
