Re: Dynamic DNS - more specifically

> I've been told by a workmate that I will
> have problems associated with the delay in DNS propagation of my address
> every time it changes.

Depends what you set the various DNS settings to.  Eg. you could drop the 
refresh time down to as little as 60sec which would mean people are meant to 
refresh records if they're older than that.

On a side note, I was contemplating hosting my server somewhere and getting 
a residential service for home so I actually setup GNUDIP which is a dynamic 
DNS package to run a DynDns service.

Not too hard to do - it has a client for win and linux.

http://206gti.net/gnudip/cgi-bin/gnudip.cgi 

if you want to have a look.  Not guaranteed to work or be a viable service 
mind you.

GnuDip2 itself can be found at: http://gnudip2.sourceforge.net/
