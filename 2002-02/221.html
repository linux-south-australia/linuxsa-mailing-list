Re: mount help

Ben Williams wrote:
> the relevant line from /etc/fstab:
> -------
> //remote_comp/home /mnt/home smbfs user,noauto,username=benw,password=XXXX 0 0
> -------

[...]

> [benw@orion benw]$ mount /mnt/home
> smbmnt must be installed suid root for direct user mounts (500,500)
> smbmnt failed: 1

to set the uid on execution, you can do
chmod u+s $(which smbmnt)

you should know that this can introduce a security hole if there's
anything wrong with smbmnt

rather than do this, i would install and configure sudo, then
sudo mount /mnt/home
or
sudo smbmount //remote_comp/home /mnt/home ...

hope this helps
