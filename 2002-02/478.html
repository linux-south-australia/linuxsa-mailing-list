If by screen dump you mean a screenshot, then the easiest is probably to
use the gimp. Go to Acquire from the main menu.

There are command line tools however like xwd, but the options in the
man page may seem a little daunting and confusing. Image Magick has a
nice utility called import though. Read man import for all the options,
but something like "import -window root root.jpg" is probably what you
want.

Hope this helps.
