Depends on what you want to do and what equipment you have to work with. 
  :-)  Everybody's goals are different, but here are some general things 
you might want to do:

1. Be able to set up your Linux box to do everything your Windows box does.

If I were going to do this, I would start out by making a list of things 
that you like to do in Windows, and then you can ask people how to do 
the same thing in Linux.

I'm not going to promice that you can do that in smoothwall, but I can 
say that I can tell you where to look instead. A Smoothwall setup makes 
a nice firewall, but probably wouldn't make a good Windows 
replacement...but I could be wrong too.

2. Set up a computer that all your other computers can put files onto so 
documents can be shared and your data can be backed up.

3. Learn how to make your own web page

4. Learn about the other ways people talk to each other over the Internet.

I understand where you are coming from, and truth be told, there are a 
lot of commands that I don't know too. Let me tell you what I did, and 
why I would recommend that you do the same.

When I first started out, one of the first goals I had was to be able 
access some files that were on another computer in my office.  There was 
a program in Windows that allowed me to access the files on the server 
if I logged into the machine from my Windows desk.

I wanted to do the same thing in Linux, so I did a lot of research to 
see how it was supposed to work in Linux.  During that process, I 
learned the command ncpmount.  I found a lot of answers using the Google 
search engine, but I was also never shy to ask for help if I couldn't 
find the information I needed.

The same thing will happen to you once you decide what your immediate 
goal is. All you have to do is ask, and someone will probably help you 
get started.
