Peter,

> My next question is probably a "walk the fine line" type of question.

You next question could be about skunks in the zoo...

> If in principle this was to be developed because I have reason to use it
> as part of my overall operations, then is that "tell the time" program an
> open scource ?

That depends on how the programmer writes the program:

 1. If the program modifies a GPL licenced program
  - then it must be released under the GPL itself

 2. If the program modifies a BSD licenced program
  - then it must be released under the BSD licence

If the program links *against* an LGPL library then you can do whatever you
like with the program (include keep it proprietary).

This is a legal question.

You have to answer:

 1. Who actually owns the copyright?
 2. When writing the program did I/someone use an already licenced library
    or whole program to implement it?
