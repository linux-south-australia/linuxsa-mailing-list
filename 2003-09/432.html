Sounds like you may want to add Samba to your Smoothwall Linux box, so 
you can store the text file on the Samba/Smoothwall/Linux server. That 
way you can access it from the program.

You will probably want to move a copy of the text file to the Smoothwall 
box. After you install Samba on the Smoothwall box, you will be able to 
map the Smoothwall box as another hard drive in your work box.  Just 
save the text from your program to the text file.

Now if the work box cannot run the program on the 486, a Linux program 
called DOSEMU can, at least that's what I do. I am able to save the text 
file on the server running Samba and Linux from my Linux work box.

This can be tough to do however, so the very first thing I would do is 
set up a Windows networking client on the 486, and Samba on the 
Smoothwall box. The firewall may be tricky, so ask questions if you need 
help.

Here is a guide I wrote when I learned how to network. You already have 
Linux installed, but I am not sure if it will be as easy to set up Samba 
as it would in Mandrake, but Mandrake may be harder to set up a 
firewall.  Anyway here it is:

http://arcom.com.au/~gossner/curriculum/docs/easysamba/easysambaweb.html

If you run the Samba client from the 486, you will have to set up a 
network drive on that box.
