Re: network drive

We had Red Hat embedded running on an 8MB Flash (it had 128MB RAM though!). It 
was an Altera System-On-Chip box with an ARM core and a pretty big FPGA. Was 
very sweet. I believe my friend got NFS working on there no probs. Was cool 
because you could write out your hardware image using cat :) Something like 
"cat myhardware.bin > /dev/pld0". The box had a soft ethernet core so we had 
to do that in order to bring up eth0 (was SRAM based FPGA, so all was lost 
when power went off).
