Re: Benefits of Samba vs Win2K Server

I think one of the biggest benefits is that no one else can come along 
and install some #$^%$ piece of software written with something like 
Access and send the stability of the whole server down the  tubes.  I 
have  NT4 servers that are running just fine as long as no one loads 
anything else on them.
When they see a Samba box  they don't know what to do, which is great.

One problem I do have with Samba however is setting user rights on 
Domain users. It would be nice to be able to add the domain/user group 
to the local admin group on the local PC. Not sure if this is being 
address in Ver 3
