James, hint: If you want to find out whether a port exists for a piece of software then a good
             'first' option would be to use whereis(1).

             e.g. shell>whereis antiword
		  antiword: /usr/ports/textproc/antiword

             This will only work if the variable "PORTNAME" in the Makefile of the port is the
             same argument you give to whereis:

             e.g. shell>grep -i portname /usr/ports/textproc/antiword/Makefile
                  PORTNAME=       antiword

             Option 2 could be to grep through /usr/ports/INDEX [I use this option if I only
             know a portion of the name of the software].

             e.g. shell>whereis vmware
                  vmware:

             e.g shell>grep -i vmware INDEX | awk -F\| '{print $2}'

                 /usr/ports/emulators/linux-vmware-toolbox2
                 /usr/ports/emulators/linux-vmware-toolbox4

            Makefile.
