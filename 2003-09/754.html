Re: Windows 2K Active Directory + PDC

/me answers self

> However, is it possible to get a Windows 2K Active Directory Server (with
> no Windows NT 4.0 servers on the network or equivalents) to act as an NT
> 4.0 PDC? I've heard of a "PDC Emulator" but I'm not quite sure whether
> that's only for a BDC or whether it can be a PDC too.

Yes, apparently it's the default behaviour.

Anyway, I mentioned my customised version of pam_smb_auth.so. If one checks
out:

 cvs -d :ext:anonymous@cvs.sourceforege.net:/cvsroot/pamsmb checkout \
 pam_smb -rDSL

..and then updates it to:

 cvs update -rDSL_001

..you'll get it.

Basically my modifications enable one to make "password and shadow" entries
on the fly. It's very, very, very beta at the moment, though.
