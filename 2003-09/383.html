I can't help you with e-Smith.  But I can with the
modem, as that's what my laptop uses.  See

  http://www.linuxant.com/drivers/

I use a RPM-based distribution.  I download the source RPM
I can then immediately rebuild the RPM after a kernel
upgrade and get my modem working again:

# rpmbuild --rebuild hsflinmodem...
# rpm -ivh /usr/src/redhat/RPMS/i386/hsflinmodem......rpm
# hsfconfig
Should we attempt to automatically configure your hardware? [yes]
Please enter your country name [AUSTRALIA]:
Where is the linux source build directory that matches your running
kernel?
[/lib/modules/2.4.20-20.9/build]
# vi /etc/modules.conf
  alias /dev/ttySHSF* hsfserial
  alias char-major-240 hsfserial
  alias /dev/ttyCUA* hsfserial
  alias char-major-241 hsfserial
  options hsfserial serialmajor=240 calloutmajor=241
# depmod -a
