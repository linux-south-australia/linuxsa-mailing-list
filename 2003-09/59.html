A frequency between tracks? not so far as I know. at the start of the disc 
there's a table of contents, that's where any CD player will find out how 
many tracks, when they start, how long they go for, etc.
If there actually is a sound between tracks, I sould suspect that it's a 
'null' sound, when no real signal is going through. Maybe a little line 
noise, if anything.

if you're getting the CD as one single track, then you're using the wrong xmms 
input plugin. any CD player or plugin (except for the ones designed to treat 
is as oneentity) or whatever should be able to give you each track on the CD 
individually
