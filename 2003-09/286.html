Re: computerworld article on AUUG conference Sydney

Also at the AUUG conference, Greg Rose pointed out to O'Shaughnessy that
he, back when he was at UNSW, had written code which had made its way
into the UNIX codebase, but he had never assigned copyright nor given a
licence for it to be used.  He asked O'Shaughnessy how SCO might react
if he offered a licence to IBM.

There is a certain ironic symmetry there: The rest of the world doesn't
know what code SCO claims to have found; and SCO don't know what code
Greg Rose owns.
