Re: Wireless and windows xp-home

Not sure what kind of card but:

- I modified the "/etc/pcmcia/wireless.opts" file which seems to set 
things. [look at /etc/pcmcia/wireless to see what it does]
- "iwconfig" is the command you need to modify things.

I had this working on a wierd 7.1/7.2 combo so 7.3 def. should work. 
More recent versions of RH make it even easier.
