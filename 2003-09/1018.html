Re: Broadband vs Cable - survey results

ADSL generally isn't all that hard to setup.

However, I find the provider provided programs make it so much more painful.

Telstra's pppoe software, for example, is absolutely horrible.  Anyone
using windows should at least be using raspppoe (I think that's its
name), as it is much easier to use, and far less temperamental.

However, we're quite happily using Telstra ADSL on rp-pppoe with a Linux
router =)  Since changing over, the link seems much more stable.
