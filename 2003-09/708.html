---
Q: Why does the Red Hat Update Agent no longer work?

A: The Red Hat Update Agent (up2date) requires a valid System Profile on Red
Hat Network and a Software Manager subscription to function properly. To
create a System Profile, run the Red Hat Network Registration Client
(rhn_register) on the system that you wish to register with Red Hat Network.
Every user receives one free subscription to Software Manager. To purchase
additional subscriptions, go to rhn.redhat.com/purchase_info.pxt. 
---

Now, go to:

 * http://www.redhat.com/

..and search for "up2date free" and you'll find other interesting
information.
