I have had this as well. A very helpful person once taught me this part....

There is a file called initrd that is located in /boot called a "ram 
disk." 
Anyway, this "ramdisk" provides Reiserfs support when the initial
boot takes place....

So...how do you make this "ramdisk" ?

In SuSE 8.2, its like so: mkinitrd -k "name_of_kernel" -i 
"name_to_give_ramdisk"

They say that payback is a female dog, so I decided to make a script to 
help
other people compile their kernels, make ramdisks, etc:

http://prdownloads.sourceforge.net/automagic/kernelmagic.tar.gz?download

Unzip the file, go into kernelconfig and run sh Kernelconfig2.8.sh.  It 
is designed
to not pave over an existing kernel or "ramdisk."
