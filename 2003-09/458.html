GNOPPIX

http://www.gnoppix.org/
GNOPPIX is a linux live cd based upon Debian GNU/Linux 3.0 (woody). It can be 
compared to Knoppix but GNOPPIX uses GNOME as desktop environment.
 We're planning to create a GUI installation tool to make installing GNU/Linux 
with a GNOME desktop as easy as possible.
