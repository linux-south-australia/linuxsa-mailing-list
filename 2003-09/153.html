Re: Winbind and Domains

On Wed, 2003-09-03 at 11:28, David Lloyd wrote:

> The obvious part that I am missing is:
> 
> I don't want to have to "join" each machine to the domain manually either
> by using the Windows tool or the smbpasswd method. 

It sounds like you're asking if it's possible to join a domain without
joining a domain. You can't. I'm not sure if winbindd supports
authentication against a logon server (a la Windows 9x). You may be able
to reduce the amount of work by creating a machine account on the PDC
for the domain member (the winbind box). Joining the domain will then
not require an admin password and can therefore be wedged into an
rc.firstboot or something similar (depending on the situation). You'd
want to test that sort of thing before rolling it out though :-)
