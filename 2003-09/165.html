Dump v. tar/cpio; only on pay per view.

Given the recent debate over dump vs. cpio/tar, I've just found (or
been bitten by) one advantage that dump has over the other two;  dump
can be run as a non-root user.  All it needs is read access to the
raw disk devices, which can be achieved by running dump in the
appropriate group, and is handy when you're remote dumping over an
ssh channel that doesn't allow root logins.  Thanks, Linus.  Now I'll
have to resort to some blerky kludge like setuid perl scripts.
