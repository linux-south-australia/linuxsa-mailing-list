Re: skunks in the zoo (was next question) - actually it's about licencing

On Wed, Sep 17, 2003 at 11:50:50AM +0930, David Lloyd wrote:

 > > If in principle this was to be developed because I have reason to use it
 > > as part of my overall operations, then is that "tell the time" program an
 > > open scource ?
 > 
 > That depends on how the programmer writes the program:
 > 
 >  1. If the program modifies a GPL licenced program
 >   - then it must be released under the GPL itself
 >  2. If the program modifies a BSD licenced program
 >   - then it must be released under the BSD licence
 
Part (2) is not true.  It can be released under any other license 
that's consistent with the BSD license.

(e.g., you can take BSD-licensed code, make some changes to it, and
release the resulting conglomeration as GPL code if that's what floats
your boat.  That's happened lots of times before -- e.g., inclusion of
BSD-licensed network code in GPL kernels)
