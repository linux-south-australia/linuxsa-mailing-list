RE: bplogin with e-smith 5.5/6

Steve,

You will need the BPALOGIN even with the latest version of e-smith. 

E-smith will do your normal ADSL / PPPOE connection with a bog standard
install no problems, but grab the RPM to get a cable connection going.

http://www.e-smith.org/contrib/rpm-index/RPM_by_name-B.html

There are two version listed, however there is only a binary package for
the 1.3 version, grab that if you are not happy building from source. 

Local links below for the 1.3 binary RPM and 2.0 source RPM

http://ftp.filearena.net/e-smith/contrib/CharlieBrady/RPMS/i386/bpalogin
-1.3-2.i386.rpm
http://ftp.filearena.net/e-smith/contrib/CharlieBrady/SRPMS/bpalogin-2.0
-1.src.rpm
