Re: Benefits of Samba vs Win2K Server

In summary:


PROS

 * Unix/Linux functionality
 * Configuration is easier due to text based nature
 * Able to run without a keyboard, mouse or monitor
 * Stability
   - Windows can be stable provided noone disturbs its configuration
   - Exchange can destabilise Windows
   - Some reported Windows boxes performance would degrade in time
 * Security
   - Linux/Unix may have less severe security issues than Windows
 * Lack of a GUI interface is a reasonable step against stopping non-IT
   types from breaking the system by using the "easy to use [break?]"
   interfaces
 * The open source is good argument

CONS

 * Unix deployments acrosss a large number of machines can be painful
 * Active Directory can provide a lot of benefit to a large organisation
 * No ACLs and extended attributes can be bad
