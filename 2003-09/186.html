RE: PHP IDE

This could open a can of worms (ie the great editor debate), but my
choice of editor is jEdit.  Nice and quick, no huge memory leaks
(although like any java swing app, requires about 30-50MB of your RAM to
run).  It supports php syntax colouring and indentation, etc.  Has good
code folding support, flexible searching (inc regexp), huge array of
available plugins, and most of the functions can be accessed using quick
keyboard shortcuts (a must for a good editor).  Plus, it's open source.
I could go on, but I won't bore you with the details.  

All you 733t vi users, please don't flame me for this   :)
