Re: two network at network neighbourhood

Essy,

You need to set your samba box (which I presume is also the gateway between
networks) to be a WINS server, and then you need to tell *all* of the hosts
on both networks to use that for WINS. You can do that in the Windows
Networking control panel if you are not using DHCP to issue IP addresses.

The samba documentation on your system should help, as I believe it does
cover this topic: That's how I set it up. (I got it working over 6 different
networks, most of which were connected via modem connections).

Darryl
