Re: Linux FTP Install Server Setup

> I was wondering if someone could point me towards some doco for
> setting up a linux O/S server so that machines I connect to my network
> may do a net install of redhat 7.0.

Don't do an FTP install, as the process falls over if the FTP
server logs you off (as during a long idle period whilst disks
are being formatted).  Do an HTTP install instead.

Basically you go to the web site, copy the network install
image to floppy using rawrite, and reboot from floppy.
Give your IP address/mask/router/DNS (or use DHCP) and
the web site name and the directory of the Red Hat
software.  It then downloads and installs the RPMs.

Note that the AARNet mirror doesn't have the network install
RPMs available yet for RH7.1.  We are in the process of moving the
mirror to the ACT and the machine in Qld doesn't have enough
disk.
