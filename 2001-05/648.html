We're using the first 4 letters of surname, first initial, then 01, 02 etc

ie:  your username would be smita01

if u had a sister called Andrea, she'd be smita02, and so on.

so far out of about 250 users, we only have one "02" in a school 
environment obviously there'd be more possible "02", "03"

We've found it works quite well, and its very easy to explain to users as
well
(its also easier to find someone's username in a list (ie: user admin tools)
