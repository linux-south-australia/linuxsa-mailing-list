Hi,

> over the past few months I have done a few Redhat 6.2 installs and have 
> found that Make and GCC have not been installed thus I am finding it very 
> hard to compile things!!!! now I have two questions:
> 
> 
> 1. how do I install them on a system which does not have them installed?
> 
> 2. what option under the custom install of RH6.2 do I have to check to 
> ensure that they are installed. ie. the bit about the options like

Its a while since I've done a RH 6.2 install, but from memory you its in
the software development section.

If its not installed use the RM program (or gnorpm if you run X) and
install from the RedHad\RPM\i386 directory on the RH CD (or similar
location on network distribution).
