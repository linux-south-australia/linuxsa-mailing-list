On 05 May 2001 09:51:53 +0930, Richard Watkins wrote:
 
> The Win2K server allows me to dial-in remotely...
> authenticate...and access resources...how will I now
> be able to do this once I put in place the firewall

One would assume that you have one modem for Internet access and another
for remote access.

>   ----------
>  | Win2KPro |---
>   ----------    |    ----------     -----------     -------
>                 |---| Win2KSrv |---| RedHat7.1 |---| Modem |---
>   ----------    |    ----------     -----------     -------
>  | Mandrake |---|   * Email
>   ----------    |   * Proxy
>                ETC  * DHCP
>                     * WINS/DNS
>                     * RAS

That can continue to happen -- you just set the default gateway on the
Windows 2000 box to the RedHat 7.1 Firewall. You'll need to setup NAT
for this to work.
