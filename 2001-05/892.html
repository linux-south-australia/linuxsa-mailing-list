RE: Dual boot - Win 2000 and Linux

Win2K boot arrangements are just like WinNT.  I would suggest you use the
Win2K boot manager to boot both Win2K and Linux.  There is a Red Hat HowTo
on using the NT loader to boot Linux.  I had NT booting RH Linux (5.2 I
think) and NT and Win95 on my previous machine.  I use FreeBSD now which
works differently and at present I have BootMagic booting FreeBSD or Win2K
which then allows boot of Win98 too.  Later I hope to get is all on the one
Win2K boot menu but my first try at that did not work.

You can use Lilo too but then you get the Lilo boot menu followed by the
NT/Win2K boot menu. If you get  WinNT/2K to do everything you just get one
menu.  Essentially you install Lilo to the root partition, capture the boot
sector off the Linux partition to a file using Linux and edit WinNT/2Ks
BOOT.INI file to add the this boot sector file to the WinNT/2K boot menu.
The HowTo tells how to set it all up.
