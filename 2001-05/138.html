Re: Slackware Linux & NETjet PCI ISDN Modem

>I'd like to ask for help on an 'issue' I'm having with a NETjet ISDN PCI
ISDN modem
>under Slackware Linux. I have followed all the instructions on the homepage
of
>Traverse Technologies but have not been able to get it running
successfully. Under
>Windows ... no probs.

I think 2.2.16 and better already have the traverse drivers in the kernel,
all you need to do is tick the relevant option in make menuconfig.

If you want to do DOV, that's a different story, the patches to make it
possible weren't accepted into the kernel source. I found with the 2.4
series kernel (with the exception of the redhat 7.1 source) all you need to
do is replace the file isdn_net.c with the traverse-supplied one, and it all
works fine.

Perhaps establish that the card works fine with the default drivers before
getting into the trickier DOV mods.
