Re: Regarding ISO Image

On 06 May 2001 20:23:55 -0700, Chandra Sekar wrote:

> Can anyone help me in Installing RHL using a ISO Image.

An ISO image alone isn't much use when you're trying to get Linux on a
computer. You really need to combine this with some other tool. These
may include:

1) Mounting the image as loopback and NFS exporting it.
2) Burn the ISO to a CD and boot from said CD.
3) Mount the image  as loopback and setup FTP/HTTP and install from
that.

Note that the ISO image is only a snapshot of a filesystem (or the
contents of a drive; whatever :-), it's not much use on it's own.[1]

   - andrew

[1] Note that you could probably install it without the above options,
but that would require a Network connection and another computer.
