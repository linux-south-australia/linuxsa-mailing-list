>Can anyone help me in Installing RHL using a ISO Image.

Well, it is really the same as installing from CD, after you have burned
the ISO image on a CD ...

Additionally, you can actually mount the ISO image on an existing Linux
system (of reasonable vintage) and extract the bits you need to make a
network boot floppy and then install over the network ...

  mount -o loop rhl71.iso /mnt/rh71
  # rummage around in the image if looking for things
  dd if=/mnt/rh71/images/net.img of=/dev/fd0  # check that img file name

And then walk up to a PC with your new boot floppy and boot it and install
the new OS over the network.

Oh, btw, cdrecord is the command you want to copy an IOS to CD. The demon's
spawn OS probably uses a different command :-)
