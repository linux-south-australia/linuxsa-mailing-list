Re: two network at network neighbourhood

If you're using SMB it generally can't cross sub net boundaries without
setting up a WINS server. Samba can act as a Wins server. Off the top of
my head, you set up a machine in one network to be a "WINS" server and
then it should work, however look up the FAQs etc at
http://www.samba.org/ or a book on Samba if you own one for the really
gory details.
