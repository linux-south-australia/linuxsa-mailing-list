Re: IP PortFW

On 10 May 2001 12:22:25 +0930, Darryl Ross wrote:

> Unfortunately, people who are 'internal' can not access the web server by
> using the 'external' address. Is there any work around for this? Due to
> various reasons, I _do_ need to be able to do this.

I've used rinetd to do this. It's a fairly simple program, with a
configuration file like this:

#Live Addr  Port   Int. Addr   Port
 203.x.x.x   80   192.168.x.x   80

http://www.suse.de/~grimmer/rinetd/rinetd.html

Worked for me quite nicely.
