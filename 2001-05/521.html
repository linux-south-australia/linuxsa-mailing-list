The operating system not found message would occur no matter which
operating system was on there. What it is, is when Linux was installed,
no partition was made active. The BIOS then doesn't know which partition
to boot. You can fix this using a standard MS-DOS boot disk containing
fdisk. Just set the correct partition active and reboot.
