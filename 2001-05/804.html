Re: [OT] Script help

bmonten wrote:
> 
> Let me apologise in advance. I know this probably isn't the real theme of the list, but I need help doing a script, and you guys
> (and I mean that in a non-gender-specific way) are bound to know.
> 
> 1) I have a file, called Rpt001.txt. If I "cat -v" the file, line 1 shows as:
> Truck Manifest^M
> How do I clean out the ^M (carriage return, I think) ? I've tried using sed, but don't know how to supply the ^M, so it's the
> control character.
> 
> 2) Same file, I have a line similar to:
> Q EXP  QTY;RIV 201 547;RIV 201 557   ANO RS   ANO
> within the script, I want to convert all semi-colons to newlines. I tried:
> 
> MSG_RAW=`head -5 $FILE | tail -1`
> MSG=`echo $MSG_RAW|sed -e "s/;/\n/g"`
> 
> but the \n is coming out as an 'n'.

sed -e 's/\r\n/\n/' Should change all of the CRLFs to \ns. You could
probably use tr as well (some older dists had a dos2unix command which
did this).

Hope it helps,
