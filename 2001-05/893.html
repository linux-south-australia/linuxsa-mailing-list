Re: Indial Server options [was: Re: ISDN PRI capacity]

Any router that can terminate an L2TP tunnel session can be used as 
the 'home gateway' for VPDN services.

That includes any Cisco router running IP Plus, and a whole pile of 
other completely non-Cisco things (though I've only ever bothered to 
do it on Cisco routers).

What size device you choose depends on your intended total throughput 
- as always. But there is no other particular issue (apart from 
straight-line grunt) to consider - the router only needs one ethernet 
port to do the work.

Cheers,
Simon
