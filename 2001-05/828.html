Re: Sendmail Question

RH 7.1 also ships sendmail configured to only accept connections from
localhost.  Check /etc/mail/sendmail.mc for

DAEMON_OPTIONS(`Port=smtp,Addr=127.0.0.1, Name=MTA')
replace it with
dnl DAEMON_OPTIONS(`Port=smtp,Addr=127.0.0.1, Name=MTA')
run m4, and restart sendmail if you want accept mail from hosts !=
127.0.0.1

.cfg
