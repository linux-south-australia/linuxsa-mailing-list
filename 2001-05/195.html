Re: Red Hat 7.1 Networking

On Mon, 7 May 2001, Jim O'Halloran wrote:

> Hi,
>
> When I installed RH7.1 I selected "medium" firewall protection believing
> that this would allow access to the services on the machine, and I've since
> use lokkit to change the firewalling to "No Protection" but that doesn't
> seem to have helped either...

7.1 changed the default behaviour of sendmail. If you have a look in
/etc/mail/sendmail.mc you will find something like this:

dnl This changes sendmail to only listen on the loopback device 127.0.0.1
dnl and not on any other network devices. Comment this out if you want
dnl to accept email over the network.
DAEMON_OPTIONS(`Port=smtp,Addr=127.0.0.1, Name=MTA')

Comment out/remove that last line, build a sendmail.cf file (m4 <
sendmail.mc > sendmail.cf) and put the sendmail.cf file in the appropriate
place (/etc).

Restart sendmail and all should be well.
