1) USER ADMINISTERED
Boxes expected wto be on network A boot with one DHCP option (say 'a'),
boxes expecting to be on network B boots with another DHCP option. (say 'b')

DHCP server on network A only accepts requests with option 'a'.

DHCP server on network B only accepts requests with option 'b'.

2) BOFH ADMINISTERED
On the central switch, set two MAC address access-lists, one
for the hosts on network A, another for the hosts on network B.

Apply access list a to VLAN A, access list b to VLAN B.
