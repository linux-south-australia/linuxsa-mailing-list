Re: Determining network traffic on an ethernet link

If the nodes are running snmp support (e.g. reasonable unix systems, 
or whatever), *or* you are running them via a switch which supports 
snmp, then its easy to monitor the load level using a snmp-speaking 
monitoring/graphing solution.

Probably the most popular (and by no means the only) answer to that 
is MRTG (http://www.mrtg.org). Its certainly what we run here.
