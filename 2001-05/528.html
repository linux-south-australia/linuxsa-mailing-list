Hmmm, there appears to be some confusion here still.

1. Some BIOSes will refuse to jump to a perfectly valid MBR because there
are no partitions in the partition table marked bootable.

2. LILO and LINUX do not care about bootable flags ...

3. DOS does care, so even if you have a BIOS that does not care, you 
   can still have problems if there are no partitions marked bootable.

4. I don't think that WinNT or Win2K care.
