Re: Linux FTP Install Server Setup

If you run a ftp site, and host a mirror of the release you want them to
install, you can do an FTP install fairly easily.  If the ftp server isn't
anonymous, then the machines will need to know username, password, and
path to the release.  They will also need a net boot disk.
