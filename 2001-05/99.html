> Question 1:
> It sounds like this means that a UPS is now of doubtful value,

No, you haven't wasted any money. I would still be using UPS on critical
(or important) boxes. I haven't had a chance to fully test the xfs
system
on linux, but under irix I have powered off boxes without flinching
(though I would still not recommend this as the norm though!!). When
repowering, they came up ok.

Q: Does LILO work with XFS? 

This depens on where you install LILO. For MBR installation: Yes. For
root
partitions: No, because the XFS superblock goes where LILO would be 
installed. This is to maintain compatibility with the Irix on-disk
format. 

Q: Does GRUB work with XFS? 

No - so far there is no direct support in GRUB to boot from an XFS 
filesystem but you may still boot an mostly XFS
system with a small ext2 /boot partition with all the GRUB stuff 
on it using GRUB. Btw. writing XFS support for GRUB
would be a good way to learn more about XFS if anyone is interested :-).
