Firstly,  a clarification. OpenOffice isn't a word processor. OpenOffice
is a project which has been directly derived from Sun's StarOffice
providing several components, one of which is a Word Processor. Have a
look at http://www.openoffice.org/projects_and_owners.html to see what I
mean.

know prefer LyX or use DocBook to document things. While DocBook can be
generated by using any text editor, I think Dale was asking whether
OpenOffice can be used to generate universally accepted documentation
formats. Correct me if I'm wrong though, Dale.

functionality of the Word Processor component.
