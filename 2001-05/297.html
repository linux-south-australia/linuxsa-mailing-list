> Does anyone know a way of getting squid to check the CGI string in a

Not a direct answer, but:

Squid knows about the following types of ACL elements:

(from the http://www.squid-cache.org/Doc/FAQ/FAQ-10.html)

<begin class="quote">

Squid knows about the following types of ACL elements:
<snip/>
method: HTTP request method (get, post, etc)
</begin>

Which implies that you may be able to setup separate ACL's for anything
which is a POST method...
