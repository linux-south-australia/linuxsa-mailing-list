If you have a linux boot disk like the one at http://www.toms.net/rb/ or the
linuxcare boot cd-rom you can.

mount your root drive it 'mount /dev/hda? /mnt' and if you have a boot
partition that would also need mounting 'mount /dev/hda /mnt/boot' THEN

cd /mnt
chroot . bash
/sbin/lilo -v
exit
umount all the paritions in reverse order.

reboot on you old system :)
