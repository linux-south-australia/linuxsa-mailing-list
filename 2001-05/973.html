Aaah. I've never had a problem with my LPRng setup.

My [global] section of my smb.conf file contains:

    printcap name = /etc/printcap
    printing = LPRNG
    print command = lpr -h -r -P%p %s

And my printers look like:

[Library]
        comment = Library printer
        printer = prn1
        public = no
        writable = no
        printable = yes

And my printcap:

prn1:\
	:sd=/var/spool/prn1:\
	:sh:\
	:rm=jet1:\
	:rp=lp:\
	:af=/var/log/lpd/prn1:

Hope it helps,
