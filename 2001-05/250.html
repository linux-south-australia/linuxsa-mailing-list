Richard Sharpe wrote:
> 
> Is it real XML? Can it be fed into DocBook/XML?

Yes it is real XML.  The DTD is published and should be
easily found by Google.

It can't be fed into DocBook, as DocBook is a differing
DTD to the OpenOffice DTD.

You can transform any XML document in a DTD to another DTD,
losslessly if that is supprted by the DTDs.  See tools like
xalan.  The transformation tools are configured using XML
(of course), the DTD for that is XSSLT.
