Re: $Id in files exported from CVS

On Thu, May 24, 2001 at 03:14:50PM +0930, Andrew Reid wrote:

 > $Id: index.php,v 1.217 2000/12/12 21:16:57 pfalcon Exp $
 > It looks like a good way of telling what version the file you're editing
 > is at. How do I go about pulling this sort of thing into my code?
 
Insert the string $Id$ somewhere in your source file.  CVS will update
it each time you check the file out.

There are other $something$ expansions too;  Check the CVS documentation
for a full list.
