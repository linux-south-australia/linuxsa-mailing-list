It actually means that you can block POST requests (or only allow 
certain users/machines to make them).

I used nc, netscape and a quick CGI script I hacked up, to see how the 
requests work. With a GET method, it GETs the URL, appending the CGI 
query string on the end. These sites will be denied, because the CGI 
string is part of the URL. The POST method, on the other hand looks more 
like:

POST [url] HTTP/[vers]
[other headers]

[CGI string]

As you can see, the CGI string is actually a seperate part of the request.
