Re: Sharing an Internet Connection

> I have two machines - one running windows ME (yuk!) and one running RedHat
> Linux 6.2.
>
> They are quite happily networked together, but there is one more thing I
> need to know how to do.
>
> This is why I need to share my internet connection. Is there any software
> out there (preferably freeware) that can do routing on my WinME(yuk!)
> machine so that my Linux machine can access the internet that way? I would
> probably rather have a proper setup using my Linux machine as a gateway to
> the internet, but I don't have a great amount of knowledge about Linux and
> don't have the money for a modem.

Try winroute from www.winroute.co.uk - its indiscriminate as to what devices
are on the other end of the gateway (as opposed to WinICS, which only likes
microsoft boxen and is illogical to setup) - install it onto the laptop, and
tell it what dial-up connections and network adapters to use.

Or you could install linux onto the laptop and dualboot it  :D

> Also, how would I set up my Linux machine to go through that gateway for
> internet access?

Edit the file /etc/sysconfig/network
You need to add/modify the following line:

GATEWAY=192.168.0.1  (or whatever your laptop's IP is)
