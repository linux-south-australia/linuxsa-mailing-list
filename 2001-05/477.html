Re: GhostScript Help Needed

Andrew Halliday wrote:
> 
> All I want to do is convert .eps to .pdf, but i need ghost-script to be
> installed...so wouldnt you know it ...ghostscript decides not to work,

It worked, it was just trying to open a display so you can view the
result.  RTFM, -dBATCH.

epstopdf --nogs --outfile=fred.ps fred.eps
gs -q -dNOPAUSE -dBATCH -sDEVICE=pdfwrite -sPAPERSIZE=a4 -sOutputFile=fred.pdf -c save pop -f fred.ps

epstopdf will call GhostScript itself, but you're better off calling
it explicitly with the correct paper size.

Finally, if you are producing PDF then you should upgrade
to a later GPL version of GhostScript or use the newer Aladdin-
licensed version if you can.  The ouptut PDF is significantly
better.
