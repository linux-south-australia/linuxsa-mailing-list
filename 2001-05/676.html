Re: Email addresses.

On Fri, 18 May 2001, Glen Turner wrote:

> The rules about the left of the @ are pretty liberal,
> as this simplifies gatewaying into non-SMTP mail systems.
> 
> It also means that the text left of the @ is case
> sensitive 

That's true. And with some SMTP servers you can switch on case sensitivity
for this part of the address if you really want to. However Glen do you
know of an SMTP server installation that actually distinguishes case? I've
never done other than ad-hoc testing but I suspect it is one of those SMTP
things where what is in the RFC is completely subsumed by what is
practiced.

(Reading the connection and message acceptance code for a good-quality MTA
is a good way to learn about rules that aren't written in any RFC.
