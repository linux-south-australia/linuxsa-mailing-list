Re: Samba Log Question

Mike O'Connor wrote:
> 
> Hi All
> 
> The following is a log from my samba server, you can see the PPro server is
> becoming the master browser, but then there are error about some type of
> browersync error.

PPRO is becoming the local master browser not the domain master browser
(unless that's further up the log file). Check your [global] section of
the machine you wish to be the Domain Master Browser (usually the PDC)
and make sure that it has a line like:

domain master = yes

Also check to make sure it has a high os level parameter.

You may also get the browsesync message on a routed network, in which
case you will want to look at the remote browse sync option. Check you
smb.conf man page for details.
