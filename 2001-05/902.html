Re: sharing an internet connection - an added note

Note on Internet Connection Sharing with WinME.

If you have a DHCP server on the linux box, there is a .reg file you need on
the WinME CD (read the readme.txt file).

I have done this and it's extremely simple and works quite well.

Win98SE and RH7.0 combo (also played with WinME as well, both have the same
issue that they want to be a DHCP server and you need to tell it to play
nice and bugger off!)
