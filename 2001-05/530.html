Re: [OT] SQL syntax

On 15 May 2001 13:48:53 +0930, James Mclean wrote:
 
> What is the SQL syntax for MySQL, when inserting a password into the database, 
> that I want encrypted?

I believe something like this:

INSERT INTO table ("username", "password") VALUES ('$username',
password('$password')

Have a look at MySQL's documentation (available online) for more
information on this topic.

The other option is to use PHP's md5 (string string) function to encrypt
it. MySQL's password() thing may use md5, but I'm not sure off the top
of my head.

md5( ) seems to have worked for me in the past. You just have to change
your authentication scripts to work with MD5:

if ($password !== md5 ($db->record['password'])) {
    die ("Password Incorrect") ;
}

Not too hard at all.
