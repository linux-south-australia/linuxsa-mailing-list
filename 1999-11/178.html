Re: Kernel compiling

On Tue, 9 Nov 1999, Jo\ao Campos wrote:

> Regarding my problems on making a stable kernel (I'm trying to trace
> them):
> 
> Can anyone tell me where I can find compiled kernels along with the
> .config configuration file needed to configure it?

If your using redhat you can download precompiled kernels which would have
the .config file used to make them. 

Ie:    kernel-2.2.25-5.rpm         = precompiled kernel
       kernel-2.2.25-5-source.rpm  = source to make the kernel and this 
                                     will have the .config file
