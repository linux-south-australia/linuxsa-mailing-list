Re: Tape operation question

> mt -f /dev/nst0 fsf 1 ; tar -xvf /dev/nst0
> 
> I'm not 100% sure if you need to use nst0 (the non-rewinding device) 
> for a fsf (skip forward). But it can't hurt :)

You really need it. If you have a fast tape, you may also use 'mt -f /dev/nst0 
asf n' to skip to the n-th file on the tape (fsf n skips n files forward). This 
is sometimes easier to handle, esp. when you have stopped a running tar.

btw, can someone explain the difference between nst0, nst0a, nst0l and nst0m ?
