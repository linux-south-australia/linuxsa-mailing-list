The XFree86 server has supported wheel mouses (or
mice if you wish) since about version 3.3.2, but the
latest stable version of KDE (1.1.2) still doesn't. 
According to KDE, wheel mouses are not implemented
in the main building blocks of KDE, called QT.

Having said this, recent versions of GNOME, and
perhaps other desktop environments support them.
KDE is said to support wheel mouses in KDE 2.0,
which is based on a newer version of QT.  Hopefully
KDE
2.0 will be out around December/January, so you'll
either have to wait until KDE 2, or switch desktop
environments.

Some applications already claim to support wheel
mouses such as Applixware.  There is also a very good
guide to using wheel mouses at
<http://www.inria.fr/koala/colas/mouse-wheel-scroll/>;.

Hope this helps.
