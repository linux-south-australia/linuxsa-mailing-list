Make sure that /dev/pts is mounted... From my /etc/fstab (RH6.0)

none            /dev/pts     devpts  gid=5,mode=620   0 0

The other thing to do might be to do an 'strace -f' of the inetd process
and see if it is coming across any permissions anywhere.  Understanding
strace output is an artform, but it is well worth learning.
