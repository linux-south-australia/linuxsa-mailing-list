Re: Esound aware sound players

Note:
I use dma on all my harddrives, it's great. BUT I had a 10Gb Seagate
that, if you enable DMA, promptly trashes itself if you start writing to
it.
To this day I've never figured out why.
It's not a linux problem, as enabling DMA in windows trashes the drive
too.
So be warned that while 95% of people get enjoyment out of using DMA,
there's a tiny chance it could cause heartache.

I reccommend remounting your drive read-only, and then enabling DMA.
Hammer the drive a bit, and watch out for errors turning up in reads.
This way if there is a problem then you'll find it without trashing
anything.
If it all seems well, then remount it read-write ;)

Toby
