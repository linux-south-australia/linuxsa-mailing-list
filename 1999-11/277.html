In the HTTP / 1.0 specs there are is a optional client sent message
"Host: <some text>"

If you use a modern browser (i.e.  IE 4+ or NS 3+ ) it sends across a
basic header
similar to:

Get / HTTP/1.0
Host: www.microsoft.com.au

The server looks for any configured Virtual servers matching that name.
and then offers up the appropriate page.

Of course if you have an early browser which doens't send the Host: line
across,
you'll get the standard "catchall" site displayed.
