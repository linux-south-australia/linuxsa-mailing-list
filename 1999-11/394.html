Re: UID problems

Jeffrey Borkent wrote:

> I have a Mandrake 6.1 linux machine running as a file server to a Macintosh
> ( using Dave-Samba ) and Nt (Samba ) environment. I am having problems with
> the logins of users. The linux computer lets users login but does not
> assign a username to the UID. When you use the whoami command you get
> "cannot find username for UID" error mesage.

Sounds like you've changed PAM (so that the users can login
in) but not NSS (so that the user's name can be resolved from
the UID).  Are you authenticating against NIS+, RADIUS or
LDAP?

If you do get stuck there's a rumour that the University has
a Systems and Networks team[1] that can help computing officers
solve problems like this :-)
