Re: Tape operation question

Robert Wuttke wrote:

 > As a rule I always use the non-rewinding version and manually tell the tape
 > to rewind itself when all required ops have been finished.

Indeed.  Incidentally, it helps lots if you "setenv TAPE /dev/nst0"
(or whatever equivalent syntax your shell uses for setting environment
variables).

$TAPE is used by commands like "mt", "tar", "dump", "restore", etc as
a default source/destination for I/O operations.  So if you just type
"tar tv" you'll get a tar listing from /dev/nst0 -- saves lots of typing!
