Ah, if you want to fix your Mickeysofts Windoom partition:

0. you have to fix you Win partition before fixing LILO.

1. boot with a Win95/98 Boot Disk (DOS Boot will not work if you use the damn FAT32). 

2. type the following command:

     FDISK /MBR

of course the FDISK.EXE must be in the Win9x Boot Disk.

YOU HAVE TO FIX THE WINDOOM PARTITION BEFORE FIXING LILO, 'cause FDISK /MBR destroys LILO information, and you'll have to fix it again.
