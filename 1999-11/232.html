Re: Braodcasting software for Linux

The page:

http://graphicswiz.com/png/pngapbr.html

Gives a list of PNG-capable browsers.
And a big list of PNG links.
The PNG home page 
http://graphicswiz.com/png/png.html
has heaps of PNG examples, which all have MIME type image/png.

My browser (Communicator 4.61) shows all PNG files with a black
background. In particular, a transparent background shows up as black,
which means that the alpha-layer feature is not much use
in this browser.
Worse than that, PNG doesn't even define animation.
So the GIF-to-PNG convertors generate a PNG file for
each frame of an animated GIF.

That's why it isn't practical to convert all GIFs to PNG files yet.
You might as well convert to JPEG instead, because at least
the vast majority of browsers will cope with that.
(Of course you can't do steganography with JPEGS.)

Because of the erroneous/incomplete showing of PNG files by most
browsers, I've been converting my GIF files for JPEG, except for
the animated ones, and a few transparent ones.
If the patent bounty hunters come after me, I'll just delete
all GIFs and convert as many as possible to JPEG.
