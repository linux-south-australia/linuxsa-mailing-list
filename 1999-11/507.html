Re: Mail server problem continued ...

On Wed, 24 Nov 1999, Justin Hennessy wrote:

> About a week ago I asked about a problem that I was having with my mail
> server. I now believe that I know that the problem is (having found out
> with a great deal of help from Kevin Hancock). The mail server is
> not relaying any mail. Is there a config file that allows me to configure
> this?
> 
> Of coruse I don't want to have my mail server relaying all mail, just the
> ones that are for it.

look in /etc/mail/access

and add a line like:

hostiwantrelayed     RELAY
