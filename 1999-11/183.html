RED HAT AND ORACLE TEAM TO IMPROVE LINUX
by Dave Murphy, member@itrain.org

Red Hat and Oracle announced a teaming arrangement
whereby Durham, NC-based Red Hat will update and
improve Linux to make it the premier e-commerce
platform operating system, and Oracle will help promote
the version. The new version of Red Hat Linux will be
optimized for Internet e-commerce business
applications.

Among Red Hats improvements, some of  which are
already under development:
  - ability to execute Java programs
  - larger memory address space
  - journaling file system (to help recover faster from a
crash)
  - support for Intel's upcoming 64-bit chips

Oracle and Red Hat will jointly promote the Oracle 8i
(O8i) database development applications.

As an aside, I just received my O8i CD this week, and
I'm looking forward to testing the development
environment on our corporate and web servers.

All of these modifications will be made under Gnu's
General Public License (GPL), the open-source license
which governs Linux distributions.
