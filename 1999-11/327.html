Corel have done it!

Hi!

I've just downloaded and installed Corel Linux. My first comment is it 
works! It is as easy as Windblows and as stabel as Linux (as we know it :-) 
). It hides everything it needs to hide from non technical users. This IS 
Linux ready for the desktop! They have stolen som desktop stuff from MAC 
too, ok by me as long as the kernel is Linux.

My experiense is mainly from administrating Win95 and NT. In Corel Linux it 
was no problem using my Windblows skills to configure it. Just as I want 
it. Easy graphical admin for the masses combined with the command line 
features for those who really want to get their hands dirty.

This distro might bring Linux one step further in breaking M$ dominance. (I 
hope, yet to belive...)
