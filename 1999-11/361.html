the dreaded gif -> png saga solved

Thanks to all the helpful suggestions. This list is truly a
knowledge-base made in heaven.

As was suggested earlier, the solution is at the web server end.
Modifying config to set mime type for .png files to image/png fixed the
problem. I am unsure how many web servers around the place might
encounter a similar need with happenings like 'world no-gif day', but
FWIW this instance is now solvered.
