Since I haven't seen a reply..
> To all Linux SA experts on setting up internet radio stations:
> Is linux a good choice (1) for an internet radio station(2)?

2. Applications are appearing.. 
   Have a look at gphone (gnome-o-phone).   
   http://www.math.okstate.edu/~droland/gphone/gphone.html

   This is in development and uses the
   standards in RFC1889 ('real time protocol' and 'real time control
   protocol', which uses UDP to transport media with reception
   reporting back to the source) and RFC1890 ('audio extention').  
   It also uses GSM (yes, same as mobile phones) compression, so don't
   expect great audio quality, but you could use another compression
   library in it's place (not sure what's out there though.)
   
   It reportably will talk to speakfreely, but I don't know about
   this application or it's license arrangements.

   The standards are designed for broadcast, so this would be a good
   candidate to turn into a radiostation type application.

RFC site: http://www.faqs.org/rfcs/index.html
