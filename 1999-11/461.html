What I would do is have only one card in the machine at the time, boot
using a DOS disk, and run the EEPROM programming program that came with
the cards to set up the IO port and IRQ. Write down the settings for that
card, then do the same for the other card, giving it different settings.

Then with both of the cards in the machine, you should be able to pass
init some paramaters at boot time to set both of them up. I'm not exactly
sure of the syntax. Someone else on the list should be able to help you.

For everyone else on the list, the Netgear EA201 cards pretty much run as
standard NE2K. Their website is at http://www.netgearinc.com/
