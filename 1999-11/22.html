Hi,

> NT itself has a bit of a problem going onto a partition bigger than 2 gigs
> initially anyway.
> So what you have hit is the installer of NT which is limiting the size.
> 
> The same problem occurs when installing NT directly without VMWare.

This isn't quite correct either. When you install NT for the first time on
a clean un-partitioned disk, what ever file system you chose, it will
partition and format a FAT file system, then convert to NTFS if you chose.
Under NT the max size of a FAT file system is 4 GB.

If you have a convenient computer already setup with NT, or have a program
like Ghost, then you can pre-partition and format the disk as an Up to 8
GB HDD, and install into that.

NT will not boot to a drive larger than 8 GB. I can't remember the max
NTFS partition size, but its something like 16 GB off the top of my head.
