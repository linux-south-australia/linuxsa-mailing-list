> This time the systems loads but stops at Starting sendmail: for a little

> services, Starting NMB Services, INND System.. for a very long time.

That's to do with name resolution..  it's probably been discussed a hundred
times on the list.  Basically, a solution is to add your machine's name and
IP to /etc/hosts.  To do this:

> When I left it for several hours, it ended up in a GUI with greenish blank
> screen and a cursor which did not give me an option of  left/right
clicking
> on anything.

You'll need to get out of that.  It sounds like it's started the X Window
System [: -- to kill it forcibly, you can usually use CTRL-ALT-BACKSPACE (as
opposed to CTRL-ALT-DELETE..), or you can use CTRL-ALT-F[1-6] to switch to
the other (text-mode) tty's.

Once you've done that, it's a trivial task to adjust /etc/hosts using vi or
emacs or joe or whatever.
