Some thoughts:

Name servers can return different IP addresses for the same domain name
(assuming that a particular name is configured with multiple addresses),
and this has become a common way of load balancing.  There could be
multiple servers with the same name but with different web content.

Browsers can indicate language preference.  I didn't notice whether you
said that you got these odd results on a single machine, or from different
machines.  Obviously if it's the latter case I would suggest you look at
your language preferences.
